Вход в систему с одноразовым кодом

Одноразовый код позволяет войти в систему без ввода имени пользователя и пароля. В настоящее время одноразовые коды доступны только некоторым пользователям.

Примечание: для использования этой функции адрес электронной почты на твоём аккаунте должен быть подтверждён. Ты можешь перейти на страницу с информацией об аккаунте‎, чтобы добавить и подтвердить свой адрес электронной почты (или адрес электронной почты родителей для пользователей младше 13 лет). Эта справочная статья содержит информацию о том, как подтвердить адрес электронной почты аккаунта.

Как войти в систему с помощью одноразового кода

  1. Нажми кнопку Получить одноразовый код по эл. почте на странице входа.
  2. Введи адрес электронной почты, подтверждённый на твоём аккаунте. Мы вышлем шестизначный одноразовый код по электронной почте.
  3. Введи одноразовый код для входа в систему (если твоя электронная почта подтверждена на нескольких аккаунтах, ты можешь выбрать один из них).

Одноразовый код не пришёл

Попробуй выполнить следующие действия, если письмо с одноразовым кодом не пришло:

  • Проверь папку со спамом.
  • Нажми кнопку «‎Отправить повторно»‎, а затем снова проверь входящие сообщения.
  • Убедись, что адрес электронной почты совпадает с адресом, указанным в аккаунте.
  • Убедись, что адрес электронной почты подтверждён. Эта справочная статья содержит информацию о том, как подтвердить адрес электронной почты аккаунта.

Если ни один из этих шагов не сработал, войди в систему, используя свой пароль. Возможно, придётся подождать некоторое время, прежде чем одноразовые коды снова начнут работать.

Полученный одноразовый код не работает

Попробуй выполнить следующие действия, если одноразовый код не сработал:

  1. Убедись, что ты вводишь одноразовый код на том же устройстве, с которого он был запрошен.
  2. Нажми кнопку Переслать и повтори попытку, используя новый код.

Если ни один из этих шагов не сработал, войди в систему, используя свой пароль. Возможно, придётся подождать некоторое время, прежде чем одноразовые коды снова начнут работать.

Рекомендации по безопасности

  • Никогда не сообщай никому одноразовый код. Roblox никогда не запрашивает у пользователей эту информацию.
  • Если тебе пришло электронное письмо без твоего запроса, можешь смело его игнорировать.
  • Если тебе пришло несколько нежелательных писем, содержащих одноразовые коды, мы настоятельно рекомендуем сменить электронную почту на твоём аккаунте и установить двухшаговую проверку с помощью приложения-аутентификатора .
