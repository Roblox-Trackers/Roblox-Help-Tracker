Добавь двухшаговую проверку в свой аккаунт

Двухшаговая проверка - это дополнительный уровень безопасности для твоего аккаунта. Это особенно важно для защиты от злоумышленников, которые не смогут получить доступ к твоему аккаунту в случае кражи пароля.

Методы двухшаговой проверки

Roblox предлагает различные методы двухшаговой проверки. На данный момент ты можешь выбрать только один метод двухшаговой проверки за раз.

Чтобы включить двухшаговую проверку на твоём аккаунте:

  1. Войди в свой аккаунт Roblox
    • Если ты не можешь войти в систему, попробуй выполнить следующие действия для восстановления пароля
  2. Перейди в Настройки аккаунта
    • Браузер : Найди значок шестерёнки в правом верхнем углу сайта
    • Мобильное приложение : Найди значок с тремя точками и нажми на значок шестерёнки
  3. Открой вкладку Безопасность
  4. Включи тумблер для желаемого метода двухшаговой проверки и следуй инструкциям на экране. Инструкция зависит от выбранного тобой варианта двухшаговой проверки
    • ВАЖНО : Чтобы включить двухшаговую проверку, на твоём аккаунте должен быть подтверждённый адрес электронной почты

Настройка двухшаговой проверки по адресу электронной почты

Если ты включишь функцию двухшаговой проверки по электронной почте, тебе будет предложено ввести код каждый раз, когда ты будешь входить в систему, используя свой пароль.

Код можно найти во входящих сообщениях, подтвержденной электронной почты, указанной на твоём аккаунте Roblox. Как только ты введёшь код на экране Roblox, ты успешно войдёшь в игру.

Обрати внимание, что при входе с помощью OTP или ключа электронной почты - двухшаговая проверка будет пройдена.

Настройка двухшаговой проверки с помощью приложения-аутентификатора

Чтобы включить двухшаговую проверку с помощью приложения-аутентификатора, на твоём мобильном устройстве должно быть установлено приложение аутентификатор. Если у тебя нет приложения аутентификатор, ты можешь загрузить его из App Store. Ты можешь выбрать такие приложения, как Google Authenticator, Microsoft Authenticator, Duo, Twilio Authy.

После включения функции двухшаговой проверки с помощью приложения Аутентификатор, тебе будет предложено ввести код каждый раз, когда ты будешь входить в систему с паролем или логином с помощью Email OTP.

Код можно найти в выбранном тобой приложении аутентификатор на твоём мобильном устройстве. Как только ты введёшь код на экране Roblox, ты успешно войдёшь в игру.

Обрати внимание, что при входе с помощью OTP или ключа электронной почты - двухшаговая проверка будет пройдена.

Настройка двухшаговой проверки с помощью ключей безопасности

Чтобы включить двухшаговую проверку с помощью ключей безопасности, в твоём аккаунте также должна быть включена двухшаговая проверка с помощью приложения Аутентификатор. Это связано с тем, что ключи безопасности поддерживаются только в веб-браузерах.

Ключ безопасности может быть физическим ключом, например Yubikey 5, Google Titan Security Key, или биометрическим методом, управляемым аппаратным обеспечением твоего устройства, например распознаванием лица или сканером отпечатков пальцев. Вариант ключа безопасности зависит от того, с какого браузера, операционной системы и устройства ты входишь в систему.

Обрати внимание, что при входе с помощью OTP или ключа электронной почты - двухшаговая проверка будет пройдена.

Двухшаговая проверка с помощью подсказок на разных устройствах

Примечание : Это экспериментальная функция, которая пока доступна не всем пользователям.

Как пользователь, ты не можешь напрямую включить двухшаговую проверку с помощью подсказок на разных устройствах. Ты увидишь эти подсказки, если на твоём аккаунте не включен ни один из перечисленных выше методов двухшаговой проверки, а также если Roblox обнаружит подозрительную попытку входа в твой аккаунт.

Если ты получишь запрос на двухшаговую проверку с помощью кросс-устройств, тебе будет предложено использовать мобильное устройство, с которого ты уже вошёл в свой аккаунт Roblox, чтобы одобрить или отклонить попытку входа. Ты сможешь просмотреть подробную информацию о попытке входа в систему, например тип устройства, региональное местоположение, IP-адрес, дату и время.

Выбор другого способа проверки попытки входа в систему

Если тебе будет предложено пройти двухшаговую проверку с помощью кросс-устройства, у тебя будет возможность подтвердить свой вход в систему другими способами.

Отправить код по электронной почте

Если в твоём аккаунте есть подтверждённый адрес электронной почты, ты увидишь опцию "Отправить код по электронной почте". При выборе этой опции на твой адрес электронной почты будет отправлен уникальный код. Введи этот код на экране Roblox, чтобы успешно войти в игру. Помни, не сообщай этот код никому.

Резервный код

У тебя также есть возможность ввести сохранённый резервный код, чтобы успешно войти в систему. Помни, не сообщай этот код никому.

Обратиться в службу поддержки

Если ты не видишь никаких опций, которые можно использовать для успешного входа в систему, обратись в службу поддержки .

Мы настоятельно рекомендуем добавить к твоему аккаунту подтверждённый адрес электронной почты. Это поможет тебе в случае, если ты захочешь подтвердить попытку входа в систему с помощью кода электронной почты, а не с помощью подсказки двухшаговой проверки кросс-устройства.

Попытка входа отклонена

Если ты получил двухшаговую проверку с помощью кросс-устройства, но не пытался войти в систему, отклони попытку входа.

Это означает, что кто-то может завладеть твоим паролем и использовать его для входа в твой аккаунт. Мы рекомендуем изменить пароль , чтобы помочь защитить аккаунт.

Попытка входа истекла

Если ты будешь ждать слишком долго, чтобы одобрить или отклонить попытку входа, она истечёт. Просто повтори попытку входа в систему.

Получение резервных кодов

  1. Войди в свой аккаунт.
  2. Перейди в Настройки .
    • Веб-браузер : Найти значок шестерёнки, расположенный в правом верхнем углу сайта.
    • Мобильное приложение : Найти значок с тремя точками Подробнее .
  3. Выбери вкладку Безопасность .
  4. Нажми Генерировать в разделе коды восстановления.
    • Веб-браузер : Находится в правой части страницы
    • Мобильное приложение : Находится в разделе коды восстановления
  5. Введи пароль твоего аккаунта в процессе настройки.
  6. Обязательно сохрани коды в надежном месте, потому что ты сможешь увидеть их только один раз.

Каждый код может быть использован только один раз. Если ты потерял коды восстановления, можешь повторить этот процесс для создания новых кодов. Любые ранее созданные коды восстановления будут деактивированы при создании новых кодов.
