Как добавить или удалить PIN-код?

У тебя есть возможность добавить четырёхзначный PIN-код для блокировки настроек. После добавления PIN-кода он будет необходим всегда для внесения любых изменений в настройки аккаунта. Эта функция защищает твой пароль, электронный адрес и настройки конфиденциальности.

Как добавить PIN-код?

  1. Войди в свой аккаунт
    (Если ты не можешь войти, попробуй выполнить следующие действия для восстановления пароля .)
  2. Выбери Настройки
    • В браузере - найди значок шестерёнки, расположенный в правом верхнем углу сайта
    • В мобильном приложении - найди значок с тремя точками и нажми Дополнительно
  3. Выбери вкладку Родительский Контроль
  4. Кнопка переключения для активации PIN-кода
  5. Создай и подтверди новый PIN-код

Никогда никому не сообщай свой PIN-код аккаунта - так как он важнее, чем твой пароль.

Как удалить PIN-код?

  1. Войди в свой аккаунт
    (Если ты не можешь войти, попробуй выполнить следующие действия для восстановления пароля .)
  2. Выбери Настройки
    • В браузере - найди значок шестерёнки, расположенный в правом верхнем углу сайта
    • Мобильные приложения - найди значок с тремя точками Подробнее
  3. Выбери вкладку Родительский контроль
  4. Кнопка переключения для отключения PIN-кода
  5. Введи текущий PIN-код и выбери Разблокировать

Как изменить PIN-код?

Чтобы изменить PIN-код, сначала выполни шаги Как удалить PIN-код . Затем следуй шагам Как добавить PIN-код для создания нового PIN-кода.
