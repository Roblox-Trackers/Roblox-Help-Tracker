Как добавить или изменить свой номер телефона?

Эта функция предназначена для игроков старше 13 лет. Телефонный номер можно использовать только в одном аккаунте Roblox.

Как добавить номер телефона?

  1. Войди в свой аккаунт
    (Если ты не можешь войти, попробуй выполнить следующие действия для восстановления пароля .)
  2. Перейди в раздел Настройки
    • Браузер - найди значок шестерёнки, расположенный в правом верхнем углу сайта
    • В мобильном приложении - найди значок с тремя точками для Дополнительно
  3. Выбери вкладку Информация аккаунта
  4. Нажми кнопку Добавить телефон
    1. Выбери твоё местоположение из выпадающего меню
    2. Введи номер телефона и пароль аккаунта
    3. Выбери Добавить телефон

Как изменить или удалить номер телефона?

  1. Войди в свой аккаунт
    (Если ты не можешь войти, попробуй выполнить следующие действия для восстановления пароля .)
  2. Перейди в раздел Настройки
    • Браузер - найди значок шестерёнки, расположенный в правом верхнем углу сайта
    • В мобильном приложении - найди значок с тремя точками для Дополнительно
  3. Выберите вкладку Информация аккаунта
  4. Нажми кнопку Обновить телефон (значок выглядит как карандаш и бумага)
    1. Выбери твоё местоположение из выпадающего меню
    2. Введи номер телефона и пароль аккаунта
    3. Выбери опцию Изменить телефон или Удалить номер телефона

Команды для текстовых сообщений

Вот несколько полезных команд для запуска и остановки текстовых сообщений, которые ты можешь получить от Roblox:

  • STOP, STOPALL, UNSUBSCRIBE, CANCEL, END и QUIT
    Эти однословные команды остановят будущие текстовые сообщения. Обрати внимание, что только однословные сообщения будут вызывать блокировку. Например, ответ «STOP» гарантирует, что ты больше не будешь получать сообщений, а ответ «STOP PLEASE» не будет.
  • START, YES и UNSTOP
    Эти команды позволят тебе вернуться к получению текстовых сообщений.
  • HELP и INFO
    Эти команды предоставят информацию, которую ты можешь использовать вышеупомянутыми командами для управления доставкой сообщений.
