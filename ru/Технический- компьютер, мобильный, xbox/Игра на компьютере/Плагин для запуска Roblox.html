Плагин для запуска Roblox

Если Roblox не может запуститься или выдает приглашение загрузить Roblox, даже если он уже установлен. Вы должны проверить, разрешен ли плагин Roblox в выбранном вами браузере вместе с Internet Explorer, поскольку Roblox использует Internet Explorer для выполнения некоторых своих функций. Для проверки выполните следующее:

Microsoft Edge

Примечание: Microsoft Edge не поддерживает плагины

Internet Explorer

  1. Нажмите на значок шестеренки, расположенный в правом верхнем углу.
  2. Нажмите Управление надстройками.
  3. Во всплывающем окне, расположенном слева под Показать, нажмите на выпадающее меню.
  4. Из выпадающего меню нажмите выбрать Все надстройки.
  5. В правой части всплывающего окна переместите полосу прокрутки вниз.
  6. В Roblox Corporation убедитесь, что Launcher Class включен.

Mozilla Firefox

  1. Нажмите кнопку меню в правом верхнем углу вашего окна Firefox.
  2. В меню нажмите кнопку Надстройки.
  3. В появившемся окне нажмите Плагины.
  4. Найдите Roblox Launcher Plugin и выберите в выпадающем меню Всегда разрешать .
  5. Закройте вкладку Менеджер надстроек и запустите игру Roblox.

Google Chrome

  1. В адресной строке введите: chrome://plugins/
  2. Нажмите Enter.
  3. Убедитесь, что там указан плагин Roblox Launcher Plugin.

Примечание: Начиная с версии 42, Chrome больше не поддерживает плагины. Эти шаги будут работать только в старых версиях Chrome.

Apple Safari

  1. Откройте Safari и щелкните значок шестеренки в правом верхнем углу.
  2. Нажмите Настройки, затем вкладку Безопасность.
  3. Убедитесь, что подключаемые модули, Java и JavaScript включены.
