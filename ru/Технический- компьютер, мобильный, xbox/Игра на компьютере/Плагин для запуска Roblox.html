Плагин для запуска Roblox

Если Roblox не может запуститься или выдает запрос на загрузку Roblox, хотя он уже установлен. Чтобы проверить, выполни следующее:

Mozilla Firefox

  1. Нажми кнопку меню в правом верхнем углу вашего окна Firefox
  2. В меню нажми кнопку Надстройки
  3. В появившемся окне нажмите Плагины
  4. Найди Roblox Launcher Plugin и выбери в выпадающем меню Всегда разрешать
  5. Закрой вкладку Менеджер надстроек и запусти Roblox

Apple Safari

  1. Открой Safari и нажми значок шестерёнки в правом верхнем углу
  2. Нажми Настройки, затем вкладку Безопасность
  3. Убедись, что подключаемые модули JavaScript включены

Google Chrome

Примечание: Начиная с версии 42, Chrome больше не поддерживает плагины.

Microsoft Edge

Примечание: Microsoft Edge не поддерживает плагины
