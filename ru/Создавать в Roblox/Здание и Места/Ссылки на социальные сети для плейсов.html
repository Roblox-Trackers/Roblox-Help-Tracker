Ссылки на социальные сети для плейсов

Разработчики могут пожелать связать свои другие профили в социальных сетях непосредственно со своим плейсом. Roblox теперь предлагает официальную поддержку для этого через функцию Social Media Link! Разработчики смогут включать до трёх ссылок на социальные сети в свои описания плейсов, аналогично методу, используемому в профиле пользователя. Каждый тип социальных сетей может быть добавлен только один раз.

Вот ссылки на социальные сети, которые можно добавить к плейсу:

  • Discord
  • Youtube
  • Twitter
  • Twitch
  • Facebook
  • Группа Roblox

Как добавить ссылки на социальные сети в плейс:

  1. Войди в свой аккаунт
  2. Перейди к странице Создать
  3. Нажми на плейс
  4. Нажми на кнопку Социальные ссылки из списка меню слева
  5. Введи URL адрес, выбери тип медии и введи заглавие
  6. Нажми кнопку Сохранить
  7. Как только ссылки будут сохранены,  они будут отображаться на странице плейса таким образом.

Важные заметки:

  • Ссылки на социальные сети будут отображаться только пользователям аккаунтов достигших 13 лет. Игрокам младше 13 лет, такие данные не будут доступны.
  • Хотя ссылки на Discord  разрешены в рамках функции ссылок на социальные сети, но всё же, пользователям не разрешается обсуждать или делиться ссылками на Discord где-либо ещё.
