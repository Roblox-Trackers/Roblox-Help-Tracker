Ссылки на социальные сети для опытов

Теперь разработчики опытов могут связать свои профиль в социальных сетях с ним, так как Roblox  предлагает официальную поддержку для этого через ссылку в социальные сети! Разработчики смогут включать до трех ссылок на социальные сети в свои описания опытов, аналогично методу, используемому в профиле пользователя. Каждый тип социальных сетей может быть добавлен только один раз.

Вот ссылки на социальные сети, которые можно добавить к опыту:

  • Discord
  • Youtube
  • Twitter
  • Twitch
  • Facebook
  • Группа Roblox

Как добавить ссылки на социальные сети в опыт:

  1. Войти в свою учетную запись;
  2. Перейти к странице Создать;
  3. Нажать на опыт;
  4. Нажать на кнопку Социальные ссылки из списка меню слева;
  5. Ввести URL адрес, выбрать  тип медии и ввести заглавие;
  6. Нажать кнопку Сохранить;

Важные заметки:

  • Ссылки на социальные сети будут отображаться только пользователям учетных записей достигших 13 лет. Игрокам младше 13 лет, такие данные не будут доступны.
  • Хотя ссылки на Discord  разрешены в рамках функции ссылок на социальные сети, но все же, пользователям не разрешается обсуждать или делиться ссылками на Discord где-либо еще. Такие учетные записи будут подлежать действиям модерации, если Discord будет упомянут в другом месте.
