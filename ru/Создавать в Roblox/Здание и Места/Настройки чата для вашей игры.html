Настройки чата для вашей игры

Ваша игра может быть настроена так, чтобы отображать сообщения в игре. Это позволяет легко настраивать параметры чата, игрокам во время игры.

Режимы чата

У вас есть два стиля чата в игре, по умолчанию:

Классический

Он отображает сообщения чата в верхнем левом окне экрана игры по умолчанию. Эти сообщения перечислены в хронологическом порядке, самые новые и самые старые, снизу вверх. Это позволяет игрокам общаться без каких-либо ограничений физического местоположения в мире вашей игры.

Выноска (облачко с текстом)

Она отображет сообщения чата над головами персонажей игроков, которые разговаривают. Это позволяет ограничивать разговоры физическим местоположением игроков на месте вашей игры.

При желании оба режима чата могут быть включены одновременно.

Настройки режима чата

По умолчанию включен только классический режим чата. Чтобы настроить какой режим включить, пожалуйста, прочитайте нашу статью для разработчиков о Чат-системе Roblox.
