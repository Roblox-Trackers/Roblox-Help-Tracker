Требования к компьютерному оборудованию и операционной системе

Минимальные требования для Roblox к компьютерному оборудованию и операционной системе перечислены ниже. Если у вас возникли затруднения с игрой, скорее всего, вам необходимо проверить отвечает ли требования вашего компьютера необходимым требованиям. Прочитайте нашу справочную статью о системных требованиях для мобильных устройств.

Требования к операционной системе:

  • ПК с ОС Windows: Roblox клинет может быть установлен на Windows 7, Windows 8 / 8.1 или Windows 10. Для Windows 8 / 8.1 вам потребуется запустить Roblox на рабочем столе, поскольку Metro Mode (мозаичный стартовый экран) в настоящее время не поддерживается.

Примечание: Для Windows 7 браузер должен быть IE10 или выше.

  • Mac : Roblox клинет может быть установлен на 10.7 (Lion) и выше - и Intel только на Mac, тогда как Roblox Studio устанавливается на Mac OS 10.11 (El Capitan) и выше.
  • Linux: Roblox не поддерживается на ОС Linux.
  • Мобильное устройство: Нажмите здесь, чтобы ознакомиться с системными требованиями для Roblox Mobile.
  • ОС Chrome: требуется Chrome OS версии 53 или выше, и вы должны сначала включить игровой магазин Google Play на вашем устройстве.

Примечание: Функция видеозаписи Roblox работает только в Windows 7, Windows 8 / 8.1 и Windows 10. Недавно выпущенные операционные системы или предварительно выпущенные версии операционных систем могут не поддерживаться сразу.

Системные требования к оборудованию:

  • Видеокарта : для Roblox требуется минимум DirectX 9 и Shader Модели 2.0. Для достижения максимальной производительности мы рекомендуем использовать компьютер младше 5 лет с отдельной видеокартой или ноутбук младше 3 лет со встроенной видеокартой.
  • Процессор: Roblox рекомендует использовать новейший процессор (2005+) с тактовой частотой 1,6 ГГц или лучше. Были некоторые проблемы со старыми процессорами AMD.
  • RAM или Память : Roblox рекомендует иметь как минимум 1 ГБ памяти в Windows 7, Windows 8 или Windows 10.
  • Место для хранения : Roblox рекомендует иметь не менее 20 МБ системного дискового пространства для установки Roblox.
  • Мобильное устройство: Нажмите здесь, чтобы ознакомиться с системными требованиями для Roblox Mobile.

Интернет требования

Если выявляетесь единственным пользователем интернета, то Roblox рекомендует подключение к Интернету со скоростью не менее 4-8 Мбит/с. Если есть несколько дополнительных пользователей, рекомендуется увеличить пропускную способность.
