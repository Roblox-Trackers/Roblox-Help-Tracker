Требования к компьютерному оборудованию и операционной системе

Минимальные требования Roblox к компьютерному оборудованию и операционной системе перечислены ниже. Если у тебя возникли проблемы, возможно, тебе необходимо обновить свой компьютер. Прочитай эту справочную статью о системных требованиях для мобильных устройств.

Требования к операционной системе:

  • PC/Windows: Roblox можно установить на 64-битные версии Windows 7, Windows 8.1 и Windows 10. Для Windows 8.1 придётся запускать Roblox в режиме рабочего стола, поскольку режим Metro (начальный экран с плитками) в настоящее время не поддерживается.
    Примечание. Веб-сайт Roblox поддерживает следующие браузеры для PC/Windows: Chrome , Firefox и Microsoft Edge .
  • Mac: клиент Roblox устанавливается на 10.10 (Yosemite) и выше, а Roblox Studio — на Mac OS 10.13 (High Sierra) и выше.
    Примечание. Веб-сайт Roblox поддерживает следующие браузеры для Mac: Chrome , Firefox и Safari .
  • Linux: Roblox не поддерживается на ОС Linux.
  • Мобильные устройства: нажми здесь для системных требований Roblox для мобильных устройств.
  • ОС Chrome: требуется Chrome OS версии 53 или выше, и сначала необходимо включить магазин Google Play на своём устройстве.

Примечание. Функция видеозаписи Roblox работает только в Windows 7, Windows 8.1 и Windows 10. Недавно выпущенные операционные системы или предварительно выпущенные версии операционных систем возможно будут поддерживаться не сразу.

Системные требования к оборудованию

  • Видеокарта : на PC/Windows приложение Roblox требует поддержки DirectX 10 или более высокого уровня функциональности. Для наилучшей производительности мы рекомендуем либо компьютер возрастом менее 5 лет с выделенной видеокартой, либо ноутбук возрастом менее 3 лет с интегрированной видеокартой.
  • Процессор: Roblox рекомендует тебе иметь современный процессор (2005+) с тактовой частотой 1,6 ГГц или выше. С более старыми процессорами AMD могут возникнуть ошибки.
  • RAM или Память: Roblox рекомендует иметь не менее 1 Гб памяти в Windows 7, Windows 8.1 или Windows 10.
  • Хранилище : Roblox рекомендует иметь не менее 20 МБ системного дискового пространства для установки Roblox.
  • Мобильное приложение: нажми здесь для системных требований для мобильного приложения Roblox.

Требования к интернет-подключению

Для использования Roblox на одном устройстве рекомендуется интернет-соединение со скоростью не менее 4-8 Мб/с. При использовании одновременно несколькими людьми целесообразно использовать большую пропускную способность. В больших семьях целесообразно использовать большую пропускную способность.
