Требования к компьютерному оборудованию и операционной системе

Минимальные требования для Roblox к компьютерному оборудованию и операционной системе перечислены ниже. Если у тебя возникли проблемы с игрой, возможно, тебе необходимо обновить свой компьютер. Прочитай эту справочную статью о системных требованиях для мобильных устройств.

Требования к операционной системе:

  • ПК с ОС Windows: Roblox устанавливается на Windows 7, Windows 8/8.1 или Windows 10. Для Windows 8/8.1 вам потребуется запустить Roblox в режиме рабочего стола, поскольку режим Metro (начальный экран с панелями) в настоящее время не поддерживается.
  • Mac : Roblox Client устанавливается на 10.10 (Yosemite) и выше, а Roblox Studio - на Mac OS 10.11 (El Capitan) и выше.
  • Linux: Roblox не поддерживается на ОС Linux.
  • Мобильная: Нажми здесь для системных требований для Roblox Mobile.
  • ОС Chrome: требуется Chrome OS версии 53 или выше, и ты должен сначала включить игровой магазин Google Play на своём устройстве.

Примечание: Функция видеозаписи Roblox работает только в Windows 7, Windows 8 / 8.1 и Windows 10. Недавно выпущенные операционные системы или предварительно выпущенные версии операционных систем могут не поддерживаться сразу.

Системные требования к оборудованию

  • Видеокарта : На PC/Windows приложение Roblox требует поддержки DirectX 10 или более высокого уровня функциональности. Для наилучшей производительности мы рекомендуем либо компьютер возрастом менее 5 лет с выделенной видеокартой, либо ноутбук возрастом менее 3 лет с интегрированной видеокартой.
  • Процессор: Roblox рекомендует тебе иметь современный процессор (2005+) с тактовой частотой 1,6 ГГц или выше. Существуют некоторые проблемы с более старыми процессорами AMD.
  • RAM или Память : Roblox рекомендует иметь не менее 1 Гб памяти в Windows 7, Windows 8 или Windows 10.
  • Место для хранения : Roblox рекомендует иметь не менее 20 МБ системного дискового пространства для установки Roblox.
  • Мобильный: Нажми здесь для системных требований для Roblox Mobile.

Интернет требования

Для одноразового домашнего хозяйства Roblox рекомендует интернет-соединение не менее 4-8 Мб/с. В больших семьях целесообразно использовать большую пропускную способность.
