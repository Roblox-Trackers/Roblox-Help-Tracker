File audio

Roblox offre agli sviluppatori molti modi per rendere i loro giochi coinvolgenti e dinamici. Uno di questi modi è attraverso l'uso dell'audio che i giocatori possono aggiungere alle proprie esperienze come musica di sottofondo, narrazione, effetti sonori e altro ancora!

Nota: tutto l'audio caricato su Roblox deve soddisfare i nostri Standard della Community Roblox .

In quale formato posso caricare?

I giocatori possono caricare file mp3 o OGG . Tieni presente che il tentativo di caricare in qualsiasi altro formato comporta il rischio di un errore in cui il file non funzionerà correttamente.

Come caricare un file audio

Il caricamento può essere effettuato facilmente tramite il sito Web roblox.com. Per fare questo:

  1. Accedi al tuo account su roblox.com
  2. Clicca Crea sulla barra della parte superiore dello schermo
  3. Clicca su Audio
  4. Clicca Sfoglia e seleziona un file audio
  5. Una volta selezionato, fare clic sul pulsante "Carica".

Note: I caricamenti audio devono essere inferiori a 7 minuti, più piccoli di 20,0 MB, chiari e di facile comprensione. Ciò significa che l'audio non può essere accelerato, rallentato o avere un volume eccessivamente alterato.

Limite di caricamento file audio

  • Le soglie del limite di caricamento differiranno tra gli account. Il conteggio dei caricamenti può essere visualizzato nel banner verde della pagina di caricamento dell'audio. Per maggiori informazioni, clicca qui .

Configurare l'audio

Una volta caricato, il file audio può essere ulteriormente personalizzato nella pagina Configura. Per individuare questa pagina, fai clic sull'icona a forma di ingranaggio situata a destra del file caricato nella scheda Audio della sezione Crea.

La pagina Configura ha le seguenti opzioni:

  • Nome: consente di modificare il nome del file audio,
  • Descrizione: un riepilogo del file audio,
  • Attiva/disattiva commenti: scegli se questo articolo è aperto o meno ai commenti,
  • Genere: classifica l'audio per aiutare le persone a trovarlo,
  • Consenti copia: scegli se questo elemento è disponibile gratuitamente per tutti gli utenti. Se questa opzione è deselezionata, l'audio sarà disponibile solo per te,
  • Archivia: l'archiviazione dell'audio ne impedirà l'utilizzo in un'esperienza. Le risorse archiviate possono essere ripristinate.

Nota: i nomi e le descrizioni sono soggetti ai nostri filtri consenti/blocca. Per ulteriori informazioni su questi filtri, clicca qui .
