Come acquistare e configurare i server VIP?

Se il creatore ha abilitato i server privati sulla propria esperienza, puoi utilizzare Robux per acquistare un server privato che puoi goderti con gli amici che inviti.

Come posso creare e modificare il mio server?

Come posso crearne uno?

  • Fai clic sulla scheda Server nella pagina dei dettagli dell'esperienza.
  • Se questa funzione è stata attivata, vedrai una sezione intitolata Server Privati . Mostrerà tutti i server che hai creato o di cui fai parte. Se ne vedi qualcuno, puoi andare avanti e giocare subito!
  • Per crearne uno nuovo, fai clic sul pulsante Crea Server Privato .
  • Dai un nome al tuo nuovo server. Leggi il contratto di abbonamento. Dopo aver letto tutte le informazioni nel box e aver scelto il nome perfetto, puoi completare l'acquisto facendo clic su Acquista Ora .
  • Una volta acquistato, potrai configurare il server subito o in un secondo momento.

Come configuro il mio server VIP?

  • Puoi accedere alla pagina di configurazione dalla finestra di notifica dell'acquisto o selezionando Configura dal pulsante Menu (che appare come tre punti) per il server nella scheda Server .
  • Puoi rinominare il server, stabilire chi può giocare al suo interno, inclusi fino a 50 utenti singoli o consentire al tuo elenco di amici di partecipare, e persino creare un collegamento al tuo server.
  • Puoi anche attivare e disattivare l'accesso al server e annullare l'abbonamento ad esso.

Come funzionano gli Abbonamenti al Server e quanto costano?

Quanto costano gli abbonamenti ai server privati?

  • I prezzi del server privato sono stabiliti dallo sviluppatore dell'esperienza.
  • Ti verrà addebitato al momento dell'acquisto e nuovamente ogni mese per mantenerlo attivo. La quota di rinnovo mensile verrà prelevata automaticamente dal tuo saldo Robux, quindi non devi preoccuparti di dimenticare di rinnovare ogni mese.
  • Se non desideri che il tuo server si rinnovi, puoi farlo in qualsiasi momento prima della data di rinnovo. Per cancellare il tuo server VIP, leggi la sezione Come posso cancellare il mio server VIP di seguito.

Come posso cancellare il mio server VIP?

  • Passa al server per l'esperienza. Clicca sul pulsante Menu (che appare come tre punti) all'estrema destra della sezione,
  • Clicca su Configura,
  • Per annullare la tariffa mensile, clicca sull'icona verde Attiva accanto a Stato abbonamento,
  • Clicca su Annulla pagamenti nella finestra di conferma per la cancellazione dell'abbonamento.

    Nota: potrai comunque utilizzare il tuo server per il resto del mese per il quale hai già pagato, ma scadrà alla fine di questo periodo.

    Cosa succede se non ho abbastanza Robux alla data di rinnovo?

    Se ciò accade, il server scadrà e non sarà più utilizzabile, ma puoi acquistare un nuovo server in qualsiasi momento.

    Come posso rendere inattivo il mio server?

  • Passa al tuo server privato per l'esperienza. Fai clic sul pulsante Menu (che appare come tre punti) all'estrema destra della sezione,
  • Fai clic su Configura ,
  • Per rendere temporaneamente inattivo il server per un periodo di tempo, fare clic sull'icona verde Attivo accanto a Stato del server VIP ,
  • Fare clic su Disabilita nella finestra di conferma della disattivazione del server,
  • Puoi fare nuovamente clic per riattivarlo in qualsiasi momento. Tuttavia, è importante tenere presente che il periodo di rinnovo non verrà esteso per questo periodo di inattività e si rinnoverà o scadrà comunque alla data originariamente programmata.
