Come acquistare e configurare i server VIP?

Se il creatore del gioco ha abilitato i server VIP sul proprio gioco, puoi utilizzare Robux per acquistare un server privato che puoi divertirti con gli amici che inviti.

Come posso creare e modificare il mio server VIP?

Come posso crearne uno?

Quando visualizzi la pagina dei dettagli di un gioco, clicca sulla scheda Server . Se questa funzione VIP è stata attivata, verrà visualizzata una sezione intitolata Server VIP . Qui puoi trovare qualsiasi server privato che hai creato o di cui fai parte. Se ne vedi, puoi andare avanti e gioca subito! Se desideri crearne uno nuovo, puoi farlo cliccando il pulsante blu Crea server VIP .

Quando hai scelto di creare un nuovo server, dovresti vedere una finestra che appare come l'immagine qui sotto. Assegnerai al tuo nuovo server un nome personalizzato e ripasserai il contratto di abbonamento. Dopo aver letto tutte le informazioni su la scatola e scelto il nome perfetto, è possibile completare l'acquisto cliccando Acquista ora .

Come configuro il mio server VIP?

Dopo aver acquistato un server VIP, puoi scegliere di visualizzare la configurazione del tuo server schermo o no.

Nella schermata di configurazione del server, è possibile rinominare il server e invitare specifici utenti a partecipare inserendo i loro nomi utente nell'apposita casella e clicca Aggiungi. Puoi invitare fino a 50 singoli utenti oppure puoi consentire a tutti nell'elenco amici per unirti selezionando Amici consentiti scatola.

Questa pagina include anche una funzione per generare un collegamento che è possibile utilizzare direttamente accedi al tuo server specifico. Per fare ciò, clicca sul pulsante blu Genera . Dopo aver configurato tutto come vuoi, clicca su Salva per iniziare a usarlo!

Una volta che tutto è stato impostato, il tuo nuovo acquisto dovrebbe ora apparire nel privato Sezione Server della pagina dei dettagli del gioco. Per giocare, clicca sul pulsante Partecipa . Se desideri apportare ulteriori modifiche, clicca sul pulsante menu (sembra tre punti) nell'angolo in alto a destra del server casella e selezionare Configura .

Come funzionano gli abbonamenti al server privato e quanto costano?

Quanto costano i server privati?

I prezzi del server privato sono stabiliti dallo sviluppatore del gioco. Questo sarà addebitato quando inizialmente effettui l'acquisto e ti verrà addebitato nuovamente ogni mese per conservarlo è attivo. La tassa di rinnovo mensile verrà automaticamente prelevata dal tuo saldo Robux, quindi non devi preoccuparti di dimenticare di rinnovare ogni mese.

Se non vuoi che il tuo server si rinnovi, puoi farlo in qualsiasi momento prima della data di rinnovo. Per cancellare il tuo server VIP, leggi il Come posso cancellare il mio server VIP di seguito.


Come posso cancellare il mio server VIP?

  • Passa al tuo server VIP per il gioco. Clicca sul pulsante Menu (sembra tre punti) all'estrema destra della sezione.
  • Clicca su Configura .
  • Per annullare la tariffa mensile, clicca sull'icona verde Attiva accanto a Stato abbonamento .
  • Dovrai cliccare su Annulla pagamenti nella conferma finestra per la cancellazione dell'abbonamento.

    Nota: dovrai comunque utilizzare il tuo server per il resto del file mese per il quale hai già pagato, ma scadrà alla fine di questo periodo.

    Cosa succede se non ho abbastanza Robux alla data di rinnovo?

    In questo caso, il server scadrà e non sarà più possibile usato, ma può essere riacquistato in qualsiasi momento.

    Come posso rendere inattivo il mio server?

    • Passa al tuo server VIP per il gioco. Clicca sul pulsante Menu (sembra tre punti) all'estrema destra della sezione.
    • Clicca su Configura .
    • Per rendere temporaneamente inattivo il server per un periodo di tempo, clicca sull'icona verde Attiva accanto a Stato server VIP .
    • Clicca su Disabilita nella finestra di conferma per il server da disattivare.
    • Puoi cliccare di nuovo per riattivarlo in qualsiasi momento. Tuttavia, è importante per tenere presente che il tempo di rinnovo non sarà prolungato per questo periodo di inattività e si rinnoverà o scadrà ancora sull'originale data programmata.
