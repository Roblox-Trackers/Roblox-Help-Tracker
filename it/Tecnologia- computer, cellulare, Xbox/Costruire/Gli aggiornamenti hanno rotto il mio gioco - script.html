Gli aggiornamenti hanno rotto il mio gioco / script

Quando effettuiamo aggiornamenti a Roblox, cerchiamo di assicurarci che gli script creati dal giocatore continuino a funzionare correttamente. Tuttavia, a volte questi script devono essere riscritti o modificati in modo che funzionino correttamente. Dai un'occhiata agli script per vedere dove potrebbe trovarsi il problema. Puoi anche consultare il Libro degli script Roblox per ulteriore aiuto.
