Problemi generali di Roblox Studio

Il seguente articolo suggerirà alcuni metodi che si sono dimostrati efficaci nel risolvere i seguenti problemi:

  • Si tenta di accedere a Roblox Studio ma non è possibile,
  • Si tenta di modificare un luogo solo per caricare uno skybox vuoto o qualcosa chiamato Place1,
  • Le sezioni di Studio sono vuote quando non dovrebbero esserlo,
  • Hai difficoltà a pubblicare su Roblox,
  • Roblox Studio ti dice di accedere anche se hai già acceduto,
  • Questi passaggi possono aiutare alcuni problemi non correlati a Studio come l'accesso a un gioco come Ospite anche quando hai effettuato l'accesso,
  • Altro: se riscontri un problema diverso con Studio, i passaggi seguenti potrebbero anche aiutarti a risolverlo.

Cosa fare - Windows

Prova ogni passaggio di seguito, uno alla volta, in ordine. Se una soluzione non funziona, passa alla successiva!

Riavvia

Per prima cosa riavvia il computer e prova a giocare. A volte questo può essere tutto ciò che è necessario per risolvere i problemi relativi a Roblox o Roblox Studio.

Controlla le tue impostazioni del proxy

Se i passaggi precedenti non hanno risolto il problema di Roblox Studio, è possibile che tu debba aver bisogno di assicurarti che il tuo computer non stia tentando di connettersi tramite un server proxy. Si prega di seguire questi passaggi:

Windows 7 o versioni precedenti

    1. Vai al menu Start,
    2. Digita "proxy" e apparir à "Configura Server Proxy" , clicca su di esso,
    3. Clicca su Impostazioni LAN e deseleziona tutti i segni di spunta in quella scheda,
    4. Clicca OK.

Windows 8 o 8.1

    1. Vai alla schermata iniziale di Windows facendo clic sull'angolo in basso a sinistra dello schermo
    2. Digita "proxy" e apparirà "Configura Proxy Server", cliccaci
    3. Clicca su Impostazioni LAN e deseleziona tutti i segni di spunta in quella scheda
    4. Clicca OK

Windows 10

    1. Premi il tasto Windows + S sulla tastiera. In alternativa, se la barra "Chiedi qualsiasi cosa" di Cortana è disponibile nella barra delle attività, è sufficiente cliccare su di essa
    2. Digita "proxy" e apparir à "Configura Server Proxy", cliccaci
    3. Assicurati che "Usa script di installazione" e "Usa un server proxy" siano disattivati
    4. Scorri verso il basso e fai clic su Salva

Controlla l'orologio di sistema di Windows

Può sembrare strano, ma diversi problemi possono effettivamente essere causati dall'orologio di sistema di Windows impostato sull'ora o sulla data sbagliata. Se questa è la causa, la semplice correzione di tutto dovrebbe porre rimedio alla situazione. Mentre questo può essere fatto manualmente, puoi anche impostarlo per l'aggiornamento automatico con un server, assicurandoti così che non rimanga indietro e ricominci a causare problemi. Ecco come fare:

Windows 8 o versioni precedenti

    1. Cliccare con il tasto destro sull'orologio di Windows nell'angolo in basso a destra,
    2. Clicca su Regola data/ora,
    3. Seleziona la scheda Orario Internet,
    4. Qui dovresti essere in grado di sincronizzare l'ora e la data con un server Internet.

Windows 10

    1. Clicca con il tasto destro sull'orologio di Windows nell'angolo in basso a destra
    2. Assicurarsi che "Imposta ora automaticamente" sia attivato e che sia stato selezionato il fuso orario corretto
