Come reinstallare il software Roblox?

Alcuni problemi tecnici possono essere risolti reinstallando Roblox. Assicurati di provare a riavviare prima di reinstallare. Ecco come reinstallare:

Utenti Windows:

Ripristina opzioni Internet per Windows

Il ripristino delle opzioni Internet ti aiuta nel caso in cui non siano sincronizzati con Roblox. Se questo risolve i tuoi problemi, è tutto ciò che devi fare. Mentre questi passaggi utilizzano Internet Explorer, risolveranno molti problemi che si verificano in altri browser come Chrome o Firefox. Ecco come:

    1. Apri Internet Explorer
    2. Apri l'icona a forma di ingranaggio > Opzioni Internet
    3. Apri la scheda Avanzate
    4. Fai clic sul pulsante Ripristina
    5. Chiudi il browser Internet Explorer
    6. Prova a utilizzare il tuo browser Web preferito per utilizzare Roblox.

Per reinstallare Roblox

Se il ripristino delle Opzioni Internet non è stato utile, prova a reinstallare Roblox. La reinstallazione dovrebbe sovrascrivere e sostituire qualsiasi file cancellato o danneggiato. Assicurarsi che tutti gli altri programmi siano chiusi quando si esegue la reinstallazione.

    • Reinstallazione per Chrome o Edge
    • Reinstallazione per Altri browser

Per disinstallare Roblox

Il più delle volte, non è necessario disinstallare il programma prima di reinstallarlo. Se la reinstallazione non funziona, potrebbe essere necessario disinstallare prima di reinstallare. Segui le istruzioni di disinstallazione di Roblox per Windows . Assicurati di riavviare il computer dopo la disinstallazione prima di reinstallare.

Cancella i tuoi file temporanei Internet

Un passaggio importante in qualsiasi processo di disinstallazione o reinstallazione è quello di cancellare i file temporanei di Internet del browser. Ciò aiuterà con eventuali problemi di corruzione che potrebbero essere sorti. Segui le istruzioni della sezione Cancella i tuoi file e cookie temporanei in Internet. Ripetere le istruzioni di reinstallazione di cui sopra al termine.

Modifica le impostazioni del tuo firewall/programma di sicurezza

Molti software antivirus o di sicurezza Internet cercano di controllare quali programmi possono accedere a Internet. Questi programmi, per impostazione predefinita, non sono autorizzati all'accesso. Leggi le informazioni nell'articolo della guida Problemi di firewall e router per assistenza su come consentire l'accesso a Roblox. Ripetere i passaggi di reinstallazione dopo aver riconfigurato il firewall.

Utenti Mac:

Per disinstallare

Assicurati di essere completamente uscito dal programma Roblox. Segui le istruzioni per la disinstallazione di Roblox per Mac .

Per reinstallare

Apri Safari, accedi a Roblox e gioca. Ti verrà richiesto di installare l'applicazione. Si prega di fare questo seguendo tutte le istruzioni sullo schermo. Se l'installazione non avviene automaticamente, eseguire le seguenti operazioni:

    1. Vai su Trova > Oggi e clicca su Roblox.dmg - questo dovrebbe eseguire il programma di installazione per te e posizionare l'icona Roblox nelle tue app.
    2. Selezionalo dalle tue App e dovrebbe andare nel tuo tray
