Reimpostazione dell'orologio o impossibile ottenere l'URL: Httpsendrequest failed,err=0x2f05

Vari problemi in diversi programmi o siti Web possono effettivamente essere causati dall'orologio di sistema di Windows impostato su ora, data o fuso orario errati.

Potresti aver ricevuto uno dei seguenti messaggi Roblox:

  • L'ora del sistema non è corretta
  • Impossibile ottenere l'URL ... Richiesta di invio HTTP non riuscita,err=0x2f05

Correggere semplicemente tutto dovrebbe porre rimedio alla situazione. È possibile impostare questi aspetti manualmente o impostarlo per l'aggiornamento automatico con un server. L'impostazione automatica assicura che non scivoli indietro e ricomincia a causare problemi. Ecco come impostarlo automaticamente:

Windows 8.1 e versioni precedenti

  1. Fare clic con il tasto destro sull'orologio di Windows nell'angolo in basso a destra
  2. Fai clic su Regola data/ora
  3. Seleziona la scheda Ora Internet
  4. Qui dovresti essere in grado di sincronizzare l'ora e la data con un server Internet

Windows 10

  1. Fare clic con il tasto destro sull'orologio di Windows nell'angolo in basso a destra
  2. Assicurarsi che "Imposta ora automaticamente" sia attivato e che sia stato selezionato il fuso orario corretto
