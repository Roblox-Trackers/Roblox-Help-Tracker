Problemi generali sul desktop Roblox

In caso di problemi con Roblox, consultare le seguenti informazioni.

Alcuni sintomi comuni

  • Roblox non viene trovato dal tuo browser – “impossibile trovare Roblox”
  • Endless Install Loop - il giocatore è diretto all'installazione di Roblox, anche dopo averlo fatto
  • Roblox o Roblox Studio si arresta in modo anomalo quando si tenta di connettersi a un gioco online o quando si apre un luogo

Perché questi possono accadere

  • Roblox o un altro programma è ancora in esecuzione durante l'installazione
  • I file temporanei sono danneggiati
  • Il firewall impedisce l'avvio di Roblox
  • Altri file Roblox sono stati danneggiati

Cosa fare

Prova ogni passaggio di seguito, uno alla volta, in ordine. Se una soluzione non funziona, passa alla successiva!

Riavvia

Per prima cosa riavvia il computer e prova a giocare. A volte questo può essere tutto ciò che è necessario per risolvere i problemi relativi a Roblox o Roblox Studio.

Ripristina opzioni Internet per Windows

Abbiamo scoperto che molti problemi di Roblox e Roblox Studio possono essere risolti ripristinando le Opzioni Internet del tuo computer. Tieni presente che questi passaggi dovranno utilizzare Internet Explorer , ma risolveranno molti problemi anche per i giocatori che utilizzano altri browser come Chrome o Firefox. Si prega di provare quanto segue :

    1. Apri Internet Explorer
    2. Apri l'icona a forma di ingranaggio > Opzioni Internet
    3. Apri la scheda Avanzate
    4. Fai clic sul pulsante Ripristina

Dopo il ripristino, chiudere Internet Explorer per rendere effettive le modifiche. Al termine, dovresti essere in grado di tornare a utilizzare il tuo browser Web preferito.

Cancella i tuoi file temporanei Internet

Un passaggio importante in qualsiasi processo di disinstallazione o reinstallazione è quello di cancellare i file temporanei di Internet del browser. Ciò aiuterà con eventuali problemi di corruzione che potrebbero essere sorti. Per assistenza in questo processo, clicca qui e  vedere la sezione "Cancella file e cookie temporanei di Internet".

Controlla il tuo firewall

Molti software antivirus o di sicurezza Internet cercano di controllare quali programmi possono accedere a Internet. Spesso, questi programmi errano con cautela e per impostazione predefinita nessun programma è autorizzato ad accedere. Per assistenza su come esaminare questo aspetto e consentire a Roblox di superare eventuali blocchi, clicca qui .

Aggiorna Internet Explorer

Prova ad aggiornare Internet Explorer all'ultima versione.  Puoi trovarla cliccando qui. Anche se al momento non usi IE come browser, dovrai comunque assicurarti di avere la versione più aggiornata sul tuo computer Windows affinché Roblox funzioni correttamente.

Ridurre al minimo i problemi futuri

Assicurati sempre che tutti i programmi siano chiusi sul tuo computer prima di installare o aggiornare Roblox. Assicurati inoltre che Internet Explorer sia aggiornato con l'ultima versione.
