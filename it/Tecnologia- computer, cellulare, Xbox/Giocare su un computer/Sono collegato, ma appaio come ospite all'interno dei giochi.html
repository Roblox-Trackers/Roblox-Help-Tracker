Sono collegato, ma appaio come ospite all'interno dei giochi

Se scopri di essere inserito nei giochi come ospite anche se hai effettuato l'accesso al tuo account, ci sono alcune cose da esaminare.

Cosa fare - Windows

Prova ogni passaggio di seguito, uno alla volta, in ordine. Se una soluzione non funziona, passa alla successiva!

Riavvio

Per prima cosa riavvia il computer e prova a giocare. A volte questo può essere tutto ciò che è necessario per risolvere i problemi relativi a Roblox o Roblox Studio.

Ripristina le opzioni Internet in Windows

Anche se non si utilizza Internet Explorer, il ripristino delle opzioni Internet tramite questo browser potrebbe comunque risolvere il problema. Si noti che mentre è necessario eseguire questi passaggi utilizzando Internet Explorer, al termine è possibile tornare al browser preferito. Si prega di provare quanto segue:

    1. Apri Internet Explorer
    2. Apri l'icona a forma di ingranaggio > Opzioni Internet
    3. Apri la scheda Avanzate
    4. Fai clic sul pulsante Ripristina

Dopo il ripristino, chiudere Internet Explorer per rendere effettive le modifiche.

Nota: se si utilizza Windows 10, assicurarsi di provare questi passaggi con Internet Explorer e non con Microsoft Edge. Le icone sembrano simili. Per trovare IE in Windows 10, premi il tasto Windows + S sulla tastiera e digita "Internet Explorer".

Controlla l'orologio di sistema di Windows

Può sembrare strano, ma diversi problemi possono effettivamente essere causati dall'orologio di sistema di Windows impostato sull'ora o sulla data sbagliate. Se questa è la causa, la semplice correzione di tutto dovrebbe porre rimedio alla situazione. Mentre questo può essere fatto manualmente, puoi anche impostarlo per l'aggiornamento automatico con un server, assicurandoti così che non scivoli indietro e ricomincio a causare problemi. Ecco come fare:

Windows 8.1 e versioni precedenti

    1. Fare clic con il tasto destro sull'orologio di Windows nell'angolo in basso a destra
    2. Fai clic su Regola data/ora
    3. Seleziona la scheda Ora di Internet
    4. Qui dovresti essere in grado di sincronizzare l'ora e la data con un server Internet

Windows 10

    1. Fare clic con il tasto destro sull'orologio di Windows nell'angolo in basso a destra
    2. Assicurarsi che l'opzione Imposta ora automaticamente sia attivata e che sia stato selezionato il fuso orario corretto

Controlla le tue impostazioni proxy

Se i passaggi precedenti non hanno risolto il problema, è possibile che sia necessario assicurarsi che il computer non stia tentando di connettersi tramite un server proxy. Si prega di seguire questi passaggi:

Windows 7 e versioni precedenti

    1. Vai al menu Start
    2. Cerca "proxy" e configurare il server proxy apparirà, fai clic su di esso
    3. Fai clic su Impostazioni LAN e deseleziona tutti i segni di spunta in quella scheda
    4. Clicca su OK

Windows 8 o 8.1

    1. Vai alla schermata iniziale piastrellata facendo clic sull'angolo in basso a sinistra dello schermo
    2. Digita "proxy" e apparirà l'opzione Configura server proxy . Cliccaci sopra
    3. Fai clic su Impostazioni LAN e deseleziona tutti i segni di spunta in quella scheda
    4. Clicca su OK

Windows 10

    1. Premi il tasto Windows + S sulla tastiera. In alternativa, se la barra "Chiedi informazioni" di Cortana è disponibile nella barra delle attività, è sufficiente fare clic su di essa
    2. Digita "proxy" e apparirà un'opzione Modifica impostazioni proxy. Cliccaci sopra
    3. Assicurati che "Usa script di installazione" e "Usa un server proxy" siano disattivati
    4. Scorri verso il basso e fai clic su Salva
