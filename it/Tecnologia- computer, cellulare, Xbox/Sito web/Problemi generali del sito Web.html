Problemi generali del sito Web

Il seguente articolo suggerirà alcuni metodi che si sono dimostrati efficaci nel risolvere i seguenti problemi:

  • Il sito Web viene visualizzato in modo errato,
  • Alcune pagine non si caricano affatto o non visualizzano errori durante il caricamento,
  • Pulsanti o collegamenti non ti consentono di fare clic su di essi,
  • Si tenta di accedere al sito Web Roblox ma non è possibile nonostante sia corretto nome utente e password,
  • Altro: se stai affrontando un problema diverso con il sito Web, i passaggi sottostanti possono comunque aiutare a risolverlo. Inoltre alcuni articoli di siti web non correlati possono essere collegati qui per alcuni passaggi.

Cosa Fare

Prova ogni passaggio di seguito, uno alla volta, in ordine. Se una soluzione non funziona, passa alla prossima!

Cancella i tuoi file e cookie temporanei da Internet

I seguenti collegamenti ti aiuteranno a rimuovere i file temporanei di Internet (anche denominati "cache") e cookie per i principali browser Web per PC. Per favore, assicurati di fai questo per Internet Explorer e qualsiasi altro browser da cui si desidera avviare Roblox.

      • Mozilla Firefox - File temporanei di Internet (da Mozilla.com)
      • Mozilla Firefox - Cookies (da Mozilla.com)
      • Google Chrome - Entrambi (da Google.com)
      • Microsoft Edge - Entrambi (da Microsoft.com)
      • Safari - Cookies (da Apple.com)
      • Altri browser: Se stai usando un altro browser o una versione precedente di un browser, cerca informazioni su come eliminare la cache/i dati del browser/i file temporanei di Internet per le tue versioni specifiche del browser.

Dopo aver completato quanto sopra descritto per tutti i browser Web pertinenti, riavvia il tuo computer e verifica se il problema è stato risolto.

Se desideri eliminare manualmente i tuoi file temporanei Internet di Windows, puoi farlo andando direttamente alla cartella dei file temporanei Internet sul tuo disco rigido. Il percorso e il nome della cartella dipendono dalla versione di Windows e dall'utilizzo o meno dei profili utente.

Nota: potrebbe essere necessario modificare le impostazioni di Windows Explorer per visualizzare tutti i tipi di file (inclusi i file di sistema protetti) per accedere a queste cartelle.

Per Windows 10

Se hai Windows 10, i file temporanei di Internet si trovano nella seguente posizione (nota che sul tuo PC possono trovarsi su un'altra unità anziché sull'unità C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Per Windows 7 and 8

Se hai Windows 7 o Windows 8, i file temporanei Internet sono in queste posizioni (si noti che possono trovarsi su un'altra unità anziché sull'unità C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Disabilita le componenti aggiuntive del tuo browser web

Molte volte i problemi relativi al sito Web sono il risultato di componenti aggiuntive di terze parti per il tuo browser web. Queste possono anche essere definite "Estensioni" o "Plugin" a seconda del tuo browser preferito. Per vedere se questo è il tuo caso, prova a disabilitare tutti i componenti aggiuntivi. Se il problema si risolve, prova a riabilitarli uno alla volta fino a quando il problema non si ripresenta. Una volta che il problema si e' ripresentato , dovresti aver trovato quello incompatibile con Roblox.

I seguenti link possono aiutarti con i browser Web PC più popolari:

        • Internet Explorer (From Microsoft.com)
        • Firefox (From Mozilla.com)
        • Chrome (From Google.com)
        • Altri browser: se usi un diverso browser o una versione precedente di un browser, cerca informazioni su come disabilitare le sue Estensioni/Plugin/ Componenti aggiuntive per il tuo browser/versione specifici.
