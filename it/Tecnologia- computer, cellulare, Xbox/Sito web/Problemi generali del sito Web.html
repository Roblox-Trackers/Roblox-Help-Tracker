Problemi generali del sito Web

Il seguente articolo suggerirà alcuni metodi che si sono dimostrati efficaci in risolvendo i seguenti problemi:

  • Il sito Web viene visualizzato in modo errato
  • Alcune pagine non si caricano affatto o non visualizzano errori durante il caricamento
  • Pulsanti o collegamenti non ti consentono di fare clic su di essi
  • Si tenta di accedere al sito Web Roblox ma non è possibile nonostante sia corretto nome utente e password
  • Altro: se si verifica un problema diverso con il sito Web, i passaggi sotto può anche aiutare a risolverlo. Inoltre alcuni siti web non correlati gli articoli possono essere collegati qui per alcuni passaggi

What to Do

Prova ogni passaggio di seguito, uno alla volta, in ordine. Se una soluzione non funziona lavoro, passa al prossimo!

Cancella i tuoi file e cookie temporanei da Internet

I seguenti collegamenti ti aiuteranno a rimuovere i file temporanei di Internet (anche denominato "cache") e cookie per i principali browser Web per PC. Per favore assicurati di fai questo per Internet Explorer e qualsiasi altro browser da cui si desidera avviare Roblox.

      • Internet Explorer - Temporary Internet Files (From Microsoft.com)
      • Internet Explorer - Cookies (From Microsoft.com)
      • Mozilla Firefox - Temporary Internet Files (From Mozilla.com)
      • Mozilla Firefox - Cookies (From Mozilla.com)
      • Google Chrome - Both (From Google.com)
      • Microsoft Edge - Both (From Microsoft.com)
      • Safari - Cookies (From Apple.com)
      • Altri browser: Se stai usando un altro browser o una versione precedente di un browser, cercare informazioni su come eliminare la cache / i dati del browser / i file temporanei di Internet per il tuo browser / versione specifici.

Dopo aver completato quanto sopra per tutti i browser Web pertinenti, riavviare il tuo computer e verifica se il problema è stato risolto.

Se si desidera eliminare Internet Explorer / Internet temporaneo di Windows File manualmente, puoi farlo andando direttamente sul Temporaneo Cartella File Internet sul disco rigido. La posizione e il nome della cartella dipendono sulla versione di Windows e se si utilizzano o meno i profili utente.

Nota: potrebbe essere necessario modificare le impostazioni di Esplora risorse per mostrare tutti i tipi di file (inclusi i file di sistema protetti) per accedere a queste cartelle.

For Windows 10

Se hai Windows 10, i file temporanei Internet si trovano nella seguente posizione (nota che sul tuo PC possono trovarsi su un'altra unità anziché sull'unità C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

For Windows 7 and 8

Se hai Windows 7 o Windows 8, i file temporanei Internet sono in questi posizioni (si noti che possono trovarsi su un'altra unità anziché sull'unità C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Disable Your Web Browser's Add-Ons

Molte volte i problemi relativi al sito Web sono il risultato di componenti aggiuntivi di terze parti per il tuo browser web. Questi possono anche essere definiti "Estensioni" o "Plugin" a seconda del tuo browser preferito. Per vedere se questo è il tuo caso, prova a disabilitare tutti i componenti aggiuntivi. Se il problema si risolve, prova a riabilitarli uno alla volta fino a quando il problema non si ripresenta. Una volta che il problema inizia backup, avresti dovuto trovare quello incompatibile con Roblox.

I seguenti link possono aiutarti con i browser Web PC più popolari:

        • Internet Explorer (From Microsoft.com)
        • Firefox (From Mozilla.com)
        • Chrome (From Google.com)
        • Altri browser: se ne usi uno diverso browser o una versione precedente di un browser, cercare informazioni su come disabilitare le sue Estensioni / Plugin / Componenti aggiuntivi per il tuo browser / versione specifici.

Reset Internet Options for Windows

Abbiamo scoperto che molti problemi di Roblox, Website e Roblox Studio possono essere risolti ripristinando le Opzioni Internet del tuo computer. Si prega di tenere presente che questi i passaggi dovranno utilizzare Internet Explorer anche se in genere si utilizza un altro programma di navigazione in rete. Dopo aver tentato questi passaggi, è possibile tornare a utilizzare qualunque browser tu preferisca. Si prega di provare quanto segue:

      1. Apri Internet Explorer (in particolare questo browser Web)
      2. Apri l'icona a forma di ingranaggio > Internet options
      3. Apri la scheda Avanzate
      4. Fai clic sul pulsante Ripristina

Dopo il ripristino, chiudere Internet Explorer per consentire le modifiche effetto. Al termine, dovresti essere in grado di tornare a utilizzare il tuo browser preferito.

Update Internet Explorer

Prova ad aggiornare Internet Explorer all'ultima versione. Potete trovare quello cliccando qui. Even se al momento non utilizzi IE come browser, dovrai comunque assicurarti hai la versione più aggiornata sul tuo computer Windows per Roblox per funzionare correttamente.
