一般的なウェブサイトの問題

次の記事では、次の問題に対処するのに効果的であることが証明されているいくつかの方法を提案します。

  • ウェブサイトが正しく表示されない
  • 一部のページがまったく読み込まれない、または読み込み時にエラーが表示される
  • ボタンまたはリンクをクリックできない
  • Roblox ウェブサイトにログインしようとしましたが、正しいユーザー名とパスワードを持っているにも関わらずログインできない
  • その他：ウェブサイトに別の問題がある場合は、次の手順も役立ちます。 さらに、いくつかの関連性のないウェブページは、いくつかの手順についてここに記事をリンクする場合があります

対処方法

以下の各ステップを順番に1つずつ試してください。 1つのソリューションが機能しない場合は、次のソリューションに進んでください

インターネット一時ファイルとCookieをクリア

以下のリンクは、インターネット一時ファイル（「キャッシュ」とも呼ばれます）と主要なPCウェブブラウザーのCookieを削除するのに役立ちます。 Internet Explorerだけでなく、Robloxを起動する他のブラウザーでもこれを実行してください。

      • Internet Explorer - Temporary Internet Files (Microsoft.comから)
      • Internet Explorer - Cookies (Microsoft.com から )
      • Mozilla Firefox - Temporary Internet Files (Mozilla.com から )
      • Mozilla Firefox - Cookies (Mozilla.com から )
      • Google Chrome - Both (Google.com から )
      • Microsoft Edge - Both (Microsoft.com から )
      • Safari - Cookies (Apple.com から )
      • その他のブラウザ ：別のブラウザまたは古いバージョンのブラウザを使用している場合は、特定のブラウザ/バージョンのキャッシュ/ブラウザデータ/一時インターネットファイルを削除する方法に関する情報を検索してください。

関連するすべてのウェブブラウザーについて上記を完了したら、コンピューターを再起動して、問題が解決したかどうかを確認してください。

Internet Explorer / Windows Temporary Internet Filesを手動で削除する場合は、ハードドライブのTemporary Internet Filesフォルダーに直接移動してください。 場所とフォルダー名は、Windowsのバージョンと、ユーザープロファイルを使用しているかどうかによって異なります。

注：これらのフォルダーにアクセスするには、Windowsエクスプローラーの設定を変更して、保護されたシステムファイルを含むすべての種類のファイルを表示する必要がある場合があります。

Windows 10の場合

Windows 10を使用している場合、インターネット一時ファイルは次の場所にあります。（PCのドライブCではなく別のドライブにある可能性があることに注意してください）

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Windows 7もしくは8の場合

Windows 7またはWindows 8を使用している場合、一時インターネットファイルは次の場所にあります。（ドライブCではなく別のドライブにある可能性があることに注意してください）

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

ウェブブラウザのアドオンを無効にします

多くのサイトの問題は、ウェブブラウザのサードパーティアドオンの結果です。 選択したブラウザに応じて、これらは拡張機能またはプラグインとも呼ばれます。 これがあなたに当てはまるかどうかを判断するには、すべてのアドオンを無効にします。 問題が解決した場合は、問題が再発するまで一度に1つずつ有効にしてみてください。 問題が再発する場合は、Robloxと互換性のないものを見つけてください。

以下のリンクは、最も一般的なPC ウェブブラウザーの使用に役立ちます。

        • Internet Explorer (Microsoft.comから)
        • Firefox (From Mozilla.com から )
        • Chrome (From Google.com から )
        • その他のブラウザー ：別のブラウザーまたは古いブラウザーを使用している場合は、特定のブラウザー/バージョンの拡張機能/プラグイン/アドオンを無効にする方法に関する情報を検索してください。
