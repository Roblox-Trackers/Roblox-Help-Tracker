ログインしていますが、エクスペリエンス内でゲストとして表示されます

アカウントにログインしていても、ゲストとしてエクスペリエンスに参加していることがわかった場合は、いくつかの検討事項があります。

対処方法 － Windows

以下の各ステップを順番に1つずつ試してください。 1つのソリューションが機能しない場合は、次のソリューションに進んでください！

再起動

最初にコンピューターを再起動して、プレイしてみます。 RobloxまたはRoblox Studio関連の問題を解決するために必要なのはこれだけで十分な場合があります。

Windowsシステム時計を確認する

奇妙に聞こえるかもしれませんが、実際には、Windowsシステム時計が間違った時刻または日付に設定されているために、さまざまな問題が発生する可能性があります。 これが原因である場合、単にすべてを修正することで状況を改善する必要があります。 これは手動で行うことができますが、サーバーで自動的に更新するように設定することもできます。これにより、サーバーが遅れて問題が発生しなくなることを確認できます。 これを行う方法は次のとおりです：

Windows 8.1およびそれ以前

    1. 右下隅のウィンドウの時計を右クリックします。
    2. 日付/時刻の調整 をクリックします。
    3. インターネット時間 タブを選択します。
    4. ここで、時刻と日付をインターネットサーバーと同期できるはずです。

Windows 10

    1. 右下隅のウィンドウの時計を右クリックします。
    2. 時間を自動的に設定がオンになっており、正しいタイムゾーンが選択されていることを確認します。

プロキシ設定を確認してください

上記の手順で問題が解決しない場合、コンピューターがプロキシサーバー経由で接続しようとしていないことを確認する必要がある可能性があります。 次の手順に従ってください：

Windows 7以前

    1. スタートメニューに移動します。
    2. 「プロキシ」を検索し、プロキシサーバーを表示し、クリックします。
    3. LAN設定 をクリックし、そのタブのすべてのチェックマークを解除します。
    4. OK をクリックします。

Windows 8または8.1

    1. 画面の左下隅をクリックして、タイル表示されたスタート画面に移動します。
    2. 「プロキシ」と入力すると、プロキシサーバーの構成オプションが表示されます。 それをクリックします。
    3. LAN設定 をクリックし、そのタブのすべてのチェックマークを解除します。
    4. OK をクリックします。

Windows 10

    1. キーボードのWindowsキー+ Sを押します。 または、コルタナの「何でも聞いてください」バーがタスクバーで使用可能な場合は、それをクリックするだけです。
    2. 「プロキシ」と入力すると、プロキシ設定の変更オプションが表示されます。 クリックして
    3. 「セットアップスクリプトを使用する」と「プロキシサーバーを使用する」がオフに設定されていることを確認します。
    4. 下にスクロールして 保存 をクリックします。
