Robloxをアンインストールする方法

Roblox アプリまたは スタジオで技術的な問題が発生している場合は、アプリケーションを再インストールする前にアンインストールすると解決する場合があります。 この記事では、アンインストールのプロセスについて説明します。

Windowsの場合の手順

プログラムをアンインストールする

Windows 7

  • [スタート]メニュー > [コントロールパネル] > [プログラムと機能] に移動します
  • Robloxが表示されるまで下にスクロールします
  • そこでプログラムをアンインストールします

Windows 8または8.1

  • 画面の左下隅をクリックして、タイル表示されたスタート画面を表示させます
  • 「プログラムと機能」と入力し、検索結果の右側に表示される [プログラムと機能] アイコンをクリックします
  • 表示されるウィンドウで、Robloxが表示されるまで下にスクロールします
  • プログラムをアンインストールします

Windows 10

  • 画面の左下にあるスタートボタンまたはCortana検索ボックスをクリックします
    • または、キーボードでWindowsキーを押したままSキーを押すこともできます
  • 検索ボックスに 「プログラムと機能」 と入力し、検索リストの上部に表示されるプログラムと機能アイコンをクリックします
  • Robloxが表示されるまで下にスクロールします
  • プログラムをアンインストールします

Robloxフォルダを削除

特に継続的な問題がある場合は、アンインストール後にファイルや古いファイルが残っていないことを確認して、Robloxフォルダを完全に削除すると良いかも知れません。 これを行うには、次の手順を実行します。

  1. スタートボタンを右クリックし、 [エクスプローラ] または [ファイルエクスプローラ] をクリックします
  2. このフォルダに移動しますC:\Users\(Your Windows Username)\AppData\Local
  3. Robloxフォルダを削除する

注：このフォルダに保存するプレースを手動で選択していない限り、Studioで保存したプレースは削除されません。

アンインストールできない場合

プログラムのアンインストールに問題がある場合は、次を試してください。

インターネット一時ファイルを削除する

アンインストールまたは再インストールの重要な手順は、ブラウザのインターネット一時ファイルを削除することです。これは、データ破損が起きた場合の問題の解決に役立ちます。 このプロセスに関してサポートが必要な場合は、 「インターネット一時ファイルとCookieを削除する」 をクリックしてセクションをご覧下さい。

ファイアウォールを確認

多くのウイルス対策ソフトウェアまたはインターネットセキュリティソフトウェアは、どのプログラムがインターネットにアクセスできるかを制御しようとします。このようなプログラムは警戒のしすぎによるエラーを引き起こし、デフォルトではどのプログラムにもアクセスできません。この問題に関するヘルプと障害なくRobloxにアクセスできる方法については、 こちら をクリックしてください。

Robloxを再インストール

次の可能性は再インストールです。分かりにくいことかも知れませんが、問題が以前に失敗したアンインストールによる破損ファイルである場合、これによりコンピューター上のプログラムのコピーが上書きされ、削除または破損したファイルが置き換えられます。 その場合、[スタート]メニューに移動してプログラムをアンインストールできます。 再インストールのサポートについては、 ここ をクリックしてください。

Macの場合の手順

Robloxプログラムの完全終了

  • Command + Option + Escキーを押して、タスク管理プロンプトを開きます
  • RobloxまたはRoblox Studioが表示されていないことを確認してください
  • プログラムのいずれかがリストされている場合は、RobloxまたはRoblox Studioを選択し、「強制シャットダウン」をクリックします

[Finder]> [アプリケーション]に移動します

  • Robloxアイコンをゴミ箱にドラッグします
    • Finderのアイコンには、インストールされているすべてのRobloxファイルが含まれています

ゴミ箱からファイルを削除する方法

  • Dockの[ごみ箱]アイコンをクリックし、[ごみ箱を空にする]を選択します。
  • または、Command + Shift + Deleteの3つのキーを同時に押してゴミ箱を空にします。
  • このアクションは元に戻せないため 、「ごみ箱内のアイテムを削除してもよろしいですか」という警告が表示されます。削除したい場合は「ごみ箱を空にする」をクリックすると、ごみ箱は空になります。


「ごみ箱内のアイテムを完全に削除してもよろしいですか？」というメッセージをバイパスするには、次を押します。

  • Command + Option/Alt + Shift + Delete
