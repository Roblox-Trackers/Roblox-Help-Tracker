バーチャル空間内での設定とヘルプ

バーチャル空間をプレイしているときに、設定を変更したり、ジャンプの仕方やカメラの回転などの動作をする方法が分かります。

設定

こちらのメニューには、音量、カメラ、グラフィックなどの機能の設定が含まれています。

バーチャル空間の開発者がカメラモードを特定の設定に固定していない限り、「定番」と「フォロー」を切り替えることができます。

  • クラシック モード では、手動で調整しない限り、カメラは 1 か所に固定されたままになります。 アバターの視点を変更するには、マウスの右ボタンを押したままドラッグします。
  • フォロー モード では、右または左に移動するとカメラがアバターと一緒に回転し、意図したターゲットを常に視界に留めることができます。
    • Shift Lockスイッチを使用すると、キーボードのShiftキーを押してカメラモードを切り替えることができます。

現在、Roblox でのエクスペリエンスをより適切に調整するために 3 つのアクセシビリティ設定が提供されています。

  • モーションを減らす は、アプリ内およびエクスペリエンス内の特定の画面効果を無効化または軽減します。 たとえば、モーションの削減が有効になっている場合、ゲーム内メニューを開いたときの画面上で、メニューがスライドするのではなくフェードインします。
  • 背景画面の透明化 は、チャット ウィンドウの背景などの UI 要素の背景をより不透明にすることで、コントラストと読みやすさを向上させます。 これにより、UI 要素にて通常より透明度が増すことはありません。
  • UI ナビゲーション切り替え は、\ キーを使用して UI ナビゲーション モードを開始および終了できるかどうかを制御します。 誤って UI ナビゲーション モードに頻繁に入ってしまう場合、または UI ナビゲーション モード機能を使用したくない場合は、この設定を無効にすることができます。

ヘルプ

このメニューには、キーボードショートカット、マウスの設定、モバイルデバイスのジェスチャーが含まれているため、バーチャル空間をナビゲートして、使用可能なすべての機能を使用できます。
