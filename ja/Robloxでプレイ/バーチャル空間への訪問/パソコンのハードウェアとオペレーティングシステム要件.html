パソコンのハードウェアとオペレーティングシステム要件

Robloxの最小システム要件は以下のとおりです。 バーチャル空間のプレイ中に問題が発生した場合は、パソコンのアップグレードが必要になる場合があります。 モバイルデバイスのシステム要件については、この ヘルプ記事 をお読みください。

オペレーティングシステム要件

  • PC/Windows: Robloxは64ビットバージョンのWindows 7、Windows 8.1、またはWindows10にインストールされます。 Windows 8.1では、Metroモード（タイル化されたスタート画面）に現在、対応していないため、Robloxをデスクトップモードで実行する必要があります。
    ご注意：Robloxウェブサイトは、PCとWindowsでは次のブラウザに対応しています: Chrome 、 Firefox 、 Microsoft Edge 。
  • Mac: Robloxクライアントは10.10（Yosemite）以降にインストールされますが、Roblox StudioはMac OS 10.13 (High Sierra) 以降にインストールされます。
    ご注意：Robloxウェブサイトは、Macでは次のブラウザに対応しています。 Chrome , Firefox と Safari 。
  • Linux ：RobloxはLinuxには対応していません。
  • モバイル ：Roblox Mobileのシステム要件については、 ここ をクリックしてください。
  • Chrome OS ：Chrome OSバージョン53以降が必要です。 最初にデバイスで Google Playストア を有効にする必要があります。

ご注意：Robloxビデオ録画機能は、Windows7、Windows 8.1、Windows10でしか機能しません。 新しくリリースされたオペレーティングシステムまたはオペレーティングシステムのプレリリースバージョンには、すぐに対応しない場合があります。

システムのハードウェア要件

  • グラフィックカード ：PC およびWindowsでは、RobloxアプリケーションはDirectX10以上の機能レベルのサポートを必要とします。 最高のパフォーマンスを得るには、専用ビデオカードを搭載した5年未満のパソコン、またはビデオカードを内蔵した3年未満のノートパソコンをお勧めします。
  • プロセッサ ：Robloxは1.6 Ghz以上のクロック速度を備えた最新のプロセッサ(2005+)をお勧めします。 古いAMDプロセッサーにはいくつかの問題があります。
  • RAMまたはメモリ： RobloxはWindows 7、Windows 8.1、またはWindows 10に少なくとも1 GBのメモリを搭載することをおすすめします
  • ストレージスペース ：Robloxのインストールには、20 Mb以上のシステムストレージスペースが必要です。
  • モバイル ：Roblox Mobileのシステム要件については、 ここ をクリックしてください。

インターネット要件

一人しか使用しない家庭では、少なくとも4〜8 Mbpsのインターネット接続をおすすめします。 大人数の家庭では、帯域幅を増やすことをおすすめします。
