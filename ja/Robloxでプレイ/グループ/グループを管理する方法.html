グループを管理する方法

グループを作成したら、次は管理します。 これを行うには、ページの左側のメニューにある [ グループ ] リンクをクリックします。 ここから、グループアクティビティとグループコントロールパネルを確認できます。

グループのメインページ

これは、グループを選択するとすぐに表示されるものです。以下のオプションが用意されています。

  • シャウト : グループの重要な更新情報をここに掲載できます。
  • 説明 : このセクションでは、グループがどのようなものかを示しています。
  • エクスペリエンス : あなたやあなたのメンバーがグループのために何かプレイスまたはエクスペリエンスを作った場合、ここに表示されます。詳細は こちら をご覧ください。
  • メンバー : このタブでは、メンバーをランク別に見ることができます。
  • ソーシャルリンク : このセクションでは、追加されたソーシャルリンクが表示されます。詳しくは こちら をご覧ください。
  • 提示版 : このセクションは、グループの継続的なフォーラムスレッドのようなものです。グループのディスカッションをここに投稿することができます。投稿した内容は、この権限を持つグループメンバーが削除することができます。また、グループ管理画面でメンバーをグループから追放することもできます。不適切なコメントがなされた場合（ Robloxコミュニティースタンダード を参照）、右側に不正を報告するためのオプションがあります。

グループコントロール

これらの コントロール はグループのページの右上にあり、グループを管理するための多くのオプションを提供します。

  • グループ管理者 ：このページでは、グループの基本情報と機能に多くの調整を加えることができます。 このページについては、以下で詳しく説明します。
  • プライマリ作成 ：このボタンを使用して、グループをプライマリグループにします。 プライマリグループの詳細については、 こちら をクリックしてください。
  • グループ退会 ：このボタンを使用すると、参加したくないグループを退会できます。
    • 注意：所有しているグループを離れた場合、後でグループに再度参加しても所有権を取り戻すことはできません。 自分が所有するグループを離れる前に、所有権を別のプレーヤーに譲渡することを選択しない場合（以下の「グループ管理ページ」セクションの「設定」を参照）、そのグループは誰にも所有されません。
  • グループ広告 ： このページでは、自分のグループの広告を作成することができます。詳細は こちら をご覧ください。
  • 不正を報告する ：不正使用の報告：グループの名前または説明、シャウト、および役割を報告するには、[ 不正使用の報告 ]ボタンを選択し、それを入力して[ 報告 ]ボタンを選択します。

グループ管理ページ

このページでは、メンバーとグループ自体を効果的に管理するために必要なツールを提供します。 アクセスするには、グループのページの右側にある [コントロール] ボックスの [グループ設定] ボタンをクリックします。 そこに、次のオプションが表示されます：

  • グループ情報 ：ここで、グループの名前、所有者、画像、説明を変更できます。
  • 設定 : ここでは、グループに参加するための要件を設定できます。これには、手動での承認が必要な場合も含まれます。また、敵の宣言を許可したり、グループの資金やエクスペリエンスの可視性を設定するオプションもあります。
    • このセクションでは、オーナーが他のメンバーに所有権を譲渡することもできます。
      • このプロセスは元に戻せませんのでご注意ください。
  • ソーシャルリンク : このセクションでは、グループの説明内にソーシャルリンクを追加することができます。詳しくは こちら をご覧ください。
  • 収入 : のセクションでは、あなたのグループが過去1日、1週間、1ヶ月、1年で獲得したRobuxを追跡することができます。また、「 収入 」セクションに「 支払い 」オプションが追加され、グループの資金を特定のメンバーに 一回限りの支払い で提供したり、 定期的な支払い を設定したりすることができます。
  • メンバー : ここでは、現在のメンバーを検索または閲覧できます。また、ランク別にフィルタリングすることもできます。さらに、ここでメンバーのランクを変更したり、保留中の参加リクエストを承認または拒否することができます。また、メンバーのアイコンの横にある歯車をクリックして「ユーザーをキック」を選択すると、そのメンバーをグループから外すことができます。
  • 役割 : ここでは、新しいランクを追加したり、既存のランクの名前や権限を変更したりすることができます。詳しくは こちら をご覧ください。
  • アフィリエイト : このセクションでは、他のグループを自分のグループの公式な敵または味方として宣言することができます。詳しくは こちら をご覧ください。
  • 監査ログ : メンバーがグループに関して行った全ての行為を確認することができます。詳しくは こちら をご覧ください。
