Meta Quest オープンベータ版についてよくあるご質問

Meta Quest オープンベータ版とは？

Roblox（ロブロックス）のアプリが Meta の App Lab 上にオープンデータ版として Meta Quest にやってきます。 オープンベータ版とは、当社の開発者が自作のバーチャル空間に参加するVRユーザーからのフィードバックを集めるチャンスが得られるものです。 Meta Quest ヘッドセットの所有者は、RobloxアプリをインストールしてXbox、iOS、Android、デスクトップデバイスのユーザーと一緒にバーチャル空間を楽しめます。

Meta Quest 2 と Quest Pro デバイスでしかオープンベータ版には対応していませんのでご注意ください。 Meta Quest 上でのRobloxは、13歳以上のユーザー限定です。

Meta Quest上のRobloxオープンベータ版にはどうやってアクセスしますか？

Meta Quest ヘッドセット（Meta Quest 2 または Pro）が必要です。

*ご注意: Robloxは、Meta Quest 1 のヘッドセットには対応していません。

オープンベータ版をインストールするには、以下の手順に従う必要があります。

  • Meta Quest デバイスが最新の状態かどうかを確認してください。 （Robloxには、Meta Quest ソフトウエアのv55が必要です。）
  • Quest Store で「Roblox」を検索して、そこからインストールしてください。
  • お持ちの Quest デバイスでRobloxを始めましょう。
  • お持ちのパソコンやスマホで クイックログイン オプションを使って既存のRobloxアカウントにサインインしてください。

*ご注意: 現在のところ、Meta Quest デバイス上でのアカウント作成やパスワード作成には対応していません。

Robloxのバーチャル空間内では、どうやって Meta Quest コントローラーを使えばいいですか？

Robloxにはお楽しみいただけるバーチャル空間がたくさんあります。 Meta Quest 上のRobloxにあるバーチャル空間には、少しずつ違ったルールや操作方法、バーチャル空間の仕組みがあります。 ほとんどのバーチャル空間では、一般的な操作方法は以下に表示してあるのと同じになります。

オープンベータ版はどのくらいの期間、利用できますか？

オープンベータ版は、Meta Quest 上のRobloxが一般公開されるまで続きます。

Meta Quest オープンベータ版でのRobloxアカウントのログイン方法は？

お持ちのパソコンやスマホで クイックログイン オプションを使って既存のRobloxアカウントにサインインしてください。


*ご注意: 現在のところ、Meta Quest デバイス上でのアカウント作成やパスワード作成には対応していません。

どのRobloxのバーチャル空間がMeta Quest上で利用できますか？

VRユーザー向けに公開されたバーチャル空間は、すべてMeta Quest上でご利用いただけます。 今後、何千ものVR体験が登場する予定です。 オープンベータ版とは、開発者コミュニティがVR向けバーチャル空間をテストし、VR向けに公開するかどうかを判断するチャンスを得られるものです。 オープンベータ版の実施に伴い、ご利用いただけるVR体験が今後さらに登場する予定です。

Meta Quest ヘッドセットを使ってプレイしたオープンベータ版でのバーチャル空間を評価することは可能ですか？

残念ながら、ユーザーはMeta Quest ヘッドセット内でバーチャル空間を評価することはできません。 評価を得る前に、開発者にVR向けバーチャル空間を最適化する機会を設けたいと考えています。

Meta Quest ヘッドセット付きRobuxを追加購入する

オープンベータ版でRobuxはご購入いただけません。 ユーザーはアイテムを購入することはできますが、Meta Questを使ってRobuxを追加購入することはできません。 Meta Quest デバイスの使用によるRobuxの購入は、近日中に対応する予定です。

RobloxはパソコンへのVR接続に対応していますか？

はい、Robloxは今後も引き続きVRヘッドセットのパソコンへの接続に対応する予定です。 詳しくは、 こちら をご覧ください。

オープンベータ版 Meta Quest上で利用できる安全対策ツールはありますか？

はい、RobloxユーザーはMeta Quest上で、ミュート、ブロック、悪質行為の報告といった機能をご利用いただけます。 詳しくは以下をごらんください。

他のユーザーのボイスチャットをミュートする

「音声でチャット」では、相手のアバター上、またはバーチャル空間のメニューにある名前の横に表示されるマイクのアイコンをクリックすると、他のユーザーをミュートできます。 ミュートは「音声でチャット」のみに適用され、テキストチャットではご利用いただけません。またミュートは、参加中のセッションにしか適用されません。

他のユーザーをブロックする

バーチャル空間内で他のユーザーをブロックすると、音声チャットやテキストチャットで関わらないようにすることができます。 こちらのヘルプ記事をご覧ください： 他のユーザーのブロック方法 。

規約違反の報告

他のユーザーによる 利用規約 や コミュニティ基準 の違反が判明した場合は、「規約違反を報告」する機能を利用して、内容審査チームに報告してください。 こちらのヘルプ記事をご覧ください： ルール違反の報告方法 。

セーフティバブル

セーフティバブル機能がオンの場合、過度に接近するアバターは次第に消えて不可視状態になります。 セーフティバブル機能による他のプレイヤーへの影響はありません。

詳しくは、安全性に関する よくあるご質問サポート をご覧ください。
