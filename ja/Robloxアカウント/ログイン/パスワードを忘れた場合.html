パスワードを忘れた場合

パスワードを思い出せない場合は、パスワードをリセットし、パスワードを忘れた場合新しいパスワードを作成する必要があります。 パスワードを忘れた場合 の記事をご覧下さい。 モバイルデバイスを介してパスワードをリセットする場合、デフォルトで電話番号を入力するように求められますが、必要に応じて電子メールに変更できます。

  • アカウントにメールアドレスを追加することを強くお勧めします。 詳細については、 メールアドレスの検証 と アカウントの安全性に関する記事 を参照してください。

メールアドレスを使用してパスワードをリセットする

アカウントに電話番号が添付されていないがメールアドレスがある場合は、[ 送信 ]ボタンの下にある[ メールを使用してパスワードをリセット ]リンクをクリックして、メールアドレスでパスワードをリセットする必要があります。

  1. Roblox ログインページ から パスワードまたはユーザーネームをお忘れですか ？ を選択。
  2. ボックスにメールアドレスを入力します。 [ 送信 ]ボタンを押します。
    • 必ず有効なメールアドレスを追加してください。 たとえば、アドレスが「name.example@domain.com」で、Robloxアカウントでピリオドなしの「nameexample@domain.com」の場合、ピリオドなしで入力する必要があります。
  3. 手順が記載されたメールが、現在アカウントに関連付けられているメールアドレスに送信されます。 メール本文の[ パスワードのリセット ]ボタンを押します。
    • パスワードのリセットメールが届かない場合は、スパム/ジャンクフォルダを確認してください。 また、「password_reset@roblox.com」と「@appmail.roblox.com」が連絡先リストに含まれていることを確認して、ブロックされないようにしてください。
  4. [ パスワードのリセット ]ページが開きます。
    • 1つのアカウント：これにより、新しいパスワードを入力するためのページが開きます。
    • 複数のアカウント：パスワードをリセットするアカウントを選択します。
  5. 新しいパスワードを入力して確認します。 [ 送信 ]ボタンを押します。
    • 必ず新しいパスワードを作成してください。 古いパスワードや別のアカウントで使用されているパスワードは使用しないでください。

電話番号を使用してパスワードをリセットする

アカウントにメールアドレスが添付されていないが電話番号がある場合は、[ 送信 ]ボタンの下にある[ 電話番号を使用してパスワードをリセットする ]リンクをクリックして、電話番号でパスワードをリセットする必要があります。

  1. Roblox ログインページ から パスワードまたはユーザーネームをお忘れですか ？ を選択。
  2. [ 電話番号を使用してパスワードをリセットする ]を選択します
  3. 国番号 を選択してください
  4. ボックスに電話番号を入力します。 [ 送信 ]ボタンを押します。
  5. [検証]を選択して、プロセスを完了します。 これにより、6桁の番号が電話に送信されます。
  6. ボックスにコード(6桁)番号を入力します。 検証 ボタンを押します。
  7. 新しいパスワードを入力して確認します。 [ 送信 ]ボタンを押します。

まだ問題がある場合

  • 技術的な問題のトラブルシューティング
  • アカウントにメールアドレスまたは電話番号を追加せずにパスワードをリセットしようとした場合、パスワードをリセットすることはできません。
    • ログインの問題については カスタマーサポート に連絡してください。 アカウントをリセットしたり、情報を変更したりする前に、すべてのプレイヤーがアカウントの所有権の確認をする必要があります。

更新：Facebookログインが削除されました

Facebookの資格情報を使用してログインする機能は、 2021年11月28日 をもってRobloxプラットフォームから削除されます。Facebookにログインしているユーザーは、アカウントにパスワードを設定するように通知されました。 この機能が削除されたためにログインできない場合は、上記の手順を試して パスワードをリセット してください。 これらの手順を実行してもアカウントにアクセスできない、またはパスワードをリセットできない場合は、 カスタマーサポートチーム にご連絡ください。

セキュリティの強化

他の人をアカウントに入れないようにするための追加のセキュリティ対策として、2段階認証プロセス（2SV）が有効になっていることを確認してください。 有効にする方法については、アカウントに 2段階認証プロセスを追加する方法 をご覧ください。
