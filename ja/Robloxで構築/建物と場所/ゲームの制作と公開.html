ゲームの制作と公開

1つのゲームを制作するだけでは、創造性を満足させることができない場合があります。 幸いなことに、当社はユーザーの満足を満たすためのお手伝いすることができます！ アカウントは常に200の公開ゲーム（他のユーザーがプレイできる）に制限されていますが、すべてのユーザーは、好きなだけゲームを制作できます。

ゲームを公開または非公開にする方法

ゲームを公開または非公開にする方法は2つあります

注：ゲームを非公開にすると、実行中のゲームがシャットダウンされ、すべてのサーバーサブスクリプションがキャンセルされます。

方法1：

      1. ウェブサイトの上部にある青いバーの 制作 をクリックします。
      2. [ マイクリエーション ]タブで、[ ゲーム ]がハイライト表示されていない場合はクリックします
      3. 公開または非公開にしたいプレイスを見つけ、そのプレイスのタイトルの下にある灰色または緑色のアイコンをクリックして、2つのステータスを切り替えます。
        • 公開 （緑色のアイコン）は、すべてのユーザーがゲームを表示およびプレイできるようにします。
        • 非公開 （灰色のアイコン）は、グループゲームでない限り、あなた以外の誰もがゲームを表示またはプレイできないようにします。その場合、あなたと任命された制作者（ゲームの編集権限を持つ必要なランクを持つ）だけがゲームをプレイできます。詳しくは、グループゲームに関する ヘルプ記事 をご覧ください。

方法2：

      1. ゲームのページに移動します。
      2. 3つのドットのアイコンをクリックして、ゲームの設定メニューを表示します。
      3. このプレイスの環境設定 または このゲームの環境設定 を選択します。
      4. 上記の手順でどちらを選択しても、表示される画面で、まだハイライト表示されていない場合は、 基本設定 タブに移動します。
      5. 公開 で、 公開 または 非公開 を選択します。
        • 公開 （緑色のアイコン）は、すべてのユーザーがゲームを表示およびプレイできるようにします。
        • 非公開 （灰色のアイコン）は、グループゲームでない限り、あなた以外の誰もがゲームを表示またはプレイできないようにします。その場合、あなたと任命された制作者（ゲームの編集権限を持つ必要なランクを持つ）だけがゲームをプレイできます。詳細については、 グループゲーム に関するヘルプ記事をご覧ください。

新しいゲームを制作する方法

    1. ウェブサイトの上部にある青いバーの 制作 をクリックします。
    2. マイクリエーションタブで、 ゲーム がまだハイライト表示されていない場合はクリックします。
    3. 新しいゲームを制作 をクリックします。
    4. 新規作成の設定とテンプレートを選択します。
    5. ゲームを制作 をクリックします。

これにより、以前の制作ページに戻ります。 必要に応じて、ゲームを今すぐ公開することも、作業を完了するまで待つこともできます。 構築を開始するには、 ゲーム リストで新しいゲームのエントリの右側にある 編集 ボタンをクリックします。

注：いつでも新しいゲームを制作できますが、「 ゲームを公開する 」プロセスを使用して公開するまで、他のRobloxプレイヤーは楽しむことはできません。
