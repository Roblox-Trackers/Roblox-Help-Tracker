エクスペリエンスの制作と公開

1つのエクスペリエンスを制作するだけでは、創造性を満足させることができない場合があります。 幸いなことに、当社はユーザーの満足を満たすためのお手伝いすることができます！ アカウントは常に200の公開エクスペリエンス（他のユーザーがプレイできる）に制限されていますが、すべてのユーザーは、好きなだけエクスペリエンスを制作できます。

エクスペリエンスを公開または非公開にする方法

エクスペリエンスを公開または非公開にする方法は2つあります

注：エクスペリエンスを非公開にすると、実行中のエクスペリエンスがシャットダウンされ、すべてのサーバーサブスクリプションがキャンセルされます。

メゾット1：

      1. ウェブサイトの上部にある青いバーの 制作 をクリックします。
      2. マイクリエーション タブで、 エクスペリエンス がハイライト表示されていない場合はクリックします
      3. 公開または非公開にしたいプレイスを見つけ、そのプレイスのタイトルの下にある灰色または緑色のアイコンをクリックして、2つのステータスを切り替えます。
        • 公開 （緑色のアイコン）は、すべてのユーザーがエクスペリエンスを表示およびプレイできるようにします。
          • 非公開 （灰色のアイコン）は、グループエクスペリエンスでない限り、あなた以外の誰もがエクスペリエンスを表示またはプレイできないようにします。その場合、あなたと任命された制作者（エクスペリエンスの編集権限を持つ必要なランクを持つ）だけがエクスペリエンスをプレイできます。詳しくは、グループエクスペリエンスに関する ヘルプ記事 をご覧ください。

メゾット2：

      1. エクスペリエンスのページに移動します。
      2. 3つのドットのアイコンをクリックして、エクスペリエンスの設定メニューを表示します。
      3. このプレイスの環境設定 または このエクスペリエンスの環境設定 を選択します。
      4. 上記の手順でどちらを選択しても、表示される画面で、まだハイライト表示されていない場合は、 基本設定 タブに移動します。
      5. 公開 で、 公開 または 非公開 を選択します。
        • 公開 （緑色のアイコン）は、すべてのユーザーがエクスペリエンスを表示およびプレイできるようにします。
        • 非公開 （灰色のアイコン）は、グループエクスペリエンスでない限り、あなた以外の誰もがエクスペリエンスを表示またはプレイできないようにします。その場合、あなたと任命された制作者（エクスペリエンスの編集権限を持つ必要なランクを持つ）だけがエクスペリエンスをプレイできます。詳細については、 グループクエクスペリエンス に関するヘルプ記事をご覧ください。

新しいエクスペリエンスを制作する方法

    1. ウェブサイトの上部にある青いバーの 制作 をクリックします。
    2. マイクリエーションタブで、 エクスペリエンス がまだハイライト表示されていない場合はクリックします。
    3. 新しいエクスペリエンスを制作 をクリックします。
    4. 新規作成の設定とテンプレートを選択します。
    5. エクスペリエンスを制作 をクリックします。

これにより、以前の制作ページに戻ります。 必要に応じて、エクスペリエンスを今すぐ公開することも、作業を完了するまで待つこともできます。 構築を開始するには、 エクスペリエンス リストで新しいエクスペリエンスのエントリの右側にある 編集 ボタンをクリックします。

注：いつでも新しいエクスペリエンスを制作できますが、エクスペリエンスを公開するプロセスを使用して公開するまで、他のRobloxプレイヤーは楽しむことはできません。
