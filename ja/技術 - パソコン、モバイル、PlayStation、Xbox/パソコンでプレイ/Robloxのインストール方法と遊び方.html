Robloxのインストール方法と遊び方

Robloxの遊び方

  1. Roblox プレーヤーをインストールする
    • Microsoft Edge、Firefox、Chrome などの最新のブラウザを使用して、 Roblox ウェブサイト にアクセスします。
    • Robloxにログインしたら、任意のバーチャル空間にアクセスして、緑色のプレイボタンをクリックします。
    • Roblox プレーヤーがインストールされていることを通知するポップアップ ウィンドウが表示されます。
  2. Microsoft Edge、Firefox、Chromeなどの最新のブラウザを介してプレーヤーを起動します。
    • Robloxウェブサイトで任意のバーチャル空間にアクセスする。
    • [ 再生 ] を選択します。
    • プログラムの選択を求める 2 番目のポップアップ ウィンドウが表示される場合があります。 Robloxを選択します。 選択を記憶するを選択した場合、今後このアクションを実行する必要はありません。

MacOS上の Firefox 経由で Roblox をインストールしてプレイする方法

1. Robloxプレーヤーをインストールする

  • Firefox を使用して Roblox Web サイトにアクセスします。
  • Roblox にログインしたら、いずれかのエクスペリエンスにアクセスし、緑色の [ プレイ ] ボタンをクリックします。
  • Roblox Player がインストールされていることを通知するポップアップ ウィンドウが表示されます。

2. プレーヤーを起動する。

  • [ 常にこのアプリケーションを使用して roblox-player リンクを開く ] をオンにし、[ 選択 ] をクリックして Finder ウィンドウを表示します。
  • Finder ウィンドウでキーボード ショートカット CMD+SHIFT+G を使用してパス名入力を表示します。
  • パス名入力に「 /Applications/Roblox.app/Contents/MacOS/Roblox.app 」と入力し、 RETURN
    を押します。
  • 次に、Finder ウィンドウで Roblox.app を選択し、 開く
    をクリックします。
  • [アプリケーションの選択] ダイアログで、Roblox.app を選択し、 リンクを開く
    をクリックします。
  • Roblox が起動され、再生ボタンをクリックして Roblox エクスペリエンスを開くとご利用できるようなります。

Roblox Studioのインストール方法

独自のRobloxのバーチャル空間を制作するには、Roblox Studio（ロブロックス・スタジオ）をインストールする必要があります。 その方法は次のとおりです。

  1. Roblox ウェブサイト にアクセスし、アカウントにログインします。
  2. Robloxにログインしたら、任意のバーチャル空間にアクセスし、右上隅にある をクリックして、 編集 を選択します。
  3. Roblox Studio がインストールされていることを通知するポップアップ ウィンドウが表示されます。
  4. インストールすると、Studio が自動的に開きます。
