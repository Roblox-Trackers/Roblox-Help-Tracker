Roblox Studioの一般的な問題

次の記事では、以下の問題の修正に効果的であることが証明されているいくつかの方法を提案します。

  • Roblox Studioにログインしようとするが、ログインできない場合
  • 空のスカイボックスまたはPlace1と呼ばれるものを読み込むためだけにプレースを編集しようとした場合
  • 空白であるはずがないところで、Studioの一部が空白の場合
  • Robloxへの公開に問題がある場合
  • Roblox Studioにすでにログインしている場合でもログインするように指示される場合
  • これらの手順は、ゲストとしてログインしている場合やゲームに参加するなど、Studioに関連しない問題の解決の一部に役立つ場合があります
  • その他：Studioで別の問題が発生している場合は、以下の手順も解決に役立つ場合があります

以下の各ステップを順番に1つずつ試してください。 1つの解決策が機能しない場合は、次の解決策に進んでください！

再起動

最初にコンピューターを再起動して、プレイしてみます。 RobloxまたはRoblox Studio関連の問題を解決するために必要なのは再起動だけで十分な場合があります。

プロキシ設定の確認

上記の手順でRoblox Studioの問題を解決できなかった場合、コンピューターがプロキシサーバー経由で接続しようとしていないことを明確にする必要があるかも知れません。 次の手順に従ってください。

Windows 7 またはそれ以前

    1. [スタート]メニューに移動します
    2. 「プロキシ」と検索すると、「プロキシ設定の変更」が表示されます。 それをクリックします
    3. [LAN設定] をクリックし、そのタブのすべてのチェックマークを解除します
    4. OK をクリック

Windows 8 または 8.1

    1. 画面の左下隅をクリックして、タイル表示されたスタート画面に移動します
    2. 「プロキシ」と入力すると、「プロキシ設定の変更」が表示されます。 それをクリックします
    3. [LAN設定] をクリックし、そのタブのすべてのチェックマークを解除します
    4. OK をクリック

Windows 10

    1. キーボードのWindowsキーSを押します。 または、Cortanaの「Ask Me Anything」バーがタスクバーで使用できる場合は、それをクリックするだけです。
    2. 「プロキシ」と入力すると、「プロキシ設定の変更」が表示されます。 それをクリックします
    3. 「セットアップスクリプトを使用する」と「プロキシサーバーを使用する」がオフに設定されていることを明確にしてくだい
    4. 下にスクロールして 保存 をクリックします

Windows のシステム クロックを確認する

不思議に思われるかも知れませんが、Windowsシステム時刻が間違った時刻または日付に設定されているために、さまざまな問題が発生する可能性があります。 これが原因である場合は、修正するだけですべての問題を改善できます。 これは手動で行うこともできますが、サーバーで自動的に更新されるように設定することもできます。 これを行う方法は次のとおりです。

Windows 8 またはそれ以前

    1. 右下隅のウィンドウの時計を右クリック
    2. [日付および時刻の調整] をクリックします
    3. [インターネットタイム] タブを選択します
    4. 時刻と日付をインターネットサーバーと同期できます。

Windows 10

    1. 右下隅のウィンドウの時計を右クリック
    2. 「時刻を自動的に設定」がオンになっていて、正しいタイムゾーンが選択されていることを確認してください

対処方法 - Mac

Roblox アプリケーションと Studio は 10.13 ( High Sierra ) 以降にインストールされます。 Mac でこの問題を解決するには、再インストールしてください。 この問題に関するヘルプが必要な場合は、 こちら をクリックしてください。

今後の問題を最小限に抑えるには

Robloxをインストールまたはアップデートする前に、コンピューター上のすべてのプログラムが閉じられていることを常に確認してください。
