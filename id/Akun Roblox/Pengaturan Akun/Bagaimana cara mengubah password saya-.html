Bagaimana cara mengubah password saya?

Mengubah Password Anda

  1. Masuk ke akun anda
    (jika anda tidak dapat masuk, coba langkah-langkah ini untuk memulihkan password anda .)
  2. Buka Pengaturan akun
    • Peramban - temukan ikon roda gigi yang terletak di pojok kanan atas situs
    • Aplikasi Seluler - temukan ikon tiga titik untuk Lainnya
  3. Pilih tab Info Akun
  4. Pilih tombol Ubah Password (ikon terlihat seperti pensil dan kertas)
  5. Masukkan password Roblox anda saat ini
  6. Masukkan password baru Anda dua kali untuk keperluan konfirmasi

Untuk keamanan tambahan, buat password baru yang belum pernah digunakan di situs web lain.

Menambahkan Keamanan

Jika Anda belum melakukannya, lanjutkan dan tambahkan alamat email lalu verifikasi. Alamat email akan sangat meningkatkan kemampuan kami untuk membantu apabila akun Anda hilang. Anda dapat mengikuti langkah-langkah dalam artikel verifikasi email ini.

Pastikan Anda telah mengaktifkan Verifikasi 2 Langkah (2SV) sebagai langkah keamanan tambahan untuk mencegah orang lain masuk ke akun Anda. Lihat Tambahkan Verifikasi 2 Langkah ke Akun Anda untuk informasi tentang cara mengaktifkannya.
