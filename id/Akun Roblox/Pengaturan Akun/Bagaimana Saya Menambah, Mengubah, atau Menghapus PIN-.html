Bagaimana Saya Menambah, Mengubah, atau Menghapus PIN?

Anda memiliki opsi untuk menambahkan PIN empat digit untuk mengunci pengaturan Anda. Setelah PIN ditambahkan, PIN akan diperlukan untuk melakukan perubahan apa pun pada pengaturan akun. Fitur ini melindungi password, alamat email, dan pengaturan privasi Anda.

Bagaimana cara menambahkan PIN?

  1. Masuk ke akun Anda
    (Jika Anda tidak dapat masuk, coba langkah-langkah ini untuk memulihkan password Anda .)
  2. Buka akun Pengaturan
    • Peramban - temukan ikon roda gigi yang terletak di pojok kanan atas situs
    • Aplikasi Seluler - temukan ikon tiga titik untuk Lainnya
  3. Pilih tab Kontrol Orang Tua
  4. Tombol sekelar untuk mengaktifkan PIN
  5. Buat dan konfirmasi PIN baru Anda

Jangan pernah memberi tahu siapa pun PIN Anda - ini bahkan lebih penting daripada password Anda.

Bagaimana cara menghapus PIN?

  1. Masuk ke akun Anda
    (Jika Anda tidak dapat masuk, coba langkah-langkah ini untuk memulihkan password Anda .)
  2. Buka akun Pengaturan
    • Peramban - temukan ikon roda gigi yang terletak di pojok kanan atas situs
    • Aplikasi Seluler - temukan ikon tiga titik untuk Lainnya
  3. Pilih tab Kontrol Orang Tua
  4. Tombol sakelar untuk mematikan PIN
  5. Masukkan PIN saat ini dan pilih Buka Kunci

Bagaimana cara mengubah PIN?

Untuk mengubah PIN, pertama-tama ikuti langkah-langkah Bagaimana cara menghapus PIN . Lalu, ikuti langkah-langkah Bagaimana cara menambahkan PIN untuk membuat PIN baru.
