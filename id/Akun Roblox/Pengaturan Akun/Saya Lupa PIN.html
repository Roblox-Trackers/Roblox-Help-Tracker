Saya Lupa PIN

Setelah PIN ditambahkan ke akun, PIN akan diperlukan untuk melakukan perubahan pada pengaturan akun.

Saat ini, pengguna dan orang tua/walinya tidak dapat mengatur ulang PIN secara manual jika lupa. Dukungan Pelanggan perlu membantu mengatur ulang PIN yang terlupakan. Untuk menghubungi Dukungan Pelanggan, harap gunakan formulir kontak .

Catatan: Dukungan Pelanggan akan meminta informasi khusus untuk memastikan bahwa mereka berbicara dengan orang tua/wali sebelum PIN disetel ulang.
