Saya Lupa PIN

Setelah PIN ditambahkan ke akun, PIN akan diperlukan untuk melakukan perubahan pada pengaturan akun.

Saat ini, jika lupa, para pengguna dan orang tua/wali tidak dapat mengatur ulang PIN secara manual. Dukungan Pengguna perlu membantu mengatur ulang PIN yang terlupakan. Untuk menghubungi Dukungan Pengguna, silakan gunakan formulir kontak .

Catatan: Dukungan Pengguna akan meminta informasi khusus untuk memastikan bahwa mereka berbicara dengan orang tua/wali sebelum PIN disetel ulang.
