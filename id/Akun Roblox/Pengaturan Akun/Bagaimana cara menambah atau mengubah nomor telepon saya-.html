Bagaimana cara menambah atau mengubah nomor telepon saya?

Fitur ini tersedia untuk pemain berusia 13 tahun atau lebih. Suatu nomor telepon hanya dapat digunakan di satu akun Roblox.

Menambahkan Nomor Telepon

  1. Masuk ke akun Anda
    (Jika Anda tidak dapat masuk, coba langkah berikut untuk memulihkan kata sandi Anda )
  2. Buka Pengaturan akun
    • Peramban - temukan ikon roda gigi yang terletak di pojok kanan atas situs
    • Aplikasi Seluler - temukan ikon tiga titik untuk Lainnya
  3. Pilih label Info Akun
  4. Pilih tombol Tambah Telepon
    1. Pilih lokasi Anda dari menu tarik-turun
    2. Masukkan nomor telepon dan kata sandi akun
    3. Pilih Tambah Telepon

Mengubah atau Menghapus Nomor Telepon

  1. Masuk ke akun Anda
    (jika Anda tidak dapat masuk, coba langkah-langkah ini untuk memulihkan kata sandi Anda .)
  2. Buka Pengaturan akun
    • Peramban - temukan ikon roda gigi yang terletak di pojok kanan atas situs
    • Aplikasi Seluler - temukan ikon tiga titik untuk Lainnya
  3. Pilih label Info Akun
  4. Pilih tombol Perbarui Ponsel (ikon terlihat seperti pensil dan kertas)
    1. Pilih lokasi Anda dari menu tarik-turun
    2. Masukkan nomor telepon dan kata sandi akun
    3. Pilih Edit Telepon atau Hapus Nomor Telepon

Perintah untuk Pesan Teks

Berikut adalah beberapa perintah berguna untuk memulai dan menghentikan pesan teks yang mungkin Anda terima dari Roblox:

  • STOP, STOPALL, UNSUBSCRIBE, CANCEL, END, dan QUIT
    Perintah satu kata ini akan menghentikan pesan teks di masa mendatang. Perhatikan bahwa hanya pesan satu kata yang akan memicu pemblokiran. Misalnya, membalas "STOP" akan memastikan Anda tidak menerima pesan lebih lanjut, tetapi membalas "STOP PLEASE" tidak.
  • MULAI, YA, dan BERHENTI
    Perintah ini akan memungkinkan Anda memilih kembali untuk menerima pesan teks.
  • BANTUAN dan INFO
    Perintah ini akan memberikan informasi Anda dapat menggunakan perintah di atas untuk mengontrol pengiriman pesan.
