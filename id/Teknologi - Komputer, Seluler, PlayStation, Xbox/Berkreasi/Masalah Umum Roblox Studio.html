Artikel berikut akan menyarankan beberapa metode yang terbukti efektif untuk memperbaiki masalah berikut:

  • Anda mencoba masuk ke Roblox Studio tetapi tidak bisa
  • Anda mencoba mengedit tempat hanya untuk memuat skybox kosong atau sesuatu yang disebut Place1
  • Bagian Studio kosong yang seharusnya tidak
  • Anda mengalami kesulitan menerbitkan ke Roblox
  • Roblox Studio memberi tahu Anda untuk masuk meskipun Anda sudah melakukannya
  • Langkah-langkah ini dapat membantu beberapa masalah yang tidak terkait dengan Studio seperti memasuki pengalaman sebagai Tamu bahkan saat Anda masuk
  • Lainnya: Jika Anda mengalami masalah lain dengan Studio, langkah-langkah di bawah juga dapat membantu menyelesaikannya

Menyalakan ulang

Pertama reboot komputer Anda dan coba mainkan. Terkadang hanya ini yang diperlukan untuk menyelesaikan masalah terkait Roblox atau Roblox Studio.

Periksa Pengaturan Proksi Anda

Jika langkah-langkah di atas tidak menyelesaikan masalah Roblox Studio Anda, mungkin Anda perlu memastikan bahwa komputer Anda tidak mencoba terhubung melalui server proxy. Silakan ikuti langkah-langkah ini:

Windows 7 atau Sebelumnya

    1. Buka menu Mulai
    2. Cari "proxy" dan "Configure Proxy Server" akan muncul, klik di atasnya
    3. Klik Pengaturan LAN dan hapus semua tanda centang di tab tersebut
    4. Klik OK

Windows 8 atau 8.1

    1. Pergi ke layar awal ubin dengan mengklik sudut kiri bawah layar Anda
    2. Ketik "proxy" dan "Configure Proxy Server" akan muncul, klik di atasnya
    3. Klik Pengaturan LAN dan hapus semua tanda centang di tab tersebut
    4. Klik OK

Windows 10

    1. Tekan Tombol Windows + S di keyboard Anda. Atau, jika bilah "Ask Me Anything" Cortana tersedia di bilah tugas Anda, Anda cukup mengekliknya
    2. Ketik "proxy" dan "Ubah Pengaturan Proxy" akan muncul, klik di atasnya
    3. Pastikan "Use Setup Script" dan "Use a Proxy Server" dinonaktifkan
    4. Gulir ke bawah dan klik Simpan

Periksa Jam Sistem Windows Anda

Ini mungkin terdengar aneh, tetapi berbagai masalah sebenarnya dapat disebabkan oleh jam sistem Windows Anda yang disetel ke waktu atau tanggal yang salah. Jika ini penyebabnya, cukup perbaiki semuanya untuk memperbaiki situasi. Meskipun hal ini dapat dilakukan secara manual, Anda juga dapat mengaturnya untuk memperbarui secara otomatis dengan server, sehingga memastikannya tidak ketinggalan dan mulai menimbulkan masalah lagi. Inilah cara melakukannya:

Windows 8 atau Sebelumnya

    1. Klik kanan pada jam windows di pojok kanan bawah
    2. Klik Sesuaikan Tanggal/Waktu
    3. Pilih tab Waktu Internet
    4. Here you should be able to synchronize your time and date with an internet server

Windows 10

    1. Klik kanan pada jam windows di pojok kanan bawah
    2. Pastikan "Atur Waktu Secara Otomatis" diaktifkan dan Anda telah memilih zona waktu yang benar
