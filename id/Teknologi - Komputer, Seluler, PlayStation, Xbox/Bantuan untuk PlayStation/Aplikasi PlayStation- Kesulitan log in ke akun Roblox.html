Aplikasi PlayStation: Kesulitan log in ke akun Roblox

Untuk menghubungkan akun Roblox kamu ke akun PlayStation Network di aplikasi, kamu harus mengetahui nama pengguna dan kata sandi Roblox-mu. Jika lupa kata sandi, dan mengalami kesulitan untuk log in, kamu harus mengubahnya terlebih dahulu. Klik di sini untuk mengetahui cara mereset kata sandi.

Catatan: Harap diingat, fitur Facebook Connect tidak tersedia di PlayStation, jadi kamu harus log in dengan kata sandi akun Roblox.

Memutuskan Koneksi antara Akun Roblox dan Akun PlayStation

      1. Di aplikasi, sorot lalu pilih tombol Menu ( ... )
      2. Sorot lalu pilih opsi Log Out

Kamu juga dapat memutuskan koneksi akun dari pengaturan Roblox .

      1. Buka tab Keamanan
      2. Buka bagian Lokasi Log In Akun Kamu
      3. Cari Aplikasi PlayStation Roblox lalu klik ikon panah di sampingnya
      4. Klik Putuskan Koneksi PlayStation untuk memutuskan koneksi
