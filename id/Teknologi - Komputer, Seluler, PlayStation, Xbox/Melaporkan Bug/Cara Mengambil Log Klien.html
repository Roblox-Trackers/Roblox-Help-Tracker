Cara Mengambil Log Klien

Mungkin ada beberapa situasi di mana karyawan Roblox meminta klien melakukan log melalui email Layanan Pelanggan atau postingan DevForum untuk membantu menyelidiki masalah. Artikel ini menjelaskan cara mengambil log tersebut.

Catatan: Kirimkan log ini hanya jika diminta oleh karyawan Roblox!

Cara Mengambil Log Roblox

Roblox Player dan Roblox Studio Log keduanya ditulis ke folder yang sama.

  • Di PC, tekan tombol WINDOWS dan R secara bersamaan. Kotak bernama Jalankan akan muncul. Salin dan tempel atau ketik berikut ini tanpa tanda kutip:

    • Untuk Win32: "%localappdata%\Roblox\logs"

      \Packages\ROBLOXCorporation.ROBLOX_55nm5eh3cm0pr\LocalState\logs”\Packages\ROBLOXCorporation.ROBLOX_55nm5eh3cm0pr\LocalState\logs”

  • Di Mac, navigasikan ke Macintosh HD > Pengguna > [profil pengguna Anda] > Perpustakaan > Log > Roblox

  • Saat ini kami tidak memiliki cara mudah bagi Anda untuk mengunggah log Android atau iOS. Maaf, kami sedang mengusahakannya!

Kemungkinan akan ada banyak file log di folder ini

  • Log Roblox akan terlihat seperti: “0.<major ver>.<minor ver>.<revision>_<date>_Player_<garbage>_last.log

    • Misalnya 0.555.1.5550874_20221214T162601Z_ Pemain _9aaf3_last.log

  • Log Roblox Studio akan terlihat sama kecuali tertulis "Studio", bukan "Player

    • Misalnya 0.555.1.5550874_20221214T162601Z_ Studio _9aaf3_last.log

  • Log bootstrapper akan terlihat seperti bootstrapper_<garbage>_1.txt

    • Kecuali masalah terjadi saat meluncurkan Roblox atau RobloxStudio dari browser web Anda, atau selama pembaruan, kemungkinan besar Anda tidak perlu mengirim log bootstrapper.

Untuk mengidentifikasi file log yang benar untuk dikirim, Anda juga bisa

  • Gunakan stempel waktu file untuk menentukan file log mana yang terkait dengan insiden yang ingin Anda laporkan
  • Hapus semua file di direktori ini, buat ulang masalah yang ingin Anda laporkan, lalu kirimkan semua konten di folder ini
  • Kalau bingung, tinggal sortir berdasarkan tanggal dan kirim 10 file terbaru.

Termasuk file log saat melaporkan bug

Saat melaporkan bug di Roblox Forum Pengembang , Anda akan memiliki opsi untuk mengunggah file log bersama dengan laporan bug Anda. Jika Anda perlu mengunggah log setelah membuat laporan bug awal, kirim DM ke grup file bug , sertakan link ke postingan laporan bug asli, dan tandai karyawan Roblox yang relevan:

Unggah log hanya dengan mengirim ke grup file bug dan lakukan hanya jika Karyawan Roblox meminta Anda. Untuk mengidentifikasi Karyawan Roblox di Forum Pengembang, pastikan tertulis "Staf Roblox" di sebelah nama akun mereka:
