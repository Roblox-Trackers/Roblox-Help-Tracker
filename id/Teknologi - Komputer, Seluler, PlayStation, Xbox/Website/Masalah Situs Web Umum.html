Masalah Situs Web Umum

Artikel berikut akan menyarankan beberapa metode yang terbukti efektif untuk memperbaiki masalah berikut:

  • Situs web ditampilkan dengan tidak benar
  • Beberapa halaman tidak dimuat sama sekali atau menampilkan kesalahan saat memuat
  • Tombol atau tautan tidak memungkinkan Anda untuk mengkliknya
  • Anda mencoba masuk ke situs web Roblox tetapi tidak bisa meskipun memiliki nama pengguna dan password yang benar
  • Lainnya: Jika Anda mengalami masalah lain dengan situs web, langkah-langkah di bawah juga dapat membantu menyelesaikannya. Selain itu, beberapa artikel yang tidak terkait dengan situs web dapat ditautkan ke sini untuk beberapa langkah

Apa yang harus dilakukan

Silakan coba setiap langkah di bawah ini, satu per satu, secara berurutan. Jika satu solusi tidak berhasil, lanjutkan ke solusi berikutnya!

Bersihkan File dan Cookie Internet Sementara Anda

Tautan berikut akan membantu Anda menghapus file internet sementara (juga disebut sebagai "cache") dan cookie untuk browser web utama PC. Pastikan untuk melakukan ini untuk browser apa pun tempat Anda ingin meluncurkan Roblox. (Roblox tidak mendukung Internet Explorer)

      • Mozilla Firefox - File Internet Sementara (Dari Mozilla.com)
      • Mozilla Firefox - Cookie (Dari Mozilla.com)
      • Google Chrome - Both (From Google.com)
      • Microsoft Edge - Keduanya (Dari Microsoft.com)
      • Safari - Cookie (Dari Apple.com)
      • Browser lain: Jika Anda menggunakan browser lain atau browser versi lama, harap telusuri informasi tentang cara menghapus cache/data browser/berkas internet sementara untuk browser/versi spesifik Anda.

Setelah Anda menyelesaikan hal di atas untuk semua browser web yang relevan, harap mulai ulang komputer Anda dan lihat apakah masalah Anda telah teratasi.

Jika Anda ingin menghapus File Internet Sementara Windows secara manual, Anda dapat melakukannya dengan langsung membuka folder File Internet Sementara di hard drive Anda. Lokasi dan nama folder bergantung pada versi Windows dan apakah Anda menggunakan profil pengguna atau tidak.

Catatan: Anda mungkin perlu mengubah pengaturan Windows Explorer untuk menampilkan semua jenis file (termasuk file sistem yang dilindungi) untuk mengakses folder ini.

Untuk Windows 10

Jika Anda memiliki Windows 10, maka File Internet Sementara ada di lokasi berikut (perhatikan bahwa pada PC Anda file tersebut dapat berada di drive lain, bukan drive C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Untuk Windows 7 dan 8

Jika Anda memiliki Windows 7 atau Windows 8 maka File Internet Sementara ada di lokasi ini (perhatikan bahwa mereka dapat berada di drive lain, bukan drive C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Jika masalah teratasi, coba aktifkan kembali satu per satu hingga masalah Anda mulai terjadi lagi. Setelah masalah dimulai kembali, Anda seharusnya menemukan yang tidak kompatibel dengan Roblox.

Tautan berikut dapat membantu Anda dengan browser web PC paling populer:

        • Microsoft Edge (Dari Microsoft.com)

        • Browser lain: Jika Anda menggunakan browser yang berbeda atau versi browser yang lebih lama, harap telusuri informasi tentang cara menonaktifkan Ekstensi/Plugin/Add-On untuk browser/versi spesifik Anda.
