Masalah Masuk atau Keluar di Browser

Jika Anda mengalami masalah saat masuk atau keluar dari Roblox di browser, pertama-tama cobalah langkah-langkah berikut untuk menyelesaikan masalah umum situs web .

Jika itu tidak membantu, coba langkah-langkah ini:

  1. Jika Anda mengalami kesulitan dalam verifikasi captcha atau kotak captcha Anda tidak dapat dimuat, pastikan untuk menambahkan domain berikut ke daftar yang diizinkan:
    • .arkoselabs.com
    • .funcaptcha.com
  2. Jika langkah-langkah di atas tidak membantu, periksa apakah jam komputer Anda saat ini akurat. Walaupun kelihatannya aneh, jam komputer yang sedikit mati dapat menyebabkan berbagai masalah. Halaman-halaman berikut dapat membantu proses ini:
    • Windows (From Microsoft.com)
    • Mac (From Apple.com)
  3. Jika Anda masih tidak dapat masuk, Anda mungkin perlu menyetel ulang sandi .
