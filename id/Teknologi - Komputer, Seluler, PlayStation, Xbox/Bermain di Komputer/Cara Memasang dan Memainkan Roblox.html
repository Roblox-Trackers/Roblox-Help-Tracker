Cara Memasang dan Memainkan Roblox

Cara Memainkan Roblox

  1. Instal Pengguna Roblox
    • Buka situs web Roblox menggunakan browser modern seperti Microsoft Edge, Firefox, atau Chrome
    • Setelah masuk ke Roblox, kunjungi pengalaman apa saja dan klik tombol hijau Mainkan
    • Jendela pop-up akan muncul memberi tahu Anda bahwa Roblox Player sedang menginstal
  2. Luncurkan pemutar melalui browser modern apa pun seperti Microsoft Edge, Firefox, atau Chrome
    • Buka game apa pun di Situs Web Roblox
    • Pilih Mainkan
    • Jendela pop-up sekunder mungkin muncul meminta Anda untuk memilih program. Pilih Roblox dan konfirmasi. Jika Anda memilih untuk mengingat pilihan Anda, Anda tidak perlu melakukan tindakan ini di masa mendatang

Cara Memasang dan Memainkan Roblox melalui Firefox di MacOS

1. Instal Pengguna Roblox

  • Buka situs web Roblox menggunakan Firefox
  • Setelah masuk ke Roblox, kunjungi pengalaman apa saja dan klik tombol hijau Mainkan
  • Jendela pop-up akan muncul memberi tahu Anda bahwa Roblox Player sedang menginstal

2. Luncurkan pemutar

  • Centang Selalu gunakan aplikasi ini untuk membuka tautan roblox-player , dan klik Pilih untuk membuka jendela Finder
  • Di jendela Finder, gunakan pintasan keyboard CMD+SHIFT+G untuk memunculkan input pathname
  • Di input pathname, masukkan /Applications/Roblox.app/Contents/MacOS/Roblox.app , dan tekan RETURN
  • Kemudian di jendela Finder, Pilih Roblox.app, dan klik Buka
  • Dalam dialog Pilih aplikasi, pilih Roblox.app dan klik Buka Tautan
  • Roblox harus berjalan, dan mengklik tombol putar untuk membuka pengalaman Roblox akan berfungsi di masa mendatang.

Cara Memasang Roblox Studio

Anda harus menginstal Roblox Studio untuk membuat pengalaman Roblox Anda sendiri. Inilah cara Anda melakukannya.

  1. Buka situs web Roblox dan masuk ke akun Anda
  2. Setelah masuk ke Roblox, kunjungi salah satu pengalaman <0>Anda , klik <1>di sudut kanan atas, dan pilih <0>Edit
  3. Jendela pop-up akan muncul memberi tahu Anda bahwa Roblox Studio sedang menginstal
  4. Setelah diinstal, Studio akan terbuka secara otomatis
