Pesan Hyperion

Artikel ini berisi pesan anti-kecurangan Hyperion yang mungkin ditampilkan saat terjadi crash, diikuti dengan langkah-langkah pemecahan masalah untuk membantumu dalam memecahkan masalahnya.

Tipe Crash : Kegagalan waktu habis

Pesan Pengguna : Roblox has timed out. Please restart your client. (Roblox kehabisan waktu. Mulai ulang klien.)

Langkah-Langkah Pemecahan Masalah :

Jika menemui pesan "Roblox has timed out. Please restart your client" (Roblox kehabisan waktu. Mulai ulang klien.), itu artinya klien Roblox perlu dimulai ulang. Jika masalah tetap terjadi meskipun klien sudah dimulai ulang pada penggunaan biasa, hubungi tim dukungan pelanggan kami untuk mendapatkan bantuan lebih lanjut.

Tipe Crash : Mesin virtual terdeteksi

Pesan Pengguna : Roblox cannot run inside of a virtual machine. (Roblox tidak dapat dijalankan di dalam mesin virtual.)

Langkah-Langkah Pemecahan Masalah :

Jika kamu menemui pesan yang menyatakan “Virtual machine detected” (Mesin virtual terdeteksi), itu artinya Roblox tidak mendukung mesin virtual (VM) yang digunakan. Roblox memerlukan passthrough GPU agar bisa berfungsi dengan benar di dalam VM. Passthrough GPU memberi VM akses langsung ke GPU fisik, tanpa melalui lapisan hypervisor. Berikut cara mengaktifkan passthrough GPU:

      1. Pastikan bahwa perangkat keras yang digunakan mendukung passthrough GPU.
      2. Update pengaturan BIOS/UEFI untuk mengaktifkan ekstensi virtualisasi (misalnya, Intel VT-x atau AMD-V).
      3. Konfigurasikan passthrough GPU di dalam perangkat lunak VM.

Ikuti langkah-langkah tersebut untuk memastikan Roblox berjalan lancar di dalam lingkungan yang divirtualisasi sehingga memberikan akses yang diperlukan ke sumber daya GPU.

Tipe Crash : Badware terdeteksi

Pesan Pengguna : Roblox has crashed due to incompatible software: <path to offending software> (Roblox mengalami crash karena perangkat lunak yang tidak kompatibel: <jalur ke perangkat lunak penyebab masalah>)

Langkah-Langkah Pemecahan Masalah :

Jika menemui pesan "Roblox has crashed due to incompatible software:" (Roblox mengalami crash karena perangkat lunak yang tidak kompatibel:) diikuti dengan jalur ke perangkat lunak penyebab masalah, itu berarti perangkat lunak yang disebutkan dalam jalur file yang ditampilkan sedang menyebabkan terjadinya masalah pada Roblox. Untuk memperbaiki masalah ini, kamu perlu menghentikan aktivitas perangkat lunak yang bersangkutan saat Roblox sedang digunakan. Kamu bisa melakukannya secara manual dengan mematikan perangkat lunak atau melakukan clean boot di Windows. Metode ini memastikan hanya perangkat lunak yang diperlukan saja yang aktif di sistem guna meminimalkan konflik. Berikut cara melakukan clean boot sesuai rekomendasi Windows :

      1. Tekan "tombol Windows + R" untuk membuka dialog Jalankan.
      2. Ketik "msconfig", lalu tekan tombol Enter.
      3. Di jendela Konfigurasi Sistem, buka tab "Layanan".
      4. Centang kotak berlabel "Sembunyikan semua layanan Microsoft".
      5. Klik "Nonaktifkan semua".
      6. Pindah ke tab "Startup".
      7. Klik "Buka Manajer Tugas".
      8. Nonaktifkan semua item yang berjalan pada saat startup.
      9. Tutup Manajer Tugas dan Konfigurasi Sistem.
      10. Mulai ulang komputer.

Lakukan langkah-langkah ini untuk memastikan tidak ada perangkat lunak lain yang mengganggu Roblox sehingga bisa berjalan lancar tanpa mengalami crash.

Tipe Crash : Integritas boot menurun

Pesan Pengguna : Critical system files are corrupt. Roblox cannot run. Please repair or reset your Windows installation. (File sistem penting rusak. Roblox tidak bisa dijalankan. Lakukan perbaikan atau reset instalasi Windows.)

Langkah-Langkah Pemecahan Masalah :

Roblox tidak bisa berfungsi dengan benar ketika Windows diatur dalam mode debug. Agar Roblox bisa digunakan, pastikan Windows berjalan normal. Ikuti langkah-langkah berikut untuk memverifikasi dan menonaktifkan mode debug di Windows .

      1. Tekan "Tombol Windows+R" untuk membuka kotak Jalankan.
      2. Ketik "msconfig", lalu tekan tombol Enter.
      3. Pilih "tab Boot", lalu pilih opsi Tingkat Lanjut.
      4. Hapus centang di kotak Debug.
      5. Pilih "OK".
      6. Pilih Terapkan, lalu "OK".
      7. Mulai ulang komputer.

Langkah-langkah ini akan memastikan Windows tidak menjalankan mode debug sehingga memungkinkan kamu menggunakan Roblox tanpa masalah.

Tipe Crash : (OBS) Open Broadcaster Software

Pesan Pengguna : OBS in-game capture is known to have issues with Roblox. (Perekam dalam-game OBS diketahui mengalami masalah saat digunakan dengan Roblox.) Gunakan perekam jendela atau desktop.

Langkah-Langkah Pemecahan Masalah :

Jika menemui pesan, "OBS in-game capture is known to have issues with Roblox" (Perekam dalam-game OBS diketahui mengalami masalah saat digunakan dengan Roblox.), itu artinya Roblox tidak mendukung fitur perekam dalam-game dari Open Broadcaster Software (OBS). Untuk melakukan streaming atau merekam Roblox, kamu perlu menggunakan mode perekam jendela atau desktop. Ikuti instruksi berikut untuk menyiapkan perekam desktop :

      1. Buka Open Broadcaster Software (OBS).
      2. Di bagian "Sources" (Sumber), klik ikon "+".
      3. Pilih "Display Capture" (Perekaman Layar) dari menu.
      4. Pilih layar yang ingin direkam (biasanya monitor utama).
      5. Sesuaikan pengaturan, seperti resolusi dan frame rate sesuai kebutuhan.
      6. Klik "OK" untuk mengonfirmasi sumber perekaman layar.
      7. Atur sumber perekaman layar sesuai yang diinginkan di adegan OBS kamu.
      8. Mulai streaming atau rekaman seperti biasa.

Dengan mode perekaman desktop, kamu bisa secara efektif melakukan streaming atau merekam Roblox tanpa mengalami masalah kompatibilitas dengan fitur perekaman dalam-game OBS.
