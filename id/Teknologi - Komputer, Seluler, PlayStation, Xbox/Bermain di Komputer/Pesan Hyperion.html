Pesan Hyperion

Artikel ini menginformasikan pesan crash anti kecurangan Hyperion yang mungkin ditampilkan saat terjadi crash, diikuti dengan langkah pemecahan masalah untuk membantumu dalam menyelesaikan masalah tersebut.

Tipe Crash : Kegagalan waktu habis

Pesan Pengguna : Roblox has timed out. Please restart your client. (Roblox kehabisan waktu. Mulai ulang Roblox.)

Langkah Pemecahan Masalah :

Jika kamu melihat pesan "Roblox has timed out. Please restart your client" (Roblox kehabisan waktu. Mulai ulang aplikasi.), ini artinya aplikasi Roblox perlu dimulai ulang. Jika masalah tetap terjadi meskipun aplikasi sudah dimulai ulang pada penggunaan biasa, hubungi tim Dukungan Pelanggan kami untuk mendapatkan bantuan lebih lanjut.

Tipe Crash : Mesin virtual terdeteksi

Pesan Pengguna : Roblox cannot run inside of a virtual machine. (Roblox tidak dapat dijalankan di dalam mesin virtual.)

Langkah Pemecahan Masalah :

Jika kamu melihat pesan “Virtual machine detected” (Mesin virtual terdeteksi), ini artinya Roblox tidak mendukung mesin virtual (VM) yang digunakan. Roblox memerlukan passthrough GPU agar dapat berfungsi dengan benar di dalam VM. Passthrough GPU memberi VM akses langsung ke GPU fisik, tanpa melalui lapisan hypervisor. Berikut cara mengaktifkan passthrough GPU:

      1. Pastikan bahwa hardware yang digunakan mendukung passthrough GPU.
      2. Update pengaturan BIOS/UEFI untuk mengaktifkan ekstensi virtualisasi (misalnya, Intel VT-x atau AMD-V).
      3. Konfigurasikan passthrough GPU di dalam software VM.

Ikuti langkah tersebut untuk memastikan Roblox berjalan lancar di dalam lingkungan yang divirtualisasi sehingga memberikan akses yang diperlukan ke sumber daya GPU.

Tipe Crash : Badware terdeteksi

Pesan Pengguna : Roblox has crashed due to incompatible software: <path to offending software> (Roblox mengalami crash karena software yang tidak kompatibel): <jalur ke software penyebab masalah>

Langkah Pemecahan Masalah :

Jika kamu melihat pesan "Roblox has crashed due to incompatible software:" (Roblox mengalami crash karena software yang tidak kompatibel:) diikuti dengan jalur ke software yang menyebabkan masalah, ini berarti software yang ditampilkan di jalur file yang menyebabkan terjadinya masalah dengan Roblox. Untuk memperbaiki masalah ini, kamu perlu menghentikan software tersebut saat Roblox sedang digunakan. Kamu dapat melakukannya secara manual dengan menghentikan software atau melakukan boot bersih di Windows. Metode ini memastikan hanya software yang diperlukan saja yang aktif di sistem untuk meminimalkan konflik. Berikut cara melakukan boot bersih yang direkomendasikan oleh Windows :

      1. Tekan "tombol Windows + R" untuk membuka dialog Jalankan.
      2. Ketik "msconfig" lalu tekan tombol Enter.
      3. Di jendela Konfigurasi Sistem, buka tab "Layanan".
      4. Centang kotak "Sembunyikan semua layanan Microsoft".
      5. Pilih "Nonaktifkan semua".
      6. Pindah ke tab "Startup".
      7. Klik "Buka Manajer Tugas".
      8. Nonaktifkan semua item Startup.
      9. Tutup Manajer Tugas dan Konfigurasi Sistem.
      10. Mulai ulang komputer.

Lakukan langkah ini untuk memastikan tidak ada software lain yang mengganggu Roblox agar Roblox dapat berjalan lancar tanpa mengalami crash.

Tipe Crash : Proses ditangguhkan

Pesan Pengguna : Roblox was stopped. Please ensure third-party software does not interfere with Roblox. (Roblox terhenti. Pastikan software pihak ketiga tidak mengganggu Roblox.)

Langkah Pemecahan Masalah :

Jika kamu melihat pesan "Roblox was stopped. Please ensure third-party software does not interfere with Roblox." (Roblox terhenti. Pastikan software pihak ketiga tidak mengganggu Roblox.), ini artinya software lain di komputermu menyebabkan masalah dengan Roblox. Untuk menyelesaikan masalah ini, kamu perlu menghentikan software lain saat menggunakan Roblox. Cara terbaik adalah dengan melakukan boot bersih di Windows untuk memastikan hanya software penting saja yang berjalan di sistem. Berikut cara melakukan boot bersih yang direkomendasikan oleh Windows :

      1. Tekan "tombol Windows + R" untuk membuka dialog Jalankan.
      2. Ketik "msconfig" lalu tekan tombol Enter.
      3. Di jendela Konfigurasi Sistem, buka tab "Layanan".
      4. Centang kotak "Sembunyikan semua layanan Microsoft".
      5. Pilih "Nonaktifkan semua".
      6. Pindah ke tab "Startup".
      7. Klik "Buka Manajer Tugas".
      8. Nonaktifkan semua item yang berjalan pada saat startup.
      9. Tutup Manajer Tugas dan Konfigurasi Sistem.
      10. Mulai ulang komputer.

Lakukan langkah ini untuk memastikan tidak ada software lain yang mengganggu Roblox agar Roblox dapat berjalan lancar tanpa mengalami crash.

Tipe Crash : Integritas boot terkompromi

Pesan Pengguna : Critical system files are corrupt. Roblox cannot run. Please repair or reset your Windows installation. (File sistem penting rusak. Roblox tidak dapat dijalankan. Lakukan perbaikan atau reset instalasi Windows.)

Langkah Pemecahan Masalah :

Roblox tidak dapat berfungsi dengan benar saat Windows diatur dalam mode debug. Agar Roblox dapat digunakan, pastikan Windows berjalan secara normal. Ikuti langkah berikut untuk memverifikasi dan menonaktifkan mode debug di Windows .

      1. Tekan "Tombol Windows+R" untuk membuka dialog Jalankan.
      2. Ketik "msconfig" lalu tekan tombol Enter.
      3. Pilih "tab Boot" lalu pilih opsi Tingkat Lanjut.
      4. Hapus centang di kotak Debug.
      5. Pilih "OKE".
      6. Pilih Terapkan lalu "OKE".
      7. Mulai ulang komputer.

Langkah ini akan memastikan Windows tidak menjalankan mode debug untuk memungkinkan kamu menggunakan Roblox tanpa masalah.

Tipe Crash : (OBS) Open Broadcaster Software

Pesan Pengguna : OBS in-game capture is known to have issues with Roblox. Please use window or desktop capture instead. (Perekam dalam game OBS mengalami masalah dengan Roblox. Gunakan perekam jendela atau desktop.)

Langkah Pemecahan Masalah :

Jika kamu melihat pesan, "OBS in-game capture is known to have issues with Roblox" (Perekam dalam game OBS mengalami masalah saat dengan Roblox.), ini artinya Roblox tidak mendukung fitur perekam dalam game dari Open Broadcaster Software (OBS). Untuk streaming atau merekam Roblox, kamu perlu menggunakan mode perekam jendela atau desktop. Ikuti instruksi berikut untuk menyiapkan perekam desktop :

      1. Buka Open Broadcaster Software (OBS).
      2. Di bagian "Sources" (Sumber), klik ikon "+".
      3. Di menu, pilih "Display Capture" (Perekaman Layar).
      4. Pilih layar yang ingin direkam (biasanya monitor utama).
      5. Sesuaikan pengaturan, seperti resolusi dan frame rate sesuai kebutuhan.
      6. Klik "OK" untuk mengonfirmasi sumber perekaman layar.
      7. Atur sumber perekaman layar sesuai yang diinginkan di adegan OBS kamu.
      8. Mulai streaming atau perekaman seperti biasa.

Dengan mode perekaman desktop, kamu dapat secara efektif melakukan streaming atau merekam Roblox tanpa mengalami masalah kompatibilitas dengan fitur perekaman dalam game OBS.
