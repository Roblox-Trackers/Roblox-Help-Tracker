Masalah Firewall dan Router

Program Firewall/Keamanan

Banyak perangkat lunak antivirus atau keamanan internet coba mengontrol program mana yang dapat mengakses internet. Seringkali, program ini melakukan kesalahan karena kewaspadaan dan secara default banyak program tidak diizinkan untuk diakses. Hal termudah untuk dicoba adalah mematikan perangkat lunak firewall internet Anda, untuk melihat apakah itu memungkinkan Anda bergabung dengan pengalaman. Jika ya, Anda dapat menyalakannya kembali dan mengikuti langkah-langkah ini untuk mengizinkan Roblox mengakses internet dengan aman:

  1. Temukan panel kontrol program firewall Anda. Seringkali Anda dapat menemukannya di bilah tugas di sudut kanan bawah desktop Anda. Jika tidak, panel kontrol program firewall mungkin dapat ditemukan di Panel Kontrol Windows Anda.
  2. Temukan bagian izin program .
  3. Temukan Roblox dan izinkan akses tak terbatas ke internet.

Catatan: Untuk bantuan dengan perangkat lunak tertentu, silakan lihat manual bantuan atau situs web program Anda.

Program Antivirus

Jika Anda memiliki program antivirus/antimalware yang berjalan di latar belakang, ada kemungkinan program tersebut bertindak sebagai semacam firewall yang terlalu berhati-hati yang mungkin mencegah beberapa komunikasi penting antara Roblox/Roblox Studio dan server kami. Jika demikian, Anda mungkin perlu memperbarui program atau menambahkan Roblox ke daftar yang diizinkan. Untuk bantuan terkait hal ini, silakan klik di sini .

Router

Klien Roblox tidak terikat untuk menggunakan port tertentu untuk data keluar, melainkan memindai untuk menemukan port percuma di mesin Anda. Jadi jika router Anda memblokir paket UDP di port APAPUN, Anda mungkin mengalami masalah. Port untuk klien Roblox dipilih dan ditetapkan secara dinamis untuk setiap sesi pengalaman.

Jika Anda menjalankan router, lihat manual router dan lihat cara membuka blokirnya. Jika Anda menjalankan modem DSL yang memblokir port, hubungi ISP Anda dan minta mereka memandu Anda tentang cara membuka blokirnya.
