aplikasi iOS: Kesalahan Memori

Jika terjadi kesalahan memori saat Anda mencoba memutar pengalaman melalui aplikasi Roblox Mobile, Anda mungkin menjalankan terlalu banyak aplikasi bersamaan. Untuk memperbaikinya, Anda mungkin perlu menutup beberapa program agar pengalaman dapat berjalan dengan baik atau memulai ulang perangkat. Jika Anda membagikan iDevice Anda dengan orang lain, pastikan untuk memeriksanya sebelum menutup aplikasi atau memulai ulang.
