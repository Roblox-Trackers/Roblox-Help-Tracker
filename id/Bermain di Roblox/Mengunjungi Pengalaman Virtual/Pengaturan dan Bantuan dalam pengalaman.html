Pengaturan dan Bantuan dalam pengalaman

Saat memainkan pengalaman, Anda dapat mengubah setelan dan mencari tahu cara melakukan tindakan seperti melompat atau memutar kamera.

Pengaturan

Menu ini berisi pengaturan untuk fungsi seperti volume, kamera, grafik, dan aksesibilitas.

Selama pengembang pengalaman belum mengunci mode kamera ke sesuatu yang spesifik, Anda dapat beralih antara Klasik dan Ikuti.

  • Dalam Mode Klasik , kamera tetap diam di satu tempat kecuali disesuaikan secara manual. Untuk mengubah sudut pandang avatar Anda, tahan tombol kanan pada mouse Anda dan seret ke sekeliling.
  • Dalam Mode Ikuti , kamera akan berputar mengikuti avatar Anda saat Anda bergerak ke kanan atau ke kiri untuk membantu target yang Anda tuju tetap terlihat.
    • Shift Lock Switch memungkinkan Anda beralih di antara mode kamera dengan menekan tombol Shift pada keyboard Anda.

Saat ini kami menyediakan tiga pengaturan aksesibilitas untuk menyesuaikan pengalaman Anda di Roblox dengan lebih baik.

  • Kurangi Gerakan menonaktifkan atau mengurangi efek layar tertentu dalam aplikasi dan dalam pengalaman. Misalnya, alih-alih menggeser menu dalam game ke layar Anda saat dibuka, menu tersebut akan memudar saat Kurangi Gerakan diaktifkan.
  • Transparansi Latar Belakang meningkatkan kontras dan keterbacaan dengan membuat latar belakang pada elemen UI menjadi lebih buram, seperti latar belakang jendela obrolan. Ini tidak akan membuat elemen UI lebih transparan dari biasanya.
  • Tombol Navigasi UI mengontrol apakah Anda dapat menggunakan tombol \ untuk masuk dan keluar dari mode navigasi UI atau tidak. Anda dapat menonaktifkan pengaturan ini jika Anda tidak sengaja sering memasuki mode navigasi UI, atau jika Anda tidak ingin menggunakan fungsi mode navigasi UI.

Bantuan

Menu ini berisi pintasan keyboard, instruksi mouse, dan gerakan perangkat seluler sehingga Anda dapat menavigasi pengalaman dan menggunakan semua fungsi yang tersedia.
