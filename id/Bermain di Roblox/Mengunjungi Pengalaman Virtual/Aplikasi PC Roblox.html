Aplikasi PC Roblox

Aplikasi Roblox PC mengubah alur cara pemain bergabung dalam pengalaman dan di mana mereka dapat membeli Premium dan Robux.

Bagaimana Cara Meng-Install

  • Anda akan melihat spanduk untuk aplikasi tersebut.
  • Klik tombol Buka di spanduk.
  • Sebuah jendela pop-up akan muncul yang mengatakan bahwa Roblox Player sedang menginstal
  • Seandainya, penginstalan otomatis tidak dimulai, c klik tombol Unduh dan Instal Roblox di jendela pop-up.
  • Jendela pop-up sekunder mungkin muncul meminta Anda untuk memilih program, Pilih Buka Roblox .

Perbedaan Aplikasi Roblox

Meskipun Anda mungkin akrab dengan sebagian besar antarmuka jika Anda memainkan Roblox secara normal, ada beberapa perbedaan utama yang perlu diperhatikan saat bermain di versi Aplikasi.


Menu utama

Anda akan melihat bahwa beberapa fitur langsung tersedia di menu utama Aplikasi Roblox, seperti Halaman Beranda, Halaman Pengalaman, Editor Avatar, dan Jendela Obrolan. Anda dapat membuka salah satu dari ini di panel kiri saat pertama kali membuka Aplikasi.

Tinggalkan Pengalaman (L)

Tombol ini membuat pemain keluar dari pengalaman, membuat pemain terputus dari server tempat mereka berada. Setelah mengklik tombol tersebut, ada pesan lain untuk mengonfirmasi jika mereka ingin pergi. Meninggalkan pengalaman dengan cara ini tidak akan menutup jendela pengalaman, dan sebaliknya akan mengembalikan Anda ke menu utama.

Untuk segera keluar sepenuhnya, Anda dapat mengeklik tombol tutup di bilah atas Jendela Aplikasi Roblox .
