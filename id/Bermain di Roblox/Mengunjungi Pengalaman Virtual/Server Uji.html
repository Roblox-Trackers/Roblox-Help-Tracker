Server Uji

Kadang-kadang para insinyur mungkin perlu menguji beberapa fitur baru atau peningkatan lain untuk Roblox sebelum meluncurkannya ke semua orang. Pada saat ini, kami dapat mengizinkan pengguna untuk masuk dan membantu! Namun, penting untuk dicatat bahwa server pengujian ini akan dibuka dan ditutup pada berbagai waktu tanpa peringatan.

Jika para insinyur ingin menguji sesuatu secara publik, biasanya akan ada spanduk pengumuman di situs utama kami yang meminta Robloxian yang membantu untuk menguji fitur dan perubahan baru bersama dengan tautan ke server tertentu tempat pengujian berlangsung.

Selain itu, harap diingat bahwa keanggotaan di situs pengujian hanya untuk tujuan pengujian dan bukan pembelian yang sebenarnya. Apa pun yang dibeli di situs uji akan tetap terhubung ke server itu saja dan tidak dapat ditransfer ke situs utama.

Harap perhatikan bahwa Dukungan Pelanggan tidak dapat membantu dengan akun di situs pengujian.
