Bagaimana cara Membeli dan Mengonfigurasi Server Pribadi/VIP?

Jika kreator pengalaman telah mengaktifkan server pribadi di pengalamannya, Anda dapat menggunakan Robux untuk membeli server pribadi yang dapat Anda nikmati bersama teman yang Anda undang.

Bagaimana cara membuat dan mengubah server saya?

Bagaimana cara membuatnya?

  • Klik tab Server di halaman detail pengalaman.
  • Jika fitur ini telah diaktifkan, Anda akan melihat bagian berjudul Server Pribadi . Ini akan menampilkan server apa pun yang telah Anda buat atau bagian darinya. Jika Anda melihatnya, Anda dapat langsung bermain!
  • Untuk membuat yang baru, klik tombol Buat Server Pribadi .
  • Beri nama server baru Anda. Baca perjanjian berlangganan.Â Setelah membaca semua informasi di kotak dan memilih nama yang tepat, Anda dapat menyelesaikan pembelian dengan mengeklik Beli Sekarang .
  • Setelah Anda membeli, Anda akan dapat mengonfigurasi server segera atau nanti.

Bagaimana cara mengkonfigurasi server saya

  • Dari jendela pemberitahuan pembelian atau dengan memilih Konfigurasi dari tombol Menu (terlihat seperti tiga titik) untuk server pada tab Server , Anda dapat masuk ke halaman konfigurasi.
  • Anda dapat mengganti nama server, menyesuaikan siapa yang dapat bermain di dalamnya termasuk sampai 50 pengguna individu atau memberikan akses kepada pengguna dalam daftar teman Anda, dan bahkan membuat tautan ke server Anda.
  • Anda juga dapat menghidupkan dan mematikan akses ke server serta membatalkan langganan Anda untuk server.

Bagaimana cara kerja Langganan Server dan berapa biayanya?

Bagaimana cara kerja langganan server pribadi?

  • Harga Server Pribadi ditentukan oleh developer pengalaman.
  • Anda akan ditagih saat melakukan pembelian dan setiap bulan agar tetap aktif. Biaya perpanjangan bulanan akan diambil dari saldo Robux Anda secara otomatis, jadi Anda tidak perlu khawatir lupa memperbarui setiap bulan.
  • Jika Anda tidak ingin server Anda diperpanjang, ini dapat dilakukan kapan saja sebelum tanggal perpanjangannya. Untuk membatalkan server VIP Anda, silakan baca Bagaimana cara membatalkan server VIP saya di bawah ini.

Bagaimana cara membatalkan server saya?

  • Temukanlah server yang Anda ingin batalkan untuk pengalaman itu. Klik tombol Menu (terlihat seperti tiga titik) di bagian ujung kanan.
  • Klik Konfigurasi .
  • Untuk membatalkan tagihan bulanan Anda, klik ikon hijau Aktif di samping Status Langganan .
  • Klik Batalkan Pembayaran pada jendela konfirmasi untuk membatalkan langganan.

Catatan: Anda masih dapat menggunakan server Anda untuk sisa bulan yang telah Anda bayar, tetapi akan kedaluwarsa pada akhir waktu ini.

Apa yang terjadi jika saya tidak memiliki Robux yang cukup pada tanggal perpanjangan?

Jika ini terjadi, maka server akan kadaluarsa dan tidak dapat digunakan lagi, tetapi Anda dapat membeli server baru kapan saja.

Bagaimana cara membuat server saya tidak aktif?

  • Temukanlah server pribadi yang Anda untuk pengalaman itu. Klik tombol Menu (terlihat seperti tiga titik) di bagian ujung kanan.
  • Klik Konfigurasi .
  • Untuk membuat server tidak aktif untuk sementara waktu, klik ikon hijau Aktif di samping Status Server VIP .
  • Klik Nonaktifkan pada jendela konfirmasi untuk menonaktifkan server.
  • Anda dapat mengkliknya lagi untuk mengaktifkannya kembali kapan saja. Namun, penting untuk diingat bahwa waktu perpanjangan Anda tidak akan diperpanjang untuk periode tidak aktif ini dan akan tetap diperpanjang atau berakhir pada tanggal yang dijadwalkan semula.
