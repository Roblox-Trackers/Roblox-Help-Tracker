Aplikasi PlayStation: Cara Bermain Pengalaman Virtual Roblox

Kamu memerlukan konsol PlayStation dan koneksi internet untuk bermain pengalaman virtual Roblox. Roblox dapat ditemukan di PlayStation Store. Bermain secara gratis!

Ada banyak pengalaman virtual berbeda yang dapat kamu mainkan di Roblox. Saat memulai pertama kalinya, pengalaman virtual akan ditemukan di layar Beranda dan Pengalaman Virtual. Setelah kamu bermain beberapa pengalaman virtual, pengalaman virtual yang baru-baru ini dimainkan akan ditampilkan di layar Beranda.

Setiap pengalaman virtual Roblox akan memiliki peraturan, kontrol, dan mekanisme yang sedikit berbeda. Kontrol umum akan tetap sama, seperti ditunjukkan di bawah ini.

Catatan: Peta kontroler ini dapat diakses kapan saja saat berada dalam pengalaman virtual dengan mengklik tombol "Options" (Opsi) di kontroler PlayStation untuk membuka menu Roblox. Tekan bumper kanan (R1) untuk melihat tab "Help" (Bantuan). Tekan bumper kiri (L1) untuk melihat tab lain.

Dengan peluncuran platform baru apa pun, seperti PlayStation, pengguna mungkin melihat tombol untuk konsol/platform lain di PlayStation saat developer Roblox meng-update pengalaman virtual mereka untuk menyertakan pemetaan tombol PlayStation, update antarmuka pengguna, atau lainnya .
