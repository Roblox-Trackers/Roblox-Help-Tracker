Roblox VR

Tingkatkan pengalaman Roblox Anda dengan realitas maya (VR)! Roblox mendukung Oculus Rift dan HTC Vive.

Bagaimana Mengaktifkan VR

Untuk mengaktifkan pengalaman VR Anda dengan Roblox, Anda harus memastikan terlebih dahulu jika Anda telah menginstal Roblox ( http://www.roblox.com/download ) kemudian silakan ikuti langkah berikut ini:

Harap pastikan bahwa perangkat Anda telah terhubung dengan benar ke PC/komputer Anda dan Mode VR dialihkan ke " Aktif " dalam pengaturan dalam pengalaman (Menu > Pengaturan > VR > Aktif). Untuk lebih lengkapnya, lihat Cara Beralih ke Mode VR dibawah ini.

HTC Vive:

    1. Dari komputer Anda, buka aplikasi SteamVR .
    2. Pastikan stasiun pangkalan Anda dinyalakan.
    3. Pastikan headset Anda terhubung ke komputer melalui kotak tautan.
    4. Nyalakan kontrolir :
      ( Untuk menyalakan kontrolir , tekan tombol System sampai Anda mendengar bunyi bip )
      (Untuk mematikan kontrolir , tekan dan tahan tombol System sampai Anda mendengar bunyi bip)
    5. Ikon status pada aplikasi SteamVR dan lampu status pada hardware semuanya harus berwarna hijau.

      Untuk memunculkan Sistem Dashboard , tekan tombol Sistem pada salah satu kontrolir .

      Catatan: Saat Anda keluar dari aplikasi SteamVR , kontrolir akan mati secara otomatis. kontrolir juga akan mati secara otomatis setelah berdiam selama jangka waktu tertentu.

Oculus Rift:

    1. Buka aplikasi Oculus di komputer Anda dan pilih ikon gear dibagian kanan atas
    2. Pilih Pengaturan Gear
    3. Pilih General di menu kiri
    4. Alihkan pengaturan Unknown Sources sehingga muncul sebagai tanda centang , bukan X
    5. Konfirmasi pilihan Anda untuk mengaktifkan pengaturan ini
    6. Pilih Play di pengalaman Roblox: http://www.roblox.com/discover
    7. Anda akan melihat layar persegi di monitor Anda yang menunjukkan mode VR (cth. Gambar di bawah)
    8. Pakai headset Anda dan Mainkan!

Cara Menggunakan Menu VR Radial

Semua pengaturan sistem yang penting dapat diakses dengan menggunakan Menu VR Radial .

    1. Pilih tombol Menu untuk menampilkan Menu VR Radial
    2. Pilih salah satu opsi berikut:
      • Menu Sistem - Opsi akses ke sistem
      • Daftar Pemain - Melihat daftar pemain di server pengalaman
      • Kamera Tengah - Memusatkan kembali kamera pada karakter Anda
      • Pemberitahuan - Lihat pemberitahuan Roblox
      • Keluar Pengalaman - Keluar dari pengalaman
      • Ransel - Lihat inventaris dalam pengalaman Anda
      • Pengalihan 2D UI - Aktifkan/nonaktifkan fitur UI 2D, sebaiknya tetap aktif untuk hasil yang optimal

Opsi Menu VR Radial yang dijelaskan di atas sesuai dengan tombol pada gambar berikut, dimulai dengan "Menu Sistem" di bagian atas dan berlawanan arah jarum jam.

Cara Mengalihkan Mode VR

Anda dapat mengaktifkan atau menonaktifkan Mode VR didalam pengalaman, mengalihkan tampilan antara headset VR dan monitor komputer Anda.

Saat Mode VR Aktif , Anda dapat mengalihkannya ke Nonaktif engan melakukan hal berikut:

    1. Pilih tombol Menu untuk menampilkan Menu VR Radial
    2. Pilih opsi System Menu
    3. Alihkan opsi VR ke Nonaktif

Saat Mode VR Nonaktif , Anda dapat mengalihkannya ke Aktif dengan melakukan hal berikut:

    1. Buka System Menu dengan menekan ESC pada keyboard Anda, atau dengan klik tombol menu hamburger dibagian sudut kiri atas layar
    2. Alihkan opsi VR ke Aktif
