Persyaratan Perangkat Keras Komputer & Sistem Operasi

Persyaratan sistem minimum untuk Roblox tercantum di bawah ini. Jika Anda mengalami masalah dalam memainkan pengalaman tersebut, Anda mungkin perlu memutakhirkan komputer Anda. Baca artikel bantuan ini untuk persyaratan sistem perangkat seluler.

Persyaratan sistem operasi

  • PC/Windows: Roblox akan diinstal pada versi 64-bit Windows 7, Windows 8.1, atau Windows 10. Untuk Windows 8.1/ Anda harus menjalankan Roblox dalam Mode Desktop, karena Mode Metro (layar mulai berubin) saat ini tidak didukung.
    Catatan: Situs web Roblox mendukung browser berikut untuk PC/Windows: Chrome , Firefox , dan Microsoft Edge< /a>.
  • Mac: Roblox Client akan diinstal pada 10.10 (Yosemite) dan lebih tinggi sedangkan Roblox Studio akan diinstal pada Mac OS 10.13 (High Sierra) dan lebih tinggi.
    Catatan: Situs web Roblox mendukung browser berikut untuk Mac: Chrome , Firefox , dan Safari .
  • Linux: Roblox tidak didukung di Linux.
  • Seluler: Klik di sini untuk persyaratan sistem untuk Roblox Mobile.
  • Chrome OS : Chrome OS versi 53 atau lebih tinggi diperlukan, dan Anda harus mengaktifkan Google Play Store di perangkat Anda.

Catatan: Fitur perekaman video Roblox hanya berfungsi di Windows 7, Windows 8/8.1, dan Windows 10. Sistem operasi yang baru dirilis atau versi pra-rilis dari sistem operasi mungkin tidak segera didukung.

Persyaratan perangkat keras sistem

  • Kartu Grafik: Di PC/Windows, aplikasi Roblox memerlukan dukungan tingkat fitur DirectX 10 atau lebih tinggi. Untuk performa terbaik, kami merekomendasikan komputer yang berusia kurang dari 5 tahun dengan kartu video khusus, atau laptop yang berusia kurang dari 3 tahun dengan kartu video terintegrasi.
  • Prosesor: Roblox menyarankan Anda memiliki prosesor terbaru (2005+) dengan kecepatan clock 1,6 Ghz atau lebih baik. Ada beberapa masalah dengan prosesor AMD yang lebih tua.
  • RAM atau Memori: Roblox menyarankan Anda memiliki setidaknya 1 GB memori di Windows 7, Windows 8.1, atau Windows 10
  • Ruang Penyimpanan: Roblox menyarankan Anda memiliki setidaknya 20 Mb ruang penyimpanan sistem untuk memasang Roblox.
  • Seluler: Klik di sini untuk persyaratan sistem untuk Roblox Mobile.

Persyaratan internet

Untuk rumah tangga sekali pakai, Roblox merekomendasikan koneksi internet minimal 4-8 Mb/s In larger households, greater bandwidth is advisable.
