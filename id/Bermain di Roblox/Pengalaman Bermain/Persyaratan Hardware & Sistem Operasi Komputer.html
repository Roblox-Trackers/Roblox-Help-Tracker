Persyaratan Hardware & Sistem Operasi Komputer

Di bawah ini adalah daftar persyaratan sistem minimum untuk Roblox. Jika kamu mengalami masalah dalam bermain pengalaman virtual, kamu mungkin perlu meng-upgrade komputer kamu. Baca artikel ini untuk persyaratan sistem perangkat seluler.

Persyaratan sistem operasi

  • PC/Windows: Roblox dapat diinstal di versi 64-bit Windows 7, Windows 8.1, atau Windows 10. Untuk Windows 8.1, kamu harus menjalankan Roblox dalam Mode Desktop karena Mode Metro (layar berubin) belum didukung saat ini.
    Catatan: Situs web Roblox mendukung browser berikut untuk PC/Windows: Chrome , Firefox , dan Microsoft Edge .
  • Mac: Roblox Client dapat diinstal di 10.10 (Yosemite) dan lebih tinggi, sedangkan Roblox Studio dapat diinstal di Mac OS 10.13 (High Sierra) dan lebih tinggi.
    Catatan: Situs web Roblox mendukung browser berikut untuk Mac: Chrome , Firefox , dan Safari .
  • Linux: Roblox tidak didukung di Linux.
  • Seluler: Klik di sini untuk persyaratan sistem Roblox Seluler.
  • Chrome : Chrome versi 53 atau lebih tinggi diperlukan dan kamu harus mengaktifkan Google Play Store terlebih dahulu di perangkatmu.

Catatan: Fitur rekaman video Roblox hanya berfungsi di Windows 7, Windows 8/8.1, dan Windows 10. Sistem operasi yang baru dirilis atau versi pra-rilis mungkin belum didukung.

Persyaratan sistem hardware

  • Kartu Grafik: Di PC/Windows, aplikasi Roblox memerlukan dukungan tingkat fitur DirectX 10 atau lebih tinggi. Untuk performa terbaik, kami merekomendasikan komputer yang berusia kurang dari 5 tahun dengan kartu video terpisah atau laptop yang berusia kurang dari 3 tahun dengan kartu video terintegrasi.
  • Prosesor: Roblox merekomendasikan prosesor terbaru (2005+) dengan clock speed 1,6 Ghz atau lebih tinggi. Prosesor AMD yang lebih lama memiliki beberapa masalah.
  • RAM atau Memori: Roblox merekomendasikan setidaknya 1 GB memori di Windows 7, Windows 8.1, atau Windows 10
  • Ruang Penyimpanan: Roblox merekomendasikan setidaknya 20 Mb ruang penyimpanan sistem untuk menginstal Roblox.
  • Seluler: Klik di sini untuk persyaratan sistem Roblox Seluler.

Persyaratan internet

Untuk penggunaan satu rumah tangga, Roblox merekomendasikan koneksi internet dengan kecepatan minimum 4-8 Mb/d. Untuk rumah tangga yang lebih besar, disarankan untuk memiliki bandwidth yang lebih besar.
