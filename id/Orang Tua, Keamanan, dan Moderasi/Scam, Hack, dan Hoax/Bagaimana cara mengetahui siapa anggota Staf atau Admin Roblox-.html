Bagaimana cara mengetahui siapa anggota Staf atau Admin Roblox?

Ada beberapa cara untuk mengetahui apakah orang lain adalah anggota Staf atau Admin Roblox.

Lencana

Salah satu cara untuk mengetahui apakah seseorang adalah staf atau bukan adalah dengan memeriksa lencana di profil mereka. Semua anggota staf Roblox akan memiliki lencana Admin Roblox yang terletak di daftar lencana mereka. Jika mereka tidak memiliki lencana ini dan mengaku sebagai staf, pastikan untuk memberi tahu kami dengan menggunakan tautan Laporkan Penyalahgunaan di profil mereka.Â Lencananya terlihat seperti ini:

Grup Admin

Untuk menemukan daftar Admin Roblox, Anda hanya perlu mengunjungi grup admin Roblox. Meskipun Anda dapat menemukan tautan ke ini di semua profil admin, Anda juga dapat membukanya secara langsung dengan mengeklik di sini .

Penting juga untuk diperhatikan bahwa administrator Roblox tidak akan pernah menanyakan password Anda. Jika ada yang menanyakan password Anda atau mengaku sebagai staf Roblox tetapi tidak memiliki lencana Admin, harap laporkan menggunakan tautan Laporkan Penyalahgunaan yang ada di sekitar situs.
