Saya memiliki masalah dengan mainan, kode, atau benda virtual

Saya memiliki masalah dengan kode permainan

Anda dapat menghubungi kami di sini untuk bantuan lebih lanjut. Pastikan untuk menyertakan informasi berikut:

  • Deskripsi masalah:
  • Nama Pengguna
  • Nomor kode
  • Gambar bagian depan dan belakang kode kartu item virtual (hanya format .jpg atau .png)
  • Tanggal pembelian
  • Toko dan kota pembelian
  • Gambar atau pindaian struk pembelian (hanya gunakan format .jpg atau .png saja)
  • Nama mainan (jika Anda tidak yakin dengan nama mainannya, berikan kami gambar dalam format .jpg atau .png dari Daftar Panduan Kolektor Jazwares )

Saya memiliki masalah dengan item virtual

Jika Anda tidak yakin apakah Anda telah menerima barang virtual untuk mainan Anda, pastikan untuk mengikuti langkah-langkah berikut untuk memeriksa inventaris Anda.

  • Pastikan Anda masuk ke akun Roblox tempat Anda menukarkan kode mainan.
  • Jika Anda tidak yakin item virtual mana yang disertakan dengan mainan Anda, Anda dapat mengunjungi situs web Jazwares di Panduan Kolektor dengan menemukan mainan Anda dan mengklik gambar benda virtual atau pada kata benda maya untuk mengakses halaman artikel di katalog Roblox.
  • Saat masuk ke akun tempat kode ditukarkan, halaman artikel katalog akan menunjukkan apakah akun tersebut adalah pemilik artikel dan menawarkan beberapa opsi. Lihat di bawah ini:
  • Anda juga dapat melihat inventaris akun dan meninjau kategori tempat item virtual berada. Jika item muncul di inventaris akun, Anda memilikinya.

Jika Anda tidak dapat menemukan item virtual di inventaris akun Anda atau yakin bahwa Anda telah menerima item yang salah, harap hubungi kami di sini untuk bantuan lebih lanjut. Pastikan untuk menyertakan informasi berikut:

  • Deskripsi masalah
  • Nama Pengguna
  • Nomor kode
