Bagaimana cara membuat Avatar saya menjadi perempuan atau laki-laki?

Memilih avatar menjadi laki-laki atau perempuan terjadi pertama kali saat kamu membuat akun, tetapi kamu selalu dapat meng-update-nya.

Mengubah jenis kelamin akun

Kamu dapat mengubah jenis kelamin avatar dengan mengeditnya di halaman pengaturan akun kamu. Ikuti langkah ini untuk mengubah jenis kelamin:

  • Pilih ikon roda gigi di sudut kanan atas layar lalu pilih Pengaturan
  • Di halaman Pengaturan, scroll ke bawah ke bagian Pribadi
  • Pilih ikon jenis kelamin yang sesuai

Catatan: Mengubah pengaturan ini tidak akan mengubah tampilan avatar kamu.

Mengubah avatar

Untuk mengubah penampilan kamu, kamu dapat menggunakan paket avatar gratis dari Marketplace:

  • Paket Laki-Laki
  • Paket Perempuan
  • Paket Anak Perempuan
  • Paket Anak Laki-Laki

Untuk mengubah paket tubuh:

  • Browser: Mengustomisasi Avatar
  • Seluler: Mengustomisasi Avatar
