Bagaimana Saya Membuat Sesuatu Domain Publik?

Item Kreator Marketplace ( Model , Stiker , Audio atau Plugin ) semuanya dapat dibuat untuk Domain Publik yang artinya penyalinan diperbolehkan. Namun, aksesori, pakaian, topi, dan perlengkapan tidak dapat dijadikan Domain Publik.

Menerbitkan aset ke Kreator Marketplace adalah satu-satunya cara agar aset Anda tersedia secara publik bagi semua pengguna untuk digunakan dalam pengalaman mereka sendiri di Studio. Lihat artikel Menerbitkan aset untuk informasi lebih lanjut.

Catatan: Untuk informasi selengkapnya tentang berbagi konten, silakan tinjau Persyaratan Penggunaan Roblox .
