File Audio

Platform Roblox menyediakan pencipta dan developer untuk mengunggah audio untuk membuat pengalaman mereka imersif dan dinamis. Menambahkan audio ke pengalaman Anda seperti musik latar, narasi, efek suara, dan banyak lagi!

Catatan: Semua audio yang diunggah ke Roblox harus memenuhi Standar Komunitas Roblox kami.

Cara Mengunggah Audio

Pengunggahan dapat dilakukan melalui Studio atau Buat di peramban web. Ikuti langkah berikut.

Studio

Ikuti langkah di bagian Mengimpor Audio Khusus .

Buat

  1. Masuk ke akun Roblox Anda.
  2. Klik Buat di bilah biru di bagian atas layar.
  3. Klik Audio .
  4. Klik Jelajahi dan pilih file audio.
  5. Setelah dipilih, klik pada tombol "Unggah" .

Setelah file audio diunggah, audio tersebut dapat disesuaikan lebih lanjut di halaman Konfigurasi. Baca artikel izin melihat untuk informasi tentang cara mengonfigurasi file audio Anda.
