Testing di Studio

Mode Testing adalah fitur di Roblox Studio yang memungkinkan kamu untuk menguji tempatmu setelah selesai dibuat atau melihat fungsi sesuatu sebelum dipublikasikan. Jika kamu ingin menguji tempatmu dengan memainkannya secara normal di situs Roblox, atur pengalaman virtualmu ke privat agar kamu dapat melakukannya tanpa pemain lain bergabung ke pengalaman virtualmu.

Baca artikel Dokumentasi Kreator tentang Mode Testing Studio untuk informasi selengkapnya.
