Tautan Media Sosial untuk Pengalaman

Para developer mungkin ingin menautkan profil media sosial mereka yang lain secara langsung ke pengalaman mereka. Roblox sekarang menawarkan dukungan resmi untuk menautkan media sosial melalui fitur Tautan Media Sosial! Para developer akan dapat menyertakan hingga tiga tautan media sosial dalam deskripsi pengalaman mereka, serupa dengan metode yang digunakan dalam profil pengguna. Setiap jenis media sosial hanya dapat ditambahkan satu kali.

Lihat artikel dokumentasi Kreator untuk mempelajari lebih lanjut tentang Tautan Media Sosial untuk Pengalaman .
