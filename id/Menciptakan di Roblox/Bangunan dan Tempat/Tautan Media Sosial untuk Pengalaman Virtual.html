Tautan Media Sosial untuk Pengalaman Virtual

Developer mungkin ingin menautkan profil media sosial mereka di pengalaman virtual mereka. Roblox sekarang menawarkan kemampuan ini melalui fitur Tautan Media Sosial. Developer akan dapat menyertakan hingga tiga tautan media sosial di deskripsi pengalaman virtual mereka, serupa dengan metode yang digunakan di profil pengguna. Setiap tipe media sosial hanya dapat disertakan satu kali.

Baca artikel dokumentasi Kreator untuk mempelajari lebih lanjut tentang Tautan Media Sosial untuk Pengalaman Virtual .
