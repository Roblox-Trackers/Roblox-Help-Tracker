Avatar Saya Ditampilkan dengan Salah atau sebagai X Abu-abu

Kadang-kadang Anda dapat mengenakan pakaian atau perlengkapan baru yang mewah hanya untuk menemukan bahwa avatar Anda belum berhasil diperbarui atau sekarang muncul sebagai X abu-abu. Jika demikian, ada beberapa hal yang dapat Anda coba.

Tarik Ulang Avatar Anda

Ada kemungkinan bahwa server mengalami sedikit gangguan saat membuat thumbnail untuk avatar Anda. Jika itu masalahnya, yang perlu Anda lakukan hanyalah memperbaruinya! Untuk melakukannya, buka bagian Avatar dan klik tautan "Klik di sini untuk menggambar ulang!" yang terletak di bawah gambar avatar Anda. Mungkin butuh beberapa menit untuk menyelesaikannya, namun melakukan hal ini bisanya sebagian besar akan memperbaiki masalah.

Bersihkan File Internet Sementara Peramban Anda

Jika menarik ulang tidak membantu, menghapus file sementara peramban Anda dapat membantu. Untuk bantuan dalam hal ini, silakan klik di sini dan ikuti langkah-langkah dalam solusi "Hapus Berkas dan Cookies Internet Sementara Anda".
