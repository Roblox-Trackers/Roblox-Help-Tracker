Mac: Penyelesaian Masalah

Hapus / Instal Ulang Aplikasi Roblox

Jika kamu mengalami masalah dan belum mencoba menginstal ulang baru-baru ini, coba uninstal program dan instal ulang dengan mengikuti langkah yang terdapat di dua artikel berikut:

  • Uninstal
  • Instal Ulang

Mulai Ulang Komputer

Jika kamu mengalami masalah saat menggunakan Aplikasi Roblox, coba mulai ulang komputer kamu untuk menyelesaikan masalah tersebut.

Reset Log

Jika memulai ulang komputer kamu tidak menyelesaikan masalah, ikuti langkah berikut ini:

  1. Tekan Command + Spasi
  2. Salin dan tempel: ~/Library/Logs/Roblox/
  3. Buka dan hapus semua yang ada dalam folder.
  4. Coba ulangi masalah tersebut

Langkah Lain yang Dapat Dicoba

Jika uninstall dan menginstal ulang tidak menyelesaikan masalah, coba periksa berikut ini:

  • Aplikasi Roblox dapat diinstal dengan OS 10.10 (Yosemite) dan yang lebih tinggi dan Roblox Studio dapat diinstal dengan OS 10.13 (High Sierra) dan yang lebih tinggi.
  • Saat ini, kami mendukung chip Apple Silicon / M1.
  • Jika saat ini kamu menggunakan browser Safari dan mengalami masalah, coba gunakan salah satu browser yang kami dukung seperti Firefox atau Google Chrome.
  • Periksa untuk memastikan bahwa ikon Roblox saat ini ada di Dock kamu dan Aplikasi Roblox ditampilkan di folder Aplikasi.
  • Coba luncurkan program dari Dock jika kamu mengalami masalah saat menjalankan pengalaman virtual dari situs web.
  • Coba reset sepenuhnya pengaturan Roblox kamu dengan menghapus file pengaturan. Untuk melakukannya, buka Perpustakaan-> Preferensi dan hapus semua file Roblox com. Saat meluncurkan Studio/Player kembali, semua file tersebut akan diinstal ulang.

Jika setelah mengikuti langkah di atas, kamu tetap mengalami masalah, hubungi Dukungan Pelanggan dan berikan informasi berikut:

  • Nama pengguna Roblox
  • Setiap pesan eror yang diterima
  • Tipe model Mac
  • Versi OS Mac
  • Langkah yang kamu lakukan hingga mengalami masalah

Sertakan juga detail tambahan apa pun yang dapat membantu kami lebih memahami masalah yang kamu alami dan setiap perubahan terbaru yang kamu lakukan pada Mac atau jaringan kamu.
