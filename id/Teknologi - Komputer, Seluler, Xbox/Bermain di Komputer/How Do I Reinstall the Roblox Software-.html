How Do I Reinstall the Roblox Software?

Beberapa masalah teknis dapat diselesaikan dengan menginstal ulang Roblox. Pastikan untuk mencoba memulai ulang sebelum menginstal ulang. Berikut cara menginstal ulang:

Pengguna Windows:

Menginstal ulang Roblox

Penginstalan ulang harus menimpa dan mengganti file yang terhapus atau rusak. Pastikan semua program lain ditutup saat menginstal ulang.

    • Instal ulang untuk Chrome atau Edge
    • Instalasi ulang untuk Browser Lain

Menghapus instalan Roblox

Sering kali, Anda tidak perlu mencopot pemasangan program sebelum memasangnya kembali. Jika penginstalan ulang tidak berhasil, Anda mungkin perlu mencopot pemasangan sebelum menginstal ulang. Ikuti petunjuk untuk mencopot pemasangan Roblox untuk Windows . Pastikan untuk me-restart komputer Anda setelah menghapus dan sebelum menginstal ulang.

Hapus file Internet sementara Anda

Langkah penting dalam proses pencopotan atau penginstalan ulang adalah menghapus file Internet sementara browser Anda. Ini akan membantu mengatasi masalah korupsi yang mungkin muncul. Ikuti petunjuk di Menghapus file internet sementara dan bagian cookie . Ulangi petunjuk penginstalan ulang di atas setelah selesai.

Ubah pengaturan program firewall/keamanan Anda

Banyak perangkat lunak antivirus atau keamanan Internet mencoba mengontrol program mana yang dapat mengakses Internet. Program-program ini tidak bekerja dengan hati-hati dan secara default tidak ada program yang diizinkan untuk diakses. Lihat informasi di artikel bantuan Masalah Firewall dan Router untuk mendapatkan bantuan dalam mengizinkan akses ke Roblox. Ulangi langkah penginstalan ulang setelah mengkonfigurasi ulang firewall.

Pengguna Mac:

Menghapus instalan

Pastikan Anda benar-benar keluar dari program Roblox. Ikuti petunjuk untuk mencopot pemasangan Roblox untuk Mac .

Menginstal ulang

Buka Safari, masuk ke Roblox dan mainkan. Anda akan diminta untuk menginstal aplikasi. Silakan lakukan ini dengan mengikuti semua petunjuk di layar. Jika penginstalan tidak terjadi secara otomatis, lakukan hal berikut:

    1. Buka Pencari > Hari ini dan klik Roblox.dmg - ini akan menjalankan penginstal untuk Anda dan menempatkan ikon Roblox di aplikasi Anda.
    2. Pilih dari aplikasi Anda dan itu harus masuk ke baki Anda
