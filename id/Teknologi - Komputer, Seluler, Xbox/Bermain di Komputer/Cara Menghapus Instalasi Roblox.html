Cara Menghapus Instalasi Roblox

Jika Anda mengalami masalah teknis pada Aplikasi atau Studio Roblox, terkadang menghapus instalan sebelum menginstal ulang aplikasi mungkin berguna. Artikel ini akan memandu Anda melalui proses mencopot pemasangan.

Instruksi Jendela

Uninstal Program

Windows 7

  • Buka menu Mulai > Panel Kontrol > Program dan Fitur
  • Gulir ke bawah hingga Anda melihat Roblox
  • Hapus instalan program di sana

Windows 8 atau 8.1

  • Tarik layar awal ubin dengan mengklik di sudut kiri bawah layar Anda
  • Ketik "Program dan fitur" dan klik ikon Program dan Fitur yang muncul di sebelah kanan hasil penelusuran Anda
  • Di jendela yang muncul, gulir ke bawah hingga Anda melihat Roblox
  • Hapus instalan program di sana

Windows 10

  • Klik tombol mulai atau kotak pencarian Cortana di kiri bawah layar
    • Atau, Anda dapat menahan tombol Windows pada keyboard Anda diikuti dengan menekan tombol S
  • Di kotak pencarian, ketik "Program dan fitur" dan klik ikon Program dan Fitur yang muncul di bagian atas daftar pencarian Anda
  • Di jendela yang muncul, gulir ke bawah hingga Anda melihat Roblox
  • Hapus instalan program di sana

Hapus folder Roblox

Jika Anda mengalami masalah yang terus-menerus, menghapus folder Roblox sepenuhnya setelah mencopot pemasangan mungkin berguna untuk memastikan tidak ada file yang rusak atau usang yang tersisa. Anda dapat melakukannya dengan mengikuti langkah-langkah di bawah ini:

  1. Klik kanan pada tombol mulai dan klik Jelajahi atau File Explorer
  2. Arahkan ke folder ini : C:\Users\(Nama Pengguna Windows Anda)\AppData\Local
  3. Hapus folder Roblox

Catatan: Ini seharusnya tidak menghapus tempat apa pun yang mungkin telah Anda simpan di Studio kecuali Anda telah memilih secara manual untuk menyimpannya di folder ini.

Kesulitan Menghapus Instalasi

Jika Anda mengalami kesulitan menghapus instalan program, coba yang berikut ini:

Bersihkan File Internet Sementara Anda

Langkah penting dalam proses pencopotan atau penginstalan ulang adalah menghapus file internet sementara browser Anda. Ini akan membantu dengan masalah korupsi yang mungkin muncul. Untuk bantuan dalam proses ini, harap klik di sini dan lihat bagian Hapus File dan Cookie Internet Sementara Anda .

Periksa Firewall Anda

Banyak perangkat lunak antivirus atau keamanan internet mencoba mengontrol program mana yang dapat mengakses internet. Seringkali, program ini melakukan kesalahan karena berhati-hati dan secara default tidak ada program yang diizinkan mengaksesnya. Untuk mendapatkan bantuan dalam menyelidiki hal ini dan mengizinkan Roblox melewati segala rintangan, klik di sini .

Instal ulang Roblox

Kemungkinan selanjutnya adalah Instal ulang. Ini mungkin tampak kontra-intuitif tetapi jika masalahnya adalah file yang rusak dari upaya pencopotan yang gagal sebelumnya, ini akan menimpa salinan program di komputer Anda, dan mengganti file yang terhapus atau rusak. Anda kemudian dapat masuk ke menu Mulai dan menghapus instalan program. Untuk bantuan dalam menginstal ulang, harap klik di sini .

Instruksi Mac

Tutup Sepenuhnya dari Program Roblox

  • Tekan tombol Command + Option + Esc untuk membuka prompt manajemen tugas
  • Pastikan Anda tidak melihat Roblox atau Roblox Studio di dalamnya
  • Jika salah satu program terdaftar, pilih Roblox atau Roblox Studio dan klik Force Quit

Buka Penemu > Aplikasi

  • Seret ikon Roblox ke tempat sampah
    • Ikon di finder berisi semua file Roblox yang telah diinstal

Cara menghapus file dari Sampah

  • Klik ikon Sampah di Dock dan pilih Kosongkan Sampah.
  • Atau, kosongkan tempat sampah dengan menekan tiga tombol secara berurutan: Command + Shift + Delete.
  • Anda akan melihat peringatan: "Apakah Anda yakin ingin menghapus item di Sampah" karena Anda tidak dapat membatalkan tindakan ini , tetapi dengan asumsi Anda memang ingin menghapus, setelah mengeklik Kosongkan Sampah, Anda harus bebas dari mereka.


Untuk mengabaikan pertanyaan "Apakah Anda yakin ingin menghapus item di Sampah secara permanen?" pesan, tekan:

  • Perintah + Opsi/Alt + Shift + Hapus
