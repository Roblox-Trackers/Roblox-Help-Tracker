Masalah Obrolan/Keyboard Umum di Komputer

Keyboard internasional diketahui terkadang menyebabkan masalah saat mengakses fitur obrolan dalam pengalaman. Jika Anda mengalami masalah ini, coba gunakan tombol garis miring terbalik ("\") pada keyboard untuk menggunakan fitur obrolan.

Jika ini tidak berhasil, Anda juga dapat mengikuti langkah berikut untuk mengubah pengaturan wilayah keyboard Anda, yang mungkin dapat menyelesaikan masalah Anda:

  1. Buka menu Mulai
  2. Buka Panel Kontrol
  3. Buka tab Jam, Bahasa, dan Wilayah
  4. Buka Wilayah dan Bahasa
  5. Sesuaikan Format dan Lokasi ke bahasa Inggris dan Amerika Serikat, lalu terapkan atau tekan OK

Harap diketahui bahwa meskipun langkah ini mungkin dapat menyelesaikan masalah keyboard Anda, melakukan ini dapat menyebabkan program normal lainnya pada mesin Anda untuk bertindak berbeda, atau memiliki kontrol keyboard yang canggung.
