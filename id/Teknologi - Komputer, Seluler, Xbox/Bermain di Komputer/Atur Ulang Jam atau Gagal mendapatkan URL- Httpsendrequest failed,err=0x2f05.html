Atur Ulang Jam atau Gagal mendapatkan URL: Httpsendrequest failed,err=0x2f05

Berbagai masalah di beberapa program atau website sebenarnya bisa disebabkan jam sistem Windows Anda disetel ke waktu, tanggal, atau zona waktu yang salah.

Anda mungkin telah menerima salah satu pesan Roblox berikut:

  • Waktu sistem Anda salah
  • Gagal mendapatkan URL...Http send request failed,err=0x2f05

Cukup mengoreksi semuanya untuk memperbaiki situasi. Anda dapat mengatur aspek-aspek ini secara manual atau mengaturnya agar diperbarui secara otomatis dengan server. Setel jam secara otomatis untuk memastikannya jam tidak tertinggal dan mulai menimbulkan masalah lagi. Berikut cara mengaturnya secara otomatis:

Windows 8.1 dan Sebelumnya

  1. Klik kanan pada jam windows di pojok kanan bawah
  2. Klik Sesuaikan Tanggal/Waktu
  3. Pilih label Waktu Internet
  4. Di sini, Anda harus dapat menyinkronkan waktu dan tanggal Anda dengan server internet

Windows 10

  1. Klik kanan pada jam windows di pojok kanan bawah
  2. Pastikan "Atur Waktu Secara Otomatis" diaktifkan dan Anda telah memilih zona waktu yang benar
