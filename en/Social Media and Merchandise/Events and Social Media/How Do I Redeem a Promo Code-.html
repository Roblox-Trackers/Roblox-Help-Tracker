How Do I Redeem a Promo Code?

How do I get a Roblox promo code?

You may get a Roblox promo code from our events or giveaways. See How Do I Get Promotional or Contest Items? for more information on getting promotional items.

Note: Remember that promo codes may expire or only be active for a short period of time, so make sure to use your code right away.

How do I redeem my promo code for the virtual item?

  1. Make sure that you're logged into your Roblox account on which you want to redeem the code.
  2. Go to Redeem Roblox Codes .
  3. Enter your code in the box.
  4. Click Redeem .
  5. A success message will appear once you've successfully redeemed the Code.
  6. You will get Credit, Robux, or a Virtual item. It depends upon the type of Code you have redeemed.

Issues with Redeeming Promo Codes

If you're having an issue redeeming a promo code, please try the following:

  1. Check to make sure you're logged into your Roblox account before redeeming the code
  2. Navigate to the promo code redemption page
  3. Make sure you've put in the code accurately
  4. Check your inventory to see if you already have the exclusive item

Note: Promo codes are often available for a limited time only. If the time frame for a particular code has passed, you will not be able to redeem the code.
