How to Redeem a Toy Code

Where do I find my code for the virtual item?

The code is found on the front of the package or on the code card inside. Check out the Jazwares video on the code cards.

How do I redeem my code for the virtual item?

  1. If your code is covered, gently scratch off the covering.
  2. Make sure that you're logged into your Roblox account on which you want to redeem the code.
  3. Go to the Toy Code Redemption Page .
  4. Enter your code in the Enter Code box.
  5. Click the Redeem button.
  6. Check out your awesome new item in your account's inventory. Make sure you check the correct inventory category.

Note: An account can only receive each toy's virtual item once . If you have a code for a virtual item that your account already owns, you can share the code with another user.
