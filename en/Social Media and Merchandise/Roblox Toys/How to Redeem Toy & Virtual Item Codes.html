How to Redeem Toy & Virtual Item Codes

Where do I find my code for the virtual item?

  • For Jazwares action figures: The code is found on the front of the package on a token or on a code card stuck to the inside of the plastic packaging. Check out the Jazwares video on where to find the virtual item code.
  • For Nerf blasters and Monopoly: The code is found in the package on the code card insert.
    • Examples of what the code card inserts look like:
  • For all other virtual item codes (e.g. Prime Gaming, Verizon, etc.), go to the redemption instructions below.

How do I redeem my code for the virtual item?

Note: If you have a digital code, skip to step 2.

  1. If your code is covered, gently scratch off the covering.
  2. Make sure that you're logged into your Roblox account on which you want to redeem the code.
  3. Go to the Toy Code Redemption Page .
  4. Enter your code in the Enter Code box.
  5. Click the Redeem button.
  6. Check out your awesome new item in your account's Inventory. Make sure you check the correct Inventory category (e.g. hat, backpack, head, accessories, etc.).
