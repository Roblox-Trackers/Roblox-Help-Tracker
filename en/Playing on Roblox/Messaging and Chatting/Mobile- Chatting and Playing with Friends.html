Mobile: Chatting and Playing with Friends

You can chat with friends on Mobile devices and easily join experiences with friends directly from the chat.

How do I chat with my friends on mobile?

Chatting with your friends on mobile is similar on an Android or an iOS device, but there may be some visual differences.

To chat with friends on Mobile, follow these instructions:

  1. Open the Roblox App
  2. In the navigation bar at the bottom of the screen, select the chat icon:
  3. The screen will show a history of your recent chats and friends with whom you can chat. If any of your friends are playing experiences and their privacy settings allow it, you can see the experiences.
  4. Select the friend or group with whom you would like to chat and the chat window will open

How do I start a new chat group on mobile?

  1. In the navigation bar at the bottom of the screen, select the chat icon
  2. Select the icon that looks like a pencil and paper
  3. Select up to 5 friends by scrolling through your friends list or searching their username
  4. Click on Chat in the upper right to start the chat
  5. The new chat group page will load

Note: If your friend adds you to a chat group, you will be added to it automatically. When you are added to a chat group, you may see an unread message notification indicating there are unread messages from the new chat. If you don't want to be in the chat, you can leave the chat at any time. If someone continues to add you to chat groups you don't want to be in, we encourage you to block them. For further information, please see our help articles about How to Report Rule Violations and How to Block Another User .

How do I change the name of a chat group?

To change the name of a chat group, please do the following:

  1. Navigate to the chat group you wish to edit
  2. Select the info icon in the top right of the screen
  3. Click on Not Set
  4. Type in a new name for the chat group
  5. When done, select the Save option

How do I report or remove someone from a chat group I created?

  1. Navigate to the chat group you wish to edit
  2. Select the info icon in the top right of the screen
  3. Select the button that looks like three dots next to the user you wish to report and/or remove from the chat group
  4. Select either Report User or Remove from Group .

Note: If the user said anything rude or inappropriate, or in any way violated Roblox's Terms of Use or Rules of Conduct, please make sure to select Report User and then select Remove from Group afterward.

How do I add a friend to a chat group I previously created?

  1. Navigate to the chat group you wish to edit
  2. Select the info icon in the top right of the screen
  3. Select Add Friends
  4. Select a user to join your group by scrolling through your friend's list or searching their username
  5. Click Add in the upper right when you've selected the users you want to add

How do I leave a chat group on mobile?

  1. Navigate to the chat group you wish to leave
  2. Select the info icon in the top right of the screen
  3. Select the Leave Group option and confirm your choice
