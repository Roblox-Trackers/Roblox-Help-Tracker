Animation System (Emotes)

The Roblox Animation System allows a measured amount of fluidity and realism for movements. Part of that system are Emotes, which are specific actions your Roblox avatar can perform. To see the default emotes, hit the “/” key to chat and try typing in the following commands:

  • /e wave
  • /e point
  • /e cheer
  • /e laugh
  • /e dance
  • /e dance2
  • /e dance3

Note: This feature is not available if in-experience chat is disabled. For more information, click here .
