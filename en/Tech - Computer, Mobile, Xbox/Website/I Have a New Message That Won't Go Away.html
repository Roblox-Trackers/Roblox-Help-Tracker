I Have a New Message That Won't Go Away

If your inbox shows that you have new messages, but all of your messages have been read, please try the following:

  • Try refreshing the page, which will fix the problem if you have (or have had) more than one browser window open.
  • Archive all of your messages from Roblox (system messages).
