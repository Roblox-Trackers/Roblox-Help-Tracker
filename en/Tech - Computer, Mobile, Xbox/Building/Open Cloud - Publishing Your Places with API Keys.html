Open Cloud - Publishing Your Places with API Keys

Open Cloud

Open Cloud aims to provide a secure and flexible platform that allows you to access your Roblox resources e.g. your assets, places, player data, etc. through standardized web APIs. Open Cloud enables you to build applications and tools to access your Roblox resources through web APIs. To learn more on how to create an API Key, please visit our Tutorial article.

The API key allows your script or application to authenticate into Roblox Cloud and do things on your behalf. Security is on top of our mind; when creating an API key, you can set granular permissions to the API keys by selecting experiences > resources > specific permissions like read or write.

Please note that API keys are highly sensitive information and you should try your best to keep them in secure places.

The Place Publishing API allows you to automate the release workflow. For example, it can be hooked into a GitHub action 68 and automatically pushes an rbxl file to Roblox Cloud after successful integration testing. For API reference, please view our article here .
