Add 2-Step Verification to Your Account

What is 2-Step Verification?

This feature makes sure no one else can login to your account, even if they know the password. When you log in from a new device you'll enter a unique security code from an authenticator app or Roblox sends you via email. Since only you have access to these sources, only you will be able to get the security code.

Turning On 2-Step Verification

  1. Log into your account
    (If you can't log in, try these steps for recovering your password .)
  2. You must have a verified email to enable this feature
    (Here are steps for verifying your email address .)
  3. Go to account Settings
    • Browser - find the gear-icon located at the upper-right corner of the site
    • Mobile Apps - find the three dots icon for More
  4. Select the Security tab
  5. Turn 2 Step Verification on for the type you would like.
    • If you choose an Authenticator app, make sure you have it on your device. Some Authenticator apps include Google Authenticator, Microsoft Authenticator, and Twilio Authy.
  6. Follow the steps on the screen to complete set-up.
    • If the account has an account PIN, you will be asked to enter this.
    • To set up an Authenticator app, you will need to scan the QR code or manually enter the set-up code into it.

Turning Off 2-Step Verification

  1. Log into your account
    (If you can't log in, try these steps for recovering your password .)
  2. Go to account Settings
    • Browser - find t he gear-icon located at the upper-right corner of the site
    • Mobile Apps - find the three dots icon for More
  3. Select the Security tab
  4. Select turn 2 Step Verification off
  5. Enter your account password to finish turning the feature off

Any previously created recovery codes will also be deactivated if you turn off 2 step verification. For help on resetting the account password, check out the Forgot My Password article.

Getting Recovery Codes

  1. Log into your account
  2. Go to account Settings
    • Browser - find t he gear-icon located at the upper-right corner of the site
    • Mobile Apps - find the three dots icon for More
  3. Select the Security tab
  4. Click Generate under the recovery codes section.
  5. Enter your account password as part of the set-up process.
  6. Make sure to save the codes in a safe place because you can only see them this one time.

Each code can only be used once. If you lose your recovery codes, you can repeat this process to create new codes. Any previously created recovery codes will be deactivated when you create new codes.

Account Restores Eligibility
