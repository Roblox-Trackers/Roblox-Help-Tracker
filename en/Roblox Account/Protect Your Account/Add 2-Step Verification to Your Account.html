Add 2-Step Verification to Your Account

We strongly recommend enabling the 2-Steps Verification feature to improve your account security. When you log in from a new device, you'll enter a unique security code from an authenticator app or the email Roblox sends you.

Why is it Important?

This feature has several benefits to protect your account.

  • This feature makes sure no one else can login to your account, even if they know the password.
  • Since only you have access to the authenticator app or your email account, only you will be able to get the security code.
  • Extra security so that you don't lose access to your experiences, Robux limiteds and more.

Turning On 2-Step Verification

  • Log into your account.
    • If you can't log in, try these steps for recovering your password .
  • You must have a verified email to enable this feature.
    • Here are steps for verifying your email address .
  • Go to account Settings .
    • Web Browser: Find the gear icon located at the upper right corner of the site.
    • Mobile Apps: Find the three dots icon for More .
  • Select the Security tab.
  • Follow the steps below to turn on 2-Step Verification for the type you like.

Note : You can only enable either an authenticator app or use an email address for the verification codes.

Set up 2-Step Verification by Authenticator App

  1. Toggle Authenticator App (Very Secure) .
    • If the account has an account PIN, you will be asked to enter it.
  2. Enter your account password to confirm your request.
  3. Scan the QR code or manually enter the setup code into it on the Authenticator app.
    • Make sure you have an Authenticator app available on your mobile device.
      • Examples: Google Authenticator, Microsoft Authenticator or Twilio Authy
  4. Recovery codes will be automatically generated for you. Make sure you save these in a secure place and do not share them with anyone. If you lose your Authenticator app and your recovery codes, you will not be able to access your account.

Set up 2-Step Verification by Security Keys

Note that Security Keys are currently only supported on web browsers. You will only be able to use your security key when you have the Authenticator app enabled, and when logging in on a web browser and not on the Roblox app.

Follow the instructions under “Set up 2-Step Verification by Authenticator App”. Authenticator 2FA must be enabled to set up security keys.

  1. Toggle “Security Keys on Web Only (Very Secure)”
  2. Follow the instructions prompts from your browser to verify the security key of your choice
    • You can use a physical key or a method managed by your device hardware like facial recognition or a fingerprint scanner
    • Examples of physical security keys: YubiKey 5, YubiKey 5C, Google Titan Security Key
    • The options for security keys will vary depending on what browser (Safari, Chrome, Firefox) and what operating system (Windows or MacOS) you use
  3. Enter a name for your key so that you can remember which key you used
    • Example key name: “Blue YubiKey 5”
  4. You may choose to repeat these steps to add more keys or close the window if you are done.

Set up 2-Step Verification by Email Address

  1. Toggle Email (Secure) .
    • If the account has an account PIN, you will be asked to enter it.
  2. Enter your account password to confirm your request.

Turning Off 2-Step Verification

  1. Log into your account.
    • If you can't log in, try these steps for recovering your password .
  2. Go to account Settings .
    • Web Browser: Find t he gear icon located at the upper right corner of the site.
    • Mobile Apps: Find the three dots icon for More .
  3. Select the Security tab.
  4. Select turn 2 Step Verification off.
  5. Enter your account password to finish turning the feature off.

Any previously created recovery codes will also be deactivated if you turn off 2 step verification. For help on resetting the account password, check out the Forgot My Password article.

Getting Recovery Codes

  1. Log into your account.
  2. Go to account Settings .
    • Web Browser: Find t he gear icon located at the upper right corner of the site.
    • Mobile Apps: Find the three dots icon for More .
  3. Select the Security tab.
  4. Click Generate under the recovery codes section.
  5. Enter your account password as part of the setup process.
  6. Make sure to save the codes in a safe place because you can only see them this one time.

Each code can only be used once. If you lose your recovery codes, you can repeat this process to create new codes. Any previously created recovery codes will be deactivated when you create new codes.
