계정에 2단계 인증 추가하기

2단계 인증이 뭔가요?

이 기능은 다른 사람이 비밀번호를 알고 있더라도 사용자의 계정에 로그인할 수 없도록 합니다. 사용자는 새로운 기기를 통해 로그인할 때 Roblox 메일로 받는 특별한 보안 코드를 입력해야 합니다. 사용자만 이메일 계정에 접속할 수 있으므로 사용자 본인만 보안 코드를 얻을 수 있습니다.

2단계 인증 켜기

  1. 계정에 로그인합니다.
    ( 로그인 할 수 없으면, ' 비밀번호를 복구하기 '를 따르세요.)
  2. 이 기능을 사용하려면 인증된 이메일이 있어야합니다.
    ( 이메일 주소 인증하기 를 확인하세요.)
  3. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  4. 보안 탭을 선택합니다.
  5. 2단계 인증 을 켭니다.

2단계 인증 끄기

  1. 계정에 로그인합니다.
    ( 로그인 할 수 없으면, ' 비밀번호를 복구하기 '를 따르세요.)
  2. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 탭을 선택합니다.
  4. 2단계 인증 을 선택하고 끕니다.
  5. 이 기능을 끄기 위해 계정 비밀번호를 입력합니다.

2단계 인증을 잠그려면 계정 비밀번호를 반드시 알고 있어야 합니다. 계정 비밀번호 재설정 도움이 필요하면, ' 비밀번호를 잊어버렸어요 ' 도움말을 확인하세요.

문제 해결

자세한 내용은 2단계 인증 문제 해결 도움말을 확인하세요.
