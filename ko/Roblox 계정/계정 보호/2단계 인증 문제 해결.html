2단계 인증 문제 해결

2단계 인증 이메일을 받지 못했어요

  1. 여러 개의 이메일 주소를 가지고 있는 경우, Roblox에 등록한 이메일과 동일한지 확인합니다.
  2. 이메일의 스팸, 정크, 프로모션, 소셜 및 기타 폴더를 확인해 필터링 여부를 확인합니다.
    • 스팸 폴더에서 이메일을 찾았을 경우, 이후에는 필터링되지 않도록 '스팸 아님' 으로 표시합니다.
    • 이메일 주소 no-reply@roblox.com 을 주소록에 추가합니다.
  3. 2단계 인증 로그인 화면에서 코드 재전송 또는 다시 시작 기능을 사용합니다.

여전히 문제가 계속될 경우, 설정해 둔 다른 2단계 인증 수단을 사용해 보세요.

위 단계를 수행한 후에도 여전히 2단계 코드를 받을 수 없다면 지원 센터 에 문의를 제출하세요.

이메일로 받은 2단계 인증 코드가 작동하지 않아요

이메일로 전송된 2단계 인증 코드는 15분 후에 만료되며 코드가 이미 해당 계정에 잘못 입력된 적이 있을 경우 더 이상 작동하지 않습니다.

이 경우 2단계 인증 로그인 화면에서 코드 재전송 또는 다시 시작 기능을 사용해 새 코드를 요청해야 합니다.

새 코드를 사용해도 여전히 로그인이 불가능할 경우 지원 센터 에 문의를 제출하세요.

인증 앱의 코드가 작동하지 않아요

2단계 인증 앱의 코드는 앱 화면에 표시 중일 동안에만 작동합니다. 코드 입력 시 해당 코드가 현재 앱에 표시되고 있는지를 확인하세요. 이전 코드는 입력해도 작동하지 않으므로, 반드시 현재 앱에 나타나 있는 새 코드를 입력해야 합니다.

여전히 문제가 계속될 경우, 설정해 둔 다른 2단계 인증 수단을 사용해 보세요.

새 코드로도 여전히 로그인이 불가능할 경우 지원 센터 에 문의를 제출하세요.

복구 코드를 분실했어요

'계정에 2단계 인증 추가하기' 도움말을 참고해 새로운 복구 코드를 생성하세요.
