2단계 인증 문제 해결

2단계 인증 이메일을 받지 못했습니다

  1. 여러 개의 이메일 주소를 가지고 있는 경우, Roblox에 등록한 이메일과 동일한지 확인하세요.
  2. 스팸, 정크, 프로모션, 소셜 및 기타 폴더를 확인하고, 이메일 제공 업체에서 필터링하였는지 확인하세요.
    • 스팸 폴더에서 이메일을 발견하면, 나중에 필터링하지 못하도록 스팸이 아님 으로 표시하세요.
    • 이메일 주소 no-reply@roblox.com를 이메일 주소록에 추가하세요.
  3. 2단계 인증 로그인 화면에서 코드 재전송 또는 다시 시작 기능을 사용하세요.

위 단계를 수행 한 후에도 여전히 2단계 코드가 수신되지 않으면 고객 지원 서비스 에 문의하세요.

2단계 인증 코드가 작동하지 않습니다

2단계 인증 코드는 15분 후에 만료되며 코드가 이미 해당 계정에 잘못 입력된 경우 더 이상 작동하지 않습니다.

2단계 인증 로그인 화면에서 코드 재전송 또는 다시 시작 기능을 사용하여 새 코드를 요청해야 합니다. 만약, 새 코드로 로그인할 수 없으면 고객 지원 서비스 에 문의하세요.
