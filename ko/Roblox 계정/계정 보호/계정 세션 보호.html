계정 세션 보호

계정 세션 보호 기능은 기기별 로그인 세션 확인을 통해 계정 보안을 지원합니다. 본 기능은 유해 행위자가 세션 하이재킹을 통해 사용자의 계정을 탈취하는 행위를 방지할 수 있게 해 줍니다.

참고 : 이 기능은 2단계 인증 을 대체하는 기능이 아닙니다 .

계정 세션 보호 비활성화

계정 세션 보호 기능은 계정을 안전하게 유지하기 위해 설계되었지만, 일부 크리에이터는 자신의 스크립트나 브라우저 확장 프로그램에서 Roblox API를 호출하기를 원할 수도 있습니다. 계정 세션 보호 기능은 이러한 도구를 손상시킬 수 있으므로, 크리에이터는 이 기능을 비활성화할 수 있습니다. 계정 세션 보호 기능의 비활성화는 Robux나 기타 Robux와 관련된 가치가 없는 테스트 계정에서만 하는 것이 좋습니다.

경고 : 계정 세션 보호 기능을 비활성화하면 계정은 보안 위협에 취약한 상태가 됩니다. 계정 세션 보호 기능을 끈 상태에서 계정이 손상된 경우 Roblox의 지원을 받을 수 없습니다. 이러한 변경 사항은 되돌릴 수 없습니다 .

계정 세션 보호를 끄려면 다음을 수행하세요.

    1. create.roblox.com으로 이동합니다.
    2. 설정 으로 이동합니다.
    3. 고급 탭을 클릭합니다.

계정 세션 보호를 끄고 나면 다시 켤 수 없습니다 . 만일 누군가 여러분이 알지 못하는 사이에 계정 세션 보호 기능을 끈 경우, Roblox 지원 센터로 문의 하세요.
