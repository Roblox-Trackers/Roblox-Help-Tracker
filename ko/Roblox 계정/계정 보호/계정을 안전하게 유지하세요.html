계정을 안전하게 유지하세요

Roblox는 모든 이들에게 안전하고 즐거운 환경을 제공하기 위해 최선을 다하고 있습니다. 하지만 무엇보다도 사기 및 도용으로부터 계정을 최고로 잘 지킬 수 있는 사람은 바로 나 자신입니다. 항상 조심하고 안전을 유지하며, 아래 유용한 정보를 통해 잠재적인 위험으로부터 계정을 보호할 수 있는 방법을 알아보세요. 인포그래픽 차트 를 눈여겨보는 것도 좋은 방법입니다.

참고: 계정이 이미 도용 된 경우, 계정 복구 안내 를 따르세요.

계정을 안전하게 유지하는 방법

  • 계정에 이메일 주소를 추가하세요.

    본인의 계정에 이메일 주소를 추가해 두세요. 이메일 주소를 추가해 두면 비밀번호 분실 시 비밀번호를 재 설정 할 수 있고, 2단계 확인(2SV) 계정을 보호할 수 있으며, 지원 센터 문의 시 보다 원활하게 도움을 받을 수 있습니다.

  • 비밀번호, 쿠키, 2단계 인증 코드, 백업 코드 등을 절대 공유하지 마세요.

    계정 해킹은 로그인 정보를 직접적으로 다른 사람과 공유하거나 온라인 설문조사를 통해 입력하는 경우 가장 흔히 발생합니다. 누군가 가 본인의 로그인 정보를 요구하는 경우 절대 제공하지 말고 신고 기능을 사용해 검열 팀으로 신고 하세요.

    비밀번호, 브라우저 쿠키, 2단계 인증 보안 코드, 백업 코드, 개인정보, 혹은 컴퓨터의 민감한 세부 정보 등을 타인에게 제공하지 마세요. 또한 Roblox 직원은 절대 사용자의 비밀번호를 요구하지 않는다는 것 을 알아 두세요.

    계정 도용을 목적으로, 계정의 특정 정보를 제공할 경우 플레이에 적용할 수 있는 굉장한 트릭 또는 핵이 있다며 접근하는 사람들이 있습니다. 대표적인 속임수로 이들은 선물이나 무료 Robux를 미끼로 계정의 세부 정보를 요청합니다. Roblox의 공식 이벤트는 블로그, 메시지 섹션의 알림 탭 또는 사이트 상단의 배너를 통해서 공지됩니다.

  • 2단계 인증을 활성화하세요.

    2단계 인증을 활성화해 두면 다른 사람이 내 계정의 비밀번호를 알더라도 로그인할 수 없습니다. 2단계 인증 방법에는 인증 앱, 이메일, 보안 키를 사용하는 방법 등이 있으며, 가장 권장되는 방법은 인증 앱 사용입니다. 사용 방법은 계정에 2단계 인증 추가하기 도움말을 통해 확인하세요.

  • 체험 종료 후 반드시 계정에서 로그아웃하세요.

    로그아웃하지 않은 상태로 컴퓨터에서 자리를 뜰 경우, 그 컴퓨터를 사용하는 누구나 해당 계정에 접근할 수 있게 됩니다. 특히 학교 혹은 친구 집에서 와 같이 컴퓨터를 다른 사람과 함께 사용하는 경우 반드시 체험 종료 후 로그아웃을 하는 것을 잊지 마세요.

  • 어려운 비밀번호를 만드세요.

    다른 사람이 알아내기 어려운 비밀번호를 생성하세요. 다른 계정과 동일한 비밀번호를 사용하지 말고, 다른 사람이 알아내기 어려운 비밀번호를 생성하세요. pass123과 같은 일반적인 비밀번호를 사용하거나 사용자 이름 일부를 비밀번호로 설정하면 다른 사람이 쉽게 알아낼 수 있습니다.

  • 계정에 절대 다른 사람의 이메일 주소를 추가하지 마세요.

    계정에는 본인 또는 보호자의 이메일 주소만 등록해야 합니다. 만약 다른 사람의 이메일 주소를 등록할 경우 해당 사용자가 비밀번호 재 설정 이메일을 받을 수 있게 됩니다. 따라서 비밀번호를 변경해 사용자 본인을 차단할 수도 있습니다.

  • 사기 및 피싱 사이트 식별 방법을 확인하세요.

    일반적으로 모르는 사이트에 대한 링크를 클릭해서는 안 됩니다. Roblox에서 허용되는 링크는 roblox.com 에서 다른 페이지로 직접 연결되는 링크 또는 Roblox 커뮤니티 기준 에 명시되어 있는 링크 뿐입니다. 링크를 보내온 사용자가 누구인지 알고 있더라도, 공백이나 특이한 문자가 포함되어 있거나 익숙하지 않은 사이트에 대한 링크는 신뢰하지 마세요.

    사기 메시지의 예는 다음과 같습니다.

    • 사용자에게 링크를 보내는 사용자를 알고 있더라도 공백이나 특이한 문자를 포함하거나 익숙하지 않은 사이트에 대한 링크는 신뢰하면 안됩니다.

      사기 메시지의 예는 다음과 같습니다:

      f*ree*.ro*b*ux.o*mg.c*om (*을 제거하고 해당 사이트로 가서 무료 robux, 멤버십 그리고 치트키 받으세요.)
      "당신의 정보가 유출될 것입니다. 여기서 모든 정보를 볼 수 있습니다.

    • 귀하의 정보가 유출되고 있으며 이 링크에서 확인할 수 있습니다. 링크는 www.scamsite. com입니다. 사이트 링크에 "."와 "com" 사이의 공간을 제거주세요.

위의 예시는 사용자 이름 및/또는 비밀번호를 도용하기 위해 생성된 사이트입니다. 이러한 사이트에 접속할 경우 계정의 Robux나 아이템을 잃을 수 있습니다.

  • 체험 중 나타나는 팝업을 믿지 마세요.

    로그인 정보는 오로지 공식 roblox.com 웹사이트 또는 공식 Roblox 앱 로그인 화면에서만 입력해야 합니다. 만약 로그인 정보를 요청하는 체험이 있을 경우, Roblox로부터의 공식적인 요청처럼 보일지라도 절대 따르지 마세요. 이 경우 바로 해당 체험을 종료하고, 체험 세부 정보 페이지에 있는 신고하기 링크를 통해 신고하세요.

  • 프로그램을 다운로드하기 전에 보호자의 허락을 받으세요.

    인터넷 상에는 많은 종류의 악성 프로그램과 키로거가 있습니다. 절대로 보호자의 허락 없이 프로그램(.exe 파일) 을 다운로드하거나 실행하지 마세요. 다운로드하는 프로그램은 신뢰할 수 있는 회사의 것이어야 합니다. 악성 프로그램은 비밀번호와 같은 개인정보를 수집할 수 있습니다. 또한 브라우저에 설치된 확장 프로그램을 확인하고 삭제하는 것이 좋습니다.

    브라우저 확장 프로그램에 주의하세요 : Roblox는 브라우저 확장 프로그램을 만들지 않습니다. 따라서 현재 이용 가능한 Roblox 공식 브라우저 확장 프로그램은 존재하지 않습니다. 만약 타 사 개발자가 만든 브라우저 확장 프로그램을 사용할 경우 다음 지침을 따르는 것이 좋습니다.

    • Safari Extensions, Firefox Extensions 또는 Google Chrome Store와 같은 알려진 플랫폼을 통해서 만 확장 프로그램을 다운로드합니다.
    • 많은 사용자가 설치하지 않았거나 리뷰가 없는 브라우저 확장 프로그램을 피하십시오. 리뷰를 읽고 설치 후 문제에 대해 알아보십시오.
    • 확장에 필요한 권한을 주의 깊게 읽고 개인 정보, 전자 메일 주소 또는 기타 데이터에 대한 액세스를 제공하거나 허용하지 마십시오.
    • 의심스러운 경우 프로그램을 다운로드하지 마세요. Roblox 계정이 도용 되거나 다른 보안 위험에 노출될 수 있습니다.

  • 계정 정보를 화면으로 공유하지 마세요.

    계정 정보(예: 생년월일, PIN 등)에 액세스 중이거나 로그인하는 동안 컴퓨터 화면이나 화면의 이미지를 공유하지 마세요. 로그인 정보가 타인과 공유되어 계정 액세스 권한을 잃게 될 수도 있습니다. Roblox 직원은 절대 사용자의 로그인 정보를 묻지 않습니다. Roblox에서 이러한 요청을 받게 될 경우 신고 기능을 통해 모두 신고 해야 합니다.
