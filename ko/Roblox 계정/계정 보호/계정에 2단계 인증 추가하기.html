계정에 2단계 인증 추가하기

계정 보안을 강화하기 위해 2단계 인증 을 활성화하세요. 2단계 인증을 활성화하면, 새로운 기기에서 로그인할 때 인증 앱 혹은 Roblox 이메일로 보안 코드를 받아 입력하게 됩니다.

왜 2단계 인증이 중요한가요?

2단계 인증 기능에는 계정 보안의 이점이 있습니다.

  • 기능을 설정해 두면, 다른 누군가가 내 비밀번호를 알게 되어도 계정에 로그인하는 것이 불가능해집니다.
  • 인증 앱이나 이메일 계정에는 사용자 본인만 접근할 수 있어, 보안 코드 수령은 본인만이 가능합니다.
  • 체험 이용 권한 유지 및 Roblox 한정판 아이템 보호 등을 위한 추가적인 보안 수단이 되어 줍니다.

2단계 인증 켜기

  • 계정에 로그인합니다.
    • 로그인할 수 없을 경우, 다음 단계를 따라 비밀번호를 복구 하세요.
  • 이 기능을 사용하려면 먼저 이메일을 인증해야 합니다.
    • 다음은 이메일 주소를 인증하는 방법 입니다.
  • 계정 설정 으로 이동합니다.
    • 브라우저: 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱: 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  • 보안 탭을 선택합니다.
  • 다음 단계를 따라 원하는 유형의 2단계 인증을 켭니다.

참고 : 2단계 인증 수단으로 이메일과 인증 앱을 동시에 사용할 수는 없으므로, 둘 중 하나를 선택해야 합니다.

인증 앱을 통한 2단계 인증 설정하기

  1. 인증 앱 을 켭니다 (매우 안전).
    • 계정에 PIN이 있는 경우, 안내에 따라 PIN을 입력합니다.
  2. 계정에 비밀번호를 입력해 요청을 확인합니다.
  3. QR 코드를 스캔하거나 인증 앱의 코드를 수동으로 입력합니다.
    • 이를 위해 먼저 휴대폰에 인증 앱을 설치해 두어야 합니다.
      • 예: Google Authenticator, Microsoft Authenticator, Twilio Authy
  4. 복구 코드가 자동으로 생성됩니다. 이 코드를 안전한 곳에 보관하고 타인과 공유하지 말아야 합니다. 인증 앱과 복구 코드를 잃으면 계정에 접근할 수 없게 됩니다.

보안 키를 통한 2단계 인증 설정하기

보안 키는 현재 웹 브라우저에서만 지원됩니다. 인증 앱이 활성화되어 있고, Roblox 앱이 아닌 웹 브라우저로 로그인할 때만 보안 키를 사용할 수 있습니다.

'인증 앱을 통한 2단계 인증 설정하기' 단계를 따르세요. 보안 키를 설정하려면 2단계 인증을 설정해야 합니다.

  1. '웹 전용 보안 키'를 켭니다(매우 안전).
  2. 브라우저의 안내에 따라 선택한 보안 키를 확인합니다.
    • 일반적인 보안 키를 이용하거나 얼굴 인식, 지문 인식 등 기기에 설정해 둔 방법을 사용할 수 있습니다.
    • 일반적인 보안 키 예시: YubiKey 5, YubiKey 5C, Google Titan Security Key
    • 보안 키는 사용하는 브라우저(Safari, Chrome, Firefox) 및 운영 체제(Windows 또는 MacOS)에 따라 옵션이 다를 수 있습니다.
  3. 사용했던 키를 기억할 수 있도록 키 이름을 입력합니다.
    • 예: '파란 Yubikey 5'
  4. 이 단계를 반복해 다른 키를 추가하거나, 완료했을 경우 창을 닫습니다.

이메일 주소를 통한 2단계 인증 설정하기

  1. 이메일 을 켭니다(안전).
    • 계정에 PIN이 있는 경우, 안내에 따라 PIN을 입력합니다.
  2. 계정에 비밀번호를 입력해 요청을 확인합니다.

2단계 인증 끄기

  1. 계정에 로그인합니다.
    • 로그인할 수 없을 경우, 다음 단계를 따라 비밀번호를 복구하세요.
  2. 계정 설정 으로 이동합니다.
    • 브라우저: 사이트 오른쪽 상단에 있는 기어 아이콘을 누릅니다.
    • 모바일 앱: 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 탭을 선택합니다.
  4. 2단계 인증 을 끄기로 설정합니다.
  5. 계정 비밀번호를 입력해 설정을 완료합니다.

2단계 인증을 끄면 이전에 생성된 복구 코드도 비활성화됩니다. 계정 비밀번호 재설정과 관련한 도움이 필요할 경우 ‘비밀번호를 잊어버렸어요’ 도움말을 참고하세요.

복구 코드 받기

  1. 계정에 로그인합니다.
  2. 계정 설정 으로 이동합니다.
    • 브라우저: 사이트 오른쪽 상단에 있는 기어 아이콘을 누릅니다.
    • 모바일 앱 : 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 탭을 선택합니다.
  4. 복구 코드 하단의 생성 을 선택합니다.
  5. 설정 단계를 진행하며 계정 비밀번호를 입력합니다.
  6. 코드는 이번 한 번만 확인할 수 있으므로, 안전한 곳에 따로 저장합니다.

각 코드는 한 번만 사용할 수 있습니다. 복구 코드를 잊어버린 경우, 이 과정을 반복해 새 코드를 생성할 수 있습니다. 이전에 생성해 둔 복구 코드는 새 코드 생성 시 모두 비활성화됩니다.

계정 복구 요건

인증 앱을 통한 2단계 인증을 활성화해 계정 보안을 강화하세요. 이 경우 비밀번호나 이메일이 노출되더라도 계정을 안전하게 유지할 수 있습니다. 또한 계정이 도용되었을 경우 이를 30일 이내에 Roblox에 알려야 합니다. 지원 센터에서는 손해 복구를 돕는 과정에서 사용자가 설정해 둔 보안 조치를 확인하게 됩니다.  만약 이 기한 내에 계정 도용 사실을 알리지 못했을 경우 계정의 인벤토리를 복구하는 것이 불가능합니다.
