계정에 2단계 인증 추가하기

계정의 보안을 강화하기 위해서 2단계 인증 을 활성화하세요. 사용자는 새로운 기기를 통해 로그인할 때, 인증 앱 혹은 Roblox 이메일로 전송되는 고유한 보안 코드를 입력해야 합니다.

2단계 인증이 중요한가요?

이 기능은 귀하의 계정을 보호할 수 있는 장점이 있습니다.

  • 2단계 인증을 설정하면, 다른 누군가 비밀번호를 알게 되어도 내 계정에 로그인할 수 없습니다.
  • 인증 앱이나 이메일 계정은 본인만 접근할 수 있고 보안 코드는 사용자 본인만 받을 수 있습니다.
  • 사용자가 체험 이용과 Roblox 한정판 아이템 등을 잃지 않게 추가 보안을 진행합니다.

2단계 인증 켜기

  • 계정에 로그인합니다.
    • 로그인할 수 없는 경우, ' 비밀번호를 복구하기 '를 따르세요.
  • 이 기능을 사용하려면 인증된 이메일이 있어야 합니다.
    • 다음은 이메일 주소 인증하는 방법 입니다.
  • 계정 설정 으로 이동합니다.
    • 브라우저 : 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 : 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  • 보안 탭을 선택합니다.
  • 다음 단계를 따라 원하는 유형의 2단계 인증을 켭니다.

참고: 2단계 인증 수단으로 이메일과 인증 앱을 동시에 사용할 수는 없으므로, 둘 중 하나를 선택해야 합니다.

인증 앱에 2단계 인증 설정하기

  1. 인증 앱 을 켭니다 (매우 안전).
    • 계정에 PIN이 있는 경우, 안내에 따라 PIN을 입력합니다.
  2. 요청을 확인하기 위해 계정의 비밀번호를 입력합니다.
  3. QR 코드를 스캔하거나 인증 앱에서 설정 코드를 직접 입력합니다.
    • 모바일 장치에서 사용할 수 있는 인증 앱인지 확인합니다.
      • 예: Google Authenticator, Microsoft Authenticator, Twilio Authy
  4. 복구 코드가 자동으로 생성됩니다. 이 코드를 안전한 곳에 보관하고 타인과 공유하지 않습니다. 인증 앱과 복구 코드를 잃으면 계정에 접근할 수 없습니다.

보안 키를 통한 2단계 인증 설정하기

보안 키는 현재 웹 브라우저에서만 지원됩니다. 인증 앱이 활성화되어 있고, Roblox 앱이 아닌 웹 브라우저로 로그인할 때만 보안 키를 사용할 수 있습니다.

아래 "인증 앱을 통한 2단계 인증 설정하기" 단계를 따릅니다. 보안 키를 설정하려면 2단계 인증을 설정해야 합니다.

  1. "웹 전용 보안 키"를 켭니다 (매우 안전).
  2. 브라우저의 안내에 따라 선택한 보안 키를 확인합니다.
    • 일반적인 보안 키를 이용하거나 얼굴 인식, 지문 인식과 같은 하드웨어 장치에서 운영하는 방법을 사용할 수 있습니다.
    • 일반적인 키 예시: YubiKey 5, YubiKey 5C, Google Titan Security Key
    • 보안 키는 사용하는 브라우저 (Safari, Chrome, Firefox) 와 운영 체제 (Windows 또는 MacOS) 에 따라 옵션이 다를 수 있습니다.
  3. 사용했던 키를 기억할 수 있도록 키 이름을 입력합니다.
    • 예시: "파란 Yubikey 5"
  4. 이 단계를 반복해 다른 키를 추가하거나 완료했으면 창을 닫습니다.

이메일 주소로 2단계 인증 설정하기

  1. 이메일 을 켭니다 (안전).
    • 계정에 PIN이 있는 경우, 안내에 따라 PIN을 입력합니다.
  2. 요청을 확인하기 위해 계정의 비밀번호를 입력합니다.

2단계 인증 끄기

  1. 계정에 로그인합니다.
    • 로그인 할 수 없으면, ' 비밀번호를 복구하기 '를 따르세요.
  2. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 탭을 선택합니다.
  4. 2단계 인증 을 선택하고 끕니다.
  5. 계정 비밀번호를 입력해 기능을 끕니다.

2단계 인증을 끄면 이전에 생성된 복구 코드도 비활성화됩니다. 계정 비밀번호 재설정에 대한 도움말은 ‘ 비밀번호를 잊어버렸어요. ’ 도움말을 확인하세요.

복구 코드 받기

  1. 계정에 로그인합니다.
  2. 계정 설정 으로 이동합니다.
    • 브라우저 : 사이트 오른쪽 상단에 있는 기어 아이콘을 누르세요.
    • 모바일 앱 : 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 탭을 선택합니다.
  4. 복구 코드 하단의 생성 을 선택합니다.
  5. 설정의 단계 중 계정 비밀번호를 입력합니다.
  6. 코드는 한 번만 확인할 수 있으므로 안전한 곳에 저장하세요.

각 코드는 한 번만 사용할 수 있습니다. 복구 코드를 잊어버린 경우, 이 과정을 반복하여 새 코드를 생성할 수 있습니다. 이전에 생성된 복구 코드는 새 코드를 생성할 때 비활성화됩니다.

계정 복원 자격

계정을 복원하기 위해선 계정을 보호할 수 있는 인증 앱이 설치되어 있어야 합니다. 또한 계정이 도용된 이후 30일 이내에 Roblox에 알려야 합니다. 이 시간 내에 알리지 못하면, 계정의 인벤토리를 복구할 수 없습니다.

문제 해결

자세한 내용은 2단계 인증 문제 해결 도움말을 확인하세요.
