계정에 2단계 인증 추가하기

계정을 보안을 강과하기 위해 2단계 인증은 사용자의 계정에 대한 추가 보안 기능입니다. 이는 비밀번호가 도난당한 경우에 고경자가 계정에 액세스하는 것을 방지하는 데 특히 도움이 됩니다.

2단계 인증 확인하는 방

Roblox가 다양한 2단계 인증 추가하는 방법이 있습니다. 이 때는, 사용자가 한 번에 하나의 2단계 인증 방법만 선택할 수 있습니다.

계정에 2단계 인증을 추가하려면:

  1. Roblox 계정에 로그인합니다.
    • 계정에 로그인 안 된다면, 비밀번호를 재설정 을 위해 이 단계를 따라하세요.
  2. 계정 설정 으로 이동합니다.
    • 브라우저: 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 : 3점 아이콘을 찾고 기어 아이콘을 클릭합니다.
  3. 보안 탭으로 이동합니다.
  4. 원하는 2단계 인증 방법의 토글을 켜고 화면에 지침을 따릅니다. 2단계 인증 선택에 따라 지침이 다릅니다.
    • 중 : 2단계 인증을 활성화하려면 계정에 확인된 이메일 주소 가 있어야 합니다.

이메일 주소를 통한 2단계 인증 설정하기

이메일을 통한 2단계 인증을 활성화하면 비밀번호로 로그인할 때마다 코드를 입력하라는 메시지가 표시됩니다.

Roblox 계정에 있는 확인된 이메일의 받은 편진함에서 코드를 찾을 수 있습니다. Roblox 화면에서 코드를 입력한 다음에 계정에 성공적으로 로그인됩니다.

이메일 OTP나 비밀번호를 통해 로그인하면 2단계 인증이 우회됩니다.

인증 앱을 통한 2단계 인증 설정하기

인증 앱을 통한 2단계 인증을 활성화하려면 모바일 기기에 인증 앱이 있어야 합니다. 인증 앱이 없다면 App Store에서 다운로드할 수 있습니다. 다른 앱은 Google Authenticator, Microsoft Authenticator, Duo, Twilio Authy등 선택할 수 있습니다.

인증 앱을 통한 2단계 인증을 활성화하면 비밀번호 또는 이메일 OTP로 로그인할 때마다 코드를 입력하라는 메시지가 표시됩니다.

모바일 장치에 있는 선택한 인증 앱을 통해 코드를 찾을 수 있습니다. Roblox 화면에서 코드를 입력한 다음에 계정이 성공적으로 로그인됩니다.

패스키를 통해 로그인하면, 2단계 인증이 우회됩니다.

보안 키를 통한 2단계 인증 설정하기

보안 키를 통한 2단계 인증을 활성화하려면 계정에 인증 앱을 통한 2단계 인증도 있어야 합니다. 이는 보안키는 웹 브라우저에서만 지원되기 때문입니다.

보안 키는 Yubikey 5, Google Titan 보안 키와 같은 물리적 키일 수도 있습니다. 또한, 얼굴 인식이나 지문 스캐너와 캍은 하드웨어에서 관리하는 생체 인식 방법일 수도 있습니다. 보안 키 옵션은 로그인하는 브라우저, 운영 체제, 장치에 따라 달라집니다.

패스키를 통해 로그인하면, 2단계 인증이 우회됩니다.

교차 기기 팝업을 통한 2단계 인증 설정하기

참고 : 이 기능은 실험적인 기능이며 모든 사용자가 사용할 수 있는 것은 아닙니다.

사용자가 교차 기기 팝업을 통한 2단계 인증을 명시적으로 활성할 수 없습니다. 계정에 위에 나열된 2단계 인증 방법이 활성화되어 있지 않거나 Roblox가 계정에 의심스러운 로그인 시도가 있음을 감지한 경우 이러한 메시지가 표시됩니다.

교차 기기 팝업에 의한 2단계 확인을 받으면 Roblox 계정에 로그인한 모바일 장치를 사용하여 로그인 시도를 승인 또는 거부하라는 메시지가 표시됩니다. 사용자가 장치의 유형, 지역 위치, IP 주소, 날짜 및 시간 등 로그인 시도에 관련 세부 사항을 검토할 수 있습니다.

로그인을 확인하는 다른 방법 선택하기

2단계 인증 교차 기기 팝업이 표시되는 경우 다른 방법으로 로그인을 인증할 수 있는 옵션이 있습니다.

이메일로 로그인 코드 받기

계정에 확인된 이메일 주소가 있는 경우 "코드를 이메일로 보내기" 선택이 표시됩니다. 이 옵션을 선택하면 이메일 주소로 고유 코드가 전송됩니다. 성공적으로 로그인하려면, Roblox 화면에 이 코드를 입력하세요. 코드를 다른 사람에게 공유하지 마세요.

백업 코드

계정에 성공적으로 로그인하려면 저장된 백업 코드를 입력할 수도 있습니다. 이 코드를 다른 사람에게 공유하시면 안 됩니다.
