일회용 코드로 로그인하기

일회용 코드를 사용하면 사용자 이름과 비밀번호를 입력하지 않고 로그인할 수 있습니다. 현재 일회용 코드는 일부 사용자만 이용이 가능합니다.

참고: 이 기능을 사용하려면 먼저 계정의 이메일 주소를 인증해야 합니다. 이메일 주소(13세 미만 사용자의 경우 보호자 이메일)를 추가하고 인증하려면 계정 정보 페이지로 이동하세요. 계정 이메일 주소를 인증하는 방법은 이 도움말 을 통해 확인할 수 있습니다.

일회용 코드로 로그인하려면 어떻게 해야 하나요?

  1. 로그인 페이지에서 이메일로 일회용 코드 받기 버튼을 누릅니다.
  2. 계정의 인증된 이메일 주소를 입력합니다. 이메일에서 6자리 일회용 코드를 확인합니다.
  3. 일회용 코드를 입력하고 로그인합니다(이메일이 여러 계정에서 인증되어 있는 경우 하나를 선택).

일회용 코드를 받지 못했어요

일회용 코드가 포함된 이메일을 받지 못한 경우 다음 단계를 시도해 보세요.

  • 스팸 또는 정크 폴더를 확인합니다.
  • 재전송 버튼을 누른 다음 받은 편지함을 다시 확인합니다.
  • 계정에 등록된 정확한 이메일 주소를 입력했는지 확인합니다.
  • 이메일 주소 인증이 완료되어 있는지 확인합니다. 이메일 주소를 인증하는 방법은 이 도움말 에서 확인할 수 있습니다.

이러한 단계를 수행하고 난 후에도 코드를 받지 못했다면 비밀번호로 로그인하세요. 일회용 코드를 해당 이메일로 받을 수 있기까지는 어느 정도 시간이 소요될 수 있습니다.

일회용 코드를 받아서 입력했지만 작동하지 않아요

일회용 코드가 작동하지 않으면 다음 단계를 시도하세요.

  1. 일회용 코드를 요청한 기기와 동일한 기기에 코드를 입력했는지 확인합니다.
  2. 재전송 버튼을 누르고 새 코드로 다시 시도합니다.

이러한 단계를 수행하고 난 후에도 로그인이 불가능할 경우 비밀번호로 로그인하세요. 해당 이메일로 전송되는 코드가 사용 가능해지기까지는 어느 정도 시간이 소요될 수 있습니다.

안전 수칙

  • 일회용 코드를 타인과 공유하지 마세요. Roblox는 절대 여러분의 일회용 코드를 요청하지 않습니다.
  • 요청하지 않은 일회용 코드를 이메일로 전송받은 경우, 해당 이메일을 무시하는 것이 좋습니다.
  • 의도치 않게 일회용 코드가 포함된 다수의 이메일을 전송받는 경우, 계정의 이메일 주소를 변경하고 인증 앱을 통한 2단계 인증을 설정 할 것을 적극 권장합니다.
