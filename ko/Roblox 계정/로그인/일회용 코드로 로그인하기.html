일회용 코드로 로그인하기

일회용 코드를 사용하면 사용자 이름과 비밀번호를 입력하지 않고 로그인할 수 있습니다. 현재 일회용 코드는 일부 사용자만 이용이 가능합니다.

참고: 이 기능을 사용하려면 먼저 계정의 이메일 주소를 인증해야 합니다. 계정 정보 페이지로 이동하여 이메일 주소를 추가하고 확인할 수 있습니다. (3세 미만 사용자의 경우 보호자 이메일) 아래에서 계정 이메일 주소를 확인하는 방법에 대한 도움말 을 확인할 수 있습니다.

일회성 코드로 로그인하려면 어떻게 해야 하나요?

  1. 로그인 페이지에서 일회성 코드를 이메일로 보내기 버튼을 누릅니다.
  2. 계정에서 확인된 이메일 주소를 입력합니다. 6자리 일회성 코드를 이메일로 확인할 수 있습니다.
  3. 일회성 코드를 입력하여 로그인합니다. (이메일이 여러 계정에서 확인된 경우 하나만 선택할 수 있습니다.)

일회성 코드를 받지 못한 경우

일회성 코드가 포함된 이메일을 받지 못한 경우 다음 단계를 시도해 보세요.

  • 스팸 또는 정크 폴더를 확인합니다.
  • 재전송 버튼을 누른 다음 받은 편지함을 다시 확인합니다.
  • 계정에 있는 이메일 주소를 정확히 입력했는지 확인합니다.
  • 이메일 주소가 인증되었는지 확인하세요. 필요한 경우 계정 이메일 주소를 인증하는 방법에 대한 도움말 을 확인할 수 있습니다.

이러한 단계가 작동하지 않으면 비밀번호로 로그인하세요. 일회성 코드가 이메일에 다시 작동하려면 잠시 기다려야 할 수 있습니다.

일회성 코드를 받았지만 작동하지 않습니다.

일회성 코드가 작동하지 않으면 다음 단계를 시도하세요.

  1. 일회성 코드를 요청한 기기와 동일한 기기로 일회성 코드를 입력했는지 확인합니다.
  2. 재전송 버튼을 누르고 새 코드로 다시 시도합니다.

이러한 단계가 작동하지 않으면 비밀번호로 로그인하세요. 일회성 코드가 이메일에 다시 작동하려면 잠시 기다려야 할 수 있습니다.

안전 수칙

  • 누구와도 일회성 코드를 공유하지 않아야 합니다. Roblox는 절대 회원님에게 일회성 코드를 요청하지 않습니다.
  • 요청하지 않은 일회성 코드를 이메일로 받은 경우, 해당 이메일을 무시하는 것이 좋습니다.
  • 일회성 코드가 포함된 원치 않는 이메일을 여러 번 받는 경우, 계정의 이메일을 변경하고 인증 앱을 통하여 2단계 인증을 설정 할 것을 적극 권장합니다.
