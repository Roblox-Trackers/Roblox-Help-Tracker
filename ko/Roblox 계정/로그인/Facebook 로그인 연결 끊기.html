Facebook 로그인 연결 끊기

Facebook 로그인 연결을 끊으면 다시는 Facebook 로그인을 할 수 없습니다. 신중히 결정하세요.

  1. 계정에 로그인합니다.
    (로그인이 되지 않으면 다음 단계를 수행하여 비밀번호를 복구 하세요.)
  2. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트 오른쪽 상단 모서리에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 소셜 로그인 섹션에서 연결 해제 버튼 을 선택합니다.

만약 아직 이메일 주소를 등록한 적이 없다면, 먼저 이메일 주소를 등록하고 인증하세요. 이메일 주소를 등록해 두면 추후 계정 분실 시 더 효율적인 문제 해결이 가능합니다. 더 자세한 내용은 ' 이메일 주소 인증하기 ' 도움말에서 확인하세요.
