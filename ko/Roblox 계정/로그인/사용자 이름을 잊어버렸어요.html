사용자 이름을 잊어버렸어요

이 도움말에서는 인증된 이메일 주소 또는 전화번호가 있는 경우 Roblox 사용자 이름을 찾는 단계를 안내합니다.

이메일을 사용해 사용자 이름 찾기

  1. 로그인 페이지에 있는 ' 비밀번호 또는 사용자 이름을 잊으셨나요? ' 링크를 클릭합니다.
  2. 기본 페이지 설정은 ‘ 비밀번호를 잊어버렸어요 ’ 탭으로 되어 있습니다. 아래 그림과 같이 사용자 이름 탭을 선택합니다.
  3. 이메일 박스에 이메일 주소를 입력하고 파란색 제출 버튼을 누릅니다. 아래와 같은 확인 메시지가 나타납니다.
    • 계정에 입력한 대로 이메일 주소를 정확히 입력해야 합니다.
  4. 모든 계정 목록이 포함된 이메일이 전송됩니다.
  5. 사용자 이름을 이용해 로그인합니다.

혹시 몇 분 후에도 이메일이 도착하지 않으면 정크/스팸 폴더를 확인해 보세요.

전화번호를 사용해 사용자 이름 찾기

  1. 로그인 페이지에 있는 ' 비밀번호 또는 사용자 이름을 잊으셨나요? ' 링크를 클릭합니다.
  2. ' 전화번호를 사용해 사용자 이름을 찾습니다 '를 선택합니다.
  3. 해당 국가 번호 를 선택합니다 .
  4. 전화번호 박스에 전화번호를 입력하고 파란색 제출 버튼을 누릅니다.
  5. 사용자 이름과 함께 SMS 가 휴대 전화로 전송됩니다.
  6. 사용자 이름을 이용해 로그인합니다.

그 외 다른 방법

이 방법은 사용자 이름을 잊어버리기 이전에, 이메일 또는 전화번호를 계정에 추가해 둔 경우에만 사용할 수 있습니다. Roblox에서 결제를 한 적이 있는 경우, 일부 구매 영수증에는 사용자 이름이 포함되어 있을 수 있으며, 이 경우 사용자 이름을 찾을 수 있는 유일한 방법이 될 수 있습니다.

안타깝게도 이메일, 전화번호 또는 구매 내역이 없으면 계정을 복구할 수 있는 방법이 없습니다.
