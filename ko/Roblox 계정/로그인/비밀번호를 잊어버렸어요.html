비밀번호를 잊어버렸어요

비밀번호가 기억나지 않으면 비밀번호 찾기 기능을 사용하여 비밀번호를 재설정하고 새 비밀번호를 만들어야 합니다. 모바일 기기를 통해 비밀번호를 재설정하는 경우 기본적으로 전화번호를 입력하라는 메시지가 표시되지만 원하는 경우 이메일로 변경할 수 있습니다.

  • 계정에 이메일 주소를 추가하는 것이 좋습니다. 자세한 내용은 이메일 인증하기 및 계정을 안전하게 유지하세요 도움말을 참고하세요.

이메일을 사용하여 비밀번호 재설정하기

비밀번호 재설정 버튼을 누르거나 혹은 이메일 링크를 클릭합니다.

  1. 로그인 화면에서 ' 비밀번호 또는 사용자 이름을 잊으셨나요? '를 클릭합니다.
  2. 상자에 이메일 주소를 입력합니다. 제출 버튼을 누릅니다.
    • 유효한 이메일 주소를 추가했는지 확인하세요. 예를 들어 주소가 "name.example@domain.com"이고 Roblox 계정에서 마침표 없이 "nameexample@domain.com"인 경우 마침표 없이 입력해야 합니다.
  3. 지침이 포함된 이메일이 현재 계정과 연결된 이메일 주소로 전송됩니다. 이메일에서 비밀번호 재설정 버튼을 누르세요.
    • 만약 이메일이 받은 편지함으로 안 들어 온다면, 정크 또는 스팸 폴더를 확인하세요. 또한 "password_reset@roblox.com" 및 "@appmail.roblox.com"이 차단되지 않도록 연락처 목록에 있는지 확인하세요.
  4. 암호 재설정 페이지가 열립니다.
    • 하나의 계정: 새 비밀번호를 입력하는 페이지가 열립니다.
    • 여러 계정: 암호를 재설정하려는 계정을 선택합니다.
  5. 새 비밀번호를 입력하고 확인합니다. 제출 버튼을 누릅니다.
    • 새 암호를 만드는 것인지 확인하세요. 이전 비밀번호나 다른 계정에서 사용한 비밀번호를 사용하지 말아주세요.

휴대폰 번호를 이용하여 비밀번호 재설정하기

이메일 입력 칸에 이메일 을 입력하고 제출 버튼을 클릭합니다.
계정에 등록된 이메일 주소와 일치하는지 다시 한번 확인하세요.

  1. 로그인 화면에서 ' 비밀번호 또는 사용자 이름을 잊으셨나요? ' 를 클릭합니다.
  2. 전화번호를 이용한 비밀번호 재설정 을 선택합니다.
  3. 국가 코드 를 선택합니다.
  4. 상자에 코드를 입력합니다. 제출 버튼을 누릅니다.
  5. 인증 을 선택하고 프로세스를 완료하세요. 해당 프로세스는 6 자리 인증 코드 를 휴대폰 문자로 발송합니다.
  6. 상자에 코드(6자리) 번호를 입력합니다. 인증 버튼을 누릅니다.
  7. 새 비밀번호를 입력하고 확인합니다. 제출 버튼을 누릅니다.

아직도 문제가 있나요?

  • 기술적인 문제 해결
  • 계정에 이메일 주소나 전화번호를 추가하지 않고 비밀번호 재설정을 시도한 경우 이 기능으로 비밀번호를 재설정할 수 없습니다.
    • 지원 센터 로 로그인 문제를 자세히 설명해주세요. 모든 사용자는 계정 관련하여 초기화하거나 정보 변경을 하기 전에 계정 소유권을 반드시 확인해 주셔야 합니다.

업데이트: Facebook 로그인 제거됨

Facebook 자격 증명으로 로그인하는 기능은 2021년 11월 28일 부로 Roblox 플랫폼에서 제거됩니다. Facebook 로그인이 있는 사용자는 자신의 계정에 비밀번호를 설정하라는 알림을 받게 됩니다. 이 기능이 제거되어 로그인할 수 없는 경우 위의 다음 단계에 따라 비밀번호를 재설정 하세요. 이 단계를 따른 후에도 계정에 액세스할 수 없거나 비밀번호를 재설정할 수 없는 경우 고객 지원 팀에 문의하세요.
