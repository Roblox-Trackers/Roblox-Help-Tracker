비밀번호를 잊어버렸어요

만약 비밀번호가 기억이 안 나는 경우, 비밀번호를 재설정하고 새롭게 생성해야 합니다.

이메일을 사용하여 비밀번호 재설정하기

  1. 로그인 화면에서 ' 비밀번호 또는 사용자 이름을 잊으셨나요? '를 클릭합니다.
  2. 이메일 입력 칸에 이메일 을 입력하고 제출 버튼을 클릭합니다.
    계정에 등록된 이메일 주소와 일치하는지 다시 한번 확인하세요.
  3. 비밀번호 재설정 버튼을 누르거나 혹은 이메일 링크를 클릭합니다.
    • 만약 하나의 계정만 존재할 경우 바로 비밀번호 변경 페이지가 나타납니다.
    • 만약 한 개 이상의 계정이 존재할 경우 존재하는 계정 리스트가 보입니다. 비밀번호 변경을 원하는 계정을 선택한 후 비밀번호 변경을 하세요.
  4. 원하는 비밀번호를 입력한 후 비밀번호 확인까지 완료되면, 제출 을 누릅니다.

만약 이메일이 받은 편지함으로 안 들어 온다면, 정크 또는 스팸 폴더를 확인하세요.

휴대폰 번호를 이용하여 비밀번호 재설정하기

  1. 로그인 화면에서 ' 비밀번호 또는 사용자 이름을 잊으셨나요? ' 를 클릭합니다.
  2. 전화번호를 이용한 비밀번호 재설정 을 선택합니다.
  3. 국가 코드 를 선택합니다.
  4. 휴대폰 번호를 휴대폰 번호 입력란에 기입하고 제출 버튼을 선택합니다.
  5. 인증 을 선택하고 프로세스를 완료하세요. 해당 프로세스는 6 자리 인증 코드 를 휴대폰 문자로 발송합니다.
  6. 입력란에 6 자리 인증 코드 를 입력하시고, 인증 버튼을 누릅니다.
  7. 원하는 비밀번호를 입력한 후 비밀번호 확인까지 완료되면, 제출 을 누릅니다.

아직도 문제가 있나요?

  • 기술적인 문제 해결
  • 지원 센터 로 로그인 문제를 자세히 설명해주세요. 모든 사용자는 계정 관련하여 초기화하거나 정보 변경을 하기 전에 계정 소유권을 반드시 확인해 주셔야 합니다.
