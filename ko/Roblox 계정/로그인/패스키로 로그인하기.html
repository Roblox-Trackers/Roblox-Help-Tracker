패스키로 로그인하기

패스키가 무엇인가요?

패스키는 기존의 비밀번호보다 더 안전하고 사용하기 쉬운 로그인 수단입니다. 패스키를 사용하면 기기의 잠금 해제 방식을 그대로 사용해 로그인할 수 있기 때문에 비밀번호를 따로 기억할 필요가 없습니다. 패스키의 원리는 공개키 암호화입니다. 이는 고유한 한 쌍의 암호 키를 생성해 사용자 계정과 Roblox가 나눠 갖는 방식입니다.


패스키가 생성되면, 같은 Apple, Google, Windows 플랫폼 계정이나 비밀번호 관리자를 사용하는 기기에서 동기화되어 언제든 바로 쓸 수 있습니다. 패스키에 관한 더 자세한 정보는 Google , Apple , Microsoft 에서 제공하는 각 페이지에서 확인하세요.

패스키는 피싱이나 크리덴셜 스터핑 공격 방지에 뛰어납니다. 번거로움을 줄이고 보안을 강화하기 위해 지금 바로 패스키를 설정하세요.

패스키 추가하기

다음의 간단한 3단계 절차로 패스키를 추가할 수 있습니다.

  • iOS 앱이나 웹 브라우저에서 설정 으로 이동합니다. 화면이 뜨면 로그인합니다.
    • 참고 : Android는 곧 지원될 예정입니다.
  • 계정 정보 > 로그인 방식에서 관리 버튼을 누릅니다.
    • 참고 : 비밀번호나 이메일로 전송된 일회용 코드로 사용자 인증을 해야 할 수도 있습니다.
  • 패스키 추가를 완료하기 위해 기기를 잠금 해제하라는 요청을 받게 됩니다.

참고 : 보안을 위해 개인용 기기에만 패스키를 추가하세요.

패스키로 로그인하기

패스키를 추가한 후에는 Face ID, Touch ID, 숫자 입력 방식과 같이, 사용 중인 기기 잠금 해제 방식을 그대로 사용해 Roblox에 로그인할 수 있습니다.

다른 기기에서 패스키로 로그인하기

모바일 기기에 패스키를 추가했다면 다음 절차를 수행해 같은 패스키로 컴퓨터에 로그인할 수 있습니다.

  • 웹 브라우저 로그인 화면에서 사용자 이름 입력 부분을 클릭하고 주변 기기의 패스키… 또는 다른 패스키 사용 을 선택합니다.
  • 화면에 QR 코드가 뜹니다. 모바일 기기로 해당 QR 코드를 스캔한 후 화면에 뜨는 지시 사항을 따릅니다.

패스키 관리하기

패스키를 사용하면 보안을 강화할 수 있지만 공유 기기에 패스키를 추가하는 것은 권장되지 않습니다. 공유 기기에 패스키를 설정할 경우, 해당 기기를 잠금 해제할 수 있는 사람이면 누구나 사용자의 계정에 액세스할 수 있게 되기 때문입니다.

이러한 경우, 최대한 빨리 '설정'에서 패스키를 삭제하세요.

  • iOS 앱이나 웹 브라우저에서 설정 으로 이동합니다. 화면이 뜨면 계정에 로그인합니다.
    • 참고 : Android는 곧 지원될 예정입니다.
  • 계정 정보 > 로그인 방식에서 관리 버튼을 누릅니다.
    • 참고 : 이미 패스키를 추가한 경우 '관리' 버튼이 대신 표시됩니다.
  • 삭제하려는 패스키 옆에 위치한 휴지통 아이콘을 누릅니다.
    • 비밀번호나 이메일로 전송된 일회용 코드로 사용자 인증을 해야 할 수도 있습니다.

참고 : Roblox에서만 패스키를 삭제하고 기기 설정에서는 삭제하지 않으면, 이미 삭제한 패스키가 로그인 옵션으로 뜰 수 있습니다.

2단계 인증을 활성화한 사용자의 경우

계정에 대한 2단계 인증이 활성화되어 있는 상태에서 패스키로 로그인하기를 선택하면 2단계 인증을 진행하라는 메시지는 뜨지 않습니다. 패스키를 통한 로그인에 성공할 경우 이는 실제 기기 소유자가 로그인했음을 인증하는 것이기 때문입니다. 계정은 안전하게 보호되고 있으니 안심하셔도 됩니다.

참고 : 패스키가 아닌 방식으로 로그인하면 평소처럼 2단계 인증을 진행하라는 메시지가 뜹니다.

자주 묻는 질문

패스키를 사용할 수 있는 기기에는 어떤 것들이 있나요?

현재 iOS 기기와 웹 브라우저에서 패스키로 Roblox 계정에 로그인할 수 있습니다. 패스키를 지원하는 기기는 다음과 같습니다.

  • iOS 16 이상이 설치된 모바일 기기.
  • Windows 10, macOS Ventura, ChromeOS 109 이상이 설치된 컴퓨터.
    • 지원 브라우저: Chrome 버전 109 이상, Safari 버전 16 이상, Edge 버전 109 이상.
  • FIDO2 표준을 준수하는 하드웨어 키

로그인 화면에서 패스키가 안 보이면 어떻게 하나요?

이미 패스키를 계정에 저장했는데도 로그인 시 뜨지 않는다면 기기에 따라 다음과 같은 방법으로 해결할 수 있습니다.

  • 현재 기기와 패스키가 저장된 기기의 운영 체제가 같다면(예: 패스키가 저장된 기기와 현재 기기 모두 iOS 기기인 경우), 패스키를 저장했을 때와 동일한 iCloud나 Google 계정을 사용했는지 확인해 보세요. 기기 버전을 최신으로 업데이트해 볼 수도 있습니다.
  • 현재 기기와 패스키가 저장된 기기의 운영 체제가 서로 다르다면(예: 저장된 기기는 iOS, 현재 기기는 Windows 컴퓨터) 패스키는 동기화되지 않습니다. 하지만 빠른 로그인, 이메일을 통한 일회용 코드, 비밀번호 등의 다른 방식으로 계정에 다시 액세스할 수 있습니다. 로그인에 성공했다면, 추후 더 쉬운 액세스를 위해 새 기기에 패스키를 추가하는 것이 좋습니다.

패스키로 로그인이 안 되면 어떻게 하나요?

Roblox에서 이미 삭제한 패스키로 로그인을 시도할 때 이러한 문제가 발생합니다. 기기 설정에서도 패스키를 삭제하지 않으면 해당 패스키가 계속 로그인 옵션에 뜰 것입니다. 각 플랫폼의 패스키 관리 방법에 대한 더 자세한 정보는 Apple , Google , Microsoft 의 도움말에서 확인하세요.

빠른 로그인, 이메일을 통한 일회용 코드, 비밀번호 등 다른 방식으로 로그인해 보세요. 필요하다면 계정 복구를 진행하세요.
