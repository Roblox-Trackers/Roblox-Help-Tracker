비밀번호는 어떻게 변경하나요?

비밀번호 변경하기

  1. 계정에 로그인합니다.
    (로그인이 되지 않으면 다음 단계를 수행하여 비밀번호를 복구 하시기 바랍니다.)
  2. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트 오른쪽 상단 모서리에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 계정 정보 탭을 선택합니다.
  4. 비밀번호 변경 버튼을 선택합니다.(연필과 종이처럼 보이는 아이콘)
  5. 현재 Roblox 비밀번호를 입력합니다.
  6. 확인을 위해 새 비밀번호를 두 번 입력합니다.

보안을 강화하려면 다른 웹 사이트에서 사용하지 않은 새 비밀번호를 만드시기 바랍니다.

보안 추가하기

추가 이메일 보안 설정을 아직 하지 않은 경우, 이메일 주소를 추가한 다음 이메일 인증을 하시기 바랍니다. 이메일 주소 인증은 계정이 분실된 경우 지원 센터에서 문제 해결을 하는데 큰 도움이 됩니다. 이메일 주소를 인증하기 도움말 에서 이메일 인증 방법을 확인할 수 있습니다.

또한 2단계 인증 (2SV)을 추가 보안 조치로 사용하여 다른 사용자를 계정에 접근하는 것을 차단할 수 있습니다. 해당 기능은 보안 설정 탭에서 찾을 수 있습니다. 이 뛰어난 보안 기능에 대한 자세한 내용은 FAQ 도움말 을 참고하시기 바랍니다.
