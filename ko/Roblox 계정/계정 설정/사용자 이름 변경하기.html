사용자 이름 변경하기

사용자 이름은 계정 설정 메뉴에서 변경 가능하며, 이때 Robux가 필요합니다. 하지만 체험이나 게임 내에서 나타나는 표시 이름 은 무료로 변경할 수 있습니다.

사용자 이름을 변경하려면 어떻게 하면 되나요?

  1. 계정에 로그인합니다.
    (로그인이 되지 않으면 다음 단계를 수행해 비밀번호를 복구 하세요.)
  2. 계정 설정 으로 이동합니다.
    • 브라우저 - 사이트 오른쪽 상단 모서리에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 계정 정보 탭을 선택합니다.
  4. 사용자 이름 변경 버튼(연필과 종이처럼 보이는 아이콘)을 선택합니다.
    • 새로운 사용자 이름을 입력합니다.
    • 현재 사용 중인 비밀번호를 입력합니다.
  5. 구매 버튼을 누릅니다.

이후부터는 새로운 사용자 이름과 현재 비밀번호로 로그인할 수 있습니다.

이전에 사용했던 사용자 이름을 되찾고 싶어요

이전에 사용했던 사용자 이름 목록은 사용자 검색과 더불어 계정 설정 및 프로필 페이지에서 찾을 수 있습니다. 따라서 이러한 방법을 이용해 사용자 이름을 이전에 사용했던 이름으로 되돌릴 수 있습니다. 이 또한 변경에는 Robux가 필요합니다.

사용자 이름 변경과 관련해서 더 알아야 할 사항이 있나요?

다음과 같은 제한 사항이 있습니다.

  • 다른 사용자의 계정과 똑같은 이름을 사용할 수 없습니다.
  • 사용 중인 또 다른 계정과 이름을 서로 바꿀 수 없습니다.
  • 사용자 이름을 대문자로 변경하는 것과 같은 간단한 변경에는 할인 혜택이 제공되지 않습니다.

중요 참고 사항

  • 생성일의 경우, 원래 계정의 생성일이 새로운 사용자 이름으로 이관됩니다.
  • 다른 플레이어가 자신의 이전 사용자 이름을 볼 수 있습니다.
