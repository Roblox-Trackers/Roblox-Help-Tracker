기프트 카드 사용 및 적용 방법

  • Roblox 기프트 카드는 Roblox.com/redeem 브라우저에서만 사용할 수 있습니다. Roblox 모바일 앱이나 비디오 게임 콘솔에서는 사용이 불가능합니다.
  • Roblox 기프트 카드는 다음과 같은 두 가지 유형의 통화를 제공합니다.
    • 크레딧 - Robux 또는 Roblox Premium 멤버십을 구매할 수 있는 실제 현금 가치. 여기 에서 크레딧을 구매할 수 있습니다.
    • Robux - 계정의 잔액에 추가되는 가상 통화. Amazon.com 의 Roblox 스토어에서만 구매할 수 있습니다.

계정에서 기프트 카드 사용

  1. 브라우저에서 Roblox.com/redeem 으로 이동합니다.

  2. 로그인하거나 계정을 새로 만듭니다.

  3. PIN 또는 코드를 찾아 웹사이트의 흰색 상자에 입력합니다.

  4. 사용 을 클릭합니다.

  5. 계정에 크레딧 추가가 완료되었다는 메시지가 나타납니다.

    때로는 코드의 문자가 숫자처럼 보이거나 숫자가 문자처럼 보일 수 있습니다. 예를 들어 기프트 카드의 숫자 영(0)은 사실 알파벳 'O'일 수 있습니다. 기프트 카드 사용 시 문제가 발생할 경우, 아래와 같이 비슷해 보이는 문자나 숫자를 바꿔 입력해 보세요.

0(숫자 영) - O

1(숫자 일) - I(알파벳 대문자 아이)

2(숫자 이) - Z

5(숫자 오) - S

6(숫자 육) - G/Q

8(숫자 팔) - B

Roblox 크레딧 사용하기

계정에서 Roblox 크레딧을 적용하면 Roblox Premium 멤버십 또는 Robux를 구매할 수 있습니다. Roblox 크레딧은 아래의 3곳에서 전환할 수 있습니다.

  • Roblox 기프트 카드 사용 페이지 (사용 가능한 크레딧 아래에 위치)
  • 멤버십 페이지
  • Robux 페이지
  1. 구매하려는 제품을 선택합니다.
  2. 구매 를 클릭합니다.
  3. 확인 메시지가 나타납니다.
  4. Roblox 크레딧이 남아 있을 경우, Robux를 더 구매할 것인지 묻는 메시지가 나타납니다(계정에 충분한 크레딧을 보유하고 있을 경우).
      • 구매를 클릭하면 남은 Roblox 크레딧이 Robux로 전환됩니다.
      • 취소를 클릭하면 계정에 크레딧이 그대로 유지됩니다.
