그룹 설명에 관한 소셜 미디어 링크

그룹 관리자는 사용자 프로필에서 사용한 방법과 마찬가지로 그룹의 설명 내에서 소셜 미디어 링크를 포함할 수 있습니다.

다음은 추가할 수 있는 소셜 미디어 링크입니다:

  • Facebook
  • Twitter
  • Youtube
  • Twitch
  • Discord

그룹 서명에 소셜 미디어 링크를 추가하기

  1. 계정에 로그인합니다.
  2. 그룹 페이지로 이동합니다.
  3. 링크를 추가하고 싶은 해당 그룹을 선택합니다.
  4. 페이지 오른쪽의 점 3개 메뉴를 클릭하고 그룹 구성 버튼 을 클릭합니다.
  5. 그룹 페이지에서 소셜 미디어 메뉴 옵션을 선택합니다.
  6. URL을 입력하고 미디어 타입을 선택한 다음 제목을 입력합니다.


  7. 저장 을 클릭합니다
  8. 링크를 저장한 다음 그룹 페이지에서 소셜 미디어 링크가 이렇게 표시됩니다.

중요 사항

  • 소셜 미디어 링크는 13세가 넘는 계정에서만 볼 수 있습니다. 13세 미만인 계정에서는 볼 수 없습니다.
  • 소셜 미디어 링크 기능 내에서 Discord 링크가 허용되지만 사용자는 Discord 링크를 다른 곳에서 토론하거나 공유할 수 없으며 다른 곳에서 언급할 경우 컴토를 받을 수 있습니다.
