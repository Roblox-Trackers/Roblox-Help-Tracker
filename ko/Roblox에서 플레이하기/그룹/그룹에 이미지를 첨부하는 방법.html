그룹에 이미지를 첨부하는 방법

그룹 소유자는 다음 단계를 수행하여 이미지 (엠블럼이라고도 함)를 그룹에 첨부할 수 있습니다:

  1. 화면 왼쪽에 있는 탐색 모음에서 그룹 을 클릭합니다
  2. 해당 그룹을 찾아서 클릭합니다
  3. 페이지 오른쪽 상단에 있는 그룹 관리자 버튼을 클릭합니다
  4. 왼쪽에서 그룹 정보 클릭합니다
  5. 파일 선택 을 클릭하고 사용할 이미지를 찾아 선택합니다
  6. 이미지를 선택한 후 열기 를 클릭합니다
  7. 그런 다음 녹색 저장 버튼을 클릭합니다

이미지 파일 제한 사항

  • Roblox 커뮤니티 기준 을 충족해야 합니다.
  • 파일 형식은 jpg 또는 png여야 합니다.
  • 최대 크기의 이미지 크기는 256 x 256입니다. 더 크게 업로드할 수 있지만 이미지의 크기가 256 x 256으로 조정됩니다.
  • 이미지가 정사각형이 아닌 경우 사각형이 되도록 패딩됩니다
