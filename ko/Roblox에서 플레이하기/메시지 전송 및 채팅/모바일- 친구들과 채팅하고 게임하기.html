모바일: 친구들과 채팅하고 게임하기

모바일 장치에서 친구와 채팅하고 채팅에서 직접 친구들과 체험에 쉽게 참여할 수 있습니다.

모바일에서 친구와 채팅을 어떻게 하나요?

모바일에서 친구와 채팅하는 것은 안드로이드 또는 iOS 기기와 유사하지만, 몇 가지 시각적 차이가 있을 수 있습니다.

모바일에서 친구와 채팅하려면 다음 지침을 따르세요.

  1. Roblox 앱을 엽니다
  2. 화면 아래쪽의 탐색 모음에서 채팅 아이콘을 선택합니다:
  3. 이 화면에는 최근 대화한 대화 내용과 함께 대화할 수 있는 친구의 기록이 표시됩니다. 친구 중 누군가 체험에서 게임을 하고 있는데 개인정보 설정이 허용하면 해당 체험을 볼 수 있습니다.
  4. 채팅할 친구 또는 그룹을 선택하면 채팅 창이 열립니다

모바일에서 새 채팅 그룹을 시작하려면 어떻게 해야 하나요?

  1. 화면 하단의 탐색 모음에서 채팅 아이콘을 선택합니다
  2. 종이와 연필 아이콘을 선택합니다
  3. 사용자 이름을 검색하거나 친구 목록을 스크롤하여 최대한 5명을 선택합니다
  4. 채팅을 시작하도록 오른쪽 상단의 채팅 을 클릭합니다
  5. 새로운 채팅 그룹 페이지가 불러옵니다

참고: 친구가 채팅 그룹에 사용자를 추가하면 자동으로 해당 그룹에 추가됩니다. 채팅 그룹에 추가할 때 새 채팅에서 읽지 않은 메시지가 있음을 나타내는 읽지 않은 메시지 알림이 표시될 수 있습니다. 채팅에 참여하고 싶지 않다면 언제든지 채팅 그룹에서 나갈 수 있습니다. 만약 누군가가 계속해서 본인이 있고 싶지 않은 채팅 그룹에 본인을 추가하는 경우, 그들을 차단하는 것이 좋습니다. 자세한 내용은, 규칙 위반 신고하는 방법 및 다른 사용자 차단하는 방법 도움말을 참고하세요.

채팅 그룹의 이름을 어떻게 변경하나요?

채팅 그룹의 이름을 변경하려면 다음 단계를 따르세요.

  1. 이름을 변경하고 싶은 채팅 그룹으로 이동합니다
  2. 화면의 오른쪽 상단에 있는 정보 아이콘을 누릅니다
  3. 설정 안 됨 을 클릭합니다
  4. 채팅 그룹의 새로운 이름을 입력합니다
  5. 완료하면 저장 을 누릅니다

내가 만든 채팅 그룹에서 누군가를 신고하거나 삭제하려면 어떻게 하나요?

  1. 설정 변경하고 싶은 채팅 그룹으로 이동합니다
  2. 화면의 오른쪽 상단에 있는 정보 아이콘을 누릅니다
  3. 신고하고자 하는 사용자 혹은 채팅 그룹에서 삭제하려고 하는 사용자 옆에 있는 세 개의 점으로 표시되는 더보기 버튼을 누릅니다
  4. 사용자 신고 혹은 그룹에서 삭제 를 선택합니다

참고: 해당 사용자가 모욕하거나 부적절한 말투를 사용하거나 어떠한 방법으로든 Roblox 이용 약관 또는 규칙을 위반한 경우 우선 사용자 신고 를 선택하고 그 다음에 그룹에서 제거 를 선택합니다.

이전에 만든 채팅 그룹에 친구를 어떻게 추가하나요?

  1. 설정 변경하고 싶은 채팅 그룹으로 이동합니다
  2. 화면의 오른쪽 상단에 있는 정보 아이콘을 누릅니다
  3. 친구 추가 를 선택합니다
  4. 채팅 그룹에 추가하고자 하는 사용자의 이름을 검색하거나 친구 목록을 스크롤하여 선택합니다
  5. 추가하고자 하는 사용자를 선택한 후 오른쪽 상단의 추가 버튼을 누릅니다

모바일에서 채팅 그룹을 어떻게 탈퇴하나요?

  1. 탈퇴할 채팅 그룹으로 이동합니다
  2. 화면의 오른쪽 상단에 있는 정보 아이콘을 누릅니다
  3. 그룹 나가기 옵션을 선택하고 확인합니다
