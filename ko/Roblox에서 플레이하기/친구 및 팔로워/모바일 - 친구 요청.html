모바일 : 친구 요청

Roblox 모바일 앱에서도 친구 요청을 수락하고 보내는 기능을 사용할 수 있습니다.

친구 요청을 수락하거나 거부하기 :

휴대 전화:

  1. 화면의 맨 밑에 있는 추가 ( More) 버튼을 누릅니다.
  2. 친구 (Friends) 를 누릅니다 (현재 요청 수가 오른쪽에 표시됩니다).
  3. 여기에서 요청 (Requests) 를 누릅니다.
  4. 수락 (Accept) 또는 거부(Decline) 을 선택합니다.

태블릿:

  1. 화면의 맨 밑에 있는 친구 ( Friends) 버튼을 누릅니다.
  2. 요청 탭 (Request Tab) 을 누릅니다
  3. 해당하는 요청에 수락 ( Accept) 또는 거부(Decline) 을 선택합니다

친구 요청을 보내기 :

휴대 전화:

  1. 화면의 맨 밑에 있는 추가 ( More) 버튼을 누릅니다.
  2. 친구 (Friends) 를 누릅니다.
  3. 맨 위에 있는 검색 바 아이콘 을 누릅니다.
  4. 친구의 사용자 이름을 입력하고 검색합니다.
  5. 사용자 이름 오른쪽에 있는 추가 버튼을 클릭합니다.

참고: 친구 요청을 이미 보냈으면 다시 보낼 수는 없으며 해당 프로필에 "Friend Request Pending"(친구 요청 대기 중)으로 표시됩니다.

태블릿:

  1. 화면의 맨 밑에 있는 홈 (Home) 버튼을 누릅니다.
  2. 화면의 맨 위에 있는 아이콘을 누릅니다.
  3. 친구 요청을 보내고 싶은 플레이어의 이름을 입력합니다.
  4. 리스트에서 친구의 이름을 누릅니다.
  5. 친구 요청 보내기(Send Friend Request) 버튼을 누릅니다.

참고: 친구 요청을 이미 보냈으면 다시 보낼 수는 없으며 해당 프로필에 "대기 중" (Pending) 또는 "로딩"(Loading) 아이콘이 표시됩니다.
