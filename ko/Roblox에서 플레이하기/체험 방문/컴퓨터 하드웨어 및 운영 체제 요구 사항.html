컴퓨터 하드웨어 및 운영 체제 요구 사항

Roblox의 최소 시스템 요구 사항은 아래와 같습니다. 체험 참가 시 오류가 발생할 경우 컴퓨터를 업그레이드해야 할 수 있습니다. 모바일 기기의 시스템 요구 사항에 대한 정보는 이 도움말 에서 확인하세요.

운영체제 요구 사항

  • PC 및 Windows : Roblox는 64비트의 Windows 7, Windows 8.1 또는 Windows 10에 설치가 가능합니다. Windows 8.1의 경우 메트로 모드(바둑판식 시작 화면)는 현재 지원되지 않으므로 데스크톱 모드에서 Roblox를 실행해야 합니다.
    참고: PC 및 Windows의 경우 Roblox 웹사이트는 Chrome , Firefox , Microsoft Edge 브라우저를 지원합니다.
  • Mac: Roblox 앱 & Roblox Studio는 10.13(High Sierra) 및 그 이상이어야 설치가 가능합니다.
    참고: Mac의 경우 Roblox 웹사이트는 Chrome , Firefox , Safari . 브라우저를 지원합니다.
  • Linux : Roblox는 Linux에서 지원되지 않습니다.
  • 모바일 : 모바일에서의 Roblox 시스템 요구 사항을 확인하려면 여기를 클릭 하세요.
  • Chrome OS : Chrome OS 버전 53 이상이 필요하며, 먼저 기기에서 Google Play 스토어 를 활성화해야 합니다.

참고: Roblox 비디오 녹화 기능은 Windows 7, Windows 8.1, Windows 10에서만 작동합니다. 새로 출시된 운영 체제 또는 시험판 버전의 운영 체제에서는 즉시 지원되지 않을 수 있습니다.

시스템 하드웨어 요구 사항

  • 그래픽 카드 : PC 및 Windows에서 Roblox 애플리케이션은 DirectX 10 이상의 기능 수준 지원이 필요합니다. 최고의 성능을 위해서는 전용 비디오 카드를 사용하는 5년 미만의 컴퓨터 또는 비디오 카드가 내장된 3년 미만의 노트북을 사용하는 것이 좋습니다.
  • 프로세서 : 클럭 속도가 1.6GHz 이상인 최신 프로세서(2005+)를 사용하는 것이 좋습니다. 구형 AMD 프로세서에서는 문제가 발생할 수 있습니다.
  • RAM 또는 메모리 : Windows 7, Windows 8.1, Windows 10 컴퓨터에서 최소 1GB의 메모리를 사용하는 것이 좋습니다.
  • 저장 공간 : Roblox 설치를 위해 최소 20Mb의 시스템 저장 공간을 확보해 두는 것이 좋습니다.
  • 모바일 : 모바일에서의 Roblox 시스템 요구 사항을 확인하려면 여기를 클릭 하세요.

인터넷 요구 사항

1인 가구의 경우 최소 4~8Mb/s의 인터넷 연결이 필요합니다. 2인 이상의 가구에서는 대역폭을 늘리는 것이 좋습니다.
