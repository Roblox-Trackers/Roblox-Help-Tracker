Roblox VR

가상 현실로 Roblox 체험을 향상시킬 수 있습니다! Roblox는 Oculus Rift 및 HTC Vive을 지원합니다.

다른 기종은 (Pico, Reverb 등) SteamVR에서 비공식적으로 지원합니다.

VR 사용 방법

Roblox로 VR 체험을 활성화하려면 먼저 Roblox, ( http://www.roblox.com/download )를 설치했는지 확인한 다음 아래의 단계를 따라야합니다.

기기가 PC에 제대로 연결되어 있고 체험 내에서 VR 모드 가 " 켜짐 "으로 전환되어 있는지 확인하세요. (메뉴>세팅>VR>켜기). 자세한 내용은 아래의 VR 모드를 켰다 껐다 하는 방법 을 참고하세요.

Oculus VR 또는 Steam VR의 경우 기기를 위한 해당 VR앱을 설치한 다음 구성합니다.

HTC Vive

지원되는 기기 : HTC Vive, Valve Index 등등

  1. 컴퓨터에서 SteamVR 앱 을 엽니다.

  2. SteamVR에서 OpenXR 가 표준이 되도록 설정합니다.

  3. 컨트롤러 켜기: (컨트롤러를 켜려면 신호음이 들릴 때까지 시스템 버튼을 누르세요.)
    (컨트롤러를 끄려면 신호음이 들릴 때까지 시스템 버튼을 길게 누릅니다.)

  4. SteamVR 앱 의 상태 아이콘과 하드웨어의 상태 표시 등이 모두 녹색이여야 합니다.

    System Dashboard 를 작동시키기 위해서 시스템 대시보드에서 컨트롤러의 시스템 버튼을 누릅니다.

    참고: SteamVR 앱 을 종료하면 컨트롤러가 자동으로 꺼집니다. 또한 컨트롤러는 일정 시간 동안 유휴 상태가 되면 자동으로 꺼집니다.

  5. 플레이어 리스트 (Player List) - 체험 서버의 플레이어 리스트를 봅니다

  6. 이제 헤드셋을 착용하고 플레이해보세요!

Oculus._1920x1080_v01.jpg

지원되는 기기 : Oculus Quest2 지원되는 기기 : Oculus Quest2 링크, Meta Quest 2, Meta Quest Pro 및 Oculus Rift

  1. 컴퓨터에서 Oculus 앱 을 열고 오른쪽 상단의 톱니바퀴 아이콘을 선택합니다.
    • 참고 : 컴퓨터에 최소 NVidia GTX1060 그래픽 카드 또는 이에 상응하는 장치가 장착되어 있지 않으면 Oculus 드라이버가 로드되지 않습니다.
  2. 기기가 잘 연결되어 있고 사용 가능한 상태로 있는지 확인합니다.
  3. Oculus App의 설정 / 일반 에서 OpenXR Runtime 을 선택합니다.
  4. Roblox 체험에서 Play 를 선택합니다: https://www.roblox.com/discover/
  5. 헤드셋이 링크 박스를 통해 컴퓨터에 연결되어 있는지 확인하세요.

링크선이 있는 Quest2를 사용하는 경우 헤드셋 안에서 링크선을 활성화시켜야 합니다.

기기가 잘 연결되어 있는지 확인하려면 다음을 수행합니다:

  • 기기 탭을 불러오고 헤드셋을 찾습니다.
  • 상태가 녹색으로 되어 있고 오류가 없는 것으로 표시되는지 확인합니다.

Meta Quest 2 및 Meta Quest Pro를 사용하는 경우 다음을 수행합니다:

  1. 알림 (Notifications) - Roblox 알림을 봅니다
  2. Make sure your Meta Quest 기기가 최신의 버전으로 되어 있는지 확인합니다. (Roblox는 Meta Quest 소프트웨어의 v55 버전이 필요합니다.)
  3. Quest 기기에서 Roblox를 실행합니다.
  4. 컴퓨터 또는 모바일에서 빠른 로그인 옵션을 통해서 사용하고 있는 Roblox 계정에* 로그인합니다.

*참고: 현재로서는 Meta Quest 기기에서 회원가입하거나 비밀번호로 로그인할 수 없습니다.

VR 모드를 켰다 껐다 하는 방법

체에서 VR 모드 를 켜거나 끄거나 VR 헤드셋과 컴퓨터 모니터 간에 디스플레이를 전환할 수 있습니다.

참고: 이 설정이 적용되도록 게임 엔진을 재부팅해야 합니다. 사용자 설정에서 VR 모드를 비활성화하면 VR에서 엔진이 작동할 수 없게 합니다.

VR 모드 가 꺼져 있을 때 다음을 수행하여 이 모드를 켜짐 으로 설정할 수 있습니다:

  1. 시스템 메뉴 옵션을 선택합니다
  2. VR 옵션을 켜 도록 설정합니다
  3. VR 옵션을 끄도록 설정합니다

VR 모드 가 켜져 있을 때 다음을 수행하여 이 모드를 꺼짐 으로 설정할 수 있습니다:

  1. System Menu 나타나도록 ESC 키를 누르거나 화면의 왼쪽 상단 모서리에 있는 햄버거 메뉴 버튼을 클릭합니다
  2. 기어 설정 을 선택합니다.
