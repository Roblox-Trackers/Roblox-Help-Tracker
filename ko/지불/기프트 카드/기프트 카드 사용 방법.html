기프트 카드 사용 방법

기프트 카드는 웹 브라우저에서만 사용할 수 있으며, Roblox 앱에서는 사용할 수 없습니다.

Roblox 기프트 카드 에는 두 가지의 유형이 있습니다.

  • 크레딧 - Robux 또는 멤버십을 구매하기 위한 실제 통화
  • Robux - 가상 통화는 계정 잔액에 추가됩니다. - Amazon.com 에서만 사용 가능해요.

계정에 기프트 카드 사용하기

  1. 웹 브라우저에서 계정에 로그인합니다.
  2. 기프트 카드 사용 페이지 로 이동합니다.
  3. 박스에 있는 기프트 카드 코드를 입력합니다.
  4. 사용 을 클릭합니다.

  5. 계정에 크레딧이 성공적으로 추가되면 완료 메시지가 나타납니다.
    PIN 입력 시 숫자 0 혹은 1 입력이 불가능 하다면, 알파벳 대문자 O 또는 I로 입력해보세요.
  6. 크레딧 사용 을 클릭하여 크레딧을 사용합니다. 추가적인 도움이 필요하면 게임카드 크레딧 사용하기 도움말 을 확인하거나 혹은 닫기 를 클릭하여 메시지를 닫은 후 다른 게임카드를 사용해 보세요.

잔액을 확인하세요.

  • 크레딧 잔액 찾기
  • Robux 잔액 찾기

구매 시 크레딧 사용하기

  1. 브라우저에서 계정에 로그인합니다.
  2. 멤버십 페이지 또는 Robux 페이지 로 이동합니다.
  3. 녹색 버튼을 클릭하여 구매하려는 제품을 선택합니다.

  4. 결제 방법으로 Roblox 카드 사용 을 선택하고 계속 을 클릭합니다.
  5. PIN을 입력하고 사용 을 클릭합니다.

  6. Roblox 크레딧 잔액이 업데이트되면 주문 확인 을 클릭합니다.
  7. 결제 확인 페이지가 나타납니다.

기프트 카드를 사용하는데 문제가 있어요.

기프트 카드를 사용하는 데 문제가 있는 경우 아래에 있는 추가 정보를 제공하여 문의 해 주세요.

추가 정보:

  • 사용자 이름
  • 일련 번호(카드 오른쪽 하단에 있는 번호)
  • 기프트 카드 구매 날짜 및 금액
  • 구매 매장 및 도시
  • 구매한 모든 카드의 PIN 번호(성공적으로 사용한 카드 포함)
  • 구매 영수증의 이미지 또는 스캔
  • 기프트 카드 앞면과 뒷면의 이미지 혹은 스캔 또는 전자 기프트 카드의 전체 이미지
  • 사용하는 동안 수신된 오류 메시지의 이미지 (해당하는 경우)

참고: 제공된 이미지가 .jpg 또는 .png 형식인지 확인해주세요.
