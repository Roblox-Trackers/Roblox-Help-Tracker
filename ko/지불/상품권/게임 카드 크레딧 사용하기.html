게임 카드 크레딧 사용하기

게임 카드 크레딧은 웹 브라우저에서만 사용할 수 있으며, Roblox 앱에서는 사용할 수 없습니다

멤버십 또는 Robux 구매 페이지에서

  1. 브라우저에서 게임 카드를 사용했던 Roblox 계정에 로그인합니다.
  2. 멤버십 구매 페이지 또는 Robux 구매 페이지 로 이동합니다.
  3. 구매할 제품을 선택합니다.
  4. 지불 유형은 Roblox 크레딧 을 선택하고 계속 을 클릭합니다.

  5. 주문 제출 을 클릭합니다. 이미 게임 카드를 사용한 경우 게임 카드 PIN을 다시 입력하지 마세요 .
  6. 전체 구매 가격에 대해 충분한 크레딧이 있다면, 경우 확인 메시지가 표시됩니다.

사용하고 남은 크레딧을 Robux로 전환

Robux 또는 멤버십을 구매한 후 남은 크레딧이 $4.99 미만인 경우, 해당 크레딧을 Robux로 전환할 수 있습니다. 해당 옵션은 Robux 패키지를 구매할 때 크레딧이 부족한 경우에만 사용할 수 있습니다.

  1. 브라우저에서 게임 카드를 사용했던 Roblox 계정에 로그인합니다.
  2. 우측 상단에 있는 기어 아이콘을 누르고 설정 페이지로 이동하세요.
  3. 청구 탭을 클릭하세요.
  4. Robux 전환 버튼을 클릭하세요.
  5. 전환 확인창을 볼 수 있습니다. 적용 을 클릭하고 전환을 완료하세요.
  6. Robux의 잔액은 해당 페이지를 새로고침 또는 새 창을 열면 확인 가능합니다.
