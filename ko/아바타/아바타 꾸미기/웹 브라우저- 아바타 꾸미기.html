웹 브라우저: 아바타 꾸미기

내 아바타는 어디에서 편집할 수 있나요?

내비게이션 메뉴에서 아바타 를 누르면 편집 가능한 페이지로 이동할 수 있습니다.

내 아바타를 편집하려면 어떻게 하면 되나요?

아이템 추가하기

  1. 아바타 편집기 페이지로 이동한 다음, 사용 가능한 풀다운 메뉴를 클릭하고 카테고리(예: 모자, 헤어, 얼굴 등)를 선택합니다.
  2. 특정 파트에 대해 여러 아이템이 있을 경우, 원하는 아이템을 찾을 때까지 아래로 스크롤합니다.
  3. 적용할 아이템을 결정했다면 해당 아이콘을 탭합니다. 이때 아이템은 녹색으로 강조 표시됩니다.

아이템 제거하기

  1. 아바타 편집기 페이지에서 최근 을 누르거나 해당되는 특정 카테고리(예: 모자, 헤어, 얼굴 등)를 선택합니다.
  2. 최근 풀다운 메뉴에서 현재 착용한 아이템이 첫 번째 목록에 있어야 합니다. 특정 카테고리를 선택한 경우, 제거할 아이템을 찾을 때까지 아래로 스크롤합니다.
  3. 녹색으로 강조 표시된 아이템을 누릅니다.

참고: 아이템을 추가하거나 제거한 후에 아바타가 제대로 나타나지 않는다면, 아바타 사진 아래의 ‘다시 그리기’ 링크를 클릭하세요.

피부 색상 변경하기

아이템과 복장을 변경하는 것도 재미있는 일이지만, 때로는 아바타의 피부색을 다양하게 변경하고 싶을 때도 있죠. 웹사이트에서 아바타의 피부색을 변경하려면 다음 단계를 따르세요.

  1. 페이지 왼쪽에서 계정 메뉴를 열고 아바타 섹션으로 이동합니다.
  2. 커서를 신체 풀다운 메뉴 위로 올린 다음, 피부 색깔 을 선택합니다.
  3. 원하는 색상을 눌러 아바타의 피부색 전체를 변경합니다.
  4. 만약 특정 신체 부위에 다른 색상을 적용하고 싶으면 오른쪽 아래 모서리에 있는 고급 을 클릭합니다.
  5. 해당 화면에서 원하는 신체 부위를 누르고 원하는 색상을 선택합니다.
