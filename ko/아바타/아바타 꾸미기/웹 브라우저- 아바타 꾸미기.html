아바타는 어떻게 편집하나요?

아이템 추가하기

  1. 아바타 편집기 페이지로 이동한 다음, 풀다운 메뉴를 누르고 카테고리(예: 모자, 헤어, 얼굴 등)를 선택합니다.
  2. 특정 파트에 대해 여러 아이템이 있을 경우, 원하는 아이템을 찾을 때까지 아래로 스크롤합니다.
  3. 선택한 후에는 아이템 아이콘을 탭하여 강조 표시합니다.

주어진 항목의 소프트 제한(예: 모자 3개 이상) 이상을 추가하려면 오른쪽 하단에서 고급 을 탭하세요. 아이템 링크에 있는 애셋 ID를 입력해 최대 10개의 아이템을 추가할 수 있습니다. 아래 예를 참고하세요.

https://www.roblox.com/catalog/ 62234425 /Brown-Hair <- 링크에 포함되어 있는 숫자가 애셋 ID입니다.

아이템 제거하기

  1. 아바타 편집기 페이지에서 최근 을 누르거나 제거하고자 하는 아이템의 카테고리(예: 모자, 헤어, 얼굴 등)를 선택합니다.
  2. '최근'의 풀다운 메뉴를 누르면 현재 장착 중인 아이템이 맨 위에 나타납니다. 이외의 특정 카테고리를 선택한 경우, 제거할 아이템을 찾을 때까지 아래로 스크롤합니다.
  3. 강조 표시된 아이템 아이콘을 탭합니다.

참고: 아이템을 추가하거나 제거한 후에 아바타가 제대로 나타나지 않을 경우, 아바타 이미지 아래의 ‘다시 그리기’ 링크를 누르세요.

피부색 변경하기

다른 아이템이나 복장을 장착하는 것도 재미있는 일이지만, 때로는 아바타의 피부색을 다양하게 변경하고 싶을 때도 있죠. 웹사이트에서 아바타의 피부색을 변경하려면 다음 단계를 따르세요.

  1. 페이지 왼쪽에서 계정의 아바타 메뉴로 이동합니다.
  2. 이 페이지에서 머리 & 몸에 커서를 이동하고 드롭다운 메뉴에 스킨 톤 을 선택합니다.
  3. 원하는 색상을 눌러 아바타의 피부색 전체를 변경합니다.
  4. 만약 특정 신체 부위에 다른 색상을 적용하고 싶으면 오른쪽 아래 모서리에 있는 고급 을 누릅니다.
  5. 해당 화면에서 원하는 신체 부위를 누르고 원하는 색상을 선택합니다.
