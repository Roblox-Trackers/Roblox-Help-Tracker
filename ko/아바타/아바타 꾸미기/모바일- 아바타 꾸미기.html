모바일: 아바타 꾸미기

모바일:인벤토리에 보기 도움말 에서 인벤토리에 접근하는 방법을 확인하세요.

인벤토리 품목을 착용하거나 제거하는 방법

모바일

  1. 인벤토리에서 착용하거나 제거할 아이템을 선택하세요.
  2. 아이템을 착용하려면, 착용 슬라이드 바를 YES 로 이동하세요.
  3. 아이템을 제거하려면 착용 슬라이드 바를 NO 로 이동하세요.

태블릿

  1. 화면 하단의 추가 버튼을 누르세요.
  2. 아바타 를 선택하세요.
  3. 해당 페이지에서 웹 사이트를 통해 아바타를 꾸밀 수 있습니다. 도움이 필요하시면 여기를 클릭하세요.

아바타의 피부색을 바꾸는 방법

모바일

현재 휴대폰의 모바일 앱을 사용하여 스킨 색상을 변경할 수는 없지만, 웹 브라우저를 통해 피부색 변경은 가능합니다.

  1. 휴대폰의 웹 브라우저를 실행하세요.
  2. Roblox.com로 이동하세요.
  3. 로그인하지 않은 경우, 전체 사이트 보기 를 누르고 5단계 스텝을 건너뛰세요. 만약 로그인한 경우, 4단계 스텝을 확인하세요.
  4. 로그인한 경우, 좌측 상단 모서리에 있는 줄 세 개가 표시된 아이콘을 눌러 메뉴를 불러온 다음 아래로 스크롤 하여 전체 사이트 를 누르세요.
  5. 전체 사이트가 로드되면 좌측 상단의 세 줄로 표시된 아이콘을 눌러 메뉴를 열어주세요. (아직 열려 있지 않은 경우)
  6. 아바타 를 선택하세요.
  7. 해당 페이지에서 컴퓨터와 같이 피부색을 변경할 수 있습니다. 도움이 필요하시면 여기를 클릭하세요.

태블릿

  1. 화면 하단의 추가 버튼을 누르세요.
  2. 아바타 를 선택하세요.
  3. 해당 페이지에서 웹 사이트를 통해 피부색을 변경할 수 있습니다. 도움이 필요하시면 여기를 클릭하세요.
