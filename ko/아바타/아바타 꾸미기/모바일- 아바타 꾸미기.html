모바일: 아바타 꾸미기

모바일:인벤토리에 보기 도움말 에서 인벤토리에 접근하는 방법을 확인하세요.

인벤토리 품목을 착용하거나 제거하는 방법

모바일

  1. 인벤토리에서 착용하거나 제거할 아이템을 선택하세요.
  2. 아이템을 착용하려면, 착용 슬라이드 바를 YES 로 이동하세요.
  3. 아이템을 제거하려면 착용 슬라이드 바를 NO 로 이동하세요.

태블릿

  1. 화면 하단의 추가 버튼을 누르세요.
  2. 아바타 를 선택하세요.
  3. 해당 페이지에서 웹 사이트를 통해 아바타를 꾸밀 수 있습니다. 도움이 필요하시면 여기를 클릭하세요.

아바타의 피부색을 바꾸는 방법

모바일

  1. 모바일 애플리케이션에서 Roblox 계정에 로그인합니다..
  2. 아바타 탭으로 이동합니다.
  3. ' 꾸미기 '를 클릭하고 ' 머리 및 몸 '을 선택합니다.
  4. '피부' 탭에서 원하는 피부색을 선택할 수 있습니다.

태블릿

  1. 화면 하단의 추가 버튼을 누르세요.
  2. 아바타 를 선택하세요.
  3. 해당 페이지에서 웹 사이트를 통해 피부색을 변경할 수 있습니다. 도움이 필요하시면 여기를 클릭하세요.
