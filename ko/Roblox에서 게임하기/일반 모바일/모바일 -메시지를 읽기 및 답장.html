모바일 :메시지를 읽기 및 답장

Roblox 메시지는 Roblox 모바일 앱에서 사용할 수 있으며 화면 하단의 네비게이션 바를 통해 액세스 할 수 있습니다.

개인 메시지 읽기

  1. 스크린에 하단의 모서리에 있는 " ... " 표시 된 아이콘을 클릭합니다
  2. Messages (메세지) 를 선택합니다
  3. 메시지를 스크롤합니다. 일단 하단에는, 더 많은 자동으로 로드됩니다.
  4. 읽고 싶은 메시지를 선택합니다.

개인 메시지 답장

  1. 스크린에 하단의 모서리에 있는 " ... " 표시 된 아이콘을 클릭합니다
  2. Messages (메세지) 를 선택합니다
  3. 메시지를 스크롤합니다. 일단 하단에는, 더 많은 자동으로 로드됩니다.
  4. 답장을 보낼 메시지를 선택합니다.
  5. Reply (답장) 을 살짝 누르고 메시지를 입력 한 다음 Send (보내기) 를 클릭합니다.

참고: 개인 메시지를 신고 할 수 있습니다 ( Roblox 커뮤니티 규칙 참고). Report this message
