모바일 :메시지를 읽기 및 답장하기

Roblox 메시지는 Roblox 모바일 앱에서 사용할 수 있으며 화면 하단의 탐색 모음을 이용하여 사용할 수 있습니다.

개인 메시지 확인하기

  1. 스크린에 하단의 모서리에 있는 " ... " 표시 된 아이콘을 클릭합니다.
  2. 메세지 를 선택합니다.
  3. 메시지를 스크롤합니다. 메시지 맨 하단에 있으면, 자동으로 더 많은 메시지가 자동으로 생성됩니다.
  4. 읽고 싶은 메시지를 선택합니다.

개인 메시지 답장하기

  1. 스크린에 하단의 모서리에 있는 " ... " 표시 된 아이콘을 클릭합니다
  2. 메세지 를 선택합니다
  3. 메시지를 스크롤합니다. 메시지 맨 하단에 있으면, 자동으로 더 많은 메시지가 자동으로 생성됩니다.
  4. 답장을 보낼 메시지를 선택합니다.
  5. 답장 을 누르고 메시지를 입력 한 다음 보내기 를 클릭합니다.

참고: 개인 메시지를 신고 할 수 있습니다. 이 메시지를 신고 누르면, 검열팀에 메시지 내용을 신고할 수 있습니다.
