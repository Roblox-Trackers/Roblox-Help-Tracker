비공개/VIP 서버는 어떻게 구매하고 구성하나요?

게임 개발자가 VIP 서버를 활성화해 둔 경우, Robux를 사용하면 친구를 초대해 함께 즐길 수 있는 비공개 서버를 구매할 수 있습니다.

나의 서버는 어떻게 만들고 변경하나요?

나의 서버는 어떻게 만드나요?

  • 게임의 세부 정보 페이지를 볼 때 서버 탭을 클릭해보세요.
  • 이 기능이 켜져 있으면 비공개 서버 섹션이 나타납니다. 이곳에서 생성한 모든 비공개 서버 혹은 참석할 수 있는 비공개 서버를 찾을 수 있습니다. 어떤 비공개 서버라도 보이면, 바로 게임을 즐길 수 있어요.
  • 직접 비공개 서버를 만들고 싶은 경우, 비공개 서버 만들기 버튼을 클릭하세요.
  • 새 서버에 원하는 이름을 입력하고 가입 계약을 읽습니다. 상자에 있는 모든 정보를 읽고 완벽한 이름도 정했다면, 지금 구매하기 를 클릭하여 구매를 완료하세요.
  • 구매가 완료되면 언제든지 서버를 구성할 수 있습니다.

나의 서버의 구성 방법은 어떻게 되나요?

  • 서버 탭에서 해당 서버의 구매 알림 창에서 혹은 메뉴 버튼(점 세 개)에서 구성 을 선택하면 구성 페이지로 이동할 수 있습니다.
  • 최대 50 명의 개별 사용자를 초대하거나, 허용된 친구 박스를 선택하여 친구 목록의 모든 사용자가 참가하도록 허용할 수 있습니다.그리고 서버의 링크도 생성할 수 있습니다.
  • 또한 서버에 대한 액세스를 켜고 끌 수 있을 뿐만 아니라 서버에 대한 구독을 취소할 수도 있습니다.

비공개 서버 가입 절차와 비용은 어떻게 되나요?

비공개 서버의 가입 비용은 어떻게 되나요?

  • 비공개 서버 가격은 게임 개발자가 정합니다.
  • 처음 구매할 때 비용이 청구되며, 이후에는 서버가 유지되도록 매달 갱신됩니다. 이때 비용은 Robux 잔액에서 자동으로 빠져나가기 때문에 매번 결제일을 기억하지 않아도 되어서 편리합니다.
  • 서버 갱신을 원하지 않으면, 갱신 날짜 이전에 언제든지 취소할 수 있습니다. VIP 서버를 취소하려면 아래에 있는‘ VIP 서버를 취소하려면 어떻게 해야 하나요? ’ 섹션을 확인하세요.

VIP 서버의 취소 방법은 어떻게 되나요?

  • 해당 체험의 비공개 서버로 이동합니다. 섹션 맨 오른쪽의 메뉴 버튼(점 세 개)을 클릭합니다.
  • 구성 을 클릭합니다.
  • 월별 청구를 취소하려면 가입 상태 옆의 녹색 토글 버튼 을 클릭합니다.
  • 가입을 취소하려면 확인 창에서 결제 취소 를 클릭합니다.

참고: 이미 결제한 달의 남은 기간 동안은 서버를 계속 사용할 수 있지만, 해당 기간이 지나면 만료됩니다.

갱신 날짜에 Robux가 부족하면 어떻게 되나요?

갱신 날짜에 Robux가 부족하다면 서버가 만료되고 더 이상 사용할 수 없지만, 재가입은 언제든지 가능합니다.

서버를 비활성화하려면 어떻게 해야 하나요?

  • 해당 체험의 비공개 서버로 이동합니다. 섹션 맨 오른쪽의 메뉴 버튼(점 세 개)을 클릭합니다.
  • 구성 을 클릭합니다.
  • 일정 기간 동안 서버를 일시적으로 비활성화하려면 VIP 서버 상태 옆의 녹색 활성 아이콘을 클릭합니다.
  • 서버를 비활성화하려면 확인 창에서 비활성화 를 클릭합니다.
  • 해당 버튼을 다시 클릭하면 언제든지 다시 활성화할 수 있습니다. 하지만 비활성화 기간 동안 가입이 연장되는 것은 아니므로 , 원래 예정된 날짜에 계속 갱신되거나 만료된다는 것을 알아 두세요.
