게임 내 채팅 방법

다른 플레이어와 통신할 수 있는 많은 좋은 방법 중 하나는 게임 내 채팅 기능을 사용하는 것입니다.

대화를 시작하려면 키보드의 / 키를 누르십시오. 그러면 채팅 창이 열리고, 개발자가 사용할 수 있는 채팅 창과 원하는 내용을 입력할 수 있는 채팅 표시줄이 열립니다. 해당 표시줄에 텍스트를 입력한 다음 Enter 을 누릅니다.

화면 왼쪽 위에 있는 채팅 아이콘을 클릭하여 채팅 창/바도 끌 수 있습니다:

참고: 채팅은 계정의 개인 정보 모드/필터가 허용하는 것으로 제한됩니다. 자세한 내용은 여기를 클릭 하시기 바랍니다.

게임 내 채팅 비활성화 방법

게임 내 채팅 기능을 비활성화하려면 다음을 수행합니다:

  1. 계정에 로그인
  2. 페이지 오른쪽 위에 있는 기어 아이콘을 클릭한 다음 설정을 클릭하여 계정 설정 페이지로 이동합니다. 여기를 클릭 해도 됩니다
  3. 개인 정보 탭을 클릭하고 "게임할 때 누구와 채팅할 수 있습니까?"아래에서 없음 으로 선택하시기 바랍니다
  4. 화면 아래로 스크롤하고 저장 을 클릭합니다
