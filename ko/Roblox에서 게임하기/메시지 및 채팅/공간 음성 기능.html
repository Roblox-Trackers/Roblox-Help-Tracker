공간 음성 기능

공간 음성 기능이 무엇인가요?

공간 음성 베타 기능은 사 람들이 Roblox에서도 실제 세상에서 이야기하는 것처럼 자연스럽게 소통할 수 있게 해 주는 새로운 기능입니다.

현재 개발자들은 원할 경우 공간 음성 기능이 각자의 체험에서 작동하도록 선택할 수 있습니다.

나이 인증을 완료한 13세 이상의 사용자는 설정 페이지에서 이 기능을 사용할지 여부를 선택할 수 있습니다. 공간 음성 채팅을 지원하는 Roblox 체험에서 이 기능을 사용해 보실 수 있습니다.

공간 음성 기능을 활성화하려면 어떻게 하면 되나요?

공간 음성 기능을 활성화하려면 아래 단계를 따르세요.

  • 계정 정보 로 이동합니다.
  • 나이 인증 을 완료합니다.
  • 개인정보 를 선택합니다.
  • 공간 음성 선택 옵션을 켜기로 전환합니다.
    • 선택란이 회색에서 초록색으로 변하면 계정에서 음성 채팅 기능이 활성화된 것입니다.

Roblox 및 Roblox 지원 센터에서 각 사용자별로 기능을 추가해 드릴 수는 없습니다.

다만, 모든 Roblox 체험에서와 마찬가지로 이용 약관 과 커뮤니티 규정 을 준수해야 합니다. 규정을 위반할 경우 공간 음성 기능 이용 정지 또는 기능 삭제 처분을 받을 수 있으며, 해당 계정은 추가적인 검열 조치를 받게 될 수 있습니다. Roblox는 재량에 따라 각 회원의 공간 음성 기능을 차단할 수 있는 권리를 보유합니다.

누가 공간 음성 기능을 사용 중인지 어떻게 알 수 있나요?

나이 인증을 완료한 13세 이상의 사용자의 경우 사용자 이름 옆에 마이크 아이콘이 나타나며, 이를 통해 활성화 여부를 확인할 수 있습니다. 다른 사용자의 경우 아바타 머리 위에서 마찬가지로 아이콘을 확인할 수 있습니다. 이외에는 사용 여부를 확인할 수 있는 방법이 없습니다.

공간 음성 기능에 대한 부적절한 사용 사례를 신고하거나, 특정 사용자를 차단하거나 음소거하려면 어떻게 하면 되나요?

다른 사용자 음소거하기

아바타 머리 위에 위치한 마이크 모양 아이콘을 누르면 다른 사용자를 음소거할 수 있습니다. 음소거는 문자 채팅이 아닌 음성 채팅에만 적용되며, 해당 체험에 머무르는 동안에만 지속됩니다.

다른 사용자 차단하기

다른 사용자와의 음성 채팅 또는 문자 채팅을 원하지않을 경우 공간 음성 체험에서 해당 사용자를 차단할 수 있습니다. 좀 더 자세한 정보가 필요할 경우 다른 사용자를 차단하는 방법 을 참고하세요.

부적절한 사용 사례 신고하기

이용 약관 또는 커뮤니티 가이드라인 위반 사례를 발견할 경우, 신고 기능을 이용해 검열팀에 신고해 주시기 바랍니다. 좀 더 자세한 정보가 필요하다면 규칙 위반을 신고하는 방법 을 참고하세요.

Roblox 안전 기능에 대해 더 자세히 알고 싶다면 Roblox 안전 을 읽어 보세요.

공간 음성 기능을 끄려면 어떻게 하면 되나요?

공간 음성 기능을 비활성화하려면 아래 단계를 따르세요.

  • 계정 정보 로 이동합니다.
  • 개인정보 를 선택합니다.
  • 공간 음성 선택 옵션을 끄기 로 전환합니다.
    • 선택란이 회색에서 초록색으로 변하면 계정에서 음성 채팅 기능이 비활성화된 것입니다.
    • 공간 음성 기능 사용이 가능한 경우 언제든지 기능을 켜서 음성 채팅을 다시 활성화할 수 있습니다.
