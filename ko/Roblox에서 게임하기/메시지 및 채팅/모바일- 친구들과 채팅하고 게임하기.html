모바일: 친구들과 채팅하고 게임하기

모바일 장치에서 친구와 채팅하고 채팅에서 직접 친구들과 게임에 쉽게 참여할 수 있습니다.

모바일에서 친구와 채팅을 어떻게 합니까?

모바일에서 친구와 채팅하는 것은 안드로이드 또는 iOS 기기와 유사하지만, 몇 가지 시각적 차이가 있을 수 있습니다. 자세한 내용은, 저희의 안드로이드 앱: 기본정보 및 iOS 앱: 기본정보 도움말을 참고하시기 바랍니다.

모바일에서 친구와 채팅하려면 다음 지침을 따르시기 바랍니다:

    • Roblox 앱을 엽니다
    • 화면 아래쪽의 탐색 모음에서 채팅 아이콘을 선택합니다:
    • 이 화면에는 최근 대화한 대화 내용과 함께 대화할 수 있는 친구의 기록이 표시됩니다. 친구 중 누군가 게임을 하고 있는데 개인 정보 설정이 허용하면 해당 게임을 볼 수 있습니다.
    • 채팅할 친구 또는 그룹을 선택하면 채팅 창이 열립니다

모바일에서 새 채팅 그룹을 시작하려면 어떻게 해야 합니까?

    • 사용자 그룹의 아이콘과 이와 같은 더하기 기호를 선택합니다
    • 새 채팅 그룹 페이지가 로드됩니다
    • 채팅 그룹의 이름 입력합니다
    • 친구 목록을 스크롤하거나 사용자 이름을 검색하여 그룹에 참여할 최대 5명 친구를 선택합니다

참고: 친구가 채팅 그룹에 사용자를 추가하면 자동으로 해당 그룹에 추가됩니다. 대화 그룹에 추가할 때 새 대화에서 읽지 않은 메시지가 있음을 나타내는 읽지 않은 메시지 알림이 표시될 수 있습니다. 대화에 참여하고 싶지 않다면 언제든지 대화를 중단할 수 있습니다. 만약 누군가가 계속해서 고객님이 있고 싶지 않은 채팅 그룹에 고객님을 추가하는 경우, 그들을 차단하는 것이 좋습니다. 자세한 내용은, 규칙 위반 신고하는 방법 및 다른 사용자 차단하는 방법 도움말을 참고하시기 바랍니다.

채팅 그룹의 이름을 어떻게 변경합니까?

채팅 그룹의 이름을 변경하려면 다음을 수행합니다:

    • 편집할 채팅 그룹으로 이동합니다
    • 화면 오른쪽 상단에서 다음과 같은 또는 아이콘을 선택합니다
    • 채팅 그룹 이름 선택합니다

    • 채팅 그룹의 새 이름 입력합니다
    • 완료되면 저장 옵션을 선택합니다

내가 작성한 채팅 그룹에서 누군가를 신고하거나 삭제하려면 어떻게 합니까?

    • 편집할 채팅 그룹으로 이동합니다
    • 화면 오른쪽 상단에서 다음과 같은 또는 아이콘을 선택합니다
    • 보고하거나 대화 그룹에서 제거할 사용자 옆에 있는 점 세 개처럼 보이는 단추 선택합니다
    • Report User (사옹자 신고) 또는 Remove from Group (그룹에서 제거하기) 선택을 합니다

Note: If the user said anything rude or inappropriate, or in any way violated Roblox's Terms of Use or Rules of Conduct, please make sure to select 사용자 신고 and then select 그룹에서 제거 afterward.

이전에 만든 채팅 그룹에 친구를 추가하려면 어떻게 해야 합니까?

    • 편집할 채팅 그룹으로 이동합니다
    • 화면 오른쪽 상단에서 다음과 같은 또는 아이콘을 선택합니다
    • 친구 추가 를 선택합니다
    • 친구 목록을 스크롤하거나 사용자 이름을 검색하여 그룹에 참여할 사용자를 선택합니다
    • 추가 할 사용자를 선택하면 추가 를 클릭합니다

모바일에서 그룹을 어떻게 탈퇴합니까?

    • 탈퇴할 채팅 그룹으로 이동합니다
    • 화면 오른쪽 상단에서 다음과 같은 또는 아이콘을 선택합니다
    • 그룹 탈퇴하기 옵션을 선택합니다
      iOS


      안드로이드

모바일에서 친구들과 어떻게 게임하나요?

    • 게임을하고 싶은 대화 상대 나 채팅 그룹으로 이동합니다
    • 친구가 하고 있는 게임 목록을 보고 선택합니다.
    • 다른 사람들이 고객님을 따라 게임을하고 싶으면 게임에 참여할 수 있습니다. 게임에 참여하면 채팅 그룹에 게임의 참여 버튼이 표시됩니다. 이렇게하면 친구들이 게임에 쉽게 참여할 수 있습니다. 게임을 채팅 그룹에 고정시킬 수도 있습니다.
