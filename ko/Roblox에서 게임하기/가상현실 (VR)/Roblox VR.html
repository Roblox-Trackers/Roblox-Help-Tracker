Roblox VR

가상 현실로 Roblox 체험을 향상시킬 수 있습니다! Roblox는 Oculus Rift 및 HTC Vive을 지원합니다.

VR 사용 방법

Roblox로 VR 체험을 활성화하려면 먼저 Roblox, ( http://www.roblox.com/download )를 설치했는지 확인한 다음 아래의 단계를 따라야합니다.

기기가 PC에 제대로 연결되어 있고 체험 내에서 VR 모드 가 " 켜짐 "으로 전환되어 있는지 확인하세요. (메뉴>세팅>VR>켜기). 자세한 내용은 아래의 VR 모드 전환 방법 을 참고하세요.

HTC Vive

  1. 컴퓨터에서 SteamVR 앱 을 엽니다.
  2. 베이스 스테이션에 전원이 공급되는지 확인하세요.
  3. 헤드셋이 링크 박스를 통해 컴퓨터에 연결되어 있는지 확인하세요.
  4. 컨트롤러 켜기:
    ( 컨트롤러를 켜려면 신호음이 들릴 때까지 시스템 버튼을 누르세요)
    (컨트롤러를 끄려면 신호음이 들릴 때까지 시스템 버튼을 길게 누릅니다.)
  5. SteamVR app 의 상태 아이콘과 하드웨어의 상태 표시 등이 모두 녹색이여야 합니다.

    작동시키기 위해서 시스템 대시보드 에서 컨트롤러의 시스템 버튼을 누릅니다.

    참고: SteamVR 앱 을 종료하면 컨트롤러 가 자동으로 꺼집니다. 또한 컨트롤러 는 일정 시간 동안 유휴 상태가 되면 자동으로 꺼집니다.

오큘러스 리프트:

  1. 컴퓨터에서 Oculus 앱 을 열고 오른쪽 상단의 톱니바퀴 아이콘을 선택합니다.
  2. 기어 설정 을 선택합니다.
  3. 왼쪽 메뉴에서 일반 을 선택합니다.
  4. 알 수 없는 출처 설정을 전환하여 X 대신 체크 로 표시되도록 합니다.
  5. 이 설정을 활성화하려면 선택을 확인하세요.
  6. Roblox 체험에서 Play 를 선택합니다: https://www.roblox.com/discover/
  7. 모니터에 VR 모드를 나타내는 정사각형 화면이 표시됩니다(예: 아래 이미지).
  8. 이제 헤드셋을 착용하고 플레이해보세요!

VR 원형 메뉴판(VR Radial Menu) 사용 방법

모든 중요한 시스템 설정을 VR 원형 메뉴판(VR Radial Menu) 으로 액세스할 수 있습니다.

  1. VR 원형 메뉴판이 나타나도록 Menu 버튼을 누릅니다
  2. 다음 옵션들 중 하나를 선택합니다 :
    • 시스템 메뉴 (System Menu) - 시스템의 옵션에 액세스합니다
    • 플레이어 리스트 (Player List) - 체험 서버의 플레이어 리스트를 봅니다
    • 카메라 재보정 (Recenter Camera) - 아바타에 카메라를 재보정합니다
    • 알림 (Notifications) - Roblox 알림을 봅니다
    • 게임 종료 (Exit Game) - 체험을 종료합니다
    • 배낭 (Backpack) - 체험 내의 인벤토리를 봅니다
    • 2D UI 토글 (2D UI Toggle) - 2D UI 기능을 토글 키로 켰다 껐다 합니다.

위에서 설명한 VR 원형 메뉴판 옵션은 아래에 있는 이미지가 표시된 듯이 맨 위에 있는 "시스템 메뉴" 부터 시작하여 반 시계 방향으로 버튼들에 해당됩니다.

VR 모드를 켰다 껐다 하는 방법

체에서 VR 모드 를 켜거나 끄거나 VR 헤드셋과 컴퓨터 모니터 간에 디스플레이를 전환할 수 있습니다.

VR 모드 가 켜져 있을 때 다음을 수행하여 이 모드를 꺼짐 으로 설정할 수 있습니다:

  1. VR 원형 메뉴판이 나오도록 Menu 버튼을 누릅니다
  2. VR 옵션을 켜 도록 설정합니다
  3. VR 옵션을 끄도록 설정합니다

VR 모드 가 꺼져 있을 때 다음을 수행하여 이 모드를 켜짐 으로 설정할 수 있습니다:

  1. System Menu 나타나도록 ESC 키를 누르거나 화면의 왼쪽 상단 모서리에 있는 햄버거 메뉴 버튼을 클릭합니다
  2. 시스템 메뉴 옵션을 선택합니다
