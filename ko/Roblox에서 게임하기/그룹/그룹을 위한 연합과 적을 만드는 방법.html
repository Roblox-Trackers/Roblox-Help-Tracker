그룹을 위한 연합과 적을 만드는 방법

이제 그룹을 생성했으므로 다른 그룹과 동맹 또는 적이 되고 싶을 수 있습니다. 그룹 소유자가 아닌 경우 다른 그룹과의 관계를 관리하려면 그룹 권한이 필요합니다. 권한에 대한 자세한 내용은 여기를 클릭 하시기 바랍니다.

다른 그룹과 제휴하기

다른 그룹이 자신의 그룹과 동맹이 되도록 요청하려면 다음을 수행합니다:

  1. 화면 왼쪽에 있는 탐색 모음에서 그룹 을 클릭합니다
  2. 해당 그룹을 찾아서 클릭합니다
  3. 페이지 오른쪽에 있는 Group Admin (그룹 관리자) 버튼을 클릭합니다
  4. 왼쪽에 있는 Relationship (관계) 탭을 선택합니다.
  5. 제휴 요청을 보낼 그룹의 이름을 입력합니다
  6. 녹색 Send Ally Request (알리 요청 보내기) 버튼을 누릅니다

적의 허용 및 선언

그룹에 대한 적을 허용하거나 방지하려면 다음을 수행합니다:

  1. 화면 왼쪽에 있는 탐색 모음에서 그룹 을 클릭합니다
  2. 해당 그룹을 찾아서 클릭합니다
  3. 페이지 오른쪽에 있는 Group Admin (그룹 관리자) 버튼을 클릭합니다
  4. 왼쪽의 Settings (설정) 탭을 선택합니다
  5. Allow enemy declarations (적의 선언 허용 확인) 상자를 선택합니다
  6. 적을 허용하지 않으려면 [적 선언 허용] 상자의 선택을 취소합니다
  7. 녹색 Save (저장) 버튼을 클릭합니다
  8. Group Admin Page (그룹 관리자 페이지) 에서 왼쪽에 있는 Relationship (관계) 탭을 선택합니다
  9. 적으로 선언할 그룹의 이름을 입력합니다
  10. 녹색 Declare Enemy! (적을 선언하라!) 버튼을 누릅니다
