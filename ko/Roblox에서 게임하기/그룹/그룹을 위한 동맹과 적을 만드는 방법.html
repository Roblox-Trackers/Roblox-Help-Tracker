그룹을 위한 동맹과 적을 만드는 방법

이제 그룹을 생성했으므로 다른 그룹과 동맹 또는 적이 되고 싶을 수 있습니다. 그룹 소유자가 아닌 경우 다른 그룹과의 관계를 관리하려면 그룹 권한이 필요합니다. 권한에 대한 자세한 내용은 그룹 역할과 등급 및 권한 부여하기 도움말을 참고하세요.

다른 그룹과 동맹 되기

다른 그룹이 자신의 그룹과 동맹이 되도록 요청하려면 다음을 수행합니다:

  1. 화면 왼쪽에 있는 탐색 모음에서 그룹 을 클릭합니다
  2. 해당 그룹을 찾아서 클릭합니다
  3. 페이지 오른쪽에 있는 Configure Group (그룹 구성) 버튼을 클릭합니다
  4. 왼쪽에 있는 Affiliates (파트너) 탭을 클릭합니다
  5. Allies (동맹) 을 선택합니다
  6. 오른쪽 상단에 있는 Send Ally request (동맹 요청 보내기) 버튼을 클릭합니다
  7. 동맹 요청을 보내고 싶은 그룹의 이름을 입력합니다
  8. Request (요청) 버튼을 누릅니다


참고: 다른 그룹에서 대기 중인 동맹 요청을 수락하거나 거절하려면 해당 페이지의 요청 섹션으로 이동하여 수락 혹은 거절을 클릭하면 됩니다. 또한 모든 요청을 한번에 수락하거나 거절하려면 오른쪽에 있는 모두 수락/모두 거절 버튼을 클릭할 수도 있습니다.

적의 허용 및 선언

그룹에 대한 적을 허용하거나 방지하려면 다음을 수행합니다:

  1. 화면 왼쪽에 있는 탐색 모음에서 그룹 을 클릭합니다
  2. 해당 그룹을 찾아서 클릭합니다
  3. 페이지 오른쪽에 있는 Configure Group (그룹 구성) 버튼을 클릭합니다
  4. 왼쪽의 Settings (설정) 탭을 선택합니다
  5. Allow enemy declarations ( 적 선포 허용 ) 토글 버튼을 클릭합니다
  6. 적을 허용하지 않으려면 Allow enemy declarations ( 적 선포 허용 ) 토글의 선택을 취소합니다
  7. 그룹 구성 페이지 에서, 왼쪽에 있는 Affiliates (파트너) 탭을 클릭합니다
  8. Enemies (적) 을 선택합니다
  9. 페이지의 오른쪽 상단에 있는 Declare Enemy (적 선포) 버튼을 클릭합니다
  10. 적 선포를 보내고 싶은 그룹의 이름을 입력합니다
  11. Send (보내기) 버튼을 누릅니다
