그룹을 광고하는 방법

멋진 새 그룹을 만들었다면, 새로운 멤버들을 끌어들이기 위해 단어를 꺼내도록 도울 수 있을 것입니다! 광고는 좋은 방법이고 그룹 광고는 Roblox에 다른 물건들을 광고하는 것과 비슷합니다.

광고를 만드는 방법

Advertise the Group (그룹 광고) 권한이 (권한에 대한 자세한 정보를 보려면 여기를 클릭 ) 부여된 모든 그룹 구성원은 광고를 만들 수 있습니다.

시작하려면 그룹의 페이지로 이동하여 오른쪽에 있는 알림 Advertise Group (그룹 광고) 버튼을 클릭하여 Create a User Ad (사용자 광고 만들기) 페이지를 엽니다. 여기서:

  1. 만들고자 하는 광고의 크기를 선택하세요. 현재 세 가지 옵션이 있습니다. 각 파일에는 시작을 돕기 위해 다운로드할 수 있는 템플릿이 있습니다:

    728 x 90 배너 템플릿 다운받기

    160 x 600 초고층 빌딩 템플릿 다운받기

    300 x 250 대형 직사각형 템플릿 다운받기

  2. 이미지 편집 소프트웨어를 사용하여 매력적인 광고를 제작하도록 합니다.
  3. 사용자 지정된 광고를 컴퓨터에 저장합니다.
  4. 다시 Create a User Ad ( 사용자 광고 만들기) 페이지에서 Choose File (파일 선택) 버튼을
    클릭합니다.
  5. 2단계에서 생성한 이미지를 찾아 선택합니다.
  6. 이름을 입력하고 녹색 Upload (업로드) 버튼을 클릭합니다.

업로드한 광고는 우리 중간 관리자 팀이 검토해서 그것이 Roblox 커뮤니티 기준 에 부합하는지 확인할 것이다. 이 프로세스에 몇 시간을 할애하시기 바랍니다. 광고가 승인되면 광고 인벤토리 페이지에서 광고를 시작할 수 있습니다. 광고 시스템 운영 및 광고 시스템 작동에 대한 자세한 내용은 여기를 클릭 하시기 바랍니다.

참고: 그룹 기금 지출 권한이 부여된 경우 그룹 펀드로 광고를 입찰할 수 있습니다.
