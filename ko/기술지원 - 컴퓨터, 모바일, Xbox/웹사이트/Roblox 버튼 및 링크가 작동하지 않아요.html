Roblox 버튼 및 링크가 작동하지 않아요

Roblox 웹사이트에 있는 버튼 또는 링크가 작동하지 않아(전혀 응답하지 않을 경우) 플레이 및 커뮤니티와의 소통이 불가능할 경우, 잘못된 브라우저 보안 설정에 그 원인이 있을 수 있습니다.

아래 내용은 Internet Explorer에서 보안 설정을 변경하는 방법입니다. 다른 선호하는 웹 브라우저가 있다면, 해당 웹 브라우저의 웹사이트에 접속해서 보안 설정을 어떻게 변경하는지 확인하세요. 보안 설정 레벨을 체크해 보거나, Roblox가 신뢰할 수 있는 사이트 목록에 포함되어 있는지 확인해 보세요.
  1. Internet Explorer에서 오른쪽 상단에 있는 기어 아이콘을 클릭하고, 인터넷 옵션 을 엽니다.
  2. 보안 탭으로 이동합니다.
  3. 보안이 높음 으로 설정되어 있는지 확인합니다.

보안이 높음으로 설정된 경우, 보안 설정을 낮추거나 신뢰할 수 있는 사이트 목록에 Roblox를 추가해야 합니다. 이 설정을 변경하는 방법은 다음과 같습니다.

  1. Internet Explorer에서 오른쪽 상단에 있는 기어 아이콘을 클릭하고. 인터넷 옵션 을 엽니다.
  2. 보안 탭으로 이동합니다.
  3. 체크 아이콘으로 보이는 신뢰할 수 있는 사이트 를 클릭합니다.
  4. 사이트 버튼을 클릭합니다.
  5. Roblox 웹 주소( http://www.roblox.com )를 입력합니다.
  6. ‘ 이 영역에 있는 모든 사이트에 대해 서버 검증 (https:) 필요 ’ 상자에 체크가 되어 있다면, 체크를 해제합니다 .
  7. 추가 버튼을 클릭합니다.
  8. 닫기 버튼을 클릭합니다.
  9. 모든 브라우저 창을 닫습니다.
  10. Internet Explorer 창을 열고 문제가 해결되었는지 다시 확인합니다.

Safari 보안 환경 설정에 접속하려면 Safari를 클릭하고 기본 설정에서 보안 을 클릭하세요. 보안 옵션에 대한 정보가 필요하면, 여기 를 클릭하세요.
