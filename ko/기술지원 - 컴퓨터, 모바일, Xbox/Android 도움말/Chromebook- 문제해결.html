Chromebook: 문제해결

요건

Chromebook에서 Roblox를 사용하기 전에 Chrome OS가 모두 최신이며 Google Play 스토어 가 모바일 앱의 Android 버전을 사용할 때 기기 설정에서 활성화되어야 합니다.

참고: Roblox 앱은 블루 투스 마우스 또는 기타 Bluetooth 포인팅 장치에서는 작동하지 않습니다.

일반적인 문제해결

앱을 닫고 다시 엽니다

    1. 오른쪽 상단에 있는 Chromebook의 Chrome 브라우저에 있는 세 줄처럼 보이는 menu(메뉴) 버튼을 클릭합니다.
    2. 팝업 메뉴에서 More Tools(추가 도구) 를 클릭합니다.
    3. Task Manager(작업 관리자) 를 클릭합니다.
    4. 강제로 End Process(프로세스 종료) 를 선택하고 화면 오른쪽 아래에 있는 프로세스 종료를 클릭합니다.

참고: 이 프로그램이 Chromebook에서 중요하다고 생각하는 프로그램인 경우 Chromebook에서 화면 오른쪽 아래에 있는 팝업을 통해 앱을 다시 시작하라는 메시지를 표시합니다.

장치를 다시 시작

    1. 계정 사진을 클릭한 다음 Sign out(로그아웃) 한 다음 Turn off(꺼짐) 을 클릭합니다.
    2. 30초 동안 장치의 전원을 끈 후 전원 버튼을 눌러 다시 켭니다.
    3. 앱을 다시 엽니다.

앱 다시 설치하기

    1. Launcher(시작 프로그램) 아이콘을 클릭하고 All Apps(모든 앱) 을 클릭한 다음 제거할 앱을 찾습니다.
    2. 앱 아이콘을 마우스 오른쪽 버튼으로 클릭하고 Remove from Chrome (Chrome에서 제거하기) 를 선택합니다.
    3. 확인 창에서 Remove(제거) 를 클릭합니다

날짜와 시간이 올바른지 확인

    1. Taskbar(작업 표시줄) 의 status(상태) 영역을 클릭한 다음 Settings(설정) 을 클릭합니다.
    2. 설정 페이지 하단으로 스크롤하고 Show advanced settings(고급 설정 보기) 를 클릭합니다.
