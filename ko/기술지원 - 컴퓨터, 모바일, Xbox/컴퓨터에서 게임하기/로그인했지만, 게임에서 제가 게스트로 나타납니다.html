Windows에서 인터넷 옵션 재설정

Internet Explorer를 사용하지 않더라도 이 브라우저를 통해 인터넷 옵션을 재설정하면 여전히 문제가 해결될 수 있습니다. Internet Explorer를 사용하여 이러한 단계를 수행해야 하지만 완료되면 원하는 브라우저로 돌아갈 수 있습니다. 다음을 시도해 보시기 바랍니다:

    1. Internet Explorer 열기
    2. 기어 아이콘 > 인터넷 옵션 열기
    3. 고급 탭 열기
    4. 리셋 버튼을 클릭하기

재설정 후에는 Internet Explorer를 닫아 변경 사항을 적용하시기 바랍니다.

참고: Windows 10을 사용하는 경우 Microsoft Edge가 아닌 Internet Explorer에서 이러한 단계를 시도했는지 확인하시기 바랍니다. 아이콘도 비슷해 보입니다. Windows 10에서 IE를 찾으려면 키보드의 Windows 키 + S를 누르고 "Internet Explorer"를 입력합니다.

Windows 시스템 시계 확인

이상하게 들릴 수도 있지만 Windows 시스템 시계가 잘못된 시간이나 날짜로 설정되어 있기 때문에 여러 가지 문제가 발생할 수 있습니다. 만약 이것이 원인이라면, 모든 것을 바로잡기만 하면 그 상황을 해결할 수 있을 것입니다. 이 작업을 수동으로 수행할 수 있지만 서버로 자동 업데이트하도록 설정할 수도 있습니다. 따라서 서버가 지연되지 않도록 하고 문제를 다시 일으키기 시작합니다. 이를 수행하는 방법은 다음과 같습니다:

Windows 8.1 및 이전 버전

    1. 오른쪽 아래 모서리에 있는 윈도우 클록을 마우스 오른쪽 버튼으로 클릭합니다.
    2. 날짜/시간 조정 을 클릭합니다.
    3. 인터넷 시간 탭 선택합니다.
    4. 여기서 시간과 날짜를 인터넷 서버와 동기화할 수 있습니다.

Windows 10

    1. 오른쪽 아래 모서리에 있는 윈도우 클록을 마우스 오른쪽 버튼으로 클릭합니다.
    2. 시간 설정이 자동으로 켜져 있고 올바른 표준 시간대를 선택했는지 확인합니다.

프록시 설정 확인

위의 단계로 문제가 해결되지 않으면 컴퓨터가 프록시 서버를 통해 연결을 시도하지 않는지 확인해야 할 수 있습니다. 다음 단계를 수행하시기 바랍니다:

Windows 7 및 이전 버전

    1. 시작 메뉴로 이동
    2. "프록시" 서버를 검색하고 프록시 서버를 구성합니다.
    3. LAN 설정 을 클릭하고 해당 탭에서 모든 체크 표시를 해제합니다.
    4. 확인을 클릭합니다.

Windows 8 or 8.1

    1. 화면의 왼쪽 하단 모서리를 클릭하여 타일 시작 화면으로 이동합니다.
    2. "프록시"를 입력하면 프록시 서버 구성 옵션이 나타납니다. 클릭하시기 바랍니다.
    3. LAN 설정 을 클릭하고 해당 탭에서 모든 체크 표시를 해제합니다.
    4. 확인을 클릭합니다.

Windows 10

    1. 키보드의 Windows 키 + S를 누릅니다. 또는 작업 표시줄에 Cortana의 "Asking me anything" 막대가 있는 경우 해당 막대를 클릭하면 됩니다.
    2. "프록시"를 입력하면 프록시 설정 변경 옵션이 나타납니다. 클릭하시기 바랍니다.
    3. "설치 스크립트 사용" 및 "프록시 서버 사용"이 꺼짐으로 설정되었는지 확인합니다.
    4. 아래로 스크롤하여 Save(저장)을 클릭합니다.
