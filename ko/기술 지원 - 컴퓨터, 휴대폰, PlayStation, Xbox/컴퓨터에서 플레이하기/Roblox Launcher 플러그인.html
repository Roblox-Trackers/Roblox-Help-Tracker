Roblox Launcher 플러그인

Roblox가 이미 설치되어 있는데도 실행할 수 없거나 다운로드하라는 안내가 나타나는 경우가 있습니다. 확인하려면 다음을 따르세요.

Mozilla Firefox

  1. Firefox 창의 오른쪽 상단에 있는 메뉴 버튼을 클릭합니다.
  2. 메뉴에서 부가 기능 버튼을 클릭합니다.
  3. 나타난 창에서 플러그인을 클릭합니다.
  4. ‘Roblox Launcher 플러그인’을 찾아서 오른쪽 드롭다운 메뉴에 있는 항상 허용을 선택합니다.
  5. 추가 기능 관리 탭을 닫고 Roblox 체험을 실행합니다.

Apple Safari

  1. Safari를 열고 오른쪽 상단에 있는 톱니바퀴 아이콘을 클릭합니다
  2. 환경 설정을 클릭하고 보안 탭을 선택합니다.
  3. 보안 탭 밑에 ‘Plug-ins, Java, and JavaScript’가 활성화되어 있는지 확인합니다.

Google Chrome

참고: 42 버전부터 Chrome은 더 이상 플러그인을 지원하지 않습니다.

Microsoft Edge

참고: Microsoft Edge는 플러그인을 지원하지 않습니다.
