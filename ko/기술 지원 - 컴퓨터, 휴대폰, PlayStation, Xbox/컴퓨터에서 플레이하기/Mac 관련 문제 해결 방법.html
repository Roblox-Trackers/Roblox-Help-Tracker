Mac 관련 문제 해결 방법

Roblox 앱 삭제 또는 재설치

최근에 앱을 재설치한 적이 없을 경우, 다음 두 가지 도움말의 하단에 위치한 안내에 따라 프로그램을 삭제하고 재설치해 보세요.

  • 삭제하기
  • 재설치하기

컴퓨터 다시 시작

Roblox 앱 사용 시의 문제일 경우, 컴퓨터를 다시 시작하면 문제가 해결됩니다.

로그 재설정

컴퓨터를 다시 시작해도 해결이 되지 않으면 아래 단계를 따르세요.

  1. Command + Space를 누릅니다.
  2. ~/Library/Logs/Roblox/를 복사해서 붙여 넣습니다.
  3. 폴더의 모든 항목을 열고 삭제합니다.
  4. 다시 시도해 봅니다.

기타 방법

삭제 후 재설치를 완료했는데도 문제가 해결되지 않는 경우 아래 방법을 시도해 보세요.

  • Roblox 앱은 10.10(Yosemite) 이상에 설치가 가능하지만, Roblox Studio는 Mac OS 10.13(High Sierra) 이상이어야 설치가 가능합니다.
  • 현재 Apple Silicon / M1 칩은 지원이 가능합니다.
  • Safari 웹브라우저 사용 중 문제가 발생한 경우, Firefox 또는 Google Chrome 등 Roblox에서 지원하는 브라우저 중 하나를 사용해 보세요.
  • Roblox 아이콘이 현재 Dock에 있고, Roblox 응용 프로그램이 애플리케이션 폴더에 위치해 있는지 확인하세요.
  • 웹사이트에서 체험을 하는 데 문제가 있을 경우 Dock에서 프로그램을 열어 보세요.
  • 설정 파일을 삭제해 Roblox 설정을 완전히 재설정하세요. 라이브러리 -> 환경 설정으로 이동해 모든 Roblox.com 파일을 삭제하면 됩니다. Studio 또는 플레이어를 다시 열면 모든 프로그램이 다시 설치됩니다.

위의 사항들을 모두 따른 후에도 계속 오류가 발생한다면 다음 세부 정보들을 기재해 Roblox 지원 센터에 문의 해 주세요.

  • Roblox 사용자 이름
  • 수신한 모든 오류 메시지
  • 이용 중인 Mac의 모델명
  • Mac OS 버전
  • 오류를 재현하는 법(순서)

지원 센터에서 해당 오류를 이해하는 데 도움이 될 만한 추가 세부 정보, 그리고 Mac 또는 네트워크의 최신 변경 사항도 제공해 주세요.
