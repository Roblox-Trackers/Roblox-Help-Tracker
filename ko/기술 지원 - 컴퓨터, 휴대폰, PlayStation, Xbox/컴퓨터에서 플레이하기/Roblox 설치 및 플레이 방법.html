Roblox 설치 및 플레이 방법

Roblox 플레이 방법

  1. Roblox 플레이어 설치하기
    • Microsoft Edge, Firefox 또는 Chrome과 같은 최신 브라우저를 사용하여 Roblox 웹사이트 로 이동합니다.
    • Roblox에 로그인한 후 체험을 방문해 초록색 플레이 버튼을 클릭합니다.
    • Roblox 플레이어가 설치 중임을 알리는 팝업 창이 나타납니다.
  2. Microsoft Edge, Firefox 또는 Chrome과 같은 최신 브라우저를 통해 Roblox 플레이어 실행하기
    • Roblox 웹사이트에서 체험을 하나 실행합니다.
    • 플레이 버튼을 클릭합니다.
    • 프로그램을 선택하라는 보조 팝업 창이 나타날 수 있습니다. Roblox를 선택하고 확인합니다. 선택 사항을 기억하도록 저장해 두면 나중에 이 작업을 다시 수행할 필요가 없습니다.

MacOS에서 Firefox를 통해 Roblox를 설치하고 플레이하는 방법

1. Roblox 플레이어 설치하기

  • Firefox를 사용해 Roblox 웹사이트로 이동합니다.
  • Roblox에 로그인한 후 체험을 방문해 초록색 플레이 버튼을 클릭합니다.
  • Roblox 플레이어가 설치 중임을 알리는 팝업 창이 나타납니다.

2. Roblox 플레이어 실행하기

  • 항상 이 애플리케이션을 사용하여 roblox 플레이어 링크 열기 를 선택하고, 선택 을 클릭하여 Finder 창을 불러옵니다.
  • Finder 창에서 키보드 단축키 CMD+SHIFT+G 를 사용하여 경로 이름 입력을 불러옵니다.
  • 경로 이름 입력에 /Applications/Roblox.app/Contents/MacOS/Roblox.app 을 입력하고 RETURN
    을 누릅니다.
  • 그런 다음 Finder 창에서 Roblox.app을 선택하고 열기
    를 클릭합니다.
  • 응용 프로그램 선택 대화 상자에서 Roblox.app를 선택하고 링크 열기
    를 클릭합니다.
  • Roblox가 실행되며, 플레이 버튼을 클릭하여 Roblox 체험을 실행하면 추후에 동일하게 작동합니다.

Roblox Studio 설치 방법

자신만의 Roblox 체험을 만들려면 Roblox Studio를 설치해야 합니다. 방법은 다음과 같습니다.

  1. Roblox 웹사이트 로 이동해 계정에 로그인합니다.
  2. Roblox에 로그인한 후 본인 의 체험에 들어가서 오른쪽 상단에 있는 을 클릭하고, 편집 을 선택합니다.
  3. Roblox Studio가 설치 중임을 알리는 팝업 창이 나타납니다.
  4. 설치가 끝나면 Studio가 자동으로 실행됩니다.
