Hyperion 메시지

이 문서에는 표시될 수 있는 잠재적인 Hyperion 안티 치트 충돌 메시지와 문제 해결을 돕기 위한 문제 해결 단계가 포함되어 있습니다.

충돌 유형 : 시간 초과 실패

사용자 메시지 : Roblox가 시간을 초과했습니다. 클라이언트를 다시 시작하세요.

문제 해결 단계 :

'Roblox가 시간을 초과했습니다. 클라이언트를 다시 시작하세요.'라는 메시지가 나타나면 Roblox 클라이언트를 다시 시작해야 한다는 의미입니다. 정기적으로 사용하다가 클라이언트를 다시 시작해도 문제가 지속되면 지원팀에 문의해 도움을 받으세요.

충돌 유형 : 가상 머신 감지

사용자 메시지 : 가상 머신 내에서 Roblox를 실행할 수 없습니다.

문제 해결 단계 :

'가상 머신 감지'라는 메시지가 나타나면 Roblox가 해당 가상 머신(VM)을 지원하지 않는다는 의미입니다. Roblox를 플레이하려면 VM 내에서 제대로 작동하는 GPU 패스스루가 필요합니다. VM은 GPU 패스스루를 통해 하이퍼바이저를 우회하여 실제 GPU에 직접 접근할 수 있습니다. GPU 패스스루 활성화 방법은 다음과 같습니다.

      1. 하드웨어의 GPU 패스스루 지원 여부를 확인합니다.
      2. BIOS/UEFI 설정을 업데이트해 가상화 기술(예: Intel VT-x 또는 AMD-V)을 활성화합니다.
      3. VM 소프트웨어 내에서 GPU 패스스루를 구성합니다.

이 단계를 진행하면 가상화된 환경에서 Roblox가 원활하게 실행되며 GPU 리소스에 필수적인 접근이 제공됩니다.

충돌 유형 : 배드웨어 감지

사용자 메시지 : 호환되지 않는 소프트웨어로 인해 Roblox에 충돌 발생: <문제를 유발한 소프트웨어 경로>

문제 해결 단계 :

'호환되지 않는 소프트웨어로 인해 Roblox에 충돌 발생'이라는 메시지와 문제를 유발한 소프트웨어 경로가 나타나면 표시된 파일 경로에 언급된 소프트웨어가 Roblox에 문제를 일으켰다는 의미입니다. 이를 해결하려면 Roblox를 사용하는 동안 해당 소프트웨어가 실행되지 않게 해야 합니다. 수동으로 해당 소프트웨어를 종료하거나 Windows에서 클린 부팅을 실행하면 되죠. 이 방법을 통해 시스템에서 필수적인 소프트웨어만 활성화하고 충돌을 최소화할 수 있습니다. Windows에서 권장하는 클린 부팅 방법은 다음과 같습니다 :

      1. 'Windows 키 + R'을 눌러 실행 대화 상자를 엽니다.
      2. 'msconfig'를 입력하고 Enter를 누릅니다.
      3. 시스템 구성 창에서 '서비스' 탭으로 이동합니다.
      4. '모든 Microsoft 서비스 숨기기'라고 적힌 상자에 체크합니다.
      5. '모두 사용 안 함'을 클릭합니다.
      6. '시작' 탭으로 전환합니다.
      7. '작업 관리자 열기'를 클릭합니다.
      8. 모든 시작 항목을 비활성화합니다.
      9. 작업 관리자와 시스템 구성을 닫습니다.
      10. 컴퓨터를 다시 시작합니다.

이 단계를 완료하면 다른 소프트웨어의 방해 없이 Roblox가 충돌하지 않고 원활하게 실행되는 데 도움이 됩니다.

충돌 유형 : 과정 일시 중단

      1. 'Windows 키 + R'을 눌러 실행 대화 상자를 엽니다.
      2. 'msconfig'를 입력하고 Enter를 누릅니다.
      3. 시스템 구성 창에서 '서비스' 탭으로 이동합니다.
      4. '모든 Microsoft 서비스 숨기기'라고 적힌 상자에 체크합니다.
      5. '모두 사용 안 함'을 클릭합니다.
      6. '시작' 탭으로 전환합니다.
      7. '작업 관리자 열기'를 클릭합니다.
      8. 모든 시작 항목을 비활성화합니다.
      9. 작업 관리자와 시스템 구성을 닫습니다.
      10. 컴퓨터를 다시 시작합니다.

이 단계를 완료하면 다른 소프트웨어의 방해 없이 Roblox가 충돌하지 않고 원활하게 실행되는 데 도움이 됩니다.

충돌 유형 : 부팅 무결성 손상

사용자 메시지 : 중요 시스템 파일이 손상되었습니다. Roblox를 실행할 수 없습니다. Windows 설치를 복구하거나 초기화하세요.

문제 해결 단계 :

Windows가 디버그 모드로 설정되면 Roblox가 제대로 작동하지 않습니다. Roblox를 이용하려면 Windows가 정상적으로 실행되어야 합니다. 다음 단계에 따라 Windows의 디버그 모드를 확인하고 비활성화하세요 .

      1. 'Windows 키+R'을 눌러 실행 상자를 엽니다.
      2. 'msconfig'를 입력하고 Enter를 누릅니다.
      3. '부팅 탭'을 선택한 다음, 고급 옵션을 선택합니다.
      4. 디버그 선택 상자의 선택를 해제합니다.
      5. '확인'을 선택합니다.
      6. 적용 그리고 '확인'을 선택합니다.
      7. 컴퓨터를 다시 시작합니다.

이 단계를 통해 Windows가 디버그 모드에서 실행되지 않게 하고, 문제없이 Roblox를 이용할 수 있습니다.

충돌 유형 : 오픈 브로드캐스터 소프트웨어(OBS)

사용자 메시지 : OBS 인게임 캡처 이용 시 Roblox에 문제가 발생하는 것으로 알려졌습니다. 창 또는 데스크톱 캡처를 이용하세요.

문제 해결 단계 :

'OBS 인게임 캡처 이용 시 Roblox에 문제가 발생하는 것으로 알려졌습니다.'라는 메시지가 나타나면 Roblox가 오픈 브로드캐스터 소프트웨어(OBS)의 인게임 캡처 기능을 지원하지 않는다는 의미입니다. Roblox를 스트리밍 또는 녹화하려면 창 또는 데스크톱 캡처 모드를 이용해야 합니다. 다음 지침에 따라 데스크톱 캡처를 설치하세요 .

      1. 오픈 브로드캐스터 소프트웨어(OBS)를 엽니다.
      2. '소스' 섹션에서 '+' 아이콘을 클릭합니다.
      3. 메뉴에서 '화면 캡처'를 선택합니다.
      4. 캡처하려는 화면을 선택합니다(보통 주로 사용하는 모니터).
      5. 필요한 경우 해상도와 프레임률 등의 설정을 조정합니다.
      6. '확인'을 클릭해 화면 캡처 소스를 확정합니다.
      7. OBS 화면에서 화면 캡처 소스를 원하는 대로 정렬합니다.
      8. 평소대로 스트리밍 또는 녹화를 시작하세요.

데스크톱 캡처 모드를 이용해 OBS의 인게임 캡처 기능의 호환성 문제를 겪지 않고 효율적으로 Roblox를 스트리밍하거나 녹화할 수 있습니다.
