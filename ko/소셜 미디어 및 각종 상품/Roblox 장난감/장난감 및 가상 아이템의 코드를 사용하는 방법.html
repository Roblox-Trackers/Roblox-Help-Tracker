장난감 및 가상 아이템의 코드를 사용하는 방법

가상 아이템 코드는 어디에서 찾을 수 있나요?

  • Jazwares 액션 피규어: 코드는 패키지 전면 또는 내부에 있는 코드카드 혹은 토큰에 있습니다. 가상 아이템 코드 찾기에 대한 자세한 내용은 Jazwares 동영상 을 통해 확인하세요.
  • 너프건 및 모노폴리: 코드는 패키지에 있는 코드 카드에 있습니다.

  • 기타 가상 아이템 코드(예: Prime Gaming, Verizon, 등)의 경우 다음 사용 방법을 참고하세요.

가상 아이템에 코드를 어떻게 사용할 수 있나요?

참고: 디지털 가상 코드를 가지고 있으면 2번 단계로 넘어가세요.

  1. 코드가 가려져 있다면, 가려진 부분을 부드럽게 긁습니다.
  2. 코드를 사용할 Roblox 계정에 로그인합니다.
  3. 코드 사용 페이지 로 이동합니다.
  4. 상자에 코드를 입력합니다.
  5. 사용 버튼을 클릭합니다.
  6. 계정 인벤토리에 새 아이템이 나타납니다. 아이템 확인 시 해당 아이템에 맞는 카테고리(예: 모자, 배낭, 머리, 장신구, 등)의 인벤토리를 선택해야 합니다.

참고: 한 계정당 각 장난감의 가상 아이템을 한 번만 받을 수 있습니다. 계정에 이미 소유한 가상 아이템에 대한 코드가 있는 경우, 다른 사용자와 코드를 공유할 수 있습니다. 이러한 가상 아이템은 판매하거나 거래할 수 없습니다.
