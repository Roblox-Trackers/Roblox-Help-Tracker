프로모션 코드는 어떻게 받을 수 있나요?

Roblox에서 진행하는 다양한 이벤트 혹은 증정 행사를 통해 프로모션 코드를 받을 수 있습니다.

가상 아이템에 프로모션 코드를 사용하는 방법을 알려 주세요.

  1. 코드를 사용할 Roblox 계정에 로그인합니다.
  2. 프로모션 코드 사용 페이지 로 이동합니다.
  3. 아래와 같이 코드 입력란에 코드를 입력 합니다.
  4. 사용 을 클릭합니다.
    • 코드가 성공적으로 사용된 경우, 아래와 같이 표시됩니다.
    • 코드가 비활성 상태이거나 프로모션 코드가 아닌 경우, 아래와 같이 표시됩니다.
  5. 프로모션 코드 사용이 완료되면 계정 인벤토리에 새 아이템이 나타납니다. 아이템 확인 시 해당 아이템에 맞는 카테고리의 인벤토리를 선택해야 합니다.

참고: 프로모션 코드는 단기간에 만료 또는 비활성화될 수 있습니다. 따라서 받는 즉시 바로 사용하세요.
