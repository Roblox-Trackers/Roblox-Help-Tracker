프로모션 코드를 사용하는 방법

프로모션 코드는 어떻게 받을 수 있나요?

Roblox에서 진행하는 다양한 이벤트 혹은 증정 행사를 통해 프로모션 코드 를 받을 수 있습니다. 프로모 아이템에 대해서 더 알아보시려면 프로모션 또는 콘테스트 아이템을 얻으려면 어떻게 해야 하나요? 라는 도움말을 참고하실 수 있습니다.

참고: 프로모션 코드는 단기간에 만료 또는 비활성화될 수 있습니다. 따라서 받는 즉시 바로 사용하세요.

가상 아이템을 위한 프로모션 코드를 어떻게 사용할 수 있나요?

  1. 코드를 사용할 Roblox 계정에 로그인합니다.
  2. 코드 사용 페이지 로 이동합니다.
  3. 상자에 코드를 입력합니다.
  4. 사용 을 클릭합니다.
  5. 코드를 성공적으로 사용되면 확인 메시지가 나타납니다.
  6. 사용한 코드의 종류에 따라 크레딧, Robux 혹은 가상 아이템을 받게 됩니다.

프로모 코드 사용 관련 문제

프로모 코드를 사용하는 데에 어려움이 있으시면 다음을 따라해 보세요:

  1. 코드를 사용하기 전에 정확한 Roblox 계정에 로그인되어 있는지 확인해 보세요.
  2. 프로모 코드 사용 페이지 로 이동합니다.
  3. 코드를 정확하게 입력하셨는지 확인하세요.
  4. 특별 아이템을 이미 잘 받으셨는지 인벤토리를 확인해 보세요.

참고: 프로모 코드는 제한된 기간에만 사용 가능합니다. 특정 코드의 사용 가능한 기간이 지났으면 해당 코드를 사용 불가능할 것입니다.
