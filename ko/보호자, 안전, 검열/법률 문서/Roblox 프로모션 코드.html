Roblox 프로모션 코드

가상 아이템 증정을 위한 무료 프로모션 코드를 제공하려면 어떻게 해야 하나요?

개발자는 본인의 체험 아이템에 한해 프로모션 코드를 구현할 수 있습니다. 이때 다음을 따라야 합니다.

  1. 각 아이템은 코드가 사용되는 체험에서만 사용할 수 있어야 합니다.
  2. 체험 아이템은 Robux 또는 실제 통화 가치가 없어야 합니다.
  3. 각 코드는 공개적으로 액세스할 수 있는 방식으로 게시되어야 합니다.
  4. 코드에 액세스하거나 코드를 사용하는 데 구매가 요구되지 않아야 합니다.
  5. 위의 모든 내용이 모든 홍보 자료에서 사용자에게 명확하게 공개되어야 합니다.

소매용 번들에는 고객이 동봉된 코드에 무료로 액세스하고 이를 사용할 수 있음을 명확하게 공개해, 이를 구매에 앞서 확인할 수 있도록 해야 합니다(예: '가상 체험 내 아이템에는 구매가 요구되지 않습니다. [URL]에서 프로모션 코드를 사용할 수 있습니다.').

또한 개발자는 모든 홍보 자료 작성 시 Roblox 브랜드 가이드라인 을 준수해야 합니다.

Roblox 마켓플레이스에서 이용 가능한 아이템 코드를 증정해도 되나요?

프로모션 코드는 본인의 체험 내에서만 특별히 액세스하고 사용할 수 있는 아이템에 대해서만 구현해야 합니다.

소매 아이템의 패키지에 프로모션 코드를 포함해 제공했습니다. 이렇게 하면 사용자가 '공개적으로 액세스할 수 있어야 한다'는 요건에 부합하는 건가요?

공개적으로 액세스 가능한 코드는 인터넷에서 사용할 수 있는 것이어야 합니다(예: 공개 웹사이트, 본인의 공개 소셜 미디어 게시물 중 하나, Roblox의 세부 정보 페이지 내 설명). 모든 소매 패키지에는 코드 사용에 추가적인 구매가 요구되지 않음을 고객이 구매에 앞서 확인할 수 있도록 명확하게 공개하고 코드를 찾을 수 있는 위치를 명시해야 합니다. (예: '가상 체험 내 아이템에는 구매가 요구되지 않습니다. [URL]에서 프로모션 코드를 사용할 수 있습니다.')

다른 개발자가 마켓플레이스 아이템의 프로모션 코드를 만드는 걸 본 적이 있어요. 이건 왜 가능한 건가요?

Roblox는 최근 마켓플레이스의 일부 개발자 아이템을 현실 세계의 제품과 연계할 수 있는 제한적인 파일럿 프로그램을 시작했습니다. 이 실험을 통해 메타버스 내 본 기능의 잠재력을 확인하고 전체 커뮤니티로 확장할 수 있는지 여부에 대해 본격적으로 파악할 예정입니다. 현재로서는 프로그램 확장에 대한 계획이 없으나, 작업이 완료되는 즉시 알려드리겠습니다.

Roblox 파트너십 프로그램의 일부가 아닌 코드를 플랫폼 외부에서 판매하는 개발자를 목격했습니다. 해당 개발자는 어떤 조치를 받게 되나요?

Roblox는 정책 위반 행위에 대해 매우 엄중한 조치를 취하고 있습니다. 해당 개발자는 DevEx 서비스, 계정 액세스, 체험 이용 가능성에 대한 정지 처분을 받을 수 있습니다.

'무료' 아이템이란 무엇을 의미하나요?

체험의 아이템이 Robux를 대가로 판매하는 것이거나 Robux를 요구하는 게임 내 통화의 대용물이 되어서는 안 됩니다. 특정 아이템 획득에 높은 수준의 기술이나 참여 시간이 필요하기 때문에 가치 있는 것으로 인식되는 경우, 해당 아이템을 모든 사용자가 체험 내에서 찾을 수 있고 자유롭게 사용할 수 있도록 명확하게 게시할 경우에 한해 프로모션 코드를 사용할 수 있습니다.

프로모션이 시즌 한정인 경우 어떻게 해야 하나요?

코드를 홍보하는 모든 위치에 코드 사용 가능 기간을 명시해야 합니다. 소매 패키지에 시즌 한정 프로모션 코드를 포함할 경우, 제품의 전체 소매 수명주기 동안 사용 코드 및 관련 체험 내 아이템을 지원하거나, 프로모션 종료 시기를 패키지에 명시해야 합니다(소매점에서 여전히 해당 패키지가 판매 중일 경우에도 해당).
