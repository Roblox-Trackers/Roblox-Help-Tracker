규칙 위반을 신고하는 방법

Roblox는 다양한 연령의 사용자와 개발자로 구성된 훌륭한 커뮤니티를 가지고 있으며, Roblox를 사용하는 모든 사용자를 대상으로 안전하고 즐거운 환경을 만들기 위해 노력하고 있습니다. 이와 관련하여 다른 사용자가 규칙을 위반하는 것을 보게 되면, 부적절한 행동을 신고할 수 있는 방법이 있습니다.

다른 사용자와의 연락 또는 채팅을 차단하는 방법에 대한 자세한 내용은 다른 사용자를 차단하는 방법 도움말 을 확인하세요.

규칙 위반 신고 기능은 규칙 위반자와 유해한 콘텐츠에 대해 알리는 가장 좋은 방법입니다. 해당 신고 기능은 검열팀으로 위반 내용을 보내고, 검열팀은 반복적으로 규칙을 위반하고 있는 사용자를 추적하여, 해당 사용자의 계정에 대한 추가 조치를 하고 있습니다. 규칙 위반 신고를 할 때는 Roblox의 규칙을 위반하는 특정 채팅, 개인 메시지, 체 또는 기타 콘텐츠를 신고해야 합니다. 이러한 내용이 없이 단순히 특정 사용자의 프로필에서 해당 사용자만을 신고할 경우, 검열팀에서는 해당 사용자가 무엇을 잘못했는지 알 수 없습니다.

체험 내 사용자 신고하기

체 내에서 신고 기능을 사용하여, Roblox 규칙을 위반하는 특정 채팅 또는 기타 콘텐츠를 신고하세요. 해당 내용은 바로 검열팀으로 전달이 되며, 검열팀에서는 사용자가 어떤 규칙을 위반했는지 확인할 수 있습니다.

  1. 화면 왼쪽 상단에 있는 메뉴 버튼을 클릭합니다. 해당 아이콘은 서로 겹쳐진 세 개의 줄처럼 보입니다.
  2. 사용자 이름 옆에 있는 깃발 아이콘을 클릭하거나 메뉴 맨 위에 있는 신고 탭을 클릭합니다.
  3. 체험 또는 사용자 창이 뜨면 사용자 탭을 누릅니다.
  4. 특정 사용자 풀다운 메뉴를 클릭하고, 신고할 사용자를 선택합니다.
  5. 위반 유형 풀다운 메뉴를 클릭하고, 신고할 사용자를 선택합니다.
  6. 추가 세부정보는 간단한 설명 창에서 작성할 수 있습니다.
  7. 제출 을 클릭하고 완료합니다.

만약, 체험 자체가 규칙을 위반하고 있다면, 해당 체험을 하는 동안 신고가 가능합니다. 위의 두 번째 단계에서 설명된 것과 같이 신고 탭을 열고, 사용자 대신에 체험 을 선택하세요.

체험 이외 위반 신고하기

Roblox 규칙을 위반하는 사용자 또는 콘텐츠를 발견하면, 앱에 있는 신고 기능을 사용하여 검열팀에 보내주세요. 신고 방법은 신고 대상에 따라 달라질 수 있지만, 부적절하다고 생각되는 것은 무엇이든 신고할 수 있습니다.

체험 신고하기

체험을 신고하려면 체험 정보 페이지의 설명 박스 오른쪽 하단에 있는 빨간색 신고 버튼을 클릭하세요.

기타 콘텐츠 신고하기

카탈로그나 라이브러리에 규칙을 위반하는 아이템이 있는 경우(예: 셔츠, 체험 패스 또는 모델) 아이템 정보 박스의 오른쪽 상단 모서리에 있는 … 버튼을 클릭하여 아이템 신고 를 선택하고 관련 내용을 적어주세요.

채팅 신고하기

웹 브라우저에 있는 다른 사용자와의 대화를 신고하려면, 채팅 창의 오른쪽 상단에 있는 기어 아이콘을 눌러 채팅 세부 정보를 열고난 후, …으로 보이는 사용자 이름 옆에 있는 버튼을 누르고 신고 를 선택하세요. 그 다음, 빨간색 신고 버튼을 누르고 신고서 양식으로 계속 진행한 후, 해당 양식을 작성 그리고 녹색 신고 버튼을 눌러주세요.

모바일 앱 에서는 신고하는 방법이 조금 다릅니다. 다른 사용자와의 대화를 신고하려면, 화면의 우측 상단에 있는 기어 아이콘을 누른 다음, 신고 를 선택하세요.

사용자가 만든 광고 신고하기

사이트에서 다른 사용자가 만든 광고를 신고하려면, 광고 이미지 우측 아래에 있는 신고 를 누릅니다. 페이지가 다시 로드되면 양식을 작성한 다음 녹색 신고 버튼을 누르세요.
