사용자 차단을 해제하는 방법

실수로 친구를 차단했거나, 차단 목록에서 누군가를 삭제해야 할 경우 아래와 같이 하면 됩니다.

  1. 계정에 로그인합니다.
  2. 계정 설정으로 이동합니다.
    • 브라우저 - 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 개인 정보 를 선택합니다.
  4. 화면 하단에 있는 차단된 사용자 보기 를 선택합니다.
  5. 사용자 이름 옆에 있는 차단 해제 를 선택합니다.
  6. 저장 을 선택합니다.

차단 해제가 안돼요!

차단이 해제되지 않는 이유는 계정 설정의 계정 PIN 기능이 활성화되어 있기 때문입니다. 차단 해제를 가능하게 하려면 아래 단계를 따르세요.

  1. 계정에 로그인합니다.
  2. 계정 설정으로 이동합니다.
    • 브라우저 - 사이트의 오른쪽 상단에 있는 기어 아이콘을 찾습니다.
    • 모바일 앱 - 점 세 개 아이콘으로 표시된 더 보기 를 찾습니다.
  3. 보안 을 선택합니다.
  4. 계정 PIN 기능을 비활성화합니다(보호자의 도움이 필요할 수도 있습니다.)
  5. 개인 정보 를 선택합니다.
  6. 화면 하단에 있는 차단된 사용자 보기 를 선택합니다.
  7. 사용자 이름 옆에 있는 차단 해제 를 선택합니다.
  8. 저장 을 선택합니다.
  9. 보안 탭으로 다시 이동 후, 계정 PIN 기능을 다시 활성화합니다.
