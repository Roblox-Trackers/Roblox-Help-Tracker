가상 장비가 무엇인가요?

가상 장비 는 아바타를 꾸밀 수 있는 각종 아이템으로, 아바타 상점 에서 Robux를 사용해 구매할 수 있습니다. 아이템 정보 페이지 에서 각 장비에 대한 세부 사항을 확인할 수 있습니다.

대부분의 장비들은 체험에서 사용할 수 없으나 일부 가능한 경우가 있습니다. 장비 사용 제한 페이지를 방문해 체험에서 사용 가능한 장비를 찾는 방법에 대해 알아보세요.

장비의 장르

장비는 해당되는 아바타 또는 체험의 유형에 따라 다양한 장르로 분류되어 있습니다. 예를 들면 다음과 같습니다.

  • 건설
  • 공포
  • 마을 및 도시
  • 군사
  • 코미디
  • 중세
  • 모험
  • SF
  • 해군
  • FPS
  • RPG
  • 스포츠
  • 격투
  • 서부극

장비 사용 제한

대부분의 체험에서 장비 사용은 불가능합니다. 특정 체험에서 장비 사용이 가능한지 확인하려면 아래 단계를 따르세요.

  1. 둘러보기 를 클릭해 체험 목록을 엽니다.
  2. 장비 사용 여부를 확인하고자 하는 체험을 클릭합니다.
  3. 체험의 설명 페이지에서 스크롤을 내려 사용 가능 장비 메뉴를 찾습니다.
  4. 사용 가능 장비 로 명시된 부분에 아이콘이 표시되어 있을 경우, 해당 체험에서는 외부 장비 사용이 불가능합니다.
    1. 각 아이콘의 의미를 알고 싶을 경우, 해당 아이콘 위에 커서를 올려놓으면 장비의 장르가 표시됩니다.
