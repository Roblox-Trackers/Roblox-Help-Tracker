게임 복사 허용하지 않기

게임을 복사하지 못하게 하면 다른 사람들이 만든 게임 모든 것을 직접 복사 할 수 없게됩니다. 그러나 다른 사람들이 고객의 게임을 자신의 게임을 위한 템플릿으로 복사하도록 허용하려면 복사를 허용을 선택할 수 있습니다.

복사를 허용하거나 허용 안 하는지 두 가지 방법이 있습니다.

첫번째 방법:

    1. 로그인하고 만들기( Create) 탭을 클릭합니다
    2. 게임 이름 오른쪽에있는 기어 아이콘을 클릭합니다. 그 다음은 표시되는 메뉴에서 시작하는 장소 구성( Configure Start Place) 선택합니다.
    3. 허가 (Permissions) 클릭합니다.
    4. 기타 허가 (Other Permission) 에서 복사를 허용하려면 복사 허용 ( Allow Copying) 옆의 확인란을 선택합니다. 복사를 허용하지 않으려면 확인란을 선택하지 않은 채로 둡니다.
      • 참고:이 확인란을 선택하면 Roblox의 모든 다른 사용자에게 현재 공유하고있는 콘텐츠를 (다양한 방법으로) 권한을 부여하게됩니다. 이 권한을 부여하지 않으려면이 상자를 선택하지 마시기 바랍니다. 콘텐츠 공유에 대한 자세한 내용은 Roblox 이용 약관 을 참고해야 합니다.
    5. 저장 (Save) 클릭합니다.

두번째 방법:

    1. 게임 페이지로 이동합니다.
    2. 세 점 아이콘을 클릭하면 게임의 설정 메뉴가 표시됩니다.
    3. 이 장소 구설 (Configure this Place) 를 선택합니다.
    4. 화면이 나타나면 허가 ( Permissions) 탭으로 이동합니다.
    5. 기타 허가 (Other Permission) 에서 복사를 허용하려면 복사 허용 ( Allow Copying) 옆의 확인란을 선택합니다.복사를 허용하지 않으려면 확인란을 선택하지 않은 채로 둡니다.
      • 참고: 이 확인란을 선택하면 Roblox의 모든 다른 사용자에게 현재 공유하고있는 콘텐츠를 (다양한 방법으로) 권한을 부여하게됩니다. 이 권한을 부여하지 않으려면이 상자를 선택하지 마시기 바랍니다. 콘텐츠 공유에 대한 자세한 내용은 Roblox 이용 약관 을 참고해야 합니다.
    6. 저장 (Save) 을 클릭합니다.
