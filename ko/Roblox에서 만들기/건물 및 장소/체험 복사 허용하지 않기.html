체험 복사 허용하지 않기

체험을 복사하는 것을 허용하지 않는 것은 다른 사람들을 본인이 만든 체험의 모든 것을 복사할 수 없도록 하는 것입니다. 그러나 다른 사람들이 본인의 체험을 자신의 체험을 위한 템플릿으로 복사하도록 허용하려면 복사 허용을 선택할 수 있습니다.

복사를 허용하거나 허용하지 않도록 하기 위해 두 가지 방법이 있습니다.

첫번째 방법:

    1. 로그인하고 만들기( Create) 탭을 클릭합니다
    2. 체험 이름 오른쪽에 있는 톱니바퀴 아이콘을 클릭합니다. 그 다음은 표시되는 메뉴에서 시작하는 플레이스 구성( Configure Start Place) 선택합니다.
    3. 허가 (Permissions) 클릭합니다.
    4. 기타 허가 (Other Permission) 에서 복사를 허용하려면 복사 허용 ( Allow Copying) 옆의 확인란을 선택합니다. 복사를 허용하지 않으려면 확인란을 선택하지 않은 채로 둡니다.
      • 참고:이 확인란을 선택하면 Roblox의 모든 다른 사용자에게 현재 공유하고 있는 콘텐츠를 (다양한 방법으로) 권한을 부여하게 됩니다. 이 권한을 부여하지 않으려면 이 상자를 선택하지 마세요. 콘텐츠 공유에 대한 자세한 내용은 Roblox 이용 약관 을 참고해야 합니다.
    5. 저장 (Save) 클릭합니다.

두번째 방법:

    1. 체험 페이지로 이동합니다.
    2. 세 점 아이콘을 클릭하면 체험의 설정 메뉴가 표시됩니다.
    3. 이 플레이스 구성 (Configure this Place) 를 선택합니다.
    4. 화면이 나타나면 허가 ( Permissions) 탭으로 이동합니다.
    5. 기타 허가 (Other Permission) 에서 복사를 허용하려면 복사 허용 ( Allow Copying) 옆의 확인란을 선택합니다.복사를 허용하지 않으려면 확인란을 선택하지 않은 채로 둡니다.
      • 참고: 이 확인란을 선택하면 Roblox의 모든 다른 사용자에게 현재 공유하고 있는 콘텐츠를 (다양한 방법으로) 권한을 부여하게 됩니다. 이 권한을 부여하지 않으려면 이 상자를 선택하지 마세요. 콘텐츠 공유에 대한 자세한 내용은 Roblox 이용 약관 을 참고해야 합니다.
    6. 저장 (Save) 을 클릭합니다.
