공개 게임을 만들고 창출하기

한 개 게임을 개발하는 것만으로는 창조성에 대한 고객님의 갈망을 충족시키기에 충분하지 않다는 것을 알 수 있습니다. 다행히 Roblox는 고객님들을 도와 드릴 수 있습니다! 모든 사용자는 원하는 만큼의 게임을 만들 수 있지만 계정은 주어진 시간에 200개의 공개 게임 (타인이 플레이 할 수 있음)으로 제한됩니다.

공개 또는 비공개 게임을 만들기:

공개 또는 비공개 게임을 만들려면 두 가지 방법이 있습니다.

참고: 게임을 비공개로 설정하면 실행 중인 게임이 종료되고 모든 VIP 구독이 취소됩니다.

방법 1:

  1. 사이트 상단에 있는 바에서 만들기 를 클릭합니다
  2. 아직 강조 표시되어 있지 않은 경우에는 My Creation 탭에서 Games(게임) 을 클릭합니다
  3. 공개 혹은 비공개로 설정하고 싶은 장소를 찾고 장소 제목 아래에 회색 또는 녹색 아이콘을 클릭하여 상태를 전환합니다.
    • 공개 (녹색 아이콘)는 모든 사용자에게 게임을 보고 재생할 수 있습니다
    • 비공개 (회색 아이콘) 를 그룹 게임이 아닌 다른 사람이 게임을 보거나 재생 할 수 없게 합니다. 어떤 경우에는, 예를 들어서 게임 편집 권한이 있는 필요한 순위로에서 사용자와 개발자만 게임을 할 수 있습니다. 더 자세한 내용은 그룹 게임 을 참조하시기 바랍니다.

방법 2:

  1. 체험 페이지로 이동합니다.
  2. 세 점 아이콘을 클릭하면 게임의 설정 메뉴가 표시됩니다.
  3. Configure this Place(이 장소를 구성하기) 또는 Configure this Game(이 체험을 구성하기) 선택합니다.
  4. 위의 단계에서 선택하는 것에 관계없이, 나타나는 스크린에서 이미 강조 표시되어 있지 않으면 Basic Setting(기본 설정) 탭으로 이동하셔야 합니다.
  5. Privacy에서 , Public(공개) 또는 Private(비공개) 선택합니다
    • Public(공개)는 모든 사용자에게 게임을 보고 재생할 수 있습니다
    • Private(비공개) 는 그룹 게임이 아닌 다른 사람이 게임을 보거나 재생 할 수 없게합니다. 어떤 경우에는, 예를 들어서 게임 편집 권한이 있는 필요한 순위로에서 사용자와 개발자만 게임을 할 수 있습니다. 더 자세한 내용은 그룹 게임 도움말을 참조하시기 바랍니다.

새 게임을 만들기:

  1. 사이트 상단에 있는 바에서 만들기 를 클릭합니다
  2. 아직 강조 표시되어 있지 않은 경우에는 My Creation 탭에서 Games(게임) 을 클릭합니다
  3. Create New Game(새 게임 만들기) 클릭합니다
  4. 새 창작을 위한 설정 및 템플릿을 선택합니다
  5. Create Game(게임 만들기) 클릭합니다

그래서 이런 것은 이전에 있던 Create(창출) 페이지로 돌아갑니다. 고객님이 원한다면 게임을 공개할 수도 있고 아니면, 게임을 할 때까지 기다릴 수도 있습니다. 게임을 만드는 것을 시작하려면 Games( 게임) 목록에서 새 게임의 항목 오른쪽에 있는 Edit(수정) 버튼을 클릭하셔야 합니다.

참고: 언제든지 새 게임을 만들 수 있지만 "Make Game Public(게임 공개하기)"프로세스를 사용하여 공개 할 때까지는 다른 Roblox 플레이어가 즐길 수 없습니다.
