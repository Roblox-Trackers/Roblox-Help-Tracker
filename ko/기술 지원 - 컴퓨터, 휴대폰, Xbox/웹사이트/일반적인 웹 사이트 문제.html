일반적인 웹 사이트 문제

다음 문서에서는 다음 문제를 해결하는 데 효과적인 것으로 입증된 몇 가지 방법을 제안합니다:

  • 웹 사이트가 잘못 표시됩니다
  • 일부 페이지가 로드하지 않거나 로드할 때 오류 발생 메시지가 표시됩니다
  • 버튼이나 링크를 클릭할 때 효과가 없습니다
  • 정화하게 유저이름과 비밀번호를 입력해도 로블록스에 로그인 해보지 못했습니다
  • 기타: 웹 사이트에서 다른 문제가 발생하면 다음 제안된 방법으로도 문제를 해결하는 데 도움이 될 수 있습니다. 추가적우로 웹 사이트와 관련되지 않은 도움말의 링크도 있습니다

해결 방법

아래의 각 단계를 한 번에 하나씩 순서대로 시도하셔야 합니다. 한 가지 해결책이 효과가 없다면 다음 단계로 넘어갑니다!

임시 인터넷 파일 및 쿠키를 지우기

다음 나열된 링크들이 주요 웹브라우저의 임시 인터넷 파일 ("캐시" 라고도 부름) 및 쿠키를 지우는 데 도움이 됩니다. Internet Explorer 및 Roblox를 실행하고 싶은 브라우저에서 이 방법을 시도해 보시기 바랍니다:

      • Internet Explorer - 임시 인터넷 파일 (From Microsoft.com)
      • Internet Explorer - 쿠키 (From Microsoft.com)
      • Mozilla Firefox - 임시 인터넷 파일 (From Mozilla.com)
      • Mozilla Firefox - 쿠키 (From Mozilla.com)
      • Google Chrome - Both (From Google.com)
      • Microsoft Edge - Both (From Microsoft.com)
      • Safari - 쿠키 (From Apple.com)
      • 기타 브라우저: 다른 부라우저를 사용하시거나 이전 버전의 브라우저를 사용하시면 그 특정 브라우저/버전에서  지우는 방법에 대해서 검색해 보시기 바랍니다.

해당 브라우저에서 캐시/브라우저 데이터/임시 인터넷 파일을 지우셨으면 컴퓨터를 다시 시작하고 문제가 해결 되었는지 확인해 보시기 바랍니다.

수동으로 Internet Explorer/Windows 임시 인터넷 파일을 지우고 싶으면 바로 하드 드라이브에 있는 임시 인터넷 파일 폴더로 갑니다. 유저 프로필을 사용하는지 안 하는지 또는 Windows의 버전에 따라 임시 인터넷 파일의 위치와 이름이 다릅니다.

참고: 모든 파일의 형식이 보이도록 (보호된 시스템 파일 포함) Windows Explorer의 설정을 변경하셔야 됩니다. 그렇게 하시면 그 폴더들에 액세스하실 수 있습니다.

Windows 10

Windows 10을 사용하시면 임시 인터넷 파일이 다음과 같은 위치에 있습니다 (PC에서 C드라이브 아닌 다른 드라이브에 있을 수도 있습니다):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Windows 7, 8 및 Vista

Windows 7 또는 Windows Vista를 사용하시면 임시 인터넷 파일이 다음과 같은 위치에 있습니다 (PC에서 C드라이브 아닌 다른 드라이브에 있을 수도 있습니다):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Windows XP

Windows XP를 사용하시면 임시 인터넷 파일이 다음과 같은 위치에 있습니다:

        • C:\Documents and Settings\<username>\Local Settings\Temporary Internet Files\

참고: PC에서 C드라이브 아닌 다른 드라이브에 있을 수도 있습니다. Windows XP에 사용자 계정이 하나만 있는 경우 <username>을 Administrator로 바꿔 임시 인터넷 파일 폴더의 경로를 가져옵니다.

웹 브라우저의 추가 기능(Add-Ons)을 사용하지 않도록 설정하기

종종 웹사이트와 관련된 문제들이 제 3자 회사에서 나온 웹 브라우저의 추가 기능으로 인해 발생합니다. 선택한 브라우저에 따라 "Extensions" 또는 "Plugins" 라고 부르기도 합니다. 추가 기능 때문에 문제가 발생했는지를 화인하기 위해서 추가 기능을 사용하지 않도록 설정해 보시기 바랍니다. 그렇게 하고 나서 문제가 해결되면 문제가 다시 발생할 때까지 추가 기능을 하나씩 다시 켜봅니다. 문제가 다시 발생하면 로블록스와 호환되지 않은 추가 기능이 발견했습니다.

가장 인기 있는 웹 브라우저에 대해서 다음 도움말을 참조하실 수 있습니다:

        • Internet Explorer (From Microsoft.com)
        • Firefox (From Mozilla.com)
        • Chrome (From Google.com)
        • 기타 브라우저: 기타 브라우저 또는 이전 버전의 브라우저를 사용하시면 그 특정 브라우저/버전에서 Extensions/Plugins/추가 기능을 사용하지 않도록 설정하는 것에 대해서 검색해 보시길 바랍니다.

Windows의 인터넷 옵션을 재설정하기

Roblox, 웹사이트 또는 Roblox Studio에 대한 문제를 컴퓨터의 인터넷 옵션을 재설정하는 것으로 해결할 수 있다는 것을 발견했습니다. 다른 웹 브라우저를 사용하셔도 다음 방법들을 Internet Explorer에서도 시도해 보셔야 합니다. 시도한 후 원하시는 웹 브라우저로 돌아가도 됩니다. 다읍 방법을 시도해 보시기 바랍니다:

      1. Internet Explorer를 엽니다 (이 웹 브라우저만)
      2. 기어 아이콘을 엽니다 > 인터넷 옵션
      3. 고급 탭을 엽니다
      4. 리셋 버튼을 클릭합니다

재설정하고 나서 변경사항을 적용하려면 Internet Explorer를 닫으셔야 합니다. 적용이 완료된 후 원하시는 웹 브라우저를 다시 사용하실 수 있습니다.

Internet Explorer를 업데이트하기

최신의 버전으로 Internet Explorer를 업데이트해 보시기 바랍니다. 여기를 클릭하시면 할 수 있습니다. IE를 사용하지 않으셔도 최신의 버전인 IE가 있서야 로블록스를 제대로 실행할 수 있습니다.
