Roblox 실행 플러그인

Roblox가 이미 설치되어 있는데도 실행할 수 없거나 다운로드하라는 안내가 나타나는 경우가 있습니다. Roblox 의 일부 기능은 Internet Explorer 에서만 작동하기 때문에 , Internet Explorer에서와 마찬가지로 주로 사용하는 다른 브라우저에서도 Roblox 플러그인이 허용되어 있는지 확인해야 합니다 . 확인하려면 다음을 따르세요.

Microsoft Edge

참고: Microsoft Edge는 플러그인을 지원하지 않습니다.

Internet Explorer

  1. 오른쪽 상단 모서리에 있는 기어 아이콘을 클릭합니다.
  2. 추가 기능 관리 를 클릭합니다.
  3. 팝업창에서 왼쪽에 있는 ‘표시’ 풀다운 메뉴를 클릭합니다.
  4. 모든 추가 기능을 선택합니다.
  5. 팝업창 오른쪽에 있는 스크롤 막대를 맨 아래까지 스크롤합니다.
  6. Roblox Corporation 밑에 있는 ‘Launcher Class’가 활성화되어 있는지 확인합니다.

Mozilla Firefox

  1. Firefox 창의 오른쪽 상단에 있는 메뉴 버튼을 클릭합니다.
  2. 메뉴에서 부가 기능 버튼을 클릭합니다.
  3. 나타난 창에서 플러그인을 클릭합니다.
  4. ‘Roblox 실행 플러그인’을 찾아서 오른쪽 드롭다운 메뉴에 있는 항상 허용을 선택합니다.
  5. 추가 기능 관리 탭을 닫고 Roblox Game을 실행합니다.

Google Chrome

  1. 주소창에 다음과 같이 입력합니다. ‘chrome://plugins/’
  2. Enter 를 클릭합니다.
  3. 'Roblox 실행 플러그인'이 허용되어 있는지 확인합니다.

참고: 42 버전부터 Chrome은 더 이상 플러그인을 지원하지 않습니다. 이 단계는 이전 버전의 Chrome에서만 작동합니다.

Apple Safari

  1. Safari를 열고 오른쪽 상단에 있는 기어 아이콘을 클릭합니다
  2. 환경 설정을 클릭하고 보안 탭을 선택합니다.
  3. 보안 탭 밑에 ‘Plug-ins, Java, and JavaScript’가 활성화되어 있는지 확인합니다.
