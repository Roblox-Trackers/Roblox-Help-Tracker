컴퓨터 그래픽 문제

Roblox에서 그래픽 문제가 발생하는 경우 아래 정보를 참고해서 해결해 보세요.

그래픽 오류를 나타내는 일반적인 현상

  • 체험을 실행하려고 할 때 ‘그래픽 카드가 너무 오래되었습니다’라는 오류 발생
  • 게임 플레이 중 Roblox 작동이 중단되고 그래픽 오류 보고 발생
  • ‘그래픽을 초기화하지 못했습니다’라는 메시지 수신
  • 이상하게 보이는 텍스처, 이미지 뒤틀림 또는 알아보기 어려운 문구와 같은 체험 내의 비정상적인 디스플레이

이러한 현상은 왜 나타나나요?

  • PC에 DirectX 10 이상의 기능 지원이 없을 경우
  • 그래픽 카드에 최신 드라이버가 설치되어 있지 않을 경우
  • 그래픽 카드가 Roblox를 지원하지 않을 경우(극히 드문 경우임)
  • 플레이어가 Roblox 그래픽 설정을 변경한 경우

문제를 해결하려면 어떻게 해야 하나요?

아래의 각 단계를 문제가 해결될 때까지 하나씩 순서대로 수행해 보세요.

Roblox 그래픽 재설정하기

이 단계는 Roblox가 작동 중단 없이 실행되는 경우에만 적합한 방법입니다.

    1. Roblox Studio를 엽니다.
    2. 파일 메뉴를 열고 설정 을 클릭합니다.
    3. 확인 을 클릭합니다.
    4. 윈도우 팝업창 좌측 하단에 있는 모두 다시 설정 버튼을 클릭합니다.

지원되는 DirectX 기능 수준 확인하기

지원되는 DirectX 기능 수준을 다음과 같이 확인해 보세요.

    1. Windows 키 + R 을 눌러 실행 프롬프트를 불러옵니다.
    2. ‘dxdiag’(작은따옴표 제외)를 입력하고 Enter 키를 누릅니다.
    3. DirectX 창의 패널 보기 탭에서 디스플레이 또는 디스플레이 1, 디스플레이 2 (사용 가능한 경우)를 클릭합니다.
      • Windows 8.1/10의 경우 기능 수준 을 찾습니다. Windows 7의 경우 DDI 버전 을 찾습니다.

‘DDI 버전’ 또는 ‘기능 수준’ 줄에 ‘10’ 이상이 표시되어 있지 않으면 그래픽 카드가 DX10 이상을 지원하지 않는 것이므로 응용 프로그램에 액세스할 수 없습니다.

디스플레이의 색상 품질 낮추기

그래픽 카드가 일반 데스크톱 작업에 사용하는 색상 품질의 3D 렌더링을 지원하지 않을 수 있습니다. 비트 수준을 16비트로 낮추어 문제 해결에 도움이 되는지 확인해 보세요.

그래픽 카드 드라이버 업그레이드하기

Robox가 현재 사용하고 있는 그래픽 카드가 무엇인지 확인하려면 다음을 수행하세요.

    1. Roblox Studio를 엽니다.
    2. Studio 설정 을 엽니다.
    3. 진단 을 클릭합니다.
    4. 아래로 스크롤해 GfxCard 라고 표시된 줄을 찾습니다.
    5. 현재 Roblox에서 사용 중인 그래픽 카드 목록을 확인합니다.

그래픽 카드 제조업체에서 제공하는 지원 페이지를 방문해 특정 모델에 맞는 최신 드라이버를 다운로드하세요.

호환 모드 확인하기

그래픽이 최신 버전이 아니지만 시스템이 최소 사양 요구 사항을 충족한다는 메시지가 표시되면, 응용 프로그램이 호환 모드에서 실행 중인지 확인해 보세요.

    1. Roblox 또는 Roblox Studio 애플리케이션을 마우스 오른쪽 버튼으로 클릭하고 속성 을 선택합니다.
    2. 호환성 탭을 선택합니다.
    3. 이 프로그램을 실행할 호환 모드 상자가 선택되어 있는 경우, 선택을 해제합니다.

그래픽 카드 제조업체에서 제공하는 지원 페이지를 방문해 특정 모델에 맞는 최신 드라이버를 다운로드하세요.
