Mac 관련 문제 해결 방법

제거 또는 재설치하기

최근에 Mac을 재설치한 적이 없을 경우, 다음 두 가지 도움말의 하단에 위치한 단계에 따라 프로그램을 삭제하고 재설치해 보세요.

  • 삭제하기
  • 재설치하기

기타 방법

삭제 후 재설치를 완료했는데도 문제가 해결되지 않는 경우 아래 방법을 시도해 보세요.

  • Roblox 앱은 10.10(Yosemite) 이상에 설치되지만, Roblox Studio는 Mac OS 10.13(High Sierra) 이상이어야 설치가 가능합니다.
  • 현재 Apple Silicon / M1 칩은 지원됩니다.
  • 현재 Safari 웹브라우저를 사용 중이며 문제가 발생한 경우, Internet Explorer, Firefox 또는 Google Chrome 등 Roblox 에서 지원하는 브라우저 중 하나를 사용해 보세요.
  • Roblox 아이콘이 현재 Dock에 있고, Roblox 응용 프로그램이 애플리케이션 폴더에 위치해 있는지 확인하세요.
  • 웹사이트에서 게임을 하는 데 문제가 있는 경우 Dock에서 프로그램을 열어 보세요.
  • 설정 파일을 삭제하여 Roblox 설정을 완전히 재설정하세요. 라이브러리 -> 환경 설정 으로 이동하여 모든 Roblox.com 파일을 삭제하면 됩니다. Studio 또는 플레이어를 다시 열면 모든 프로그램이 다시 설치됩니다.

위의 사항들을 모두 따른 후에도 계속 오류가 발생한다면 Roblox 고객 지원 센터 에 연락하여 다음 세부 정보를 알려주세요.

  • Roblox 사용자 이름
  • 수신된 모든 오류 메시지
  • Mac의 모델 이름
  • 맥 OS 버전
  • 문제 재현 단계

더불어 보고된 문제를 더 잘 이해할 수 있게 도움 될 만한 추가 세부 정보와 Mac 또는 네트워크에 대한 최신 변경 사항을 포함시키기 바랍니다.
