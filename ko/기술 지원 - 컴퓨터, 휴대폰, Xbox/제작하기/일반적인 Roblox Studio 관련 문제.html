일반적인 Roblox Studio 관련 문제

본 도움말에서는 다음과 같은 문제를 해결하는 데 효과적인 것으로 입증된 몇 가지 방법을 안내합니다.

  • Roblox Studio에 로그인이 안됩니다.
  • 비어 있는 스카이 박스 또는 Place1이라는 항목만 불러오도록 장소를 편집하려고 합니다.
  • 비어 있어서는 안되는 Studio의 섹션이 비어 있습니다.
  • Roblox에 게시하는 데 어려움을 겪고 있습니다.
  • Roblox Studio에 이미 로그인했는데도 로그인하라는 메시지가 나타납니다.
  • 본 단계들은 Studio 이외의 일부 문제에도 도움이 될 수 있습니다 . 예를 들면 로그인을 했음에도 불구하고 게임이 게스트 모드로 시작되는 경우 등입니다 .
  • 기타: Studio에서 다른 문제가 발생하는 경우 아래 단계를 따르면 문제 해결에 도움이 될 수 있습니다.

Windows 사용자의 경우

아래 각 단계를 하나씩 순서대로 따라해 보세요. 먼저 시도한 해결책이 효과가 없다면 다음 방법으로 넘어가 또다시 시도하세요.

재부팅하기

먼저 컴퓨터를 재부팅하고 게임을 다시 실행합니다. 때로는 이 작업이 Studio뿐 아니라 Roblox 내 그 외 다른 문제의 해결책이 되기도 합니다.

Windows에서 인터넷 옵션 재설정하기

Internet Explorer를 사용하지 않더라도 이 브라우저의 인터넷 옵션을 재설정하면 문제가 해결될 수 있습니다. 본 단계는 Internet Explorer를 위한 것이지만, 완료 후 원래 사용하던 브라우저로 돌아가도 됩니다. 방법은 다음과 같습니다.

    1. Internet Explorer를 엽니다.( 다른 브라우저 제외 )
    2. 기어 아이콘에서 인터넷 옵션 을 엽니다.
    3. 고급 탭을 엽니다.
    4. 원래대로 버튼을 클릭합니다.

설정을 마치고 나면 변경 사항을 적용하기 위해 Internet Explorer를 닫습니다.

프록시 설정 확인하기

위의 단계를 수행하고 난 후에도 Roblox Studio 문제가 해결되지 않았다면, 컴퓨터가 프록시 서버를 통해 연결되지 않도록 확인하는 것이 도움이 될 수 있습니다. 방법은 다음과 같습니다.

Windows 7 또는 이전 버전

    1. 시작 메뉴로 이동합니다.
    2. ‘ 프록시 ’ 라고 검색하면 ‘ 프록시 서버 구성 ’ 이 나타납니다 . 이를 클릭합니다 .
    3. LAN 설정 을 클릭하고 해당 탭의 모든 체크 표시를 클릭 해제합니다.
    4. 확인 을 클릭합니다.

Windows 8 또는 8.1

    1. 화면의 왼쪽 하단을 클릭하여 타일 모양의 시작 화면으로 이동합니다.
    2. ‘ 프록시 ’ 라고 검색하면 ‘ 프록시 서버 구성 ’ 이 나타납니다 . 이를 클릭합니다 .
    3. LAN 설정 을 클릭하고 해당 탭의 모든 체크 표시를 클릭 해제합니다.
    4. 확인 을 클릭합니다.

Windows 10

    1. 키보드의 Windows 키 + S를 누릅니다. 또는 Cortana의 ‘Ask Me Anything’ 바를 작업 표시 줄에서 사용할 수 있으면 클릭합니다.
    2. ‘ 프록시 ’ 라고 검색하면 ‘ 프록시 서버 구성 ’ 이 나타납니다 . 이를 클릭합니다 .
    3. ‘설정 스크립트 사용’ 및 ‘프록시 서버 사용’이 해제되어 있는지 확인합니다.
    4. 아래로 스크롤하여 저장 을 클릭합니다.

Windows 시스템 시계 확인하기

흔한 일은 아니지만, 컴퓨터 시계가 부정확하면 여러 가지 문제가 발생할 수 있습니다. 만약 이것이 원인이라면, 시스템 시계 시간을 원래대로 변경해주면 문제점이 해결될 수 있습니다. 해당 작업은 수동으로 할 수 있지만, 서버로 자동 업데이트되도록 설정하여 다시는 문제가 발생하지 않도록 방지할 수 있습니다. 방법은 아래와 같습니다.

Windows 8 또는 이전 버전

    1. 우측 하단에 있는 Windows 시계 위에서 마우스 오른쪽 버튼을 클릭합니다.
    2. 날짜 혹은 시간 조정 을 클릭합니다.
    3. 인터넷 시간 탭을 선택합니다.
    4. 여기에서 인터넷 서버 시간과 날짜를 동기화할 수 있습니다.

Windows 10

    1. 우측 하단에 있는 Windows 시계 위에서 마우스 오른쪽 버튼을 클릭합니다.
    2. ‘자동으로 시간 설정’이 켜져 있고 올바른 시간대를 선택했는지 확인하세요.

Mac 사용자의 경우

Roblox 클라이언트는 Mac OS 10.7(Lion) 이상 및 Intel 전용 Mac에 설치 가능하지만, Roblox Studio는 10.10(Yosemite) 이상이어야 설치가 가능합니다.

Mac에서 이 문제를 해결하려는 경우, 재설치를 해야 합니다. 재설치에 도움이 필요하다면 여기를 클릭 하세요.

앞으로의 문제 최소화하기

Roblox를 설치하거나 업데이트하기 전에 항상 컴퓨터에서 모든 프로그램을 종료하세요. Windows 사용자라면 Internet Explorer가 최신 버전으로 업데이트되어 있는지 확인하는 것도 잊지 마세요.

위의 사항들을 모두 따른 후에도 계속 오류가 발생한다면 Roblox 지원 센터 에 연락하여 다음 세부 정보를 알려주세요.

  • Roblox 사용자 이름
  • 수신된 오류 메시지
  • 기기 모델 이름
  • OS 버전
  • 문제 재현 단계
  • 영향을 받은 특정 프로젝트 링크

더불어 보고된 문제를 더 잘 이해할 수 있게 도움 될 만한 추가 세부 정보와 기기, 인터넷 연결 또는 Studio 프로젝트에 대한 최신 변경 사항을 포함시키기 바랍니다.
