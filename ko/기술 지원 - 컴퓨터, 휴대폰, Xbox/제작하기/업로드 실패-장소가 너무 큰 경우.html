업로드 실패/장소가 너무 큰 경우

스튜디오에서 업로드에 실패했다는 메시지를 받을 때마다, 이는 일반적으로 장소가 너무 크거나 항목/스크립트가 너무 많은 결과입니다. 테스트하려면 다음을 시도해 보시기 바랍니다:

  1. 먼저 로블록스 스튜디오에서 열고 File (파일) 메뉴 > Save (저장) 옵션을 사용하여 컴퓨터에 위치를 저장합니다
  2. File (파일) 메뉴에서 New (새로) 만들기를 선택하여 로블록스 스튜디오를 열고 새 체험
    만들기
  3. 새 플레이스에 벽돌을 추가하고 File (파일) 메뉴 > Publish to Roblox As (다른 이름으로 Roblox에 게시)
  4. 현재 문제가 있는 장소를 선택하여 비어 있는 새 장소를 업로드합니다.
  5. 작동합니까?

만약에 이것이 효과가 있다면, 여러분은 집을 더 작게 만들거나 그 안에 있는 몇몇 물건들을 줄여야 할 겁니다.
이렇게 하려면 컴퓨터에 저장한 버전을 편집하고 위의 빈 플레이스에서 했던 것과 동일하게 게시하면 됩니다. 또는 플레이스를 구성하고 문제가 발생하지 않았을 수 있는 이전 버전으로 되돌릴 수 있습니다. 더 자세한 내용을 확인하려면 체험 설정 을 확인해보세요.
