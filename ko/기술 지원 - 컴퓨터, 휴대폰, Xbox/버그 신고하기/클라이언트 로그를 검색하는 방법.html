클라이언트 로그를 검색하는 방법

Rblox직원은 문제를 조사하는 데 도움이되도록 고객 지원 메일 또는 DevForum의 게시물을 통해 클라이언트 로그를 요청하는 상황이 있을 수 있습니다.이 안내는 이런 로그들을 검색하는 방법에 대한 설명으로 가득합니다.

참고: Roblox 직원이 요청하지 않으면 이 로그들을 검색하거나 보내지 마십시오.

클라이언트 로그를 검색하기

클라이언트 로그를 검색하는 프로세스는 사용중인 운영 체제에 따라 다릅니다. 다음과 같은 정보는 휴대폰에는 적용되지않은 것에 주의 하십시오.

  1. Roblox Studio 를 엽니다:

    Windows XP / Vista / 7 / 10: Start > All Programs (or All Apps) > Roblox > Roblox Studio

    Windows 8: 왼쪽 하단에 있는 Windows 로고를 클릭하고 타일 모양의 시작 화면이 나타날 때 Roblox Studio를 입력합니다. 화면 오른쪽에 나타나는 검색 리스트에서 Roblox Studio를 선택하십시오.

    Mac: Command + Space Bar > Type in RobloxStudio > Enter

  2. 파일을 클릭하여 설정 을 엽니다 > 설정

  3. 왼쪽 창에서 네트워크(Network) 를 클릭하십시오 . 표시되지 않으면 Roblox Studio를 닫았다가 다시 엽니다.

  4. Diagnostics 이란 아래에,다음 확인란이 선택되어 있는지 확인하십시오:

    PrintEvents
    PrintInstances
    PrintPhysicsErrors
    PrintProperties
    PrintTouches

  5. Roblox를 닫다.

  6. 키보드의 WINDOWS KEY하고 R와 함께 누르십시오.

    Mac이용 : Macintosh HD > Users > [your user profile] > Library > Logs > Roblox

    참고: Mac의 사용자는  7번의 단계를  껀너뛰어도 됩니다.

  7. Run 이라는 상자가 나타납니다. 따옴표 없이 다음을 복사하여 붙여 넣거나 입력하십시오.

    Windows Vista / 7 / 8 / 10를 사용하면 : "%localappdata%\Roblox\logs"

    Windows XP를 사용하면 : "%USERPROFILE%\Local Settings\Application Data\Roblox\logs"

  8. 폴더에 있는 모든 항목을 삭제합니다.

  9. Roblox를 게임하는 일반적인 방법으로 한 번 해보십시오.

  10. 작동하지 않으면 Roblox를 닫고 6번 단계와 7번 단계를 다시 해보십시오.

    현재는 폴더에 여러 파일이 있습니다.  로그를 필요로하는 문제 상황을 확실히 확인하려면 info@roblox.com 으로 보내주십시오.
