기프트 카드 크레딧 사용하기

Roblox 기프트 카드는 Roblox.com/redeem 의 브라우저에서만 사용할 수 있습니다. Roblox 모바일 앱이나 비디오 게임 콘솔에서는 사용할 수 없습니다.

멤버십 또는 Robux 구매 페이지

계정에 충분한 Roblox 크레딧이 있다면, Roblox 크레딧은 Roblox 프리미엄 멤버십 또는 Robux를 구매하는 데 사용할 수 있습니다. 다음 3곳에서 Roblox 크레딧을 전환할 수 있습니다.

  • Roblox 기프트 카드 사용 페이지 (사용 가능한 크레딧 아래)
  • 멤버십 페이지
  • Robux 페이지
  1. 구매하려는 제품을 선택합니다.
  2. 구매 를 클릭합니다.
  3. 확인이 나타납니다.
  4. Roblox 크레딧이 남아 있는 경우 Robux를 더 구매할 것인지 묻습니다. (Robux가 계정에 충분한 경우)
      • 구매를 클릭하면 남은 Roblox 크레딧이 Robux로 전환됩니다.
      • 취소를 클릭하면 사용할 준비가 될 때까지 Roblox 크레딧이 계정에 유지됩니다.

남은 크레딧을 Robux로 전환하기

사용하다 남은 크레딧을 Robux로 전환할 수 있습니다.

  1. 브라우저에서 기프트 카드를 사용한 Roblox 계정에 로그인합니다.
  2. 오른쪽 상단 모서리에 있는 기어를 클릭하여 설정 페이지로 이동합니다.
  3. 청구 탭을 선택합니다.
  4. Robux로 변환 버튼을 클릭합니다.
  5. 확인 화면이 표시됩니다. 전환을 완료하려면 교환 을 클릭하고 크레딧을 전환하지 않으려면 취소 를 클릭하세요.
  6. 현재 페이지를 새로고침하거나 새 페이지로 이동하면 Robux 잔액이 업데이트됩니다.
