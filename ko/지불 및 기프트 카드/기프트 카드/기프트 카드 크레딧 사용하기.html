기프트 카드 크레딧 사용하기

Roblox 기프트 카드는 Roblox.com/redeem 브라우저에서만 사용할 수 있습니다. Roblox 모바일 앱이나 비디오 게임 콘솔에서는 사용이 불가능합니다.

멤버십 또는 Robux 구매 페이지에서 사용 방법

계정에서 Roblox 크레딧을 적용하면 Roblox Premium 멤버십 또는 Robux를 구매할 수 있습니다. Roblox 크레딧은 아래의 3곳에서 전환할 수 있습니다.

  • Roblox 기프트 카드 사용 페이지 (사용 가능한 크레딧 아래)
  • 멤버십 페이지
  • Robux 페이지
  1. 구매하려는 제품을 선택합니다.
  2. 구매 를 클릭합니다.
  3. 확인 메시지가 나타납니다.
  4. Roblox 크레딧이 남아 있는 경우, Robux를 더 구매할 것인지 묻는 메시지가 나타납니다(계정에 충분한 크레딧을 보유하고 있는 경우).
      • 구매를 클릭하면 남은 Roblox 크레딧에서 Robux로 전환됩니다.
      • 취소를 클릭하면 계정에 크레딧은 계속 그대로 유지됩니다.

남은 크레딧을 Robux로 전환하기

그 대신에, 이 크레딧을 Robux로 전환할 수 있습니다.

  1. 브라우저에서 기프트 카드를 사용했던 Roblox 계정에 로그인합니다.
  2. 우측 상단에 있는 기어 아이콘을 누르고 설정 페이지로 이동합니다.
  3. 청구 탭을 클릭합니다.
  4. Robux 전환 버튼을 클릭합니다.
  5. 확인 화면이 나타나고 전환하기를 완료 하려면 사용 을 클릭합니다. 크레딧을 전환하지 않으면 취소 를 선택합니다.
  6. 현재 페이지를 새로 고치거나 새 페이지로 이동하면 Robux 잔액이 업데이트됩니다.
