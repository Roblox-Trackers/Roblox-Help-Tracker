How to Report Rule Violations

Roblox is a great community of players and developers of many ages, and we like to encourage a safe and fun environment for everyone. If you see other users breaking the rules, we have tools in place to help you report any inappropriate behavior.

For information about blocking another user from contacting or chatting, read the Blocking A User help article .

The Report Abuse feature is the best way to let us know about rule-breakers and bad content. It sends reports directly to our moderators and provide a way for us to keep track of repeat offenders so further steps can be taken against their account. When reporting abuse, be sure to report the specific Chat, Private Message, Experience or other Content that violates Roblox rules. If you just report a user from their profile page, our moderators won't be able to see what they did wrong.

Reporting a user in-experience

Use the Report Abuse feature in an experience to report the specific chat or other content that violates Roblox rules. It sends the information directly to our moderators and allows them to see what the other user is doing wrong.

  1. Select the menu button, located at the upper left of the screen. This icon looks like the Roblox logo stacked on the top left corner of Roblox player.
  2. Select the flag icon located next to a user's name, or select the Report tab at the top of the menu.
  3. For Experience or Person? select the Person tab.
  4. Select the Which Person? pull-down menu, and find the username to report.
  5. Select the Reason for Abuse? pull-down menu, and find the inappropriate action.
  6. Additional details can be written in the Short Description box.
  7. Select Submit to finalize.

If the experience itself is breaking the rules, you can report it while playing that experience. Open the Report tab as described in Step 2 above, then choose Experience instead of Player to complete the report form.

Reporting violations outside of experiences

If you find users or content breaking the Roblox rules, use the Report Abuse feature located throughout the apps to report this to our moderators. There are different ways to report based on what is being reported, but anything you see can be reported.

Reporting an experience

To report an entire experience, use the red Report Abuse button in the lower right-hand corner of the Description box on the experience's About page.

Reporting other content

To report an item in the catalog or library (such as a shirt, experience pass, or model) that is breaking the rules, select the three dots button in the upper right hand corner of the item's information box. Select Report item and fill out the form.

Reporting app chat

To report chat made by another user in the browser app , select the gear icon in the upper-right corner of the chat window to open the Chat Details. Select the button next to the user name that looks like the three dots, then Report . Select the red Report button to continue to the report form then fill it out and select the green Report Abuse button.

It's a bit different in the mobile apps . Select the information (i) icon in the upper-right hand corner of the screen, select the user from the list of participants in the conversation or the (...) icon next to their name, and then select Report Abuse .

Reporting a user-created advertisement

To report an Ad made by another user on the site, select Report below the right hand side of the Ad image. When the page reloads, fill out the form and select the green Report Abuse button.
