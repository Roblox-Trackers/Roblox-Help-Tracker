學習如何創建體驗

Creator Documentation 是對於學習用 Roblox Studio 創建體驗並查找大多數問題解答的好地方。您可以創建類似以下的體驗。

  • Beyond the Dark
  • City People Cars

下面的連結能協助您入門！

入門

  • Intro to Roblox Studio : 關於如何使用 Studio 創建你第一個體驗的課程。
  • Getting Started for Developers : 創建體驗的 創作者 提供的資源文章

腳本

  • Intro to Coding and Game Design : 學習如何編碼您的第一個體驗的課程。
  • Lua Learning : 是個互動的 Roblox 體驗，展示如何使用 Lua 編寫代碼。
  • Roblox Developer 101 : 教您如何編碼和建造體驗的教程。

設計

  • Animate in Roblox : 學習動畫並研究身體運動的教程。
  • Galactic Speedway : 學習 三D 模型和編碼的課程。
