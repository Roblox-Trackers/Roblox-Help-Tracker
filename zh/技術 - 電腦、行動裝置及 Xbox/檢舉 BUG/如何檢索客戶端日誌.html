如何檢索客戶端日誌

在某些情況下，Roblox員工可能會通過客戶關懷電子郵件或DevForum公佈請求客戶端日誌，以幫助調查某個問題。本文描述如何檢索這些日誌。

注意 : 除非Roblox員工要求，否則請不要檢索或發送這些日誌。

檢索客戶端日誌

檢索客戶端日誌的過程因您使用的操作系統而異。請注意以下信息不適用於移動設備。

  1. 打開 Roblox Studio :

    Windows 7 / 10: Start >所有程序（或者所有應用程序） > Roblox > Roblox Studio

    Windows 8: 點擊左下角的Windows標誌，並在平舖的”開始“屏幕出現時輸入 Roblox Studio 。從屏幕右側的搜索列表中選擇 Roblox Studio 。

    Mac: Command + Space Bar >輸入 RobloxStudio > Enter

  2. 通過點擊”設置“>”設置“以打開” 設置 “

  3. 在左側窗格中，點擊” 網絡 “ 。如果沒有顯示，關閉並和重新打開Roblox Studio。

  4. 在 " 診斷 " 下面，確保選中這些複選框：

    PrintEvents
    PrintInstances
    PrintPhysicsErrors
    PrintProperties
    PrintTouches

  5. 關閉Roblox

  6. 同時按下鍵盤上的WINDOWS鍵和R鍵。

    適用於Mac: Macintosh HD > Users > [your user profile] > Library > Logs > Roblox

    注意：Mac用戶跳過步驟7

  7. 將出現一個名為 Run 的框。複製粘貼或鍵入以下不帶引號的內容:

    如果使用Windows 7 / 8 / 10: "%localappdata%\Roblox\logs"

    刪除文件夾中的所有內容。

  8. 試著用正常的方式玩Roblox。

  9. 當它不起作用時，請關閉Roblox並重複步驟6和7。

    現在該文件夾中將有幾個文件。 請發送給他們作為對您的支持消息的回复，確保確保參考了需要日誌的問題情況。
