一般Roblox Studio問題

这文章提出一些有效方法解决以下問題：

  • 您嘗試登錄Roblox Studio但無法啟動，
  • 您試著編輯一個地方但它加載了一個空白的Skybox或叫做Place1
  • Studio的一部分不應该是空的。
  • 在Roblox上發佈時遇到了問題
  • 當你已經在線，Roblox Studio還是要求登錄
  • 這些步驟能助於解決一些與非Studio相關的問題，列如玩家已登錄，也可以作為訪客進入遊戲
  • 其他：如果您遇到與Studio額外問題，則以下步驟也可能有助於解決問題

如何操作- Windows

請使用按順序嘗試下面的每個步驟。如果一個解決方案無效，請繼續下一步！

重啟

首先重新啟動電腦並嘗試播放。有時這会解決Roblox或Roblox Studio相關問題。

在Windows中重置您的互聯網選項

即使您不使用Internet Explorer，通過你目前的瀏覽器重置Internet選項仍可解決您的問題。請注意，雖然這些步驟需要使用Internet Explorer來完成它，一旦完成后，您照样可以回到你想用的瀏覽器。請嘗試以下操作：

    1. 打開Internet Explorer（這指定的瀏覽器）
    2. 打開齒輪圖標> Internet選項
    3. 打開 高級選項卡
    4. 點擊 重置 按鈕

重置後，請關閉Internet Explorer以使更改生效。

查观您的代理設置

如果上述步驟不能解決您的Roblox Studio問題，您需要確保電腦不是在嘗試通過代理服務器進行連接。請按照以下步驟操作：

Windows 7或更早版本

    1. 轉到 Start “開始”菜單
    2. 搜索“代理”和“配置代理服務器”將会出現，并點擊它
    3. 點擊 LAN “ 局域網”設置 ，然後取消選中該選項卡中的所有復選標記
    4. 點擊 OK

Windows 8 or 8.1

    1. 點擊屏幕左下角，以便進入平鋪開始屏幕
    2. 输入Proxy“代理”然后“配置代理服務器”將会出現，并點擊它
    3. 點擊 LAN“ 局域網”設置 ，然後取消選中該選項卡中的所有復選標記
    4. 點擊 OK

Windows 10

    1. 按下鍵盤上的Windows鍵 +S鍵。或者，如果Cortana的“問我任何事情”欄在您的任務欄中可用，您可以單擊它
    2. 搜索“代理”然后“配置代理服務器”將会出現，并點擊它
    3. 確保“使用安裝腳本”和“使用代理服務器”設置為關閉
    4. 向下滾動並點擊 保存

查观 您的Windows系統時鐘

這聽起來很奇怪，但實際上各種問題是由於Windows系統時鐘的時間或日期被設置成錯誤而導致的。如果這是原因，那麼糾正一切就應該解決情況。雖然可以手動完成它，但也可以將其設置為使用服務器自動更新，從而確保它不會滑落並再次引發問題。以下是如何做到這一點：

Windows 8或更早版本

    1. 右鍵點擊右下角的窗口時鐘
    2. 點擊 調整日期/時間
    3. 選擇 互聯網時間 選項卡
    4. 在這裡，您應該能夠將您的時間和日期與互聯網服務器進行同步

Windows 10

    1. 在這裡，您應該能夠將您的時間和日期與互聯網服務器進行同步
    2. 確保“自動設置時間”已打開，並且您已選擇了正確的時區

如何操作 - Mac

Roblox客戶端將安裝在10.7 (Lion)及以上版本 - 只能安裝在Intel版本的Mac電腦上，而Roblox Studio將安裝在Mac OS 10.10 (Yosemite)及以上版。

要在Mac上解決此問題，請重新安裝。如需幫助，請 點擊此處 。

最大限度地減少未來的問題

始終確保在安裝或更新Roblox之前，您的電腦上的所有程序都已關閉。對於Windows，請確保Internet Explorer與最新版本保持同步。
