上傳失敗或地點太大

無論何時您在Studio中收到關於上傳失敗的消息時，通常都是由於您的位置太大或者其中包含太多項目/腳本。要進行測試，請嘗試以下操作：

  1. 首先打開Roblox Studio，然後使用 File 菜單> Save 選項將您的地點儲存到電腦中
  2. 通過 File 菜單中選擇 New ，使Roblox Studio保持打開狀態並創建新體驗
  3. 將磚塊添加到新地點並選擇 File 菜單> Publish to Roblox As
  4. 選擇您目前遇到困難的地點，然後上傳新的空白地點。
  5. 它起作用嗎？

如果這個方法有效，那麼您可能需要將您的地點縮小，或者減少其中的一些項目。您可以通過編輯儲存到電腦上的版本，並像我們對上面空白部分所做的那樣發布它來實現這一點。或者，您可以配置您的地點，將它恢復到還沒有遇到問題的早期版本。如需協助，請 點擊此處 。
