Roblox 啟動器插件

若 Roblox 無法運行或提供 Roblox 下載提示，即使它已經安裝好了。若要進行檢查，請執行以下操作：

Mozilla Firefox

  1. 點擊 Firefox 視窗 右上方的菜單按鈕
  2. 在菜單中，點擊「 加載項」 按鈕
  3. 在出現的 視窗 中，點擊 「 插件」
  4. 查找 「 Roblox Launcher Plugin 」 ，然後在右邊的下拉菜單中選擇「 總是允許」
  5. 關閉 「 加載項管理器 」 標籤 並啟動 Roblox 體驗

Apple Safari

  1. 打開 Safari 並點擊右上角的「齒輪」 圖示
  2. 點擊首選項，然後點擊「 安全」 標籤
  3. 在 「 安全 」 標籤 下，確保啟用插件，Java 和 JavaScript

Google Chrome

注意：從版本 42 開始，Chrome 不再支援插件。

Microsoft Edge

注意：Microsoft Edge 不支援插件
