如何安裝和玩 Roblox

如何玩 Roblox

  1. 安裝 Roblox 播放器
    • 使用 Microsoft Edge、Firefox 或 Chrome 等任何現代瀏覽器存取 Roblox 網站
    • 登入 Roblox 後，存取任何體驗並點擊綠色的 開始 按鈕
    • 將出現一個彈出窗口，通知您 Roblox Player 正在安裝
  2. 透過任何現代瀏覽器（例如​​ Microsoft Edge、Firefox 或 Chrome）啟動播放器
    • 前往 Roblox 網站上的任何遊戲
    • 選擇 播放
    • 可能會出現第二個彈出窗口，要求您選擇一個程式。 選擇 Roblox 並確認。 如果您選擇記住您的選擇，則以後無需執行此操作

如何在 MacOS 上通過 Firefox 安裝和運行 Roblox

1. 安装 Roblox 播放器

  • 使用 Firefox 訪問 Roblox 網站
  • 登錄 Roblox 後，訪問任何體驗並點擊綠色的 播放 按鈕
  • 將出現一個彈出窗口，通知您 Roblox Player 正在安裝

2. 啟動播放器

  • 勾選 始終使用此應用程式開啟 roblox-player 連結 ，然後點擊 選擇 開啟 Finder 窗口
  • 在 Finder 視窗中，使用鍵盤快速鍵 CMD+SHIFT+G 調出路徑名稱輸入
  • 在路徑名輸入中，輸入 /Applications/Roblox.app/Contents/MacOS/Roblox.app ，然後按 RETURN
  • 然後在 Finder 視窗中，選擇 Roblox.app，然後按一下 開啟
  • 在「選擇應用程式」對話方塊中，選擇 Roblox.app 並點選 開啟連結
  • Roblox 應該正在運行，並且將來應該可以點擊播放按鈕來開啟 Roblox 體驗。

如何安裝 Roblox Studio

您需要安裝 Roblox Studio 才能創建您自己的 Roblox 體驗。 以下是您可以這樣做的方法。

  1. 造訪 Roblox 網站 並登入您的帳戶
  2. 登入 Roblox 後，造訪任何 您的 體驗，點擊右上角的 ，然後選擇 編輯
  3. 將出現一個彈出窗口，通知您 Roblox Studio 正在安裝
  4. 安裝後，Studio 將自動打開
