如何重新安裝 Roblox 軟體？

重新安裝 Roblox 後，可以解決某些技術問題。請確保在重新安裝之前嘗試重新開機。以下是重新安裝的方式：

Windows 使用者：

重設 Windows 的網際網路選項

重設網際網路選項有助於防止它們與 Roblox 不同步。如果這樣做可以解決您的問題，那便足夠了。雖然這些步驟使用的是 Internet Explorer，但還將解決其他瀏覽器（例如 Chrome 或 Firefox）中發生的許多問題。以下是操作方法：

    1. 開啟 Internet Explorer
    2. 開啟裝備圖示 > 網際網路選項
    3. 開啟 進階 標籤
    4. 按一下 重設 按鈕
    5. 關閉 Internet Explorer 瀏覽器
    6. 嘗試使用您最愛的網頁瀏覽器來使用 Roblox。

重新安裝 Roblox

如果重設網際網路選項沒有幫助，請嘗試重新安裝 Roblox。重新安裝應覆寫並取代所有已刪除或損壞的檔案。執行重新安裝時，請確保所有其他程式均已關閉。

    • 針對 Chrome 或 Edge 重新安裝
    • 針對其他瀏覽器重新安裝

解除安裝 Roblox

在大多數情況下，重新安裝程式之前無需先將其解除安裝。如果重新安裝不起作用，則可能需要先解除安裝，然後才能重新安裝。請按照 適用於 Windows 的 Roblox 解除安裝指示 進行操作。確保在解除安裝後重新啟動電腦，然後再重新安裝。

清除 Internet 暫存檔案

在任何解除安裝或重新安裝程序中，清除瀏覽器的 Internet 暫存檔案都是一個重要步驟。這將有助於解決任何可能已出現的損毀問題。請按照 清除 Internet 暫存檔案和 Cookie 部分中的指示進行操作。完成後，請重複上述重新安裝指示。

變更防火牆／安全程式設定

許多防毒軟體或網際網路安全性軟體會試圖控制哪些程式可以存取網際網路。出於謹慎考慮，這些程式會出錯，依預設不允許任何程式存取。閱讀 防火牆和路由器問題說明文章 中的資訊，以獲取允許 Roblox 存取的說明。重新設定防火牆後，請重複重新安裝步驟。

Mac 使用者：

解除安裝

確保您已完全退出 Roblox 程式。請按照 適用於 Mac 的 Roblox 解除安裝指示 進行操作。

重新安裝

開啟 Safari，登入 Roblox 開始玩遊戲。系統將提示您安裝應用程式。請按照螢幕上的指示進行操作。如果安裝不會自動進行，請執行以下操作：


    1. 前往 Finder > 今天 ，然後按一下 Roblox.dmg，隨即為您執行安裝程式並將 Roblox 圖示放置在您的應用程式中
    2. 從您的應用程式中選擇它，將隨即進入您的系統匣
