我登入了，但我在體驗中以客人的身份出現

如果您發現即使您已登入了您的帳號，但還是以客人的身份進入體驗中， 有幾件事需要調查一下。

怎麼辦- Windows

請按順序嘗試下面的每一步。如果一個解決方案不起作用，那就繼續下一個步驟！

重啟

首先重新重啟您的電腦並嘗試玩。有時，這可能是解決 Roblox 或 Roblox Studio 相關問題所需要的全部內容。

檢查您的 Windows 系統時鐘

這聽起來很奇怪，但實際上由於 Windows 系統時鐘被設定為錯誤的時間或日期而導致各種問題。如果這就是原因，那麼簡單地糾正所有的錯誤就能解決問題。雖然這可以手動完成，但您也可以將其設定為使用服務器自動更新，從而確保它不會落後並再次引發問題。以下是如何做到這一點：

Windows 8.1 和更早版本

1. 右鍵點擊右下角的 Windows 時鐘

2.點擊 調整日期/時間

3.選擇 互聯網時間 選項卡

4. 在這裡，您應該能夠將您的時間和日期與互聯網服務器進行同步

Windows 10

1. 右鍵點擊右下角的 Windows 時鐘

2. 確保“自動設定時間”已開啟，並且您已選擇了正確的時區

檢查您的代理設定

如果上述步驟無法解決您的問題，那麼您可能需要確保您的電腦沒有嘗試通過代理服務器進行 連接。請按照以下步驟：

Windows 7 或更早版本

  1. 轉到“開始”菜單
  2. 搜索 “proxy，”它將顯示一個”更改代理設定“。點擊它
  3. 點擊 局域網設定 ，並取消選中該選項卡中的所有復選標記
  4. 點擊 確定

Windows 8 或 8.1

  1. 點擊屏幕左下角進入平鋪開始屏幕
  2. 輸入 “PROXY” 它將顯示一個”更改代理設定“選項。點擊它
  3. 點擊 局域網設定 ，並取消該選項卡中的所有復選標記
  4. 點擊 確定

Windows 10

  1. 按下鍵盤上的 Windows 鍵+ S。或者，如果 Cortana 的“問我任何東西”欄在您的任務欄中是可使用，您可以點擊它
  2. 輸入 “Proxy”，它將顯示一個”更改代理設定“選項。點擊它
  3. 確保“使用設定腳本”和“使用代理服務器”都已設定為“關閉”
  4. 向下滾動並點擊 儲存

怎麼辦- Mac

如果您正在運行 Mac，請嘗試重新安裝。如需協助， 請點擊此處 。
