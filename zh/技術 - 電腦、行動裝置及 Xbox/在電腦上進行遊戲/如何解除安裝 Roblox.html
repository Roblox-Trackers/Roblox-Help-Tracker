如何解除安裝 Roblox

如果您在 Roblox App 或 Studio 上有技術問題，有時先解除安裝 Roblox 並重新安裝可能會有所幫助。本文將指導您完成解除安裝程序。

Windows 指示

解除安裝程式

Windows 7

  • 前往「開始」功能表 >「控制台」> 程式和功能
  • 向下捲動直到出現 Roblox
  • 在那裡解除安裝程式

Windows 8 或 8.1

  • 按一下螢幕左下角來存取並排的開始螢幕
  • 輸入「程式和功能」，然後按一下搜尋結果右側顯示的 程式和功能圖示
  • 在彈出的視窗中，向下捲動直到出現 Roblox
  • 在那裡解除安裝程式

Windows 10

  • 按一下「開始」按鈕或螢幕左下方的 Cortana 搜尋方塊
    • 也可以按住鍵盤上的 Windows 鍵，然後再按 S 鍵
  • 在搜尋方塊中，輸入「程式和功能」，然後按一下搜尋清單上方顯示的 程式和功能圖示
  • 在彈出的視窗中，向下捲動直到出現 Roblox
  • 在那裡解除安裝程式

刪除 Roblox 資料夾

如果您遇到的問題持久存在，則在解除安裝後完全移除 Roblox 資料夾可能會很有幫助，可以確保沒有遺留損毀或過時的檔案。您可以按照以下步驟進行操作：

    1. 在「開始」按鈕上按一下滑鼠右鍵，然後按一下 瀏覽 或 檔案總管
    2. 導覽到此資料夾： C:\Users\(Your Windows Username)\AppData\Local
    3. 刪除 Roblox 資料夾

附註：除非您已手動選擇將其儲存在此資料夾中，否則不應刪除您可能已儲存在 Studio 中的任何位置。

解除安裝困難

如果在解除安裝程式時遇到困難，請嘗試以下操作：

清除 Internet 暫存檔案

在任何解除安裝或重新安裝程序中，清除瀏覽器的 Internet 暫存檔案都是一個重要步驟。這將有助於解決任何可能已出現的損毀問題。如需此程序的協助，請 按一下此處 ，並參閱 清除 Internet 暫存檔案和 Cookie 部分。

檢查防火牆

許多防毒軟體或網際網路安全性軟體會試圖控制哪些程式可以存取網際網路。通常出於謹慎考慮，這些程式會出錯，依預設不允許任何程式存取。如需進一步調查並讓 Roblox 排除任何障礙的協助，請 按一下此處 。

重新安裝 Roblox

接下來可以執行重新安裝。這似乎違反常理，但如果問題是先前解除安裝失敗的嘗試導致檔案損毀，則這將覆寫電腦上程式的複本，並取代所有已刪除或損毀的檔案。然後，您應該能夠進入「開始」功能表並解除安裝程式。如需重新安裝的協助，請 按一下此處 。

Mac 指示

完全退出 Roblox 程式

  • 按下 Command + Option + Esc 鍵開啟工作管理提示
  • 確保您在其中沒有看到 Roblox 或 Roblox Studio
  • 如果列出了任一程式，請選擇 Roblox 或 Roblox Studio，然後按一下「Force Quit（強制結束）」

前往 Finder > 應用程式

  • 將 Roblox 圖示拖到垃圾桶
    • Finder 中的圖示包含所有已安裝的 Roblox 檔案

如何從垃圾桶刪除檔案

    • 按一下 Dock 中的垃圾桶圖示，然後選擇「Empty Trash（清空垃圾桶）」。
    • 也可以透過連續按下三個按鍵來清空垃圾桶：Command + Shift + Delete。
    將出現以下警示：「您確定要刪除垃圾桶中的項目嗎」， 因為此動作無法復原 。但假設您要刪除，則按一下「Empty Trash（清空垃圾桶）」後，這些內容將會清空。

若要略過「您確定要永久清除垃圾桶中的項目嗎?」訊息，請按下：

  • Command + Option/Alt + Shift + Delete
