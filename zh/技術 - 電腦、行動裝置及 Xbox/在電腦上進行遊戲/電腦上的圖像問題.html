電腦上的圖像問題

如果您遇到 Roblox 的圖形問題，請閱讀以下信息。


圖像 問題的常見問題

  • 嘗試啟動體驗時顯示“顯示卡太舊”錯誤
  • 在遊戲過程中 Roblox 崩潰並報告圖像錯誤
  • 接收信息：“圖像初始化失敗”
  • 在玩的時候，所有遊戲看起來很不尋常，例如看起來奇怪的紋理，扭曲的圖像或者難以理解的文字

為什麼會發生

  • 電腦不支持 DirectX10 或更高的特性級別
  • 顯卡沒有最新的驅動程序
  • Roblox 不支援顯卡（很少出現這種情況）
  • 玩家已更改 Roblox 圖像設定

該怎麼做

試試下面的方，一步一步來。 看看問題是否仍然存在。如果是，繼續前進。

重置 Roblox 圖像

這僅適用於您可以在不崩潰的情況下真正啟動 Roblox

  1. 打開 Roblox Studio
  2. 打開 “文件” 菜單，然後點擊 設定
  3. 點擊 OK 以進行
  4. 點擊位於彈出窗口左下方的 重置所有設定 按鈕。

檢測支持的 DirectX 特性級別

檢查你所支持的 DirectX 特性級別 ：

  1. 按下 Windows 鍵+ R 彈出運行提示符。
  2. 輸入 dxdiag，然後按下鍵盤的 Enter 鍵
  3. 在 DirectX 窗口的面板視圖中單擊 “Display”或“Display 1”，如果可用，單擊 “Display 2” 選項卡
    • 對於 Windows 8.1/10，尋找特性級別。對於 Windows 7 查找 DDI 版本。

如果你在 “DDI 版本” 或“功能級別”中沒有看到 10 或以上，說明你的顯卡不支持 DX10 或以上，你將無法訪問應用程序。

降低顯示器的色彩質量

您的顯卡可能無法支持用於普通桌面操作的顏色質量的 3D 渲染。您可以嘗試將位深度降低到16位，以查看是否有幫助。

升級您的顯卡驅動程序

要了解 Roblox 目前使用的顯卡，請嘗試以下操作：

  1. 打開 Roblox Studio
  2. 打開 Studio 設定
  3. 點擊 診斷
  4. 向下滾動查找標記為 GfxCard 的行
  5. 這將列出 Roblox 當前使用的顯卡

請訪問顯卡製造商提供的支持頁面，並下載特定型號的最新驅動程序。

檢查兼容模式

如果您收到一條消息，指出您的圖像已過期，但您的計算機通過了最低規格要求，您可以檢查應用程序是否在兼容模式下運行。

    1. 右鍵單擊 Roblox 或 Roblox Studio 應用程序，然後選擇屬性。
    2. 選擇兼容性選項卡。
    3. 如果選中以兼容模式運行此程序框，請取消選中它。

請訪問您的顯卡製造商提供的支持頁面，並為您的特定型號下載最新的驅動程序。
