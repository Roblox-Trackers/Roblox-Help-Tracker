常見網站問題

下面的文章將提出一些已被證明有效的方法來解決以下的問題：

  • 該網站顯示不正確
  • 有些頁面根本無法加載或加載時顯示錯誤
  • 按鈕或鏈接不允許您點擊它們
  • 您在Roblox網站上嘗試使用正確的用戶名和密碼但無法登錄
  • 其它：如果您在網站上遇到了不同的問題，下面的步驟也可能有助於解決這個問題。此外， 一些與網站無關的文章可以鏈接到這里以進行一些步驟

如何操作

請按順序嘗試下面的每一步。如果一個解決方案不起作用，請繼續使用下一步！

清除臨時互聯網文件和Cookie

以下鏈接將幫助您刪除主要電腦網頁瀏覽器的臨時互聯網文件（也稱為“緩存”）和Cookie。請務必執行您想要啟動Roblox的 Internet Explorer以及其它瀏覽器。

  • Internet Explorer - 臨時Internet文件（來自Microsoft.com）
  • Internet Explorer - Cookies（來自Microsoft.com）
  • Mozilla Firefox - 臨時Internet文件（來自Mozilla.com）
  • Mozilla Firefox - Cookies（來自Mozilla.com） \ 谷歌瀏覽器-兩者（來自Google.com）
  • 谷歌瀏覽器- 兩者（來自Google.com）
  • Microsoft Edge - 兩者（來自Microsoft.com）
  • Safari - Cookies（來自Apple.com）
  • 其他瀏覽器 ：如果您使用的是其它瀏覽器或舊版本的瀏覽器，請搜索有關如何刪除特定瀏覽器/版本的緩存/瀏覽器數據/臨時Internet文件的信息。

完成上述所有相關網絡瀏覽器後，請重新啟動計算機並查看您的問題是否已解決。

如果您想手動刪除Internet Explorer / Windows臨時Internet文件，可以直接進入硬盤驅動器上的Temporary Internet Files文件夾。位置和文件夾名稱取決於Windows的版本以及您是否使用用戶配置文件

注意：您可能需要更改Windows資源管理器的設置才能顯示各種文件（包括受保護的系 統文件）以訪問這些文件夾。

對於Windows 10

如果您有Windows 10，則臨時Internet文件位於以下位置（請注意，在PC上，它們可以位於 其他驅動器上而不是驅動器C上）：

  • C：\ Users \ <用戶名> \應用程序數據\本地\微軟\的Windows \ INetCache

對於Windows 7和8

如 果您有Windows 7或Windows 8，則臨時Internet文件位於這些位置（請注意，它們可以 位於另一個驅動器上而不是驅動器C上）：

  • C：\ Users \ <用戶名> \ AppData \ Local \ Microsoft \ Windows \ Temporary Internet Files \
  • C：\ Users \ <用戶名> \ AppData \ Local \ Microsoft \ Windows \ Temporary Internet Files \ Low \

禁用您的Web瀏覽器的插件

很多時候與網站相關的問題都是第三方附加組件對於您的Web瀏覽器的結果。根據您選擇的瀏覽器，這些也可能被稱為“擴展”或“插件”。要查看是否屬於這種情況，請嘗試禁用所有附加組件。如果問題得到解決，請嘗試每次重新啟用它們，直到問題再次出現。一旦問題重新開始，您應該找到與Roblox不兼容的問題。

以下鏈接可以幫助您使用最流行的PC網頁瀏覽器：

  • Internet Explorer（來自Microsoft.com）
  • Firefox（來自Mozilla.com）
  • Chrome（來自Google.com）
  • 其它瀏覽器 ：如果您使用的是其他瀏覽器或較舊版本的瀏覽器，請搜索有關如何禁用其特定瀏覽器/版本的擴展/插件/附加組件的信息。

重置Windows的互聯網選項

我們發現許多Roblox，網站和Roblox Studio問題都可以通過重置計算機的Internet選項來解決。請記住，即使您通常使用其他網絡瀏覽器，這些步驟也需要使用Internet Explorer。嘗試完這些步驟後，您可以返回使用您喜歡的任何瀏覽器。請嘗試以下操作：

  1. 打開Internet Explorer（特別是這個網頁瀏覽器）
  2. 打開齒輪圖標> Internet選項
  3. 打開 高級選項卡
  4. 點擊 重置 按鈕

重置後，請關閉Internet Explorer以使更改生效。完成此操作後，您應該能夠返回使用您選擇 的網絡瀏覽器。

更新Internet Explorer

請嘗試將您的Internet Explorer更新到最新版本。您可以 點擊這裡 找到它。即使您目前沒有將IE瀏覽器用作瀏覽器，您仍然需要確保您的Windows計算機上具有最新版本，以便Roblox正常工作。
