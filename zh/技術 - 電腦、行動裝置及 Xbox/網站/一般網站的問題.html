一般網站的問題

下面的文章將提出一些已被證明有效的方法來解決以下的問題：

  • 該網站顯示不正確
  • 有些頁面根本無法加載或加載時顯示錯誤
  • 按鈕或鏈接不允許您點擊它們
  • 您在 Roblox 網站上嘗試使用正確的使用者名稱和密碼但無法登入
  • 其它：若您在網站上遇到了不同的問題，下面的步驟也可能有助於解決這個問題。此外， 一些與網站無關的文章可以鏈接到這里以進行一些步驟

如何操作

請按順序嘗試下面的每一步。若一個解決方案不起作用，請繼續使用下一步！

删除 Internet 临时 檔案 和 Cookie

以下連結將協助您刪除主要電腦網頁瀏覽器的 Internet 临时檔案（也稱為緩存）和 Cookie。請務必執行您想要啟動 Roblox 的任何瀏覽器。（Roblox 不支援 Internet Explorer）

      • Mozilla Firefox - Internet 臨時 檔案 （來自 Mozilla.com）
      • Mozilla Firefox - Cookies（來自 Mozilla.com）
      • Google Chrome - 兩者（來自 Google.com）
      • Microsoft Edge - 兩者（來自 Microsoft.com）
      • Safari - Cookies（來自 Apple.com）
      • 其他瀏覽器 ：若您使用的是其它瀏覽器或舊版本的瀏覽器，請搜索有關如何刪除特定瀏覽器、版本的緩存、瀏覽器數據或 Internet 臨時檔案的資訊。

完成上述所有相關網絡瀏覽器後，請重新啟動電腦並查看您的問題是否已解決。

若您想手動刪除 Windows 的 Internet 臨時檔案，可以直接進入硬盤驅動器上的 Internet 臨時檔案資料夾。位置和資料夾名稱取決於 Windows 的版本以及您是否使用使用者配置檔案。

注意：您可能需要更改  Windows 資源管理器的設置才能顯示各種檔案（包括受保護的系 統檔案）以訪問這些資料夾。

對於 Windows 10

若您有 Windows 10，則 Internet 臨時檔案位於以下位置（請注意，在電腦上，它們可以位於 其他驅動器上而不是驅動器 C 上）：

          • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

對於 Windows 7 和 8

若您有 Windows 7 或 Windows 8，則臨時 Internet 檔案位於這些位置（請注意，它們可以 位於另一個驅動器上而不是驅動器C上）：

          • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
          • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

停用瀏覽器的附加組件

很多時候與網站相關的問題都是第三方附加組件對於您的瀏覽器的結果。根據您選擇的瀏覽器，這些也可能被稱為擴展或插件。要查看是否屬於這種情況，請嘗試禁用所有附加組件。若問題得到解決，請嘗試每次重新啟用它們，直到問題再次出現。一旦問題重新開始，您應該找到與Roblox不兼容的問題。

以下鏈接可以協助您使用最流行的 PC 網頁瀏覽器：

      • Internet Explorer（來自 Microsoft.com）
      • Firefox（來自 Mozilla.com）
      • Chrome（來自 Google.com）
      • 其它瀏覽器 ：若您使用的是其他瀏覽器或較舊版本的瀏覽器，請搜索有關如何禁用其特定瀏覽器/版本的擴展/插件/附加組件的資訊。
