創建和把遊戲製成公開

您可能會覺得，只開發一款遊戲無法滿足您對創意的渴望。幸好的是，我們可以協助！所有用戶都可以隨著自己的意思，構建盡可能更多的遊戲，但是賬戶在任何給定時間，有限制200個公共遊戲。（那些可以由其他人播放）。

製作遊戲為公開或私人：

製作公開或私人遊戲方式有兩種。

注意：讓您的遊戲保密將關閉所有正在運行的遊戲並取消所有服務器。

第一方法：

  1. 點擊網站頂部藍色欄中的 創作
  2. 在 我的創作 標籤中，如果遊戲尚未突出顯示，請點擊Games（ 遊戲 ）
  3. 找到您想要改為公共/私密的場所，然後點擊場所標題下方的灰色或綠色圖標，在兩種狀態之間切換
    • “公共” （綠色圖標）允許任何用戶查看和玩遊戲
    • ”私密“ （灰色圖標）會阻止除您之外的任何人查看或玩遊戲，除非是小組遊戲，在這種情況下，只有您和指定的開發人員（具有必要的遊戲編輯權限）才能夠玩這遊戲。有關更多信息，請參閱關於 群組遊戲 的幫助文章。

第二方法：

  1. 轉到您的遊戲頁麵
  2. 點擊三點圖標來拉起遊戲的設置菜單
  3. 選擇 空間設定 或 遊戲設定
  4. 無論您在上述步驟中選擇哪一個，在屏幕上，如果尚未突出顯示，請導航至Basic Settings （ 基本設置） 選項卡
  5. 在Privacy （ 隱私 ） 下，選擇Public（ 公共 ） 或（ 私人 ）
    • Public 公共 將允許任何用戶查看和玩遊戲
    • Private 私密 会阻止除您之外的任何人查看或玩游戏，除非是群组游戏，在这种情况下，只有您和指定的开发者（具有必要的游戏编辑权限）才能玩游戏。 有关更多信息，请参阅关于 Group Games . 的帮助文章。

創建新遊戲：

  1. 點擊網站頂部藍色欄中的 創作
  2. 如果它尚未明亮顯示，請在 My Creations （我的創作）選項卡中單擊 Games （ 遊戲 ）
  3. 點擊 Create New Game （創造新遊戲）
  4. 為新創造選擇Settings（設置）和 Template（模板）
  5. 點擊 Create Game （創作遊戲）

這會帶您回到您之前的創建頁面。要的話，您可以現在把遊戲造作公共，或者您可以等到您已做了一些工作。要開始創建，請點擊遊戲列表中新 遊戲 條目右側的 Edit（ 編輯） 按鈕。

注意：您可以任何時創建一款新遊戲，但只直到使用“製作遊戲為公共”流程將它改為公開之後，才能讓所有 Roblox 玩家享用它。
