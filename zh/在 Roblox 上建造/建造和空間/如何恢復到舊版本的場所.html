如何恢復到舊版本的場所

Roblox保留所有上傳場所的副本。這意味著，當您對某個地方或場所進行任何更改並將它發佈到Roblox時，沒有更改的副本將保存為舊的版本。通過以下操作，您可以隨時查找並恢復到這些舊版本之一。

更改為早期版本：

  1. 點擊位於屏幕頂部的 創作 。
  2. 從這裡查找您想要配置的場所
  3. 點擊場所名稱右側的”齒輪“圖標，並在顯示的菜單中點擊 Configure Start Place
  4. 點擊 Version History 。在這裡，您將看到按日期排序的過去版本列表
  5. 選擇您想要成為當前的版本
  6. 點擊 Revert to this version 以使它成為當前場所
