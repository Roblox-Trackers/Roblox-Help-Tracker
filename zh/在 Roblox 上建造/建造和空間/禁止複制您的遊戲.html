禁止複制您的遊戲

禁止複制您的遊戲可以防止其他人直接複製您創建的所有內容。但是，如果您希望允許其他人將您的遊戲作為自己遊戲的模板，則可以選擇允許複製。

有兩種方法可以用來允許或禁止複制。

第一方式：

    1. 登錄並點擊 創作 選項卡
    2. 點擊遊戲名稱左邊的齒輪圖標，然後在菜單中選擇 Configure Start Place
    3. 點擊 Permissions
    4. 在 Other Permissions 下，如果您希望允許複製，您可以 打鉤 允許複製 旁邊的複選框。如果您不想被複製， 您不要打鉤。
      • 注意： 通過選中此框，您是在允許Roblox其他會員來使用（以各種方式）您正在分享的內容。如果您不想授予此權利，請不要選中此框。關於分享內容的更多訊息，請查看 Roblox使用條款 。
    5. 點擊 Save

第二方式：

    1. 導航到您的遊戲頁面
    2. 點擊三點圖標來拉起遊戲的設置菜單
    3. 選擇 Configure this Place
    4. 在出現的屏幕上，請導航到 Permissions 選項卡
    5. 在 Other Permissions 下，如果您希望允許複製，您可以 打鉤 允許複製 旁邊的複選框。如果您不想被複製， 您不要打鉤。
      • 注意： 通過選中此框，您是在允許Roblox其他會員來使用（以各種方式）您正在分享的內容。如果您不想授予此權利，請不要選中此框。關於分享內容的更多訊息，請查看 Roblox使用條款
    6. 點擊 Save
