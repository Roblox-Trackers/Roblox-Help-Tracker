在Studio中編輯和測試（編輯模式）

編輯模式是Roblox Studio中的一項功能，它允許您在沒有任何動靜時 （類似於時間停止） 而編輯全部或大部分的地方。提供了大量工具和選項來幫助您構建和定制您的地方。在退出編輯模式之前，推薦您把正在工作的內容存儲在電腦上。

進入編輯模式

編輯模式可以通過以下任何方式到達：

  • 通過訪問遊戲的詳細信息頁面，點擊播放按鈕上方看起來像三個點的圖標，然後從彈出的菜單中選擇 編輯 。請注意，這將需要您自己的遊戲之一，或者如果它屬於其他人，它將不需要 被複製鎖定
  • 通過轉到 創建 部分，找到您想要編輯的地方並點擊相應的 編輯 按鈕
  • 通過打開在Studio的“文件”菜單中選擇 打開 保存在電腦中的地放
  • 通過在Studio內點擊File> 選擇 New。 這會打開了一個空白的地方，可以變成幾乎任何您想要的東西

選項卡

Studio提供各種不同選項，位於各個標籤內的快速可點擊按鈕中。這些標籤是：

  • HOME ：包含各種最常用的快速參考按鈕
  • MODEL ：在這裡您可以找到代表大部分物體插入和修改選項的按鈕
  • TERRAIN ：這有所有的工具允許您修改地形
  • TEST ：您所需要的所有幫助您測試改變如何在遊戲中運行的東西都在這裡。此外，此選項卡的“仿真”區域可讓您了解您的遊戲在多種移動設備或電視機上的顯示效果。
  • VIEW ：允許您在Studio中顯示或隱藏各個部分/窗口
  • PLUGINS ：在這裡您可以找到您已經下載或添加到您的帳戶的任何插件。有關此功能的更多信息， 點擊這裡

探險者和屬性部分

一旦您進入 Edit( 編輯)模式（無論是新創的或在一個已經被建好的場所），您都會在屏幕右側看到兩個標籤為Explorer（ 深测器 ）和Properties（ 屬性 ）的部分。若您沒有看到它們，您可以通過單擊“View（ 視圖 ）”選項卡下左側的相應圖標重新打開它們。

  • Explorer(探險者)
    本節將列出位於您場所中的所有內容。它將被分解成各種組，如Workspace(工作區)，Lighting(燈光)，StarterPack(入門包)，Players(玩家)等等。要查看服務組內部的內容，只需點擊它左側的小灰色箭頭即可。在您的地方和大多數腳本中看到的所有物體都會位於Workspace組裡面。如果一個物體包含其它物體（如貼花），它也會有一個小箭頭。要關閉組並隱藏其內容，只需再次點擊箭頭即可。
  • Properties (屬性)
    本節向您顯示物體具有的Size（大小），Rotation（旋轉）等屬性列表，如果它可以移動（錨定）等）。必須選擇該物體才能查看其屬性。

使用鏡頭

鏡頭操作非常簡單。這與在線玩遊戲時不同，相機不限於一個物體（例如您的頭）。相反，它可以讓您在任何角度移動它。使用W，A，S，D或箭頭鍵向前，向後，向左和向右移動相機。點擊並按住鼠標右鍵並移動鼠標以旋轉視圖。向上滾動鼠標的滾輪會放大，而向下滾動則會縮小。

添加和移動物體

在編輯模式下添加 Bricks（磚塊），Scripts（腳本）和Models（模型）（以及許多其它物體）很容易，並且有多種方法可以實現這一點。第一個選項是簡單地右鍵點擊遊戲窗口中的空白區域，選擇插入物體，然後選擇要添加到您地方的東西。

或者，您可以使用“模型”選項卡中的許多按鈕快速插入許多內容，例如零件，磚塊，效果，生成點，腳本等。最後，為了便於放置，您可以使用屏幕左側的工具箱將許多不同的磚塊，模型或腳本拖放到遊戲窗口中。如果這不可見，您可以在“查看”選項卡中重新打開它。

將已經添加到遊戲中的東西移動更容易！首先，只需點擊位於屏幕頂部“型號”選項卡上的“移動”圖標即可。然後，您可以通過在遊戲窗口中單擊並拖動來自由移動對象。如果您只想向上/向下，向左/向右或向內/向外移動它，只需單擊對象即可顯示一系列箭頭。點擊指向您想移動的方向並拖動鼠標的那個。

注意：某些物體（如火花和腳本）是無形，因此無法移動。

Toolbox工具箱

工具箱允許開發人員將模型，貼花等由其他玩家創建的拖放到他們的地方置。這與右鍵點擊菜單中的插入物體選項類似，但提供了預先創建的材質和播放器創建的內容。

調整大小和旋轉物體

要調整磚塊的大小，單擊位於Model （模型）選項卡中的Scale （縮放）按鈕，選擇一個磚，並在對象的頂部，底部，左側，右側，前部和後部出現小點。單擊並拖動對應於您想要縮小或拉伸的一側的點。

注意：模型無法調整大小。

要旋轉一個物體，只需點擊它，然後點擊位於模型選項卡中的旋轉按鈕。這將用六個點和三個圓圈圍繞選定的項目。單擊其中一個點並拖動鼠標以扭曲或傾斜與點的顏色相匹配的圓內部的選擇。

添加和修改地形

Roblox工作室現在還允許您輕鬆添加或操縱世界地形，創造出美麗的風景。您可以在Terrain選項卡中找到適當的工具。有關這些工具各自所具有的功能的詳細信息， 點擊這裡 。

測試您的場所

一旦您完成了您的地方，或者您想在發布之前看看它做了什麼，點擊 Test tab（ 測試標籤） 中的 Play（ 播放） 按鈕。這會將您的頭像加載到遊戲窗口中，並允許您四處走動並玩耍。要停止測試並將游戲恢復到單擊“播放”之前的狀態，點擊 Stop “ 停止 ”按鈕。

注意：如果您想通過在Roblox網站上的正常播放功能播放來測試您的地方，您可以在沒有其他玩家加入您的遊戲的情況下將游戲地方為私人遊戲來進行測試。有關更多信息，請參閱幫助文章 創建和製作公共遊戲 。

測試您的遊戲如何與其它玩家合作

如果您想看看您的地方與其他幾個參與者的相似之處，您可以模擬一個服務器。為此，轉到Test tab（測試） 選項卡，然後在位於“客戶端和服務器”區域的下拉框中選擇要模擬的玩家數量。完成後，點擊Start（開始）。這將打開其他幾個Studio窗口。其中一個窗口將作為服務器，而其他窗口將允許您控制將添加到遊戲中的各種假冒玩家之一。要停止測試，單擊原始Studio窗口中的Cleanup （清理）按鈕。

注意：如果您想通過Roblox網站上的常規播放功能與您指定的其他開發人員一起測試您的地方，則可以在未經批准的玩家加入遊戲時將其作為群組遊戲加入，並將編輯權限限制為指定的等級，並將您的遊戲設置為私人。有關更多信息，請參閱幫助文章 創建和製作公共遊戲 和 組遊戲 。
