群組遊戲

擁有正確 權限 的組成員能夠為組創建和編輯遊戲，這可以成為收入來源。

群組擁有的遊戲：創建和編輯

群組遊戲是專門由組擁有的遊戲。任何從這些遊戲中獲得的資金和從銷售或廣告中獲得Robux都將應用於群組本身，不管是誰創造它。有關群組收入的更多信息，請點擊 這裡 。

具有“創建和編輯群組遊戲" 權限 的任何成員都可以參與群組場所的構建過程。要創建新的群組遊戲，請執行以下操作：

  1. 在網站頂部的藍色欄中點擊 創作
  2. 點擊 Group Creations 選項卡
  3. 從左側導航欄頂部的下拉框中選擇”群組“
  4. 點擊 Create New Place 按鈕
  5. 瀏覽每個選項卡，然後按照自己的意願設置遊戲。有有關配置遊戲的幫助，請 點擊這裡
  6. 點擊 Create Place 按鈕

這將在組的位置列表中創建一個私有空的地方/場所。要使遊戲公開並可玩，可以點擊左側導航欄中的 遊戲 ，然後點擊場所標題下方的灰色 私人按鈕 。您可以通過以下步驟進行編輯或配置該場所或者其他組擁有的遊戲：

  1. 在網站頂部的藍色欄中點擊 創作
  2. 點擊 Group Creations 選項卡
  3. 從左側導航欄頂部的下拉框中選擇”群組“
  4. 在列表中找到您想要編輯或配置的場所
  5. 要編輯場所本身，請點擊其 Edit 按鈕。這將在Studio中打開它，這樣您就可以進行任何必要的調整
  6. 要配置該場所的基本設置，您可以在編輯按鈕的右邊點擊”齒輪“，然後選擇 Configure
