如何複製遊戲

有時候您可能會想開始新遊戲，但是想編輯以前存在的遊戲來節省一些時間。您也可能只想將其中一個遊戲複製到另一個賬戶。以下步驟將幫助您做這些操作。

如何複製您自己的遊戲

  1. 運行Roblox Studio
  2. 在Studio中登錄您的賬戶
  3. 點擊 My Games
  4. 找到您想要複製的遊戲，點擊它，然後點擊 Edit 位於屏幕底部
  5. 一旦您編輯遊戲後，您可以用兩種複制方式：
  • 如果您只想將備份保存到您的電腦上，請選擇 file > Save As
  • 如果您想將游戲副本上傳到Roblox，請選擇 file> Publish to Roblox As ，然後通過現有遊戲上傳或創建一個新遊戲

注意：要將游戲複製到您的其他賬戶之一，請選擇步驟5中的 Save As 方法。保存遊戲後，登出Studio，然後重新登錄為新賬戶。然後您可以使用 Publish to Roblox As 上傳到該賬戶。

如何使用其人的複制允許遊戲作為模板

您可能會看到一款您認為可以作為您自己遊戲的良好初始點的遊戲。好消息是，一些建設者鼓勵您為此使用他們的遊戲！但是，只有在遊戲的 About 選項卡上列出了“可以復制此遊戲的源代碼”通知時才能複制遊戲。請參閱我們的 幫助文章，了解如何允許或禁止複制遊戲 以獲取更多詳細信息。

如果允許複製遊戲，那麼以下步驟將幫助您將其用作自己遊戲的起點：

  1. 轉到遊戲的詳細信息頁面
  2. 點擊標題右上角的三個點的圖標
  3. 在彈出的菜單中單擊 edit
  4. 這將在Studio中打開遊戲，您可以執行以下兩件事之一：
  • 選擇 file > Save As ，然後將游戲保存到您的電腦上。最後您可以根據自己的喜好編輯遊戲。
  • 選擇 file > Publish to Roblox As ，然後通過現有遊戲上傳或創建一個新遊戲。

注意：在決定發布複製的遊戲之前，請務必先徵求創作者的許可。如果玩家的遊戲不允許被複製，並且您想製作自己的版本，請務必首先向創建者請求許可。這是比較有禮貌的。
