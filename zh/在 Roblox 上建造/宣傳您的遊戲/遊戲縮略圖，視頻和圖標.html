遊戲縮略圖，視頻和圖標

您可以使用縮略圖和圖標功能來幫助您的“場景/遊戲”受到關注並脫穎而出。

什麼是縮略圖？

縮略圖是在細節頁面上的圖像，用於代表您的遊戲並可以用來幫助它脫穎而出。同時還可以向新玩家展示他們點擊“播放”時的期望。遊戲最多可以有10個縮略圖而每一個都可以是圖像或視頻。遊戲 發佈 時將添加一個默認縮略圖，但如果您願意，您可以稍後更新它。

您如何添加新的縮略圖或視頻？

您的第一個縮略圖將自動添加為默認圖像。不過，您可以添加新的免費自定義縮略圖或者YouTube視頻為500個Robux。要做到這一點 ，請按照以下步驟操作：

  1. 使用您選擇的圖像或視頻編輯器來創建圖像或視頻。
  2. 如果您正在製作一個視頻，把它上傳到您的YouTube賬戶上。
  3. 點擊位於屏幕頂部藍色欄中的 創作 。
  4. 找到您想要添加圖庫項目的位置，並點擊右側的“齒輪”。
  5. 點擊 Configure Start Place 。
  6. 在顯示的頁面左側欄中，點擊 Thumbnails 。
  7. 選擇是否要使用屏幕右側的 Image, Video 或 Auto generated Image 。
  8. 如果您正在上傳圖像，點擊 Choose file ，查找並選擇您的圖像文件，按 Open ，然後按藍色的 Upload an Image 按鈕。然後將看到一個彈出框，確認您想購買縮略圖。按綠色的 Buy Now 按鈕來完成 交易。
  9. 如果您正在鏈接YouTube視頻，在該框中輸入URL網址，然後點擊藍色的 Add Video 按鈕。然後您將 看到一個彈出框，確認您要購買視頻縮略圖。按綠色的 Buy Now 按鈕來完成交易。
  10. 如果您想使用 Auto generated Image ，那麼此選項是免費的和您將在選擇此選項時看到預覽。按藍色的 Set Thumbnail 按鈕選擇該圖像。

注意：圖像和視頻必須符合 Roblox社區規則 。當上傳的圖像/視頻正在等待我們的版主批准時 ，圖像或視頻將暫時不可用。如果縮略圖由於違反了這些規則而被刪除或拒絕，那麼所花費的Robux是不可退還 。

視頻有什麼限制（長度，內容等）？

視頻最長可達30秒。如果您的視頻超過這個長度，您將收到一個錯誤信息，並且不會上傳。有關其它 視頻限制和認可的完整列表，請 點擊這裡 。

什麼是遊戲圖標和您如何製作它們？

圖標是正方形的圖像，除了在詳細頁面以外也可以在某些地方用來代表您的遊戲。這包括“遊戲”頁面， 個人檔案 頁面，“收藏夾列表等。在默認情況下，遊戲的圖標將是我們在上一節中提到的遊戲標準縮略圖的裁剪版本。 也就是說，您可以使用您選擇的圖片編輯器創建自己的圖像並免費上 傳。有關此過程的幫助以及如何創建引人注目的圖標的些技巧，請 點擊這裡 。

注意:圖標圖片必須符合 Roblox社區規則 。
