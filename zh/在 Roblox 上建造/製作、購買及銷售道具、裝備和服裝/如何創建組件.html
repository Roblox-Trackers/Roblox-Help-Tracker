如何創建組件

Package（我們稱它為“組件”）基本上是將多個物件組合在一起形成一個物體。組件可以包含磚塊或工具的任何東西，並且它可以大大地增強了您操作一組物體的難易程度。此外，通過創建組件，您還可以選擇將它上傳到Roblox。此功能允許其他玩家通過使用“ Studio Toolbox ”中的“All Models”選項或 瀏 覽“道具庫” 時訪問組件。

如何製作組件

為了創建組件，請遵循以下五個簡單的步驟。

    1. 在Roblox Studio中打開一個場所。
    2. 創建您想要組合在一起的物件，並按照您希望的方式排列它們。
    3. 要組合您的組件，按下鍵盤上的 CTRL + G 。
    4. 從菜單欄中選擇 File ，然後選擇 Publish Selection to Roblox 。
    5. 完成所需要的信息，然後選擇 Finish 。

如果正確執行了這五個步驟，那麼您應該已經成功地創建了一個組件。
