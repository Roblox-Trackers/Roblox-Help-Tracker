Roblox 生物特徵辨識隱私權聲明

您為什麼會收到這份聲明？

這份生物特徵辨識隱私權聲明說明 Roblox 如何收集、使用、共享、保存及銷毀 (下文統稱為「處理」) 您使用我們服務時的生物特徵辨識資料。 「生物特徵辨識資料」包括：(i)「生物特徵辨識特徵」是為您測量生物特徵時產生的資料，例如視網膜或虹膜掃描、指紋、聲紋，或是手或臉部幾何結構的掃描；及 (ii)「生物特徵辨識資訊」是以可用來識別您身分之生物辨識特徵為基礎的資訊。

請仔細閱讀這份聲明。 若使用 Roblox 身分驗證程序，即表示您同意如本聲明所述的方式，收集、使用、共享及保存您的生物特徵辨識資料。

我們處理哪種類型的生物特徵辨識資料？

  • 臉部幾何結構。

我們如何處理資料？

  • 我們僱用服務供應商 Veriff 和 Persona，協助我們的身分與年齡驗證程序。
  • Veriff 或 Persona 會擷取您 ID 的照片和自拍照，確認您的身分。 接著，Veriff 或 Persona 會審核從您的 ID 和自拍照擷取的臉部幾何結構，並且分析您臉部的年齡，然後比對這項資料和您 ID 上的出生日期。 比對是為了安全確認您的年齡和身分，以及偵測詐騙。
  • 少數 Roblox 員工得存取 Veriff 或 Persona 基於法規遵循目的所儲存的 ID 照片和自拍照。

您的生物特徵辨識資料我們會保存多久？

  • Roblox 不會收到您的生物特徵辨識資料副本，亦沒有存取權。
  • 我們的服務供應商 Veriff 或 Persona 分別會在收集後 90 天內刪除您的生物特徵辨識資料，除非法律規定 (例如搜索票或傳票) 必須將您的生物特徵辨識資料保存得更久。

保存期限結束時，您的生物特徵辨識資料會怎麼處理？

  • 我們的服務供應商 Veriff 和 Persona 皆會在上述保存期限結束時永久銷毀您的生物特徵辨識資料，使其無法復原或修復。

其他隱私權揭露與您的權利

如需進一步瞭解我們如何處理個人資訊，以及針對我們資料處理方式，您有哪些權利和選擇，請查閱我們在 https://www.roblox.com/info/privacy 的隱私權政策。如需進一步瞭解 Veriff 的資料處理方式，請查閱 https://www.veriff.com/privacy-policy 的 Veriff 隱私權政策。 如需進一步瞭解 Persona 的資料處理方式，請查閱 https://withpersona.com/legal/privacy-policy 的 Persona 隱私權政策。
