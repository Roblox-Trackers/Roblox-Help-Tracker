我們不允許使用者在 Roblox 上分享自己或他人的個人資訊，包括：
  • 全名
  • 電子郵件地址
  • 密碼
  • 實際地址
  • 電話號碼
  • 其他獨特的身分識別特徵
  • 自己或其他人的私人影像
  • 未經授權的未成年人錄音或影像
引導使用者離開平台
使用 Roblox 時，不得連結到任何外部網站或服務，但下列除外：YouTube、Facebook、Discord、Twitter 及 Twitch。 前往上述任一核准網站的連結必須包含在 適用的 PolicyService API 中，或是列入個人資料或體驗說明頁面的社群網路及社群連結設定內。 出現在無法部署 API 的位置之連結 (包括前往核准網站的連結) 一律禁止，例如文字聊天或內嵌在媒體中。 禁止任何其他連結或試圖規避連結偵測，包括：
  • 連結包含不適合 Roblox 的內容
  • 部分連結、破壞篩選條件，以及前往非允許網站的許可網站連結
  • 鼓勵他人前往未經許可的外部連結

注意：為了隱私和安全，無論 API 用途為何，均禁止 13 歲以下的使用者分享連結
濫用 Roblox 系統 Roblox 維護許多系統來確保使用者的安全，並且禁止試圖規避這些系統、未經授權存取 Roblox 或他人的 Roblox 帳號或資訊，或是以其他方式違反我們的平台安全規定，包括：
規避安全和保全系統
  • 試圖繞過我們的安全系統（包括隱藏素材中的違規元素）存取受禁止的內容
  • 開立新帳號來規避對先前帳號所採取的執法行動
  • 使用 VPN 掩蓋所在位置，以便未經授權存取 Roblox，包括存取特定功能或帳號
  • 對外提供您 Roblox 帳戶的存取權，或誘使他人以存取別人的 Roblox 帳戶，用以規避執行措施或安全系統。
  • 在您的體驗中無法使用所需的 PolicyService API
破壞性內容和行為
  • 使用程式化機器人執行破壞性任務
  • 上傳包含混淆之程式碼或隱藏之指令碼的內容，且內容會誤導終端使用者，或是中斷他們的體驗
  • 將背景噪音插入底層音訊，使預期的音訊無法辨識
  • 在上傳的音訊中或透過語音聊天時插入過大的音量、尖叫聲或刺耳噪音，藉此擾亂他人的體驗
  • 不實檢舉其他使用者或體驗，或是鼓勵他人這麼做
  • 入侵或大量湧入體驗或團體，以破壞體驗或其聲譽
