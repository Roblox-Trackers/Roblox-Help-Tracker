如何檢舉違規

Roblox 是一個由多年齡段的玩家和開發人員組成的強大社群，我們希望為所有人提供一個安全愜意的環境。如果您發現其他使用者違反了規則，我們將提供工具來協助您檢舉任何不當行為。

如需有關阻止其他使用者聯絡或聊天的資訊，請閱讀 阻止使用者說明文章 。

「Report Abuse（檢舉濫用）」功能是讓我們瞭解違規者和不良內容的最佳方法。系統會直接將檢舉內容傳送給仲裁者，並為我們提供一種方式來追蹤屢犯者，以便可以針對他們的帳號採取進一步措施。檢舉濫用時，請務必檢舉特定的聊天、私人訊息、體驗或其他違反 Roblox 規則的內容。如果只是從他們的個人檔案頁面檢舉使用者，仲裁者將看不到他們有哪些不當行為。

在體驗中檢舉使用者

使用體驗中的「Report Abuse（檢舉濫用）」功能來檢舉違反 Roblox 規則的特定聊天或其他內容。系統會直接將資訊傳送給仲裁者，並允許他們查看其他使用者有哪些不當行為。

  1. 選擇位於螢幕左上方的 Menu（功能表） 按鈕。此圖標看起來像 Roblox 徽標堆疊在 Roblox 播放器的左上角。
  2. 選擇位於使用者名稱旁邊的標幟圖示，或選擇功能表頂端的 Report（檢舉） 標籤。
  3. 對於 Game or Player?（體驗還是玩家?） ，選擇 Player（玩家） 標籤。
  4. 選擇 Which Player（哪個玩家） 下拉式功能表，然後找到要檢舉的使用者名稱。
  5. 選擇 Type of Abuse（濫用類型） 下拉式功能表，然後找到不當動作。
  6. 其他詳細資料會寫入「Short Description（簡短說明）」方塊中。
  7. 選擇 Submit（提交） 即可完成。

如果體驗本身違反規則，則可以在玩體驗時進行檢舉。按照上面第 2 步所述開啟「Report（檢舉）」標籤，然後選擇 Game（體驗） 而不是 Player（玩家） 來完成檢舉表單。

在體驗之外檢舉違規

如果您發現有使用者或內容違反了 Roblox 規則，請使用遍佈於整個應用程式中的「Report Abuse（檢舉濫用）」功能向我們的仲裁者檢舉此行為。根據所檢舉的內容，有不同的檢舉方法，但是您看到的任何內容都可以檢舉。

檢舉體驗

若要檢舉整個體驗，請使用體驗 About（關於） 頁面上 Description（說明） 方塊右下角的紅色「Report Abuse（檢舉濫用）」按鈕。

檢舉其他內容

若要檢舉型錄或圖庫中違反規則的道具（例如襯衫、體驗通行證或模型），請選擇該道具的資訊方塊右上角的三個點按鈕。選擇 Report item（檢舉道具） 並填寫表單。

檢舉應用程式聊天

若要檢舉其他使用者在瀏覽器應用程式中進行的聊天，請選擇聊天視窗右上角的裝備圖示以開啟「Chat Details（聊天詳細資料）」。選擇使用者名稱旁邊看起來像三個點的按鈕，然後選擇 Report（檢舉） 。選擇紅色的 Report（檢舉） 按鈕以繼續前往檢舉表單並進行填寫，然後選擇綠色的 Report Abuse（檢舉濫用） 按鈕。

但在行動應用程式中略有不同。選擇螢幕右上角的裝備圖示，然後選擇 Report Abuse（檢舉濫用） 。

檢舉使用者創作的廣告

若要檢舉網站上由其他使用者製作的廣告，請選擇位於廣告影像右側下方的 Report（檢舉） 。重新載入頁面後，填寫表單並選擇綠色 Report Abuse（檢舉濫用） 按鈕。
