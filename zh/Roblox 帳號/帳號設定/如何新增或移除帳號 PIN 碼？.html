如何新增或移除帳號 PIN 碼？

您可以選擇在 帳號 中新增一個四位數 PIN 碼鎖定您 的 設定。一旦 啟用 了 PIN 碼，您需要輸入它才能對帳號設定 做出 更新 。此功能可保護您 的 密碼、電子郵件地址及隱私權設定。

如何新增 PIN 碼？

  1. 登入 您的帳號
    （ 如果您無法登 入 ，請嘗試 找回密碼 的步驟。）
  2. 前往 帳號 「 設定 」
    • 瀏覽器： 尋找 位於網頁右上角的齒輪圖示
    • 行動裝置 App： 尋找 三個點圖示裡的「更多」
  3. 選擇「 安全」 標籤
  4. 選擇 啟用 PIN 碼
  5. 創建並確認您的新 PIN 碼

永遠不要向任何人透露您的 PIN 碼，它甚至比您的密碼還重要。

如何移除帳號 PIN 碼？

  1. 登入 您的帳號
    （ 如果您無法登 入 ，請嘗試 找回密碼 的步驟。）
  2. 選取帳號 「 設定 」
    • 瀏覽器： 尋找 位於網頁右上角的齒輪圖示
    • 行動裝置 App： 尋找 三個點圖示裡的「更多」
  3. 選擇「 安全」 標籤
  4. 選擇 停用 PIN 碼
  5. 輸入 目前 的 PIN 碼並選擇 「 解鎖 」

如何變更 PIN 碼？

若要變更 PIN 碼，首先按照 如何移除 PIN 碼 步驟操作。然後，按照 如何新增 PIN 碼 的步驟建立新的 PIN 碼。
