電腦硬件和操作系統要求

下面列出了 Roblox 的最低系統要求。如果您在玩遊戲時遇到問題，您可能需要對您的電腦進行升及。參閲 此文章 以了解行動裝置的系統要求。

操作系統要求

  • PC / Windows: Roblox 將安裝在 Windows 7、Windows 8/8.1 或者 Windows 10上。對於 Windows 8/8.1，您需要在桌面模式下運行 Roblox，因為目前不支援 Metro 模式（平鋪啟動屏幕）。
    注意 ：對於 Windows 7，瀏覽器必須是 IE10 或更高版本。
  • Mac ：Roblox 客戶端將安裝在 10.7（Lion）及以上版本 - 而 Roblox Studio 將安裝在 Mac OS 10.11（ El Capitan ）及以上版本。
  • Linux ：Roblox 不支援 Linux 操作系統。
  • 手機： 點擊這裡 查看 Roblox 手機的系統要求。
  • Chrome OS ：Chrome OS 版本 53 或更高的版本是必需的。您必須先在您的設備上啟用 Google Play 商店 。

注意：Roblox 視頻錄製功能僅適用於 Windows 7、Windows 8/8.1 和 Windows 10。最新發布的操作系統或預發布的操作系統版本可能不會立即得到支援。

系統硬件要求

  • 顯卡：Roblox 至少需要 DirectX 版本 10.0 和 Shader Model 版本 2.0。為了獲得最佳表現，我們建議使用電腦少於 5 年的專用顯卡，或者使用筆記本電腦未滿 3 年的內置顯卡。
  • 處理器：Roblox 建議您使用最新的處理器（2005+），時鐘速度為 1.6 Ghz 或更高。舊 AMD 處理器會出現一些問題。
  • RAM 或記憶體：Roblox 建議您在 Windows 7、Windows 8 或 Windows 10 上至少有 1 GB 內存。
  • >存儲空間：Roblox 建議您至少有 20mb 的系統存儲空間來安裝 Roblox。
  • 行動裝置： 點擊此處 查看 Roblox 行動裝置的系統要求。

互聯網要求

對於單次使用的家庭，Roblox 建議使用最少為 4-8 Mbps 的網絡連接。在較大的家庭中，更高帶寬是可取的。

其他説明

技術支援：電腦、行動裝置、Xbox、虛擬現實文章
