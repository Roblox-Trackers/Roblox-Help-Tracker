體驗設定和協助

在玩體驗時，您可以變更設定，並了解如何執行操作，如跳躍或旋轉像機。

設定

此選單包含音量、相機、圖形和輔助功能等功能的設定。

只要體驗開發人員沒有鎖定特定的相機模式，您就可以在經典模式和追蹤模式之間切換。

  • 在 經典模式 下，除非手動調整，否則相機將保持固定在一個位置。 若要變更頭像的有利位置，請按住滑鼠右鍵並拖曳它。
  • 在 跟隨模式 下，當您向右或向左移動時，相機將隨著您的頭像一起旋轉，以幫助您保持預期目標在視野中。
    • Shift 鎖定開關可讓您按下鍵盤上的 Shift 鍵在相機模式之間切換。

我們目前提供三種輔助功能設置，以更好地自訂您在 Roblox 上的體驗。

  • 減少運動 停用或減少應用程式內和體驗內的某些螢幕效果。 例如，當遊戲選單開啟時，它不會滑到螢幕上，而是在啟用「減少運動」時淡入。
  • 背景透明度 透過讓 UI 元素上的背景（例如聊天視窗背景）更加不透明來提高對比度和易讀性。 這不會使 UI 元素比平常更透明。
  • UI 導航切換 控制是否可以使用 \ 鍵進入和退出 UI 導航模式。 如果您經常意外進入 UI 導航模式，或者不想使用 UI 導航模式功能，則可以停用此設定。

協助

此菜單包含鍵盤快捷鍵、鼠標指令、以及行動裝置手勢，因此您就可以在體驗中導航並使用所有可用的功能。
