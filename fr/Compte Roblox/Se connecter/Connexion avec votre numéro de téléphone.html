Connexion avec votre numéro de téléphone

Vous pouvez vous connecter à votre compte Roblox avec votre numéro de téléphone dans les applications mobiles. C'est une option supplémentaire, car vous pouvez également vous connecter avec votre nom d'utilisateur et votre mot de passe.

Entrez le numéro de téléphone et le mot de passe du compte.

Remarque : Cette fonctionnalité est uniquement disponible dans certaines applications Roblox, et vous devez disposer d'un numéro de téléphone vérifié sur votre compte.

Je n'arrive pas à me connecter avec le numéro de téléphone

Veuillez essayer ces instructions si vous rencontrez un souci :

  • Assurez-vous d'entrer votre numéro de téléphone exactement comme vous l'avez saisi sur le compte
  • Connectez-vous avec votre nom d'utilisateur ou avec votre adresse e-mail, comme alternative
