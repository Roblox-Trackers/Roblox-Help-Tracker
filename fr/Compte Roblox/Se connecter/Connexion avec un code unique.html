Connexion avec un code unique

Le code unique vous permet de vous connecter sans saisir votre nom d'utilisateur et votre mot de passe. Pour le moment, les codes uniques ne sont disponibles que pour quelques utilisateurs.

Remarque : Pour utiliser cette fonctionnalité, votre compte doit comporter une adresse électronique vérifiée. Vous pouvez accéder à la page Informations sur votre compte pour ajouter et vérifier votre adresse e-mail. Vous pouvez lire cet article d'aide sur la façon de vérifier l'adresse e-mail du compte.

Comment accéder au compte avec un code unique

  1. Appuyez sur le bouton Envoyer un code unique sur la page de connexion.
  2. Saisissez l'adresse e-mail qui est vérifiée sur votre compte. Nous vous envoyons un code unique à six chiffres.
  3. Saisissez le code unique pour vous connecter (si votre adresse électronique est vérifiée sur plusieurs comptes, vous pouvez en choisir un).

Je n'ai pas reçu de code unique

Essayez ces étapes si vous n'avez pas reçu l'e-mail contenant le code unique.

  • Vérifiez votre dossier spam ou courrier indésirable.
  • Appuyez sur le bouton Renvoyer, puis vérifiez à nouveau votre boîte de réception.
  • Assurez-vous que vous avez saisi votre adresse e-mail exactement comme elle figure sur votre compte.
  • Assurez-vous que votre adresse e-mail est vérifiée. Vous pouvez lire cet article d'aide sur la façon de vérifier l'adresse électronique du compte.

Si aucune de ces étapes ne fonctionne, veuillez vous connecter en utilisant votre mot de passe. Vous devrez peut-être attendre un peu avant que les codes à usage unique ne fonctionnent à nouveau pour votre courrier électronique.

J'ai reçu un code unique mais il ne fonctionne pas

Essayez ces étapes si votre code unique ne fonctionne pas.

  1. Assurez-vous de saisir le code à usage unique sur le même appareil que celui à partir duquel vous l'avez demandé.
  2. Appuyez sur le bouton Renvoyer et réessayez avec un nouveau code.

Si aucune de ces étapes ne fonctionne, veuillez vous connecter en utilisant votre mot de passe. Vous devrez peut-être attendre un peu avant que les codes à usage unique ne fonctionnent à nouveau pour votre courrier électronique.

Conseils de sécurité

  • Ne partagez jamais un code unique avec qui que ce soit. Roblox ne vous contactera jamais pour vous le demander.
  • Si vous recevez un e-mail contenant un code à usage unique sans l'avoir demandé, vous pouvez l'ignorer sans risque.
  • Si vous recevez plusieurs courriels indésirables contenant des codes uniques, nous vous conseillons vivement de modifier l'adresse électronique de votre compte et d'ajouter la vérification en 2 étapes par l'application Authenticator. .
