Connexion rapide

Notre fonction de connexion rapide vous permet de vous connecter à un nouvel appareil à partir d'un autre appareil auquel vous êtes déjà connecté, sans avoir à saisir à nouveau votre mot de passe.

Pour connecter à un nouvel appareil

Pour vous connecter, vous devrez suivre les deux séries d'instructions

Générer un code à usage unique

  1. Sur votre nouvel appareil, ouvrez la page de connexion.
  2. Cliquez sur le bouton Connecter avec un autre appareil .
  3. Un code à usage unique s'affichera, qui n'est valable que 5 minutes.

Authentifier le code à usage unique

  1. Sur votre appareil connecté, accédez à Paramètres du compte , puis Connexion rapide .
  2. Après avoir tenu compte des avertissements, saisissez le code à usage unique affiché sur votre nouvel appareil, puis cliquez sur Entrer . .
  3. Vérifiez tous les détails avant de sélectionner Accorder l'accès complet .
  4. Un message de réussite apparaîtra une fois que votre nouvel appareil sera connecté.

Dépannage des codes QR

Si vous voyez une option pour ouvrir une session avec un code QR et que vous rencontrez des difficultés, procédez comme suit :

  1. Assurez-vous que la dernière version de l'application Roblox est installée.
    1. Vous avez également besoin d'une autre application pour scanner le code QR. Essayez l'application appareil photo de votre dispositif ou n'importe quel scanner de code QR à votre choix !
  2. Assurez-vous que vous êtes connecté à l'appareil à partir duquel vous scannez le code QR.
  3. Sur certains téléphones (par ex. Google Pixel), l’application intégrée de l’appareil photo pourrait ne pas gérer le code QR correctement. Essayez d’installer un autre scanner de code QR ou d’ouvrir une session en saisissant plutôt le code à six caractères.

REMARQUE : Cette fonctionnalité est actuellement indisponible sur Xbox. Cette fonctionnalité fonctionne seulement si vos appareils sont situés à proximité les uns des autres.

Et enfin, cette fonctionnalité peut ne pas être disponible pour tous les joueurs en ce moment.
