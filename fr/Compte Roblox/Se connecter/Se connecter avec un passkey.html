Se connecter avec un passkey

Qu'est-ce qu'un passkey ?

Le passkey est une alternative simple et sécurisée aux mots de passe. Avec le passkey, vous pouvez vous connecter de la même manière que pour débloquer votre appareil, sans avoir à vous rappeler votre mot de passe. Le passkey se sert d'une cryptographie asymétrique afin de générer des paires de clés cryptographiques uniques à votre compte et spécifiques à Roblox.


Une fois créés, les passkeys se synchronisent sur tous vos appareils avec les mêmes comptes Apple, Google ou Windows ou avec le gestionnaire de mots de passe que vous utilisez, ce qui garantit que vos clés numériques sont toujours à portée de main. Pour plus d'informations sur les passkeys, veuillez vous référer aux ressources fournies par Google , Apple et Microsoft .

Les passkeys vous protègent des tentatives de hameçonnage et bourrage d'identifiants. Nous encourageons tout le monde à utiliser les passkeys pour éviter les conflits et améliorer la sécurité.

Comment ajouter un passkey

L'ajout d'un passkey est un processus simple en trois étapes :

  • Allez dans les réglages de votre navigateur Web ou app iOS. Connectez-vous si nécessaire.
    • Remarque : l'assistance Android sera bientôt disponible.
  • Sous Informations sur le compte > Méthodes de connexion, appuyez sur le bouton Ajouter une clé d’accès .
    • Remarque : vous devrez probablement vérifier votre identité avec votre mot de passe et un code unique envoyé par e-mail.
  • Pour terminer l'ajout d'un passkey, vous devrez débloquer votre appareil.

Remarque : pour votre sécurité, n'ajoutez des passkeys que sur vos appareils personnels.

Utiliser votre passkey pour vous connecter

Une fois le passkey ajouté, vous pourrez vous connecter à Roblox de la même façon que vous débloquez votre appareil : avec Face ID, Touch ID ou à l'aide du pavé numérique.

Utiliser votre passkey pour vous connecter sur un autre appareil

Si vous possédez un passkey sur votre appareil mobile, vous pouvez l'utiliser pour vous connecter sur votre ordinateur :

  • Cliquez sur le champ Nom d'utilisateur et sélectionnez Passkey d'un appareil à proximité... ou Utiliser un passkey différent sur le formulaire de connexion du navigateur Web.
  • Un code QR sera affiché à l'écran. Scannez ce code QR avec votre appareil mobile et suivez les instructions à l'écran.

Gestion de vos passkeys

Bien que les passkeys offrent une sécurité additionnelle, nous ne recommandons pas leur ajout sur des appareils partagés. Si vous avez un passkey configuré sur un appareil partagé, les personnes pouvant débloquer l'appareil pourront avoir accès à votre compte.

Dans ce cas, retirez immédiatement le passkey depuis les réglages.

  • Allez dans les réglages de votre navigateur Web ou app iOS. Connectez-vous à votre compte si nécessaire.
    • Remarque : la prise en charge d’Android sera bientôt disponible.
  • Sous Informations sur le compte > Méthodes de connexion, appuyez sur le bouton Ajouter une clé d’accès .
    • Note : Si vous avez déjà ajouté un passe, vous verrez le bouton "Gérer" à la place.
  • Appuyez sur l'icône de poubelle à côté du passkey pour le retirer.
    • Vous devrez probablement vérifier votre identité à l'aide de mots de passe ou de code unique par e-mail.

Remarque : après avoir retiré un passkey de Roblox, et jusqu'à ce qu'il soit retiré des réglages de votre appareil, vous pourrez toujours voir l'option de connexion avec le passkey supprimé.

Pour les utilisateurs ayant activé la vérification en 2 étapes

Si vous avez activé la vérification en 2 étapes sur votre compte et choisi de vous connecter à l'aide d'un passkey, vous n'aurez pas besoin de réaliser la deuxième étape. C'est parce qu'une connexion réussie à l'aide d'un passkey confirme que vous êtes bel et bien en possession de l'appareil. Vous pouvez avoir la certitude que la sécurité de votre compte n'est pas compromise.

Remarque : si vous vous connectez avec autre chose qu'un passkey, vous devrez compléter la vérification en 2 étapes comme d'habitude.

FAQ

Quels appareils sont compatibles avec les passkeys ?

Vous pouvez dorénavant utiliser les passkeys pour vous connecter à vos comptes Roblox sur les navigateurs Web et appareils iOS. Les appareils suivants utilisent les passkeys :

  • Appareil mobile avec iOS 16 ou supérieur.
  • Ordinateur avec Windows 10, macOS Ventura, ou ChromeOS 109.
    • Navigateurs compatibles : Chrome version 109 ou supérieure, Safari version 16 ou supérieure, Edge version 109 ou supérieure.
  • Clés matérielles conformes au standard FIDO2.

Que faire si je ne trouve aucun passkey sur l'écran de connexion ?

Si vous avez déjà configuré un passkey sur votre compte sans pouvoir l'utiliser pour vous connecter, suivez les étapes ci-dessous selon votre appareil :

  • Si votre appareil utilise le même système d'exploitation que celui avec le passkey sauvegardé (par exemple, d'un appareil iOS à un autre), vous devrez vous connecter au même iCloud ou compte Google utilisé lors de la configuration du passkey. Veillez également à ce que votre appareil soit à jour.
  • Si votre appareil utilise un système différent que celui avec le passkey sauvegardé (par exemple, d'un appareil iOS à un ordinateur Windows), les passkeys ne seront pas synchronisés. Vous pouvez néanmoins récupérer l'accès à votre compte via d'autres méthodes, comme la connexion rapide, le code unique par e-mail ou le mot de passe. Après votre connexion, nous recommandons l'ajout d'un passkey sur votre nouvel appareil pour faciliter les accès ultérieurs.

Que faire si je n'arrive pas à me connecter avec un passkey ?

Ce problème survient si vous tentez d'utiliser un passkey ayant précédemment été retiré de Roblox. À moins que vous ne supprimiez le passkey depuis les réglages de l'appareil, vous devriez le voir comme option de connexion. Consultez les articles d'aide sur Apple , Google et Microsoft pour plus d'informations sur la gestion des passkeys sur ces plateformes.

Tentez d'autres méthodes de connexion, comme la connexion rapide, le code unique par e-mail ou le mot de passe. Si besoin est, vous devrez procéder à une récupération de compte.
