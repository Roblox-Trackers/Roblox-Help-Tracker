J’ai oublié mon nom d’utilisateur

Cet article vous aidera,  avec des étapes à suivre, à retrouver votre nom d’utilisateur Roblox oublié.

Trouver le nom d’utilisateur en utilisant l’adresse email

  1. Sélectionnez Nom d'utilisateur ou mot de passe oublié sur la page de connexion
  2. La page doit utiliser par défaut l'onglet Mot de passe oublié ; vous devez sélectionner l'onglet Nom d'utilisateur.
  3. Entrez votre adresse email dans la case Email et appuyez sur le bouton bleu Soumettre . Vous allez voir ce message de confirmation.
    S’il vous plaît assurez-vous de taper votre adresse e-mail exactement comme vous l’avez entré sur votre compte.
  4. Cela enverra un message à l’adresse électronique fournie avec une liste de tous vos comptes.
  5. Après avoir récupéré votre nom d’utilisateur, cliquez ici pour vous connecter.

Si, après quelques minutes, le courriel ne semble pas arriver, s’il vous plaît assurez-vous de vérifier vos dossiers de corbeille/spam .

Trouver un nom d'utilisateur en utilisant un numéro de téléphone

  1. Sélectionnez N om d'utilisateur ou mot de passe oublié sur la page de connexion.
  2. Sélectionnez Utilisez le numéro de téléphone pour récupérer le nom d'utilisateur.
  3. Sélectionnez votre code pays.
  4. Entrez votre numéro de téléphone dans la case Numéro de téléphone et appuyez sur le bouton bleu Soumettre .
  5. Cela enverra un SMS avec vos noms d'utilisateurs sur votre téléphone s'il avait déjà été vérifié sur votre compte
  6. Après avoir récupéré votre nom d'utilisateur, cliquez ici pour vous connecter.

D'autres façons

Ces méthodes ne fonctionneront que si un email ou un numéro de téléphone a été ajouté au compte avant que vous n'ayez oublié votre nom d'utilisateur. Le seul autre moyen de localiser le nom d'utilisateur consiste à effectuer un achat sur Roblox. Certains reçus d'achat incluent le nom d'utilisateur.

Malheureusement, sans email, sans numéro de téléphone ni historique d’achat, il n’y a aucun moyen de récupérer le compte.
