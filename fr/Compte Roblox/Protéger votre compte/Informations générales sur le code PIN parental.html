Informations générales sur le code PIN parental

Les parents et les utilisateurs ont la possibilité d'ajouter un code PIN à quatre chiffres pour verrouiller le compte. Une fois un code PIN ajouté, il sera nécessaire afin d'apporter des modifications aux paramètres du compte. Cette fonctionnalité empêche la modification du mot de passe, de l'adresse e-mail, des paramètres de confidentialité, ainsi que des autres paramètres.
