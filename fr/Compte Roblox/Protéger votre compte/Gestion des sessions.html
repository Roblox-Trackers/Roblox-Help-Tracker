Gestion des sessions

Utilisez Où vous êtes connecté pour gérer toutes vos sessions connectées

La section Où vous êtes connecté dans les paramètres de sécurité de votre compte affichera toutes les sessions connectées (ainsi que l'emplacement régional approximatif, la durée de la dernière activité et l'appareil pour chaque session). À partir de cette vue, vous pouvez choisir de vous déconnecter des sessions individuelles de votre choix ou de vous déconnecter de toutes les sessions autres que celle dans laquelle vous vous trouvez actuellement.

Pourquoi est-ce utile ?

  • Dans un écran consolidé, vous pouvez afficher toutes vos sessions connectées, vous aidant ainsi à identifier les sessions potentiellement suspectes.
  • Si vous pensez que l'une de vos sessions connectées peut être suspecte, vous pouvez vous déconnecter de chacune de ces sessions individuellement pour sécuriser votre compte.

Comment trouver Où vous êtes connecté

  1. Connectez-vous à votre compte
  2. Accédez aux Paramètres du compte
    • Navigateur - l'icône d'engrenage située dans le coin supérieur droit du site
    • Applications mobiles - l'icône à trois points pour Plus
  3. Sélectionnez l'onglet Sécurité
  4. Faites défiler vers le bas pour voir la section Où vous êtes connecté

Remarque : Dans certains cas, les sessions peuvent s'afficher avec des informations incomplètes ou sans aucune information, auquel cas elles s'afficheront comme « Inconnu » Il s'agit de sessions plus anciennes avec des données inconnues en raison de leur inactivité.

Comment se déconnecter de sessions de connexion individuelles à partir de l'écran Où vous êtes connecté

  1. Sélectionnez une session dans la section Où vous êtes connecté
  2. Vérifiez les détails de l'appareil, son emplacement et l'horodatage de la dernière activité de la session
  3. Sélectionnez Se déconnecter de la session

Comment se déconnecter de toutes les sessions connectées à partir de l'écran Où vous êtes connecté

  1. Dans la section Où vous êtes connecté , sélectionnez le bouton Se déconnecter de toutes les autres sessions au bas de la section.
