Ajouter la vérification en 2 étapes à votre compte

Nous vous recommandons vivement d'activer la fonction de la vérification en deux étapes pour améliorer la sécurité de votre compte. Lorsque vous vous connectez à partir d'un nouvel appareil, vous devez saisir un code de sécurité unique provenant d'une application d'authentification ou de l'e-mail que Roblox vous envoie.

Pourquoi est-ce important ?

Cette fonction présente plusieurs avantages pour protéger votre compte.

  • Elle permet de s'assurer que personne d'autre ne peut se connecter à votre compte, même s'il connaît le mot de passe.
  • Comme vous êtes le seul à avoir accès à l'application d'authentification ou à votre compte de messagerie, vous êtes le seul à pouvoir obtenir le code de sécurité.
  • Une sécurité supplémentaire pour que vous ne perdiez pas l'accès à vos expériences, vos Robux et encore plus.

Activation de la vérification en deux étapes

  • Connectez-vous à votre compte.
    • Si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe .
  • Vous devez disposer d’une adresse e-mail vérifiée pour activer cette fonctionnalité.
    • Voici les étapes à suivre pour vérifier votre adresse e-mail .
  • Accédez aux paramètres du compte .
    • Navigateur - trouvez l’icône d’engrenage (roue dentée) située dans le coin supérieur droit du site.
    • Applications mobiles : Trouvez l'icône des trois points pour plus d'options .
  • Sélectionnez l'onglet Sécurité .
  • Suivez les étapes ci-dessous pour activer la vérification en deux étapes pour le type que vous souhaitez.

Remarque : Vous ne pouvez activer qu'une application d'authentification ou utiliser une adresse e-mail pour les codes de vérification.

Configuration de la vérification en 2 étapes par une application d'authentification

  1. Basculez sur L'application d'authentification (très sécurisée) .
    • Si le compte dispose d'un code PIN, il vous sera demandé de le saisir.
  2. Saisissez le mot de passe de votre compte pour confirmer votre demande.
  3. Scannez le code QR ou entrez manuellement le code de configuration dans l'application Authenticator.
    • Assurez-vous que vous disposez d'une application Authenticator sur votre appareil mobile.
      • Exemples : Google Authenticator, Microsoft Authenticator ou Twilio Authy
  4. Les codes de récupération seront automatiquement générés pour vous. Assurez-vous de les enregistrer dans un endroit sûr et de ne les partager avec personne. Si vous perdez votre application Authenticator et vos codes de récupération, vous ne pourrez pas accéder à votre compte.

Configurer la vérification en deux étapes par clés de sécurité

Notez que les clés de sécurité ne sont actuellement prises en charge que sur les navigateurs Web. Vous ne pourrez utiliser votre clé de sécurité que lorsque l'application Authenticator est activée et lorsque vous vous connectez sur un navigateur Web et non sur l'application Roblox.

Suivez les instructions sous "Configurer la vérification en 2 étapes par l'application Authenticator". L'Authenticator 2FA doit être activé pour configurer les clés de sécurité.

  1. Basculez sur "Clés de sécurité sur le Web uniquement (très sécurisé)"
  2. Suivez les instructions de votre navigateur pour vérifier la clé de sécurité de votre choix
    • Vous pouvez utiliser une clé physique ou une méthode gérée par le matériel de votre appareil comme la reconnaissance faciale ou un scanneur d'empreintes digitales
    • Exemples de clés de sécurité physiques : YubiKey 5, YubiKey 5C, Google Titan Security Key
    • Les options pour les clés de sécurité varient en fonction du navigateur (Safari, Chrome, Firefox) et du système d'exploitation (Windows ou MacOS) que vous utilisez.
  3. Saisissez un nom pour votre clé afin de vous rappeler quelle clé vous avez utilisée
    • Exemple de nom de clé : “Blue YubiKey 5”
  4. Vous pouvez choisir de répéter ces étapes pour ajouter plus de clés ou fermer la fenêtre si vous avez terminé.

Configuration de la vérification en deux étapes par adresse électronique

  1. Basculer sur E-mail (sécurisé) .
    • Si le compte dispose d'un code PIN, il vous sera demandé de le saisir.
  2. Saisissez le mot de passe de votre compte pour confirmer votre demande.

Désactivation de la vérification en 2 étapes

  1. Connectez-vous à votre compte.
    • Si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe .
  2. Accédez aux paramètres du compte .
    • Navigateur - trouvez l’icône d’engrenage (roue dentée) située dans le coin supérieur droit du site.
    • Applications mobiles : recherchez l’icône à trois points pour en savoir plus .
  3. Sélectionnez l'onglet Sécurité .
  4. Sélectionner désactiver Vérification en deux ėtapes .
  5. Entrez le mot de passe de votre compte pour terminer la désactivation de la fonctionnalité.

Tous les codes de récupération créés précédemment seront également désactivés si vous désactivez la vérification en 2 étapes. Pour obtenir de l’aide sur la réinitialisation du mot de passe du compte, consultez l’article mot de passe oublié

Obtenir des codes de récupération

  1. Connectez-vous à votre compte
  2. Accédez aux Paramètres du compte .
    • Navigateur web : Trouvez l'icône de la roue dentée située dans le coin supérieur droit du site.
    • Applications mobiles : trouvez l’icône à trois points successifs plus d’options .
  3. Sélectionnez l’onglet Sécurité .
  4. Cliquez sur Générer sous la section codes de récupération.
  5. Entrez le mot de passe de votre compte dans le cadre du processus de configuration.
  6. Assurez-vous d’enregistrer les codes dans un endroit sûr, car vous ne pouvez les voir qu’une seule fois.

Chaque code ne peut être utilisé qu’une seule fois. Si vous perdez vos codes de récupération, vous pouvez répéter ce processus pour créer de nouveaux codes. Tous les codes de récupération créés précédemment seront désactivés lorsque vous créerez de nouveaux codes.
