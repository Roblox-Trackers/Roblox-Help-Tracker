Ajouter la vérification en 2 étapes à votre compte

Qu’est-ce que la vérification en 2 étapes ?

Cette fonctionnalité garantit que personne d’autre ne peut se connecter à votre compte, même s’il connaît le mot de passe. Lorsque vous vous connectez à partir d’un nouvel appareil, vous entrez un code de sécurité unique à partir d’une application d’authentification ou que Roblox vous envoie par e-mail. Étant donné que vous seul avez accès à ces sources, vous seul pourrez obtenir le code de sécurité.

Activation de la Vérification en Deux-Étapes

  1. Connectez-vous à votre compte (si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe . )
  2. Vous devez disposer d’un e-mail vérifié pour activer cette fonctionnalité (voici les étapes à suivre pour vérifier votre adresse e-mail.)
  3. Aller sur paramètres du compte
    • Navigateur - trouvez l’icône d’engrenage (roue dentée) située dans le coin supérieur droit du site
    • Applications mobiles - trouvez l’icône constituée de trois points successifs pour plus d’options
  4. Sélectionnez l'onglet Sécurité
  5. Activez la vérification en 2 étapes selon le type que vous souhaitez .
    • Si vous choisissez une application « Authentificateur », assurez-vous de l’avoir sur votre appareil. Parmi les applications d’authentification vous trouverez entre autres, Google Authenticator, Microsoft Authenticator et Twilio Authy.
  6. Suivez les étapes à l’écran pour terminer la configuration.
    • Si le compte a un code PIN activé, il vous sera demandé de le saisir.
    • Pour configurer une application « Authentificateur », vous devrez scanner le code QR ou entrer manuellement le code de configuration dans l’application.

Désactivation de la vérification en 2 étapes

  1. Connectez-vous à votre compte (si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe.)
  2. Aller aux paramètres du compte
    • Navigateur - trouvez l’icône d’engrenage (roue dentée) située dans le coin supérieur droit du site
    • Applications mobiles - trouvez l’icône constituée de trois points successifs pour plus d’options
  3. Sélectionnez l'onglet Sécurité
  4. Sélectionnez Désactiver la vérification en 2 étapes
  5. Entrez le mot de passe de votre compte pour terminer la désactivation de la fonctionnalité

Tous les codes de récupération créés précédemment seront également désactivés si vous désactivez la vérification en 2 étapes. Pour obtenir de l’aide sur la réinitialisation du mot de passe du compte, consultez l’article Mot de passe oublié.

Obtention de codes de récupération

  1. Connectez-vous à votre compte
  2. Aller sur paramètres du compte
    • Navigateur - trouvez l’icône d’engrenage (roue dentée) située dans le coin supérieur droit du site
    • Applications mobiles - trouvez l’icône constituée de trois points successifs pour plus d’options
  3. Sélectionnez l’onglet Sécurité
  4. Cliquez sur Générer sous la section codes de récupération.
  5. Entrez le mot de passe de votre compte dans le cadre du processus de configuration.
  6. Assurez-vous d’enregistrer les codes dans un endroit sûr, car vous ne pouvez les voir qu’une seule fois.

Chaque code ne peut être utilisé qu’une seule fois. Si vous perdez vos codes de récupération, vous pouvez répéter ce processus pour créer de nouveaux codes. Tous les codes de récupération créés précédemment seront désactivés lorsque vous créerez de nouveaux codes.

Eligibilité à la Restauration de Compte

Pour être éligible aux restaurations de compte, nous avons besoin que vous disposiez d’une application d’authentification configurée pour protéger votre compte. En outre, vous devez informer Roblox dans les 30 jours suivant la compromission du compte. Malheureusement, si nous ne sommes pas informés dans ce délai, nous ne sommes généralement pas en mesure de récupérer l’inventaire du compte.
