Ajouter la vérification en 2 étapes à votre compte

La vérification en deux étapes est une couche de sécurité supplémentaire pour votre compte. Ceci est particulièrement utile pour empêcher les attaquants d’accéder à votre compte en cas de vol de votre mot de passe.

Méthodes de vérification en 2 étapes

Il existe différentes méthodes de vérification en deux étapes proposées par Roblox. Pour le moment, vous ne pouvez choisir qu'une seule méthode de vérification en deux étapes à la fois.

Pour activer la vérification en deux étapes sur votre compte :

  1. Connectez-vous à votre compte Roblox
    • Si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe
  2. Accédez aux Paramètres du compte
    • Navigateur : recherchez l'icône en forme d'engrenage dans le coin supérieur droit du site
    • Application mobile : recherchez l'icône à trois points et appuyez sur l'icône en forme d'engrenage
  3. Accédez à l' onglet Sécurité
  4. Activez le bouton correspondant à la méthode de vérification en deux étapes souhaitée et suivez les instructions à l'écran. Les instructions varient en fonction de votre choix de vérification en deux étapes.
    • IMPORTANT : Vous devez disposer d'une adresse e-mail vérifiée sur votre compte pour activer la vérification en deux étapes.

Configurer la Vérification en deux étapes par adresse E-mail

Une fois que vous avez activé la vérification en 2 étapes par e-mail, il vous sera demandé de saisir un code chaque fois que vous vous connecterez avec votre mot de passe.

Vous pouvez trouver le code dans la boîte de réception de l'e-mail vérifié que vous avez sur votre compte Roblox. Une fois que vous aurez entré le code sur votre écran Roblox, vous serez connecté avec succès.

Veuillez noter que si vous vous connectez via Email OTP ou mot de passe, votre vérification en 2 étapes sera contournée.

Configurer la vérification en 2 étapes par l'application Authenticator

Pour activer la vérification en 2 étapes par l'application Authenticator, vous devez disposer d'une application Authenticator sur votre appareil mobile. Si vous ne disposez pas d'une Application d'authentification, vous pouvez en télécharger une depuis le Magasin d'applications. Certaines applications que vous pouvez choisir sont Google Authenticator, Microsoft Authenticator, Duo, Twilio Authy.

Une fois que vous avez activé la vérification en 2 étapes par l'application Authenticator, il vous sera demandé de saisir un code chaque fois que vous vous connecterez avec votre mot de passe ou que vous vous connecterez avec Email OTP.

Vous pouvez trouver le code dans l'application Authenticator de votre choix sur votre appareil mobile. Une fois que vous aurez entré le code sur votre écran Roblox, vous serez connecté avec succès.

Notez que si vous vous connectez via un mot de passe, votre vérification en deux étapes sera contournée.

Configurer la Vérification en deux étapes par Clés de sécurité

Pour activer la vérification en 2 étapes par clés de sécurité, vous devez également activer la vérification en 2 étapes par l'application Authenticator sur votre compte. En effet, les clés de sécurité ne sont prises en charge que sur les navigateurs Web.

Votre clé de sécurité peut être une clé physique comme Yubikey 5, la clé de sécurité Google Titan, ou il peut s'agir d'une méthode biométrique gérée par le matériel de votre appareil comme la reconnaissance faciale ou le scanner d'empreintes digitales. Votre option de clé de sécurité varie en fonction du navigateur, du système d'exploitation et de l'appareil à partir duquel vous vous connectez.

Notez que si vous vous connectez via un mot de passe, votre Vérification en deux étapes sera contournée.

Vérification en 2 étapes par invites multi-appareils

Remarque : Il s'agit d'une fonctionnalité expérimentale et n'est pas encore disponible pour tous les utilisateurs.

En tant qu'utilisateur, vous n'avez pas la possibilité d'activer explicitement la Vérification en deux étapes par des invites multi-appareils. Vous verrez ces invites si aucune des méthodes de Vérification en deux étapes répertoriées ci-dessus n'est activée sur votre compte et si Roblox détecte qu'il y a une tentative de connexion suspecte à votre compte.

Si vous recevez une Vérification en 2 étapes par invite multi-appareils, il vous sera demandé d'utiliser un appareil mobile sur lequel vous êtes déjà connecté à votre compte Roblox pour approuver ou rejeter la tentative de connexion. Vous aurez la possibilité de consulter les détails de la tentative de connexion, tels que le type d'appareil, l'emplacement régional, l'adresse IP, la date et l'heure.

Sélection d'une autre façon de vérifier votre tentative de connexion

Si vous êtes confronté à une invite de Vérification en deux étapes sur plusieurs appareils, vous aurez la possibilité de vérifier votre connexion par d'autres moyens.

Envoyez-moi un code par e-mail

Si vous disposez d'une adresse e-mail vérifiée sur votre compte, vous verrez une option « Envoyez-moi un code par e-mail ». Choisir cette option enverra un code unique à votre adresse e-mail. Entrez ce code sur votre écran Roblox pour vous connecter avec succès. N'oubliez pas de ne partager ce code avec personne.

Code de sauvegarde

Vous avez également la possibilité de saisir votre Code de sauvegarde enregistré pour vous connecter avec succès. N'oubliez pas de ne partager ce code avec personne.

Contactez le support

Si vous ne voyez aucune option que vous pouvez utiliser pour vous connecter, contactez le Service client .

Nous vous recommandons fortement d'ajouter une adresse e-mail vérifiée à votre compte. Cela vous aidera au cas où vous souhaiteriez vérifier votre tentative de connexion avec un code e-mail au lieu de l'invite de Vérification multi-appareils en 2 étapes.

Tentative de connexion refusée

Si vous recevez une invite de Vérification en 2 étapes sur plusieurs appareils, mais que vous n'avez pas tenté de vous connecter, refusez la tentative de connexion.

Cela signifie que quelqu'un pourrait connaître votre mot de passe et l'utiliser pour se connecter à votre compte. Nous vous recommandons de modifier votre mot de passe pour sécuriser votre compte.

Tentative de connexion expirée

Si vous attendez trop longtemps pour approuver ou refuser la tentative de connexion, celle-ci expirera. Réessayez simplement la tentative de connexion.

Obtenir des codes de sauvegarde

  1. Connectez-vous à votre compte
  2. Accédez au compte Paramètres
    • Navigateur Web : recherchez l' icône en forme d'engrenage située dans le coin supérieur droit du site.
    • Applications mobiles : recherchez l'icône à trois points pour Plus .
  3. Sélectionnez l'onglet Sécurité
  4. Cliquez sur Générer dans la section des codes de récupération
    • Navigateur Web : il se trouve sur le côté droit de la page.
    • Applications mobiles : elles se trouvent dans la section des codes de récupération.
  5. Entrez le mot de passe de votre compte dans le cadre du processus de configuration.
  6. Assurez-vous d’enregistrer les codes dans un endroit sûr, car vous ne pouvez les voir qu’une seule fois.

Chaque code ne peut être utilisé qu’une seule fois. Si vous perdez vos codes de récupération, vous pouvez répéter ce processus pour créer de nouveaux codes. Tous les codes de récupération créés précédemment seront désactivés lorsque vous créerez de nouveaux codes.
