Difficultés avec la Vérification en Deux-Étapes

Je ne reçois pas l’e-mail de la Vérification en deux-étapes

  1. Si vous avez plusieurs adresses e-mail, veuillez confirmer que vous sélectionnez bien la même adresse que celle que vous avez enregistrée sur votre compte Roblox.
  2. Vérifiez votre  dossier spam, vos courriers indésirables, vos promotions, votre dossier social ou tout autre dossier pour voir si le message a été filtré par votre fournisseur de messagerie.
    • Si vous trouvez effectivement votre message dans le dossier spam, marquez-le comme Non Spam pour éviter le filtrage dans le futur.
    • Ajoutez l’adresse no-reply@roblox.com à vos contacts e-mail.
  3. Utilisez la fonction Renvoyer le Code ou Recommencer sur l'écran de connexion de la Vérification en Deux-Étapes.

Si vous ne recevez toujours pas le code en deux-étapes après avoir suivi toutes ces étapes, veuillez contacter le Service Clients.

Mon code pour la Vérification en deux-étapes ne fonctionne pas

Les codes pour la Vérification en deux-étapes expirent après 15 minutes et ne seront plus valides si le code a été entré incorrectement sur le compte en question.

Utilisez la fonction Renvoyez le Code ou Recommencer sur l'écran de connexion de la Vérification en deux-étapes afin de demander un nouveau code. Si vous n'arrivez toujours pas à vous connecter avec un nouveau code, veuillez contacter le Service Clients .
