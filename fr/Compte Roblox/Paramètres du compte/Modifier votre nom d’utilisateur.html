Modifier votre nom d’utilisateur

Sur Roblox, vous pouvez modifier le nom d'utilisateur de votre compte dans les paramètres du compte. Il en coûte Robux pour effectuer ce changement. Cependant votre Nom d'Affichage est une option gratuite pour modifier la façon dont votre nom apparaît dans une expérience ou dans un jeu.

Comment puis-je changer le nom d’utilisateur ?

  1. Connecte-toi à ton compte
    (si tu ne parviens pas à te connecter, essaye de suivre ces étapes pour récupérer ton mot de passe .)
  2. Accède aux Paramètres du compte.
    • Dans le navigateur: recherche l'icône en forme d'engrenage située dans le coin supérieur droit du site.
    • Dans les applications mobiles : recherche l’icône à trois points pour accéder à Plus.
  3. Sélectionne l’onglet Informations du compte.
  4. Sélectionne le bouton Modifier le nom d’utilisateur (l’icône ressemble à un papier et un crayon).
    • Saisis ton nouveau nom d’utilisateur.
    • Saisis ton mot de passe Roblox actuel.
  5. Sélectionne le bouton Acheter.

Tu peux désormais te connecter avec le nouveau nom d’utilisateur et ton mot de passe actuel.

Noms précédemment utilisés

Une liste des noms d’utilisateur précédemment utilisés est disponible sur les pages Paramètres du compte et Profil , ainsi qu’avec l’option de recherche d’utilisateurs. Tu peux changer à nouveau pour récupérer un nom d’utilisateur précédemment utilisé, en suivant le processus ci-dessus, ce qui te coûtera également des Robux.

Quelles autres informations dois-je connaître sur les changements de nom ?

Les options suivantes ne sont pas disponibles :

  • Tu ne peux pas utiliser un nom appartenant à un autre compte
  • Tu ne peux pas échanger des noms d’utilisateur avec un autre compte que tu possèdes
  • Aucune remise ne s’applique pour de simples changements tels que le remplacement des lettres minuscules du nom par des majuscules

Remarques importantes :

  • La date de création de compte initiale sera reportée sur ton nouveau nom d’utilisateur
  • Les autres joueurs peuvent voir les noms d’utilisateur que tu as utilisés précédemment
