Comment changer votre Nom d'Affichage

Roblox permet désormais à tous les utilisateurs du monde entier de modifier le nom d'affichage de votre compte Roblox dans les paramètres du compte en tant que fonctionnalité gratuite. Cette fonction est différente de la Modification d'un nom d'utilisateur .

Comment changer le Nom d'Affichage?

  1. Connectez-vous à votre compte
    (Si vous ne pouvez pas vous connecter, essayez ces étapes pour réinitialiser votre mot de passe .)
  2. Accédez aux Paramètres du compte
    • Navigateur - recherchez l'icône en forme d'engrenage située dans le coin supérieur droit du site
    • Applications mobiles - recherchez l'icône des trois points pour Plus
  3. Sélectionnez l’onglet Informations du compte
  4. Sélectionnez le bouton Changer le Nom d'affichage (l’icône ressemble à un papier et un crayon)
    • Saisissez votre nouveau nom d'affichage
  5. Cliquez sur Sauvegarder
  6. Lorsque vous actualisez la page, votre nom d'affichage devrait être différent de votre nom d'utilisateur.

Que dois-je savoir d'autre sur les changements de Nom d'Affichage?

  • Votre nom d'affichage et votre nom d'utilisateur seront visibles sur Roblox et les expériences associées
  • Vous ne pouvez changer votre nom d'affichage qu'une fois tous les sept jours.
  • Les noms d'affichage doivent respecter les règles de la communauté Roblox et les conditions d'utilisation ou peuvent être réinitialisés à votre nom d'utilisateur.
  • D'autres joueurs peuvent également avoir le même nom d'affichage.
