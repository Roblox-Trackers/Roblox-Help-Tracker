Comment changer mon numéro de téléphone ?

Pour les joueurs ayant plus de 13 ans, un numéro de téléphone peut être ajouté à un compte. Un numéro de téléphone ne peut être utilisé que sur un seul compte.

Pour ajouter un numéro de téléphone

  1. Connectez-vous à votre compte
    ( Si vous n’arrivez pas à accéder au compte, veuillez cliquer ici )
  2. Sélectionnez vos Paramètres du compte
    • Navigateur - trouvez l'icône en forme de roue dentée, située dans le coin supérieur droit du site
    • Applications mobiles - localisez l'icône à trois points pour Plus d'options
  3. Sélectionnez l'onglet Informations sur le compte
  4. Sélectionnez le bouton Ajouter un numéro de téléphone
    1. Sélectionnez votre emplacement dans le menu déroulant
    2. Entrez le numéro de téléphone et le mot de passe du compte
    3. Sélectionnez Ajouter un numéro de téléphone

Pour mettre à jour ou supprimer un numéro de téléphone existant

  1. Connectez-vous à votre compte
    ( Si vous n’arrivez pas à accéder au compte, veuillez cliquer cliquer ici )
  2. Sélectionnez vos Paramètres du compte
    • Navigateur - trouvez l'icône en forme de roue dentée, située dans le coin supérieur droit du site
    • Applications mobiles - localisez l'icône à trois points pour Plus d'options
  3. Sélectionnez l'onglet Informations sur le compte
  4. Sélectionnez le bouton Mise à jour du téléphone (l'icône ressemble à un crayon avec morceau de papier)
    1. Sélectionnez votre emplacement dans le menu déroulant
    2. Entrez le numéro de téléphone et le mot de passe du compte
    3. Sélectionnez Modifier le téléphone ou Supprimer le numéro de téléphone

Commandes pour les messages

Voici quelques commandes utiles pour démarrer et arrêter les messages texte que vous recevez de Roblox:

  • ARRÊTER, ARRÊTER TOUT, SE DÉSABONNER, ANNULER, METTRE FIN, et QUITTER - Ces commandes à mot unique arrêteront les futurs messages. Veuillez noter que seuls les messages à mot unique déclenchent le blocage. Par exemple, si vous répondez «ARRÊTER», vous ne recevrez plus d'autres messages,  mais «ARRÊTER, S’IL VOUS PLAÎT » ne fonctionnera pas.
  • ARRÊTER, OUI et RÉACTIVER - Ces commandes vous permettront de vous réabonner à la réception de messages texte.
  • AIDE et INFO - Ces commandes fournissent des informations au sujet des options ci-dessus, que vous pouvez utiliser pour contrôler la diffusion des messages.
