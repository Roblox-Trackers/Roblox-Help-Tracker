Comment changer mon numéro de téléphone ?

Cette fonctionnalité est réservée aux joueurs de 13 ans ou plus. Un numéro de téléphone ne peut être utilisé que sur un seul compte.

Ajouter un numéro de téléphone

  1. Connectez-vous à votre compte
    (Si vous ne parvenez pas à vous connecter, essayez ces étapes pour récupérer votre mot de passe )
  2. Sélectionnez Paramètres du compte
    • Navigateur - trouvez l'icône représentant une roue dentée, située dans le coin supérieur droit du site
    • Applications mobiles - localisez l'icône à trois points pour Plus d'options
  3. Sélectionnez l'onglet Informations sur le compte
  4. Sélectionnez le bouton Ajouter un numéro de téléphone
    1. Sélectionnez votre emplacement dans le menu déroulant
    2. Entrez le numéro de téléphone et le mot de passe du compte
    3. Sélectionnez Ajouter un numéro de téléphone

Modifier ou supprimer un numéro de téléphone

  1. Connectez-vous à votre compte
    (Si vous n’arrivez pas à accéder au compte, veuillez cliquer ici .)
  2. Sélectionnez Paramètres du compte
    • Navigateur - trouvez l'icône représentant une roue dentée, située dans le coin supérieur droit du site
    • Applications mobiles - localisez l'icône à trois points pour Plus d'options
  3. Sélectionnez l'onglet Informations sur le compte
  4. Sélectionnez le bouton Mise à jour du téléphone (l'icône ressemble à un crayon avec un morceau de papier)
    1. Sélectionnez votre emplacement dans le menu déroulant
    2. Entrez le numéro de téléphone et le mot de passe du compte
    3. Sélectionnez Modifier le numéro de téléphone ou Supprimer le numéro de téléphone

Commandes pour les Messages texte

Voici quelques commandes utiles pour démarrer et arrêter les messages texte que vous recevez de Roblox:

  • ARRÊTER, ARRÊTER TOUT, SE DÉSABONNER, ANNULER, METTRE FIN, et QUITTER
    Ces commandes à mot unique arrêteront les futurs messages. Veuillez noter que seuls les messages à mot unique déclencheront le blocage. Par exemple, répondre « ARRÊTER » garantira que vous ne recevrez plus de messages, mais répondre « ARRÊTER, S’IL VOUS PLAÎT » ne le fera pas.
  • ARRÊTER, OUI, et RÉACTIVER
    Ces commandes vous permettront de vous réabonner à la réception de messages texte.
  • AIDE et INFO
    Ces commandes fourniront des informations que vous pouvez utiliser les commandes ci-dessus pour contrôler la livraison des messages.
