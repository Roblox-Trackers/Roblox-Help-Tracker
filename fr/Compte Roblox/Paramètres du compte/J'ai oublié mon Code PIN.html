J'ai oublié mon Code PIN

Une fois qu'un code PIN est ajouté à un compte, il vous sera nécessaire de modifier les paramètres du compte.

À ce stade, les utilisateurs et leurs parents/tuteurs ne pourront pas réinitialiser manuellement le code PIN, s'il est oublié. L'assistance de notre Service Client sera nécessaire lors de la réinitialisation des codes PIN oubliés. Pour contacter le Service Client, veuillez utiliser le formulaire de contact .

Remarque : Le Service Client peut demander des informations spécifiques pour s'assurer qu'il parle au parent/tuteur avant de réinitialiser un code PIN.
