J'ai oublié mon Code PIN

Une fois qu'un code PIN est ajouté à un compte, il vous sera nécessaire de modifier les paramètres du compte.

À ce stade, les utilisateurs et leurs parents / tuteurs ne pourront pas réinitialiser manuellement le code PIN, s'il a été oublié. L'assistance de notre Service Client sera nécessaire lors de la réinitialisation des codes confidentiels oubliés. Pour contacter le service clientèle, veuillez utiliser le formulaire de contact .

Note : Il se peut que le Service Client vous demande des informations spécifiques, pour s'assurer qu'ils parlent au parent / tuteur, avant qu’il réinitialise le code PIN.
