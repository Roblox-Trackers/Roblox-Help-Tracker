Comment Changer la Langue ou le Pays de votre Compte

Pour la meuilleure experience, assurez-vous toujours que les paramètres de langue et de pays de votre compte sont corrects.

Comment changer la langue de votre compte

  1. Connectez-vous sur votre compte
    ( Si vous ne pouvez pas vous connecter, veuillez essayer ces étapes pour récupérer votre mot de passe .)
  2. Allez aux Paramètres du compte
    • Navigateur - trouvez l'icône en forme d'engrenage, située dans le coin supérieur droit du site
    • Applications mobiles - recherchez l'icône des trois points pour Plus d'options
  3. Sélectionnez l'onglet Infos sur le compte
  4. Sous la section Personnel , il s'affichera la langue actuelle de votre compte
  5. Sélectionnez la flèche vers le bas à droite et trouvez votre langue.
  6. La page se rechargera et affichera la nouvelle langue choisie.
  7. Si la page ne se recharge pas, cliquez sur le bouton Enregistrer dans le coin inférieur droit de la section Personnel.

Si votre langue n’est pas affichée dans les options, vérifiez de nouveau si celle-ci est supportée. Veuillez vérifier à une date ultérieure pour voir s'il a été ajouté.
