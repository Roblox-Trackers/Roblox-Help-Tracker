Comment Ajouter ou Supprimer un Code PIN sur mon Compte ?

Vous avez la possibilité d'ajouter un code PIN à quatre chiffres sur votre compte pour verrouiller vos paramètres. Une fois qu'un code PIN est ajouté, il vous sera demandé chaque fois que vous souhaitez modifier les paramètres du compte. Cette fonctionnalité protège votre mot de passe, votre adresse e-mail et vos paramètres de confidentialité.

Comment ajouter un Code PIN ?

  1. Connectez-vous sur votre compte
    (Si vous ne pouvez pas vous connecter, veuillez suivre ces étapes pour récupérer votre mot de passe .)
  2. Allez dans Paramètres du compte
    • Navigateur - trouvez l'icône d'engrenage située dans le coin supérieur droit du site
    • Applications mobiles - recherchez l'icône des trois points pour Plus
  3. Sélectionnez l'onglet Contrôle Parentale
  4. Basculer le bouton pour activer le code PIN
  5. Créez et confirmez votre nouveau Code PIN

Ne communiquez jamais votre code PIN à personne - c'est encore plus important que votre mot de passe.

Comment supprimer le Code PIN ?

  1. Connectez-vous sur votre compte
    (Si vous ne pouvez pas vous connecter, veuillez suivre ces étapes pour récupérer votre mot de passe .)
  2. Allez dans Paramètres du compte
    • Navigateur - trouvez l'icône d'engrenage située dans le coin supérieur droit du site
    • Applications mobiles - recherchez l'icône des trois points pour Plus
  3. Sélectionnez l'onglet Contrôle Parentale
  4. Sélectionnez désactiver le PIN
  5. Entrez le PIN actuel et sélectionnez Déverrouiller

Comment changer un Code PIN ?

Pour changer un PIN, suivez d'abord les étapes de la section Comment supprimer le Code PIN? . Ensuite, suivez les étapes de la section Comment ajouter un PIN? pour créer un nouveau PIN.
