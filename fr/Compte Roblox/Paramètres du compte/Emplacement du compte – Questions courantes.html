Emplacement du compte – Questions courantes

Comment l'emplacement de mon compte est-il déterminé ?

L'emplacement de votre compte est détecté automatiquement en fonction de divers signaux tels que votre adresse IP. Si vous utilisez une connexion VPN ou voyagez, la précision de votre localisation peut être affectée.

À quoi sert l'emplacement de mon compte ?

L'emplacement de votre compte est utilisé dans plusieurs domaines, notamment la personnalisation de votre expérience sur Roblox. Cependant, la principale raison pour laquelle les utilisateurs peuvent vouloir modifier l'emplacement est de déterminer quelle devise peut être utilisée sur le compte pour acheter des Robux et échanger des cartes-cadeaux. Par exemple, si vous souhaitez utiliser une carte-cadeau, la devise de cette carte-cadeau doit correspondre à la devise de l'emplacement de votre compte.

Comment puis-je modifier l'emplacement de mon compte ?

Si l'emplacement de votre compte est mal défini, vous pouvez le mettre à jour une fois par vous-même en accédant à Paramètres > Informations de compte. Dans la section « Personnel », choisissez l'emplacement correct dans le menu déroulant. Si vous ne savez pas pourquoi vous devez modifier votre emplacement, ne le modifiez pas. Une fois que vous aurez modifié l'emplacement, vous ne pourrez plus le modifier vous-même.

Pourquoi ne puis-je pas modifier l'emplacement de mon compte ?

Si vous changez fréquemment d'emplacement ou si vous résidez dans un endroit soumis à certaines restrictions, vous ne pourrez peut-être pas modifier votre emplacement par vous-même. Dans ce cas, veuillez contacter le Service Client si vous avez une raison valable de mettre à jour votre emplacement.
