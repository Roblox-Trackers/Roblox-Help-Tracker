Comment bloquer un autre utilisateur

Chez Roblox, nous encourageons tous nos utilisateurs à nous aider maintenir un environnement à la fois sûr et amusant pour tout le monde.  Cependant, nous savons que parfois les joueurs ne s'entendent pas. Vous voudrez peut-être empêcher certains utilisateurs de vous parler en les bloquant.

Comment bloquer un autre utilisateur

Pour bloquer un autre utilisateur sur le site Web ou l'application mobile :

  1. Visitez leur page de profil
  2. Cliquez sur les trois points dans le coin supérieur droit de la boîte contenant leur nom d'utilisateur et les informations sur leurs amis / abonnés
  3. Un menu apparaîtra, où vous pouvez sélectionner l'option Bloquer un Utilisateur

Comment bloquer un autre utilisateur dans une expérience

Pour bloquer un autre utilisateur depuis une expérience à l'aide du classement :

  1. Trouvez l'utilisateur dans le classement / liste des joueurs, au coin supérieur droit de l'écran de l'expérience. Ça ressemblera à ceci :

    Si cette liste n'est pas visible, elle est probablement fermée. Pour la rouvrir, cliquez sur votre nom d'utilisateur dans le coin supérieur droit. Remarque - il est possible que le classement n'apparait pas si vous utilisez un appareil de petite taille, tel qu'un téléphone, et dans ce cas-ci vous devrez utiliser la méthode de la page de profil décrite ci-dessus.
  2. Une fois que vous aurez trouvé le nom de l'utilisateur que vous souhaitez bloquer à l'intérieur du classement, cliquez dessus et un menu s'ouvrira.
  3. Sélectionnez l'option Bloquer un Utilisateur . Vous pouvez également choisir de les débloquer, ou de Signaler un Abus directement à partir de ce menu. Une fois que vous avez bloqué l'utilisateur, l'icône à gauche de son nom se transforme en un cercle barré pour indiquer qu'il a été bloqué.

Pour bloquer un autre utilisateur à l'intérieur d'une expérience sans classement à l'aide du menu Esc:

  1. Trouvez l'utilisateur que vous souhaitez bloquer dans le menu Esc sous l'onglet "Personnes".
  2. Une fois que vous avez trouvé le nom de l'utilisateur que vous souhaitez bloquer, veuillez cliquer sur le bouton Bloquer qui ressemble à ceci :
  3. Cliquez sur le bouton " Bloquer " dans la boîte de dialogue contextuelle pour confirmer.

Paramètres de confidentialité et contact

Roblox vous offre aussi une variété de Paramètres de Confidentialité pour limiter certaines formes de communication et d'actions à des groupes particuliers de personnes, tels que vos amis ou vos abonnés. Pour plus d'informations, vous pouvez consulter  cet l'article d'aide les Paramètres de Confidentialité .
