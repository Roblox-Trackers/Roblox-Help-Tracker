Comment bloquer un autre utilisateur

Chez Roblox, nous encourageons tous nos joueurs à nous aider à maintenir un environnement à la fois sûr et amusant pour tous. Cependant, nous savons que parfois les joueurs ne s'entendent pas. Vous voudrez peut-être empêcher certains utilisateurs de communiquer avec vous en les bloquant.

Comment bloquer un autre utilisateur

Bloquer un autre utilisateur via un navigateur ou une application mobile

  1. Visitez leur page de profil
  2. Sélectionnez les trois points dans le coin supérieur droit de la section des détails du profil pour ouvrir le menu déroulant
  3. Cliquez sur l'option Bloquer l'utilisateur pour bloquer un utlisateur

Blocage des utilisateurs dans une expérience

Bloquer un autre utilisateur en utilisant le classement

  1. Trouvez l'utilisateur dans le classement / liste des joueurs, au coin supérieur droit de l'écran de l'expérience. Cela se présentera comme suit :


    Si cette liste n'est pas visible, elle vient probablement d'être fermée. Pour la rouvrir, cliquez sur les trois points et sélectionnez Classement ou vous pouvez simplement cliquer sur la touche Tab de votre clavier

    Remarque : Le classement peut ne pas s'afficher si vous utilisez un appareil à petit écran tel qu'un téléphone, auquel cas vous devez utiliser la méthode de la page de profil décrite ci-dessus.

  2. Cliquez sur l'utilisateur que vous voulez bloquer

  3. Cliquez sur l'option Bloquer dans le menu déroulant. Vous pouvez également choisir de les débloquer ou de signaler un abus directement à partir de ce menu.



    Une fois que vous avez bloqué l'utilisateur, l'icône située à gauche de son nom se transforme en un cercle barré d'une ligne pour indiquer qu'il a été bloqué

Bloquer un autre utilisateur à l'aide du menu Esc

  1. Appuyez sur la touche Esc et recherchez l'utilisateur que vous souhaitez bloquer dans le menu Esc, sous l'onglet Personne .
  2. Une fois que vous avez trouvé le nom de l'utilisateur que vous souhaitez bloquer, veuillez cliquer sur le bouton Bloquer qui ressemble à ceci :
  3. Cliquez sur le bouton Bloquer dans la boîte de dialogue pour confirmer.

Paramètres de confidentialité et contact

Roblox vous offre aussi une variété de Paramètres de Confidentialité pour limiter certaines formes de communication et d'actions avec des groupes particuliers de personnes, tels que vos amis ou vos abonnés. Pour en savoir plus, consultez l'article d'aide Paramètres de confidentialité.
