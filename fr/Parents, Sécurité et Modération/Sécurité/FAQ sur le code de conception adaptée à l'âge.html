FAQ sur le code de conception adaptée à l'âge

  1. Comment le système de filtrage des textes de Roblox assure-t-il la sécurité des enfants ?

    Roblox fait tout son possible pour favoriser un environnement où les personnes de tout âge peuvent créer, jouer, apprendre et imaginer en toute sécurité. Lors de la conception de notre plateforme, nous avons gardé à l'esprit la confidentialité et la sécurité des enfants, notamment par la mise en place de filtres textuels avancés qui bloquent les propos inappropriés ou tout autre contenu dangereux. Plus précisément, ce filtre textuel bloquera automatiquement les gros mots, les informations permettant d'identifier une personne (comme une adresse ou un numéro de téléphone), les mots associés à l'intimidation et au harcèlement, ainsi que tout texte non reconnu.

    Par exemple, si un enfant essaie de partager son adresse, notre filtre de chat empêchera l'envoi du message. L'enfant verra son message bloqué s'afficher sous la forme d'une série de dièses (####) et recevra un avertissement indiquant que son message n'a pas été envoyé. Le ou les autres utilisateurs ne sauront jamais que l'enfant a tenté d'envoyer un message. Nous limitons en outre l'accès aux liens des médias sociaux pour les jeunes enfants, ainsi que les notifications mobiles (pop-ups).

  2. Comment puis-je contrôler qui peut chatter avec mon enfant ?

    Si votre enfant a moins de 13 ans, nous faisons en sorte qu'il soit extrêmement difficile pour des inconnus sur Roblox de le contacter. Vous pouvez restreindre davantage la capacité de votre enfant à chatter avec d'autres personnes sur Roblox en procédant comme suit :

    1. Cliquez sur l'icône de l'engrenage pour accéder à Paramètres
    2. Sélectionnez l'onglet Confidentialité
    3. Sélectionnez « Off » sous Paramètres du contact

    Voici ce que signifient les différentes options pour contrôler le contact avec votre enfant sur Roblox :

    • L'option "Qui peut m'envoyer des messages" permet de contrôler qui peut envoyer un message dans la boîte de réception "Messages" de votre enfant.
    • L'option "Qui peut chatter avec moi dans l'application" permet de contrôler qui peut envoyer un message de chat à votre enfant lorsqu'il utilise l'application ou le site Web Roblox. Ce paramètre détermine également si un utilisateur peut publier sur les murs du groupe.
    • L'option « Qui peut discuter avec moi » permet de contrôler quels utilisateurs peuvent discuter avec votre enfant en temps réel lorsqu'il se trouve dans une expérience.

    Si vous avez configuré un code parental sur le compte Roblox de votre enfant, ces paramètres de chat ne peuvent être modifiés sans ce code. Remarque : bien qu'il soit possible de désactiver le chat dans le jeu, cela peut rendre Roblox moins agréable, puisque beaucoup de nos jeux sont construits autour de la possibilité de chatter avec d'autres Robloxiens.

  3. Qu'est-ce qu'un code parental ?

    Si votre enfant possède un compte Roblox, vous pouvez configurer un code parental qui doit être saisi avant que les paramètres de confidentialité de ce compte puissent être modifiés. Suivez les instructions ci-dessous pour créer un code parental :

    1. Cliquez sur l'icône de l'engrenage pour accéder à Paramètres
    2. Sélectionnez l'onglet Sécurité
    3. Cliquez sur l'onglet à côté de « Le code PIN du compte est actuellement désactivé »
    4. Si vous ne l'avez pas encore fait, entrez et vérifiez votre adresse électronique.
    5. Une fois que vous avez suivi les instructions pour vérifier votre adresse électronique, revenez sur cette page et cliquez sur l'onglet à côté de « Le code PIN du compte est actuellement désactivé. » Vous serez ensuite invité à créer un code PIN à 4 chiffres.
    6. Votre code parental est complet une fois que vous avez saisi votre code PIN et cliqué sur « Ajouter ». Votre code PIN sera désormais requis avant de pouvoir modifier les paramètres du compte.

  4. Où puis-je trouver la politique de confidentialité de Roblox ?

    Vous pouvez trouver notre politique de confidentialité ici . Il y a également le lien au bas de toutes les pages de roblox.com.

  5. Quel type d'informations Roblox recueille-t-il ?

    Roblox peut recueillir des informations vous concernant dans les cas suivants :

    • Si vous postulez à un emploi chez Roblox via notre site web d'entreprise
    • Lorsque vous vous inscrivez pour un compte Roblox
    • Lorsque vous choisissez de vous engager sur la plateforme Roblox (par exemple, effectuer un achat sur Roblox) ou d'envoyer des messages de chat

    Pour plus de détails sur le type d'informations que Roblox peut recueillir dans les cas ci-dessus, consultez les sections 3, 4 et 5 de notre site Web Politique de confidentialité .

  6. Quels sont les droits en matière de protection de la vie privée dont je bénéficie en fonction de mon pays de résidence ?

    En général, Roblox honorera les droits de confidentialité offerts dans votre juridiction.

    Si vous êtes situé dans l'Espace économique européen (EEE), ou si vous vivez au Brésil, en Islande, en Norvège, en Suisse ou dans certains États des États-Unis, nous vous offrons les protections de confidentialité suivantes :

    • Droit d'être informé
    • Droit d'accès
    • Droit de rectification
    • Droit à l'effacement/à l'oubli
    • Droit de restreindre le traitement
    • Droit à la portabilité des données
    • Droit d'opposition
    • Droits relatifs à la prise de décision automatisée et au profilage

    Si vous vivez dans l'un des pays mentionnés ci-dessus, consultez la section 15 de notre Politique de confidentialité pour de plus amples informations.

  7. Comment puis-je exercer mes droits en matière de protection de la vie privée si je me trouve dans une juridiction qui offre des droits en la matière ?

    Pour exercer vos droits en matière de protection de la vie privée, suivez les étapes ci-dessous :

    1. Visiter la page Assistance Roblox
    2. Cliquer sur le menu déroulant sous Type de catégorie d'aide
    3. Sélectionner Demandes de confidentialité des données
    4. Sélectionner la sous-catégorie qui vous intéresse (par exemple, Droit à l'oubli)
    5. Cliquer sur Soumettre

    Se rendre ici pour plus d'informations sur la politique de confidentialité de Roblox.
