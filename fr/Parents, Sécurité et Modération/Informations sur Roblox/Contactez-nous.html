Contactez-nous

L'équipe de support client de Roblox répond aux questions liées aux problèmes de facturation et de compte. Assurez-vous de consulter nos articles d'aide Roblox avant de nous contacter.

  • Si vous avez besoin d'aide supplémentaire, consultez les informations ci-dessous et remplissez le formulaire d'assistance .
  • Si vous avez des questions sur les fonctionnalités du site, les performances de l'expérience, la construction ou les scripts, visitez notre documentation pour les créateurs et DevForum pour en savoir plus.
  • Si vous rencontrez un problème pour rejoindre une expérience, utiliser la plate-forme Roblox ou jouer à une expérience, assurez-vous de consulter la page Statut Roblox .

De quelles informations avons-nous besoin lorsque vous nous contactez?

Lorsque vous nous contactez, assurez-vous de fournir les informations suivantes ci-dessous.

  • Nom d'utilisateur Roblox : nom d'utilisateur pour lequel vous avez besoin d'aide
    Remarque : Si vous avez besoin d'aide pour plusieurs noms d'utilisateur, ajoutez-les tous dans la description du champ du problème tout en décrivant votre problème.
  • Sur quel appareil rencontrez-vous le problème ? : sélectionnez l'appareil pour lequel vous avez besoin d'aide, où vous avez effectué l'achat, etc.
  • Type de catégorie d'aide : sélectionnez le sujet pour lequel vous avez besoin d'aide

Remarque : Si nous vous demandons une capture d'écran, vous pouvez prendre des captures d'écran en suivant les étapes dans l'article Comment faire une capture d'écran?

De quelles informations supplémentaires avons-nous besoin pour des questions spécifiques ?

Lorsque vous remplissez le champ Description du problème , assurez-vous d'être aussi descriptif que possible. Cela nous aide à vous aider plus rapidement. Découvrez les informations dont nous avons besoin ci-dessous et dans les articles répertoriés déstinés aux questions spécifiques.
