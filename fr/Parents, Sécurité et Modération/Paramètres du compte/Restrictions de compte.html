Restrictions de compte

Roblox offre aux utilisateurs de tous âges la possibilité de socialiser et de vivre des expériences avec d'autres membres de la communauté. Il existe de multiples opportunités de discuter avec les autres et diverses expériences qui offrent différentes expériences, de l'aventure au combat en passant par le jeu de rôle, et plus encore. Nous autorisons les expériences avec certaines formes de violence de dessin animé, tant qu'elles ne sont pas trop graphiques ou réalistes. Cependant, Roblox offre une fonctionnalité qui permet aux parents de limiter la possibilité de discuter et les types d'expériences auxquels leurs enfants peuvent accéder. Les parents peuvent en savoir plus sur la façon dont nous assurons la sécurité des enfants et nos contrôles parentaux grâce à notre Guide des Parents .

Que se passe-t-il lorsque les restrictions de compte sont activées?

L'activation des restrictions de compte verrouille les paramètres de contact d'un compte afin qu'aucun autre utilisateur ne puisse envoyer de messages ni discuter dans l'application ou dans l'expérience. Aucun de ces paramètres de contact ne peut être ajusté individuellement lorsque les restrictions de compte sont activées. L'activation des restrictions de compte limitera également l'accès seulement aux expériences recommandées pour Tous les âges via nos Contrôles des expériences permises

Comment activer les restrictions de compte?

Sur un ordinateur

  • Connectez-vous au compte (si vous ne pouvez pas accéder au compte, veuillez cliquer ici )
  • Cliquez sur l'icône représentant un engrenage situé dans le coin supérieur droit du site
  • Sélectionnez l'option Paramètres indiquée par l'icône représentant une roue dentée
  • Sélectionnez l'option Confidentialité
  • Dans la section Restrictions de compte , appuyez sur le bouton bascule pour activer les restrictions de compte.
  • La bascule deviendra verte et le message suivant s'affichera "Restrictions de compte est actuellement activé"
  • Si vous souhaitez désactiver les restrictions de compte, appuyez simplement à nouveau sur le bouton bascule

Remarque: Si le compte a activé un code PIN parental, cela sera nécessaire pour ajuster les paramètres des restrictions de compte. Pour plus d'informations, veuillez lire cet article d'aide sur la FAQ des code PIN de compte.

Sur un appareil mobile

  • Connectez-vous au compte dans l'application (si vous ne pouvez pas accéder au compte, cliquez ici )
  • Cliquez sur le bouton Plus indiqué par trois points situés dans le coin inférieur droit de la page.
  • Dans le menu contextuel, sélectionnez l'option Paramètres
  • Sélectionnez Confidentialité dans le menu situé à gauche de l'écran.
  • Dans la section Restrictions de compte , appuyez sur le bouton bascule pour activer les restrictions de compte.
  • La bascule deviendra verte et le message suivant s'affichera "Restrictions de compte est actuellement activé"
  • Si vous souhaitez désactiver les restrictions de compte, appuyez simplement à nouveau sur le bouton bascule

Remarque: Si le compte a activé un code PIN parental, cela sera nécessaire pour ajuster les paramètres des restrictions de compte. Pour plus d'informations, veuillez lire cet article d'aide sur la FAQ des code PIN de compte.
