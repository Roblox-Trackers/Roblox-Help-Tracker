Questions fréquentes sur la restriction des dépenses mensuelles

Q : Comment Roblox sait-il que l’adresse e-mail du parent/tuteur fournie appartient bien au parent ?
R : Nous faisons confiance à nos utilisateurs pour être honnêtes quant à la saisie d’une adresse électronique parentale/tuteur. Les parents/tuteurs peuvent réinitialiser l’adresse électronique du parent/tuteur sur le compte en accédant au compte de leur enfant.
Q : Cette fonctionnalité est-elle disponible uniquement pour les utilisateurs de moins de 13 ans ?
R : Oui, cette fonctionnalité n’est actuellement disponible que pour les utilisateurs de moins de 13 ans.
Q : Pourquoi la limite de dépense apparait-elle en USD même pour les parents/tuteurs non américains
A: En raison des fluctuations du change de devises, les limites sont en dollars américains (USD). Nous travaillons à prendre en charge davantage de devises à l’avenir pour le confort des utilisateurs des pays non américains.
Q : Les parents/tuteurs sont-ils avisés lorsque la limite de dépenses mensuelles est atteinte ?
R : Non, les parents/tuteurs ne recevront pas de notification par courriel à moins qu’ils ne choisissent de recevoir des notifications par courriel de Roblox pour toutes les dépenses (assurez-vous d’ajouter votre courriel au compte utilisateur sous « Informations sur le compte » dans les paramètres).
Q : Les parents/tuteurs peuvent-ils fixer une limite de dépenses quotidiennes ou hebdomadaires ?
R : Non, ce paramètre limite le montant des dépenses autorisées sur le compte de l’utilisateur de moins de 13 ans sur la base du mois civil uniquement.
Q : Que se passera-t-il si le code PIN parent est supprimé ?
R : Si le code PIN parent est supprimé après la modification des paramètres du compte, les paramètres doivent revenir à la valeur par défaut.
Q : Si le compte n’a pas de code PIN parent, mais qu’il dispose d’une adresse e-mail parentale/tuteur vérifiée, les utilisateurs peuvent-ils apporter des modifications aux paramètres du compte ?
R : S’il n’y a pas de code PIN parent mais qu’il existe une adresse électronique parentale/tuteur vérifiée, toute tentative de modification des paramètres entraînera une demande d’ajout d’un code PIN parent au compte. Pour plus d’informations, consultez la section Restrictions relatives aux dépenses mensuelles.
