Contrôles des expériences autorisés

Lignes directrices sur l'expérience de Roblox, qui fournissent des recommandations d'âge pour le contenu approprié d’une expérience. Sur la base de ces recommandations d'âge, les utilisateurs et les parents peuvent prendre des décisions éclairées sur le contenu avec lequel ils interagissent. Les parents peuvent utiliser les expériences autorisées pour gérer l'accès au contenu auquel leur enfant a accès. Les parents peuvent en apprendre davantage sur nos contrôles parentaux dans notre Guide des Parents .

Que se passe-t-il lorsque les expériences autorisées sont définies ?

Les parents sont en mesure de choisir des expériences adaptées à l'âge en permettant des expériences autorisées. Cela permettra à votre enfant de ne rejoindre que les expériences qui correspondent ou sont en dessous de l'âge recommandé. Les expériences au-delà de la restriction d'âge activée peuvent être affichées dans les résultats de recherche ou annoncées dans les annonces. Cependant, les expériences ne seront pas visibles. Les utilisateurs recevront un message indiquant que l'expérience n’est pas disponible en raison des paramètres de restrictions du compte.

Comment une recommandation relative à l'âge est-elle dissociée ?

Les recommandations de Roblox sur l'âge sont fondées sur la recherche sur le développement de l’enfant et sont fondées sur les normes de l’industrie. Pour déterminer à quels publics une expérience convient généralement, nous avons examiné les normes mondiales de l’industrie et consulté des experts en développement de l’enfant.

Comment configurer les expériences autorisées ?

Sur un ordinateur

  • Connectez-vous au compte (si vous ne pouvez pas accéder au compte, veuillez cliquer ici )
  • Cliquez sur l'icône représentant un engrenage située dans le coin supérieur droit du site.
  • Sélectionnez l'option Paramètres indiquée par l'icône d'engrenage
  • Vous devez avoir activé un PIN parental pour configurer cette fonction
    • Voici les étapes de configuration du PIN parental
  • Sélectionnez l'option de Contrôles parentaux
  • Dans la section Expériences autorisées , sélectionnez le niveau d'expérience le plus élevé auquel vous souhaitez que le compte puisse participer
  • Une fois sélectionné, un message vert s’affichera “Enregistré avec succès !"

Remarque : Si le compte a activé les restrictions du compte, le compte sera limité au contenu pour tous les ages uniquement et désactivera les expériences autorisées. Le paramètre Restrictions de compte doit être désactivé avant que les expériences autorisées puissent être ajustées. Pour plus d'informations, veuillez lire l'article d'aide sur les Restrictions du compte .

Sur un appareil mobile

  • Connectez-vous au compte dans l’application (si vous ne pouvez pas accéder au compte, veuillez cliquer ici )
  • Cliquez sur le bouton Plus indiqué par trois points dans le coin inférieur droit de la page
  • Dans le menu contextuel, sélectionnez l’option Paramètres
  • Vous devez avoir activé un PIN parental pour configurer cette fonction
    • Voici les étapes pour configurer le PIN parental
  • Sélectionnez Contrôles Parentaux dans le menu à gauche de l'écran
  • Dans la section Expériences autorisées , sélectionnez le plus haut niveau d'expérience que vous souhaitez que le compte puisse rejoindre
  • Une fois sélectionné, un message vert s’affichera "Enregistré avec succès !"

Remarque : Si le compte a activé les Restrictions de compte, le compte sera limité au contenu pour Tous les âges et désactivera les Expériences Autorisées. Le paramètre Restrictions de compte doit être désactivé avant que l’option Expériences autorisées puisse être ajustée. Pour plus d'informations, veuillez lire l'article d'aide sur les Restrictions du compte .
