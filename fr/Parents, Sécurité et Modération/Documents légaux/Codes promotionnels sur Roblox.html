Codes promotionnels sur Roblox

Comment puis-je donner des codes promotionnels gratuits pour des articles virtuels ?

Les développeurs peuvent mettre en place des codes promotionnels uniquement pour les articles de leur propre expérience, à condition que :

  1. L'item n'est utilisable que dans l'expérience où le code est échangé,
  2. L'item de l'expérience n'a aucune valeur en Robux ou en monnaie du monde réel
  3. Le code lui-même est publié de manière à être accessible au public,
  4. Aucun achat n'est nécessaire pour accéder au code ou l'échanger, et
  5. Tous les éléments ci-dessus sont clairement divulgués aux utilisateurs dans tous les documents promotionnels

Les codes utilisés dans les offres groupées de vente au détail doivent indiquer clairement aux clients, avant l'achat, l'endroit où ils peuvent accéder au code gratuit et l'échanger (par exemple, « Aucun achat nécessaire pour l'article d'expérience virtuelle. Code promotionnel disponible sur [URL] »)

Les développeurs doivent également suivre les directives de Roblox Lignes directrices de la marque dans tout matériel promotionnel.

Puis-je donner un code pour un article disponible sur le marché de Roblox ?

Vous pouvez mettre en place des codes promotionnels uniquement pour les articles accessibles et utilisables exclusivement dans vos expériences.

Je mets un code promo sur l'emballage de mon article de détail. Est-ce que cela peut être considéré comme "accessible au public" pour les utilisateurs ?

Les codes accessibles au public doivent être disponibles sur Internet (par exemple, sur votre site Web public, dans l'un de vos messages publics sur les médias sociaux, dans la description de votre page de détails sur Roblox). Tout emballage de vente au détail doit clairement indiquer aux clients, avant l'achat, qu'aucun achat n'est nécessaire pour échanger le code, et indiquer où le code peut être trouvé. (par exemple, « Aucun achat nécessaire pour l'article d'expérience virtuelle. Code promotionnel disponible sur [URL] »)

J'ai vu d'autres développeurs créer des codes promotionnels pour des items du marketplace. Pourquoi cela est-il autorisé ?

Roblox a récemment lancé un programme pilote limité pour que les items de certains développeurs sur le Marketplace soient associés à un produit du monde réel. Cette expérience nous permettra d'en savoir plus sur le potentiel d'une telle capacité dans notre métavers et sur la possibilité de l'étendre à l'ensemble de la communauté. À ce stade, nous n'avons pas de calendrier pour cette initiative plus large, alors soyez patients.

J'ai trouvé des développeurs qui vendent des codes hors plateforme et qui ne font pas partie de votre programme de partenariat. Qu'allez-vous faire à leur sujet ?

Nous prenons très au sérieux les violations de nos politiques. Les contrevenants peuvent être soumis à des suspensions des services DevEx, de l'accès au compte ou de la disponibilité de l'expérience.

Que voulez-vous dire par un item "gratuit" ?

L'item de votre expérience ne peut pas être un item que vous vendez pour des Robux ou une monnaie proxy dans le jeu qui nécessite des Robux. Si un item est perçu comme précieux parce qu'il nécessite un haut niveau de compétence ou un temps d'engagement pour le gagner, vous pouvez utiliser un code promo pour cet item à condition qu'il soit clairement publié pour que tout utilisateur puisse le trouver et qu'il soit librement utilisable dans votre expérience.

Que dois-je faire si ma promotion est saisonnière ?

Vous devez faire en sorte que la fenêtre d'échange soit claire pour les utilisateurs partout où votre code est promu. Si votre code promotionnel saisonnier figure sur un emballage de vente au détail, vous devez soit prendre en charge le code de remboursement et l'élément d'expérience associé pendant tout le cycle de vie de vente au détail de votre produit, soit indiquer très clairement sur l'emballage la fin de la promotion, même si l'emballage est toujours disponible dans les rayons de vente au détail.
