Liens vers les médias sociaux dans les descriptions des expériences

Les développeurs peuvent souhaiter lier leurs autres profils de médias sociaux directement à leurs expériences. Roblox offre désormais un support officiel pour cela via la fonction Social Media Link ! Les développeurs pourront inclure jusqu'à trois liens de médias sociaux dans leurs descriptions d'expérience, de manière similaire à la méthode utilisée pour leurs profils d'utilisateurs. Chaque type de média social ne peut être ajouté qu'une seule fois.

Les liens vers médias sociaux qui peuvent être ajoutés sont :

  • Discord
  • Youtube
  • Twitter
  • Twitch
  • Facebook
  • Roblox Group

Comment ajouter des liens vers médias sociaux à votre expérience :

  1. Connectez-vous à votre compte
  2. Allez sur la page Créer
  3. Cliquez sur la place Expérience
  4. Cliquez sur le menu M édias sociaux dans la liste
  5. Entrez l'URL, sélectionnez le type de support et saisissez le titre
  6. Cliquez sur Enregistrer
  7. Une fois que vous enregistrez les liens, voici comment les liens vers médias sociaux apparaîtront sur la page de l'expérience.

Remarques importantes :

  • Les liens vers les médias sociaux seront visibles uniquement pour les comptes de plus de 13 ans. Ils ne seront pas visibles pour les comptes qui ont moins de 13 ans.
  • Bien que les liens Discord soient permis dans la fonctionnalité de liens vers les réseaux sociaux, les utilisateurs ne sont pas autorisés à discuter ou à partager des liens Discord ailleurs et peuvent être modérés si ceux-ci sont mentionnés dehors.
