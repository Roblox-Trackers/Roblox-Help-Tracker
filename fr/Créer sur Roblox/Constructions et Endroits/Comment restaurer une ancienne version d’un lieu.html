Comment restaurer une ancienne version d’un lieu

Roblox conserve des copies de tous les lieux téléchargés. Cela veut dire que, chaque fois que vous apportez des changements à un lieu et les publiez sur Roblox, une copie sans tous ces changements est sauvegardée comme ancienne version. Vous pouvez trouver et restaurer une de ces anciennes versions à tout moment en faisant ce qui suit.

Restaurer une ancienne version :

  1. Cliquer sur Créer , en haut de l'écran
  2. A partir de là, sélectionnez le lieu que vous voudriez configurer
  3. Cliquez sur l’icône d’engrenage, situé à droite du nom du lieu et cliquez sur Configurer le lieu de départ dans le menu qui s'affiche.
  4. Cliquez sur Historique de la version . Vous y verrez une liste avec toutes les anciennes versions, classées par date
  5. Choisissez la version que vous voulez utiliser actuellement
  6. Cliquez sur Restaurer cette version pour la rendre la version actuelle
