Questions Fréquentes sur les paiements Premium

Pour des utilisateurs :

Q : Comment puis-je savoir quels avantages j'obtiendrai dans une expérience spécifique avec mon abonnement Premium ?
R : La divulgation d'avantages spécifiques dans l'expérience est à la discrétion des développeurs de l'expérience. Les avantages peuvent être trouvés dans la description de l'expérience, dans l'expérience elle-même ou à tout autre endroit où les développeurs décrivent leur expérience.

Q : Comment mon abonnement Premium aide-t-il mes expériences préférées ?
R : Si vous êtes un abonné Premium, une partie de vos frais d'abonnement ira directement à vos expériences les plus jouées. Plus vous jouez à vos expériences préférées sur Roblox, plus les développeurs de ces expériences gagneront grâce aux paiements Premium.

Pour des développeurs :

Q : Comment le score de temps de jeu Premium est-il calculé ?
R : Le score de temps de jeu est déterminé par la proportion de temps qu'un utilisateur Premium passe dans votre expérience par rapport à d'autres expériences. Le score de temps de jeu est calculé quotidiennement.

Q : Comment les gains sont-ils distribués pour les expériences de groupe ?
R : La distribution de groupe pour les paiements Premium fonctionne comme n'importe quel autre gain provenant des passes de jeu ou des produits de développement. Par conséquent, si la distribution automatique est configurée dans votre groupe, les revenus provenant des paiements Premium seront honorés.

Q : Pourquoi y a-t-il un délai de 28 jours pour les paiements ?
R : Vous recevrez des paiements Premium quotidiennement, 28 jours après chaque jour mesuré. Par exemple, vous recevrez votre paiement pour l'engagement du 19 février le 18 mars (28 jours plus tard) et pour l'engagement du 20 février le 19 mars (également 28 jours plus tard). Les paiements sont en retard d'un mois sur l'engagement, car nous examinons l'engagement d'un joueur pendant tout le mois de son abonnement. Cela nous permet de vous donner une proportion de leur frais d'abonnement pour ce mois. Ce décalage nous permet également de nous assurer que votre paiement n'est pas affecté par des remboursements ou d'autres complications de paiement.

Q : Comment mon paiement Premium Robux gagné est-il lié à mon Score de temps de jeu Premium ?
R : Votre score de temps jeu Premium vous donne une indication précoce et directionnelle de la façon dont votre paiement Premium changera le mois prochain. Cependant, la relation n'est pas exacte car votre score de temps de jeu Premium est calculé sur une base quotidienne, mais votre paiement est calculé sur l'ensemble du mois écoulé.

Q : Pourquoi les expériences payantes ne sont-elles pas éligibles pour les Paiements Premium ?
R : Nous souhaitons permettre une expérience fluide pour les abonnés Premium, afin que les expériences payantes pour l'accès ne soient actuellement pas incluses dans les Versements Premium.

Q : Combien puis-je m'attendre à recevoir ?
R : Vous pouvez utiliser le tableau des paiements Premium pour anticiper le montant auquel vous pouvez vous attendre chaque mois.

Q : Combien de temps dois-je attendre pour voir l'impact que les versements Premium auront sur moi ?
R : Nous vous recommandons d'être patient lorsque vous testez les versements Premium et de l'expérimenter - testez de nouvelles fonctionnalités pour augmenter l'engagement, ajustez-les et utilisez des tableaux de bord pour mesurer l'impact. Dans quelques mois vous pourrez voir à quel point les paiements Premium vous sont bénéfiques.

Q : Pourquoi ma ligne de Robux de paiement Premium gagnée est-elle pointillée ?
R : Votre ligne de Robux de paiement Premium gagnée sera initialement pointillée, car nous vous montrons une prédiction du montant de Robux que vous gagnerez grâce à cela. jour de temps de jeu Premium dans votre expérience. Nous aurons besoin d'un mois pour traiter vos Robux gagnés et nous assurer que le montant est exact. Une fois l'analyse terminée, la ligne Robux gagnée pour ce jour sera incluse dans votre Robux en attente .

Q : Dois-je faire quelque chose de spécifique pour recevoir des paiements Premium ?
R : Si votre expérience est déjà populaire parmi les abonnés Premium, vous recevrez un paiement sans avoir à modifier votre expérience. Au fur et à mesure que les abonnés Premium passent plus de temps dans votre expérience, vous devriez voir les paiements augmenter.
