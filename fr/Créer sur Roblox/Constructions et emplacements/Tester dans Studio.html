Tester dans Studio

Le mode test est une fonctionnalité de Roblox Studio qui vous permet de tester votre lieu une fois que vous avez terminé votre lieu, ou que vous voulez voir ce que fait quelque chose avant de le publier. Si vous voulez tester votre emplacement en le jouant via la fonction de jeu normale sur le site Roblox, vous pouvez le faire sans que d'autres joueurs ne rejoignent votre expérience en définissant votre expérience sur Privé.

Consultez l'article Documentation du créateur sur Modes de test Studio pour obtenir des informations détaillées.
