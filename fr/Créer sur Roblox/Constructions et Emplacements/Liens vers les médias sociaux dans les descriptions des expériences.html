Liens vers les médias sociaux dans les descriptions des expériences

Les développeurs peuvent souhaiter lier leurs autres profils de médias sociaux directement à leurs expériences. Roblox offre désormais un support officiel pour cela via la fonction Lien vers les médias sociaux ! Les développeurs pourront inclure jusqu'à trois liens vers des médias sociaux dans la description de leur expérience, de la même manière que dans un profil d'utilisateur. Chaque type de média social ne peut être ajouté qu'une seule fois.

Consultez l'article de la Documentation du Créateur pour en savoir plus sur les Liens de médias sociaux pour les expériences .
