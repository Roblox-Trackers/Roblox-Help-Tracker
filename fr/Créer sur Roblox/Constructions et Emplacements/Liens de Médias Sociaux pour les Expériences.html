Liens de Médias Sociaux pour les Expériences

Les développeurs peuvent souhaiter lier leurs autres profils de médias sociaux directement à leurs expériences. Roblox offre désormais un support officiel pour cela via la fonction Lien vers les médias sociaux ! Les développeurs pourront inclure jusqu'à trois liens de médias sociaux dans leurs descriptions d'expérience, de manière similaire à la méthode utilisée pour leurs profils d'utilisateurs. Chaque type de média social ne peut être ajouté qu'une seule fois.

Les liens vers médias sociaux qui peuvent être ajoutés sont :

  • Discord
  • YouTube
  • Twitter
  • Twitch
  • Facebook
  • Groupe Roblox

Comment ajouter des liens vers médias sociaux à votre expérience :

  1. Connectez-vous à votre compte
  2. Allez sur la page Créer
  3. Cliquez sur le lieu de l'expérience
  4. Cliquez sur le menu Liens sociaux dans la liste de gauche.
  5. Entrez l'URL, sélectionnez le type de média et entrez le titre
  6. Cliquez sur Enregistrer
  7. Une fois les liens enregistrés, voici à quoi ils ressembleront sur la page d'expérience.

Remarques importantes :

  • Les liens vers les réseaux sociaux ne seront visibles que pour les comptes de plus de 13 ans. Ils ne seront pas visibles pour les comptes qui ont moins de 13 ans.
  • Bien que les liens Discord soient permis dans la fonctionnalité de liens vers les réseaux sociaux, les utilisateurs ne sont pas autorisés à discuter ou à partager des liens Discord ailleurs et peuvent être modérés s'ils sont mentionnés ailleurs.
