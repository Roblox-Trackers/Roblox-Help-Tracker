Comment rendre quelque chose public ?

Les articles de la Boutique des Créateurs ( les modèles , les décalcomanies , les audios ou les plugins ) peuvent tous être créés pour le domaine public, ce qui signifie que la copie est autorisée. Toutefois, les accessoires, les vêtements, les chapeaux et les équipements ne peuvent pas être placés dans le domaine public.

La publication des articles sur la Boutique des Créateurs est le seul moyen de rendre vos articles disponibles publiquement pour tous les utilisateurs afin qu'ils puissent les utiliser dans le cadre de leurs propres expériences dans le Studio. Consultez l'article Publication de contenus pour plus d'informations.

À noter : Pour plus d'informations sur le partage de contenu, veuillez consulter les Conditions d'utilisation Roblox .
