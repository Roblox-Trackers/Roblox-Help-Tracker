Comment puis-je faire quelque chose du Domaine Public ?

Les éléments du marché des créateurs ( Les Modèles, Les Décalques, l’Audio et Les Plugins ) peuvent tous être créés pour le domaine public, ce qui signifie que la copie est autorisée. Pour rendre votre objet du Domaine Public ( copie autorisée) , procédez comme suit :

  1. Cliquez sur Créer en haut de l'écran.
  2. Cliquez sur Les Modèles , Les Décalques , L'Audio ou Les Plugins dans l'écran qui s'affiche.
  3. Trouvez l’objet que vous aimeriez rendre disponible dans le Domaine Public et cliquez sur l’icône d’engrenage située à sa gauche.
  4. Sélectionnez Configurer dans le menu qui s'affiche.
  5. Sous Copier, cochez la case Autoriser la copie
    • En cochant cette case, vous accordez à tous les autres utilisateurs de Roblox le droit d'utiliser (de diverses manières) le contenu que vous partagez maintenant. Si vous ne voulez pas conférer ce droit, veuillez ne pas cocher cette case. Pour plus d'informations sur le partage de contenu, veuillez consulter les Conditions d'utilisation Roblox .
  6. Cliquez sur Sauvegarder .

Actifs de publication sur la place de marché des créateurs est le seul moyen de rendre vos éléments accessibles au public pour que tous les utilisateurs puissent les utiliser dans leur propre expérience dans Studio. Vous pouvez publier n'importe quel maillage ou image que vous avez importé via le Asset Manager , ou n'importe quel modèle ou plug-in que vous avez téléchargé dans votre inventaire.

Remarque : les vêtements, chapeaux et équipements ne peuvent pas être placés dans le Domaine Public.
