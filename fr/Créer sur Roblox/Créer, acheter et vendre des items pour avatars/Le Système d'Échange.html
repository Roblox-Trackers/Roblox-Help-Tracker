Le Système d'Échange

Le Système d'Échange est une fonctionnalité géniale pour les utilisateurs avec une formule d'adhésion qui leur permet d'échanger des objets Limités/Limités U et des Robux avec d'autres utilisateurs. Les deux joueurs auront besoin d'avoir cette option activée dans l'onglet Confidentialité de leur page Paramètres du Compte .

Créer un échange

Pour démarrer un nouvel échange avec un autre joueur :

  1. Accédez à la page de profil de ce joueur.
  2. Cliquez sur le bouton qui ressemble à trois points dans le coin supérieur droit de la boîte qui contient leur nom d'utilisateur et les informations sur leurs amis/abonnés
  3. Dans le menu qui apparaît, sélectionnez Échangez Objets .
  4. Cela ouvrira une nouvelle page qui montre les objets disponibles que vous et le joueur devez échanger :
  5. Sélectionnez le ou les articles que vous souhaitez échanger. (Ils apparaîtront dans la liste Votre Offre .)
  6. Sélectionnez le ou les articles que vous souhaitez échanger. (Ils apparaîtront sur votre Liste de Demandes .)
  7. Vous pouvez également choisir d'adoucir l'accord en incluant des Robux supplémentaires, mais veuillez ne pas oublier que des frais de transaction de 30% seront supprimés une fois l'offre acceptée.
  8. Passez en revue l'échange pour vous assurer que vous êtes d'accord.
    Remarque : Roblox ne peut pas annuler un échange, vous devez donc être certain d'être satisfait de l'échange.
  9. Cliquez sur le bouton Faire une offre pour soumettre l'échange. Une fois que vous avez soumis, une fenêtre contextuelle apparaîtra pour vous permettre de confirmer la demande d'échange. (La soumission d'un nouvel échange informera l'autre joueur via un message privé.)

Remarque : Le montant des Robux ne peut pas dépasser 50% de l'offre actuelle, calculé après avoir retiré les frais de transaction. Par exemple, si vous proposez un article d'une valeur de 300 Robux, vous ne pouvez pas ajouter plus de 150 Robux (frais de transaction postaux) à l'offre.

Affichage de vos transactions actuelles ou passées

Pour visualiser vos échanges, procédez comme suit :

  1. Dans le menu à gauche de n'importe quelle page, cliquez sur l'option Échanger pour accéder à la page Échanges .
  2. Sélectionnez la liste des offres d'échanges en attente ( Entrantes / Sortantes ), Terminées , et Inactives que vous souhaitez voir dans le menu déroulant Type d'Échange.

Accepter, refuser et contrer une transaction

Pour visualiser vos offres entrantes, veuillez suivre les instructions ci-dessus.

  1. Accédez à l'échange que vous souhaitez examiner.
  2. Cliquez sur Accepter ou Refuser pour l'échange tel qu'il est indiqué.
  3. Si vous n'aimez pas l'offre et souhaitez la modifier, vous pouvez cliquer sur Comptoir pour modifier l'échange. Une fois que vous êtes satisfait de la contre-offre, appuyez sur Faire Offre pour envoyer une nouvelle demande d'échange.
