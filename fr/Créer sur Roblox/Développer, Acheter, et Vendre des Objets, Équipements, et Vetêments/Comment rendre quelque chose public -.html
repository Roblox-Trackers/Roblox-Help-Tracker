Comment rendre quelque chose public ?

Les objet de la Bibliothèque ( Les Modèles, Les Décalques, l’Audio, Les Plugins) peuvent être créés pour le domaine public, ce qui veut dire qu’on en autorise la copie. Pour rendre votre objet dans le domaine public (copie autorisée), suivez ces étapes :

  1. Cliquez sur Créer dans la barre bleu en haut de l’écran ou cliquez ici
  2. Cliquez sur Les Modèles, Les Décalques, l’Audio, Les Plugins dans l’écran qui s’affiche
  3. Trouvez  l’objet que vous aimeriez rendre disponible dans le domaine public et cliquez sur l’icône d’engrenage située à sa gauche
  4. Sélectionnez Configurer dans le menu qui s’affiche
  5. Sous Copier, cochez la case Autoriser la copie
    • Si vous cochez cette case, vous autorisez tout autre utilisateur Roblox à utiliser (de manières diverses) le contenu que vous partagez. Si vous ne voulez pas conférer ce droit, veuillez ne pas cocher cette case. Pour plus d’informations sur le partage de contenu, veuillez consulter les Conditions d'utilisation Roblox .
  6. Cliquez sur Sauvegardez

Note : On ne peut pas rendre public les vêtements, les chapeaux et l'équipement.
