J'ai un problème avec un jouet, un code ou un objet virtuel

J'ai un problème avec le code jouet

Vous pouvez nous contacter ici pour plus d'assistance. Assurez-vous d'inclure les informations suivantes :

  • Description du problème :
  • Nom d'utilisateur
  • Numéro de code
  • Image du recto et du verso de la carte de code d'élément virtuel (format .jpg ou .png uniquement)
  • Date d'achat
  • Magasin et ville d'achat
  • Image ou numérisation du reçu d'achat (format .jpg ou .png uniquement)
  • Nom du jouet (Si vous n'êtes pas sûr du nom du jouet, veuillez nous fournir une image, un format .jpg ou .png uniquement à partir des listes du Guide du collectionneur Jazwares )

J’ai un problème avec un objet virtuel

Si vous n'êtes pas certain d'avoir reçu l'objet virtuel de votre jouet, veuillez suivre ces étapes pour vérifier votre inventaire.

  • Assurez-vous d'être connecté au compte Roblox sur lequel vous avez utilisé le code jouet.
  • Si vous n'êtes pas sûr de quel objet virtuel est fourni avec votre jouet, vous pouvez consulter le site Web de Jazwares dans le Guide du collectionneur en trouvant votre jouet, puis en cliquant sur l'image de l'objet virtuel ou sur les mots de l' objet virtuel pour accéder à la page de l'article dans le catalogue Roblox.
  • Une fois connecté au compte sur lequel le code a été échangé, la page du catalogue de l'élément indique si le compte est propriétaire de l'article et vous propose différentes options. Voir ci-dessous:
  • Vous pouvez également afficher l'inventaire du compte et consulter la catégorie dans laquelle se trouve l'objet virtuel. Si l'élément apparaît dans l'inventaire du compte, vous en êtes propriétaire.

Si vous ne parvenez pas à localiser l'objet virtuel dans l'inventaire du compte ou pensez que vous avez reçu le mauvais article, contactez-nous ici pour plus d'assistance. Assurez-vous d'inclure les informations suivantes :

  • Description du problème :
  • Nom d'utilisateur
  • Numéro de code
  • Image du recto et du verso de la carte de code d'élément virtuel (format .jpg ou .png uniquement)
  • Date d'achat
  • Magasin et ville d'achat
  • Image ou numérisation du reçu d'achat (format .jpg ou .png uniquement)
  • Nom du jouet (Si vous n'êtes pas sûr du nom du jouet, veuillez nous fournir une image, un format .jpg ou .png uniquement à partir des listes du Guide du collectionneur Jazwares )

J'ai un problème avec un jouet physique lui-même

Vous pouvez contacter Jazwares directement à l'adresse suivante : customer_service@jazwares.com
