Roblox Star Code

Roblox s'est engagé à favoriser la croissance des activités et des revenus de nos influenceurs dans le programme Vidéo Stars . Le Star Code facilite le support de vos Vidéos Stars préférées sans frais supplémentaires.

Qu'est-ce que le Roblox Star Code ?

Le Roblox Star Code est un code spécial que vous pouvez obtenir auprès d’un membre du programme Roblox Vidéo Stars. Lorsque vous utilisez un Star Code pour prendre en charge une Vidéo Star, ils gagnent automatiquement 5% de l'achat que vous effectuez sur le site Web de Roblox. Vous recevrez toujours le montant total de Robux que vous avez payé, avec le bonus supplémentaire de savoir que vous avez contribué à votre Vidéo Star préférée !

Remarque: les Roblox Star Codes peuvent être appliqués aux achats sur le site Web de Roblox et sur l'application Roblox.

Comment utiliser le Star Code

Entrez le Star code associé à votre Roblox Vidéo Star préférée sur la page d'achat de Robux lors d'une transaction. Les Vidéo Stars feront la promotion de leurs Star codes sur leur profil Roblox ou sur leurs pages de groupe, ainsi que sur leurs comptes de réseaux sociaux, y compris YouTube, Twitter et bien plus encore !

Via le Site Web

  • Rendez-vous sur Robux ou sur Adhésion et sélectionnez la valeur que vous souhaitez acheter. Vous serez ensuite redirigé vers la page de sélection du mode de paiement :
  • Ayez votre Roblox Vidéo Star code préférée à portée de la main et appuyez sur Cliquez ici pour ouvrir la fenêtre de saisie du code. Entrez votre code et appuyez sur Ajouter .
  • Lorsque le code est accepté, la page affichera la personne que vous soutenez.

Via l'application

  • Ayez votre code Roblox Video Star préféré à portée de main.
  • Rendez-vous sur la page d'adhésion et faites défiler jusqu'à la fin de la page pour localiser le champ Star Code. Appuyez sur Touchez ici pour ouvrir la fenêtre de saisie du code.
  • Entrez votre code et appuyez sur Ajouter.
  • Lorsque le code est accepté, la page affichera la personne que vous soutenez.

Une fois que vous aurez ajouté votre code, il sera conservé pendant 2 semaines afin que vous n'ayez pas à le ressaisir chaque fois que vous effectuez un achat.
