Comment copier un jeu

Parfois il peut arriver que vous vouliez commencer un nouveau jeu mais vous voudriez éditer un jeu existant pour gagner du temps. Vous pouvez également souhaiter copier l'un de vos jeux sur un autre compte. Les étapes suivantes vous aideront à faire ces choses.

Comment copier son propre jeu

  1. Aller sur Roblox Studio
  2. Connectez-vous à votre compte dans Studio
  3. Cliquez sur Mes Jeux
  4. Trouvez le jeu que vous voulez copier, cliquez dessus, puis cliquez sur Modifier en bas de l'écran
  5. Une fois que vous aurez modifié le jeu, vous pouvez le copier de deux façons :
  • Si vous souhaitez simplement enregistrer une sauvegarde sur votre ordinateur, choisissez Fichier > Enregistrer sous
  • Si vous souhaitez télécharger une copie du jeu sur Roblox, choisissez Fichier > Publier sur Roblox Comme et remplacez un jeu existant ou créez-en un nouveau

Remarque : pour copier le jeu sur l'un de vos autres comptes, choisissez la méthode Enregistrer sous de l'étape 5. Une fois le jeu enregistré, déconnectez-vous de Studio, puis reconnectez-vous avec les identifiant de votre nouveau compte. Vous pouvez ensuite utiliser Publier sur Roblox pour l'uploader sur ce compte.

Comment utiliser comme modèle le jeu d'un autre utilisateur qui a autorisé la copie

Vous pouvez voir un jeu qui, selon vous, fonctionnerait comme un bon point de départ pour votre propre jeu. La bonne nouvelle est que certains constructeurs vous encouragent à utiliser leurs jeux à cette fin ! Cependant, un jeu ne peut être copié que si la mention "La source de ce jeu peut être copiée" est listée dans l'onglet A propos du jeu. Veuillez consulter notre article d'aide sur la façon d'autoriser ou de refuser la copie de votre jeu pour plus de détails.

Si le jeu peut être copié, les étapes suivantes vous aideront à l'utiliser comme point de départ pour votre propre jeu :

  1. Aller à la page des détails du jeu
  2. Cliquez sur l'icône qui ressemble à trois points en haut à droite de son titre
  3. Cliquez sur Modifier dans le menu qui apparaît
  4. Cela va ouvrir le jeu dans Studio où vous pouvez faire l'une des deux choses suivantes :
  • Choisissez Fichier> Enregistrer sous et enregistrez le jeu sur votre ordinateur. Vous pouvez ensuite éditer le jeu à votre goût.
  • Choisissez Fichier> Publier sur Roblox au fur et à mesure que vous téléchargez un jeu existant ou en créez un nouveau.

Remarque: Avant de décider de publier un jeu copié, assurez-vous d'abord de demander la permission du créateur. Si le jeu du joueur n'est pas autorisé à être copié, et que vous voulez faire votre propre version, assurez-vous d'abord de demander la permission au créateur. C'est poli.
