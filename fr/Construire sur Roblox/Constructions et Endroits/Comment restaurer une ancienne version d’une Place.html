Comment restaurer une ancienne version d’une Place

Roblox garde une copie de toute Place sauvegardée. Cela veut dire que, chaque fois que vous apportez des changements à une Place et les publiez sur Roblox, une copie sans tous ces changements est sauvegardée comme ancienne version. Vous pouvez trouver et restaurer une de ces anciennes versions à tout moment en faisant les suivantes :

Restaurer une ancienne version :

  1. Cliquez sur Créer , situé en haut de l’écran
  2. A partir de là, sélectionnez la Place que vous voudriez configurer
  3. Cliquez sur l’icône d’engrenage, situé à droite du nom de la Place et cliquez sur Configurer dans le menu qui s’affiche.
  4. Cliquez sur Historique des versions . Vous y verrez une liste avec toutes les anciennes versions, classées par date, en ordre décroissant
  5. Choisissez la version que vous voulez utiliser actuellement
  6. Cliquez sur Restaurer cette version pour la rendre la version actuelle
