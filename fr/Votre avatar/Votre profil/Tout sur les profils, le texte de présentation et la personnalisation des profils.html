Tout sur les profils, le texte de présentation et la personnalisation des profils

Voir un profil

Pour afficher votre propre profil, appuyez simplement sur votre nom d'utilisateur située sur l'écran d'accueil ou sélectionnez Profil dans le menu de navigation. Afin de voir le profil de quelqu'un d'autre :

  • Utilisation de la zone de recherche : vous pouvez saisir le nom du profil que vous souhaitez afficher dans la zone de recherche en haut du site. Sélectionnez ensuite l’option pour faire la recherche dans Joueurs. Puis cliquez tout simplement sur leur nom dans la liste des utilisateurs qui apparaît.
  • Créateurs d'objets ou de jeux : Vous pouvez afficher le profil d'un créateur de jeu ou d'objets en allant sur sa page de détails et en cliquant sur son nom. Pour les jeux, le nom se trouve sous le titre et pour les objets, celui-ci se trouve à droite de son image.

La boîte d'informations de base

La première chose que vous voyez lors de la visualisation d'un profil est la boîte d'information de base. Vous trouverez ici l'icône du joueur, son nom d'utilisateur, son niveau d'adhèsion, la dernière mise à jour de son statut, ses options d'interaction et les informations liées à ses amis/abonnés.

L'onglet Créations

Vous pouvez afficher les objets ou les jeux créés par un joueur en cliquant sur l'onglet Créations du profil du joueur. Celui-ci est situé sous la boîte d'informations de base. Cet onglet affichera tout ce que l'utilisateur a créé, classé par catégorie.

Par défaut, les jeux s'affichent dans une vue détaillée qui se concentre sur un jeu à la fois et peut être parcouru en cliquant sur les icônes fléchées à gauche et à droite de la zone de description. Vous pouvez toutefois afficher plusieurs jeux à la fois en cliquant sur l'icône

Choisir ce qui s’affiche sur votre onglet Créations

Parfois, vous constaterez peut-être que certaines de vos créations reflètent mieux vos qualités de développeur que les autres. Si c’est le cas, vous pouvez choisir de ne présenter que celles-là. Pour faire ça:

  1. Visitez la page des détails du jeu / de l'objet
  2. Cliquez sur l’icône en forme d’engrenage ou sur l'icône qui ressemble à trois points à droite du titre
  3. Dans le menu qui apparaît, cliquez sur Ajouter au profil ou Supprimer du profil

L'onglet À propos - le descriptif et les réseaux sociaux

Parfois, les utilisateurs peuvent souhaiter faire savoir aux gens leurs pensées, ou peut-être qu'ils veulent tout simplement divertir leurs collègues avec des citations amusantes. Le texte de présentation, visible dans l'onglet À propos , est l'endroit idéal pour cela. Pour en créer ou pour modifier le texte existant :

  1. Connectez-vous à votre compte
  2. Aller à la page Profil
  3. Tapez votre nouveau texte dans la case À propos
  4. Cliquez sur Enregistrer

Comment ajouter des liens sociaux à un compte

Si vous êtes âgé 13 ans ou plus, vous pouvez ajouter des réseaux sociaux à votre propre profil depuis l'onglet Social de la page des paramètres de votre compte , en procédant comme suit:

  1. Connectez-vous sur votre compte
  2. Sélectionnez l'icône en forme de roue dentée, située dans le coin supérieur droit du site
  3. Dans le menu contextuel, sélectionnez Paramètres
  4. Faites défiler vers le bas pour les Réseaux sociaux
  5. Entrez l'URL, cliquez sur Enregistrer

  6. Sélectionnez l'option Visible pour pour déterminer les comptes auxquelles vous souhaitez afficher ces liens.

Remarque: les liens de réseaux sociaux ne seront visibles que pour les comptes âgés de plus de 13 ans.

Dans le coin supérieur droit de votre boîte blurb, vous verrez tous les réseaux sociaux auxquels un joueur a adhéré.

L'onglet À propos de - Tenue actuelle

Dans la section Tenue actuelle de l'onglet À propos du profil, il y aura une image décrivant l’avatar actuel du joueur.  Vous pouvez basculer entre les vues 2D et 3D en cliquant sur le bouton correspondant dans le coin supérieur droit de l'image. Dans la boîte bleue à droite de cette image, vous verrez une liste de tout ce que l'utilisateur porte. Si vous voyez quelque chose que vous aimez, vous pouvez cliquer dessus pour en acheter ou pour obtenir plus de détails.

L'onglet À propos - Autres sections

Voici les autres sections que vous trouverez dans l'onglet À propos :

Amis

Affiche jusqu'à neuf amis du joueur. S'il en a plus de neuf, vous pouvez les voir en cliquant sur le bouton bleu Voir tout.

Collections

Cela vous montrera jusqu'à six articles qu'un joueur a acheté. Par défaut, ce seront les objets qu’il vient d’obtenir, mais le joueur peut choisir ce qui s'affiche dans cette section s'il le souhaite. Pour voir l'inventaire complet d'un joueur classé par catégorie, cliquez sur le bouton bleu Inventaire. Nous allons discuter comment choisir ce qui s'affiche dans cette section un peu plus loin dans cet article.

Groupes

Cette section montrera tous les groupes auxquels le joueur appartient. Tout comme la section jeux de l'onglet Créations, les groupes s'afficheront dans une vue détaillée qui se concentre sur un seul groupe à la fois. Vous pouvez les faire défiler en cliquant sur les icônes fléchées à gauche et à droite de la boîte de description. Vous pouvez toutefois afficher plusieurs groupes à la fois en cliquant sur l'icône

Jeux préférés

Ici, vous verrez une sélection de jeux qu'un joueur a choisi comme favoris. Pour voir tous les favoris d'un joueur, cliquez simplement sur le bouton bleu Favoris. Pour plus d'informations sur la manière de jouer à un jeu, cliquez ici .

Badges de Roblox

Ceci affiche une liste de tous les badges créés par Roblox que l'utilisateur a gagnés. Pour voir comment un badge a été obtenu, il suffit de cliquer dessus.

Badges du joueur

Cela contient les badges créés par des joueurs qu'un joueur a obtenus en faisant de diverses choses dans certains jeux. Pour plus d'informations sur la façon dont un badge est gagné, il suffit de cliquer dessus.

L’onglet À propos - Choisir ce qui s'affiche dans vos collections

Parfois, vous souhaitez peut-être afficher des éléments spécifiques dans la barre des collections de votre profil. Pour faire ça:

  1. Visitez la page de détails d’un objet que vous possédez
  2. Cliquez sur l'icône en forme d’engrenage à droite du titre
  3. Dans le menu qui apparaît, cliquez sur Ajouter au profil ou Supprimer du profil
