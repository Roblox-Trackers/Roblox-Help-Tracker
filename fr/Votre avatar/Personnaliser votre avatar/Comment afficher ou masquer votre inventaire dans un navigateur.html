Comment afficher ou masquer votre inventaire dans un navigateur

Suivez ces étapes pour afficher l'inventaire de votre compte dans un navigateur Web :

  1. Cliquez sur l'icône de menu située en haut à gauche de l'écran.
  2. Cliquez sur l'option Inventaire .
  3. Utilisez le menu pour sélectionner une catégorie et une sous-catégorie (le cas échéant).
  4. Faites défiler votre inventaire sur la page. Au bas de chaque page, vous pouvez appuyer sur les boutons fléchés vers la droite ou vers la gauche pour parcourir les différentes pages.
  5. Cliquez sur l'image de l'article que vous souhaitez afficher.
  6. Une fois que vous avez fait votre sélection, la page de l'article affichera l'article comme "Article possédé" avec une coche à côté. La page comprend également des informations telles que le nom, la description et le genre (le cas échéant).

Pour masquer votre inventaire pour les autres joueurs

  1. Accédez à la page Paramètres en cliquant sur l'icône d'engrenage située dans le coin supérieur droit de votre page de profil.
  2. Dans Paramètres, cliquez sur l'option Confidentialité .
  3. Choisissez le groupe d'utilisateurs pour lequel vous souhaitez voir votre inventaire dans le menu Qui peut voir mon inventaire ? Si vous sélectionnez Personne , votre inventaire ne sera visible pour aucun utilisateur.

Veuillez noter : masquer votre inventaire n'affectera pas votre capacité à utiliser le système d'échange .
