Comment faire de mon avatar un garçon ou une fille ?

C'est au moment où vous créez votre compte que vous choisissez si votre avatar est un garçon ou une fille, mais vous pouvez aussi le changer plus tard.

Changer le genre du compte

Vous pouvez changer votre genre en l’éditant dans la page dse paramètres de votre compte. Veuillez suivre ces étapes afin de changer votre genre :

  • Allez sur l'icône d’engrenage dans le coin supérieur droit de votre écran et sélectionnez Paramètres dans le menu qui s’affiche
  • Sur la page des Paramètres, descendez en bas de la page jusqu’à ce que vous arrivez à la section Personnel
  • Sélectionnez l'icône de sexe appropriée

Note: Le changement de ces paramètres ne va pas modifier l’apparence de votre avatar.

Changer l’avatar-même

Pour changer votre apparence, vous pouvez choisir les paquets gratuits d’avatar depuis le catalogue :

  • Paquet Roblox Homme
  • Paquet Roblox Femme
  • Paquet Roblox Fille
  • Paquet Roblox Garçon

Afin de changer le paquet du corps :

  • Navigateur : Personnaliser votre avatar
  • Application Mobile : Personnaliser votre avatar
