Application Mobile : Personnaliser votre Avatar

Veuillez consulter la rubrique Application Mobile : Visualisation de votre inventaire article d'aide afin d’accéder à l’inventaire de vos articles depuis votre mobile.

Comment porter un article de l’inventaire ou comment l'enlever

Par téléphone

  1. Sélectionnez l’article que vous souhaitez porter ou que vous voulez supprimer
  2. Pour porter l'article, déplacez le curseur Porter cet article sur Oui
  3. Pour le supprimer, bouger le même curseur sur Non

Par la tablette

  1. Appuyez sur le bouton Plus situé en bas de l’écran
  2. Appuyez sur Avatar
  3. À partir d'ici, vous pouvez habiller votre avatar comme vous le feriez sur le site Web. Pour obtenir de l'aide, veuillez appuyer ici .

Comment changer la couleur de peau de votre avatar

Téléphone

  1. Connectez-vous à votre compte Roblox sur l'Application Mobile.
  2. Allez dans l'onglet Avatar.
  3. Cliquez sur " Personnaliser " et sélectionnez " Corps .
  4. Depuis l'onglet Peau, vous pourrez sélectionner la couleur de peau souhaitée.

Par la tablette

  1. Cliquez sur le bouton Plus situé en bas de l’écran
  2. Appuyez sur Avatar
  3. À partir d'ici, vous pouvez changer la couleur de votre peau comme vous le feriez sur le site Web. Pour obtenir de l'aide, veuillez appuyer ici .
