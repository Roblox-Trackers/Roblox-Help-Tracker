Navigateur: Personnaliser votre Avatar

Où puis-je modifier mon avatar ?

Appuyez sur Avatar situé dans le menu de navigation.

Comment modifier mon avatar ?

Pour ajouter un élément

  1. Sur la page Éditeur d'avatar, appuyez sur l'un des menus déroulants disponibles et sélectionnez une catégorie (par exemple: chapeau, cheveux, visage, etc.)
  2. Si vous avez plusieurs éléments pour une partie donnée, faites défiler vers le bas jusqu'à ce que l'élément souhaité soit localisé.
  3. Une fois choisi, appuyez sur l'icône de l'élément qui le mettra en surbrillance.

Si vous souhaitez ajouter plus que la limite souple (par exemple, plus de trois chapeaux) d'un article donné, appuyez sur Avancé dans le coin inférieur droit. Vous pouvez ajouter jusqu'à 10 articles en entrant l'ID d'actif qui se trouve dans le lien de l'article. Voir exemple ci-dessous :

https://www.roblox.com/catalog/ 62234425 /Cheveaux-Bruns <- Les chiffres dans le lien sont l'identifiant de l'actif

Pour supprimer un élément

  1. Sur la page de l'Éditeur d'Avatar, appuyez sur Récent ou sélectionnez la catégorie applicable (par exemple: chapeau, cheveux, visage, etc.)
  2. Dans le menu déroulant Récent, les articles actuellement portés doivent figurer parmi les premiers répertoriés. Si vous sélectionnez plutôt la catégorie applicable, faites défiler vers le bas jusqu'à ce que vous trouviez ce que vous souhaitez supprimer.
  3. Appuyez sur l'icône de l'élément en surbrillance.

Remarque : Si vous avez ajouté ou supprimé un élément et que votre avatar ne l'affiche pas correctement, appuyez sur le lien "Redessiner" sous la photo de votre avatar.

Pour changer de couleur de peau

Bien que changer vos objets/vêtements soit certainement amusant, vous souhaiterez parfois changer la couleur même de la peau de votre avatar. Les étapes suivantes vous aideront à changer la couleur de peau de votre avatar via le site Web Roblox.

  1. Accédez à la section Avatar de votre compte sur la gauche de la page.
  2. Sur cette page, déplacez votre curseur sur la rubrique Tête & Corps du menu déroulant et sélectionnez Teint de peau .
  3. À partir de là, appuyez sur une couleur pour changer l'apparence de tout votre avatar.
  4. Si vous souhaitez une couleur différente pour certaines parties du corps, appuyez sur Avancé dans le coin inférieur droit.
  5. À partir de là, appuyez sur une partie du corps, puis sélectionnez la couleur souhaitée.
