Voir votre inventaire sur le site Web

Suivez ces étapes pour afficher l'inventaire de votre compte sur le site Web:

  1. Cliquez sur l'icône de menu située en haut à gauche de l'écran.
  2. Cliquez sur l'option Inventaire.
  3. Utilisez le menu pour sélectionner une catégorie de garde-robe et une sous-catégorie (le cas échéant).
  4. Faites défiler votre inventaire sur la page. Au bas de chaque page, vous pouvez appuyer sur les flèches vers la droite ou vers la gauche pour parcourir différentes pages.
  5. Cliquez sur l'image que vous souhaitez voir.
  6. Une fois que vous avez fait votre choix, la page de l'élément affiche l'élément en tant qu'”élément possédé" avec une coche à côté. La page comprend également des informations telles que le nom, la description et le genre (le cas échéant).

Cacher votre inventaire pour d'autres joueurs

  1. Visitez votre page de profil
  2. Accédez à la page Paramètres en cliquant sur l'icône représentant un engrenage dans le coin supérieur droit de la page de votre profil.
  3. Dans les paramètres, cliquez sur l'onglet Confidentialité
  4. Dans l'onglet Confidentialité , cliquez sur Qui peut voir mon inventaire?
  5. Choisissez quel groupe d'utilisateurs vous aimerait pouvoir voir votre inventaire. Si vous sélectionnez Personne , vous garantissez que votre inventaire n'est visible par aucun utilisateur.

Remarque: masquer votre inventaire n'affectera pas votre capacité à utiliser le système de négociation .
