Comment faire de mon Avatar une fille ou un garçon ?

Le choix d'un avatar garçon ou fille se fait d'abord lors de la création de votre compte, mais vous pouvez le mettre à jour.

Changer le genre du compte

Vous pouvez changer votre genre en l’éditant dans la page des paramètres de votre compte. Veuillez suivre ces étapes afin de changer votre genre :

  • Accédez à l' icône d'engrenage dans le coin supérieur droit de votre écran et sélectionnez Paramètres dans le menu qui apparaît
  • Sur la page Paramètres, faites défiler jusqu'à la section Personnel
  • Sélectionnez l'icône du genre appropriée

Remarque : La modification de ces paramètres ne modifiera pas l'apparence de votre avatar.

Changer l’avatar-même

Pour changer de look, vous pouvez acheter des avatars gratuits sur le Marketplace :

  • Package pour homme
  • Package pour femme
  • Package Roblox fille
  • Paquet Roblox garçon
