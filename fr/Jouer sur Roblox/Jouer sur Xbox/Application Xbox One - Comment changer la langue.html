Application Xbox One : Comment changer la langue

Nous vous offrons plusieurs traductions du texte officiel Roblox sur l’Application Xbox. Cependant, le contenu dans le jeu pourrait s’afficher toujours en anglais. Pour l’instant, les langues suivantes sont disponibles :

  • Portugais brésilien
  • Anglais
  • Français
  • Allemand
  • Italien
  • Russe
  • Espagnol

Comment changer

Pour changer la langue allez sur Paramètres système > Changer la langue

Note : Vous devriez peut-être fermer et relancer l’application Roblox pour voir les changements.
