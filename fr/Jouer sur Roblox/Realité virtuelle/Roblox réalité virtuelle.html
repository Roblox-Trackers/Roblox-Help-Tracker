Roblox réalité virtuelle

Améliorez votre expérience Roblox avec la réalité virtuelle ! Roblox est compatible avec Oculus Rift et HTC Vive.

Comment activer la réalité virtuelle

Pour activer votre expérience VR avec Roblox, vous devez d'abord vous assurer que vous avez installé Roblox ( http://www.roblox.com/download ) puis suivre les étapes ci-dessous:

Veuillez vous assurer que votre appareil est correctement connecté à votre PC et que VR Mode est mis sur " On " dans les paramètres de l'expérience (Menu>Settings>VR>On). Pour plus de détails, voir Comment alterner en mode VR ci-dessous.

HTC Vive :

  1. Depuis votre ordinateur, ouvrez l' application SteamVR .
  2. Assurez-vous que vos stations de base sont alimentées.
  3. Assurez-vous que votre casque est connecté à l'ordinateur via le boîtier de liaison.
  4. Allumez les contrôleurs :
    (Pour allumer le contrôleur, appuyez sur le bouton Système jusqu'à ce que vous entendiez un bip sonore )
    (Pour éteindre le contrôleur, maintenez enfoncé le bouton Système jusqu'à ce que vous entendiez un bip sonore)
  5. Les icônes d'état sur l' application SteamVR et les voyants d'état sur le matériel doivent tous être verts.

    Pour afficher le tableau de bord du système , appuyez sur le bouton Système de l'un ou l'autre des contrôleurs.

    Remarque : lorsque vous quittez l' application SteamVR , les contrôleurs s'éteignent automatiquement. Les contrôleurs s'éteignent également automatiquement après avoir été inactifs pendant un certain temps.

Oculus brèche :

  1. Ouvrez l'application Oculus sur votre ordinateur et sélectionnez l'icône en forme d'engrenage en haut à droite
  2. Sélectionnez les Paramètres d'Engrenage
  3. Sélectionnez Général dans le menu à gauche
  4. Basculez le paramètre Sources inconnues pour qu'il apparaisse comme une vérification au lieu d'un X
  5. Confirmez votre sélection pour activer ce paramétrage
  6. Sélectionnez Jouer sur une expérience Roblox : http://www.roblox.com/discover
  7. Vous verrez un écran carré sur votre moniteur indiquant le mode RV (comme dans l'image ci-dessous)
  8. Mettez votre casque et jouez !

Comment utiliser le menu RV Radial

Vous pouvez accéder à tous les paramètres importants du système en utilisant le menu RV Radial .

  1. Appuyez sur le bouton Menu pour faire apparaître le menu RV Radial
  2. Sélectionnez une des options suivantes :
    • Menu du Système - Options du système d'accès
    • List des joueurs - Voir la liste des joueurs du serveur de l'expérience
    • Recentrer la Camera - Recentrer la caméra sur votre personnage
    • Notifications - Afficher les notifications Roblox
    • Quitter l'expérience - Quitter l'expérience
    • Sac à dos - Affichez votre inventaire dans l'expérience
    • Bouton 2D UI - Activer/désactiver la fonction d'interface utilisateur 2D, cela doit rester activée pour une expérience optimale sur Roblox.

Les options RV Radial Menu décrites ci-dessus correspondent aux boutons de l'image suivante, en commençant par "Menu du Système" en haut et an allant dans le sens contraire des aiguilles d'une montre.

Comment basculer en mode RV

Vous pouvez activer ou désactiver le mode RV dans l'expérience, en basculant l'affichage entre votre casque RV et votre écran d'ordinateur.

Lorsque le mode RV est désactivé , vous pouvez l'activer en procédant comme suit :

  1. Appuyez sur le bouton Menu pour faire apparaître le menu Radial RV
  2. Sélectionnez l'option Menu du Système
  3. Basculez l'option RV sur Désactivé

Lorsque le mode RV est activé , vous pouvez le désactiver en procédant comme suit :

  1. Affichez le menu du Système en appuyant sur Esc de votre clavier ou en cliquant sur le bouton du menu en forme de hamburger dans le coin supérieur gauche de l'écran.
  2. Basculez l'option RV sur Activé
