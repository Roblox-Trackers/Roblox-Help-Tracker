Application Mobile : Demandes d'amitié

La possibilité d'accepter et d'envoyer des demandes d'amis est également disponible sur l'application mobile Roblox.

Accepter ou refuser des demandes d'amitié :

Portables :

  1. Appuyez sur le bouton Plus en bas de l'écran
  2. Appuyez sur Amis (le nombre total actuel des demandes est affiché à droite)
  3. À partir de là, appuyez sur Demandes
  4. Appuyez sur Accepter ou Refuser

Tablettes :

  1. Appuyez sur le bouton Amis en bas de l'écran
  2. Appuyez sur l' onglet Demandes
  3. Appuyez sur le bouton Accepter ou Ignorer correspondant à la demande que vous souhaitez examiner

Envoyer une Demande d’Amitié :

Portables :

  1. Appuyez sur le bouton Plus en bas de l'écran
  2. Appuyez Amis
  3. Appuyez sur l'icône de la barre de Recherche en haut
  4. Tapez le nom d'utilisateur de votre ami et faites une recherche
  5. Cliquez sur le bouton Ajouter ami à droite de leur nom d'utilisateur

Note : Si vous avez déjà envoyé une demande d'amitié, vous ne pourrez pas en envoyer une autre et son profil indiquera "En attente" ou il affichera une icône "Chargement".

Tablettes :

  1. Appuyez sur le bouton Accueil au bas de l'écran
  2. Appuyez sur l'icône en haut de l'écran
  3. Tapez le nom du joueur auquel vous souhaitez envoyer l'invitation
  4. Appuyez sur le nom du joueur dans la liste
  5. Appuyez sur le bouton Ajouter ami

Note : Si vous avez déjà envoyé une demande d'amitié, vous ne pourrez pas en envoyer une autre et son profil indiquera "En attente"
