Configuration de VR pour Roblox

Améliorez votre expérience Roblox avec la réalité virtuelle ! Oculus Rift, Meta Quest Pro, Meta Quest 2, Quest 2 avec lien, ainsi que HTC Vive et Valve Index sur Windows PC.

D'autres appareils (Pico, Reverb, etc.) sont officieusement pris en charge via SteamVR.

Comment activer le VR

Pour activer votre expérience de réalité virtuelle avec Roblox, vous devez d’abord vous assurer que vous avez installé Roblox , puis suivre les étapes ci-dessous :

Assurez-vous que votre appareil est correctement connecté à votre PC et que le mode VR est " On " dans les paramètres de l’expérience (Menu>Paramètres >VR>On). Pour plus de détails, voir Comment basculer en mode VR ci-dessous.

Veuillez installer et configurer l’application VR pour votre appareil : Oculus VR ou Steam VR.

Steam VR:

Appareils pris en charge : HTC Vive, Valve Index et autres

  1. Depuis votre ordinateur, installez et ouvrez l’application SteamVR .

  2. Activer OpenXR comme backend pour SteamVR

  3. Allumez les contrôleurs : (Pour allumer le contrôleur, appuyez sur le bouton Système jusqu’à ce que vous entendiez un bip)
    (Pour éteindre le contrôleur, appuyez et maintenez le bouton Système jusqu’à ce que vous entendiez un bip)

  4. Les icônes d’état de l’application SteamVR et les voyants d’état du matériel doivent tous être verts.

    Pour afficher le Tableau de bord du système , appuyer sur le bouton Système de l’une des commandes.

    Remarque : Lorsque vous quittez l’application SteamVR , les contrôleurs s’éteignent automatiquement. Les contrôleurs s’éteindront également automatiquement après avoir été inactif pendant un certain temps.

  5. Sélectionnez Jouer sur une expérience Roblox : Expériences Roblox

  6. Mettez votre casque et jouez !

Oculus VR:

Appareils compatibles : Oculus Quest2 avec lien, Appareils compatibles : Oculus Quest2 avec lien, Appareils compatibles : Oculus Quest2 avec lien, Meta Quest 2, Meta Quest Pro, Oculus Rift et compatible

  1. Installez et ouvrez l’application Oculus sur votre ordinateur
    • Remarque : Les pilotes Oculus ne se chargeront pas, sauf si votre ordinateur est équipé d’au moins une carte graphique NVidia GTX1060 ou équivale
  2. Confirmez que votre appareil est connecté et que le statut est prêt
  3. Sous Paramètres / Général dans l’application Oculus, sélectionnez OpenXR Runtime
  4. Sélectionnez Jouer sur une expérience Roblox : Expériences Roblox
  5. Mettez votre casque et jouez !

Si vous utilisez Quest2 avec un câble de liaison, vous devrez activer le lien à l’intérieur du casque.

Pour confirmer que votre appareil est connecté, procédez comme suit :

  • Ouvrez l’onglet Périphériques et trouvez votre casque
  • Confirmez que l’état est vert et qu’aucune erreur n’est affichée

Si vous utilisez Meta Quest 2 et Meta Quest Pro, vous devrez faire ce qui suit :

  1. Recherchez Roblox dans le Quest Store et installez-le à partir de là
  2. Assurez-vous que votre appareil Meta Quest est à jour (Roblox nécessite la version 55 du logiciel Meta Quest.)
  3. Démarrez Roblox sur votre appareil Quest
  4. Connectez-vous à votre compte Roblox existant* en utilisant l’option de connexion rapide sur votre ordinateur ou votre téléphone.

*Remarque : Pour le moment, nous ne prenons pas en charge la création de compte ou l’ouverture de session par mot de passe sur les appareils Meta Quest.

Comment activer le mode VR

Vous pouvez activer ou désactiver le mode VR dans le jeu.

Remarque : Le moteur doit redémarrer pour que cela prenne effet. La désactivation du mode VR dans les paramètres de l’utilisateur empêchera le moteur de démarrer en VR.

Lorsque le mode VR est activé , vous pouvez le désactiver en procédant comme suit :

  1. Sélectionnez l’icône Menu pour afficher le menu en jeu
  2. Sélectionnez l’option Paramètres
  3. Basculer l’option VR sur Off et redémarrer

Lorsque le mode VR est Off , vous pouvez le basculer sur On en procédant comme suit :

  1. Affichez le Menu du Système en appuyant sur ESC de votre clavier ou en cliquant sur le bouton du menu en forme de hamburger dans le coin supérieur gauche de l'écran
  2. Basculer l’option VR sur On et redémarrer
