Système d'animation (émoticônes)

Le système d'animation Roblox permet une quantité mesurée de fluidité et de réalisme pour les mouvements. Une partie de ce système comprend les émoticônes, qui sont des actions spécifiques que votre avatar Roblox peut effectuer. Pour voir les émoticônes par défaut, appuyez sur la touche "/" pour tchatter et essayez de saisir les commandes suivantes :

  • /e signe de la main
  • /e index pointant vers l'utilisateur
  • /e acclamation
  • /e visage riant aux larmes
  • /e danseuse
  • /e danse2
  • /e danse3

Remarque : Cette fonctionnalité n'est pas disponible si le chat intégré est désactivé. Pour plus d'informations, cliquez ici.
