Comment relancer votre Avatar - Dans une expérience

Réinitialiser l'avatar vous permet de réapparaître et de recommencer à un point d'apparition. Ceci est utile si votre avatar perd un bras (en particulier le bras de votre arme) ou si vous êtes coincé dans un mur.

Pour réinitialiser votre avatar

  1. Cliquez sur le logo Roblox dans le coin supérieur gauche de l'écran pour ouvrir le menu.
  2. Cliquez sur Réinitialiser le caractère ou appuyez sur la touche R de votre clavier.
  3. Une fenêtre contextuelle apparaîtra, vous demandant de confirmer l'action. Cliquez sur Réinitialiser et votre avatar réapparaîtra à un point d'apparition.
