Application bêta Roblox

L'application Roblox modifie la façon dont les joueurs rejoignent les jeux et où ils peuvent acheter Premium et Robux.

Comment installer

  • Si votre compte a accès à l'application Roblox Beta, vous verrez une bannière pour l'application.
  • Cliquez sur le bouton Essayer maintenant dans la bannière.
  • Une fenêtre pop-up apparaîtra indiquant que le lecteur Roblox est en cours d'installation
  • Une deuxième fenêtre pop-up  peut apparaître vous demandant de sélectionner un programme, sélectionnez Ouvrir Roblox .

Comment quitter la bêta

Si vous souhaitez désactiver l'application Roblox Beta, suivez les instructions ci-dessous :

  • Ouvrez l'application bêta
  • Cliquez sur Plus, puis allez dans les paramètres
  • Au bas de l'écran des paramètres, vous verrez l'option de désactiver l'application bêta.

  • Lorsque vous cliquez sur l'option Désactiver la bêta , une enquête Roblox s'ouvrira lorsque vous quitterez, veuillez nous dire ce que vous en pensez.

Remarque : cliquer sur la bannière de la page d'accueil vous ramènera dans l'application bêta.

Différences entre les applications bêta

Bien que vous connaissiez peut-être la plupart de l'interface si vous avez joué à Roblox normalement, il y a quelques différences clés à noter lorsque vous jouez sur la version bêta.


Menu principal

Vous verrez que plusieurs fonctionnalités sont directement disponibles dans le menu principal de l'application bêta, telles que la page d'accueil, la page des jeux, l'éditeur d'avatar et la fenêtre de discussion. Vous pouvez ouvrir n'importe lequel d'entre eux sur le panneau de gauche lors de la première ouverture de l'application.

Quitter le jeu (L)

Ce bouton permet au joueur de quitter le jeu, ce qui le déconnecte du serveur sur lequel il se trouvait. Après avoir cliqué sur le bouton, il y a un autre message pour confirmer s'ils veulent partir. Quitter les jeux de cette manière ne fermera pas la fenêtre du jeu et vous ramènera au menu principal.

Pour quitter complètement immédiatement, vous pouvez cliquer sur le bouton X en haut à droite dans la fenêtre du jeu.
