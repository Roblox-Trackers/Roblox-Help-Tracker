Comment suivre ou joindre un autre joueur dans les jeux

Vous pouvez jouer avec vos amis à un jeu auquel ils jouent déjà ! Tant que votre ami a activé le paramètre "En jeu", vous pourrez le rejoindre dans un jeu en cliquant sur un bouton.

Remarque : Vous pourrez rejoindre le jeu seulement si vous avez les autorisations nécessaires. Pour plus d'informations, consultez nos articles d'aide Créer  et rendre publics ses jeux et Jeux de groupe .

Pour rejoindre un autre joueur

  • Recherchez le nom d'utilisateur ou accédez à la page de son profil
  • Si cet utilisateur est actuellement dans un jeu et permet aux utilisateurs de le suivre, une option Rejoindre le jeu s’affichera dans les résultats de la recherche ou sur la page de son profil.
  • Cliquez sur Rejoindre le jeu et vous serez placé dans le jeu avec eux

Aide supplémentaire

Fonctions de sécurité: Chat, confidentialité et filtrage (y compris l'activation / la désactivation ou la limitation de la possibilité des autres de vous rejoindre
