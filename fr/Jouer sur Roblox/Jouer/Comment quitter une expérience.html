Comment quitter une expérience

Lorsque vous avez fini de jouer à une expérience, vous pouvez partir à tout moment.

  1. Cliquez sur le logo Roblox dans le coin supérieur gauche de l'écran pour ouvrir le menu.
  2. Cliquez sur Quitter ou appuyez sur la touche L de votre clavier.
  3. Une fenêtre contextuelle vous demandera de confirmer l'action ; cliquez sur Quitter.
