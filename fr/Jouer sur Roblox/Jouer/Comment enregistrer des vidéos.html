Comment enregistrer des vidéos

Parfois vous voudriez peut-être sauvegarder un souvenir sur Roblox et une capture d'écran tout simplement ne va pas suffire. Vous avez besoin d'une vidéo ! Heureusement, Roblox a tout ce dont vous avez besoin. Pour enregister une vidéo, nous vous prions d'essayer les manipulations suivantes :

  1. Allez dans le jeu, cliquez sur le buton menu dans le coin supérieur gauche

2. Dans le menu du jeu, cliquez sur l'onglet Enregistrer

3. L'option Paramètres Vidéo est l'endroit où vous allez choisir ce que vous souhaitez faire avec la vidéo quand vous avez fini d'enregistrer :

  • Sauvegarder sur le Disque – cela va enregistrer le fichier dans votre ordinateur
  • Télécharger sur Youtube : cela va automatiquement ajouter le fichier sur votre compte Youtube, même si pour l'instant, cette option n'est pas valide

Pour commencer l'enregistrement cliquez sur Enregistrer la Vidéo .

4. Ensuite faites votre truc ! Quand vous avez terminé, cliquez sur l'icône dans le coin supérieur gauche

5. Pour visualiser votre vidéo, cliquez sur le lien Mes Vidéos dans la fenêtre qui apparait.
