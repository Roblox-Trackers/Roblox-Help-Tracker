Comment enregistrer des vidéos

Parfois, vous voudrez peut-être économiser de la mémoire Roblox et une capture d'écran ne suffira pas. Vous avez besoin d'une vidéo ! Pour enregistrer une vidéo, procédez comme suit :

  1. Entrez dans l'expérience, cliquez sur le bouton de menu dans le coin supérieur gauche
  2. Dans le menu de l'expérience, cliquez sur l'onglet Enregistrer
  3. Pour démarrer l'enregistrement, cliquez sur Enregistrer la vidéo .
  4. Alors faites votre truc ! Lorsque vous avez terminé, cliquez sur l'icône rouge dans le coin supérieur gauche
  5. Pour visionner votre vidéo, cliquez sur le lien Mes vidéos dans la fenêtre qui s'affiche

Remarque : Actuellement, la fonction d'enregistrement n'est pas disponible sur les plates-formes Mac.
