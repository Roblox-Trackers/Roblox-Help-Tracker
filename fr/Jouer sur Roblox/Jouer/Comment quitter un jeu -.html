Comment quitter un jeu ?

Quand vous avez fini de jouer un jeu, vous pouvez le quitter à n’importe quel moment.

  1. Cliquez sur le menu qui ressemble à trois lignes horizontales, situé dans le coin supérieur gauche de l'écran.
  2. Cliquer sur Quitter Jeu ou appuyez sur la touche “L” de votre clavier.>
  3. Une fenêtre pop-up apparaîtra sur votre écran pour vous demander de confirmer l'action; cliquez sur Quitter.
