Le Système d'Animations (émoticônes)

Le Système d'Animation Roblox permet une certaine fluidité et réalisme des mouvements. Les Emoticônes font partie de ce système, ce sont des actions spécifiques que l'avatar Roblox peut effectuer. Afin de voir la liste avec les émoticônes par défaut, appuyez la touche “/” et essayez de taper les commandes suivantes :

  • /e wave
  • /e point
  • /e cheer
  • /e laugh
  • /e dance
  • /e dance2
  • /e dance3

Remarque : Cette caractéristique n'est pas disponible si le chat dans le jeu a été désactivé. Pour en savoir plus, cliquez ici .
