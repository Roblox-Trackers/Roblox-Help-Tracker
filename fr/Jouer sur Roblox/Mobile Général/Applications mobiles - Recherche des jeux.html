Applications mobiles : Recherche des jeux

Même si vous pouvez rechercher des jeux sur Roblox via votre ordinateur, si vous utilisez une tablette ou un téléphone, vous pouvez également le faire sur la version d’Application Mobile de Roblox.

Comment trouver des jeux

Il y a plusieurs façons de rechercher des jeux sur Roblox.

Parcourir les catégories

Pour ce faire, appuyez sur l'icône Jeux située dans la barre de navigation en bas de l'écran. A partir de là, vous pouvez parcourir l'une des listes par défaut : Populaires, Meilleurs Gains, Meilleurs Notes, En vedette et Recommandés.

Note : Vous pouvez également utiliser les listes déroulantes situées en haut de l'écran pour restreindre les jeux à une catégorie spécifique ou pour choisir une autre liste, par exemple vos favoris, l’activité des amis, etc.

Utiliser la fonction de recherche

Vous pouvez également rechercher des jeux par titre ou par mots clé. Veuillez suivre les étapes ci-dessous :

  1. Allez à la section Jeux
  2. Appuyez sur l'icône en haut de l'écran.
  3. Entrez le titre du jeu ou un mot-clé dans la barre de recherche.
  4. Appuyez sur Retour ou Rechercher sur votre clavier. Cela peut ressembler à une flèche pointant vers la gauche ou une loupe.

Une fois que vous aurez trouvé un jeu auquel vous aimeriez jouer, appuyez sur son icône. Cela va charger sa page. Ici vous pouvez lire des informations sur le jeu, voir les classements, etc. Pour commencer le jeu, appuyez sur le bouton Jouer .
