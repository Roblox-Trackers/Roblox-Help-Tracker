Application Mobile : lire et répondre aux messages

Vos messages Roblox sont disponibles dans l'application mobile Roblox et sont accessibles depuis la barre de navigation située en bas de l'écran.

Pour lire des messages personnels

  1. Appuyez sur l'icône " ... " dans le coin inférieur droit de l'écran
  2. Sélectionnez Messages
  3. Faites défiler vos messages. Une fois en bas, la page chargera automatiquement plus de messages.
  4. Sélectionnez le message que vous souhaitez lire.

Répondre aux messages personnels

  1. Appuyez sur l'icône "..." dans le coin inférieur droit de l'écran
  2. Sélectionnez Messages
  3. Faites défiler vos messages. Une fois arrivé en bas, la page chargera automatiquement plus de messages.
  4. Sélectionnez le message auquel vous souhaitez répondre.
  5. Appuyez sur Répondre , entrez votre message, puis appuyez sur Envoyer .

Note: Les messages personnels peuvent être signalés . Appuyez sur Signaler ce message pour envoyer un rapport à nos modérateurs.
