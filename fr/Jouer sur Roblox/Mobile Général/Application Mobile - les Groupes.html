Application Mobile : les Groupes

L'Application Mobile Roblox permet aux joueurs de se tenir informés en ce qui concerne leurs groupes et de poster sur le Mur du groupe.

Accéder aux Groupes

Pour accéder aux groupes via l'Application mobile, suivez ces étapes :

  1. Appuyez sur le bouton More " ... " situé au bas de l'écran
  2. Ici cliquez sur Groupes
  3. Sélectionnez le groupe que vous souhaitez afficher

Note: Vous pouvez également rechercher des groupes en entrant un nom de groupe dans la barre de recherche située en haut de l'écran.

Interaction avec les Groupes

Tablette

Si vous utilisez une tablette, vous pouvez interagir avec des groupes comme vous le feriez sur un ordinateur de bureau ou un ordinateur portable. Pour plus d'assistance avec les groupes et les fonctions de groupe sur les tablettes et les ordinateurs, veuillez cliquer ici .

Téléphone

Si vous affichez la page du groupe sur un téléphone, vous pouvez afficher sa Sescription, ses Membres et son Mur, son Magasin, ses Affiliés et effectuer les opérations suivantes (en fonction des privilèges du groupe):

  • Publier sur le Mur - Cliquez sur le champs sur lequel il est écrit "Ecrivez quelque chose...", ajoutez votre message et cliquez sur "Publiez" .
  • Quitter votre Group - Pour ce faire, appuyez sur l'icone " ... "  dans le coin supérieur droit de l'écran. Lisez le message dans la fenêtre pop-up et appuyez sur Oui si vous êtes sûr de vouloir quitter le groupe.
  • Rendre principal - Pour faire ça, appuyez sur l'icône «...» dans le coin supérieur droit de l'écran. Appuyez sur le bouton "Rendre principal" dans le menu déroulant

Note: Utilisez le bouton Signaler un abus si jamais vous trouvez des violations de nos Règles de la Communauté parmi les messages sur le mur.
