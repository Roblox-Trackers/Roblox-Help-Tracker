Application Mobile : Affichage de votre inventaire

Suivez ces étapes pour afficher l'inventaire de votre compte sur votre téléphone portable :

  1. Appuyez sur le bouton Plus en bas de l'écran.
  2. Appuyez sur Inventaire
  3. Sur un portable , utilisez le menu déroulant pour sélectionner une catégorie de garde-robe (chapeaux, chemises, équipement, etc.) et la sous-catégorie.
  4. Sur une tablette , sélectionnez une catégorie de garde-robe dans la zone Catégorie et sous-catégorie sur la gauche de l'écran.
  5. Faites défiler vos options. En bas de chaque page vous pouvez appuyer sur les boutons fléchés vers la droite ou vers la gauche pour naviguer entre les différentes pages ou, dans certains cas, d'autres vont se charger automatiquement.
  6. Sélectionnez l'élément que vous souhaitez afficher
  7. Une fois votre sélection effectuée, la page Élément affiche plus d'informations, telles que le nom, la description et le genre (le cas échéant).

Des articles supplémentaires peuvent être achetés via le Catalogue Roblox, qui est également disponible sur l'Application Mobile Roblox. Cliquez ici pour plus d'informations.

Note: Comme les Chapeaux Roblox sont conçus autour de la forme de la Tête de base, rappelez-vous que tous les Chapeaux et Têtes ne s'emboîtent pas parfaitement, mais les joueurs peuvent toujours les utiliser pour créer des looks originaux et uniques.
