Applications mobiles : Comment chatter dans les expériences

Alors que vous pouvez discuter avec vos amis sur Roblox à partir de votre ordinateur, vous avez également la possibilité de discuter dans l'expérience sur l'Application Mobile Roblox.

Comment chatter

Si vous souhaitez discuter en expérience, veuillez suivre les étapes suivantes :

  1. Sélectionnez une expérience. Pour plus d'informations sur comment rechercher des expériences, veuillez cliquer ici .
  2. Pour envoyer vos propres messages, appuyez sur l'icône en haut à gauche de l'écran.
  3. Utilisez le clavier qui s'affiche en pop-up pour entrer votre message, puis appuyez sur sa touche Retour . Cela peut ressembler à une flèche dirigée vers la gauche à l'extrême droite du clavier.

Note: Vous pouvez également ouvrir et fermer la boîte de dialogue quand vous souhaitez en appuyant sur l'icône dans le coin supérieur gauche.

Comment désactiver le chat en cours dans l'expérience

Pour désactiver la possibilité de discuter dans l'expérience, veuillez suivre les étapes suivantes :

  1. Connectez-vous au compte
  2. Cliquez sur l'icône Plus "..." située dans le coin inférieur droit de l'écran
  3. Sélectionnez Paramètres
  4. Dans le menu déroulant, sélectionnez Confidentialité
  5. Sélectionnez Personne dans "Qui peut discuter avec moi?",

Note: Veuillez noter que ce chat sera soumis au mode / filtre de confidentialité du compte. Pour plus d'informations sur ces filtres, veuillez cliquer ici .
