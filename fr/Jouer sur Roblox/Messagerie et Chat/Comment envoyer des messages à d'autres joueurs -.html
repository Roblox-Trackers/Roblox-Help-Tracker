Comment envoyer des messages à d'autres joueurs ?

Vous pouvez contacter un créateur de jeu pour signaler un problème avec des objets en jeu ou utiliser la fonction de messagerie.

Envoyer un message

Pour envoyer un message, allez dans la section Amis et cliquez sur le nom de votre ami. Cela vous amènera à leur profil. Appuyez sur le bouton Message dans la case en haut de la page qui contient le nom d'utilisateur et les informations sur les amis et les abonnés.

Si vous n'êtes pas déjà ami avec la personne avec laquelle vous aimeriez communiquer, vous pouvez toujours leur envoyer des messages aussi longtemps que vous l'avez tous les deux autorisé dans vos paramètres de confidentialité. Pour les trouver, tapez leur nom d'utilisateur dans la barre de recherche en haut de l'écran et choisissez l'option Rechercher (nom d'utilisateur) dans Personnes dans l'ensemble des choix qui apparaîtront. Cliquez sur leur nom dans la liste des résultats de recherche et il vous mènera sur leur profil où vous pouvez cliquer sur le bouton Message .

Le bouton Message ne me laisse pas cliquer dessus

Si la personne à laquelle vous essayez d'envoyer un message a un bouton Message qui n'est pas cliquable, il se peut que ses paramètres de confidentialité ne soient pas configurés pour autoriser les messages privés à ce moment. De plus, vous devrez vous assurer que vos paramètres de confidentialité leur permettent de répondre à votre message. Par exemple, si vous autorisez seulement vos amis à vous envoyer un message et que vous essayez d'envoyer un message à quelqu'un qui n'est pas dans votre liste d'amis, le bouton de message ne pourra pas être cliqué dessus. Pour plus d'informations sur les autorisations de confidentialité, veuillez cliquer ici.
