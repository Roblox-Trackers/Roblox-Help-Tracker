Comment envoyer des messages à d'autres joueurs

Vous pouvez contacter un créateur d'expérience pour signaler un problème avec des objets ou des passes d'expérience ou avoir une conversation privée avec votre ami en utilisant la fonction de messagerie.

Envoi d'un message

Pour envoyer un message à votre ami :

  • Accédez à votre section Amis
  • Cliquez sur le nom de votre ami. Cela vous amènera à leur profil.
  • Appuyez sur le bouton Message dans la case en haut de la page qui contient le nom d'utilisateur et les informations sur les amis et les abonnés.

Pour envoyer un message à quelqu'un avec qui vous n'êtes pas ami :

  • Tapez son nom d'utilisateur dans la barre de recherche en haut de l'écran
  • Choisissez l'option Rechercher (nom d'utilisateur) dans Personnes dans l'ensemble de choix qui apparaîtra
  • Cliquez sur leur nom dans la liste des résultats de recherche
  • Cela vous mènera à leur profil où vous pourrez cliquer sur leur bouton Message .

Remarque : Les paramètres de confidentialité de votre compte et de ceux de l'autre personne doivent autoriser la messagerie en dehors des seuls amis.

Le bouton de message ne me laisse pas cliquer dessus

Si la personne à laquelle vous essayez d'envoyer un message a un bouton Message qui n'est pas cliquable, il se peut que ses paramètres de confidentialité ne soient pas configurés pour autoriser les messages privés à ce moment. De plus, vous devrez vous assurer que vos paramètres de confidentialité leur permettront de vous répondre. Par exemple, si vous autorisez uniquement vos amis à vous envoyer des messages et que vous essayez d'envoyer un message à quelqu'un qui n'est pas votre ami, vous ne pourrez pas cliquer sur le bouton de message. Pour plus d'informations sur les autorisations de confidentialité, veuillez lire Fonctionnalités de sécurité : chat, confidentialité et filtrage .
