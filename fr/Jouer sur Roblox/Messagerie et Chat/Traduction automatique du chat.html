Traduction automatique du chat

Dans certaines expériences, Roblox traduira automatiquement les messages de discussion des joueurs dans d'autres langues vers votre langue. Lorsque vous rejoignez une expérience de traduction de chat, vous verrez un message système indiquant que le chat est en cours de traduction.

Si vous souhaitez savoir quels messages spécifiques sont traduits, vous pouvez activer « Option pour voir le message non traduit » dans le menu Paramètres. Si ce paramètre est activé, une icône apparaîtra devant les messages traduits. Vous pouvez également cliquer/appuyer sur le bouton pour basculer entre le message original et la traduction.

Si vous souhaitez désactiver la traduction du chat, vous pouvez le faire en désactivant « Traduction automatique du chat » dans le menu Paramètres. Lorsque la traduction du chat est désactivée, tous les messages reviendront à leur forme d'origine et vous ne verrez plus les traductions.
