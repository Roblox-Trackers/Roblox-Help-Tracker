Comment supprimer des messages dans ma boîte de réception ?

Pour l’instant, les messages envoyés par le système de Roblox peuvent être supprimés, mais pas les messages réguliers d'autres joueurs. Si vous souhaitez supprimer les messages pour avoir une boîte de réception propre, nous avons l’option Archive.

Comment puis-je archiver des messages ?

L'action Archiver déplace les messages de votre boîte de réception vers l’onglet Archivés. Pour archiver un message, il suffit de cliquer sur le bouton Archiver situé sous les onglets de navigation.

Remarque : Vous pouvez redéplacer les messages dans votre boîte de réception plus tard en les trouvant dans l'onglet Archivés, en les affichant et en cliquant sur le bouton Déplacer vers la boîte de réception.
