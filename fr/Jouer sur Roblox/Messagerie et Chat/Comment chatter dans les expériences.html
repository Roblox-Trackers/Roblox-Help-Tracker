Comment chatter dans les expériences

L’une des meilleures façons de communiquer avec vos amis joueurs est d’utiliser le chat qui est disponible dans l'expérience.

Pour commencer à discuter, appuyez sur la touche “ / ” de votre clavier. Cela ouvrira la fenêtre de discussion, si elle est activée par le développeur, et une barre de discussion où vous pourrez taper ce que vous voulez dire. Une fois que vous avez écrit votre texte dans cette barre, appuyez sur Entrée sur votre clavier pour l'envoyer.

Vous pouvez également ouvrir là fenêtre/barre de discussion en cliquant sur l'icône de discussion située dans le coin supérieur gauche de l'écran :

Remarque : Le Chat sera limité à ce qui est autorisé par le mode/filtre de confidentialité du compte. Pour plus d'informations à ce sujet, s'il vous plaît cliquez ici .

Comment désactiver le chat dans une expérience

Pour désactiver la possibilité de discuter dans une expérience, procédez comme suit :

  1. Connectez-vous au compte
  2. Accédez sur votre page Paramètres du compte en cliquant sur l'icône en forme d’engrenage située dans le coin supérieur droit de la page suivi de Paramètres. Vous pouvez aussi cliquer ici
  3. Cliquez sur l'onglet Confidentialité et sous "Qui peut discuter avec moi ?", Sélectionnez Personne
