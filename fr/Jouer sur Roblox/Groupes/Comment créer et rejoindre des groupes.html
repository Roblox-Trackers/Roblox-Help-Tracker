Comment créer et rejoindre des groupes

Les groupes sont un moyen pour les joueurs et leurs amis de former leur propre mini-communauté avec d'autres joueurs partageant les mêmes idées. Les groupes peuvent affronter d'autres groupes, aider à la construction, engager des discussions, etc.

Conditions

Le nombre de groupes qu'un joueur peut rejoindre est limité à 100 groupes. Les joueurs peuvent également créer jusqu'à 100 groupes. La création de groupe nécessite un paiement unique de 100 Robux, par groupe.

Comment créer un groupe

Si les conditions sont remplies, la création d'un groupe est assez simple.

  1. Connectez-vous à votre compte
  2. Cliquez sur Groupes , situé dans le panneau de navigation situé à gauche
  3. Cliquez sur le bouton Créer , situé sur le côté gauche de la page

Une fois sur cette page, remplissez les sections Nom, Description et Emblème. Chacune doit refléter l'objet du groupe. Une fois cela fait, située dans la zone Paramètres sur le côté gauche de la page, choisissez la section accessibilité  de groupe. Pour un contrôle optimal, sélectionnez l'option Approbation manuelle.

Remarque : Les noms de groupes ne peuvent pas dépasser 50 caractères.

Une fois que tout a été défini comme vous le souhaitez, cliquez sur le bouton ( ) Acheter pour finaliser la création du groupe.

Comment trouver et rejoindre un groupe

Pour localiser un groupe :

  1. Cliquez sur la barre de recherche située en haut de la page
  2. Entrez un mot-clé (par exemple "Ninjas" si vous recherchez un groupe sur les ninjas)
  3. Choisissez de rechercher dans les groupes

À partir de là, la page de recherche de groupe comprend des informations telles qu'une brève description du groupe, ce qui est utile pour identifier les groupes d'intérêt.

Une fois qu'un groupe a été sélectionné, accédez à la page de ce groupe et cliquez sur le bouton Rejoindre le groupe . Si l'adhésion au groupe nécessite une approbation manuelle, attendez que la demande d'adhésion soit approuvée. En fonction du groupe, le propriétaire peut avoir des exigences spécifiques telles que l'envoi d'un message personnel avant qu'une demande d'adhésion au groupe ne soit approuvée.

Questions fréquemment posées

Question : « Puis-je supprimer mon groupe ?

Réponse : Actuellement, il n'y a aucun moyen de supprimer un groupe. Assurez-vous que le nom est correct avant de créer le groupe, car il n'existe actuellement aucune option pour modifier le titre après sa création.
