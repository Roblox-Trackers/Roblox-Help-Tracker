Application Mobile : les Groupes

L'Application Mobile Roblox permet aux joueurs de se tenir informés en ce qui concerne leurs groupes et de poster sur le Mur du groupe.

Accéder aux Groupes

Pour accéder aux groupes via l'Application mobile, suivez ces étapes :

  1. Appuyez sur le bouton More " ... " situé au bas de l'écran
  2. Ici cliquez sur Groupes
  3. Sélectionnez le groupe que vous souhaitez afficher

Note: Vous pouvez également rechercher des groupes en entrant un nom de groupe dans la barre de recherche située en haut de l'écran.

Interagir avec des groupes sur le téléphone

La page principale du groupe

C'est ce que vous voyez dès que vous sélectionnez votre groupe. Les options suivantes sont disponibles :

  • Appel : Les mises à jour importantes du groupe peuvent être publiées ici.
  • Description : Cette section montre en quoi consiste le groupe.
  • Expériences : Si vous ou vos membres avez créé des endroits/expériences pour le groupe, ils apparaîtront ici. Pour plus d'informations, cliquez ici .
  • Membres : Cet onglet permet l’affichage des membres par Rang.
  • Liens sociaux : cette section affiche vos liens sociaux ajoutés. Pour plus d'informations, cliquez ici.
  • Mur : Cette section est comme un fil de discussion continu pour le groupe. Les discussions du groupe peuvent être affichées ici. Les messages peuvent être supprimés par les membres du groupe ayant cette autorisation. Les membres peuvent également être exilés du groupe depuis la page d'administration du groupe.  S’il y a des commentaires inappropriés (voir les Règles de la Communauté Roblox ), l'option de signaler un comportement abusif est située à droite.

Commandes du groupe

Ces contrôles sont situés dans le coin supérieur droit de la page du groupe et offrent de nombreuses options pour gérer votre groupe.

  • Configuration du groupe : cette page vous permet d'apporter de nombreux ajustements aux informations et aux fonctionnalités de base de votre groupe. Cette page est détaillée ci-dessous.
  • Mettre en priorité : Ce bouton vous permet de faire d'un groupe votre groupe principal. Pour plus d'informations sur les groupes principaux, cliquez ici .
  • Quitter le groupe : Ce bouton vous permettra de quitter un groupe dont vous ne souhaitez plus faire partie.
    • Remarque : Si vous quittez un groupe dont vous êtes propriétaire, vous ne pourrez pas en reprendre la propriété en rejoignant le groupe ultérieurement. Si vous ne choisissez pas de transférer la propriété à un autre joueur (voir Paramètres dans la section Page d'administration de groupe ci-dessous) avant de quitter un groupe que vous possédez, le groupe n'appartiendra à personne.
  • Signaler un abus : pour signaler le nom ou la description , l’appel et le rôle , sélectionnez le bouton Signaler un abus , puis remplissez-le et sélectionnez le bouton Signaler.

La page d'administration du groupe

Cette page vous donnera les outils dont vous avez besoin pour gérer efficacement vos membres et le groupe lui-même. Pour y accéder, cliquez sur le bouton Configuration du groupe dans la boîte Contrôles à droite de la page du groupe.

  • Informations : C'est ici que vous pouvez modifier le nom, le propriétaire, l'image et la description de votre groupe.
  • Paramètres : Vous pouvez définir ici les conditions requises pour rejoindre le groupe. Cela inclut l'exigence d'une approbation manuelle. Vous trouverez également les options pour autoriser la déclaration d'ennemis et définir la visibilité des fonds et des expériences du groupe. Cette section permettra également aux propriétaires de transférer la propriété à un autre membre.
    • Veuillez noter que ce processus n'est pas réversible.
  • Liens sociaux : Cette section vous permet d'ajouter des liens sociaux dans les descriptions de vos groupes. Pour plus d'informations, cliquez ici .
  • Revenus : La section vous permet de suivre les Robux que votre groupe a gagnés au cours de la dernière journée, semaine, mois ou année. De plus, l'option Gains a été ajoutée dans la section Revenus pour donner les fonds du groupe à des membres spécifiques via un Gain unique ainsi que pour mettre en place un Gain récurrent .
  • Membres : Ici, vous pouvez rechercher ou parcourir vos membres actuels.Vous pouvez également choisir de filtrer par rang. De plus, c'est ici que vous pourrez modifier le classement des membres et accepter ou refuser les demandes de rejoindre le groupe en attente. Enfin, vous pouvez supprimer un membre du groupe en cliquant sur les trois points à côté de son icône et en sélectionnant "Bannir l'utilisateur".
  • Rôles : C'est ici que vous pouvez ajouter de nouveaux rangs ainsi que modifier le nom et les autorisations de ceux qui existent déjà. Pour plus d'informations, cliquez ici .

Remarque : les utilisateurs peuvent se connecter à un ordinateur / portable pour consulter les Journaux d'audit de groupe et basculer vers la section Affiliés de son groupe.

Utilisez le bouton Signaler un abus s'il y a des publications sur le mur qui enfreignent les Normes de la Communauté Roblox .
