Comment créer des alliés et des ennemis pour votre groupe

Maintenant que vous avez créé votre groupe, vous voudrez peut-être devenir soit alliés, soit ennemis, avec d’autres groupes. Si vous n’êtes pas le propriétaire du groupe, les autorisations du groupe sont requises pour gérer les relations avec d’autres groupes. Pour plus d'informations sur les autorisations, consultez Rôles/rangs de groupe et autorisations .

Devenir des alliés avec un autre groupe

Pour demander à un autre groupe de devenir un allié de votre groupe, veuillez suivre les étapes suivantes :

  1. Cliquez sur Groupes dans la barre de navigation, à gauche de l’écran
  2. Trouvez le groupe en question et cliquez dessus
  3. Cliquez sur le bouton Configurer le groupe à droite de la page
  4. Cliquez sur l'onglet Affiliés à gauche
  5. Sélectionnez les alliés
  6. Cliquez sur le bouton Envoyer une demande d'allié disponible dans le coin droit de la page
  7. Tapez le nom du groupe auquel vous souhaitez envoyer une demande d'allié
  8. Appuyez sur le bouton Demander.

Remarque (Pour accepter/refuser toute demande d'allié en attente de n'importe quel groupe, veuillez vous rendre dans la section demande de la page et cliquer sur accepter/refuser, vous pouvez également accepter ou refuser toutes les demandes en une seule fois en cliquant sur Tout refuser/Accepter tous les boutons disponibles sur la droite)

Autoriser et déclarer des ennemis

Pour autoriser ou déclarer des ennemis à votre groupe, procédez comme suit :

  1. Cliquez sur Groupes dans la barre de navigation, à gauche de l’écran
  2. Trouvez le groupe en question et cliquez dessus
  3. Cliquez sur le bouton Configurer le groupe à droite de la page
  4. Sélectionnez l'onglet Paramètres à gauche
  5. Activez le bouton Autoriser les déclarations d'ennemis
  6. Pour interdire les ennemis, activez le bouton Autoriser les déclarations d'ennemis
  7. Depuis la page Configurer le groupe, cliquez sur l'onglet Affiliés à gauche
  8. Sélectionnez les ennemis
  9. Cliquez sur le bouton Déclarer l'ennemi disponible dans le coin droit de la page
  10. Tapez le nom du groupe que vous souhaitez déclarer comme ennemi
  11. Appuyez sur le bouton Envoyer
