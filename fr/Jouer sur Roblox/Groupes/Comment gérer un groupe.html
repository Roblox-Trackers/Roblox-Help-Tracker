Comment gérer un groupe

Après avoir créé un groupe, l'étape suivante consiste à le gérer. Pour ce faire, cliquez sur le lien Groupes , situé dans le menu à gauche de la page. A partir de là, vous pouvez voir l'activité du groupe et le panneau de contrôle du groupe.

La page principale du groupe

C'est ce que vous voyez dès que vous sélectionnez votre groupe. Les options suivantes sont disponibles:

  • Shout : Des mises à jour importantes pour les groupes peuvent être postées ici.
  • Description : Cette section montre en quoi consiste le groupe.
  • Expériences : si vous ou vos membres avez créé des lieux/expériences pour le groupe, ils apparaîtront ici. Pour plus d'informations, voir Expériences de groupe.
  • Membres : Cet onglet permet l’affichage des membres par Rang.
  • Liens sociaux : cette section affiche vos liens sociaux ajoutés. Pour plus d'informations, consultez Liens sociaux sur les descriptions de groupe .
  • Mur : Cette section est comme un fil de discussion continu pour le groupe. Les discussions de groupe peuvent être publiées ici. Les publications peuvent être supprimées par les membres du groupe qui disposent de cette autorisation. Les membres peuvent également être exilés du groupe depuis la page d'administration du groupe. Si des commentaires inappropriés sont faits (voir les normes de la communauté Roblox ), l'option de signaler un abus se trouve à droite.

Contrôles du groupe

Ces commandes sont situées en haut à droite de la page du groupe et offrent de nombreuses options pour gérer votre groupe.

  • Configurer le groupe : cette page vous permet d'apporter de nombreux réglages aux informations et fonctionnalités de base de votre groupe. Cette page est traitée plus en détail ci-dessous.
  • Faire d'un groupe principal : ce bouton vous permet de faire d'un groupe votre groupe principal. Pour plus d'informations sur les groupes principaux, consultez Groupe principal.
  • Quitter le groupe : Ce bouton vous permettra de quitter un groupe dont vous ne souhaitez plus faire partie.
    • Remarque : Si vous quittez un groupe dont vous êtes propriétaire, vous ne pourrez pas en reprendre la propriété en rejoignant le groupe ultérieurement. Si vous ne choisissez pas de transférer la propriété à un autre joueur (voir Paramètres dans la section Page d'administration de groupe ci-dessous) avant de quitter un groupe que vous possédez, le groupe n'appartiendra à personne.
  • Faire la Pub d'un groupe : Cette page vous permettra de créer une annonce pour votre groupe. Pour plus d'informations, consultez Comment publier un groupe.
  • Signaler un abus : pour signaler le nom ou la description du groupe, le cri et le rôle , sélectionnez le bouton Signaler un abus , puis remplissez-le et sélectionnez le bouton Signaler .

La page d'administration du groupe

Cette page vous donnera les outils dont vous avez besoin pour gérer efficacement vos membres et le groupe lui-même. Pour y accéder, cliquez sur le bouton Configurer le groupe dans la zone Contrôles à droite de la page du groupe. Une fois là-bas, vous verrez les options suivantes :

  • Infos sur le groupe : C'est ici que vous pouvez modifier le nom, le propriétaire, l'image et la description de votre groupe.
  • Paramètres : Vous pouvez définir ici les conditions requises pour rejoindre le groupe. Cela inclut l'exigence d'une approbation manuelle. Vous trouverez également les options pour autoriser la déclaration d'ennemis et définir la visibilité des fonds et des jeux du groupe.
    • Cette section permettra également aux propriétaires de transférer la propriété à un autre membre.
      • Veuillez noter que ce processus n'est pas réversible.
  • Liens sociaux : cette section affiche vos liens sociaux ajoutés. Pour plus d'informations, consultez Liens sociaux sur les descriptions de groupe.
  • Revenus : Cette section vous permet de suivre les Robux que votre groupe a gagnés au cours des derniers jours, semaines, mois ou années. De plus, l'option Paiements a été ajoutée dans la section Revenus pour donner les fonds du groupe à des membres spécifiques via un paiement unique ainsi que pour configurer un paiement récurrent .
  • Membres : Ici, vous pouvez rechercher ou parcourir vos membres actuels. Vous pouvez également choisir de filtrer par rang. De plus, c'est ici que vous pourrez modifier le classement des membres et accepter ou refuser toute demande d'adhésion en attente. Enfin, vous pouvez supprimer un membre du groupe en cliquant sur l'engrenage à côté de son icône et en sélectionnant "Kick User".
  • Rôles : c 'est ici que vous pouvez ajouter de nouveaux rangs ainsi que modifier le nom et les autorisations de ceux qui existent déjà. Pour plus d'informations, consultez Rôles/rangs de groupe et autorisations.
  • Affiliés : Cette section vous permettra de déclarer d'autres groupes comme ennemis ou alliés officiels de votre groupe. Pour plus d'informations, voir Comment créer des alliés et des ennemis pour votre groupe.
  • Journal d'audit : cela vous permettra de voir tout ce que vos membres ont fait en ce qui concerne le groupe. Pour plus d'informations, voir Affichage de l'activité de votre groupe (Journal d'audit de groupe)
