Comment gérer un groupe

Après avoir créé un groupe, l'étape suivante consiste à le gérer. Pour ce faire, cliquez sur le lien Groupes, situé dans le menu à gauche de la page. A partir de là, vous pouvez voir l'activité du groupe et le panneau de contrôle du groupe.

La page principale du groupe

C'est ce que vous voyez dès que vous sélectionnez votre groupe. Les options suivantes sont disponibles :

  • Shout : Des mises à jour importantes pour les groupes peuvent être postées ici. Les mises à jour les plus récentes s’afficheront dans une bulle de mots jaunes sous la description du groupe.
  • Membres : Cet onglet permet l’affichage des membres par rang.
  • Alliés et ennemis : Ces onglets ne seront pas visibles si votre groupe n'a ni alliés ni ennemis. Mais si c'est le cas, ces onglets afficheront d'autres groupes qui sont vos alliés ou vos ennemis.
  • Magasin : Cet onglet vous permettra de créer des vêtements pour vendre et voir les vêtements déjà en vente. Pour plus d'informations, cliquez ici.
  • Jeux : Si vous ou vos membres avez créé des endroits/jeux pour le groupe, ils apparaîtront ici. Pour plus d'informations, cliquez ici.
  • Mur : Cette section est comme un forum permanent pour le groupe. Les discussions du groupe peuvent être affichées ici. Les messages peuvent être supprimés par les membres du groupe ayant cette autorisation. Les membres peuvent également être exilés du groupe depuis la page d'administration du groupe. S’il y a des commentaires inappropriés (voir les Règles de la Communauté Roblox ), l'option de signaler un comportement abusif est située à droite.

Commandes du groupe

Ces contrôles sont situés dans le coin supérieur droit de la page du groupe et offrent de nombreuses options pour gérer votre groupe.

  • Administrateur du groupe : cette page vous permet d'apporter de nombreux ajustements aux informations et aux fonctionnalités de base de votre groupe. Cette page est détaillée ci-dessous.
  • Publicité du groupe : cette page vous permettra de créer une annonce pour votre groupe. Pour plus d'informations, cliquez ici.
  • Groupe principal : Ce bouton vous permet d’établir votre groupe principal. Pour plus d'informations sur les groupes principaux, cliquez ici.
  • Quitter le groupe : ce bouton vous permettra de quitter un groupe dont vous ne voulez plus faire partie.
    • Veuillez noter que si vous quittez un groupe que vous possédez, vous ne serez pas en mesure de retrouver la propriété en rejoignant le groupe plus tard. Si vous ne choisissez pas de transférer la propriété à un autre joueur (voir Paramètres dans la section Page d'administration de groupe ci-dessous) avant de quitter un groupe que vous possédez, la propriété sera transférée à un autre membre du groupe avec adhésion. S'il n'y a pas d'autres membres avec adhésion, le groupe appartiendra à personne.
  • Journal d'audit : Cela vous permettra de voir tout ce que vos membres ont fait en ce qui concerne le groupe. Pour plus d'informations, cliquez ici .

La page d'administration du groupe

Cette page vous donnera les outils dont vous avez besoin pour gérer efficacement les membres et le groupe lui-même. Pour y accéder, cliquez sur le bouton Administrateur de groupe dans la zone Commandes à droite de la page du groupe. Une fois là, vous verrez les options suivantes :

  • Membres : Ici vous pouvez rechercher vos membres actuels. Vous pouvez également choisir de filtrer la liste par rang. En outre, vous pourrez modifier l'hiérarchie des membres et accepter ou refuser les demandes de rejoindre le groupe en attente. Enfin, vous pouvez supprimer un membre du groupe en cliquant sur la roue située à côté de son icône et en sélectionnant « Kicker l’utilisateur ».
  • Infos sur le groupe : C'est ici que vous pouvez modifier l'image et la description de votre groupe.
  • Paramètres : vous pouvez définir ici les conditions requises pour rejoindre le groupe. Cela inclut l'approbation manuelle et la restriction de l'accès aux seuls membres du Builders Club. Vous trouverez également les options pour permettre la déclaration des ennemis et définir la visibilité des fonds du groupe et des jeux. Cette section permettra également aux propriétaires de transférer la propriété à un autre membre. Veuillez noter que lorsque vous transférez la propriété, le joueur auquel vous la transférez doit également être membre du Builders Club pour pouvoir être propriétaire du groupe.
    • Veuillez noter que ce processus n'est pas réversible.
  • Relations : Cette section vous permettra de déclarer d'autres groupes comme ennemis ou alliés officiels de votre groupe. Pour plus d'informations, cliquez ici .
  • Rôles : Ici, vous pouvez ajouter de nouveaux rangs et changer le nom et les permissions de ceux déjà existants. Pour plus d'informations, cliquez ici.
  • Revenus : Cette section vous permet de suivre les Robux que votre groupe a gagnés au cours des derniers jours, semaines, mois ou années.
  • Paiements : Cette section vous permettra de donner une partie des fonds de votre groupe à des membres spécifiques par le biais d'un paiement unique ainsi que de mettre en place un paiement récurrent.
