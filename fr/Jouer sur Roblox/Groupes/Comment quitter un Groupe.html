Comment quitter un Groupe

Parfois, il est possible de ne plus vous sentir intéressés de faire partie d'un certain groupe. Si c'est le cas, vous pouvez le quitter à n'importe quel moment.

Comment quitter un Groupe

  • Allez dans votre section Groupes
  • Cliquez sur le groupe que vous voudriez quitter
  • Cliquez sur le bouton Quitter le groupe sur le côté droit de la page comme indiqué ci- dessous

Gestion du Groupe

Si le propriétaire quitte le groupe, alors tout membre du groupe avec adhésion peut revendiquer la gestion. S'il n'y en a plus, alors le groupe va rester sans propriétaire.
