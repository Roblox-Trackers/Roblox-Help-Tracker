Comment quitter un Groupe

Parfois, il est possible de ne plus vous sentir intéressés de faire partie d'un certain groupe. Si tel est le cas, vous pouvez le quitter à tout moment.

Comment quitter un Groupe

  • Accédez à votre section Groupes
  • Cliquez sur le groupe que vous voudriez quitter
  • Cliquez sur le bouton Quitter le groupe sur le côté droit de la page :

Propriété du groupe

Si le propriétaire quitte le groupe, tout membre du groupe possédant une adhésion peut en revendiquer la propriété. S'il n'y a aucun autre membre du groupe avec une adhésion, alors le groupe restera sans propriétaire.
