Changer le nom d'un groupe

Exigences

  • Changer le nom d'un groupe coûtera 100 Robux, payés avec le Robux du propriétaire du groupe
  • Le propriétaire du groupe doit avoir une adresse e-mail vérifiée sur son compte Roblox
  • La propriété du groupe ne doit pas avoir été modifiée au cours des 90 derniers jours
  • Le nom du groupe devrait être resté inchangé au cours des 90 derniers jours
  • Le nom souhaité pour le groupe ne soit pas déjà utilisé.

Comment changer un nom de groupe ?

  1. Accédez au groupe dont vous souhaitez modifier le nom.
  2. Cliquez sur le menu à trois points, à droite du nom du groupe, puis cliquez sur Configurer groupe .
  3. Accédez à la section Informations .
  4. Cliquez sur le bouton Modifier , à droite du nom du groupe .
    • Saisissez le nouveau nom du groupe.
    • Acceptez nos termes et conditions en cliquant sur la case à cocher.
    • Si vous ne souhaitez pas modifier le nom, cliquez sur Annuler ou sur le X en haut à droite.
  5. Cliquez sur l ' icône pour changer le nom du groupe.

Ancien nom du groupe

Via un navigateur

Affichez-le en survolant le texte des Noms de groupe précédents sur la page des détails du groupe ou sous le nom actuel dans les résultats de la recherche.

Via l'application

Affichez-le en appuyant sur le texte des Noms de groupe précédents sur la page des détails du groupe ou sous le nom actuel dans les résultats de la recherche.

Questions fréquemment posées

Question : À quelle fréquence un propriétaire de groupe peut-il modifier ce nom de groupe ?

Réponse : Il y a une période de 90 jours requise entre les changements de nom de groupe.

Question : Mon nouveau nom de groupe a été modéré, puis-je le modifier avant 90 jours ?

Réponse : La restriction de 90 jours ne s'applique pas si le nouveau nom du groupe est modéré. Les utilisateurs peuvent le modifier en suivant les étapes ci-dessus.

Question : Comment les membres du groupe seront-ils informés du nouveau nom du groupe ?

Réponse : Chaque membre du groupe recevra un message l'informant du changement de nom.
