Application PlayStation : comment changer de langue

Le texte officiel de Roblox est disponible en différentes langues dans l'application PlayStation. Voici la liste des langues disponibles à l'heure actuelle :

  • Portugais (Brésil)
  • Chinois (traditionnel et simplifié)
  • Anglais
  • Français
  • Allemand
  • Indonésien
  • Japonais
  • Coréen
  • Italien
  • Espagnol
  • Thaï

Remarque : il est possible que le contenu des expériences soit en anglais quelle que soit la langue choisie.

Comment changer de langue

À l'heure actuelle, il n'est pas possible de changer de langue dans Roblox sur PlayStation. Si vous souhaitez changer de langue, vous devrez le faire depuis votre compte Roblox. Cliquez ici pour en savoir plus.
