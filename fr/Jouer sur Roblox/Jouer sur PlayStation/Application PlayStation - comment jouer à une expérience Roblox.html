Application PlayStation : comment jouer à une expérience Roblox

Pour jouer aux expériences, vous aurez besoin d'une console PlayStation et d'une connexion Internet. Vous trouverez Roblox sur le PlayStation Store. L'application est téléchargeable gratuitement !

Il y a toutes sortes d'expériences auxquelles vous pouvez jouer sur Roblox. Après avoir lancé le jeu, vous les trouverez à l'écran d'accueil et sur l'écran des expériences. Ensuite, vous pourrez retrouver les expériences auxquelles vous aurez joué récemment sur l'écran d'accueil.

Chaque expérience Roblox a ses propres règles, commandes et mécaniques. Les commandes globales sont celles qui sont indiquées ci-dessous.

Remarque : dans les expériences, vous pouvez accéder à tout moment à ce diagramme des commandes en appuyant sur la touche Options de la manette PlayStation afin d'ouvrir le menu Roblox. Appuyez sur la touche R1 pour voir l'onglet d'aide. Appuyez sur la touche L1 pour voir les autres onglets.

Étant donné que l'application vient d'être lancée sur PlayStation, il est possible que des commandes pour d'autres plateformes ou consoles s'affichent dans les expériences, car tous les développeurs n'ont pas encore eu le temps de les mettre à jour.
