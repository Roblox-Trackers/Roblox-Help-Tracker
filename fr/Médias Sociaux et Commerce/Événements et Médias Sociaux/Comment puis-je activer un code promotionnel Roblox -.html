Comment puis-je activer un code promotionnel Roblox ?

Comment puis-je obtenir un code promotionnel Roblox ?

Vous pouvez obtenir un code promo Roblox lors de l'un de nos nombreux événements ou cadeaux.

Comment puis-je activer mon code promotionnel pour avoir l'article virtuel ?

  1. Assurez-vous que vous êtes connecté à votre compte Roblox sur lequel vous souhaitez utiliser le code.
  2. Allez sur Échanger des promotions Roblox .
  3. Entrez le code dans la case ci-dessous Saisir votre code
  4. Cliquez sur Activer .
    • Si vous l'avez activé avec succès, vous verrez ceci:
    • Si le code est inactif ou s'il ne s'agit pas d'un code promotionnel, vous verrez ceci:
  5. Une fois que vous avez activé le code promotionnel, découvrez votre nouvel article dans l'inventaire de votre compte. Assurez-vous de cocher la bonne catégorie d'inventaire.

Remarque: N'oubliez pas que les codes promotionnels peuvent expirer ou n'être actifs que pendant une courte période. Veillez donc à utiliser votre code immédiatement.
