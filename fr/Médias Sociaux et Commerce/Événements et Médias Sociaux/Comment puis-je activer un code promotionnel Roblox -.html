Comment puis-je activer un code promotionnel Roblox ?

Comment puis-je obtenir un code promotionnel Roblox ?

Vous pouvez obtenir un code promo Roblox lors de l'un de nos nombreux événements ou giveaways.

Comment puis-je activer mon code promotionnel pour avoir l'article virtuel ?

  1. Assurez-vous que vous êtes connecté à votre compte Roblox sur lequel vous souhaitez utiliser le code.
  2. Allez sur la page Activer le Code .
  3. Entrez votre code dans la case.
  4. Cliquez sur Activer .
  5. Un message de réussite apparaîtra une fois que vous aurez activé le Code avec succès.
  6. Vous obtiendrez du Crédit, du Robux ou un Objet Virtuel. Cela dépend du type de Code que vous avez activé.

Remarque: N'oubliez pas que les codes promotionnels peuvent expirer ou être actifs que pendant une courte période de temps. Veillez donc vous assurer d'utiliser votre code tout de suite.
