Comment Activer un Code de Jouet

Où puis-je trouver mon code pour l'objet virtuel ?

Le code se trouve au recto du paquet ou à l'intérieur de la carte de code. Découvrez la vidéo Jazwares sur les cartes de code.

Comment puis-je activer mon code pour l'objet virtuel ?

  1. Si votre code est couvert, grattez doucement le revêtement.
  2. Veuillez vous assurer que vous êtes connecté à votre compte Roblox sur lequel vous souhaitez activer le code.
  3. Allez sur la Page de Rachat du Code de Jouet .
  4. Entrez votre code dans la boîte Entrer Code de Jouet .
  5. Cliquez sur le bouton Activer .
  6. Découvrez votre nouvel article génial dans l'inventaire de votre compte. Assurez-vous de vérifier la bonne catégorie d'inventaire.

Remarque: Un compte ne peut recevoir qu'une seule fois l'objet virtuel de chaque jouet. Si vous avez un code pour un objet virtuel que votre compte possède déjà, vous pouvez partager le code avec un autre utilisateur.
