Comment Activer des Codes de Jouets et des Objets Virtuels

Où puis-je trouver mon code pour l'objet virtuel ?

  • Pour les figurines Jazwares : Le code se trouve au recto de l'emballage sur un jeton ou sur une carte code, collée à l'intérieur de l'emballage plastique. Regardez la video Jazwares pour savoir où trouver le code de l'objet virtuel.
  • Pour Nerf blasters et Monopoly : le code se trouve dans le paquet sur l'insert du code carte.

  • Pour tous les autres codes d'objets virtuels (par exemple, Prime Gaming, Verizon, etc.), veuillez suivre les instructions d'activation trouvées ci-dessous.

Comment puis-je activer mon code pour l'objet virtuel ?

Remarque : Si vous disposez d'un code numérique, passez à l'étape 2.

  1. Si votre code est couvert, grattez doucement le revêtement.
  2. Veuillez vous assurer que vous êtes connecté à votre compte Roblox sur lequel vous souhaitez activer le code.
  3. Allez sur la Page de Rachat du Code de Jouet .
  4. Entrez votre code dans la case.
  5. Cliquez sur le bouton Activer .
  6. Découvrez votre nouvel article génial dans l'inventaire de votre compte. Assurez-vous de vérifier la bonne catégorie d'inventaire (par exemple. chapeau, sac à dos, tête, accessoires, etc.).

Remarque : Un code d'article virtuel ne peut être échangé qu'une seule fois sur le compte d'un individu. Si vous avez un code pour un article virtuel que votre compte possède déjà, vous pouvez partager le code avec un autre utilisateur. Ces objets ne peuvent être ni échangés ni vendus.
