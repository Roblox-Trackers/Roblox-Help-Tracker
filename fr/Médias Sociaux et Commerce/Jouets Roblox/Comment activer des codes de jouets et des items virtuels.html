Comment activer des codes de jouets et des items virtuels

Où puis-je trouver le code de mon item virtuel ?

  • Pour les figurines Jazwares : Le code se trouve au recto de l'emballage sur un jeton ou sur une carte code, collée à l'intérieur de l'emballage plastique. Regardez la vidéo Jazwares pour savoir où trouver le code de l'item virtuel.
  • Pour Nerf blasters et Monopoly : le code se trouve dans le paquet sur l'insert du code carte.
    • Exemples des inserts de code carte :
  • Pour tous les autres codes d'items virtuels (par exemple, Prime Gaming, Verizon, etc.), veuillez suivre les instructions d'activation trouvées ci-dessous.

Comment puis-je activer le code de l'item virtuel ?

Remarque : Si vous disposez d'un code numérique, passez à l'étape 2.

  1. Si votre code est couvert, grattez doucement le revêtement.
  2. Veuillez vous assurer que vous êtes connecté à votre compte Roblox sur lequel vous souhaitez activer le code.
  3. Accédez à la Page de Rachat du Code. .
  4. Entrez votre code dans la case.
  5. Cliquez sur le bouton Activer .
  6. Découvrez votre nouvel article génial dans l'inventaire de votre compte. Assurez-vous de vérifier la bonne catégorie d'inventaire (par exemple. chapeau, sac à dos, tête, accessoires, etc.).

Remarque : Un code d'item virtuel ne peut être échangé qu'une seule fois sur le compte d'un individu. Si vous avez un code pour un item virtuel que votre compte possède déjà, vous pouvez partager le code avec un autre utilisateur. Ces objets ne peuvent être ni échangés ni vendus.
