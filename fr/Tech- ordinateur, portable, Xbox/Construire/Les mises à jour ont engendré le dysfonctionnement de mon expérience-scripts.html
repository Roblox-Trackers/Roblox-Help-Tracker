Les mises à jour ont engendré le dysfonctionnement de mon expérience/scripts

Lorsque nous mettons à jour Roblox, nous essayons de nous assurer que les scripts créés par les joueurs continueront de fonctionner correctement. Cependant, ces scripts doivent parfois être réécrits ou ajustés pour qu'ils fonctionnent correctement. Examinez vos scripts pour voir d'où vient le problème. Vous pouvez également consulter le Livre de Scripting Roblox pour une aide supplémentaire.
