Problèmes généraux de la page d’internet

L'article suivant proposera quelques méthodes qui se sont révélées efficaces pour résoudre les problèmes suivants :

  • La page d’internet ne s'affiche pas correctement
  • Certaines pages ne se chargent pas du tout ou affichent une erreur lors du chargement
  • Les boutons ou les liens ne vous permettent pas de cliquer dessus
  • Vous essayez de vous connecter au site Web Roblox mais ne peut pas avoir le nom d'utilisateur et le mot de passe corrects
  • Autre: Si vous rencontrez un problème différent avec la page d’internet, les étapes ci-dessous peuvent également aider à le résoudre. De plus, certains articles non liés à une page d’internet peuvent être liés ici pour certaines étapes.

Quoi faire ?

Veuillez essayer chaque étape ci-dessous, une à la fois, dans l'ordre. Si une solution ne fonctionne pas, passez à la suivante!

Effacer vos fichiers Internet temporaires et vos cookies

Les liens suivants vous aideront à supprimer les fichiers Internet temporaires (également appelés "cache") et les cookies pour les principaux navigateurs Web de PC. Assurez-vous de le faire pour Internet Explorer et pour tout autre navigateur à partir duquel vous souhaitez lancer Roblox.

      • Internet Explorer - Fichiers Internet temporaires (sur Microsoft.com)
      • Internet Explorer - Cookies (De Microsoft.com)
      • Mozilla Firefox - Fichiers Internet temporaires (à partir de Mozilla.com)
      • Mozilla Firefox - Cookies (De Mozilla.com)
      • Google Chrome - Les deux (Google.com)
      • Microsoft Edge - Both (De Microsoft.com)
      • Safari - Cookies (De Apple.com)
      • Autres navigateurs: Si vous utilisez un navigateur différent ou une ancienne version d'un navigateur, veuillez rechercher des informations sur la manière de supprimer ses données de cache / navigateur / fichiers Internet temporaires pour votre navigateur / version spécifique.

Une fois que vous avez complété ce qui précède pour tous les navigateurs Web pertinents, veuillez redémarrer votre ordinateur et voir si votre problème a été résolu.


Si vous souhaitez supprimer vos fichiers temporaires de Internet Explorer / Windows manuellement, vous pouvez le faire en accédant directement au dossier Fichiers Temporaires d’Internet de votre disque dur. L'emplacement et le nom du dossier dépendent de la version de Windows et de l'utilisation ou non de profils utilisateur.

Remarque: Vous devrez peut-être modifier les paramètres de l'Explorateur Windows pour afficher tous les types de fichiers (y compris les fichiers système protégés) afin d'accéder à ces dossiers.

Pour Windows 10


Si vous avez Windows 10, les fichiers Internet temporaires sont à l'emplacement suivant (notez que sur votre PC, ils peuvent se trouver sur un autre lecteur au lieu du lecteur C):

            • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Pour Windows 7 et 8


Si vous avez Windows 7 ou Windows 8, les fichiers Internet temporaires se trouvent à ces emplacements (notez qu'ils peuvent se trouver sur un autre lecteur au lieu du lecteur C) :

        • C: \ Users \ <nom d'utilisateur> \ AppData \ Local \ Microsoft \ Windows \ Fichiers Internet temporaires
        • C: \ Users \ <nom d'utilisateur> \ AppData \ Local \ Microsoft \ Windows \ Fichiers Internet temporaires \ Low \

Désactiver les modules complémentaires de votre navigateur Web


Plusieurs fois, les problèmes liés aux sites Web sont le résultat de modules complémentaires tiers pour votre navigateur Web. Ceux-ci peuvent également être appelés "Extensions" ou "Plugins" en fonction de votre navigateur de choix. Pour voir si c'est le cas pour vous, essayez de désactiver tous vos modules complémentaires. Si le problème est résolu, essayez de le réactiver un par un jusqu'à ce que votre problème recommence. Une fois le problème rétabli, vous devriez avoir trouvé celui qui est incompatible avec Roblox.


Les liens suivants peuvent vous aider avec les navigateurs Web PC les plus populaires :

  • Internet Explorer (à partir de Microsoft.com)
  • Firefox (à partir de Mozilla.com)
  • Chrome (à partir de Google.com)
  • Autres navigateurs : Si vous utilisez un navigateur différent ou une version antérieure d'un navigateur, veuillez rechercher des informations sur la manière de désactiver ses extensions / plug-ins / modules complémentaires pour votre navigateur / version spécifique.
