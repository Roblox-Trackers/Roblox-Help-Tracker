L’appli iOS: Erreur stockage

Si une erreur stockage se produit lorsque vous essayez de jouer à un jeu via l'application Roblox Mobile, il se peut que vous exécutiez trop d'applications concurrentes. Pour y remédier, vous devrez peut-être fermer certains programmes pour permettre aux jeux de fonctionner correctement ou même redémarrer votre appareil. Si vous partagez votre iDevice avec d'autres personnes, assurez-vous de vérifier avec elles avant de fermer une application ou de redémarrer.
