App iOS: Boucle de mise à jour

Parfois, lorsqu'un utilisateur ouvre une application mobile Roblox obsolète sur un appareil iOS, il reçoit une notice les poussant vers l'Application Boutique pour mettre à jour l'application. Cependant, sur la page Roblox Application Boutique, l'option Ouvrir l'application est disponible. Lorsqu'un utilisateur appuie sur le bouton Ouvrir , il reçoit le message pour mettre à jour l'application.

Comment résoudre

Pour résoudre ce problème, procédez simplement comme suit:

  1. Videz le cache de l'Application Boutique en tapotant 10 fois au total sur l'une des icônes de navigation (par exemple, Sélection, Catégories, Top Charts, etc.) en bas, en vous assurant que la même icône est tapée à chaque fois.
  2. Ensuite, fermez l'application Boutique, puis redémarrez-la. De là, il est recommandé de mettre à jour directement depuis l'Application Boutique.

Pour éviter que votre application ne soit obsolète à l'avenir, vous pouvez configurer iOS pour mettre à jour automatiquement les applications afin que les nouvelles versions de l'application Roblox Mobile soient automatiquement téléchargées lorsqu'elles sont disponibles.
