Application iOS : erreur de mémoire

Si une erreur de mémoire se produit lorsque vous essayez de jouer une expérience via l'application Roblox Mobile, vous exécutez peut-être trop d'applications simultanées. Pour résoudre ce problème, vous devrez peut-être fermer certains programmes pour permettre aux expériences de fonctionner correctement ou redémarrer votre appareil. Si vous partagez votre iDevice avec d'autres personnes, assurez-vous de vérifier avec eux avant de fermer une application ou de redémarrer.
