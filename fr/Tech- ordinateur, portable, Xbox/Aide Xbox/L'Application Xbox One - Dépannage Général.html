L'Application Xbox One : Dépannage Général

Si vous rencontrez des difficultés avec l'Application Roblox sur Xbox One, nous vous recommandons de parcourir les solutions fournies par Microsoft sur leur page de Support pour Xbox One : Une application ne démarre pas sur Xbox One

Remarque : Veuillez garder à l'esprit que nous recommandons ces solutions pour les inconvénients généraux et non pas seulement pour les situations où l'application ne démarre pas.

Si après avoir suivi les étapes indiquées sur cette page, les blocages persistent, veuillez contacter le Service Client et fournissez les informations suivantes :

  • GamerTag Xbox :
  • Nom d'Utilisateur Roblox :
  • Tout message d'erreur reçu :
  • Tout Code d'Erreur Reçu :

S'il y a d'autres informations supplémentaires comme les modifications récentes apportées soit aux comptes Xbox ou Roblox, soit à votre connexion d'internet, veuillez les indiquer également, ainsi que tout autre détail.

Remarque : Les GamerTags Xbox  sont sensibles aux caractères (font la différence entre les lettres MAJUSCULES et minuscules), veuillez donc vous assurer de nous donner le gamertag exact lorsque vous nous contactez.
