Réinitialisation de l'horloge ou N'a pas réussi à obtenir l’URL: Httpsendrequest failed,err=0x2f05

Divers problèmes de plusieurs programmes ou sites Web peuvent être causés par l’horloge système Windows qui est réglé sur une heure, une date ou un fuseau horaire erroné.

Vous avez peut-être reçu l'un des messages Roblox suivants:

  • Votre heure système est incorrecte
  • N'a pas réussi à obtenir l'URL...Http send request failed,err=0x2f05

Tout simplement corriger tout devrait remédier à la situation. Vous pouvez définir ces aspects manuellement ou le configurer pour une mise à jour automatique avec un serveur. Si vous le configurer automatiquement, il ne sera pas retardé et ne provoquera pas à nouveau des problèmes. Voici comment faire pour le définir automatiquement:

Windows 8.1 et versions antérieures

    1. Cliquez souris droit sur l’horloge de Windows dans le coin bas droit
    2. Cliquez sur Régler la date et l’heure
    3. Sélectionnez l’onglet Heure Internet
    4. Ici, vous devriez pouvoir synchroniser votre heure et votre date avec un serveur Internet

Windows 10

    1. Cliquez souris droit sur l’horloge de Windows dans le coin bas à droite
    2. Assurez-vous d’avoir activé Définir l'heure automatiquement et que vous avez sélectionné le fuseau horaire correct
