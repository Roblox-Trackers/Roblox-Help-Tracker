Que faire si votre antivirus a signalé Roblox

Si votre antivirus a signalé Roblox comme un problème, soyez assuré que Roblox est dispensé de virus et ne contient aucun logiciel malveillant. Parfois, les programmes antivirus peuvent signaler un programme de manière incorrecte, ce qui est appelé un faux positif. Voici quelques solutions suggérées pour éviter que cela se produise, ainsi que ce que vous pouvez faire si cela se produit:

Mettre à jour votre antivirus

Parfois, il vous suffit de vous assurer que vous disposez de la version la plus récente de votre programme antivirus. Il est possible qu'ils aient déjà corrigé le problème. Si cela ne fonctionne pas, passons à la possibilité suivante.

Signaler le rapport comme un faux positif

Si la mise à jour du logiciel n'a pas aidé, vous devrez alors signaler l'avertissement comme un faux positif. Pour vous assurer que le programme Roblox peut fonctionner correctement, vous devez le définir comme programme autorisé pour toute application antivirus ou anti-malveillant. Cela peut être différent pour chaque programme, mais les liens ci-dessous peuvent être utiles pour certains des plus courants.

Lorsque vous recherchez des exclusions sur des machines Windows, les fichiers Roblox sont stockés dans le dossier suivant:

C: \ Users \ (votre nom d'utilisateur Windows) \ AppData \ Local \ Roblox
