Je suis connecté, mais j’apparais en tant qu’invité à l'intérieur d'expériences

Si vous constatez que vous vous trouvez dans des expériences en tant qu'invité même si vous êtes connecté à votre compte, il y a quelques points à examiner.

Que faire - Windows

Veuillez essayer chacune des étapes ci-dessous, une à la fois, dans l'ordre proposée. Si une solution ne fonctionne pas, passez à la suivante!

Redémarer

Commencez par redémarrer votre ordinateur et essayez de jouer. Cela peut parfois suffire pour résoudre les soucis liés à Roblox ou au Studio Roblox.

Vérifier le système horaire de votre Windows

Cela peut sembler étrange, mais divers soucis peuvent être causés par le réglage de l'heure ou de la date de votre système horaire Windows. Si c'est la cause, tout simplement corriger tout devrait remédier la situation. Bien que cela puisse être fait manuellement, vous pouvez également le configurer pour qu'il se mette automatiquement à jour avec un serveur, garantissant ainsi qu'il ne soit pas en retard et provoquant à nouveau des problèmes. Voici comment faire ceci:

Windows 8.1 et versions antérieures

    1. Cliquez souris droit sur l’horloge de Windows dans le coin bas droit
    2. Cliquez sur Régler la date et l’heure
    3. Sélectionnez l’onglet Heure Internet
    4. Ici, vous devriez pouvoir synchroniser votre heure et votre date avec un serveur Internet

Windows 10

    1. Cliquez souris droit sur l’horloge de Windows dans le coin bas à droite
    2. Assurez-vous d’avoir activé Définir l'heure automatiquement et que vous avez sélectionné le fuseau horaire correct

Vérifiez vos paramètres de proxy

Si les étapes ci-dessus n'ont pas résolu votre souci, c’est possible que vous deviez vous assurer que votre ordinateur ne tente pas de se connecter via un serveur proxy. Veuillez suivre ces étapes:

Windows 7 ou versions antérieures

    1. Allez sur le menu Démarrer
    2. Cherchez “proxy” et une option Configurez le serveur proxy s’affichera. Cliquez dessus
    3. Cliquez sur Paramètres LAN et décochez toutes les cases de l'onglet
    4. Cliquez sur OK

Windows 8 ou 8.1

    1. Accédez à l'écran de démarrage en mosaïque en cliquant sur le coin inférieur gauche de votre écran.
    2. Saisissez “proxy” et une option “Configurer le serveur proxy”s’affichera. Cliquez dessus
    3. Cliquez sur Paramètres LAN et décochez toutes les cases de l'onglet
    4. Cliquez sur OK

Windows 10

    1. Appuyez sur la touche Windows + S de votre clavier. Sinon, si la barre "Demandez-moi quelque chose" de Cortana est disponible dans votre barre des tâches, vous pouvez simplement cliquer dessus.
    2. Saisissez “proxy” et une option “Configurer le serveur proxy”s’affichera. Cliquez dessus
    3. Assurez-vous que "Utiliser le script d'installation" et "Utiliser un serveur proxy" sont désactivés
    4. Faites descendre et cliquez sur Sauvegarder
