Comment désinstaller Roblox

Si vous rencontrez des problèmes techniques sur l'application ou le studio Roblox, il peut parfois s’avérer utile de désinstaller Roblox avant de le réinstaller. Cet article vous guidera dans le processus de désinstallation.

Instructions Windows

Désinstaller le programme

Windows 7

  • Accédez au menu Démarrer > Panneau de configuration > Programmes et fonctionnalités.
  • Faites défiler l'écran vers le bas jusqu’à trouver Roblox.
  • Désinstallez le programme depuis cette fenêtre.

Windows 8 ou 8.1

  • Désactivez l’écran de démarrage en mosaïque en cliquant dans le coin inférieur gauche de ton écran.
  • Saisissez « Programmes et fonctionnalités » et cliquez sur l’icône Programmes et fonctionnalités qui s’affiche à droite dans les résultats de recherche.
  • Dans la fenêtre qui s’ouvre, faites défiler l'écran vers le bas jusqu’à trouver Roblox.
  • Désinstallez le programme depuis cette fenêtre.

Windows 10

  • Cliquez sur le bouton de démarrage ou sur la zone de recherche Cortana dans le coin inférieur gauche de l’écran.
    • Vous pouvez également maintenir la touche Windows de votre clavier, puis appuyer sur la touche S.
  • Dans la zone de recherche, tapez « Programmes et fonctionnalités » et cliquez sur l’icône Programmes et fonctionnalités qui s’affiche en haut des résultats de recherche.
  • Dans la fenêtre qui s’ouvre, faites défiler l'écran vers le bas jusqu’à trouver Roblox.
  • Désinstallez le programme depuis cette fenêtre.

Supprimer le dossier Roblox

Si un problème persiste particulièrement, il peut être utile de supprimer complètement le dossier Roblox après la désinstallation pour s'assurer qu'il ne reste pas de fichiers corrompus ou dépassés. Pour ce faire, suivez les étapes ci-dessous :

  1. Cliquez avec le bouton droit sur le bouton de démarrage puis cliquez sur Explorer ou Explorateur de fichiers.
  2. Accédez à ce dossier : C:\Users\(votre nom d’utilisateur Windows)\AppData\Local
  3. Supprimez le dossier Roblox.

Remarque : cette opération ne devrait pas supprimer les lieux que vous avez enregistrés dans Studio, à moins que vous n'ayez choisi manuellement de les enregistrer dans ce dossier.

Difficulté lors de la désinstallation

Si vous avez des difficultés à désinstaller le programme, essayez la procédure suivante :

Effacer les fichiers Internet temporaires

Une étape importante dans toute procédure de désinstallation ou de réinstallation consiste à effacer les fichiers Internet temporaires de ton navigateur. Cela résoudra tout problème de corruption ayant pu se produire. Pour obtenir de l’aide pour réaliser cette procédure, cliquez ici et consultez la section Effacer les fichiers Internet temporaires et les cookies .

Vérifier le pare-feu

De nombreux antivirus ou logiciels de sécurité Internet tentent de contrôler les programmes pouvant accéder à Internet. Souvent, ces logiciels font preuve d’un excès de prudence et, par défaut, aucun programme ne peut y accéder. Pour obtenir de l’aide pour vérifier cela et autoriser Roblox à contourner tout blocage, cliquez ici .

Réinstaller Roblox

La possibilité suivante consiste à réinstaller le programme. Cela peut paraître contre-intuitif mais si le problème provient d’un fichier corrompu généré par un précédent échec de tentative de désinstallation, la réinstallation permettra d’écraser la copie du programme sur votre ordinateur et de remplacer tout fichier supprimé ou endommagé. Vous devriez alors pouvoir accéder à votre menu Démarrer et désinstaller le programme. Pour obtenir de l’aide lors de la réinstallation, cliquez ici .

Instructions Mac

Fermer complètement le programme Roblox

  • Appuyez simultanément sur les touches Commande + Option + Échappe pour ouvrir une invite de gestion des tâches.
  • Vérifiez que Roblox ou Roblox Studio ne figure pas dans la liste.
  • Si l’un de ces programmes y est répertorié, sélectionnez Roblox ou Roblox Studio et cliquez sur Forcer à quitter.

Accéder à Finder > Applications

  • Faites glisser l’icône Roblox dans la corbeille.
    • L’icône dans Finder contient tous les fichiers Roblox qui ont été installés.

Comment supprimer des fichiers à partir de la corbeille

  • Cliquez sur l’icône de Corbeille dans le Dock et sélectionnez Vider la corbeille.
  • Vous pouvez également vider la corbeille en appuyant sur trois touches consécutivement : Commande + Shift + Suppr.
  • Vous devriez voir un message d’avertissement : « Êtes-vous sûr de vouloir supprimer les éléments de votre corbeille » car vous ne pouvez pas annuler cette action , mais en supposant que vous vouliez les supprimer, une fois que vous aurez cliqué sur Vider la Corbeille, les fichiers devraient avoir disparu.


Pour éviter que le message « Êtes-vous sûr de vouloir supprimer les éléments de votre corbeille » apparaisse, appuyez sur :

  • Commande + Option/Alt + Shift +Suppr
