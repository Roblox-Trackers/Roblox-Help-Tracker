Comment installer et jouer à Roblox ?

Comment jouer à Roblox

  1. Installer le Lecteur Roblox
    • Accédez au site Web de Roblox à l'aide de n'importe quel navigateur moderne comme Microsoft Edge, Firefox ou Chrome
    • Une fois connecté à Roblox, visitez n'importe quelle expérience et cliquez sur le bouton vert Jouer
    • Une fenêtre pop-up apparaîtra pour vous informer que le Lecteur Roblox est en cours d'installation
  2. Lancez le lecteur via n'importe quel navigateur moderne comme Microsoft Edge, Firefox ou Chrome
    • Accédez à n'importe quel jeu sur le site Web de Roblox
    • Sélectionnez Jouer
    • Une fenêtre pop-up secondaire peut apparaître vous demandant de sélectionner un programme. Sélectionnez Roblox et validez. Si vous choisissez de vous souvenir de votre choix, vous n'aurez plus besoin d'effectuer cette action à l'avenir

Comment installer et jouer à Roblox via Firefox sur MacOS

1. Installer le Lecteur Roblox

  • Accédez au site Web de Roblox à l'aide de Firefox
  • Une fois connecté à Roblox, visitez n'importe quelle expérience et cliquez sur le bouton vert Jouer
  • Une fenêtre contextuelle apparaîtra pour vous informer que le Lecteur Roblox est en cours d'installation

2. Lancer le lecteur

  • Cochez Toujours utiliser cette application pour ouvrir les liens du lecteur Roblox , et cliquez sur Choisir pour afficher la fenêtre Chercher
  • Dans la fenêtre du Chercher, utilisez le raccourci clavier CMD+SHIFT+G pour afficher l'entrée du nom de chemin
  • Dans l'entrée du nom de chemin, saisissez /Applications/Roblox.app/Contents/MacOS/Roblox.app , et appuyez sur RETOUR
  • Ensuite, dans la fenêtre du Chercher, sélectionnez Roblox.app, et cliquez sur Ouvrir
  • Dans la boîte de dialogue Choisir une application, sélectionnez Roblox.app et cliquez sur Ouvrir le lien
  • Roblox devrait être en cours d'exécution, et en cliquant sur le bouton de lecture pour ouvrir les expériences Roblox devrait fonctionner à l'avenir.

Comment créer vos propres jeux Roblox

Vous devrez installer Roblox Studio pour créer vos propres expériences Roblox. Voici comment procéder.

  1. Accédez au site Web de Roblox et connectez-vous à votre compte.
  2. Une fois connecté à Roblox, visitez l'une de vos expériences, cliquez dans le coin supérieur droit et sélectionnez Modifier .
  3. Une fenêtre pop-up apparaîtra pour vous informer que Roblox Studio est en cours d'installation
  4. Une fois installé, Studio s'ouvrira automatiquement
