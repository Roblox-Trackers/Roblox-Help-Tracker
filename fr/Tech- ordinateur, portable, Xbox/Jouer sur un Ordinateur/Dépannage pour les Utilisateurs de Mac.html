Dépannage pour les Utilisateurs de Mac

Désinstaller/Réinstaller

Si vous rencontrez des difficultés et vous n’avez pas récemment effectué une réinstallation, essayez de désinstaller le programme, puis réinstallez-le en suivant les étapes décrites plus bas, après ces deux articles :

  • Désinstaller
  • Réinstaller

Autres astuces à essayer

Si désinstaller et réinstaller n'a pas aidé, nous vous invitons à essayer les astuces suivantes :

  • Roblox Client s'installe sur les versions 10.7 (Lion) et celles supérieures - et uniquement sur les Macs Intel, tandis que Roblox Studio s'installe sur Mac OS 10.10 (Yosemite) et les versions ultérieures.
  • Actuellement, nous prenons en charge les puces Apple Silicon / M1.
  • Si vous utilisez actuellement le navigateur Web Safari et vous rencontrez des soucis, essayez d'utiliser l'un de nos navigateurs compatibles, tels que Firefox, Internet Explorer ou Google Chrome.
  • Vérifiez bien que l’icône Roblox soit bien présente sur la barre de menu, et que l’Application Roblox apparaît dans votre dossier d’Applications.
  • Essayez de lancer le programme depuis la barre de menu si vous avez des difficultés à jouer depuis le site en ligne.
  • Essayez de re-configurer complètement vos paramètres Roblox, en supprimant les fichiers de configuration. Pour cela, allez dans la Bibliothèque-> Préférences et supprimez tous les fichiers Roblox com. Lorsque le Studio / le Player se charge de nouveau, cela devrait tous réinstaller.
