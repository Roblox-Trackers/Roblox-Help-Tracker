Roblox Launcher Plugin

Si Roblox ne parvient pas à s'exécuter ou fournit une invitation pour télécharger Roblox, même s'il est déjà installé. Vous devriez vérifier si le plug-in Roblox est autorisé dans votre navigateur préféré avec Internet Explorer, car Roblox utilise Internet Explorer pour certaines de ses fonctions. Pour vérifier, effectuez les opérations suivantes:

Mozilla Firefox

  1. Cliquez sur le bouton de menu en haut à droite de votre fenêtre Firefox

  2. Dans le menu, cliquez sur le bouton Modules complémentaires

  3. Dans la fenêtre qui apparaît, cliquez sur Plugins

  4. Recherchez "Roblox Launcher Plugin" et sélectionnez Always Allow dans son menu déroulant à droite

  5. Fermez l'onglet Gestionnaire de modules complémentaires et lancez un jeu Roblox

Internet Explorer

  1. Cliquez sur l'icône représentant un engrenage, située dans le coin supérieur droit
  2. Cliquez sur Gérer les modules complémentaires
  3. Dans la fenêtre contextuelle située à gauche sous "Afficher", cliquez sur le menu déroulant.
  4. Sélectionnez tous les modules complémentaires
  5. Sur le côté droit de la fenêtre contextuelle, déplacez la barre de défilement vers le bas.
  6. Sous Roblox Corporation, assurez-vous que «Launcher Class» est activé

Apple Safari

  1. Ouvrez Safari et cliquez sur l'icône d'engrenage dans le coin supérieur droit
  2. Cliquez sur Préférences, puis sur l'onglet Sécurité
  3. Sous l'onglet Sécurité, assurez-vous que les plug-ins, Java et JavaScript sont activés

Google Chrome

Remarque: à partir de la version 42, Chrome ne prend plus en charge les plug-ins.

Microsoft Edge

Remarque: Microsoft Edge ne prend pas en charge les plug-ins.
