Plugin du lanceur Roblox

Si Roblox ne parvient pas s'exécuter ou fournit une invite pour télécharger Roblox, même s'il est déjà installé. Pour vérifier, procédez comme suit :

Mozilla Firefox

  1. Cliquez sur le bouton de menu en haut à droite de votre fenêtre Firefox
  2. Dans le menu, cliquez sur le bouton Modules complémentaires
  3. Dans la fenêtre qui apparaît, cliquez sur Plugins
  4. Recherchez "Roblox Lanceur Plugin" et sélectionnez Toujours Activé dans son menu déroulant à droite
  5. Fermez l'onglet Gestionnaire de modules complémentaires et lancez une expérience Roblox

Apple Safari

  1. Ouvrez Safari et cliquez sur l'icône d'engrenage dans le coin supérieur droit
  2. Cliquez sur Préférences, puis sur l'onglet Sécurité
  3. Sous l'onglet Sécurité, assurez-vous que JavaScript est activé

Google Chrome

Remarque : à partir de la version 42, Chrome ne prend plus en charge les plug-ins.

Microsoft Edge

Remarque: Microsoft Edge ne prend pas en charge les plug-ins.
