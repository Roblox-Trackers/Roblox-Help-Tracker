Comment activer des Cartes Cadeaux Roblox

Les Cartes Cadeaux ne peuvent être activées que dans un navigateur - celles-ci ne peuvent pas être activées dans les applications Roblox.

Il existe deux types de Cartes Cadeaux Roblox :

  • Crédit - Valeur en argent réel pour l'achat des Robux ou des adhésions
  • Robux - Monnaie virtuelle ajoutée au solde du compte - disponible seulement sur Amazon.com

Activer une carte cadeau sur votre compte

  1. Connectez-vous à votre compte sur un navigateur
  2. Accédez à la page Activer un Code .
  3. Entrez votre code dans la case.
  4. Cliquez sur Activer.

  5. Un message de réussite apparaîtra lorsque vous aurez réussi à ajouter le crédit à votre compte.
    Si vous rencontrez des soucis en activant un code PIN qui contient un 0 (zéro) ou un 1 (un), veuillez essayer de les remplacer par les lettres majuscules O et I.
  6. Cliquez sur Utiliser le Crédit pour utiliser votre crédit. Vous pouvez lire l'article d'aide Utiliser le Crédit d'une Carte Cadeau pour obtenir de l'aide sur les étapes suivantes OU Cliquez sur Fermer pour fermer le message et utiliser une autre carte cadeau.

Activer le Crédit pour payer un achat

  1. Connectez-vous à votre compte sur un navigateur
  2. Accédez à la page d' Adhésion ou à la Page Robux
  3. Sélectionnez le produit que vous souhaitez acheter en cliquant sur le bouton correct
  4. Sélectionnez Activer votre Carte Roblox comme type de paiement et cliquez sur Continuer .
  5. Entrez le code PIN et cliquez sur Activer
  6. Une fois le solde de votre Crédit Roblox est mis à jour, cliquez sur Envoyer la Commande
  7. La page sera mise à jour pour confirmer votre paiement.
