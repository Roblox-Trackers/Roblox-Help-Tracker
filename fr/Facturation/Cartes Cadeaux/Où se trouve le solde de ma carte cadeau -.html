Où se trouve le solde de ma carte cadeau ?

La carte cadeau ne peut être utilisée que dans un navigateur - elle ne peut pas être utilisée dans les applications Roblox.

Où se trouve le solde de ma carte cadeau ?

Après avoir activé une carte cadeau, votre solde créditeur apparaît généralement sur les pages d'achat. Le solde créditeur d'un compte ne sera pas visible sur toutes les pages.

Étapes pour vérifier si vous avez reçu votre crédit de carte cadeau

  • Assurez-vous que vous êtes connecté au compte sur lequel la carte de jeu a été utilisée, dans un navigateur.
  • Accédez à la page Activer une carte Roblox ou à Page Paramètres de facturation.
  • Sur la page Activer une carte Roblox, votre solde créditeur apparaît sous le bouton Activer .
  • Sur la page de Facturation, votre solde actue l s'affiche sous Crédit Roblox .
  • Si vous ne voyez pas le solde créditeur, cela signifie que le compte ne dispose actuellement d'aucun crédit de carte cadeau Roblox utilisé.
