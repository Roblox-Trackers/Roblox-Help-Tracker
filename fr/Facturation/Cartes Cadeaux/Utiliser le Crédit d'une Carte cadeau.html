Utiliser le Crédit d'une Carte Cadeau

Le crédit de carte cadeau ne peut être utilisé que dans un navigateur - il ne peut pas être utilisé dans les applications Roblox.

Depuis les pages d'abonnement ou d'achat de Robux

  1. Connectez-vous au compte Roblox où vous avez échangé la carte cadeau sur le navigateur.
  2. Allez sur la page d'achat d'abonnement ou sur la page d'achat de Robux .
  3. Sélectionnez le produit que vous souhaitez acheter.
  4. Sélectionnez Crédit Roblox comme type de paiement et cliquez sur Continuer .

  5. Cliquez sur Commander . Veuillez ne pas resaisir le code PIN de votre carte cadeau si vous l'avez déjà utilisé.

  6. Si vous avez assez de crédit pour le prix d'achat total, vous verrez un message de confirmation.

Conversion du crédit restant en Robux

Vous pouvez également convertir ce crédit en Robux.

  1. Connectez-vous au compte Roblox où vous avez échangé la carte cadeau sur le navigateur.
  2. Cliquez sur l'engrenage dans le coin supérieur droit pour accéder à votre page de paramètres .
  3. Sélectionnez l'onglet Facturation
  4. Cliquez sur le bouton Convertir en Robux
  5. Vous recevrez un écran de confirmation, cliquez sur Activer pour terminer la conversion ou sur Annuler pour ne pas convertir le crédit.
  6. Votre solde Robux sera mis à jour après l'actualisation de votre page actuelle ou votre navigation vers une nouvelle page.
