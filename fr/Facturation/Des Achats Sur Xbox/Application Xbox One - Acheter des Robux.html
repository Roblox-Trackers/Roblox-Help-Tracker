Application Xbox One : Acheter des Robux

Les cartes-cadeaux Roblox ne peuvent pas être échangées sur Xbox pour le moment.

Comment acheter des Robux pour une utilisation sur Xbox

Assurez-vous d'être connecté à votre compte Roblox lié à Xbox Live lorsque vous achetez des Robux, car il n'est pas possible de transférer des Robux entre vos comptes.
  • Pour les articles Avatar, si vous n'avez pas assez de Robux, la page de l'article apparaîtra avec une option pour obtenir Robux . La sélection de cette option vous amènera à la page Robux où vous pourrez choisir parmi les packs disponibles.
  • Pour les articles d'expérience, si vous n'avez pas assez de Robux, un message apparaîtra avec le montant de Robux le plus bas qui vous permet toujours d'acheter l'article.
  • Vous pouvez acheter Robux directement à tout moment à partir de l'écran Robux dans l'écran principal de l'expérience.
  • Vous pouvez acheter Robux directement sur le site Web Xbox Live .

Affichage de votre solde Robux

Sur l'écran d'accueil, cliquez sur le pare-chocs droit (RB) jusqu'à ce que vous atteigniez l'écran Robux. Sur cette page, dans le coin inférieur gauche, votre total actuel se trouve à côté de Mon solde .
