Application PlayStation : problèmes de connexion au compte Roblox

Pour connecter votre compte Roblox à votre compte PlayStation Network dans l'application, vous devez connaître votre nom d'utilisateur et votre mot de passe Roblox. Si vous n'arrivez pas à vous connecter parce que vous avez oublié votre mot de passe, vous devrez le modifier. Cliquez ici pour savoir comment réinitialiser votre mot de passe.

Remarque : la fonctionnalité de connexion via Facebook n'est pas disponible sur PlayStation, vous devrez donc utiliser le mot de passe de votre compte Roblox.

Déconnecter vos comptes Roblox et PlayStation

      1. Dans l'application, sélectionnez le bouton de menu ( ... )
      2. Sélectionnez l'option Déconnexion

Vous pouvez également déconnecter vos comptes depuis les paramètres de Roblox .

      1. Allez à l'onglet Sécurité
      2. Allez à la section Où vous vous connectez
      3. Cliquez sur la flèche à côté de l'option Application Roblox sur PlayStation
      4. Cliquez sur Déconnecter PlayStation .
