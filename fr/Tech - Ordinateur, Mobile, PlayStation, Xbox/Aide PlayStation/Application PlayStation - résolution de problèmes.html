Application PlayStation : résolution de problèmes

Si vous rencontrez des problèmes avec l'application Roblox sur PlayStation, nous vous recommandons de consulter les solutions sur la page d'assistance de Sony intitulée Comment résoudre les problèmes de lancement de jeux

Remarque : ces solutions sont recommandées pour les problèmes d'ordre général, et pas seulement pour les situations où l'application ne démarre pas.

Si les problème(s) persiste(nt) après avoir suivi les instructions de la page, contactez l'assistance clientèle de Roblox en fournissant les informations suivantes :

    • Nom du compte PlayStation Network :
    • Nom d'utilisateur Roblox :
    • Message d'erreur reçu :
    • Code d'erreur reçu :

S'il y a d'autres informations pertinentes, comme des changements récents sur votre compte PlayStation ou Roblox, ou sur votre connexion Internet, indiquez-les également.

Remarque : les noms de compte PlayStation Network prennent en compte les majuscules et les minuscules, assurez-vous donc d'utiliser le format correct lorsque vous nous contactez.
