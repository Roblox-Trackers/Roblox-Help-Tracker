Problèmes généraux de Roblox Studio

L'article suivant proposera quelques méthodes qui se sont révélées efficaces pour résoudre les problèmes suivants :

  • Vous essayez de vous connecter sur Roblox Studio, mais vous n'arrivez pas
  • Vous essayez d'éditer un lieu uniquement pour qu'il charge un skybox vide ou quelque chose appelé Place1
  • Les sections de Studio sont vides et cela ne devrait pas être le cas
  • Vous rencontrez des difficultés pour publier sur Roblox
  • Roblox Studio vous dit de vous connecter même si vous y êtes déjà
  • Ces étapes peuvent résoudre certains problèmes non liés à Studio, tels que la saisie d'une expérience en tant qu'invité même lorsque vous êtes connecté
  • Autre : Si vous rencontrez un problème différent avec le Studio, les étapes ci-dessous peuvent également vous aider à le résoudre

Que faire - Windows

Veuillez essayer chaque étape ci-dessous, une à la fois, dans l'ordre. Si une solution ne fonctionne pas, passez à la suivante !

Redémarrer

Commencez par redémarrer votre ordinateur et essayez de jouer. Cela peut parfois suffire à résoudre les problèmes liés à Roblox ou Roblox Studio.

Vérifiez vos paramètres de proxy

Si les étapes ci-dessus n'ont pas résolu votre problème Roblox Studio, il est possible que vous deviez vous assurer que votre ordinateur ne tente pas de se connecter via un serveur proxy. Veuillez suivre ces étapes :

Windows 7 ou versions antérieures

    1. Allez dans le menu Démarrer
    2. Recherchez "proxy" et "Configurer le Serveur Proxy" apparaîtra, cliquez dessus
    3. Cliquez sur Paramètres de réseau et décochez toutes les options dans cet onglet
    4. Cliquez sur OK

Windows 8 ou 8.1

    1. Accédez à l’écran de démarrage en mosaïque en cliquant sur le coin inférieur gauche de votre écran
    2. Tapez "proxy" et "Configurer le Serveur Proxy" apparaîtra, cliquez dessus
    3. Cliquez sur Paramètres de réseau et décochez toutes les options dans cet onglet
    4. Cliquez sur OK

Windows 10

    1. Appuyez sur la touche Windows + S de votre clavier. Sinon, si la barre "Demandez-moi n'importe quoi" de Cortana est disponible dans votre barre des tâches, vous pouvez simplement cliquer dessus
    2. Tapez "proxy" et "Modifier les paramètres proxy" apparaîtra, cliquez dessus
    3. Assurez-vous que "Utiliser le script d'installation" et "Utiliser un serveur proxy" sont désactivés
    4. Faites défiler et cliquez sur Enregistrer

Vérifiez votre horloge système Windows

Cela peut sembler étrange, mais divers problèmes peuvent en fait être causés par le fait que votre horloge système Windows est réglée sur une heure ou une date erronée. Si c'est le cas, tout simplement en corrigeant tout devrait remédier la situation. Bien que cela puisse être fait manuellement, vous pouvez également le configurer pour qu'il se mette à jour automatiquement avec un serveur, vous assurant ainsi qu'il ne glisse pas et ne recommence pas à causer des problèmes. Voici comment procéder :

Windows 8 ou antérieur

    1. Faites un clic droit sur l'horloge de la fenêtre dans le coin inférieur droit
    2. Cliquez sur Ajuster Date/Heure
    3. Sélectionnez l'onglet Temps Internet
    4. Ici vous devriez pouvoir synchroniser votre date et votre heure avec un serveur Internet

Windows 10

    1. Faites un clic droit sur l'horloge de la fenêtre dans le coin inférieur droit
    2. Assurez-vous que "Définir l'heure automatiquement" est activé et que vous avez sélectionné le bon fuseau horaire

Que faire - Mac

Roblox Client s'installera sur 10.10 (Yosemite) et supérieur tandis que Roblox Studio s'installera sur Mac OS 10.11 (El Capitan) et supérieur.

Pour éventuellement résoudre ce problème sur un Mac, veuillez réinstaller. Pour obtenir de l'aide, cliquez ici .

Réduire les futurs problèmes
