Messages Hyperion

Cet article présente les messages d'erreur potentiels de la fonction anti-triche d'Hyperion qui peuvent apparaître, ainsi que les étapes de dépannage correspondantes pour vous aider à résoudre ces problèmes.

Type de crash : Expiration de session

Message utilisateur : La session de Roblox a expiré. Veuillez redémarrer votre client.

Étapes de dépannage :

Si vous rencontrez un message indiquant « La session de Roblox a expiré. Veuillez redémarrer votre client », cela signifie que votre client Roblox doit être redémarré. Si le problème persiste malgré le redémarrage du client dans le cadre d'une utilisation normale, contactez notre équipe d'assistance pour obtenir de l'aide.

Type de crash : Machine virtuelle détectée

Message utilisateur : Roblox ne peut pas fonctionner sur une machine virtuelle.

Étapes de dépannage :

Si vous rencontrez un message indiquant « Machine virtuelle détectée », cela signifie que votre machine virtuelle (VM) n'est pas prise en charge par Roblox. Roblox nécessite un passthrough GPU pour fonctionner correctement sur une VM. Le passthrough GPU permet à la VM d'accéder directement au GPU physique, en contournant la couche de l'hyperviseur. Voici comment activer le passthrough GPU :

      1. Confirmez que votre matériel prend en charge le passthrough GPU.
      2. Mettez à jour les paramètres de votre BIOS/UEFI pour activer les extensions de virtualisation (par exemple, Intel VT-x ou AMD-V).
      3. Configurez le passthrough GPU dans votre logiciel VM.

Le respect de ces étapes garantit le bon fonctionnement de Roblox sur un environnement virtuel, en fournissant l'accès nécessaire aux ressources GPU.

Type de crash : Badware détecté

Message utilisateur : Roblox a crashé en raison d'un logiciel incompatible : <chemin d'accès au logiciel incriminé>

Étapes de dépannage :

Si vous rencontrez un message indiquant « Roblox a crashé en raison d'un logiciel incompatible : » suivi d'un chemin d'accès au logiciel incriminé, cela signifie que le logiciel mentionné dans le chemin d'accès pose des problèmes à Roblox. Pour résoudre ce problème, vous devrez empêcher ce logiciel de fonctionner lorsque vous utilisez Roblox. Vous pouvez le faire manuellement en arrêtant le logiciel ou en effectuant un démarrage propre dans Windows. Cette méthode garantit que seuls les logiciels nécessaires sont actifs sur votre système, minimisant ainsi les conflits. Voici comment effectuer un démarrage propre, comme le recommande Windows :

      1. Appuyez sur les touches Windows + R pour ouvrir la boîte de dialogue Exécuter.
      2. Tapez « msconfig » et appuyez sur Entrée.
      3. Dans la fenêtre configuration système, allez dans l'onglet « Services ».
      4. Cochez la case « Masquer tous les services Microsoft ».
      5. Cliquez sur « Désactiver tout ».
      6. Passez à l'onglet « Démarrage ».
      7. Cliquez sur « Ouvrir le Gestionnaire des tâches ».
      8. Désactivez tous les éléments de démarrage.
      9. Fermez le gestionnaire des tâches et la configuration système.
      10. Redémarrez votre ordinateur.

En suivant ces étapes, vous vous assurerez qu'aucun autre logiciel n'interfère avec Roblox, ce qui lui permettra de fonctionner sans problème et sans crash.

Type de crash : Intégrité du lancement compromise

Message utilisateur : Des fichiers critiques du système sont corrompus. Roblox ne peut pas fonctionner. Veuillez réparer ou réinitialiser votre installation Windows.

Étapes de dépannage :

Roblox ne fonctionne pas correctement lorsque Windows est en mode débogage. Pour utiliser Roblox, assurez-vous que Windows fonctionne normalement. Suivez les étapes suivantes pour identifier et désactiver le mode de débogage dans Windows .

      1. Appuyez sur las touches Windows + R pour ouvrir la boîte de dialogue Exécuter.
      2. Tapez « msconfig » et appuyez sur Entrée.
      3. Sélectionnez l'onglet « Démarrer » puis Options avancées.
      4. Désactivez la case à cocher Débogage.
      5. Sélectionnez « OK ».
      6. Sélectionnez Appliquer, puis « OK ».
      7. Redémarrez votre ordinateur.

Ces étapes garantissent que Windows ne fonctionne pas en mode débogage, ce qui vous permet d'utiliser Roblox sans problème.

Type de crash : (OBS) Open Broadcaster Software

Message utilisateur : La capture de jeu d'OBS est connue pour avoir des problèmes avec Roblox. Veuillez utiliser la capture de fenêtre ou de bureau à la place.

Étapes de dépannage :

Si vous rencontrez un message indiquant « La capture de jeu d'OBS est connue pour avoir des problèmes avec Roblox », cela signifie que Roblox ne prend pas en charge la fonction de capture du jeu d'Open Broadcaster Software (OBS). Pour diffuser ou enregistrer Roblox, vous devrez utiliser le mode de capture de la fenêtre ou du bureau. Suivez les instructions ci-dessous pour configurer la capture de bureau :

      1. Ouvrez Open Broadcaster Software (OBS).
      2. Dans la section « Sources », cliquez sur l'icône « + ».
      3. Sélectionnez « Capture de fenêtre » dans le menu.
      4. Choisissez la fenêtre vous souhaitez capturer (en général votre moniteur principal).
      5. Ajustez les paramètres tels que la résolution et la fréquence d'images si nécessaire.
      6. Cliquez sur « OK » pour confirmer la source de capture de la fenêtre.
      7. Disposez la source de capture de fenêtre comme vous le souhaitez sur votre scène OBS.
      8. Commencez la diffusion ou l'enregistrement comme d'habitude.

En utilisant le mode de capture de bureau, vous pouvez efficacement diffuser ou enregistrer Roblox sans rencontrer de problèmes de compatibilité avec la fonction de capture de jeu d'OBS.
