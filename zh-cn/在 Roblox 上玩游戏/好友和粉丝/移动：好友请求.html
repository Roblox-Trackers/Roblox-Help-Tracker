移动：好友请求

接受和发送好友请求的功能可以在 移动 App 上使用。

接受或忽略好友请求：

手机：

  1. 点按屏幕底部的 “更多” 按钮
  2. 点按 “好友” （右侧显示目前请求数量）
  3. 从这里，点按 “请求”
  4. 点按 “接受” 或 “忽略”

平板电脑：

  1. 点按屏幕底部的 “好友” 按钮
  2. 点按 “请求” 选项卡
  3. 根据你对个别要请的决定点按 “接受” 或 “忽略” 按钮

发送好友请求：

手机：

  1. 点按屏幕底部的 “更多” 按钮
  2. 点按 “好友”
  3. 点按顶部的 “ 搜寻栏” 图标
  4. 输入好友的用户名并搜索
  5. 点按用户名右侧的添加按钮

注意：如果你已经发送好友请求，你将不能再发送另一个好友请求，并且他们的个人资料会显示为 “待处理” 或将显示 “加载” 图标。

平板电脑

  1. 点按屏幕底部的 “首页” 按钮
  2. 点按屏幕顶部的图标
  3. 输入你想要结交的玩家名称
  4. 点按清单中的玩家名字
  5. 点按 “发送好友请求” 按钮

注意：如果你已经发送好友请求，你将不能再发送另一个好友请求，并且他们的个人资料会显示为 “好友请求待处理” 。
