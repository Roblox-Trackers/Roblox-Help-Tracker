电脑硬件和操作系统要求

下面列出了 Roblox 的最低系统要求。如果您在玩游戏时遇到问题，您可能需要对您的电脑进行升及。请阅读 此帮助文章 以了解移动设备的系统要求。

操作系统要求

  • PC/Windows: Roblox 将安装在 Windows 7、Windows 8/8.1 或者 Windows 10 上。对于 Windows 8/8.1，您需要在桌面模式下运行 Roblox，因为目前不支持 Metro 模式（平铺启动屏幕）。
    注意 ：对于 Windows 7，浏览器必须是 IE10 或更高版本。
  • Mac ：Roblox 客户端将安装在 10.7（Lion）及以上版本 - 而 Roblox Studio 将安装在 Mac OS 10.11（ El Capitan ）及以上版本。
  • Linux ：Roblox 不支持 Linux 操作系统。
  • 手机 ： 点击此处 查看 Roblox 手机的系统要求。
  • Chrome OS ：Chrome OS 版本 53 或更高的版本是必需的。您必须先在您的设备上启用 Google Play商店 。

注意：Roblox 视频录制功能仅适用于 Windows 7、Windows 8/8.1 和 Windows 10。最新发布的操作系统或预发布的操作系统版本可能不会立即得到支持。

系统硬件要求

  • 显卡 ：Roblox 至少需要 DirectX 版本 10.0 和 Shader Model 版本 2.0。为了获得最佳表现，我们建议使用电脑少于 5 年的专用显卡，或者使用笔记本电脑未满 3 年的内置显卡。
  • 处理器 ：Roblox 建议您使用最新的处理器（2005+），时钟速度为 1.6 Ghz 或更高。旧 AMD 处理器会出现一些问题。
  • RAM 或记忆体 ：Roblox 建议您在 Windows 7，Windows 8 或 Windows 10 上至少有 1 GB 内存。
  • 存储空间 ：Roblox 建议您至少有 20mb 的系统存储空间来安装 Roblox。
  • 移动 ： 点击此处 查看 Roblox 移动 的系统要求。

互联网要求

对于单次使用的家庭，Roblox 建议使用最少为 4-8 Mbps 的网络连接。在较大的家庭中，更高带宽是可取的。
