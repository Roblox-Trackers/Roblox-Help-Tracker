常见网站问题

下面的文章将提出一些已被证明有效的方法来解决以下的问题：

  • 该网站显示不正确
  • 有些页面根本无法加载或加载时显示错误
  • 按钮或链接不允许您点击它们
  • 您在Roblox网站上尝试使用正确的用户名和密码但无法登录
  • 其它：如果您在网站上遇到了不同的问题，下面的步骤也可能有助于解决这个问题。此外， 一些与网站无关的文章可以链接到这里以进行一些步骤

如何操作

请按顺序尝试下面的每一步。如果一个解决方案不起作用，请继续使用下一步！

清除临时互联网文件和Cookie

以下链接将帮助您删除主要电脑网页浏览器的临时互联网文件（也称为“缓存”）和Cookie。请务必执行您想要启动Roblox的 Internet Explorer以及其它浏览器。

  • Internet Explorer - 临时Internet文件（来自Microsoft.com）
  • Internet Explorer - Cookies（来自Microsoft.com）
  • Mozilla Firefox - 临时Internet文件（来自Mozilla.com）
  • Mozilla Firefox - Cookies（来自Mozilla.com） \ 谷歌浏览器 - 两者（来自Google.com）
  • 谷歌浏览器 - 两者（来自Google.com）
  • Microsoft Edge - 两者（来自Microsoft.com）
  • Safari - Cookies（来自Apple.com）
  • 其他浏览器 ：如果您使用的是其它浏览器或旧版本的浏览器，请搜索有关如何删除特定浏览器/版本的缓存/浏览器数据/临时Internet文件的信息。

完成上述所有相关网络浏览器后，请重新启动计算机并查看您的问题是否已解决。

如果您想手动删除Internet Explorer / Windows临时Internet文件，可以直接进入硬盘驱动器上的Temporary Internet Files文件夹。位置和文件夹名称取决于Windows的版本以及您是否使用用户配置文件

注意：您可能需要更改Windows资源管理器的设置才能显示各种文件（包括受保护的系 统文件）以访问这些文件夹。

对于 Windows 10

如果您有Windows 10，则临时Internet文件位于以下位置（请注意，在PC上，它们可以位于 其他驱动器上而不是驱动器C上）：

  • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

对于Windows 7和8

如 果您有Windows 7或Windows 8，则临时Internet文件位于这些位置（请注意，它们可以 位于另一个驱动器上而不是驱动器C上）：

  • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
  • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

禁用您的Web浏览器的插件

很多时候与网站相关的问题都是第三方附加组件对于您的Web浏览器的结果。 根据您选 择的浏览器，这些也可能被称为“扩展”或“插件”。 要查看是否属于这种情况，请尝试禁用所有附加组件。 如果问题得到解决，请尝试每次重新启用它们，直到问题再次出现。 一旦问题重新开始，您应该找到与Roblox不兼容的问题。

以下链接可以帮助您使用最流行的PC网页浏览器：

  • Internet Explorer（来自Microsoft.com）
  • Firefox（来自Mozilla.com）
  • Chrome（来自Google.com）
  • 其它浏览器 ：如果您使用的是其他浏览器或较旧版本的浏览器，请搜索有关如何禁用其特定浏览器/版本的扩展/插件/附加组件的信息。

重置Windows的互联网选项

我们发现许多Roblox，网站和Roblox Studio问题都可以通过重置计算机的Internet选项来解决。请记住，即使您通常使用其他网络浏览器，这些步骤也需要使用Internet Explorer。 尝试完这些步骤后，您可以返回使用您喜欢的任何浏览器。 请尝试以下操作：

  1. 打开Internet Explorer（特别是这个网页浏览器）
  2. 打开齿轮图标> Internet选项
  3. 打开 高级选项卡
  4. 点击 重置 按钮

重置后，请关闭Internet Explorer以使更改生效。 完成此操作后，您应该能够返回使用您选择  的网络浏览器。

更新Internet Explorer

请尝试将您的Internet Explorer更新到最新版本。 您可以 点击这里 找到它。即使您目前没  有将IE浏览器用作浏览器，您仍然需要确保您的Windows计算机上具有最新版本，以便Roblox正常工作。
