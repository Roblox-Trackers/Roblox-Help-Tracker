一般网站的问题

下面的文章将提出一些已被证明有效的方法来解决以下的问题：

  • 该网站显示不正确
  • 有些页面根本无法加载或加载时显示错误
  • 按钮或链接不允许你点击它们
  • 你在 Roblox 网站上尝试使用正确的用户名和密码但无法登录
  • 其它：如果你在网站上遇到了不同的问题，下面的步骤也可能有助于解决这个问题。此外， 一些与网站无关的文章可以链接到这里以进行一些步骤

如何操作

请按顺序尝试下面的每一步。如果一个解决方案不起作用，请继续使用下一步！

删除 Internet 临时文件和 Cookie

以下链接将帮助你删除主要电脑网页浏览器的 Internet 临时文件（也称为缓存）和 Cookie。请务必执行你想要启动 Roblox 的任何 浏览器。（Roblox 不支持 Internet Explorer）

      • Mozilla Firefox - Internet 临时文件（来自 Mozilla.com）
      • Mozilla Firefox - Cookies（来自 Mozilla.com）
      • Google Chrome - 两者（来自 Google.com）
      • Microsoft Edge - 两者（来自 Microsoft.com）
      • Safari - Cookies（来自 Apple.com）
      • 其他浏览器 ：如果您使用的是其它浏览器或旧版本的浏览器，请搜索有关如何删除特定浏览器/版本的缓存/浏览器数据/Internet 临时文件的信息。

完成上述所有相关网络浏览器后，请重新启动电脑并查看你的问题是否已解决。

如果你想手动删除 Windows 的 Internet 临时文件，可以直接进入硬盘驱动器上的 Internet 临时文件夹。位置和文件夹名称取决于 Windows 的版本以及你是否使用用户配置文件

注意：你可能需要更改 Windows 资源管理器的设置才能显示各种文件（包括受保护的系 统文件）以访问这些文件夹。

对于 Windows 10

如果你有Windows 10，则 Internet 临时文件位于以下位置（请注意，在PC上，它们可以位于 其他驱动器上而不是驱动器C上）：

            • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

对于Windows 7 和 8

如 果你有 Windows 7 或 Windows 8，则临时Internet文件位于这些位置（请注意，它们可以 位于另一个驱动器上而不是驱动器C上）：

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

禁用你的 Web 浏览器的插件

很多时候与网站相关的问题都是第三方附加组件对于你的 Web 浏览器的结果。 根据你选 择的浏览器，这些也可能被称为“扩展”或“插件”。 要查看是否属于这种情况，请尝试禁用所有附加组件。 如果问题得到解决，请尝试每次重新启用它们，直到问题再次出现。 一旦问题重新开始，您应该找到与Roblox不兼容的问题。

以下链接可以帮助你使用最流行的 PC 网页浏览器：

      • Internet Explorer（来自 Microsoft.com）
      • Firefox（来自 Mozilla.com）
      • Chrome（来自 Google.com）
      • 其它浏览器 ：如果你使用的是其他浏览器或较旧版本的浏览器，请搜索有关如何禁用其特定浏览器/版本的扩展/插件/附加组件的信息。
