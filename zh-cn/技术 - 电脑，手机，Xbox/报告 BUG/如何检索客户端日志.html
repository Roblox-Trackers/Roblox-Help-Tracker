如何检索客户端日志

在某些情况下，Roblox员工可能会通过客户关怀电子邮件或DevForum公布请求客户端日志，以帮助调查某个问题。 本文描述如何检索这些日志。

注意 : 除非Roblox员工要求，否则请不要检索或发送这些日志。

检索客户端日志

检索客户端日志的过程因您使用的操作系统而异。 请注意以下信息不适用于移动设备。

  1. 通过以下方式打开 Roblox Studio ：
    Windows 7 / 10： 开始 > 所有程序（或所有应用程序）> Roblox > Roblox Studio
    Windows 8 ：单击左下角的 Windows 徽标，然后在平铺的开始屏幕出现时键入 Roblox Studio 。 从屏幕右侧显示的搜索列表中选择 Roblox Studio 。
    Mac： Command + Space Bar > 输入 RobloxStudio > 进入
  2. 通过单击“设置”>“设置”以打开“ 设置 ”
  3. 在左侧窗格中，点击” 网络 “ 。 如果没有显示，关闭并和重新打开Roblox Studio。
