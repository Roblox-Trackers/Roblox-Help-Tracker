如何检索客户端日志

在某些情况下，Roblox员工可能会通过客户关怀电子邮件或DevForum公布请求客户端日志，以帮助调查某个问题。本文描述如何检索这些日志。

注意 : 除非Roblox员工要求，否则请不要检索或发送这些日志。

检索客户端日志

检索客户端日志的过程因您使用的操作系统而异。请注意以下信息不适用于移动设备。

  1. 打开 Roblox Studio :

    Windows 7 / 10: Start > 所有程序 （或者所有应用程序） > Roblox > Roblox Studio

    Windows 8: 点击左下角的Windows标志，并在平铺的”开始“屏幕出现时输入 Roblox Studio 。从屏幕右侧的搜索列表中选择 Roblox Studio 。

    Mac: Command + Space Bar > 输入 RobloxStudio > Enter

  2. 通过点击”设置“>”设置“以打开” 设置 “

  3. 在左侧窗格中，点击” 网络 “ 。如果没有显示，关闭并和重新打开Roblox Studio。

  4. 在 " 诊断 " 下面，确保选中这些复选框：

    PrintEvents
    PrintInstances
    PrintPhysicsErrors
    PrintProperties
    PrintTouches

  5. 关闭Roblox

  6. 同时按下键盘上的WINDOWS键和R键。

    适用于Mac: Macintosh HD > Users > [your user profile] > Library > Logs > Roblox

    注意：Mac用户跳过步骤7

  7. 将出现一个名为 Run 的框。复制粘贴或键入以下不带引号的内容:

    如果使用 Windows 7 / 8 / 10: "%localappdata%\Roblox\logs"

    删除文件夹中的所有内容。

  8. 试着用正常的方式玩Roblox。

  9. 当它不起作用时，请关闭Roblox并重复步骤6和7。

    现在该文件夹中将有几个文件。 请发送给他们作为对您的支持消息的回复，确保确保参考了需要日志的问题情况。
