上传失败/场所太大

无论何时您在Studio中收到关于上传失败的信息时，通常都是由于您的场所太大或者其中包含太多项目/脚本。 要进行测试，请尝试以下操作：

  1. 首先打开Roblox Studio，然后使用 File 菜单> Save 选项将您的场所保存到电脑中
  2. 通过 File 菜单中选择 New ，使Roblox Studio保持打开状态并创建新游戏
  3. 将砖块添加到新场所并选择 File 菜单> Publish to Roblox As
  4. 选择您目前遇到困难的场所，然后上传新的空白场所。
  5. 它起作用吗？

如果这个方法有效，那么您可能需要将您的场所缩小，或者减少其中的一些项目。您可以通过编辑保存到电脑上的版本，并像我们对上面空白部分所做的那样发布它来实现这一点。或者，您可以配置您的场所，将它恢复到还没有遇到问题的早期版本。 如需协助，请 点击此处 。
