一般Roblox Studio问题

这文章提出一些有效方法解决以下问题：

  • 您尝试登录Roblox Studio但无法启动，
  • 您试着编辑一个地方但它加载了一个空白的Skybox或叫做Place1
  • Studio的一部分不应该是空的。
  • 在Roblox上发布时遇到了问题
  • 当你已经在线，Roblox Studio还是要求登录
  • 这些步骤能助于解决一些与非Studio相关的问题，列如玩家已登录，也可以作为访客进入游戏
  • 其他：如果您遇到与Studio额外问题，则以下步骤也可能有助于解决问题

如何操作- Windows

请使用按顺序尝试下面的每个步骤。 如果一个解决方案无效，请继续下一步！

重启

首先重新启动电脑并尝试播放。有时这会解决Roblox或Roblox Studio相关问题。

在Windows中重置您的互联网选项

即使您不使用Internet Explorer，通过你目前的浏览器重置Internet选项仍可解决您的问题。请注意，虽然这些步骤需要使用Internet Explorer来完成它，一旦完成后，您照样可以回到你想用的浏览器。请尝试以下操作：

    1. 打开Internet Explorer（这指定的浏览器）
    2. 打开齿轮图标> Internet选项
    3. 打开 高级选项卡
    4. 点击 重置 按钮

重置后，请关闭Internet Explorer以使更改生效。

查觀您的代理设置

如果上述步骤不能解决您的Roblox Studio问题，您需要确保电脑不是在尝试通过代理服务器进行连接。请按照以下步骤操作：

Windows 7或更早版本

    1. 转到 Start “开始”菜单
    2. 搜索“代理”和“配置代理服务器”将会出现，并点击它
    3. 点击 LAN“局域网”设置 ，然后取消选中该选项卡中的所有复选标记
    4. 点击 OK

Windows 8 or 8.1

    1. 点击屏幕左下角，以便进入平铺开始屏幕
    2. 输入Proxy“代理”然后“配置代理服务器”将会出现，并点击它
    3. 点击 LAN“局域网”设置 ，然后取消选中该选项卡中的所有复选标记
    4. 点击 OK

Windows 10

    1. 按下键盘上的Windows键+ S键。或者，如果Cortana的“问我任何事情”栏在您的任务栏中可用， 您可以单击该栏
    2. 搜索“代理”然后“配置代理服务器”将会出现，并点击它
    3. 确保“使用安装脚本”和“使用代理服务器”设置为关闭
    4. 向下滚动并点击 保存

查观您的Windows系统时钟

这听起来很奇怪，但实际上各种问题是由于Windows系统时钟的时间或日期被设置成错误而导致的。 如果这是原因，那么纠正一切就应该解决情况。 虽然可以手动完成它，但也可以将其设置为使用服务器自动更新，从而确保它不会滑落并再次引发问题。以下是如何做到这一点：

Windows 8或更早版本

    1. 右键点击右下角的窗口时钟
    2. 点击 调整日期/时间
    3. 选择 互联网时间 选项卡
    4. 在这里，您应该能够将您的时间和日期与互联网服务器进行同步

Windows 10

    1. 在这里，您应该能够将您的时间和日期与互联网服务器进行同步
    2. 确保“自动设置时间”已打开，并且您已选择了正确的时区

如何操作- Mac

Roblox客户端将安装在10.7 (Lion)及以上版本 - 只能安装在Intel版本的Mac电脑上，而Roblox Studio将安装在Mac OS 10.10 (Yosemite)及以上版本。

要在Mac上解决此问题，请重新安装。 如需帮助，请 点击此处 。

最大限度地减少未来的问题

始终确保在安装或更新Roblox之前，您的电脑上的所有程序都已关闭。 对于Windows， 请确保Internet Explorer与最新版本保持同步。
