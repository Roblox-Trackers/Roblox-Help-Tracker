我登录了，但我在游戏中以客人的身份出现

如果您发现即使您已登录了您的账户，但还是以客人的身份进入游戏中， 有几件事需要调查一下。

怎么办 - Windows

请按顺序尝试下面的每一步。如果一个解决方案不起作用，那就继续下一个步骤！

重启

首先重新重启您的电脑并尝试玩。有时，这可能是解决Roblox或Roblox Studio相关问题所需要的全部内容。

在Windows中重置您的互联网选项

即使您不使用Internet Explorer，通过此浏览器重置互联网选项也可能解决您的问题。 请注意，虽然这些步骤确实需要使用Internet Explorer来完成，但您可以在完成后返回到您选择的浏览器。请尝试以下操作：

  1. 打开Internet Explorer
  2. 打开“齿轮”图标> 互联网选项
  3. 打开 高级 选项卡
  4. 点击 重置 按钮

重置后，请关闭Internet Explorer以使这些变化生效。

注意：如果您使用的是Windows 10，确保您使用Internet Explorer而不是Microsoft Edge尝试这些步骤。 图标看起来很相似。要在Windows 10中查找IE，按下键盘上的Windows键+ S，然后输入“Internet Explorer”。

检查您的Windows系统时钟

这听起来很奇怪，但实际上由于Windows系统时钟被设置为错误的时间或日期而导致各种问题。 如果这就是原因，那么简单地纠正所有的错误就能解决问题。虽然这可以手动完成，但您也可以将其设置为使用服务器自动更新，从而确保它不会落后并再次引发问题。 以下是如何做到这一点：

Windows 8.1和更早版本

1. 右键点击右下角的Windows时钟

2. 点击 调整日期/时间

3. 选择 互联网时间 选项卡

4. 在这里，您应该能够将您的时间和日期与互联网服务器进行同步

Windows 10

1. 右键点击右下角的Windows时钟

2. 确保“自动设置时间”已开启，并且您已选择了正确的时区

检查您的代理设置

如果上述步骤无法解决您的问题，那么您可能需要确保您的电脑没有尝试通过代理服务器进行 连接。 请按照以下步骤：

Windows 7或更早版本

  1. 转到“开始”菜单
  2. 搜索“proxy，”它将显示一个”更改代理设置“。点击它
  3. 点击 局域网设置 ，并取消选中该选项卡中的所有复选标记
  4. 点击 确定

Windows 8或8.1

  1. 点击屏幕左下角进入平铺开始屏幕
  2. 输入“PROXY”它将显示一个”更改代理设置“选项。点击它
  3. 点击 局域网设置 ，并取消该选项卡中的所有复选标记
  4. 点击 确定

Windows 10

  1. 按下键盘上的Windows键+ S。 或者，如果Cortana的“问我任何东西”栏在您的任务栏中是可使用，您可以点击它
  2. 输入“Proxy”，它将显示一个”更改代理设置“选项。点击它
  3. 确保“使用设置脚本”和“使用代理服务器”都已设置为“关闭”
  4. 向下滚动并点击 保存
