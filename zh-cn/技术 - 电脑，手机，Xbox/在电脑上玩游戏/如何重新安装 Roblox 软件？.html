如何重新安装 Roblox 软件？

通过重新安装 Roblox，可以解决某些技术问题。请确保在重新安装之前尝试重新启动。以下是重新安装的方法：

Windows 用户：

重置 Windows 的“Internet Options（Internet 选项）”

重置你的“Internet Options（Internet 选项）”有助于防止它们与 Roblox 不同步。如果这解决了你的问题，这就是你需要做的。虽然这些步骤使用的是 Internet Explorer，但是它们可以解决其他浏览器（例如 Chrome 或 Firefox）中出现的很多问题。方法如下：

    1. 打开 Internet Explorer
    2. 打开齿轮图标 > “Internet Options（Internet 选项）”
    3. 打开 “Advanced（高级）” 选项卡
    4. 单击 “Reset（重置）” 按钮
    5. 关闭 Internet Explorer 浏览器
    6. 尝试使用你最喜欢的网络浏览器来使用 Roblox。

重新安装 Roblox

如果重置你的“Internet Options（Internet 选项）”没有帮助，请尝试重新安装 Roblox。重新安装会覆盖并替换所有已删除或损坏的文件。执行重新安装时，请确保 所有 其他程序都已关闭。

    • 适用于 Chrome 或 Edge 的重新安装方法
    • 适用于其他浏览器的重新安装方法

卸载 Roblox

大多数情况下，在重新安装之前，你不需要卸载该程序。如果重新安装不起作用，则可能需要先卸载，然后再重新安装。遵循 适用于 Windows 的卸载 Roblox 说明。 请确保在卸载后重新启动计算机，然后再重新安装。

清除你的临时 Internet 文件

任何卸载或重新安装过程中的一个重要的步骤是清除浏览器的临时 Internet 文件。这将有助于解决可能出现的任何损坏问题。遵循 清除你的临时 Internet 文件和 Cookie 一节的说明。完成后，重复上述重新安装说明。

更改你的防火墙/安全程序设置

很多反病毒或 Internet 安全软件都试图控制哪些程序可以访问 Internet。这些程序出错是出于谨慎，默认情况下不允许任何程序访问。阅读 “防火墙和路由器问题”帮助文章 中的信息，以获得有关允许 Roblox 的帮助。重新配置防火墙后，重复重新安装步骤。

Mac 用户：

卸载

确保你完全关闭了 Roblox 程序。遵循 适用于 Mac 的卸载 Roblox 说明

重新安装

打开 Safari，登录 Roblox 玩游戏。系统将提示你安装该应用程序。请按照屏幕上的说明进行操作。如果安装不是自动进行，请执行以下操作：

    1. 转到 Finder > “Today（今天）” 并单击 Roblox.dmg - 这将为你运行安装程序，并将 Roblox 图标放在你的 App 中
    2. 从你的 App 中选择该程序，该程序应该会进入你的任务栏
