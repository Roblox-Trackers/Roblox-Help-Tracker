如何卸载 Roblox

如果你在 Roblox App 或 Studio 上遇到技术问题，有时先卸载再重新安装 Roblox 可能会有所帮助。 本文将指导你完成卸载过程。

Windows 说明

卸载该程序

Windows 7

  • 转到“开始”菜单 >“控制面板”> “程序和功能”
  • 向下滚动，直到看到 Roblox
  • 在那里卸载该程序

Windows 8 or 8.1

  • 单击屏幕左下角，调出平铺的开始屏幕
  • 输入“程序和功能”，然后单击出现在搜索结果右侧的 “程序和功能”图标
  • 在弹出的窗口中，向下滚动，直到看到 Roblox
  • 在那里卸载该程序

Windows 10

  • 单击屏幕左下角的开始按钮或 Cortana 搜索框
    • 或者，你也可以按住键盘上的 Windows 键，然后按下 S 键
  • 在搜索框中，输入“程序和功能”，然后单击出现在搜索列表顶部的 “程序和功能”图标
  • 在弹出的窗口中，向下滚动，直到看到 Roblox
  • 在那里卸载该程序

删除Roblox文件夹

如果你遇到的问题特别持久，则在卸载后完全删除 Roblox 文件夹可能会有所帮助，这样可确保没有任何损坏或过期的文件。 你可以按照以下步骤进行操作：

  1. 右键单击开始按钮，然后单击 Explore 或 File Explorer
  2. 导航到此文件夹： C:\Users\(你的 Windows 用户名)\AppData\Local
  3. 删除 Roblox 文件夹

注意：除非你已手动选择将其保存在此文件夹中，否则这不应删除你可能已保存在 Studio 中的任何场景。

卸载时遇到问题

如果卸载该程序时遇到问题，请尝试以下方法：

清除你的临时 Internet 文件

任何卸载或重新安装过程中的一个重要的步骤是清除浏览器的临时 Internet 文件。 这将有助于解决可能出现的任何损坏问题。 有关此过程的帮助，请 单击此处 并查看 “清除你的临时 Internet 文件和 Cookie” 一节。

检查您的防火墙

很多反病毒或 Internet 安全软件都试图控制哪些程序可以访问 Internet。 这些程序出错通常是出于谨慎，默认情况下不允许任何程序访问。 有关查看此问题并允许 Roblox 通过任何阻碍的帮助，请 单击此处 。

重新安装Roblox

下一个可能性是重新安装。 这可能看起来有违直觉，但是如果问题是由于以前卸载失败导致的文件损坏，这样做会覆盖你计算机上的该程序的副本，并替换任何已删除或损坏的文件。 然后，你应该可以进入“开始”菜单并卸载该程序。 有关重新安装的帮助，请 单击此处 。

Mac 说明

完全关闭 Roblox 程序

  • 按 Command + Option + Esc 键打开任务管理提示
  • 确保你没有在其中看到 Roblox 或 Roblox Studio
  • 如果列出了任何一个程序，请选择 Roblox 或 Roblox Studio，然后单击“Force Quit（强制退出）”

转到 Finder >“应用程序”

  • 将 Roblox 图标拖进“回收站”
    • Finder 中的这个图标包含所有已安装的 Roblox 文件

如何删除“回收站”中的文件

  • 单击 Dock 中的垃圾箱图标并选择清空垃圾箱。
  • 或者，连续按以下三个键清空回收站：Command + Shift + Delete。
  • 你应该会看到警告：“是否确实要删除回收站中的项目？”， 原因是你无法撤消此操作 ，但假设你确实要删除，一旦你单击“清空回收站”，你就应该删除了这些项目。


若要绕过“是否确实要永久删除回收站中的项目？”信息，请按以下键：

  • Command + Option/Alt + Shift + Delete
