电脑上的图形问题

如果您遇到 Roblox 的图形问题，请阅读以下信息。


图形问题的常见问题

  • 尝试启动游戏时显卡“图形太旧”错误
  • 在游戏过程中 Roblox 崩溃并报告图形错误
  • 接收信息：“图形初始化失败”
  • 在玩的时候，所有游戏看起来很不寻常，例如看起来奇怪的纹理，扭曲的图像或者难以理解的文字

为什么会发生

  • 电脑不支持 DirectX10 或更高的特性级别
  • 显卡没有最新的驱动程序
  • Roblox 不支持显卡（很少出现这种情况）
  • 玩家已更改 Roblox 图形设置

该怎么做

试试下面的方，一步一步来。 看看问题是否仍然存在。 如果是，继续前进。

重置 Roblox 图形

这仅适用于您可以在不崩溃的情况下真正启动 Roblox

  1. 打开 Roblox Studio
  2. 打开 “文件” 菜单，然后点击 设置
  3. 点击 OK 以进行
  4. 点击位于弹出窗口左下方的 重置所有设置 按钮。

检测支持的 DirectX 特性级别

检查你所支持的 DirectX 特性级别：

  1. 按下 Windows 键+ R 弹出运行提示符。
  2. 输入 dxdiag，然后按下键盘的 Enter 键
  3. 在 DirectX 窗口的面板视图中单击 “Display”或“Display 1”，如果可用，单击 “Display 2” 选项卡
    • 对于 Windows 8.1/10，寻找特性级别。对于 Windows 7 查找 DDI 版本。

如果你在 “DDI 版本” 或“功能级别”中没有看到 10 或以上，说明你的显卡不支持 DX10 或以上，你将无法访问应用程序。

降低显示器的色彩质量

您的显卡可能无法支持用于普通桌面操作的颜色质量的 3D 渲染。 您可以尝试将位深度降低到16位，以查看是否有帮助。

升级您的显卡驱动程序

要了解 Roblox 目前使用的显卡，请尝试以下操作：

  1. 打开 Roblox Studio
  2. 打开 Studio 设置
  3. 点击 诊断
  4. 向下滚动查找标记为 GfxCard 的行
  5. 这将列出 Roblox 当前使用的显卡

请访问显卡制造商提供的支持页面，并下载特定型号的最新驱动程序。

检查兼容模式

如果您收到一条消息，指出您的图形已过期，但您的计算机通过了最低规格要求，您可以检查应用程序是否在兼容模式下运行。

    1. 右键单击 Roblox 或 Roblox Studio 应用程序，然后选择属性。
    2. 选择兼容性选项卡。
    3. 如果选中以兼容模式运行此程序框，请取消选中它。

请访问您的显卡制造商提供的支持页面，并为您的特定型号下载最新的驱动程序。
