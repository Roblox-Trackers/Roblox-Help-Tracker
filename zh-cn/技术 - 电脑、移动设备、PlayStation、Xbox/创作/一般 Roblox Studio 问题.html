一般 Roblox Studio 问题

这文章提出一些有效方法解决以下问题：

  • 你尝试登录 Roblox Studio 但无法启动，
  • 你试着编辑一个地方但它加载了一个空白的 Skybox 或叫做 Place1
  • Studio 的一部分不应该是空的。
  • 在 Roblox 上发布时遇到了问题
  • 当你已经在线，Roblox Studio 还是要求登录
  • 这些步骤能助于解决一些与非 Studio 相关的问题，列如玩家已登录，也可以作为访客进入游戏
  • 其他：如果你遇到与 Studio 额外问题，则以下步骤也可能有助于解决问题

怎麼辦 - Windows

请使用按顺序尝试下面的每个步骤。 如果一个解决方案无效，请继续下一步！

重啟

首先重新启动电脑并尝试运作。 有时这会解决 Roblox 或 Roblox Studio 相关问题。

檢查您的代理設定

如果上述步骤不能解决你的 Roblox Studio 问题，你需要确保电脑不是在尝试通过代理服务器进行连接。 请按照以下步骤操作：

Windows 7 或更早版本

    1. 转到 Start “开始”菜单
    2. 搜索“代理”和“配置代理服务器”将会出现，并点击它
    3. 单击 LAN“局域网”设置 ，然后取消选中该选项卡中的所有复选标记
    4. 单击 OK

Windows 8 或 8.1

    1. 单击屏幕的左下角转到平铺的开始屏幕
    2. 输入 Proxy “代理”然后“配置代理服务器”将会出现，并单击它
    3. 单击 LAN“局域网”设置 ，然后取消选中该选项卡中的所有复选标记
    4. 单击 OK

Windows 10

    1. 按下键盘上的 Windows 键+ S 键。 或者，如果 Cortana 的“问我任何事情”栏在您的任务栏中可用， 您可以单击该栏
    2. 搜索“代理”然后“配置代理服务器”将会出现，并点击它
    3. 确保“使用安装脚本”和“使用代理服务器”设置为关闭
    4. 向下滚动并点击 保存

檢查您的 Windows 系統時鐘

这听起来很奇怪，但实际上各种问题是由于 Windows 系统时钟的时间或日期被设置成错误而导致的。 如果这是原因，那么纠正一切就应该解决情况。 虽然可以手动完成它，但也可以将其设置为使用服务器自动更新，从而确保它不会滑落并再次引发问题。 以下是如何做到这一点：

Windows 8 或更早版本

    1. 在这里，您应该能够将您的时间和日期与互联网服务器进行同步
    2. 单击 调整日期/时间
    3. 选择 互联网时间 选项卡
    4. 在这里，您应该能够将您的时间和日期与互联网服务器进行同步

Windows 10

    1. 右键单击右下角的窗口时钟
    2. 确保“自动设置时间”已打开，并且您已选择了正确的时区
