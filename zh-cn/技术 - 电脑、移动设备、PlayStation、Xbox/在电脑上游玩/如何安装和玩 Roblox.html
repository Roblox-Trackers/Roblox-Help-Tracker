如何安装和玩 Roblox

如何玩 Roblox

  1. 安装 Roblox 播放器
    • 使用任何现代浏览器（例如 Microsoft Edge、Firefox 或 Chrome）访问 Roblox 网站
    • 登录 Roblox 后，访问任何体验并单击绿色的“播放”按钮
    • 将出现一个弹出窗口通知您 Roblox Player 正在安装
  2. 通过任何现代浏览器（例如 Microsoft Edge、Firefox 或 Chrome）启动播放器
    • 前往 Roblox 网站上的任何游戏
    • 选择 播放
    • 可能会第二个弹出窗口，要求您选择一个程序。 选择 Roblox 并确认。 如果您选择记住您的选择，则以后无需执行此操作

如何在 MacOS 上通过 Firefox 安装和运行 Roblox

1. 安装 Roblox 播放器

  • 使用 Firefox 访问 Roblox 网站
  • 登录 Roblox 后，访问任何作品并单击绿色的 播放 按钮
  • 将出现一个弹出窗口，通知你 Roblox Player 正在安装

2. 启动播放器

  • 选中 始终使用此应用程序打开 roblox-player 链接 ，然后单击 选择 以调出 Finder 窗口
  • 在 Finder 窗口中，使用键盘快捷键 CMD+SHIFT+G 调出路径名输入
  • 在路径名输入，输入 /Applications/Roblox.app/Contents/MacOS/Roblox.app ，然后按 RETURN
  • 然后在 Finder 窗口中，选择 Roblox.app，然后单击 打开
  • 在选择应用程序对话框中，选择 Roblox.app 并单击 打开链接
  • Roblox 应该正在运行，并且单击播放按钮打开 Roblox 作品应该在未来工作。

如何安装 Roblox Studio

您需要安装 Roblox Studio 才能创建您自己的 Roblox 体验。 以下是您可以操作的方法。

  1. 访问 Roblox 网站 并登录您的帐户
  2. 登录 Roblox 后，访问任何 您的 体验，点击右上角的 ，然后选择 编辑
  3. 将出现一个弹出窗口，通知您 Roblox Studio 正在安装
  4. 安装后，Studio 将自动打开
