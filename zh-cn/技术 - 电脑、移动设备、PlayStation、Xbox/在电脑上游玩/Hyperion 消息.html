Hyperion 消息

本文包含可能出现的 Hyperion 反作弊崩溃消息，以及帮助您解决问题的相应故障排除步骤。

崩溃类型 ：超时故障

用户消息 ：Roblox 已超时。 请重启您的客户端。

故障排除步骤 ：

如果遇到消息“Roblox 已超时。 请重启您的客户端”，即表示您的 Roblox 客户端需要重启。 在正常使用期间，如果重启客户端后问题仍然存在，请 联系我们的支持团队 寻求进一步帮助。

崩溃类型 ：检测到虚拟机

用户消息 ：Roblox 无法在虚拟机中运行。

故障排除步骤 ：

如果看到消息“检测到虚拟机”，即表示 Roblox 不支持您的虚拟机 (VM)。 Roblox 需要 GPU 直通才能在 VM 中正常运行。 GPU 直通允许 VM 绕过虚拟机管理程序层直接访问物理 GPU。 以下是启用 GPU 直通的方法：

      1. 确认您的硬件支持 GPU 直通。
      2. 更新 BIOS/UEFI 设置以启用虚拟化扩展（例如 Intel VT-x 或 AMD-V）。
      3. 在 VM 软件中配置 GPU 直通。

遵循这些步骤可确保 Roblox 在虚拟化环境中流畅运行，从而实现对 GPU 资源的必要访问。

崩溃类型 ：检测到流氓软件

用户消息 ：软件不兼容导致 Roblox 崩溃：<违规软件的路径>

故障排除步骤 ：

如果看到消息“软件不兼容导致 Roblox 崩溃：”，并且后面跟着违规软件的路径，即表示显示的文件路径中提到的软件正在导致 Roblox 出现问题。 要解决此问题，您需要阻止该软件在使用 Roblox 时运行。 您可以手动关闭软件，或在 Windows 中执行干净启动。 此方法可确保系统上只运行必要的软件，最大限度地减少冲突。 以下是 Windows 推荐的干净启动方法 ：

      1. 按“Windows 键 + R”，打开“运行”对话框。
      2. 输入 "msconfig"，然后按“回车”键。
      3. 在“系统配置”窗口中，转到“服务”选项卡。
      4. 选中“隐藏所有 Microsoft 服务”复选框。
      5. 单击“全部禁用”。
      6. 切换到“启动”选项卡。
      7. 单击“打开任务管理器”。
      8. 禁用所有启动项。
      9. 关闭“任务管理器”和“系统配置”。
      10. 重启电脑。

完成这些步骤将有助于确保没有其他软件干扰 Roblox，使其能够流畅运行而不会崩溃。

      5. 单击“全部禁用”。
      6. 切换到“启动”选项卡。
      7. 单击“打开任务管理器”。
      8. 禁用所有启动项。
      9. 关闭“任务管理器”和“系统配置”。
      10. 重启电脑。

完成这些步骤将有助于确保没有其他软件干扰 Roblox，使其能够流畅运行而不会崩溃。

崩溃类型 ：启动完整性受损

用户消息 ：关键系统文件已损坏。 Roblox 无法运行。 请修复或重置您的 Windows 安装。

故障排除步骤 ：

当 Windows 设置为调试模式时，Roblox 无法正常运行。 要使用 Roblox，请确保 Windows 正常运行。 按照以下步骤验证并禁用 Windows 的调试模式 。

      1. 按“Windows 键 + R”，打开“运行”对话框。
      2. 输入 "msconfig"，然后按“回车”键。
      3. 选择“启动”选项卡，然后选择“高级”选项。
      4. 取消选中“调试”复选框。
      5. 选择“确定”。
      6. 选择“应用”，然后选择“确定”。
      7. 重启电脑。

这些步骤将确保 Windows 不会在调试模式下运行，让您可以顺利使用 Roblox。

崩溃类型 ：(OBS) Open Broadcaster Software

用户消息 ：已知 OBS 游戏内捕获与 Roblox 之间存在问题。 请改用窗口或桌面捕获。

故障排除步骤 ：

如果收到消息“已知 OBS 游戏内捕获与 Roblox 之间存在问题”，即表示 Roblox 不支持 Open Broadcaster Software (OBS) 的游戏内捕获功能。 要串流或录制 Roblox，需要改用窗口或桌面捕获模式。 按照以下说明设置桌面捕获 ：

      1. 打开 Open Broadcaster Software (OBS)。
      2. 在“来源”部分，单击“+”图标。
      3. 从菜单中选择“显示捕获”。
      4. 选择要捕获的显示器（通常是主显示器）。
      5. 根据需要调整分辨率和帧率等设置。
      6. 单击“确定”确认显示器捕获源。
      7. 根据需要在 OBS 场景中排列显示器捕获源。
      8. 像往常一样开始串流或录制。

通过使用桌面捕获模式，可以高效串流或录制 Roblox，避免了 OBS 游戏内捕获功能的兼容性问题。
