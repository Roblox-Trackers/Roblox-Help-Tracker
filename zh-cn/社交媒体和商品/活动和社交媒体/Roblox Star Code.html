Roblox Star Code

Roblox 致力于为我们的 Video Star 项目 中有影响力的人提供业务和收入增长。 Star Code（明星代码）使它很容易支持你最喜欢的视频明星，而不需要额外的费用。

什么是 Roblox Star Code ？

Roblox Star Code 是一种特殊的代码，你可以从 Roblox Video Star 项目的成员那里获得此代码。 当你使用 Star Code 来支持视频明星时，他们将自动获得你在 Roblox 网站上购买 Robux 的 5%。 你仍然会收到你所支付的全额 Robux，也知道你已经贡献给你最喜欢的视频明星！

注意：Star Code 可以用于 Roblox 网站或 Roblox App 上的购买。

如何使用 Star Code

在完成交易时，你可以在 Robux 购买页面上输入与你最喜欢的 Roblox Video Star 相关联的明星代码。 Video Star 将在他们的 Roblox 个人资料或者群组页面上推广他们的明星代码，以及他们的社交媒体账户，包括 YouTube、Twitter 等等！

通过网络浏览器

  • 前往 Robux 或 会员资格 页面并选择你要购买的价格。 然后你将被重定向到付款方式选择页面。
  • 准备好你最喜欢的 Roblox Video Star 代码，然后按 单击此处 打开代码输入窗口。 输入您的代码并 添加 。
  • 接受代码后，页面将显示你支持的人。

通过 App

  • 随身携带你喜欢的 Roblox Video Star 代码
  • 前往 会员资格 页面并滚动到页面末尾附近以找到星号代码字段。 按 点按此处 打开代码输入窗口。
  • 输入您的代码并按 添加 。
  • 当代码被接受后，页面将显示你支持的人。

一旦你添加了你的代码，它将保存 2 周，这样你就不必每次购物时重新输入代码。
