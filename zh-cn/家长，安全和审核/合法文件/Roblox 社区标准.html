Roblox 社区标准

Roblox 旨在通过积极向上和文明守礼将十亿人连结在一起。 每天，全世界都有数以千万计的人们来到这里探索数百万种的沉浸式数字体验，与朋友一起享受乐趣。 所有的体验都有 Roblox 的数百万创作者们创作打造。 我们坚信，建立一个安全、文明并多元的社区的重要性。 这个社区能够激发并培养创作力，并让全世界的人们通过积极的沟通更紧密的联系在一起。

我们制定了这些社区标准，希望可以确保每个人都能在我们共享的探险体验中感到安全可靠、宾至如归。 这些标准适用于您在 Roblox 上及与其他 Roblox 用户互动的所有行为，包括线上活动以及由 Roblox 举办的现实世界中的活动。 针对 17+ 的体验，请查看 Roblox 17+ 政策标准 。

我们要求您一直尊重我们的社区标准并尊重彼此；并理解如果您违反了这些标准，我们可能会对您的帐户和内容采取强制措施。 如果有违反这些标准的行为，例如在现实世界中对他人构成威胁，我们保留与相关当局联系或合作以确保所有人安全无虞的权利。

我们还在不断学习，因此，随着我们社区的发展和平台的成长，这些标准也会发生相应地变化。 如果有任何变更，我们会 在此处 通知您。 如果您看到了您认为已经违反社区标准，或在其他方面让您感到不安全的内容，请使用 有关适用于您使用 Roblox 的法律条款的更多信息，请参阅 使用条款 。

我们感谢您遵守这些标准并承诺确保 Roblox 的安全和文明。

安全

无论用户的身份或信仰如何，我们希望我们的社区成为一个人们可以进行积极向上并文明沟通的地方。 为了确保 Roblox 做到让每个人都能舒适地做自己，我们禁止以下行为：

危害儿童
Roblox 绝不容忍危害未成年人的行为，包括：
  • 掠夺性行为，包括试图与未成年人交朋友以操纵和利用他们（即性诱拐）
  • 以任何方式让未成年人呈现性感
  • 与未成年人进行具有性暗示的对话，或向未成年人索取色情材料
  • 分享、请求或讨论儿童性剥削材料
恐怖主义和暴力极端主义
Roblox 绝不容忍煽动、纵容、支持、美化或宣传恐怖主义或极端主义组织或个人（外国或国内）及其意识形态或行为的内容或行为，包括：
  • 描述或支持恐怖主义或极端主义袭击
  • 描述或支持恐怖组织或极端主义组织的领导人或代表
  • 分享恐怖主义或极端主义组织的口号、图像、旗帜、宣言或图标，无论是全部还是易于识别的部分
  • 提及恐怖主义和极端主义组织的意识形态、信息或策略
  • 表达支持、纵容或美化恐怖主义或极端主义意识形态或行为
  • 为恐怖主义或极端主义组织招募成员，或鼓励他人前往 Roblox 之外的平台以查找此类信息
  • 为恐怖主义或极端主义组织、人员或支持团体筹款
  • 表示支持、纵容或美化大规模枪击事件，以及其他国内恐怖主义或暴力极端主义行为
威胁、欺凌和骚扰
Roblox 不允许使用现实世界或在线上伤害并威胁他人、煽动对人或财产的暴力、欺凌、跟踪、挑衅、骚扰、恐吓、勒索和敲诈。 同时，我们不允许描述、美化或提倡此类行为的内容，包括：
  • 威胁要进行身体或性侵犯或暴力
  • 威胁要在现实生活中伤害某人
  • 威胁要泄露他人的个人信息
  • 威胁要窃取他人的 Roblox 帐户或提交针对其他用户的虚假滥用报告
  • 挑出一个用户或群体进行公开或私下地嘲笑或辱骂
  • 性骚扰
  • 在网上或现实生活中冒充个人、团体或实体，以损害他们的声誉或导致他人伤害他们
自杀、自伤和伤害行为
Roblox 非常重视用户的福祉。 我们在这里为那些与精神健康作斗争的人提供资源，我们保留就可信的自残威胁与当局沟通的权利。 我们不允许描述、美化或鼓励自杀、自伤或可能造成极度人身伤害风险的现实世界活动的内容或行为，包括：
  • 描述自杀或自残的方法
  • 美化、支持或描绘自杀或自残实例或方法的内容
  • 现实世界中危险的身体挑战或特技

如果您或您所爱的人有自杀或自残的念头，可以寻求帮助。 请致电自杀预防热线，例如 此处 。 如需额外国际资源，请参阅 此处 。
歧视、诽谤和仇恨言论
Roblox 尊重并欢迎来自不同年龄、背景和身份的用户。 我们不允许支持、美化或宣扬仇恨团体及其意识形态或行为的内容或行为。 您也不得基于以下原因歧视、嘲笑或助长对个人或团体的仇恨，或鼓励他人直接或间接这样做：
  • 年龄
  • 种族、认知的种族或民族
  • 国籍
  • 性取向
  • 性别、性别认同或性别表达
  • 宗教或宗教信仰或信仰
  • 残疾状况，包括疾病、身体状况、毁容、行动不便和精神障碍
  • 身体或精神方面的残疾状态
  • 退伍军人身份
  • 投掷
  • 家庭状况
有害的平台外言论或行为
Roblox 用户需尊重平台上或平台外的所有人。 如果向我们举报的可信度高，我们可能会对下列用户采取行动：
  • 与有组织犯罪、恐怖分子或极端主义组织或仇恨团体有关联
  • 在 Roblox 平台外骚扰、欺凌、歧视或伤害他人
  • 在平台外分享他人的个人信息，包括向当局做出虚假报告
  • 曾因某些恶劣罪行而定罪，从事某些非法活动，或煽动他人犯罪
文明

文明是 Roblox 社区的核心。 为了促进我们的文明环境，Roblox 禁止某些类型的内容，包括：

现实世界敏感事件
Roblox 禁止以下内容：再现特定现实世界敏感事件；嘲弄这些事件的受害者；支持、美化或宣传这些事件的肇事者或结果；或利用这些事件达到商业目的，包括以下方面的内容：
  • 针对人员或财产的大规模暴力行为
  • 现实世界中特定的自然灾害事件
  • 侵犯人权或公民权利
暴力内容和血腥
尽管 Roblox 上的某些体验可能包含武器和暴力，但我们不允许包含极端暴力或严重虐待身体或心理的内容，包括：
  • 虐待和折磨动物
  • 对极端血腥、血腥暴力或死亡的实际或现实世界描述
浪漫和色情内容
Roblox 是结识在线朋友、聊天和合作创意项目的安全空间，但我们禁止描绘性活动或寻求在现实世界建立浪漫关系的内容。 这包括：
  • 浪漫或调情的姿态，或用户间在浪漫情境中交流
  • 在网上追求或引导浪漫关系
  • 做出不受欢迎的调情行为
  • 参与露骨的色情对话，或向其他 Roblox 用户索取色情材料
  • 描绘、强烈暗示或明确描述性行为的内容
  • 裸体
  • 性暗示的虚拟形象服装物品
非法和受管制的商品和活动
我们禁止用户讨论、描述或宣传非法和某些管制商品。 你也不得讨论在现实世界中从事非法活动，或鼓励他人从事此类行为，或在平台上从事或宣传这些活动。 这些商品和活动包括：
  • 非法和管制药物，包括大麻和滥用处方药或非处方药
  • 毒品、烟草或其他吸烟用具
  • 膳食补充剂和增强剂，例如减肥药和类固醇
  • 描述与饮酒或吸毒相关的沉醉行为
  • 现实世界中的炸弹/爆炸物和武器制造说明或原理图
  • 体验项目之外的仿真现代枪支
  • 提供 Robux 或其他任何价值品作为奖品的平台内竞赛
  • 抽奖式游戏

除非受当地法律或规定禁止，我们允许包括不可玩赌博内容（例如显示赌场或非可玩角色打牌）以及非赌场或赌博类的游戏中的运气和机会元素。 但是，我们不允许包含模拟赌博的体验，包括使用虚拟筹码进行游戏、模拟投注或交换真钱、Robux 或具有价值的游戏内道具。

如果您或您认识的某人因为正与药物滥用作斗争，请寻求帮助。 你可以从 此处 开始。

污言秽语
为了确保每个人都能在 Roblox 感到安全和文明，我们不允许以任意语言文本、图像或上传音频等方式污言秽语，包括：
  • 使用错误的拼写、特殊字符或其他方式以规避对污言秽语内容的检测
  • 引用或使用包含脏话歌词的音频
政治内容
关于 Roblox 用户有关的议题和主题，我们重视友好辩论。 但是，为了保持文明和尊重的环境，我们禁止讨论或描述某些政治内容，包括：
  • 当前竞选公职的候选人，包括他们的口号、竞选材料、集会或活动
  • 政党，包括官方政党附属组织
  • 当选职位的特定种族
  • 现实世界中现任的当选官员
  • 最近、之前当选的具有官方身份的官员
  • 以往以候选人身份竞选过政治职位的个人
  • 亵渎政治实体符号，包括焚烧国旗
  • 与现实世界边界、领土或管辖关系相关的煽动性内容

作弊和诈骗

  • 出售具有误导性或描述不准确的道具

垃圾信息

广告
独立广告发布 开发者可以在自己的体验中独立发布广告内容，而不是使用 Roblox 沉浸式广告。 在这些情况下，体验的开发者必须直接上传和控制广告内容的投放，并对此类内容负责。 开发者独立发布广告时，必须遵守以下规则，以及所有 Roblox 规则（包括使用条款、 广告标准，以及所有其他社区标准）和适用法律及规定。 如果你的广告内容违反了这些规则，我们可能会采取行动，包括删除内容、暂停体验或暂停帐户。
  • 开发者必须始终全权掌控独立广告的发布。 如果开发者在其体验中展示独立广告，该体验则可能不会以程序化方式调用第三方广告服务。
  • 开发者可能无法分享与广告商分享用户个人识别互动信息；在处理所有用户信息时，也须遵守 创作者分析使用条款 及 Roblox 使用条款 。
  • 禁止开发者向 13 岁以下的用户展示广告内容。 为了达成此目标，开发者必须使用 Roblox 工具，例如 PolicyService API 中的 AreAdsAllowed boolean，以向不允许看到广告的用户替换、隐藏、或屏蔽广告内容。
    请注意：应用 13+ 的体验指南将不会向 13 岁以下的用户隐藏广告内容。 所有广告内容须使用 PolicyService API 隐藏。
  • 当内容是广告时，开发者必须使用简单易懂的语言清晰明了地向用户披露。 单独提及广告商的名字是不充分的。 可接受的披露语言包括：“广告”、“付费”或“赞助”。 披露应出现在广告内容当中或附近，并以用户可能注意并理解的方式呈现。
  • 开发者可能不会误导用户，让其认为他们和 Roblox 提供的广告有关。
  • 开发者不得重复使用或改造 Roblox 沉浸式广告内容。
  • 广告不能虚假陈述所宣传的内容，以鼓励用户参与（例如，提供免费的 Robux）。

有关如何独立广告的更多信息，请参阅 常见问题 。

Roblox 经济
Roblox 维护 Robux 系统，而让所有人都能受益并享受乐趣。 为了公平起见，Roblox 经济系统中禁止进行某些活动，包括：
  • 使用第三方服务购买、销售、交易或赠送 Robux。 这不适用于 Roblox 礼品卡。
  • 参与平台外的二级市场交易，以获取帐户访问权限或从体验或商城中获取虚拟内容。 这不适用于帐户持有人在可验证的协议中出售或转让帐户创作的权利。
  • 以欺诈或误导的方式使用团体支付系统；包括在 Robux 中提供付款以充当资产模型、加入群组、推荐成员或在体验中扮演角色。
促销活动 除非获得 Roblox 明确授权，或与 Roblox 进行官方合作，开发者不得使用平台外服务或产品，直接或以捆绑方式销售平台上的项目（包括独家功能或其他体验增强）以换取金钱或 Robux。 有关可接受的促销规范详情，请访问 此处 。
诱导募捐 我们也不允许我们的社区在 Roblox 上诱导慈善事业募捐，包括：
  • 体验中提示用户为众筹活动捐赠 Robux
  • 声称交易与筹款活动有关
  • 向用户说明一定数量的销售收入将用于某项事业

注意：开发者可以为用户实施体验中的 Robux 交易，以表达对体验的欣赏或慷慨奖励创作者（例如“打赏罐”功能），前提是用户清楚他们的付款不会得到回报。
付费随机道具 付费随机道具生成器包括直接使用 Robux 购买，或使用 Robux 购买的体验内货币而购买的生成器。
  • 如果你的体验中包含付费随机道具，用户不得交易或出售由此产生的资产以换取 Robux、真实货币或 Roblox 上的其他有价之物。
  • 创作者须使用适用的 PolicyService API 来确保付费随机道具仅对符合条件的用户开放。
  • 必须在购买前向符合资格的用户准确披露收到每个可能获取的道具几率。
安全

为了让我们的社区蓬勃发展，每个人都应该对自己的数字身份和现实身份感到安全。 为了实现这一点，我们应当尊重彼此的个人信息和 Roblox 的安全系统，并保护 Roblox 帐户的安全。 本节介绍玩家和创作者应如何对待他人的信息并保护自己和他人的数字安全。

关于 Roblox 如何处理你个人信息的更多内容，请访问 隐私政策 。 如需更多关于确保你的帐户安全性的信息，包括如何启用额外的安全功能（例如两步验证），请前往 此处 访问我们的帮助页面。

为了维护你的安全的元宇宙环境，Roblox 不允许：

分享个人信息
为了维护隐私和安全，用户可能被禁止在 Roblox 上分享个人或请求他人的个人信息和唯一标识符，包括：
  • 电子邮件地址
  • 密码或访问令牌
  • 家庭地址或实体地址
  • 财务信息，包括信用卡和银行账户详细信息
  • 医疗信息
  • 电话号码
  • 未链接到 Roblox 的互联网标识符，例如用户名
  • 唯一标识符，例如社会保障号码
  • 本人或其他非公众人物的照片
  • 真实姓名，不包括针对符合资格用户所允许的昵称
  • 与未成年人有关的任何个人身份信息，包括未经授权的图像或语音及视频录像。
引导用户离开平台
使用 Roblox 时，你不能使用前往外部网站或服务的链接，以下平台除外：YouTube、Facebook、Discord、Twitter 和 Twitch。 链接至已批准的网站须包括在 适用的 PolicyService API 中，或在你的个人资料或体验描述页面的社交网络及社交链接设置中列出。 若链接（包括指向经认可网站的链接）出现在无法部署 API 的位置（例如文本聊天或媒体嵌入），则予以禁止。 禁止其他链接或试图逃避链接检测的行为，包括：
  • 包含 Roblox 不当内容的链接
