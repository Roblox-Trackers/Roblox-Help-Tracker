Roblox 生物识别隐私通知

您为什么会收到此通知？

本生物识别隐私声明解释了 Roblox 在您使用我们的服务时如何收集、使用、共享、保留和销毁（统称为“处理”）您的生物识别数据。 “生物特征数据”包括：(i) “生物特征标识符”，这是通过测量您的生物特征而生成的数据，例如您的视网膜或虹膜扫描、指纹、声纹或您的手部或脸部几何形状的扫描； (ii) “生物特征信息”，这是基于可用于识别您身份的生物特征标识符的信息。

请仔细阅读本通知。 使用 Roblox 身份验证流程，即表示您同意按照本通知所述收集、使用、共享和保留您的生物识别数据。

我们处理什么类型的生物识别数据？

  • 面部几何。

我们如何处理它？

  • 我们使用服务提供商 Veriff 和 Persona 来协助我们的身份和年龄验证过程。
  • 为确认您的身份，Veriff 或 Persona 会拍下您的 ID 和一张自拍照。 然后，Veriff 或 Persona 会检查从您的 ID 和自拍照中提取的面部几何形状，并分析您的面部年龄，将其与 ID 中的出生日期进行比较。 这样做是为了确保安全地验证您的年龄和身份，同时检测欺诈行为。
  • 出于合规性目的，部分 Roblox 人员可能会访问 Veriff 或 Persona 中存储的您的 ID 和自拍照片。

我们会保留您的生物特征数据多长时间？

  • Roblox 不会收到或访问您的生物识别数据的副本。
  • 我们的服务提供商 Veriff 或 Persona 将在收集后 90 天内分别删除您的生物识别数据，除非法律要求（例如搜查令或传票）将您的生物识别数据保留更长时间。

在保留期结束时，您的生物识别数据会怎样？

  • 我们的服务提供商 Veriff 和 Persona 都会在上述保留期结束时永久删除您的生物识别数据，使其无法恢复或重建。

额外的隐私披露和您的权利

有关我们如何处理您的个人信息的更多内容，以及您在我们的数据实践方面的权利和选择，请前往 https://www.roblox.com/info/privacy 查看我们的隐私政策。 有关 Veriff 数据实践的更多信息，请查看 https://www.veriff.com/privacy-policy 上的 Veriff 隐私政策。 有关 Persona 数据实践的更多信息，请查看 https://withpersona.com/legal/privacy-policy 上的 Persona 隐私政策。
