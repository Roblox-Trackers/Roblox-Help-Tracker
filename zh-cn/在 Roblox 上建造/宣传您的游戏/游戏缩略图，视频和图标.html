游戏缩略图，视频和图标

您可以使用缩略图和图标功能来帮助您的“场景/游戏”受到关注并脱颖而出。

什么是缩略图？

缩略图是在细节页面上的图像，用于代表您的游戏并可以用来帮助它脱颖而出。同时还可以向新玩家展示他们点击“播放”时的期望。 游戏最多可以有10个缩略图而每一个都可以是图像或视频。 游戏 发布 时将添加一个默认缩略图，但如果您愿意，您可以稍后更新它。

您如何添加新的缩略图或视频？

您的第一个缩略图将自动添加为默认图像。不过，您可以添加新的免费自定义缩略图或者YouTube视频为500个Robux。要做到这一点 ，请按照以下步骤操作：

  1. 使用您选择的图像或视频编辑器来创建图像或视频。
  2. 如果您正在制作一个视频，把它上传到您的YouTube账户上。
  3. 点击位于屏幕顶部蓝色栏中的 创作 。
  4. 找到您想要添加图库项目的位置，并点击右侧的“齿轮”。
  5. 点击 Configure Start Place 。
  6. 在显示的页面左侧栏中，点击 Thumbnails 。
  7. 选择是否要使用屏幕右侧的 Image, Video 或 Auto generated Image 。
  8. 如果您正在上传图像，点击 Choose file ，查找并选择您的图像文件，按 Open ，然后按蓝色的 Upload an Image 按钮。 然后将看到一个弹出框，确认您想购买缩略图。 按绿色的 Buy Now 按钮来完成 交易。
  9. 如果您正在链接YouTube视频，在该框中输入URL网址，然后点击蓝色的 Add Video 按钮。 然后您将 看到一个弹出框，确认您要购买视频缩略图。 按绿色的 Buy Now 按钮来完成交易。
  10. 如果您想使用 Auto generated Image ，那么此选项是免费的和您将在选择此选项时看到预览。按蓝色的 Set Thumbnail 按钮选择该图像。

注意：图像和视频必须符合 Roblox社区规则 。当上传的图像/视频正在等待我们的版主批准时 ，图像或视频将暂时不可用。 如果缩略图由于违反了这些规则而被删除或拒绝，那么所花费的Robux是不可退还 。

视频有什么限制（长度，内容等）？

视频最长可达30秒。如果您的视频超过这个长度，您将收到一个错误信息，并且不会上传。 有关其它 视频限制和认可的完整列表，请 点击这里 。

什么是游戏图标和您如何制作它们？

图标是正方形的图像，除了在详细页面以外也可以在某些地方用来代表您的游戏。 这包括“游戏”页面， 个人资料 页面，“收藏夹列表等。在默认情况下，游戏的图标将是我们在上一节中提到的游戏标准缩略图的裁剪版本。 也就是说，您可以使用您选择的图片编辑器创建自己的图像并免费上 传。 有关此过程的帮助以及如何创建引人注目的图标的些技巧，请 点击这里 。
