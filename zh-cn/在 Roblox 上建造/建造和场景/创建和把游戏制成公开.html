创建和把游戏制成公开

您可能会觉得，只开发一款游戏无法满足您对创意的渴望。幸好的是，我们可以协助！所有用户都可以随着自己的意思，构建尽可能更多的游戏，但是账户在任何给定时间，有限制200个公共游戏。 （那些可以由其他人播放）。

制作游戏为公开或私人：

制作公开或私人游戏方式有两种。

注意：让您的游戏保密将关闭所有正在运行的游戏并取消所有服务器。

第一方法：

  1. 点击网站顶部蓝色栏中的 创作
  2. 在 My Creation（ 我的创作）选项卡 中，如果游戏尚未突出显示，请点击Games （ 游戏 ）
  3. 找到您想要改为公共/私人的地方，再点击地方标题下方的灰色或绿色图标，在两种状态之间切换
    • Public 公共 （绿色图标）将允许任何用户查看和玩游戏
    • Private 私密（灰色图标）会阻止除您之外的任何人查看或玩游戏，除非是小组游戏，在这种情况下，只有您和指定的开发人员（具有必要的游戏编辑权限）才能够玩这游戏。有关更多信息，请参阅关于 群组游戏 的帮助文章。

第二方法：

  1. 转到您的游戏页面
  2. 点击三点图标来拉起游戏的设置菜单
  3. 选择 空间设定 或 游戏设定
  4. 无论您在上述步骤中选择哪一个，在屏幕上，如果尚未突出显示，请导航至 Basic Settings（ 基本设置） 选项卡
  5. 在 Privacy（ 隐私） 下，选择 Public（ 公共） 或 Private （ 私人 ）
    • 共 将允许任何用户查看和玩游戏
    • 私密 会阻止除您之外的任何人查看或玩游戏，除非是群组游戏，在这种情况下，只有您和指定的开发者（具有必要的游戏编辑权限）才能玩游戏。 有关更多信息，请参阅关于 Group Games 的帮助文章。

创建新游戏：

  1. 点击网站顶部蓝色栏中的 创作
  2. 如果它尚未明亮显示，请在My Creations （我的创作）选项卡中单击 Games ( 游戏 )
  3. 点击 Create New Game （ 创造新游戏）
  4. 为新创造选择 Settings（设置）和 Template（模板）
  5. 点击 Create Game （ 创作游戏 ）

这会带您回到您之前的创建页面。要的话，您可以现在把游戏造作公共，或者您可以等到您已做了一些工作。要开始创建，请点击游戏列表中新 游戏 条目右侧的 Edit（ 编辑） 按钮。

注意：您可以任何时创建一款新游戏， 但只直到使用“制作游戏为公共”流程将它改为公开之后，才能让所有 Roblox 玩家享用它。
