如何复制游戏

有时候您可能会想开始新的游戏，但是想编辑以前存在的游戏来节省一些时间。您也可能只想将其中一个游戏复制到另一个账户。 以下步骤将帮助您做这些操作。

如何复制您自己的游戏

  1. 运行Roblox Studio
  2. 在Studio中登录您的账户
  3. 点击 My Games
  4. 找到您想要复制的游戏，点击它，然后点击 Edit 位于屏幕底部
  5. 一旦您编辑游戏后，您可以用两种复制方式：
  • 如果您只想将备份保存到您的电脑上，请选择 file > Save As
  • 如果您想将游戏副本上传到Roblox，请选择 file > Publish to Roblox As ，然后通过现有游戏上传或创建一个新游戏。

注意：要将游戏复制到您的其他账户之一，请选择步骤5中的 Save As 方法。保存游戏后，登出Studio，然后重新登录为新账户。然后您可以使用 Publish to Roblox As 上传到该账户。

如何使用其人的复制允许游戏作为模板

您可能会看到一款您认为可以作为您自己游戏的良好初始点的游戏。好消息是，一些建设者鼓励您为此使用他们的游戏！ 但是，只有在游戏的 About 选项卡上列出了“可以复制此游戏的源代码”通知时才能复制游戏。请参阅我们的 帮助文章，了解如何允许或禁止复制游戏 以获取更多详细信息。

如果允许复制游戏，那么以下步骤将帮助您将其用作自己游戏的起点：

  1. 转到游戏的详细信息页面
  2. 点击标题右上角的三个点的图标
  3. 在弹出的菜单中单击 edit
  4. 这将在Studio中打开游戏，您可以执行以下两件事之一：
  • 选择 file > Save As ，然后将游戏保存到您的电脑上。 最后您可以根据自己的喜好编辑游戏。
  • 选择 file > Publish to Roblox As ，然后通过现有游戏上传或创建一个新游戏。

注意：在决定发布复制的游戏之前，请务必先征求创作者的许可。如果玩家的游戏不允许被复制，并且您想制作自己的版本，请务必首先向创建者请求许可。这是比较有礼貌的。
