如何添加或删除账户 PIN 码？

你可以选择在账户中添加一个四位数 PIN 码锁定你的设置。 一旦启用了 PIN 码，你需要输入它才能对账户”设置“作出任何更新。 此功能可保护你的密码、电子邮件地址及隐私设置。

如何添加 PIN 码？

  1. 登录你的账户
    （如果你无法登录，请尝试 找回密码 的 步骤 。 )
  2. 转到账户 ”设置"
    • 浏览器 - 查找位于网页右上角的 “齿轮” 图标
    • 移动 App - 查找三个点图标里的 “更多”
  3. 选择 "安全" 选项卡
  4. 选择开启 PIN 码
  5. 创建并确认你的新 PIN 码

绝对不要向任何人透露你的 PIN 码 - 它甚至比你的密码还重要。

如何删除账户PIN 码？

  1. 登录你的账户
    （如果你无法登录，请尝试 找回密码 的 步骤 。 )
  2. 转到账户 "设置"
    • 浏览器 - 查找位于网页右上角的 "齿轮" 图标
    • 移动 App - 查找三个点图标里的 "更多"
  3. 选择 "安全" 选项卡
  4. 选择关闭 PIN 码
  5. 输入当前 PIN 码并选择 "解锁"

如何更改 PIN 码？

若要更改 PIN 码，首先按照 如何移除 PIN 码 步骤操作。 然后，按照 如何添加 PIN 码 的步骤建立新的 PIN 码。
