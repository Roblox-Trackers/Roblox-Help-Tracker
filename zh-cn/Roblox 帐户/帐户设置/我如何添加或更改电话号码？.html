我如何添加或更改电话号码？

这个功能是为13岁以上的玩家提供的。 一个电话号码只能在 一个 Roblox账户上使用。

添加电话号码

  1. 登录您的账户
    (如果无法登录，请尝试这些步骤 恢复密码 .)
  2. 浏览到账户 设置
    • 浏览器 - 位于网站右上角的齿轮图标
    • 移动应用程序 - 三个点图标里的 ”更多“
  3. 选择 ”账户信息“ 选项卡
  4. 选择 ”添加手机“ 按钮
    1. 从下拉菜单中选择您的位置
    2. 输入电话号码和账户密码
    3. 选择 ”添加手机“

更改或删除电话号码

  1. 登录您的账户
    (如果无法登录，请尝试这些步骤 恢复密码. )
  2. 浏览到账户 设置
    • 浏览器 - 位于网站右上角的齿轮图标
    • 移动应用程序 - 三个点图标里的 ”更多“
  3. 选择 ”账户信息“ 选项卡
  4. 选择 ”更改手机“ 按钮(图标看起来像铅笔和纸)
    1. 从下拉菜单中选择您的位置
    2. 输入电话号码和账户密码
    3. 选择 “更改手机” 或 ”删除手机“

短信指令

下面是一些可用的指令，可以启动和停止您从Roblox 收到的短信:

  • STOP, STOPALL, UNSUBSCRIBE, CANCEL, END, 和 QUIT
    这些单字指令将停止未来的短信。 注意，只有单字信息才会发送停止。 例如，回复“STOP”将确保您不会收到进一步的信息，但是回复“STOP PLEASE”将不会。
  • START, YES, 和 UNSTOP
    这些指令将允许您选择返回到接收短信。
  • HELP 和 INFO
    这些指令将提供信息，您可以使用上面的指令来控制信息的传递。
