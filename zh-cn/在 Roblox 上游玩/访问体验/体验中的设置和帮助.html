体验中的设置和帮助

在玩体验时，您可以更改设置并了解如何执行跳跃或旋转相机等操作。

设置

此菜单包含音量、相机、图形和辅助功能等功能的设置。

只要作品开发人员没有锁定特定的镜头模式，你就可以在经典模式和跟随模式之间切换。

  • 在 经典模式 下，除非手动调整，否则相机将保持固定在一个位置。 要更改头像的有利位置，请按住鼠标右键并拖动它。
  • 在 跟随模式 下，当您向右或向左移动时，摄像头将随着您的头像一起旋转，以帮助您保持预期目标在视野中。
    • Shift 锁定开关使您可以通过按键盘上的 Shift 键在相机模式之间切换。

我们目前提供三种辅助功能设置，以更好地定制您在 Roblox 上的体验。

  • 减少运动 禁用或减少应用内和体验内的某些屏幕效果。 例如，当游戏菜单打开时，它不会滑到屏幕上，而是在启用“减少运动”时淡入。
  • 背景透明度 通过使 UI 元素上的背景（例如聊天窗口背景）更加不透明来提高对比度和易读性。 这不会使 UI 元素比平常更透明。
  • UI 导航切换 控制是否可以使用 \ 键进入和退出 UI 导航模式。 如果您经常意外进入 UI 导航模式，或者不想使用 UI 导航模式功能，则可以禁用此设置。

帮助

此菜单包含键盘快捷键、鼠标指令、以及移动设备手势，因此你就可以在作品中导航并使用所有可用的功能。
