电脑硬件和操作系统要求

下面列出了 Roblox 的最低系统要求。 如果你在游玩体验中遇到问题，你可能需要升级你的电脑。 请阅读 此帮助文章 以了解移动设备的系统要求。

作業系統要求

  • PC/Windows: Roblox 将安装在 Windows 7、Windows 8.1 或者 Windows 10。 对于 Windows 8.1，您需要在桌面模式下运行 Roblox，因为目前不支持 Metro 模式（平铺的开始屏幕）。.
    注意：Roblox 网站支持以下 PC/Windows 浏览器： Chrome 、 Firefox 和 Microsoft Edge 。
  • Mac： Roblox 應用程式和 Studio 將安裝在 10.13 (High Sierra) 及更高版本上。
    注意：Roblox 網站支援以下 Mac 瀏覽器： Chrome 、 Firefox 和 < 5>Safari 。
  • Linux ：Roblox 不支持 Linux 操作系统。
  • 移动端 ： 点按此处 查看 Roblox 在移动端上的系统要求。
  • Chrome OS ：Chrome OS 版本 53 或更高的版本是必需的。 您必须先在您的设备上启用 Google Play商店 。

注意：Roblox 视频录制功能仅适用于 Windows 7、Windows 8.1 和 Windows 10。 最新发布的操作系统或预发布的操作系统版本可能不会立即得到支持。

系統硬體需求

  • 显卡 ：在 PC/Windows 上，Roblox 应用程序需要 DirectX 10 或更高的功能级别的支持。 为了获得最佳性能，我们建议使用 5 年以下配备专用显卡的电脑，或 3 年以下配备集成显卡的笔记本电脑。
  • 处理器 ：Roblox 建议你使用最新的处理器（2005+），时钟速度为 1.6 Ghz 或更高。 旧 AMD 处理器会出现一些问题。
  • RAM或记忆体： Roblox 建议您在 Windows 7，Windows 8.1 或 Windows 10 上至少有 1 GB 内存。
  • 存储空间 ：Roblox 建议你至少有 20MB 的系统存储空间来安装 Roblox。
  • 移动端 ： 点按此处 查看 Roblox 在移动端上的系统要求。

網路需求

对于单次使用的家庭，Roblox 建议使用最少为 4-8 Mbps 的网络连接。 如果家庭成员较多，更高的带宽最好不过。
