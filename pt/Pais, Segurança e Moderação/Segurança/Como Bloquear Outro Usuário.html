Como Bloquear Outro Usuário

Na Roblox, incentivamos todos os nossos jogadores a nos ajudar a manter um ambiente seguro e divertido para todos. No entanto, sabemos que, às vezes, os jogadores simplesmente não se dão bem. Você pode impedir que certos usuários possam falar com você bloqueando-os.

Bloqueando outro usuário

Para bloquear outro usuário no navegador ou nos aplicativos móveis:

  1. Visite a página de perfil do usuário
  2. Selecione os três pontos no canto superior direito da caixa que contêm o nome de usuário e informações de amigos/seguidores
  3. Um menu será exibido, onde você pode selecionar a opção Bloquear Usuário

Bloqueando usuários na experiência

Para bloquear outro usuário de dentro de uma experiência, siga as etapas abaixo:

  1. Encontre o usuário na lista de classificação/jogadores no canto superior direito da tela da experiência. Isso parecerá assim:

    Se esta lista não estiver visível, provavelmente está fechada. Para reabri-la, selecione seu nome de usuário no canto superior direito. Observação - a tabela de classificação pode não aparecer se você estiver usando um dispositivo de tela pequena, como um telefone; nesse caso, você precisará usar o método de página de perfil descrito acima.
  2. Depois de encontrar o nome do usuário que você deseja bloquear dentro do placar, selecione-o e um menu será aberto.
  3. Selecione Bloquear Usuário . Você também pode optar por desbloqueá-los ou Denunciar Abuso diretamente neste menu. Depois de bloquear o usuário, o ícone à esquerda do nome se transformará em um círculo com uma linha para indicar que eles foram bloqueados.

Para bloquear outro usuário de dentro de uma experiência sem o placar usando o menu Esc:

  1. Encontre o usuário que você deseja bloquear no menu Esc na guia "Pessoas".
  2. Depois de encontrar o nome do usuário que você deseja bloquear, clique no botão Bloquear que tem essa aparência:
  3. Clique no botão " Bloquear " na caixa de diálogo pop-up para confirmar.

Configurações de privacidade e contato

O Roblox também oferece uma variedade de configurações de privacidade para limitar determinadas formas de comunicação e ações a grupos específicos de pessoas, como apenas seus amigos ou seguidores. Leia mais no artigo de ajuda das Configurações de Privacidade.
