Termos de Uso de Dados Analíticos para Criadores

Ao usar os Dados Analíticos para Criadores, você concorda com a coleta e uso de seus dados de usuário pela Roblox para fins de análise. Seus dados de usuário serão usados somente para calcular dados de análise agregados para cada experiência.

Além disso, ao usar os Dados Analíticos para Criadores, você concorda expressamente que irá preservar todas as informações de avaliação comparativa apresentadas nos painéis e/ou derivadas dos mesmos em confidêncialidade e que usará tais informações somente para avaliar o desempenho da sua experiência Roblox. Além disso, você concorda em não compartilhar, vender, divulgar ou disseminar as informações de avaliação comparativa de qualquer forma.
