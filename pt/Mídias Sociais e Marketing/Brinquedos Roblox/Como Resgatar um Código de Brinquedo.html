Como Resgatar um Código de Brinquedo

Onde encontro meu código para o item virtual?

O código é encontrado na frente do pacote ou no cartão de código interno. Confira o vídeo Jazwares sobre cartões de código.

Como faço para resgatar meu código para o item virtual?

  1. Se o seu código estiver coberto, raspe suavemente a proteção.
  2. Verifique se você está conectado à sua conta do Roblox na qual deseja resgatar o código.
  3. Vá para a Página de Resgate de Código de Brinquedo .
  4. Digite seu código na caixa Inserir Código de brinquedo.
  5. Clique no botão Resgatar .
    Confira seu incrível novo item no inventário da sua conta. Certifique-se de verificar a categoria de inventário correta.

Nota: uma conta só pode receber um item virtual de cada brinquedo uma vez . Se você tem um código para um item virtual que sua conta já possui, você pode compartilhar o código com outro usuário.
