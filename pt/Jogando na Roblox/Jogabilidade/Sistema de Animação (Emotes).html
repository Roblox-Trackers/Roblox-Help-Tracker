Sistema de Animação (Emotes)

O Sistema de animação Roblox permite uma quantidade medida de fluidez e realismo para os movimentos. Parte desse sistema são os Emotes, que são ações específicas que o seu avatar do Roblox pode executar. Para ver as emotes padrão, pressione a tecla “/” para bater papo e tente digitar os seguintes comandos:

  • /e wave (acenar)
  • /e point (apontar)
  • /e cheer (comemorar)
  • /e laugh (rir)
  • /e dance (dançar)
  • /e dance2 (dançar 2)
  • /e dance3 (dançar 3)
