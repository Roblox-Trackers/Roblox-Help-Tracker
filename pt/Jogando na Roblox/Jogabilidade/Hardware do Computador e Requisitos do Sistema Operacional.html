Hardware do Computador e Requisitos do Sistema Operacional

Os requisitos mínimos do sistema para a Roblox estão listados abaixo. Se você estiver com problemas para jogar, talvez seja necessário atualizar seu computador. Leia este artigo de ajuda sobre os requisitos do sistema para dispositivos móveis.

Requisitos do sistema operacional

  • PC/Windows: A Roblox será instalada no Windows 7, Windows 8/8.1 ou Windows 10. Para o Windows 8/8.1, você precisará executar o Roblox no modo Desktop, pois o modo Metro (a tela inicial lado a lado) não é suportado atualmente.
  • Mac: A Roblox irá ser instalada no 10.7 (Lion) ou superior - e Intel apenas nos Macs.
  • Linux: A Roblox não é suportada no Linux.
  • Mobile: Clique aqui para requisitos de sistema para o Roblox Mobile.
  • Chrome OS : A versão 53 ou superior do Chrome OS é necessária e você deve primeiro ativar a Google Play Store no seu dispositivo.

Nota: O recurso de gravação de vídeo da Roblox funciona somente no Windows 7, Windows 8/8.1 e Windows 10. Sistemas operacionais recém-lançados ou versões pré-lançadas de sistemas operacionais podem não ser imediatamente suportados.

Requisitos de hardware do sistema

  • Placa de vídeo: A Roblox requer o mínimo do DirectX 9 e o Shader Model 2.0. Para o melhor desempenho, recomendamos um computador com menos de 5 anos com uma placa de vídeo dedicada ou um laptop com menos de 3 anos com uma placa de vídeo integrada.
  • Processador: Roblox recomenda que você tenha um processador recente (2005+) com uma velocidade de clock de 1,6 Ghz ou superior. Houve alguns problemas com processadores AMD mais antigos
  • RAM ou Memória: A Roblox recomenda que você tenha pelo menos 1 GB de memória no Windows 7, Windows 8 ou Windows 10 computadores. Pelo menos 512 Mb de memória é recomendado para o Windows XP.
  • Espaço de Armazenamento: A Roblox recomenda que você tenha pelo menos 20 Mb de espaço de armazenamento do sistema para instalar o Roblox.
  • Mobile: Clique aqui para requisitos de sistema para o Roblox Movel.

Requisitos de Internet

Para uma família pequena, a Roblox recomenda uma conexão de internet mínima de 4 a 8 Mb/s. Em residências maiores, é aconselhável uma maior banda larga.
