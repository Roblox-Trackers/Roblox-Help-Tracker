Estou Logado, Mas eu Apareço Como Convidado Dentro dos Jogos

Se você achar que está sendo colocado em jogos como convidado mesmo que esteja conectado à sua conta, há algumas coisas para investigar.

O Que Fazer - Windows

Por favor, tente cada passo abaixo, um de cada vez, em ordem. Se uma solução não funcionar, passe para a próxima!

Reinicie

Primeiro reinicie o seu computador e tente jogar. Às vezes, isso pode ser tudo o que é necessário para resolver problemas relacionados ao Roblox ou ao Roblox Studio.

Redefina suas opções de internet no Windows

Mesmo que você não use o Internet Explorer, a redefinição das opções da Internet por meio desse navegador ainda poderá resolver seu problema. Observe que, embora essas etapas precisem ser feitas usando o Internet Explorer, você pode voltar ao seu navegador de preferência depois de terminar. Por favor, tente o seguinte:

    1. Abra o Internet Explorer
    2. Abra o ícone de roda dentada> Opções da Internet
    3. Abra a guia Avançado
    4. Clique no botão Redefinir

Após a redefinição, feche o Internet Explorer para permitir que as alterações entrem em vigor.

Nota: Se você estiver usando o Windows 10, certifique-se de executar estas etapas com o Internet Explorer e não com o Microsoft Edge. Os ícones parecem semelhantes. Para encontrar o IE no Windows 10, pressione a tecla Windows + S no teclado e digite "Internet Explorer".

Verifique o Seu Relógio do Sistema Windows

Pode parecer estranho, mas vários problemas podem ser causados ​​pelo fato de o relógio do sistema do Windows estar definido na hora ou data errada. Se esta é a causa, simplesmente corrigir tudo deve remediar a situação. Embora isso possa ser feito manualmente, você também pode configurá-lo para atualizar automaticamente com um servidor, garantindo assim que ele não escorregue para trás e comece a causar problemas novamente. Veja como fazer isso:

Windows 8.1 e Anterior

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Clique em Ajustar data / hora
    3. Selecione a guia Horário da Internet
    4. Aqui você deve ser capaz de sincronizar sua hora e data com um servidor de internet

Windows 10

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Certifique-se de que a opção Definir hora automaticamente está ativada e você selecionou o fuso horário correto

Verifique Suas Configurações de Proxy

Se as etapas acima não resolverem o problema, talvez seja necessário certificar-se de que o computador não está tentando se conectar por meio de um servidor proxy. Por favor, siga estes passos:

Windows 7 ou Anterior

    1. Vá para o menu Iniciar
    2. Procurar "proxy" e configurar o servidor proxy irá aparecer, clique nele
    3. Clique em Configurações de LAN e desmarque todas as marcas de seleção nessa guia
    4. Clique OK

Windows 8 ou 8.1

    1. Vá para a tela inicial lado a lado clicando no canto inferior esquerdo da tela
    2. Digite "proxy" e uma opção Configurar Servidor Proxy aparecerá. Clique nisso
    3. Clique em Configurações de LAN e desmarque todas as marcas de seleção nessa guia
    4. Clique OK

Windows 10

    1. Pressione Windows Key + S no seu teclado. Alternativamente, se a barra "Pergunte-me qualquer coisa" da Cortana estiver disponível na barra de tarefas, basta clicar
    2. Digite "proxy" e uma opção Alterar configurações de proxy será exibida. Clique nisso
    3. Certifique-se de que "Usar script de instalação" e "Usar um servidor proxy" estejam desativados.
    4. Role para baixo e clique em Salvar
