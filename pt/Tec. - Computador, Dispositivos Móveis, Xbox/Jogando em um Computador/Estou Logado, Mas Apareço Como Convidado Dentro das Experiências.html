Estou Logado, Mas Apareço Como Convidado Dentro das Experiências

Se você perceber que está sendo colocado em Experiências como convidado mesmo que esteja conectado à sua conta, há algumas coisas para se investigar.

O Que Fazer - Windows

Por favor, tente cada passo abaixo, um de cada vez, em ordem. Se a solução não funcionar, passe para a próxima!

Reinicie

Primeiro reinicie o seu computador e tente jogar. Às vezes, isso é tudo que é preciso para resolver problemas relacionados a Roblox ou ao Roblox Studio.

Verifique o Seu Relógio do Sistema no Windows

Pode parecer estranho, mas vários problemas podem ser causados ​​pelo fato de o relógio do sistema do Windows estar definido na hora ou data errada. Se esta for a causa, simplesmente corrigir tudo deve remediar a situação. Embora isso possa ser feito manualmente, você também pode configurá-lo para atualizar automaticamente com um servidor, garantindo assim que ele não se atrase e comece a causar problemas novamente. Veja como fazer isso:

Windows 8.1 e Anterior

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Clique em Ajustar Data / Hora
    3. Selecione a guia Horário da Internet
    4. Aqui você deve ser capaz de sincronizar sua hora e data com um servidor de internet

Windows 10

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Verifique se a opção Definir Hora Automaticamente está ativada e se você selecionou o fuso horário correto

Verifique Suas Configurações de Proxy

Se as etapas acima não resolverem o problema, talvez seja necessário verificar se o computador não está tentando se conectar por meio de um servidor proxy. Por favor, siga estes passos:

Windows 7 ou Anterior

    1. Vá para o menu Iniciar;
    2. Procure por "proxy" e surgirá a opção Configurar o Servidor Proxy. Clique nela;
    3. Clique em Configurações de LAN e desmarque todas as caixas nessa guia;
    4. Clique OK;

Windows 8 ou 8.1

    1. Vá para a tela inicial clicando no canto inferior esquerdo da tela;
    2. Digite "proxy" e surgirá a opção Configurar Servidor Proxy. Clique nela;
    3. Clique em Configurações de LAN e desmarque todas as caixas nessa guia;
    4. Clique OK;

Windows 10

    1. Pressione Windows + S no seu teclado. Alternativamente, se a barra "Pergunte-me qualquer coisa" da Cortana estiver disponível na barra de tarefas, basta clicar nela;
    2. Digite "proxy" e surgirá a opção Alterar Configurações de Proxy. Clique nela;
    3. Certifique-se de que "Usar script de instalação" e "Usar um servidor proxy" estejam desativados;
    4. Role para baixo e clique em Salvar;
