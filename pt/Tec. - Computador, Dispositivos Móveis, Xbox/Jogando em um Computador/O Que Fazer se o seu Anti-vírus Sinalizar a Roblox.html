O Que Fazer se o seu Anti-vírus Sinalizar a Roblox

Se o seu anti-vírus sinalizou a Roblox como uma preocupação, tenha a certeza de que o Roblox está livre de vírus e não contém nenhum malware. Ocasionalmente, os programas antivírus podem sinalizar um programa incorretamente, o que é conhecido como falso positivo. Aqui estão algumas soluções sugeridas para ajudar a evitar que isso aconteça, bem como o que você pode fazer se ocorrer:

Atualize seu antivírus

Às vezes tudo que você precisa fazer é apenas ter certeza de que você tem a versão mais recente do seu programa anti-vírus. É possível que eles já tenham corrigido o problema. Se isso não funcionar, então vamos passar para a próxima possibilidade

Sinalizar o relatório como falso positivo

Se a atualização do software não ajudar, você precisará sinalizar o aviso como falso positivo. Para garantir que o programa Roblox seja executado corretamente, você precisará defini-lo como um programa permitido para qualquer aplicativo antivírus ou anti-malware. Isso pode ser diferente para cada programa, mas os links abaixo podem ajudar com alguns dos mais comuns.s.

  • AVAST Anti-Virus
  • Bitdefender
  • Kaspersky
  • Malwarebytes
  • Norton
  • Windows Defender
  • Panda Security
  • McAfee

Ao procurar exclusões em máquinas Windows, os arquivos do Roblox são armazenados na seguinte pasta:

C:\Usuario\(Seu usuario do Windows)\AppData\Local\Roblox
