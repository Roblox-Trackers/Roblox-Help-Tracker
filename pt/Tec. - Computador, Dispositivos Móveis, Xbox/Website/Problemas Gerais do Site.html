Problemas Gerais do Site

O artigo a seguir irá sugerir alguns métodos que se mostraram eficazes para corrigir os seguintes problemas:

  • O site está sendo exibido incorretamente.
  • Algumas páginas não estão sendo carregadas ou exibem um erro ao carregar.
  • Botões ou links não estão permitindo que você clique neles
  • Você tenta entrar no site da Roblox, mas não consegue, apesar de ter o nome de usuário e senha corretos
  • Outros: se você tiver um problema diferente com o website, as etapas abaixo também podem ajudar a resolvê-lo. Além disso, alguns artigos não relacionados ao website podem ser vinculados aqui para algumas etapas

O que fazer

Por favor, tente cada passo abaixo, um de cada vez, em ordem. Se uma solução não funcionar, passe para a próxima!

Limpe seus arquivos temporários da Internet e cookies

Os links a seguir ajudarão você a remover os arquivos temporários da Internet (também conhecidos como "cache") e os cookies dos principais navegadores da Web para PC. Por favor, certifique-se de fazer isso para o Internet Explorer, bem como qualquer outro navegador que você deseja iniciar a Roblox.

      • Mozilla Firefox - Arquivos temporários de internet (em Mozilla.com)
      • Mozilla Firefox - Cookies (em Mozilla.com)
      • Google Chrome - Ambos (em Google.com)
      • Microsoft Edge - Ambos (em Microsoft.com)
      • Safari - Cookies (em Apple.com)
      • Outros Navegadores: Se você estiver usando um navegador diferente ou uma versão mais antiga de um navegador, pesquise informações sobre como excluir seus dados de cache / navegador / arquivos temporários da Internet para seu navegador / versão específica.

Depois de concluir o acima para todos os navegadores relevantes, reinicie o computador e verifique se o problema foi resolvido.

Se você desejar excluir manualmente os arquivos temporários da Internet do Internet Explorer / Windows,  você poderá fazer isso indo diretamente para a pasta Arquivos temporários no disco rígido. O nome do local e da pasta depende da versão do Windows e se você está ou não usando perfis de usuário.

Nota: Pode ser necessário alterar as configurações do Windows Explorer para mostrar todos os tipos de arquivos (incluindo os arquivos de sistema protegidos) para acessar essas pastas.

Para Windows 10

Se você tiver o Windows 10, os Arquivos Temporários da Internet estarão no seguinte local (observe que no seu PC eles podem estar em outra unidade em vez da unidade C):

        • C:\Users\<nomedeusuario>\AppData\Local\Microsoft\Windows\INetCache

Para Windows 7 e 8

Se você tiver o Windows 7 ou o Windows 8, os Arquivos de Internet Temporários estarão nesses locais (observe que eles podem estar em outra unidade, em vez de na unidade C):

        • C:\Users\<nomedeusuario>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<nomedeusuario>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Desativar Complementos do seu navegador da Web

Muitas vezes, problemas relacionados ao website são o resultado de complementos de terceiros para o seu navegador da web. Estes também podem ser referidos como "Extensões" ou "Plugins", dependendo do seu navegador de escolha. Para ver se este é o seu caso, tente desabilitar todos os seus complementos. Se o problema for resolvido, tente reativá-los um de cada vez até que seu problema comece a ocorrer novamente. Depois que o problema iniciar novamente, você deve ter encontrado o que é incompatível com a Roblox.

Os links a seguir podem ajudá-lo com os navegadores da Web para PC mais populares:

        • Internet Explorer (em Microsoft.com)
        • Firefox (em Mozilla.com)
        • Chrome (em Google.com)
        • Outros navegadores: Se você estiver usando um navegador diferente ou uma versão mais antiga de um navegador, pesquise informações sobre como desabilitar suas extensões / plugins / propagandas para seu navegador / versão específica.

Se o problema persistir mesmo após seguir as sugestões acima, entre em contato com o Suporte ao Cliente da Roblox e certifique-se de fornecer o seguinte:

  • Nome de usuário da Roblox

  • Qualquer mensagem de erro recebida

  • Nome do modelo do seu dispositivo

  • Versão do iOS

  • Passos para encontrar o problema

Por favor, inclua qualquer detalhe adicional que possa nos ajudar a entender melhor o problema que está sendo reportado, bem como qualquer alteração recente que você tenha feito em seu dispositivo ou redes de conexão.
