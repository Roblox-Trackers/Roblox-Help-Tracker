Hardware do computador e requisitos do sistema operacional

Os requisitos mínimos do sistema para a Roblox estão listados abaixo. Se você estiver com problemas para jogar, talvez seja necessário atualizar seu computador. Leia este artigo de ajuda sobre os requisitos do sistema para dispositivos móveis.

Requisitos do sistema operacional

  • PC/Windows: A Roblox pode ser instalada em versões de 64 bits do Windows 7, Windows 8.1 ou Windows 10. Para o Windows 8.1, você precisará executar o Roblox no modo Desktop, pois o modo Metro (a tela inicial lado a lado) não é suportado atualmente.
    Nota: O site do Roblox suporta os seguintes navegadores para PC: Chrome , Firefox , e Microsoft Edge .
  • Mac: O Roblox Application & Studio será instalado na versão 10.13 (High Sierra) e superior.
    Nota: O site da Roblox é compatível com os seguintes navegadores para Mac: Chrome , Firefox , e Safari .
  • Linux: O aplicativo Roblox não é suportado em Linux.
  • Dispositivos móveis: Clique aqui para requisitos de sistema para o aplicativo Roblox para dispositivos móveis.
  • Chrome OS : É necessário a versão 53 ou superior do Chrome OS e você deve primeiro ativar a Google Play Store em seu dispositivo.

Obs.: O recurso de gravação de vídeo da Roblox funciona somente no Windows 7, Windows 8.1 e Windows 10. Sistemas operacionais recém-lançados ou versões pré-lançadas de sistemas operacionais podem não ser imediatamente suportadas.

Requisitos de hardware do sistema

  • Placa de vídeo: Em PC/Windows, o aplicativo Roblox requer suporte a nível de recurso do DirectX 10 ou superior. Para obter o melhor desempenho, recomendamos um computador com menos de 5 anos com placa de vídeo dedicada ou um laptop com menos de 3 anos com placa de vídeo integrada.
  • Processador: A Roblox recomenda que você tenha um processador recente (2005+) com uma velocidade de clock de 1,6 Ghz ou superior. Há relatos de alguns problemas com processadores AMD mais antigos
  • RAM ou Memória: A Roblox recomenda que você tenha pelo menos 1 GB de memória no Windows 7, Windows 8.1 ou Windows 10 computadores.
  • Espaço de Armazenamento: A Roblox recomenda que você tenha pelo menos 20 Mb de espaço de armazenamento do sistema para instalar o cliente Roblox.
  • Dispositivos móveis: Clique aqui para ver os requisitos de sistema para a Roblox em dispositivos móveis.

Requisitos de Internet

Para uma moradia de pequeno porte com poucos usuários, a Roblox recomenda uma conexão de internet mínima de 4 a 8 Mb/s. Em residências maiores, é aconselhável uma banda larga maior.
