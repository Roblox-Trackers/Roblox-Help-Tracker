Mensagens de Hyperion

O artigo contém possíveis mensagens de travamento antitrapaça com que você pode se deparar, seguidas pelas etapas para solucionar os problemas respectivos.

Tipo de travamento : Falha por tempo esgotado

Mensagem ao usuário : Tempo esgotado no Roblox. Reinicie o cliente.

Etapas para solução de problemas :

Se encontrar a mensagem "Tempo esgotado no Roblox. Reinicie o cliente", é preciso reiniciar o cliente do Roblox. Se o problema persistir mesmo após reiniciar o cliente durante um uso regular, contate a nossa equipe de suporte para receber assistência.

Tipo de travamento : Máquina virtual detectada

Mensagem ao usuário : Roblox não roda em máquina virtual.

Etapas para solução de problemas :

Se encontrar a mensagem "Máquina virtual detectada", significa que a sua máquina virtual (VM) não é suportada por Roblox. Roblox requer passagem pela placa de vídeo para rodar direito dentro de uma VM. Passagem pela placa de vídeo dá à VM acesso direto à placa física, contornando a camada do hipervisor. Veja como habilitar a passagem pela placa de vídeo:

      1. Confirme que seu hardware tem suporte a passagem (GPU passthrough, em inglês)
      2. Atualize as configurações da BIO/UEFI para habilitar extensões de virtualização (por ex., Intel VT-x ou AMD-V).
      3. Configure a passagem pela placa de vídeo no software da VM.

Siga as etapas para garantir que o Roblox rode direito num ambiente virtualizado, fornecendo o acesso necessário aos recursos da GPU.

Tipo de travamento : Badware detectado

Mensagem ao usuário : Roblox travou devido a software incompatível: <caminho do software infrator>

Etapas para solução de problemas :

Se encontrar a mensagem "Roblox travou devido a software incompatível:", seguido pelo caminho do software infrator, quer dizer que o software mencionado no caminho exibido está causando problemas com o Roblox. Para corrigir o problema, evite que o software rode enquanto você usa Roblox. Para isso, você pode desativar manualmente o software ou reiniciar o Windows. O método garante que apenas softwares necessários estejam ativos no sistema, minimizando conflitos. Confira como realizar um reinício "simples", como recomendado pelo Windows :

      1. Aperte "tecla Windows + R" para abrir o Executar.
      2. Digite "msconfig" e aperte Enter.
      3. Na janela de Configurações do Sistema, acesse a aba "Serviços".
      4. Clique na caixa "Ocultar todos os serviços Microsoft".
      5. Clique em "Desativar tudo."
      6. Vá para a aba "Inicialização de Programas".
      7. Clique em "Abrir Gerenciador de Tarefas."
      8. Desabilite todos os itens de inicialização.
      9. Feche o Gerenciador de Tarefas e a Configurações do Sistema.
      10. Reinicie o computador.

Concluir as etapas garantirá que nenhum software interaja com o Roblox, possibilitando que ele rode sem travar.

Tipo de travamento : Integridade da inicialização comprometida

Mensagem ao usuário : Arquivos críticos do sistema estão corrompidos. Roblox não pode rodar. Repare ou redefina a sua instalação do Windows.

Etapas para solução de problemas :

Roblox não funciona direito quando o Windows está no modo de depuração. Para usar Roblox, certifique-se de que o Windows esteja rodando normalmente. Siga estes passos para verificar e desabilitar o modo de depuração no Windows .

      1. Aperte "tecla Windows + R" para abrir o Executar.
      2. Digite "msconfig" e aperte Enter.
      3. Selecione a aba "Inicialização do Sistema" e então "Opções avançadas".
      4. Desmarque a caixa "Depurar".
      5. Selecione "OK."
      6. Selecione "Aplicar" e então "OK".
      7. Reinicie o computador.

As etapas garantirão que o Windows não está rodando no modo de depuração, possibilitando usar Roblox sem problemas.

Tipo de travamento : (OBS) Open Broadcaster Software

Mensagem do sistema : A captura no jogo de OBS costuma dar erros com Roblox. Use captura de janela ou de área de trabalho em vez disso.

Etapas para solução de problemas :

Se encontrar a mensagem "A captura no jogo de OBS costuma dar erros com Roblox", significa que Roblox não suporta a funcionalidade de captura no jogo do Open Broadcaster Software (OBS). Para transmitir ou gravar Roblox, você terá que usar captura de janela ou de área de trabalho em vez disso. Siga as instruções para configurar captura de área de trabalho :

      1. Abra o Open Broadcaster Software (OBS).
      2. Na seção "Sources", clique no ícone "+".
      3. Selecione "Display Capture" do menu.
      4. Escolha a tela que deseja capturar (em geral, seu monitor primário).
      5. Ajuste as configurações, como resolução e taxa de quadros, conforme necessário.
      6. Clique em "OK" para confirmar a fonte de captura da exibição.
      7. Configure a fonte de captura de exibição conforme desejado na sua cena do OBS.
      8. Comece a transmitir ou gravar como de costume.

Usando o modo de captura de área de trabalho, dá para transmitir ou gravar Roblox sem se deparar com erros de compatibilidade com a funcionalidade de captura no jogo do OBS.
