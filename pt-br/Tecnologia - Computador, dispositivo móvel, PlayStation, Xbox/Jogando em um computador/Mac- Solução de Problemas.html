Mac: Solução de Problemas

Instalação/Reinstalação dos aplicativos Roblox

Se você está tendo um problema e ainda não tentou reinstalar recentemente, tente desinstalar o programa e reinstalar usando os passos localizados no final dos seguintes dois artigos:

  • Desinstalar
  • Reinstalar

Reiniciar o computador

Se você está reportando um problema usando qualquer aplicação Roblox, tente reiniciar seu computador para ver se o problema foi resolvido.

Redefinição

Se reiniciar o computador não ajudar, siga os passos abaixo.

  1. Aperte comando + espaço
  2. Copie e cole: ~/Library/Logs/Roblox/
  3. Abra e delete tudo na pasta.
  4. Tente reproduzir o mesmo erro.

Outras opções

Se desinstalar e reinstalar não ajudar, tente o seguinte

  • Aplicação Roblox & Studio será instalado na versão 10.13 ( High Sierra ) e superior.
  • Atualmente, suportamos Apple Silicon / Chips M1.
  • Se você estiver usando o navegador da Web Safari e tiver algum problema, tente usar um de nossos navegadores compatíveis, como o o Firefox ou o Google Chrome.
  • Verifique se o ícone do aplicativo Roblox está atualmente no seu Dock e se o aparece na sua pasta Aplicativos.
  • Tente iniciar o programa a partir do Dock se estiver com problemas para jogar um jogo no site.
  • Tente redefinir totalmente suas configurações Roblox excluindo os arquivos de configurações. Pra fazer isso vá para a biblioteca-> Preferências e delete todos os arquivos Roblox. Quando o Studio / Plataforma for iniciado novamente, eles devem ser reinstalados.
