Problemas Gerais do Roblox Studio

O artigo a seguir irá sugerir alguns métodos que se mostraram eficazes para corrigir os seguintes problemas:

  • Você tenta entrar no Roblox Studio, mas não consegue
  • Você tenta editar um local apenas para que ele carregue um skybox vazio ou algo chamado Place1
  • As seções do Studio estão em branco e não deviam estar
  • Você está tendo dificuldades para publicar no Roblox
  • Roblox Studio lhe diz para fazer o login mesmo que você já esteja logado
  • Essas etapas podem ajudar alguns problemas não relacionados ao Studio, como entrar em um jogo como Convidado mesmo quando você está conectado

Reiniciar

Primeiro reinicie o seu computador e tente jogar. Às vezes, isso pode ser tudo o que é necessário para resolver problemas relacionados ao Roblox ou ao Roblox Studio.

Verifique as suas definições de proxy

Se as etapas acima não resolverem o problema do Roblox Studio, talvez seja necessário certificar-se de que o computador não está tentando se conectar por meio de um servidor proxy. Por favor, siga estes passos:

Windows 7 ou anterior

    1. Vá para o menu Iniciar
    2. Digite "proxy" e "Alterar configurações de proxy" irá aparecer, clique nele
    3. Clique em Configurações LAN e desmarque todas as marcas de seleção nessa guia
    4. Clique no botão Reset (Redefinir)

Windows 8 ou 8.1

    1. Vá para a tela inicial lado a lado clicando no canto inferior esquerdo da tela
    2. Procure "proxy" e "Configurar servidor proxy" irá aparecer, clique nele
    3. Clique em Configurações LAN e desmarque todas as marcas de seleção nessa guia
    4. Clique em OK

Windows 10

    1. Pressione Windows Key + S no seu teclado. Alternativamente, se a barra "Me Pergunte Qualquer Coisa" da Cortana estiver disponível na barra de tarefas, você pode simplesmente clicar nela
    2. Procure "proxy" e "Configurar servidor proxy" irá aparecer, clique nele
    3. Certifique-se de que "Usar script de instalação" e "Usar um servidor proxy" estejam desativados.
    4. Clique em OK

Verifique o relógio do sistema Windows

Pode parecer estranho, mas vários problemas podem ser causados ​​pelo fato de o relógio do sistema do Windows estar definido na hora ou data errada. Se essa é a causa, simplesmente corrigir tudo deve remediar a situação. Embora isso possa ser feito manualmente, você também pode configurá-lo para atualizar automaticamente com um servidor, garantindo assim que ele não escorregue para trás e comece a causar problemas novamente. Veja como fazer isso:

Windows 8 ou anterior

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Clique em Ajustar Data/Hora
    3. Selecione a guia Horário da Internet
    4. Aqui você deve ser capaz de sincronizar sua hora e data com um servidor de internet

Windows 10

    1. Clique com o botão direito no relógio do Windows no canto inferior direito
    2. Certifique-se de que "Definir hora automaticamente" está ativado e você selecionou o fuso horário correto
