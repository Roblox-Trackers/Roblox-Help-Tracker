Como Eu Resgato um Código Promocional?

Como posso obter um código promocional da Roblox?

Você pode receber um código promocional do Roblox em nossos eventos ou brindes. Consulte Como obtenho itens promocionais ou de concurso? para obter mais informações sobre como obter itens promocionais.

Nota: Lembre-se de que os códigos promocionais podem expirar ou ficar ativos apenas por um curto período, portanto, use seu código imediatamente.

Como eu resgato o código promocional de um item virtual?

  1. Verifique se você está conectado à conta Roblox correta para a qual deseja resgatar o código.
  2. Vá para Resgatar Códigos Roblox .
  3. Digite seu código na caixa.
  4. Clique em Resgatar .
  5. Uma mensagem de confirmação aparecerá quando você conseguir resgatar o código com sucesso.
  6. Você receberá seu Crédito, Robux ou Item Virtual. Isso dependerá do tipo de Código que você resgatou.

Problemas com o resgate de códigos promocionais

Se estiver tendo problemas para resgatar um código promocional, tente o seguinte:

  1. Verifique se você está conectado à sua conta Roblox antes de resgatar o código
  2. Navegue até a página de resgate de código promocional
  3. Certifique-se de que você inseriu o código com precisão
  4. Verifique seu inventário para ver se você já tem o item exclusivo

Observação: Os códigos promocionais geralmente estão disponíveis apenas por tempo limitado. Se o período de tempo para um determinado código já tiver passado, você não poderá resgatar o código.
