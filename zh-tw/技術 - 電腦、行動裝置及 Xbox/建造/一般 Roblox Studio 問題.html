一般 Roblox Studio 問題

这文章提出一些有效方法解决以下問題：

  • 您嘗試登錄 Roblox Studio 但無法啟動，
  • 您試著編輯一個地方但它加載了一個空白的 Skybox 或叫做 Place1
  • Studio 的一部分不應该是空的。
  • 在 Roblox 上發佈時遇到了問題
  • 當你已經在線，Roblox Studio 還是要求登錄
  • 這些步驟可能有助於解決一些與 Studio 無關的問題，例如即使您已登入也可以以訪客身分進入體驗
  • 其他：如果您遇到與 Studio 額外問題，則以下步驟也可能有助於解決問題

請依照以下步驟解決並預防問題：

怎麼辦 - Windows

請使用按順序嘗試下面的每個步驟。 如果一個解決方案無效，請繼續下一步！

重啟

首先重新啟動電腦並嘗試播放。 有時這会解決 Roblox 或 Roblox Studio 相關問題。

檢查您的代理設定

如果上述步驟不能解決您的 Roblox Studio 問題，您需要確保電腦不是在嘗試通過代理服務器進行連接。 請按照以下步驟操作：

Windows 7 或更早版本

    1. 轉到 Start “開始”菜單
    2. 搜索“代理”和“配置代理服務器”將会出現，并點擊它
    3. 點擊 LAN “ 局域網”設置 ，然後取消選中該選項卡中的所有復選標記
    4. 點擊 OK

Windows 8 或 8.1

    1. 點擊屏幕左下角，以便進入平鋪開始屏幕
    2. 输入Proxy“代理”然后“配置代理服務器”將会出現，并點擊它
    3. 點擊 LAN“ 局域網”設置 ，然後取消選中該選項卡中的所有復選標記
    4. 點擊 OK

Windows 10

    1. 按下鍵盤上的 Windows 鍵 +S 鍵。 或者，如果 Cortana 的“問我任何事情”欄在您的任務欄中可用，您可以單擊它
    2. 搜索“代理”然后“配置代理服務器”將会出現，并點擊它
    3. 確保“使用安裝腳本”和“使用代理服務器”設置為關閉
    4. 向下滾動並點擊 保存

檢查您的 Windows 系統時鐘

這聽起來很奇怪，但實際上各種問題是由於 Windows 系統時鐘的時間或日期被設置成錯誤而導致的。 如果這是原因，那麼糾正一切就應該解決情況。 雖然可以手動完成它，但也可以將其設置為使用服務器自動更新，從而確保它不會滑落並再次引發問題。 以下是如何做到這一點：

Windows 8 或更早版本

    1. 查观 您的 Windows 系統時鐘
    2. 點擊 調整日期/時間
    3. 選擇 互聯網時間 選項卡
    4. 在這裡，您應該能夠將您的時間和日期與互聯網服務器進行同步

Windows 10

    1. 右鍵點擊右下角的窗口時鐘
    2. 確保“自動設置時間”已打開，並且您已選擇了正確的時區

做什麼 - Mac

Roblox 應用程式和 Studio 將安裝在 10.13 ( High Sierra ) 及更高版本上。 要在 Mac 上修復此問題，請重新安裝。 如需幫助，請 點擊此處 。

盡量減少未來的問題

在安裝或更新 Roblox 之前，請始終確保關閉計算機上的所有程序。
