Mac：故障排除

卸載/重新安裝 Roblox 應用程式

如果您遇到問題並且最近還沒有嘗試重新安裝，請嘗試卸載該程序，並使用以下兩篇文章底部的步驟重新安裝：

  • 卸載
  • 重新安裝

重新啟動計算機

如果您使用任何 Roblox 應用程式問題，請嘗試重新啟動電腦來解決問題。

重置日誌

如果重新啟動電腦沒有幫助，請按照以下步驟操作。

  1. 按 Command + 空白鍵
  2. 複製並貼上：~/Library/Logs/Roblox/
  3. 打開並刪除資料夾中的所有內容
  4. 嘗試重現相同的問題。

嘗試其他的事项：

如果卸載和重新安裝不起作用，請嘗試以下操作

  • Roblox 應用程式和Studio 將安裝在 10.13 ( High Sierra ) 及更高版本上。
  • 目前，我們支援Apple Silicon 或 M1芯片。
  • 如果您當前正在使用 Safari Web 瀏覽器遇到問題，請嘗試使用受支持的瀏覽器之一，如 Firefox 或 Google Chrome。
  • 檢查以確保 Roblox 圖標當前位於 Dock 上，並且 Roblox 應用程序顯示在您的應用文件夾中。
  • 如果您在玩網站體驗時遇到問題，請嘗試從 Dock 啟動程式。
  • 嘗試通過刪除設置文件來徹底的重置您的 Roblox 設置。 為此，請前往「庫」-> 「首選項」 並刪除所有 Roblox com 檔案。 當 Studio / Player 重新啟動時，將會重新安裝它們。
