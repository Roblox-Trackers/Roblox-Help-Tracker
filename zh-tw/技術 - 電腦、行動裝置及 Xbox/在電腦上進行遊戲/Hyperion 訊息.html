Hyperion 訊息

本文章包含您可能看到的 Hyperion 反作弊崩潰訊息以及對應的疑難排解步驟，以協助您解決這些問題。

崩潰類型 ：逾時失敗

使用者訊息 ：Roblox 逾時。 請重新啟動用戶端。

疑難排解步驟 ：

如果您遇到訊息顯示：「Roblox 逾時。 請重新啟動用戶端。」表示您的 Roblox 用戶端需要重新啟動。 若重新啟動用戶端並在正常使用下仍發生此問題， 請聯絡支援團隊 取得進一步協助。

崩潰類型 ：偵測到虛擬機器

使用者訊息 ：Roblox 無法在虛擬機器內執行。

疑難排解步驟 ：

如果您看到訊息顯示「偵測到虛擬機器」，表示您的虛擬機器 (VM) 不支援 Roblox。 Roblox 需要顯示卡直通 (GPU passthrough) 才能在 VM 中正常運作。 顯示卡直通能讓 VM 繞過虛擬機管理程式層，直接存取實體顯示卡。 以下是啟用顯示卡直通的方法：

      1. 確認您的硬體支援顯示卡直通。
      2. 更新 BIOS/UEFI 設定以啟用虛擬擴充（例如 Intel VT-x 或 AMD-V）。
      3. 在 VM 軟體中設定顯示卡直通。

遵循以上步驟提供必要的顯示卡資源存取權限，確保 Roblox 能在虛擬化環境中順暢運作。

崩潰類型 ：偵測到惡意軟體

使用者訊息 ：Roblox 因為不相容的軟體崩潰：<違規軟體路徑>

疑難排解步驟 ：

如果您看到訊息顯示「Roblox 因為不相容的軟體崩潰：」且後續接著違規軟體的路徑。表示顯示檔案路徑中提到的軟體會對 Roblox 造成問題。 若要修正，您需要在使用 Roblox 時避免使用該軟體。 您可以手動關閉該軟體，或是執行 Windows 乾淨啟動。 此方法能確保系統上僅執行必要的軟體，盡可能減少衝突。 以下是按照 Windows 建議，完成乾淨啟動的方式 ：

      1. 按下「Windows 鍵 + R」以開啟「執行」對話方塊。
      2. 輸入「msconfig」並按下 Enter 鍵。
      3. 在「系統設定」視窗中，前往「服務」索引標籤。
      4. 勾選標示為「隱藏所有 Microsoft 服務」的方塊。
      5. 按一下「全部停用」。
      6. 切換到「啟動」索引標籤。
      7. 按一下「開啟工作管理員」。
      8. 停用所有啟動項目。
      9. 關閉工作管理員和系統設定。
      10. 重新啟動電腦。

完成這些步驟可確保沒有其他軟體干擾 Roblox，使其順暢運作不會崩潰。

      4. 勾選標示為「隱藏所有 Microsoft 服務」的方塊。
      5. 按一下「全部停用」。
      6. 切換到「啟動」索引標籤。
      7. 按一下「開啟工作管理員」。

崩潰類型 ：啟動完整性遭破壞

使用者訊息 ：重要系統檔案已損毀。 無法執行 Roblox。 請修復或重置 Windows 安裝。

疑難排解步驟 ：

Roblox 在 Windows 設為偵錯模式時無法正常運作。 若要使用 Roblox，請確保 Windows 以正常方式運作。 遵循以下步驟以確認並停用 Windows 的偵錯模式 。

      1. 按下「Windows 鍵 + R」以開啟「執行」方塊。
      2. 輸入「msconfig」並按下 Enter 鍵。
      3. 選擇「開機」索引標籤，然後選擇「進階選項」。
      4. 取消勾選「偵錯」核取方塊。
      5. 選擇「確定」。
      6. 選擇「套用」然後「確定」。
      7. 重新啟動電腦。

這些步驟可確保 Windows 並未執行偵錯模式，讓您順暢無礙使用 Roblox。

崩潰類型 ：(OBS) Open Broadcaster Software

使用者訊息 ：OBS 遊戲內擷取已知會與 Roblox 發生問題。 請改為使用視窗或桌面擷取。

疑難排解步驟 ：

如果您看到訊息顯示：「OBS 遊戲內擷取已知會與 Roblox 發生問題。」表示 Roblox 不支援 Open Broadcaster Software (OBS) 的遊戲內擷取功能。 若要串流或錄製 Roblox，您需要改用視窗或桌面擷取模式。 遵循以下指示以設定桌面擷取 ：

      1. 開啟 Open Broadcaster Software (OBS)。
      2. 在「來源」區塊，按一下「+」圖示。
      3. 在選單中選擇「顯示器擷取」。
      4. 選擇您要擷取的顯示器（通常是您的主螢幕）。
      5. 根據需要調整如解析度和畫格速率。
      6. 按一下「確定」以確認顯示器擷取來源。
      7. 在 OBS 場景中視需要調整顯示器擷取來源。
      8. 照以往方式開始串流或錄製。

使用桌面擷取模式，您可以有效地串流或錄製 Roblox，而不會遇到 OBS 遊戲內擷取功能的相容性問題。
