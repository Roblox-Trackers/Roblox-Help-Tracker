電腦硬體和作業系統需求.

下面列出了 Roblox 的最低系統要求。 如果您在玩體驗時遇到問題，您可能需要對您的電腦進行升及。 閱讀 這篇幫助文章 了解行動裝置的系統需求。

作業系統要求

  • PC/Windows： Roblox 將安裝在 64 位元版本的 Windows 7、Windows 8.1 或 Windows 10 上。 关於 Windows 8.1，您需要在桌面模式下執行 Roblox，因為目前不支援 Metro 模式（平舖的開始畫面）。
    注意：Roblox 網站支援以下 PC/Windows 瀏覽器：<2 >Chrome< /2>、 Firefox 和 Microsoft Edge 。
  • Mac： Roblox 應用程式和 Studio 將安裝在 10.13 (High Sierra) 及更高版本上。
    注意：Roblox 網站支援以下 Mac 瀏覽器： Chrome 、 Firefox 和 < 5>Safari 。
  • Linux： Linux 不支援 Roblox。
  • 行動版： 點擊此處 了解 Roblox Mobile 的系統需求。
  • Chrome 作業系統 ：需要 Chrome 作業系統版本 53 或更高版本，而且您必須先在裝置上啟用 Google Play 商店 。

注意：Roblox 視頻錄製功能僅適用於 Windows 7、Windows 8.1 和 Windows 10。 最新發布的操作系統或預發布的操作系統版本可能不會立即得到支援。

系統硬體需求

  • 顯示卡： 在 PC/Windows 上，Roblox 應用程式需要 DirectX 10 或更高功能等級的支援。 為了獲得最佳性能，我們建議使用 5 年以下配備專用顯卡的電腦，或 3 年以下配備集成顯卡的筆記本電腦。
  • 處理器： Roblox 建議您使用時脈速度為 1.6 Ghz 或更高的最新處理器 (2005+)。 舊 AMD 處理器會出現一些問題。
  • RAM 或記憶體 Roblox 建議您在 Windows 7、Windows 8.1 或 Windows 10 上至少擁有 1 GB 內存
  • 儲存空間： Roblox 建議您至少有 20 Mb 的系統儲存空間來安裝 Roblox。
  • 行動版： 按此 了解 Roblox Mobile 的系統需求。

網路需求

對於單次使用的家庭，Roblox 建議使用最少為 4-8 Mbps 的網絡連接。 在較大的家庭中，更高帶寬是可取的。
