Configurar el estilo de chat de tu sitio

Tu lugar se puede configurar para mostrar cómo aparecen los mensajes de chat en la experiencia. Esto permite opciones flexibles para que los jugadores se comuniquen mejor mientras juegan a tu experiencia. El sistema de chat de texto en la experiencia permite a los usuarios comunicarse y socializar fácilmente entre sí en experiencias en vivo.

Consulta el artículo Documentación para creadores del chat de texto en la experiencia para obtener información detallada.
