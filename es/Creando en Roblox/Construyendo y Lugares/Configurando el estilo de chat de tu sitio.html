Configurando el estilo de chat de tu sitio

Su lugar se puede configurar para mostrar cómo se muestran los mensajes de chat en la experiencia. Esto permite opciones flexibles para permitir que los jugadores se comuniquen mejor mientras juegan su experiencia. El sistema de chat de texto en experiencia permite a los usuarios comunicarse y socializar fácilmente entre sí en experiencias en vivo.

Consulta el artículo Documentación para creadores de in-experience Text Chat para obtener información detallada.
