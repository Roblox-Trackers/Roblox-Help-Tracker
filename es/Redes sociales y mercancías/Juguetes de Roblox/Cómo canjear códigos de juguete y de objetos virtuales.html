Cómo canjear códigos de juguete y de objetos virtuales

¿Dónde encuentro el código para el objeto virtual?

  • Para las figuras de acción de Jazwares: el código se encuentra en la parte frontal del paquete en una ficha o en una tarjeta de código pegada en el interior del embalaje de plástico. Mira el video de Jazwares sobre dónde encontrar el código del objeto virtual.
  • Para pistolas Nerf y Monopoly: el código se encuentra en el paquete en el inserto de la tarjeta de código.

  • Para todos los demás códigos de objetos virtuales (por ejemplo, Prime Gaming, Verizon, etc.), consulta las instrucciones de canje a continuación.

¿Cómo canjeo el código para el objeto virtual?

Nota: si tienes un código digital, salta al paso 2.

  1. Si el código está cubierto, raspa suavemente la cubierta.
  2. Asegúrate de haber iniciado sesión en tu cuenta de Roblox en la que deseas canjear el código.
  3. Ve a a la página de canje de códigos .
  4. Introduce el código en la casilla.
  5. Haz clic en Canjear .
  6. Echa un vistazo a tu increíble objeto en el inventario de tu cuenta. Asegúrate de marcar la categoría de inventario correcta (por ejemplo, sombrero, mochila, cabeza, accesorios, etc.).

Nota: un código de objeto virtual solo se puede canjear en la cuenta de una persona una sola vez. Si tienes un código para un objeto virtual que ya tienes en tu cuenta, puedes compartir el código con otro usuario. Estos objetos no se pueden intercambiar ni vender.
