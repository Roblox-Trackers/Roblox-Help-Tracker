¿Cómo puedo canjear un código de promoción?

¿Cómo puedo obtener un código promocional de Roblox?

Puedes obtener un Código Promocional<0> de Roblox de nuestros eventos u obsequios. Lee Como Puedo Obtener Objetos Promocionales o de Concurso? para más información de como obtener objetos promocionales.

Nota: recuerde que los códigos de promoción pueden caducar o estar activos solo por un corto período de tiempo, así que asegúrese de usar su código de inmediato.

¿Cómo puedo canjear mi código de promoción para el artículo virtual?

  1. Asegúrese de haber iniciado sesión en su cuenta de Roblox en la que desea canjear el código.
  2. Ve a Canjear Códigos de Roblox .
  3. Ingrese su código en el cuadro debajo de Ingresa tu código .
  4. Hacer clic Canjear .
  5. Aparecerá un mensaje de éxito una vez que haya canjeado con éxito el código.
  6. Obtendrá crédito, Robux o un elemento virtual. Depende del tipo de Código que haya canjeado.

Problemas al Canjear Códigos Promocionales

Si estás experimentando algún problema al canjear un código promocional, por favor intenta lo siguiente:

  1. Asegúrate de haber iniciado sesión en tu cuenta de Roblox antes de canjear el código
  2. Navega a la página de canje de código promocional
  3. Asegúrate de ingresar el código correctamente
  4. Revisa tu inventario para confirmar si ya tienes el objeto exclusivo

Nota: Los códigos promocionales usualmente están disponibles por tiempo limitado. Si el periodo de tiempo para un código en particular ha finalizado, no podrás canjear el código.
