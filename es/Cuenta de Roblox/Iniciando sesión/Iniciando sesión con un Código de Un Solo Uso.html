Iniciando sesión con un Código de Un Solo Uso

Un código de Un Solo Uso le permite iniciar sesión sin ingresar su nombre de usuario y contraseña. En este momento, los códigos de un solo uso solo están disponibles para algunos usuarios.

Para utilizar esta función, debe tener una dirección de correo electrónico verificada en su cuenta. Puede ir a la página Información de su cuenta para agregar y verificar su dirección de correo electrónico (o el correo electrónico de los padres para usuarios menores de 13 años). Puedes leer este artículo de ayuda sobre cómo verificar la dirección de correo electrónico de la cuenta.

Cómo iniciar sesión con un Código de Un Solo Uso

  1. Pulse el botón Enviarme un código de un solo uso por correo electrónico en la página de inicio de sesión.
  2. Ingrese la dirección de correo electrónico que está verificada en su cuenta. Le enviaremos por correo electrónico un código único de seis dígitos.
  3. Ingrese el código único para iniciar sesión (si su correo electrónico está verificado en varias cuentas, puede elegir una).

No recibí un código de Un Solo Uso

Pruebe estos pasos si no recibió un correo electrónico que contenga un código de un solo uso.

  • Revisa tu carpeta de spam/basura.
  • Presione el botón Reenviar, luego revise su bandeja de entrada nuevamente.
  • Asegúrate de ingresar la dirección de correo electrónico exactamente como está en la cuenta.
  • Asegúrate de que tu dirección de correo electrónico esté verificada. Puedes leer este artículo de ayuda sobre cómo verificar la dirección de correo electrónico de la cuenta.

Si ninguno de estos pasos funciona, inicie sesión con su contraseña. Es posible que deba esperar un tiempo antes de que los códigos de un solo uso vuelvan a funcionar para su correo electrónico.

Recibí un Código Único pero no funciona

Pruebe estos pasos si su código único no funciona.

  1. Asegúrate de ingresar el código de un solo uso en el mismo dispositivo desde el que lo solicitaste.
  2. Pulsa el botón Reenviar e inténtalo de nuevo con un nuevo código.

Si ninguno de estos pasos funciona, inicie sesión con su contraseña. Es posible que deba esperar un tiempo antes de que los códigos de un solo uso vuelvan a funcionar para su correo electrónico.

Consejos de seguridad

  • Nunca comparta un código de un solo uso con nadie. Roblox nunca se comunicará contigo para pedirlo.
  • Si recibe un correo electrónico que contiene un código de un solo uso sin solicitarlo, puede ignorarlo con seguridad.
  • Si recibe varios correos electrónicos no deseados que contienen códigos de un solo uso, le recomendamos encarecidamente que cambie el correo electrónico de su cuenta y configure la verificación en dos pasos mediante la aplicación Authenticator .
