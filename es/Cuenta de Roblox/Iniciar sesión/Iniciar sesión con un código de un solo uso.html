Iniciar sesión con un código de un solo uso

Un código de un solo uso te permite iniciar sesión sin ingresar su nombre de usuario y contraseña. En este momento, los códigos de un solo uso solo están disponibles para algunos usuarios.

Para utilizar esta función, debes tener una dirección de correo electrónico verificada en tu cuenta. Cómo iniciar sesión con un código de un solo uso

  1. Presiona el botón Enviarme un código de un solo uso por correo electrónico en la página de Inicio de sesión.
  2. Ingresa la dirección de correo electrónico que está verificada en tu cuenta. Te enviaremos por correo electrónico un código único de seis dígitos.
  3. Ingresa el código de un solo uso para iniciar sesión (si tu correo electrónico está verificado en varias cuentas, puedes elegir una).

No recibí un código de un solo uso

Prueba estos pasos si no recibiste un correo electrónico que contenga un código de un solo uso.

  • Revisa tu carpeta de spam/basura.
  • Presiona el botón Reenviar, luego revisa tu bandeja de entrada nuevamente.
  • Asegúrate de ingresar la dirección de correo electrónico exactamente como está en la cuenta.
  • Asegúrate de que la dirección de correo electrónico de la cuenta esté verificada. Puedes leer este artículo de ayuda sobre cómo verificar la dirección de correo electrónico de la cuenta.

Si ninguno de estos pasos funciona, inicia sesión con tu contraseña. Es posible que debas esperar un tiempo antes de que los códigos de un solo uso vuelvan a funcionar para tu correo electrónico.

Recibí un código de un solo uso, pero no funciona

Prueba estos pasos si tu código de un solo uso no funciona.

  1. Asegúrate de ingresar el código de un solo uso en el mismo dispositivo desde el que lo solicitaste.
  2. Presiona el botón Reenviar y vuelve a intentarlo con un nuevo código.

Si ninguno de estos pasos funciona, inicia sesión con tu contraseña. Es posible que debas esperar un tiempo antes de que los códigos de un solo uso vuelvan a funcionar para tu correo electrónico.

Consejos de seguridad

  • Nunca compartas un código de un solo uso con nadie. Roblox nunca se comunicará contigo para pedírtelo.
  • Si recibes un correo electrónico que contiene un código de un solo uso sin solicitarlo, puede ignorarlo.
  • Si recibes varios correos electrónicos no deseados que contienen códigos de un solo uso, te recomendamos encarecidamente que cambies el correo electrónico de tu cuenta y configures la verificación en dos pasos mediante la aplicación Autenticador .
