Mensajes de Hyperion

Este artículo contiene potenciales mensajes de fallos antitrampas de Hyperion que podrías encontrar, seguidos de los pasos de solución de errores correspondientes para ayudarte a resolverlos.

Tipo de fallo : Fallo de límite de tiempo

Mensaje de usuario : La sesión de Roblox expiró. Reinicia tu cliente.

Pasos de solución de errores :

Si te encuentras con un mensaje que dice "La sesión de Roblox expiró. Reinicia tu cliente", significa que necesitas reiniciar tu cliente de Roblox. Si el problema persiste a pesar de reiniciar el cliente durante el uso regular, contacta con nuestro equipo de soporte para obtener más ayuda.

Tipo de fallo : Máquina virtual detectada

Mensaje de usuario : Roblox no se puede ejecutar en una máquina virtual.

Pasos de solución de errores :

Si ves el mensaje "Máquina virtual detectada", significa que tu máquina virtual (o VM, por sus siglas en inglés) no es compatible con Roblox. Roblox requiere transferencia de GPU para funcionar adecuadamente en una VM. La transferencia de GPU otorga acceso directo a la VM a la GPU física, omitiendo la capa de hipervisor. Sigue estos pasos para activar la transferencia de GPU:

      1. Confirma que tu hardware es compatible con la transferencia de GPU.
      2. Actualiza tus ajustes de BIOS/UEFI para habilitar las extensiones de virtualización (p. ej., Intel VT-x o AMD-V).
      3. Configura la transferencia de GPU en tu software de VM.

Seguir estos pasos asegura que Roblox funcione correctamente en un entorno virtual, otorgando el acceso necesario a los recursos de GPU.

Tipo de fallo : Badware detectado

Mensaje de usuario : Roblox falló debido a software incompatible: <ruta al software infractor>

Pasos de solución de errores :

Si ves un mensaje que dice "Roblox falló debido a software incompatible:" seguido de una ruta al software infractor, significa que el software mencionado en la ruta del archivo mostrado está causando problemas con Roblox. Para arreglar esto, necesitarás prevenir que el software se ejecute mientras usas Roblox. Puedes hacer esto manualmente al cerrar el software, o puedes realizar un arranque limpio en Windows. Este método asegura que solo el software necesario esté activo en tu sistema, minimizando los conflictos. Sigue estos pasos para completar el inicio limpio como lo recomienda Windows :

      1. Presiona la "tecla Windows + R" para abrir el diálogo Ejecutar.
      2. Escribe "msconfig" y presiona Intro.
      3. En la ventana Configuración del sistema, ve a la pestaña "Servicios".
      4. Marca la casilla etiquetada con "Ocultar todos los servicios de Microsoft".
      5. Haz clic en "Deshabilitar todos".
      6. Cambia a la pestaña "Inicio de Windows".
      7. Haz clic en "Abrir el Administrador de tareas".
      8. Deshabilita todos los elementos de inicio.
      9. Cierra el administrador de tareas y la configuración del sistema.
      10. Reinicia tu computadora.

Completar estos pasos te ayudará a asegurarte de que ningún otro software interfiera con Roblox, lo que le permitirá funcionar sin fallos.

Tipo de fallo : Integridad de arranque comprometida

Mensaje de usuario : Los archivos críticos del sistema están corruptos. Roblox no se puede ejecutar. Repara o reinicia tu instalación de Windows.

Pasos de solución de errores :

Roblox no funciona adecuadamente cuando Windows está configurado en el modo de depuración. Para usar Roblox, asegúrate de que Windows funciona con normalidad. Sigue estos pasos para verificar y deshabilitar el modo de depuración en Windows .

      1. Presiona la "tecla Windows+R" para abrir el recuadro Ejecutar.
      2. Escribe "msconfig" y presiona Intro.
      3. Selecciona la "pestaña de Arranque" y selecciona las Opciones avanzadas.
      4. Desmarca la casilla Depurar.
      5. Selecciona "Aceptar".
      6. Selecciona Aplicar y después "Aceptar".
      7. Reinicia la computadora.

Estos pasos asegurarán que Windows no se ejecute en el modo de depuración, lo que te permitirá usar Roblox sin fallos.

Tipo de fallo : Open Broadcaster Software (OBS)

Mensaje de usuario : La captura de juego de OBS suele presentar fallos con Roblox. En su lugar, usa la captura de ventana o escritorio.

Pasos de solución de errores :

Si recibes un mensaje que dice "La captura de juego de OBS suele presentar fallos con Roblox", significa que Roblox no es compatible con la función de captura de juego de Open Broadcaster Software (OBS). Para transmitir o grabar Roblox, en su lugar, necesitarás usar el modo de captura de ventana o escritorio. Sigue estas instrucciones para configurar la captura de escritorio :

      1. Abre Open Broadcaster Software (OBS).
      2. En la sección "Fuentes", haz clic en el ícono "+".
      3. Selecciona "Captura de pantalla" desde el menú.
      4. Elige la pantalla que quieres capturar (por lo general, tu monitor principal).
      5. Configura los ajustes como resolución y cuadros por segundo conforme lo necesites.
      6. Haz clic en "Aceptar" para confirmar la fuente de captura de pantalla.
      7. Arregla la fuente de captura de pantalla como lo desees en tu escena de OBS.
      8. Comienza a transmitir o grabar de forma usual.

Al usar el modo de captura de escritorio, puedes transmitir o grabar Roblox sin los fallos de compatibilidad relacionados con la función de captura de juego de OBS.
