Hardware de la Computadora y Requisitos del Sistema Operativo

Los requisitos mínimos del sistema para Roblox se enumeran a continuación. Si tiene problemas para jugar, es posible que deba actualizar su computadora. Lea este artículo de ayuda para conocer los requisitos del sistema para dispositivos móviles..

Requisitos del sistema operativo

  • PC/Windows: Roblox se instalará en Windows 7, Windows 8 / 8.1 o Windows 10. Para Windows 8 / 8.1, deberá ejecutar Roblox en modo de escritorio, ya que el modo Metro (la pantalla de inicio en mosaico) no es compatible actualmente.
    Nota: El sitio web de Roblox es compatible con los siguientes navegadores para PC / Windows: Chrome , Firefox , y Microsoft Edge .
  • Mac: Roblox Client se instalará en 10.7 (Lion) y superior, mientras que Roblox Studio se instalará en Mac OS 10.11 (El Capitan) y superior.
    Nota: El sitio web de Roblox es compatible con los siguientes navegadores para Mac: Chrome , Firefox , y Safari .
  • Linux: Roblox no es compatible con Linux.
  • Mobile: Click aqui para conocer los requisitos del sistema para Roblox Mobile.
  • Chrome OS : Se requiere Chrome OS versión 53 o superior, y primero debe habilitar Google Play Store en su dispositivo.

Nota: La función de grabación de video de Roblox solo funciona en Windows 7, Windows 8 / 8.1 y Windows 10. Es posible que los sistemas operativos recién lanzados o las versiones preliminares de los sistemas operativos no sean compatibles de inmediato.

Requisitos de hardware del sistema

  • Tarjeta gráfica : en PC / Windows, la aplicación Roblox requiere compatibilidad con DirectX 10 o un nivel de función superior. Para obtener el mejor rendimiento, recomendamos una computadora de menos de 5 años con una tarjeta de video dedicada o una computadora portátil de menos de 3 años con una tarjeta de video integrada.
  • Procesador : Roblox recomienda que tenga un procesador reciente (2005+) con una velocidad de reloj de 1.6 Ghz o mejor. Ha habido algunos problemas con los procesadores AMD más antiguos.
  • RAM or Memoria: Roblox recomienda que tenga al menos 1 GB de memoria en Windows 7, Windows 8 o Windows 10
  • Espacio de almacenamiento: Roblox recomienda que tenga al menos 20 Mb de espacio de almacenamiento del sistema para instalar Roblox.
  • Móvil: haga click aqui para conocer los requisitos del sistema para Roblox Mobile.

Requisitos de Internet

Para un hogar de un solo uso, Roblox recomienda una conexión a Internet mínima de 4-8 Mb / s. En hogares más grandes, se recomienda un mayor ancho de banda.
