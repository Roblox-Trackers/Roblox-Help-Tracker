Cómo publicitar un grupo

Una vez que hayas creado un increíble grupo nuevo, ¡quizás quieras ayudar a hacer correr la voz para comenzar a atraer nuevos miembros! Los anuncios son una buena forma de hacerlo y anunciar un grupo es similar a anunciar otros artículos en Roblox.

¿Cómo crear un anuncio?

Cualquier miembro del grupo que tenga permiso para Advertise the Group (Publicitar el Grupo) (haz clic aquí para obtener más información sobre permisos) podrá crear un anuncio.

Para comenzar, vé a la página del grupo y haz clic en el botón Advertise Group (Anunciar Grupo) que se encuentra a la derecha para abrir la página Create a User Ad (Crear un Anuncio de Usuario). Desde allí:

  1. Elije el tamaño del anuncio que deseas hacer. Actualmente hay tres opciones. Cada uno tiene una plantilla que puedes descargar para ayudarte a comenzar:

    Descarga 728 x 90 Modelo Bandera

    Descarga 160 x 600 Modelo Rascacielos

    Descarga 300 x 250 Modelo Rectangulo Largo

  2. Utiliza un software de edición de imágenes para crear un anuncio atractivo.
  3. Guarda el anuncio personalizado en tu computadora.
  4. De vuelta en la página Create a User Ad , haz clic en el botón Choose File (Elegir archivo).
  5. Encuentra y selecciona la imagen que creaste en el paso dos.
  6. Ingresa un nombre y haz clic en el botón verde Upload (Cargar).

El anuncio que subiste será revisado por nuestro equipo de moderadores para garantizar que se adhiere a las Normas de la Comunidad de Roblox . Por favor espera varias horas para este proceso. Una vez que tu anuncio haya sido aprobado, podrás lanzar el anuncio desde tu página de Ad Inventory (Inventario de Anuncios). Para obtener más información sobre de cómo ejecutar un anuncio y cómo funciona el sistema de anuncio, haz clic aquí .

Nota: puedes hacer una oferta en tu anuncio con fondos del grupo si se le has otorgado el permiso para Spend Group Funds (Fondos de Gastos del Grupo).
