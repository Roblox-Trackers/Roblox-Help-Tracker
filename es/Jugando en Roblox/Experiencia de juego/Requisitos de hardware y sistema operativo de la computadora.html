Requisitos de hardware y sistema operativo de la computadora

Los requisitos mínimos del sistema para Roblox se enumeran a continuación. Si tiene problemas para jugar la experiencia, es posible que deba actualizar su computadora. Lea este artículo de ayuda para conocer los requisitos del sistema para dispositivos móviles.

Requisitos del sistema operativo

  • PC/Windows: Roblox se instalará en Windows 7, Windows 8/8.1 o Windows 10. Para Windows 8/8.1, deberá ejecutar Roblox en modo de escritorio, ya que el modo Metro (la pantalla de inicio en mosaico) no es compatible actualmente.
    Nota: El sitio web de Roblox es compatible con los siguientes navegadores para PC/Windows: Chrome , Firefox y Microsoft Edge .
  • Mac: Roblox Client se instalará en 10.10 (Yosemite) y superior, mientras que Roblox Studio se instalará en Mac OS 10.11 (El Capitan) y superior.
    Nota: El sitio web de Roblox es compatible con los siguientes navegadores para Mac: Chrome , Firefox y Safari .
  • Linux: Roblox no es compatible con Linux.
  • Móvil: Haga clic aquí para conocer los requisitos del sistema para Roblox Mobile.
  • Chrome OS: Se requiere Chrome OS versión 53 o superior, y primero debe habilitar Google Play Store en su dispositivo.

Nota: la función de grabación de video de Roblox solo funciona en Windows 7, Windows 8/8.1 y Windows 10. Es posible que los sistemas operativos recientemente lanzados o las versiones preliminares de los sistemas operativos no sean compatibles de inmediato.

Requisitos de hardware del sistema

  • Tarjeta gráfica : En PC/Windows, la aplicación Roblox requiere compatibilidad con DirectX 10 o superior. Para obtener el mejor rendimiento, recomendamos una computadora de menos de 5 años con una tarjeta de video dedicada o una computadora portátil de menos de 3 años con una tarjeta de video integrada.
  • Procesador : Roblox recomienda tener un procesador reciente (2005+) con una velocidad de reloj de 1,6 Ghz o superior. Ha habido algunos problemas con los procesadores AMD más antiguos.
  • RAM or Memoria: Roblox recomienda tener al menos 1 GB de memoria en Windows 7, Windows 8 o Windows 10
  • Espacio de almacenamiento: Roblox recomienda tener al menos 20 Mb de espacio de almacenamiento en el sistema para instalar Roblox.
  • Móvil: haga click aqui para conocer los requisitos del sistema para Roblox Mobile.

Requisitos de Internet

Para un hogar de un solo uso, Roblox recomienda una conexión a Internet mínima de 4-8 Mb/s. En hogares más grandes, se recomienda mayor ancho de banda.
