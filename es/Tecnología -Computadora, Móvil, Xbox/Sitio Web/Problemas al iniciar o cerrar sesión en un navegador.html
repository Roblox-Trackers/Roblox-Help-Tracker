Problemas al iniciar o cerrar sesión en un navegador

Si tienes problemas para iniciar o cerrar sesión en Roblox en un navegador, primero intenta seguir estos pasos para resolver los problemas generales del sitio web .

Si necesitas ayuda adicional, intenta estos pasos:

  1. Si tienes dificultades con la verificación de captcha o el cuadro de captcha no se carga, asegúrate de incluir la lista de dominios permitidos las siguientes direcciones:
    • .arkoselabs.com
    • .funcaptcha.com
  2. Si los pasos anteriores no te ayudaron a resolver el problema , asegúrate de que el reloj de tu computadora tenga la hora correcta. Por extraño que parezca, si el reloj de la computadora está fuera de hora, esto puede conducir a varios problemas. Las siguientes páginas pueden ayudarte con este proceso:
    • Windows (De Microsoft.com)
    • Mac (De Apple.com)
  3. Si aún no puede iniciar sesión, es posible que debas restablecer tu contraseña .
