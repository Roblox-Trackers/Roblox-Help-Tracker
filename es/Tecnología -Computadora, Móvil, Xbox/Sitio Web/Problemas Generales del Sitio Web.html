Problemas Generales del Sitio Web

El siguiente artículo sugerirá algunos métodos que han demostrado su eficacia para solucionar los siguientes problemas:

  • El sitio web se muestra incorrectamente
  • Algunas páginas no se cargan o muestran un error al cargar
  • Los botones o enlaces no le permiten hacer clic en ellos
  • Intenta iniciar sesión en el sitio web de Roblox pero no puede, a pesar de tener el nombre de usuario y la contraseña correctos
  • Otro: Si experimenta un problema diferente con el sitio web, los pasos a continuación también pueden ayudar a resolverlo. Además, algunos artículos no relacionados con el sitio web pueden enlazar aquí para algunos pasos

Qué hacer?

Por favor, prueba cada paso a continuación, uno a la vez, en orden. Si una solución no funciona, pasa a la siguiente.

Borra los archivos temporales de internet y cookies

Los siguientes enlaces te ayudarán a eliminar los archivos temporales de Internet (también denominados "caché") y las cookies para los principales navegadores web de PC. Asegúrate de hacer esto para Internet Explorer, así como para cualquier otro navegador desde el que desees iniciar Roblox.

      • Internet Explorer - Archivos Temporales de Internet (De Microsoft.com)
      • Internet Explorer - Cookies (De Microsoft.com)
      • Mozilla Firefox - Archivos Temporales de Internet (De Mozilla.com)
      • Mozilla Firefox - Cookies (De Mozilla.com)
      • Google Chrome - Ambos (De Google.com)
      • Microsoft Edge - Ambos (De Microsoft.com)
      • Safari - Cookies (De Apple.com)
      • Otros navegadores: Si está utilizando un navegador diferente o una versión anterior de un navegador, busque información sobre cómo eliminar sus datos de caché/navegador/archivos temporales de Internet para su navegador/versión específica.

Una vez que haya completado lo anterior para todos los navegadores web relevantes, reinicie su computadora y vea si su problema se ha resuelto.

Si desea eliminar manualmente sus archivos temporales de Internet Explorer/Windows Internet, puede hacerlo yendo directamente a la carpeta Archivos temporales de Internet en su disco duro. La ubicación y el nombre de la carpeta dependen de la versión de Windows y de si está usando perfiles de usuario.

Nota: Es posible que necesites cambiar la configuración de Windows Explorer para mostrar todo tipo de archivos (incluidos los archivos del sistema protegido) para acceder a estas carpetas.

Para Windows 10

Si tienes Windows 10, los archivos temporales de Internet están en la siguiente ubicación (ten en cuenta que en tu PC pueden estar en otra unidad en lugar de en la unidad C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Para Windows 7 y 8

Si tiene Windows 7 o Windows 8, los archivos temporales de Internet se encuentran en estas ubicaciones (tenga en cuenta que pueden estar en otra unidad en lugar de la unidad C):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\

Deshabilite los complementos de su navegador web

Muchas veces, los problemas relacionados con el sitio web son el resultado de complementos de terceros para su navegador web. Estos también pueden denominarse "Extensiones" o "Complementos" según el navegador que elija. Para ver si este es su caso, intente deshabilitar todos sus complementos. Si el problema se resuelve, intente volver a habilitarlos uno por uno hasta que el problema comience a ocurrir nuevamente. Una vez que el problema comienza de nuevo, debería haber encontrado el que es incompatible con Roblox.
