Mac: Solución de problemas

Desinstalar/Reinstalar

Si tiene un problema y no ha intentado volver a instalarlo recientemente, intente desinstalar el programa y vuelva a instalarlo siguiendo los pasos que se encuentran en la parte inferior de los dos artículos siguientes:

  • Desinstalar
  • Reinstalar

Otras Cosas para Probar

Si la desinstalación y la reinstalación no ayudaron, intenta lo siguiente

  • Asegúrate de estar utilizando una de las versiones de Mac OS compatibles: 10.7 y superior.
  • Actualmente, solo se admite el chipset Intel. El Motorola Power PC no es compatible.
  • Si actualmente estás utilizando el navegador web Safari y tienes un problema, intenta utilizar uno de nuestros navegadores compatibles como Internet Explorer, Firefox o Google Chrome.
  • Verifica que el icono de Roblox esté actualmente en tu Dock y que la Aplicación Roblox aparezca en tu carpeta de Aplicaciones.
  • Intenta ejecutar el programa desde el Dock si tienes problemas para jugar desde el sitio web.
  • Intenta restablecer completamente la configuración de Roblox eliminando los archivos de configuración. Para hacer esto, vé a Library (Biblioteca)-> Preferencias y elimina todos los archivos de Roblox com. Cuando Studio/Player se inicie nuevamente, debería reinstalarlos todos.
