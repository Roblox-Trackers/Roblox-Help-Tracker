Nachricht erhalten: Robux von meinem Konto entfernt

Ein Betrag von Robux kann von einem Konto entfernt werden, nachdem eine Anforderung zur Rückerstattung für einen entsprechenden Kauf bearbeitet wurde. Das Benutzerkonto wird über eine direkte Nachricht benachrichtigt, die von Roblox an den Posteingang des Kontos gesendet wird. In dieser Nachricht werden Sie darüber unterrichtet, dass eine Gebühr erstattet wurde und dass die von dieser Gebühr stammenden Robux von Ihrem Konto entfernt wurden.

Gegen Konten, die das Rückerstattungssystem missbrauchen oder nicht autorisierte Käufe abschließen, können zusätzliche Moderationsmaßnahmen ergriffen werden, da es sich hierbei um einen Verstoß gegen unsere Nutzungsbedingungen handelt.  Ein Konto kann auch für einen bestimmten Zeitraum gesperrt oder von der Roblox-Website entfernt werden. Weitere Informationen zu den Roblox-Rückerstattungsrichtlinien entnehmen Sie bitte unseren Nutzungsbedingungen .

Wenn Sie weitere Fragen oder Bedenken zu dieser Nachricht oder zum Entfernen Ihres Robux haben, bitten wir Sie, sich jederzeit an den Kundendienst zu wenden.
