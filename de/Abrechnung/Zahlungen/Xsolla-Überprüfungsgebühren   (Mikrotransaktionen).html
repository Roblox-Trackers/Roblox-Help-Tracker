Xsolla-Überprüfungsgebühren (Mikrotransaktionen)

Aus Sicherheitsgründen werden Sie möglicherweise gebeten, Ihre Transaktion bei unserem Zahlungsanbieter Xsolla zu bestätigen, damit die Benutzern auf der ganzen Welt sichere Zahlungen ermöglicht werden. Die Kartenüberprüfung ist eine gängige Praxis, um die Zahlungssicherheit zu gewährleisten.

Wie funktioniert der Verifizierungsprozess:

Xsolla erstellt eine Mikrotransaktion auf Ihrem Finanzkonto (normalerweise zwischen 0,20 USD und 0,90 USD). Der Betrag dieser Mikrotransaktion ist der Bestätigungscode, den Sie in das Formular eingeben müssen. Sie sollten nur den numerischen Wert der Gebühr eingeben. Wenn Ihr Code 0,44 US-Dollar beträgt, sollten Sie im Formular 0,44 eingeben, um Ihre Transaktion zu bestätigen.

  • Bestätigungscodes sind für einen Zeitraum von 3 Stunden gültig.
  • Wenn Sie mehrere Codes erhalten, stellen Sie sicher, dass Sie den neuesten Code verwenden.
  • Wenn Ihr Code abläuft, wird ein neuer Code generiert und Ihnen zugestellt, nachdem Sie versuchen, eine weitere Zahlung mit derselben Karte vorzunehmen.

Was passiert, wenn ich meinen Kauf nicht bestätige:

Der Kauf wird abgelehnt und der Versuch, den Kauf zu tätigen, wird nicht in Rechnung gestellt. Jedoch sollten Sie bitte beachten, dass die Transaktionen bei Ihrem Finanzinstitut möglicherweise eine Sicherheitswarnung auslösen und das Geld von dem Kauf kann für einige Tage in einem ausstehenden "Halten" -Status gehalten werden. Bitte beachten Sie, dass einige Banken und Kreditkartenunternehmen nicht die Differenz zwischen "ausstehenden" und "gebuchten" Gebühren aufweisen.

Diese Sicherheitsgebühren werden nicht an Roblox weitergeleitet. Ausstehende Kosten sollten innerhalb von 1 bis 10 Werktagen auf Ihr Konto übertragen werden, sofern dies noch nicht geschehen ist.

Was passiert mit der Verifizierungsgebühr:

Der Betrag der Mikrotransaktion wird Ihrem Konto automatisch innerhalb von 1 bis 10 Werktagen gutgeschrieben, unabhängig davon, ob Sie den Kauf mit Xsolla abschließen oder nicht.

Weitere Informationen zum Überprüfungsprozess finden Sie auf der Xsolla FAQ -Seite.
