Appellieren Sie an Ihre Inhalt-Moderation

Sie können sich an das Einspruchsteam wenden, um die Situation zu überprüfen, wenn einer Ihrer Inhalte blockiert oder gelöscht wurde. Inhalte sind Dinge, die Sie erstellt haben, wie z. B. Erfahrungen, Abziehbilder, Kleidung, Gruppenlogos oder Audiodateien. Obwohl die meisten Inhalte nach der Moderation normalerweise nicht wiederhergestellt werden können, wird das Beschwerdeteam sein Bestes tun, um eine zufriedenstellende Lösung zu finden.


Moderation anfechten:

Wenden Sie sich über das Support-Formular hier an das Appeals-Team. Stellen Sie sicher, dass Folgendes enthalten ist:

  • Benutzernamen
  • Kurze Aussage zum Grund Ihrer Beschwerde
  • Ein Link zum Inhalt der Roblox-Website
  • Wenn Sie Einspruch auf eine Audiodatei einlegen, fügen Sie Ihrer Beschwerde eine Kopie der hochgeladenen Audiodatei bei.

Beim Einreichen einer Beschwerde für einen Vermögenswert bestehen einige Einschränkungen :

  • Sobald eine Moderation Aktion durchgeführt wurde, kann sie nicht erneut überprüft werden. Die Doppelte Beschwerde für die gleiche Moderation Aktion auf einem Konto werden nicht überprüft.
  • Um die Geheimhaltung unserer Benutzer zu schützen, kann nur der Eigentümer eines moderierten Inhalts eine Beschwerde für den Inhalt senden. Informationen zur Einspruch Einlegung von Inhalten, die einem Freund gehören, finden Sie unter Kann ich für einen Freund Einspruch einlegen?
  • Stellen Sie sicher, dass Sie sich an unserem Appell-Team,über eine gültige E-Mail-Adresse für das Konto wenden, welche zum Inhalt des Einspruches zugeordnet werden kann.



Um Einspruch auf die Appell Konto-Moderation einzulegen, lesen Sie die Informationen zum Einspruch der Konto-Moderation einlegen .
