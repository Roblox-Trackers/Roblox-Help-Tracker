So entsperren Sie einen Benutzer

Befolgen Sie diese Schritte, wenn Sie versehentlich einen Freund blockiert haben oder nur jemanden aus Ihrer Sperrliste entfernen möchten:

  1. Ins Konto einloggen
  2. Gehen Sie zu den Kontoeinstellungen
    • Browser - das Zahnradsymbol in der oberen rechten Ecke der Site
    • Mobile Apps - das Drei-Punkte-Symbol für More
  3. Wählen Sie die Registerkarte Datenschutz
  4. Blockierte Benutzer am unteren Bildschirmrand anzeigen
  5. Wählen Sie neben dem Spielernamen Nicht mehr sperren
  6. Wählen Sie Speichern

Entsperren funktioniert nicht!

Dies wird dadurch verursacht, dass die Konto-PIN-Funktion in Ihrem Konto aktiviert ist. Sie müssen diese Schritte ausführen, bevor Sie jemanden entsperren können.

  1. Ins Konto einloggen
  2. Gehen Sie zu den Kontoeinstellungen
    • Browser - das Zahnradsymbol in der oberen rechten Ecke der Site
    • Mobile Apps - das Drei-Punkte-Symbol für More
  3. Wählen Sie die Registerkarte Sicherheit
  4. Deaktivieren Sie die Konto-PIN-Funktion (Ihre Eltern müssen möglicherweise helfen)
  5. Wählen Sie die Registerkarte Datenschutz
  6. Blockierte Benutzer am unteren Bildschirmrand anzeigen
  7. Wählen Sie neben dem Spielernamen Nicht mehr sperren
  8. Wählen Sie Speichern
  9. Kehren Sie zur Registerkarte Sicherheit zurück und aktivieren Sie die Konto-Pin-Funktion erneut
