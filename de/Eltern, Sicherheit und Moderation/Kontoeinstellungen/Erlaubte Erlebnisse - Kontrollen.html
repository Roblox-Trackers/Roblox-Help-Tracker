Erlaubte Erlebnisse - Kontrollen

Roblox-Erlebnisrichtlinien, die Altersempfehlungen für die inhaltliche Eignung eines Erlebnisses enthalten. Auf der Grundlage dieser Altersempfehlungen können Benutzer und Eltern fundierte Entscheidungen über die Inhalte treffen, mit denen sie interagieren. Eltern können mit Erlaubten Erlebnisse den Zugriff auf die Inhalte verwalten, auf die ihr Kind Zugriff hat. Eltern können mehr über unsere Kindersicherung in unserem Elternleitfaden erfahren.

Was geschieht, wenn Erlaubte Erlebnisse eingestellt sind?

Eltern können altersgerechte Erlebnisse auswählen, indem sie Erlaubte Erlebnisse aktivieren. Dadurch kann Ihr Kind nur an Erlebnissen teilnehmen, die den eingestellten Altersempfehlungen entsprechen oder darunter liegen. Erlebnisse, die über die aktivierte Altersbeschränkung hinausgehen, können in den Suchergebnissen angezeigt oder in den Anzeigen beworben werden. Die Erlebnisse können jedoch nicht angezeigt werden. Die Benutzer erhalten eine Meldung, dass das Erlebnis aufgrund der Kontobeschränkungseinstellungen nicht verfügbar ist.

Wie wird eine Altersempfehlung festgelegt?

Die Altersempfehlungen von Roblox beruhen auf Forschungsergebnissen zur kindlichen Entwicklung und auf Branchenstandards. Um festzustellen, für welche Zielgruppen ein Erlebnis generell geeignet ist, haben wir globale Industriestandards untersucht und Experten für Kindesentwicklung konsultiert.

Wie konfiguriere ich Erlaubte Erlebnisse?

Auf einem Computer

  • Loggen Sie sich in das Konto ein (Wenn Sie keinen Zugriff auf das Konto haben, klicken Sie bitte hier )
  • Klicken Sie auf das Zahnrad-Symbol in der oberen rechten Ecke der Website
  • Wählen Sie die Option Einstellungen , die durch das Zahnradsymbol angezeigt wird.
  • Sie müssen eine Eltern-PIN eingerichtet haben, um diese Funktion zu konfigurieren. Hier sind die Schritte für die Konfiguration der Kindersicherungs-PIN
  • Wählen Sie die Option Elterliche Kontrolle
  • Wählen Sie im Abschnitt Erlaubte Erlebnisse die höchste Stufe von Erlebnissen aus, an denen das Konto teilnehmen kann.
  • Nach der Auswahl wird eine grüne Meldung angezeigt: "Erfolgreich gespeichert!''

Bitte beachten Sie: Wenn für das Konto Kontoeinschränkungen aktiviert sind, ist das Konto nur auf Inhalte für alle Altersgruppen beschränkt und die Option "Erlaubte Erlebnisse" ist deaktiviert. Die Einstellung "Kontoeinschränkungen" muss deaktiviert werden, bevor "Erlaubte Erlebnisse" angepasst werden können. Weitere Informationen finden Sie im Hilfe-Artikel Kontoeinschränkungen .

Auf einem mobilen Gerät

  • Loggen Sie sich in das Konto in der App ein (wenn Sie nicht auf das Konto zugreifen können, klicken Sie bitte hier )
  • Klicken Sie auf die Schaltfläche "Mehr" , die mit drei Punkten in der unteren rechten Ecke der Seite gekennzeichnet ist.
  • Wählen Sie im Pop-up-Menü die Option Einstellungen
  • Sie müssen eine Eltern-PIN eingerichtet haben, um diese Funktion zu konfigurieren. Hier sind die Schritte für die Konfiguration der Kindersicherungs-PIN
  • Wählen Sie im Menü auf der linken Seite des Bildschirms die Option Kindersicherung
  • Wählen Sie im Abschnitt "Erlaubte Erlebnisse" die höchste Stufe von Erlebnissen aus, an denen das Konto teilnehmen kann.
  • Nach der Auswahl wird eine grüne Meldung angezeigt: "Erfolgreich gespeichert!"

Bitte beachten Sie: Wenn für das Konto Kontoeinschränkungen aktiviert sind, ist das Konto nur auf Inhalte für alle Altersgruppen beschränkt und die Funktion "Erlaubte Erlebnisse" ist deaktiviert. Die Einstellung "Kontoeinschränkungen" muss deaktiviert werden, bevor "Erlaubte Erlebnisse" angepasst werden können. Weitere Informationen finden Sie im Hilfe- Artikel Kontoeinschränkungen .
