Benachrichtigungen Kontrollen für Ausgaben

Mit den Einstellungen für die Ausgabenbenachrichtigung können Eltern/Erziehungsberechtigte die Ausgaben ihrer Kinder in einem Roblox-Konto besser kontrollieren und einsehen, indem sie E-Mails an die verifizierte E-Mail-Adresse der Eltern/Erziehungsberechtigten eines Benutzerkontos unter 13 Jahren senden. Die Eltern/Erziehungsberechtigten können die Häufigkeit der Ausgabenbenachrichtigungen auf sichere Weise anpassen.

Was passiert, wenn Ausgabenbenachrichtigungen aktiviert sind?

Eltern/Erziehungsberechtigte können festlegen, dass Ausgabenbenachrichtigungen per E-Mail verschickt werden, wenn die Ausgaben in einem bestimmten Kalendermonat besonders hoch sind, oder dass eine E-Mail über alle ausgegebenen Beträge verschickt wird. Sie können auch festlegen, dass sie keine Benachrichtigungen erhalten.

Hinweis: Für die Kontrolle der Ausgabeneinstellungen und künftige Änderungen sind sowohl eine E-Mail-Adresse der Eltern/Erziehungsberechtigten als auch eine Eltern-PIN für das Benutzerkonto erforderlich, um die Sicherheit zu erhöhen.

Wie kann ich diese Funktion aktivieren?

  1. Melden Sie sich bei Ihrem Konto an.
    Wenn Sie sich nicht an Ihr Passwort erinnern, müssen Sie es zurücksetzen, indem Sie den Informationen unter Ich habe mein Passwort vergessen folgen.
  2. Gehen Sie zu den Kontoeinstellungen, indem Sie das Zahnradsymbol in der oberen rechten Ecke der Website auswählen.
  3. Wählen Sie die Registerkarte Kindersicherung
    • Die Eltern-/Erziehungsberechtigten-E-Mail muss verifiziert und eine Eltern-PIN für das Konto aktiviert sein. Schalten Sie die Einstellung ein oder aus.
  4. Schalten Sie die Einstellung ein oder aus.
  5. Wenn die Einstellung aktiviert ist, können Sie eine Häufigkeitseinstellung aus der Dropdown-Liste auswählen (Standard oder Alle Benachrichtigungen senden )
    • Wenn Standard ausgewählt ist:
      • Standardmäßig erhalten Eltern/Erziehungsberechtigte eine E-Mail-Benachrichtigung, wenn der Benutzer einen übermäßig hohen Betrag für den Kalendermonat erreicht. Wenn Benachrichtigungen deaktiviert sind, wird keine E-Mail gesendet
    • Wenn Alle Benachrichtigungen senden ausgewählt ist:
      • = E-Mail-Benachrichtigung wird für Transaktionen an die verifizierte E-Mail-Adresse des Kontos gesendet.

Hinweis: Bei Auswahl der Standardoption erhalten Eltern/Erziehungsberechtigte nur dann E-Mail-Benachrichtigungen, wenn die Ausgaben in einem bestimmten Kalendermonat zu hoch sind. Benutzer erhalten eine Popup-Nachricht, die sie darüber informiert, dass sie ihr monatliches Ausgabenlimit erreicht haben.
