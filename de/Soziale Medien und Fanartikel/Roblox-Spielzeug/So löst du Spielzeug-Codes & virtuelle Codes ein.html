Wie kann ich meinen Code für den virtuellen Artikel einlösen?

Hinweis: Wenn du einen digitalen Code hast, fahre mit dem zweiten Schritt fort.

  1. Wenn dein Code verdeckt ist, kratze vorsichtig den Belag ab.
  2. Stelle sicher, dass du bei dem Roblox-Konto angemeldet bist, für das du den Code einlösen möchtest.
  3. Gehe zur folgenden Seite , um deinen Geschenkgutschein einzulösen.
  4. Gib den Code im Textfeld ein.
  5. Klicke auf die Schaltfläche Einlösen .
  6. Sieh dir deinen tollen neuen Artikel im Inventar deines Kontos an. Stelle sicher, dass du die richtige Inventar-Kategorie überprüfst (z.B. Hut, Rucksack, Kopf, Accessoires usw.)

Hinweis: Der Code für einen virtuellen Artikel kann auf einem Konto nur ein einziges Mal eingelöst werden. Wenn du einen Code für einen Artikel hast, über das dein Konto schon verfügt, kannst du ihn einem anderen Nutzer oder einer anderen Nutzerin schenken. Diese Artikel können nicht getauscht oder verkauft werden.
