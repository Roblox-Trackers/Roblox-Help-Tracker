Wie man einen Spielzeug Kode einlöst

Wo finde ich meinen Kode für den virtuellen Gegenstand?

  • Für Jazwares-Actionfiguren: Der Code befindet sich auf der Vorderseite der Verpackung auf einem Token oder auf einer Codekarte, die auf der Innenseite der Kunststoffverpackung aufgeklebt ist. Schaue dir das Jazwares Video an, um zu erfahren, wo du den virtuellen Artikelcode findest.
  • Für Nerf Blaster und Monopoly: Der Code befindet sich in der Verpackung auf der Codekarteneinlage.

  • Für alle anderen virtuellen Artikelcodes (z. B. Prime Gaming, Verizon usw.) gehe zur unten stehenden Anleitung zum Einlösen.

Wie kann ich meinen Kode für den virtuellen Gegenstand einlösen?

Hinweis: Wenn du einen digitalen Code hast, fahre mit dem zweiten Schritt fort.

  1. Wenn dein Kode abgedeckt ist, zerkratze vorsichtig die Abdeckung.
  2. Stelle sicher, dass du in dem Roblox-Konto angemeldet bist, wo du den Kode einlösen möchtest.
  3. Gehe zur folgenden Seite , um deine Geschenkkarte einzulösen.
  4. Füge den Code ins Kästchen ein.
  5. Klicke auf die Schaltfläche Einlösen .
  6. Suche nach deinem neuen tollen Gegenstand im Inventar deines Kontos. Stelle sicher, dass du die richtige Inventar-Kategorie überprüfst (z.B. Hut, Rucksack, Kopf, Accessoires etc.)

Hinweis: der Code von einem virtuellen Gegenstand kann auf einem Konto nur ein einziges Mal eingelöst werden. Wenn du über einen Kode für ein virtuelles Element verfügst das dem Konto bereits gehört, kannst du den Kode mit einem anderen Benutzer teilen. Diese Gegenstände können nicht getradet oder verkauft werden.
