Wie man einen Spielzeug Kode einlöst

Wo finde ich meinen Kode für den virtuellen Gegenstand?

  • Für Jazwares-Actionfiguren: Der Code befindet sich auf der Vorderseite der Verpackung auf einem Token oder auf einer Codekarte, die auf der Innenseite der Kunststoffverpackung aufgeklebt ist. Sehen Sie sich das Jazwares video an, um zu erfahren, wo Sie den virtuellen Artikelcode finden.
  • Für Nerf Blaster und Monopoly: Der Code befindet sich in der Verpackung auf der Codekarteneinlage.

  • Für alle anderen virtuellen Artikelcodes (z. B. Prime Gaming, Verizon usw.) gehen Sie zu den unten stehenden Einlöseanweisungen .

Wie kann ich meinen Kode für das virtuelle Gegenstand einlösen?

Hinweis: Wenn Sie einen digitalen Code haben, fahren Sie mit Schritt 2 fort.
  1. Wenn Ihr Kode abgedeckt ist, zerkratzen Sie vorsichtig die Abdeckung.
  2. Stellen Sie sicher, dass Sie bei Ihrem Roblox-Konto angemeldet sind, für das Sie den Kode einlösen möchten.
  3. Gehen Sie zur Seite Spielzeug Code-Einlösung
  4. Geben Sie Ihren Kode in das Feld Spielzeug-Code eingeben ein.
  5. Klicken Sie auf die Schaltfläche Einlösen .

  6. Sehen Sie Ihren tollen neuen Gegenstand im Inventar Ihres Kontos an. Stellen Sie sicher, dass Sie die richtige Inventar Kategorie überprüfen (zB Hut, Rucksack, Kopf, Accessoires etc.) .

Hinweis: Ein Konto kann nur jedes virtuelle Spielzeugelement einmal empfangen. Wenn Sie über einen Kode für ein virtuelles Element verfügen, das Ihrem Konto bereits gehört, können Sie den Kode für einen anderen Benutzer freigeben.
