Gesponserte Spiele

Sobald Sie das Spiel Ihrer Träume erstellt haben, ist es Zeit das Wort zu verbreiten, damit andere Spieler die Früchte Ihrer Arbeit genießen können! Während die Mundpropaganda selbst ein leistungsfähiges Werkzeug ist, bietet Roblox mehrere Hilfsmittel an, die Ihnen den Einstieg erleichtern. Einer dieser Hilfsmittel ist die Funktion "Gesponserte Spiele".

Was sind gesponserte Spiele?

Gesponserte Spiele, auch bekannt als native Anzeigen, sind Werbeaktionen für Spiele die direkt in den vielen verschiedenen Spielarten/Listen zu sehen sind, wenn sie an anderen Orten erscheinen. Sie werden eindeutig als gesponsert gekennzeichnet, sehen aber anders aus und verhalten sich wie jede andere verfügbare Auswahl.

Wie funktioniert es?

Die Funktion "Gesponserte Spiele" verwendet ein Gebotssystem, das fast so ist wie das in den Nutzeranzeigen . Entwickler melden jeden Tag bei Robux, dass sie ihre Werbeaktion ausführen möchten. Sie geben auch die spezifische Plattform (Web, Tablet oder Telefon) an, auf der sie wollen, dass es angezeigt wird.

Wie oft ein Spiel in einem gesponserten Slot platziert wird, hängt davon ab wie viel der Entwickler anbietet und wie viel andere Spieler für die gleiche Plattform anbieten. Nehmen wir an, drei Entwickler bieten alle an, dass ihre Spiele auf Tablets präsentiert werden. Die Gebote lauten wie folgt:

  • Entwickler A bietet 50 Robux an
  • Entwickler B bietet 100 Robux an
  • Entwickler C bietet 300 Robux an

In diesem Beispiel bietet Entwickler A die kleinste Menge von Robux an, so dass das Spiel in den    gesponserten Slots möglichst selten erscheint. Entwickler B bieten doppelt so viele Robux an, damit ihr Spiel doppelt so häufig in diesen Slots erscheint. Entwickler C hat sechs Mal so viel Robux wie Entwickler A geboten, also wird ihr Spiel sechsmal so oft erscheinen.

Wie kann ich teilnehmen?

Wenn Sie im gesponserten Spielsystem teilnehmen möchten, können Sie dies folgendermaßen tun:

  1. Melden Sie sich in Ihrem Konto auf roblox.com an
  2. Klicken Sie auf Erstellen in der blauen Leiste am oberen Bildschirmrand
  3. Klicken Sie auf Spiele in der linken Spalte
  4. Suchen Sie das Spiel, für das Sie werben möchten und klicken Sie auf das Zahnradsymbol rechts daneben
  5. Wählen Sie Sponsor in dem Menü das erscheint
  6. Ein Bildschirm wird angezeigt, der wie folgt aussieht:
  7. Prüfen Sie das Kontrollkästchen für jede Plattform, auf die Sie bieten möchten
  8. Klicken Sie auf die Schaltfläche Erstellen

Daraufhin wird auf der Gesponserte Spiele Register im Bereich "Erstellen" für jede ausgewählte Plattform ein Eintrag erstellt. Führen Sie die folgenden Schritte aus, um die Anzeige zu aktivieren:

  1. Klicken Sie auf das Zahnradsymbol rechts daneben und wählen Sie Ausführen im angezeigten Menü
  2. Geben Sie ein, wie viele Robux Sie bieten möchten, und klicken Sie auf die Schaltfläche Ausführen

Wie kann ich feststellen, wie gut meine Promotion ist?

Sobald Sie alles eingerichtet haben, können Sie später am Tag auf den Tab "Gesponserte Spiele" im Bereich "Erstellen" zurückkehren, um zu sehen wie viel es sich ausgezahlt hat! Unter dem Eintrag der Anzeige befinden sich verschiedene Zahlen:

  • Gesamtdruck : Anzahl der Nutzer, die Ihre Anzeige gesehen haben
  • Gesamtzahl der Klicks : Anzahl der Nutzer, die auf Ihre Anzeige geklickt haben
  • Gesamte Klickrate : der Nutzer, die Ihre Anzeige gesehen haben. Dies ist der Prozentsatz von denen, die darauf geklickt haben
  • Gesamtgebot : die Menge an Robux, die Sie für die Aktion geboten haben
