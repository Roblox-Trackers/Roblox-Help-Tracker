Wie können Sie Vermögenswerte/Assets archivieren

Durch das Archivieren eines Asset soll verhindert werden, dass es in der Suche angezeigt wird, und dass andere es in ihren Spielen verwenden.

Sie können nur A udio, Mesh und Decals archivieren.  Die archivierte Elemente werden nicht in Ihrem Inventar angezeigt. Um Sie die Elemente wiederherstellen zu können, müssen Sie um auf die Elemente über die Registerkarte Erstellen zugreifen.

Archiv über Inventar

  1. Wählen Sie das Menüsymbol oben links auf dem Bildschirm.
  2. Wählen Sie die Option Inventar .
  3. Wählen Sie den Artikel im Inventar aus, den Sie archivieren möchten.
  4. Wählen Sie das Dreipunktmenü in der oberen rechten Ecke und wählen Gestalten .
  5. Scrollen Sie auf der Seite nach unten, bis Sie den Abschnitt Archiv finden, und klicken Sie auf die Schaltfläche Archivieren :
  6. Klicken Sie auf die Schaltfläche Speichern , die unten auf der Seite liegt.
  7. Sie können ein archiviertes Element wiederherstellen, indem Sie an derselben Stelle auf die Schaltfläche Wiederherstellen klicken:

Archivieren Sie über TAB Erstellen

Können Sie auch Elemente/Vermögenswerte über TAB Erstellen archivieren. Mehrere Informationen dazu, als auch zum Archivieren von Spielen finden Sie in dieser Anleitung .
