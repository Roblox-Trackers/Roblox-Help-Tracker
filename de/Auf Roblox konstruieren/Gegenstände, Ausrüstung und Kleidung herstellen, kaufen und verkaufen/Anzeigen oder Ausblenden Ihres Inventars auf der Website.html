Anzeigen oder Ausblenden Ihres Inventars auf der Website

Führen Sie die folgenden Schritte aus, um das Inventar Ihres Kontos auf der Website anzuzeigen:

  1. Klicken Sie auf das Menüsymbol oben links auf dem Bildschirm.
  2. Klicken Sie auf die Option Inventar.
  3. Verwenden Sie das Menü, um eine Kleider Kategorie und ggf. eine Unterkategorie auszuwählen.
  4. Blättern Sie durch Ihr Inventar auf der Seite. Tippen Sie unten auf jeder Seite auf die Pfeiltasten nach rechts oder nach links, um durch verschiedene Seiten zu navigieren.
  5. Klicken Sie auf das Artikelbild, das Sie anzeigen möchten.
  6. Sobald Sie Ihre Auswahl getroffen haben, wird auf der Seite des Artikels der Artikel als "Artikel im Besitz" mit einem Häkchen angezeigt. Die Seite enthält auch Informationen wie Name, Beschreibung und ggf. Genre.

Verstecken Sie Ihr Inventar für andere Spieler

  1. Besuchen Sie Ihre Profilseite
  2. Gehen Sie zur Einstellungsseite, indem Sie auf das Zahnradsymbol in der oberen rechten Ecke Ihrer Profilseite klicken
  3. Klicken Sie in den Einstellungen auf die Registerkarte Datenschutz
  4. Auf der Registerkarte Datenschutz klicken Sie auf Wer kann mein Inventar sehen?
  5. Wählen Sie aus, für welche Benutzergruppe Ihr Inventar angezeigt werden soll. Wenn Sie nichts auswählen, wird sichergestellt, dass Ihr Inventar für keinen Benutzer sichtbar ist.

Hinweis: Das Ausblenden Ihres Inventars hat keinen Einfluss auf die Nutzung des Handelssystems .
