Konfigurierung des Chat-Stils Ihres Ortes

Ihr Platz kann so konfiguriert werden, wie Sie ihre Chat-Nachrichten im Spiel angezeigt werden möchten. Dies ermöglicht flexible Optionen, mit denen die Spieler während des Spiels am besten kommunizieren können.

Chat-Modi

Es stehen Ihnen zwei Standard-Chat-Stile zur Verfügung.

Klassisch

Dies wird die Chat-Nachrichten im oberen linken Chat-Fenster des Standard-Spielbildschirms angezeigt. Diese Meldungen werden in chronologischer Reihenfolge aufgelistet. Die ältesten Meldungen sind von unten nach oben aufgelistet. Auf diese Weise können sich Spieler global auf einem Server unterhalten, ohne Einschränkungen hinsichtlich der physischen Position in der Spielwelt.

Blase

Dadurch werden Chat-Nachrichten über den Charakterköpfen der sprechenden Spieler angezeigt. Auf diese Weise können Gespräche auf die physischen Standorte der Spieler an der Stelle Ihres Spiels beschränkt werden.

Beide Chat-Modi können auf Wunsch gleichzeitig aktiviert werden.

Chat-Modi einstellen

Standardmäßig ist nur der klassische Chat-Modus aktiviert. Lesen Sie unseren Entwicklerartikel über das Roblox Chat System durch, um zu erfahren welche Modi aktiviert sind.
