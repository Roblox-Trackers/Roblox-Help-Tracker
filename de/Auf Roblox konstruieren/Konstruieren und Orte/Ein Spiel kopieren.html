Ein Spiel kopieren

Manchmal werden Sie vielleicht feststellen, dass Sie ein neues Spiel starten möchten, aber ein bereits existierendes Spiel bearbeiten wollen, um Ihnen etwas Zeit zu sparen. Sie können sich auch nur wünschen eines Ihrer Spiele auf ein anderes Konto kopieren. Die folgenden Schritte helfen Ihnen, diese Dinge zu tun.

Ein eigenes Spiel kopieren

  1. Führen Sie Roblox Studio aus
  2. Melden Sie sich in Ihrem Konto im Studio an
  3. Klicken Sie auf Meine Spiele
  4. Suchen Sie das Spiel das Sie kopieren möchten, klicken Sie darauf und klicken Sie dann auf Bearbeiten am unteren Bildschirmrand
  5. Sobald Sie das Spiel bearbeiten, können Sie auf zwei Wegen kopieren:
  • Wenn Sie nur ein Backup auf Ihrem Computer speichern möchten, wählen Sie Datei > Speichern unter
  • Wenn Sie eine Kopie des Spiels auf Roblox hochladen möchten, wählen Sie Datei > In Roblox veröffentlichen und laden Sie über ein bestehendes Spiel hoch oder erstellen Sie ein neues

Hinweis: Um das Spiel auf eines Ihrer anderen Konten zu kopieren, wählen Sie die Methode Speichern unter aus Schritt 5. Sobald das Spiel gespeichert wurde, melden Sie sich beim Studio ab und melden Sie sich dann als Ihr neues Konto erneut an. Sie können dann in Roblox veröffentlichen verwenden, um es in diesem Konto hochzuladen.

Wie verwende ich das Kopieren erlaubtes Spiel von jemandem anderen als Vorlage

Sie können ein Spiel sehen, das Ihrer Meinung nach als guter Ausgangspunkt für Ihr eigenes Spiel dienen könnte. Die gute Nachricht ist, dass einige Entwickler Sie ermutigen, ihre Spiele für diesen Zweck zu verwenden! Ein Spiel kann jedoch nur kopiert werden, wenn der Hinweis "Die Quelle dieses Spiels kann kopiert werden" im Info-Tab des Spiels aufgeführt ist. Für weitere Details, bitte lesen Sie folgenden Hilfeartikel, um das Kopieren Ihres Spiels zu erlauben oder zu untersagen .

Wenn das Spiel kopiert werden darf, helfen Ihnen die folgenden Schritte dies als Ausgangspunkt für Ihr eigenes Spiel zu verwenden:

  1. Gehen Sie zur Detailseite des Spiels
  2. Klicken Sie auf das Symbol, das rechts oben im Titel wie drei Punkte aussieht
  3. Klicken Sie im angezeigten Menü auf Bearbeiten
  4. Dies öffnet das Spiel im Studio, wo Sie eines von zwei Dingen tun können:
  • Wählen Sie Datei > Speichern unter und speichern Sie das Spiel auf Ihrem Computer. Sie können das Spiel dann nach Ihren Wünschen bearbeiten.
  • Wählen Sie Datei > In Roblox veröffentlichen und laden Sie dann entweder ein bestehendes Spiel hoch oder erstellen Sie ein neues Spiel.

Hinweis: Bevor Sie sich entscheiden ein kopiertes Spiel zu veröffentlichen, müssen Sie zunächst die Erlaubnis des Erstellers erhalten. Wenn das Spiel des Spielers nicht kopiert werden darf und Sie Ihre eigene Version erstellen möchten, fragen Sie zuerst den Entwickler um Erlaubnis. Es ist höflich.
