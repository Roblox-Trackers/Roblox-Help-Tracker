So können Sie einen Platz Veröffentlichen/Hochladen

Veröffentlichen ist, wenn Sie eine Kopie Ihres Platzes auf den Roblox-Servern speichern. Dies bedeutet, dass es Ihren Platz speichert und dann auf die Roblox-Server hochlädt, wo es gespeichert bleibt, damit andere sich an Ihren Platz leichter erfreuen können.

Ihren Platz Veröffentlichen:

  1. Erstellen Sie Ihren Platz
  2. Klicken Sie im Studio (Bearbeitungsmodus) im Menü Datei auf Auf Roblox veröffentlichen als
  3. Ein neues Fenster wird geöffnet, in dem Sie aufgefordert werden, einen vorhandenen Platz zum Überschreiben auszuwählen oder einen neuen Platz zu erstellen
  4. Wählen Sie Ihre Option und Ihr Platz wird hochgeladen

Hinweis: Achten Sie beim Erstellen Ihres Platzes darauf, eine zusätzliche Kopie auf Ihrem Computer zu speichern, damit Sie eine Sicherheitskopie haben, falls etwas passieren sollte

Wenn Sie einfach einen Platz aktualisieren, der zuvor in Roblox hochgeladen wurde, können Sie zu Datei wechseln und dann auf Auf Roblox Veröffentlichen klicken.
