Bearbeiten und Testen in Studio (Bearbeitungsmodus)

Der Bearbeitungsmodus ist eine Funktion in Roblox Studio, mit der Sie alle oder die meisten Teile eines Platzes bearbeiten können, ohne dass sich etwas bewegt ( so als würde die Zeit stillstehen ). Es bietet eine große Menge an Werkzeugen und Optionen an, die Ihnen helfen, Ihren Platz zu erstellen und anzupassen. Bevor Sie den Bearbeitungsmodus verlassen, wird empfohlen, dass Sie das, was Sie gerade bearbeitet haben, auf Ihrem Computer speichern.

Einstieg in den Bearbeitungsmodus

Der Bearbeitungsmodus kann auf eine der folgenden Arten erreicht werden:

  • Indem Sie die Detailseite eines Spiels aufrufen, klicken Sie auf das Symbol das wie drei Punkte aussieht, über der Wiedergabe Schaltfläche, und wählen Sie Bearbeiten aus dem angezeigten Menü. Bitte beachten Sie, dass dies eines Ihrer eigenen Spiele sein muss oder, wenn es jemand anderem gehört, muss es nicht Copy Locked sein
  • Gehen Sie in den Bereich Erstellen , suchen Sie den Platz den Sie bearbeiten möchten, und klicken Sie auf die entsprechende Schaltfläche Bearbeiten
  • Öffnen Sie einen Platz, der auf Ihrem Computer gespeichert ist, indem Sie im Studio Menü Datei Öffnen auswählen
  • Indem Sie in Studio Datei> Neu auswählen. Dies öffnet einen leeren Platz, der in fast alles was Sie möchten, verwandelt werden kann

Die Registerkarten

Studio bietet eine Vielzahl von verschiedenen Optionen in schnell anklickbaren Schaltflächen in verschiedenen Registerkarten an. Diese Registerkarten sind:

  • Startseite : Diese enthält eine Vielzahl der am häufigsten verwendeten Tasten für Kurzanleitung
  • Model : Hier finden Sie Schaltflächen, die die meisten Optionen zum Einfügen und Ändern von Objekten darstellen
  • Gelände : Mit diesen Werkzeugen können Sie das Gelände modifizieren
  • Test : Alle Dinge, die Sie benötigen, um zu testen, wie Ihre Änderungen innerhalb des Spiels funktionieren, finden Sie hier. Darüber hinaus können Sie im Bereich Emulation auf dieser Registerkarte sehen, wie Ihr Spiel auf einem von verschiedenen Mobilgeräten oder Fernsehgeräten aussieht.
  • Ansicht : Ermöglicht das Ein- oder Ausblenden der verschiedenen Bereiche/Fenster in Studio
  • Plugins : Hier finden Sie alle Plugins, die Sie heruntergeladen oder Ihrem Konto hinzugefügt haben. Weitere Informationen zu dieser Funktion finden Sie hier

Die Abschnitte Explorer und Eigenschaften

Sobald Sie sich im Bearbeitungsmodus befinden ( entweder an einem neuen Platz oder an einem bereits bestehenden ), sehen Sie auf der rechten Seite des Bildschirms zwei Abschnitte die Explorer und Eigenschaften benannt sind. Wenn Sie eines oder beide nicht sehen, können Sie sie erneut öffnen, indem Sie auf die entsprechenden Symbole auf der Registerkarte Ansicht klicken.

  • Explorer
    Dieser Abschnitt listet alles auf, was sich in Ihrem Platz befindet. Es wird in verschiedene Gruppen wie Arbeitsbereich, Beleuchtung, StarterPack, Spieler ... usw. aufgeteilt. Um zu sehen, was sich in einer Dienstgruppe befindet, klicken Sie einfach auf den kleinen grauen Pfeil links davon. Alle Objekte, die Sie in Ihren Platz und in den meisten Skripts sehen, befinden sich in der Workspace-Gruppe. Wenn ein Objekt andere Objekte ( wie zum Beispiel Abziehbilder ) enthält, wird es auch einen kleinen Pfeil haben. Um eine Gruppe zu schließen und ihren Inhalt zu verbergen, klicken Sie einfach erneut auf den Pfeil.
  • Eigenschaften
    Dieser Abschnitt zeigt Ihnen eine Liste der Eigenschaften, die ein Objekt hat, wie Größe, Rotation, wenn es verschoben ( verankert ) werden kann usw. Das Objekt muss ausgewählt sein, um die Eigenschaften anzuzeigen.

Die Kamera bedienen

Die Kamera ist einfach zu bedienen . Anders als wenn Sie Ihr Spiel online spielen, ist die Kamera nicht nur auf ein Objekt beschränkt ( z.B. Ihr Kopf) . Stattdessen können Sie es überall und in jedem Winkel bewegen. Verwenden Sie entweder W, A, S, D oder die Pfeiltasten, um die Kamera vorwärts, rückwärts, links und rechts zu bewegen. Klicken und halten Sie die rechte Maustaste und bewegen Sie die Maus, um Ihre Ansicht zu drehen. Wenn Sie das Mausrad mit dem Mausrad nach oben rollen, wird es vergrößert, während es beim Herunterrollen verkleinert wird.

Objekte hinzufügen und verschieben

Das Hinzufügen von Bausteinen, Skripten und Modellen ( sowie vieler anderer Objekte ) im Bearbeitungsmodus ist einfach und es gibt mehrere Möglichkeiten, dies zu erreichen. Die erste Option besteht darin, einfach mit der rechten Maustaste auf ein leeres Feld im Spielfenster zu klicken, Objekt einfügen auszuwählen und das Objekt selektieren, das Sie zu Ihrem Platz hinzufügen möchten.

Alternativ können Sie die vielen Schaltflächen auf der Registerkarte Modell verwenden, um schnell eine Vielzahl von Dingen wie Teile, Bausteine, Effekte, Spawnpunkte, Skripts usw. einzufügen . Um die Platzierung zu vereinfachen, können Sie mithilfe der Werkzeugkasten auf der linken Seite des Bildschirms viele verschiedene Bausteine, Modelle oder Skripte in das Spielfenster ziehen. Wenn dies nicht sichtbar ist, können Sie es auf der Registerkarte Ansicht erneut öffnen.

Das Verschieben von Dingen, die Sie bereits zu Ihrem Spiel hinzugefügt haben, ist noch einfacher! Klicken Sie zuerst auf das Verschieben-Symbol in der Registerkarte Modell oben auf dem Bildschirm. Sie können dann ein Objekt frei bewegen, indem Sie es anklicken und durch das Spielfenster ziehen. Wenn Sie es nur nach oben/unten, nach links/rechts oder nach innen/außen bewegen möchten, klicken Sie einfach auf das Objekt, um eine Reihe von Pfeilen anzuzeigen. Klicken Sie auf die Richtung, in die Sie sich bewegen möchten, und ziehen Sie die Maus.

Hinweis: Einige Objekte, wie Glitzern und Skripts, sind nicht sichtbar und können daher nicht verschoben werden.

Der Werkzeugkasten

Mit der Werkzeugkasten können Entwickler Modelle, Aufkleber und mehr, die von anderen Spielern erstellt wurden, per Ziehen & Ablegen an ihren Platz ziehen. Der Werkzeugkasten ermöglicht Entwickler Modelle, Aufkleber und mehr, die von anderen Spielern erstellt wurden, durch Ziehen & Ablegen an ihren Platz ziehen. Dies ähnelt der Option Objekt Einfügen innerhalb des Rechtsklick-Menüs, bietet jedoch vorgefertigtes Material und von Spielern erstellte Inhalte an.

Größe ändern und Drehen von Objekte

Um die Größe der Bausteine ​​zu ändern, klicken Sie auf die Schaltfläche Skalieren auf der Registerkarte Modell, wählen Sie einen Baustein aus, und kleine Punkte sollten oben, unten, links, rechts, vorne und hinten am Objekt erscheinen. Klicken und ziehen Sie den Punkt, der der Seite entspricht, die Sie verkleinern oder dehnen möchten.

Hinweis: Die Größe der Modelle kann nicht geändert werden.

Um ein Objekt zu drehen, klicken Sie einfach darauf und anschließend auf die Schaltfläche Drehen auf der Registerkarte Modell. Dadurch wird das ausgewählte Objekt mit sechs Punkten und drei Kreisen umgeben. Klicken Sie auf einen der Punkte und ziehen Sie die Maus, um Ihre Auswahl innerhalb des Kreises zu drehen oder zu neigen, der der Farbe des Punktes entspricht.

Gelände hinzufügen und modifizieren

Mit Roblox Studio können Sie nun auch das Gelände Ihrer Welt einfach hinzufügen oder manipulieren, um wunderschöne Landschaften zu schaffen. Sie finden die entsprechenden Werkzeuge auf der Registerkarte Gelände. Für Details zu den einzelnen Werkzeugen bitte hier klicken.

Testen Sie Ihren Platz

Wenn Sie mit Ihrem Platz fertig sind oder vor dem Veröffentlichen sehen möchten, was etwas tut, klicken Sie auf die Schaltfläche Wiedergabe auf der Registerkarte Test . Dadurch wird Ihr Avatar in das Spielfenster geladen und es ermöglicht Ihnen herumzulaufen und zu spielen. Um den Test zu beenden und das Spiel wie vor dem Klicken wiederherzustellen, klicken Sie auf die Schaltfläche Stopp.

Hinweis: Wenn Sie Ihren Platz testen möchten, indem Sie ihn über die normale Spielfunktion auf der Roblox-Seite spielen, können Sie dies tun, ohne dass andere Spieler Ihrem Spiel beitreten, indem Sie Ihr Spiel auf Privat setzen. Weitere Informationen finden Sie im Hilfeartikel Spiele Erstellen und Veröffentlichen.

Testen wie Ihr Spiel mit anderen Spielern funktioniert

Wenn Sie sehen möchten, wie Ihr Platz mit mehreren Teilnehmern aussehen würde, können Sie einen Server emulieren. Gehen Sie dazu auf die Registerkarte Test und wählen Sie in der Dropdown-Liste im Bereich Kunden und Server aus, wie viele Spieler Sie emulieren möchten. Klicken Sie danach auf Start. Dadurch werden mehrere andere Studio-Fenster geöffnet. Eines dieser Fenster dient als Server, während die anderen einen der verschiedenen Spieler Imitationen steuern, die dem Spiel hinzugefügt werden. Um den Test zu beenden, klicken Sie im ursprünglichen Studio-Fenster auf die Schaltfläche "Säubern".

Hinweis: Wenn Sie Ihren Platz testen möchten, indem Sie ihn mit anderen von Ihnen ausgewählten Entwicklern über die normale Spielfunktion auf der Roblox-Seite spielen, können Sie dies tun, ohne dass unberechtigte Spieler Ihrem Spiel beitreten, indem Sie es als Gruppenspiel hochladen, den Zugriff für editieren auf vorgegebene Ränge beschränken und Ihr Spiel auf privat setzen. Weitere Informationen finden Sie in den Hilfeartikeln Spiele Erstellen und Veröffentlichen und Gruppenspiele .
