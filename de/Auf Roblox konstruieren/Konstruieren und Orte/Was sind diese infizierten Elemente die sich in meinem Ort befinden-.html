Was sind diese infizierten Elemente die sich in meinem Ort befinden?

Leider erstellen Spieler manchmal Modelle oder Plugins, die nicht wirklich das tun, was diese versprochen haben, oder können sogar seltsames und unerwünschtes Verhalten verursachen. Diese werden als infiziert bezeichnet und sind nur normale Modelle oder Plugins mit fehlerhaften Skripts. Sie werden deswegen gemacht, um Probleme für Spieleentwickler zu verursachen.

Was passiert, wenn mein Platz bereits infiziert ist?

Während die Entfernung einer Infektion manchmal schwierig erscheint, gibt es ein paar Dinge die es zu versuchen wert sind.

Zurück zu einer älteren Version

Wenn Ihr Standort bereits in Ihr Konto hochgeladen wurde, können Sie auf der Konfigurationsseite zum Bereich Versionsverlauf wechseln und den Standort auf eine frühere Version zurücksetzen. Achten Sie darauf, ein Datum auszuwählen, bei dem die Infektion definitiv nicht aufgetreten ist. Das bedeutet, dass Sie alles, das nach diesem Zeitpunkt hinzugefügt wurde, neu erstellt/neu angeordnet werden muss, jedoch kann man möglicherweise verhindern, dass Sie alles neu erstellen müssen. Seien Sie vorsichtig, wenn Sie erneut kostenlose Modelle hinzufügen, um zu verhindern, dass die Infektion zurückkehrt. Wenn Sie Hilfe beim Zurücksetzen benötigen, klicken Sie bitte hier .

Fangen Sie mit Ihrem Platz von vorne an

Manchmal kann die Infektion so tief gehen, dass die einzige Möglichkeit, sie zu reparieren, darin besteht, von vorne zu beginnen. Hilfe zum Zurücksetzen auf eine Basisvorlage finden Sie hier .

Wie verhindere ich Infektionen von vornherein?


Um zu verhindern, dass Ihre Plätze in Zukunft infiziert werden, gibt es mehrere Möglichkeiten, die Sie tun können:

    • Speichern Sie immer Ihren Fortschritt, bevor Sie kostenlose Modelle einfügen.
    • Untersuchen Sie immer Modelle, die Sie aus dem öffentlichen Inventar entfernen. Stellen Sie sicher, dass es keine fehlerhaften Skripts oder Skripts mit fragwürdigen Namen enthält. Wenn das Modell Kommentare erlaubt hat, lesen Sie sie, um zu sehen, was andere Spieler gesagt haben.
    • Nutzen Sie Roblox Studio optimal zum Erstellen von Spielen, da fehlerhafte Skripts in Modellen standardmäßig nicht in der Studio-Umgebung ausgeführt werden können. Sie können dann Modelle überprüfen und verhindern, dass schlimme Dinge auftreten.
    • Für Plugins - versuchen Sie, das Plugin an einem Testplatz auszuführen, bevor Sie es in Ihren Spielbereich einfügen.


Hinweis: Es gibt keine Möglichkeit, dass das Skript in Ihren Kontosicherheits- oder Rechnungsdaten aufgenommen wird, also machen Sie sich keine Sorge. Sie können diese Modelle über unser Missbrauchs Meldesystem melden, indem Sie auf den Link Missbrauch melden auf der Beschreibungsseite des Modells/Skripts /Plugins klicken, damit unsere Moderatoren sie so schnell wie möglich entfernen können.
