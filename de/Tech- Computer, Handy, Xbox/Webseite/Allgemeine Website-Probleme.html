Allgemeine Website-Probleme

Der folgende Artikel wird einige Methoden vorschlagen, die sich bei der Behebung der folgenden Probleme als effektiv erwiesen haben :

  • Die Website wird nicht richtig angezeigt
  • Einige Seiten werden überhaupt nicht geladen oder zeigen beim Laden einen Fehler an
  • Die Schaltflächen oder Links öffnen sich nicht wenn man darauf klickt
  • Sie können sich auf der Roblox Website nicht anmelden, obwohl Sie die richtige Benutzernamen und das Passwort eingegeben haben
  • Sonstiges: Wenn Sie ein anderes Problem mit der Website haben, können die folgenden Schritte ebenfalls zur Lösung beitragen. Zusätzlich können einige nicht bezogene Webseiten Artikel hier für einige Schritte verlinken.

Was zu tun

Bitte versuchen Sie jeden Schritt. Wenn eine Lösung nicht funktioniert, fahren Sie mit dem nächsten Schritt fort!

Löschen Sie Ihre temporären Internetdateien und Cookies


Die folgenden Links helfen Ihnen, die temporären Internetdateien (auch als "Cache" bezeichnet) und Cookies für die wichtigsten PC-Webbrowser zu entfernen. Dies gilt für jeden Browser, die Sie benutzen werden.

      • Internet Explorer - Temporary Internet Files (From Microsoft.com)
      • Internet Explorer - Cookies (From Microsoft.com)
      • Mozilla Firefox - Temporary Internet Files (From Mozilla.com)
      • Mozilla Firefox - Cookies (From Mozilla.com)
      • Google Chrome - Both (From Google.com)
      • Microsoft Edge - Both (From Microsoft.com)
      • Safari - Cookies (From Apple.com)
      • Andere Browser : Wenn Sie einen anderen Browser oder eine ältere Version eines Browsers verwenden werden, suchen Sie nach Informationen zum Löschen der Cache-/Browserdaten/temporären Internetdateien für Ihren spezifischen Browser/Version.

Sobald Sie die oben genannten Schritte für alle relevanten Webbrowser abgeschlossen haben, starten Sie Ihren Computer neu und prüfen Sie, ob das Problem behoben wurde.

Wenn Sie Ihre Internet Explorer/Windows temporären Internetdataien manuell löschen möchten, können Sie dies tun, indem Sie direkt zum Ordner Temporäre Internetdateien auf Ihrer Festplatte gehen. Der Name des Verzeichnisses und des Ordners hängt von der Windows-Version ab, egal ob Sie die Benutzerprofile verwendet haben oder nicht.

Hinweis: Möglicherweise müssen Sie die Einstellungen von Windows Explorer ändern, um alle Arten von Dateien anzuzeigen (einschließlich der geschützten Systemdateien), um auf diese Ordner zuzugreifen.



Für Windows 10

Wenn Sie Windows 10 haben, befinden sich die Temporäre Internetdateien am folgenden Speicherort (beachten Sie, dass sie sich auf Ihrem PC auf einem anderen Laufwerk statt auf Laufwerk C befinden können):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\INetCache

Für Windows 7 und 8

Wenn Sie Windows 7 oder Windows Vista haben, befinden sich die temporäre Internetdateien an diesen Speicherorten (beachten Sie, dass sie sich auf einem anderen Laufwerk statt auf Laufwerk C befinden können):

        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\
        • C:\Users\<username>\AppData\Local\Microsoft\Windows\Temporary Internet Files\Low\


Deaktivieren Sie die Add-Ons Ihres Webbrowsers


Viele Probleme mit der Website sind das Ergebnis von Add-Ons von Drittanbietern für Ihren Webbrowser. Diese können je nach Browser Ihrer Wahl auch als Erweiterungen oder Plugins bezeichnet werden. Um festzustellen, ob dies für Sie der Fall ist, deaktivieren Sie alle Ihre Add-Ons. Wenn das Problem behoben ist, versuchen Sie, sie nacheinander zu aktivieren, bis das Problem erneut auftritt. Sobald das Problem wieder auftritt, sollten Sie den, der mit Roblox inkompatibel ist, gefunden haben.

Die folgenden Links können Ihnen bei den beliebtesten PC-Webbrowsern helfen:

  • Internet Explorer ( from Microsoft.com )
  • Firefox ( from Mozilla.com )
  • Chrome ( from Google.com )
  • Andere Browser : Wenn Sie einen anderen Browser oder eine ältere Version eines Browsers verwenden, suchen Sie nach Informationen zum Deaktivieren der Erweiterungen/ Plugins/Add-Ons für Ihren spezifischen Browser/Version.


Zurücksetzen der Internetoptionen für Windows

Wir haben festgestellt, dass viele Probleme von Roblox Website- und Roblox Studio durch Zurücksetzen der Internetoptionen Ihres Computers gelöst werden können. Bitte, beachten Sie, dass diese Schritte den Internet Explorer verwenden müssen, auch wenn Sie normalerweise einen anderen Webbrowser verwenden. Nachdem diese Schritte unternommen wurden, können Sie wieder den jeweils bevorzugten Browser verwenden. Bitte versuchen Sie  folgendes:

  1. Öffnen Sie den Internet Explorer (speziell diesen Webbrowser)
  2. Öffnen Sie das Zahnradsymbol  > Internetoptionen
  3. Öffnen Sie die Registerkarte Erweitert
  4. Klicken Sie die Zurücksetzen Taste

Schließen Sie den Internet Explorer nach dem Zurücksetzen, damit die Änderungen wirksam werden. Nachdem dies abgeschlossen ist, sollten Sie wieder in der Lage sein, Ihren bevorzugten Webbrowser zu verwenden.

Aktualisieren Sie den Internet Explorer

Bitte versuchen Sie, Ihren Internet Explorer auf die neueste Version zu aktualisieren. Sie können das finden, indem Sie hier klicken. Auch wenn Sie den Internet Explorer derzeit nicht als Browser verwenden, müssen Sie dennoch sicherstellen, dass Sie die aktuellste Version auf Ihrem Windows-Computer haben, damit Roblox ordnungsgemäß funktioniert.
