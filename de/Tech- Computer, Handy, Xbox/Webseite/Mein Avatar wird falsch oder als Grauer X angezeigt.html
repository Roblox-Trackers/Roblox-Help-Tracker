Mein Avatar wird falsch oder als Grauer X angezeigt

Gelegentlich kann es passieren dass Sie Ihre neue Kleidung oder Ausrüstung anziehen möchten, und feststellen müssen, dass Ihr Avatar nicht erfolgreich aktualisiert wurde oder jetzt als graues X erscheint. Wenn das der Fall ist, gibt es ein paar Dinge, die Sie ausprobieren können.

Ihren Avatar Neu rendern

Es ist möglich, dass die Server beim Erstellen des Miniaturbildes für Ihren Avatar auf ein kleines Hindernis stoßen. Wenn das der Fall ist, müssen Sie sie nur regenerieren! Um dies zu tun, navigieren Sie einfach zu Ihrem Avatar Bereich und klicken Sie auf den "Klicken Sie hier, um Neu zu rendern!" Link unter dem Bild Ihres Avatars. Es kann ein paar Minuten dauern, bis das Problem behoben ist, aber das sollte das Problem die meiste Zeit beheben.

Löschen Sie die temporären Internetdateien Ihres Browsers

Wenn das Neu rendern nicht hilft, können Sie die temporären Dateien Ihres Browsers löschen. Für Unterstützung, bitte hier klicken und folgen Sie den Schritten in der Lösung "Löschen Sie Ihre temporären Internetdateien und Cookies".
