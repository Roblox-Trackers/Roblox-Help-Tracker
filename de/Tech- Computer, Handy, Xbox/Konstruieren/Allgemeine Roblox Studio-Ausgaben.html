Allgemeine Roblox Studio-Ausgaben

Der folgende Artikel wird einige Methoden vorschlagen, die sich bei der Behebung der folgenden Probleme als wirksam erwiesen haben:

  • Sie versuchen sich bei Roblox Studio anzumelden, können dies aber nicht tun
  • Sie versuchen einen Ort zu bearbeiten, nur um eine leere Skybox oder etwas namens Platz1 laden zu lassen
  • Teile von Studio sind leer, die nicht sein sollten
  • Sie haben Schwierigkeiten in Roblox zu veröffentlichen
  • Roblox Studio sagt das Sie sich anmelden müssen, obwohl Sie bereits angemeldet sind
  • Diese Schritte helfen möglicherweise bei einigen Problemen die nicht mit dem Studio zu tun haben, z. B. wenn Sie ein Spiel als Gast betreten, auch wenn Sie angemeldet sind
  • Sonstiges: Wenn beim Studio ein anderes Problem auftritt, können die folgenden Schritte ebenfalls zur Behebung des Problems beitragen

Was zu tun ist - Windows

Bitte versuchen Sie jeden Schritt einzeln der Reihe nach. Wenn eine Lösung nicht funktioniert, fahren Sie mit der nächsten fort!

Neustart

Starten Sie zuerst Ihren Computer neu und versuchen Sie zu spielen. Manchmal ist dies alles was benötigt wird, um Roblox- oder Roblox Studio-Probleme zu lösen.

Setzen Sie Ihre Internetoptionen in Windows zurück

Auch wenn Sie Internet Explorer nicht verwenden, kann das Problem durch das Zurücksetzen der Internetoptionen über diesen Browser behoben werden. Bitte beachten Sie, dass diese Schritte zwar mit dem Internet Explorer durchgeführt werden müssen, Sie aber nach Abschluss des Vorgangs zu Ihrem bevorzugten Browser zurückkehren können. Bitte versuchen Sie folgendes:

    1. Öffnen Sie den Internet Explorer (speziell für diesen Browser)
    2. Öffnen Sie das Zahnradsymbol > Internetoptionen
    3. Öffnen Sie die Registerkarte Erweitert
    4. Klicken Sie auf die Schaltfläche Zurücksetzen

Schließen Sie nach dem Zurücksetzen den Internet Explorer, damit die Änderungen wirksam werden.

Überprüfen Sie Ihre Proxy Einstellungen

Wenn die obigen Schritte Ihr Roblox Studio-Problem nicht gelöst haben, müssen Sie möglicherweise sicherstellen, dass Ihr Computer keine Verbindung über einen Proxy-Server herstellt. Bitte folgen Sie diesen Schritten:

Windows 7 oder früher

    1. Gehen Sie zum Startmenü
    2. Suchen Sie "Proxy" und "Proxy Server konfigurieren" wird angezeigt, klicken Sie darauf
    3. Klicken Sie auf LAN-Einstellungen und deaktivieren Sie alle Häkchen auf dieser Registerkarte
    4. OK klicken

Windows 8 oder 8.1

    1. Gehen Sie zum gekachelten Startbildschirm, indem Sie auf die untere linke Ecke des Bildschirms klicken
    2. Schreiben Sie "Proxy" und dann wird "Proxy Server konfigurieren" angezeigt, klicken Sie darauf
    3. Klicken Sie auf LAN-Einstellungen und deaktivieren Sie alle Häkchen auf dieser Registerkarte
    4. OK klicken

Windows 10

    1. Drücken Sie die Windows-Taste + S auf Ihrer Tastatur. Alternativ, wenn Cortanas "Frag mich was" -Bar in Ihrer Taskleiste verfügbar ist, können Sie einfach darauf klicken
    2. Schreiben Sie "Proxy" und dann wird "Proxy-Einstellungen ändern" angezeigt, klicken Sie darauf
    3. Stellen Sie sicher, dass "Use Setup Script" und "Proxy Server verwenden" deaktiviert sind
    4. Scrollen Sie nach unten und klicken Sie auf Speichern

Überprüfen Sie Ihre Windows-Systemuhr

Es mag seltsam klingen, aber verschiedene Probleme können tatsächlich dadurch verursacht werden, dass Ihre Windows-Systemuhr auf die falsche Zeit oder das falsche Datum eingestellt ist. Wenn dies die Ursache ist, sollte einfach alles korrigiert werden um die Situation zu beheben. Obwohl dies manuell durchgeführt werden kann, können Sie es auch so einstellen, dass es automatisch mit einem Server aktualisiert wird. So wird sichergestellt, dass es nicht zurückfällt und erneut Probleme verursacht. So geht's:

Windows 8 oder früher

    1. Klicken Sie mit der rechten Maustaste auf die Windows-Uhr in der unteren rechten Ecke
    2. Klicken Sie auf Datum / Uhrzeit anpassen
    3. Wählen Sie die Registerkarte Internetzeit
    4. Hier sollten Sie Zeit und Datum mit einem Internetserver synchronisieren können

Windows 10

    1. Klicken Sie mit der rechten Maustaste auf die Windows-Uhr in der unteren rechten Ecke
    2. Stellen Sie sicher, dass "Zeit automatisch einstellen" aktiviert ist und Sie die richtige Zeitzone ausgewählt haben

Was zu tun ist - Mac

Roblox Client wird auf 10.7 (Lion) und höher installiert - und nur auf Intel-Macs, während Roblox Studio auf Mac OS 10.10 (Yosemite) und höher installiert wird.

Um dies auf einem Mac zu beheben, installieren Sie es bitte neu. Klicken Sie hier um Hilfe zu erhalten.

Um zukünftige Probleme zu minimieren

Stellen Sie immer sicher das alle Programme auf Ihrem Computer geschlossen sind, bevor Sie Roblox installieren oder aktualisieren. Stellen Sie unter Windows sicher, dass Internet Explorer mit der neuesten Version auf dem neuesten Stand ist.
