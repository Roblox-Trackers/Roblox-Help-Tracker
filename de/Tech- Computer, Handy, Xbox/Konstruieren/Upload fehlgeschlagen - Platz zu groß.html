Upload fehlgeschlagen / Platz zu groß

Wenn Sie im Studio eine Fehlermeldung erhalten, dass Ihr Upload fehlgeschlagen ist, ist dies in der Regel darauf zurückzuführen, dass Ihr Platz zu groß ist oder zu viele Elemente / Skripts enthält. Um dies zu testen, versuchen Sie bitte Folgendes:

    1. Speichern Sie Ihren Bereich zuerst auf Ihrem Computer, indem Sie ihn in Roblox Studio öffnen und das Menü Datei > Speichern verwenden
    2. Halten Sie Roblox Studio geöffnet und erstellen Sie ein neues Spiel, indem Sie im Datei Menü den Befehl Neu auswählen
    3. Fügen Sie Ihrem neuen Bereich einen Baustein hinzu und wählen Sie Datei > In Roblox veröffentlichen als
    4. Wählen Sie den Ort, an dem Sie gerade Probleme haben, um Ihren neuen leeren Ort damit zu überschreiben.
    5. Funktioniert es nun?

Wenn dies funktioniert, müssen Sie möglicherweise Ihren Platz verkleinern oder einige Elemente reduzieren. Sie können dies tun, indem Sie die Version bearbeiten, die Sie auf Ihrem Computer gespeichert haben, und diese auf der gleichen Art und Weise wie oben beschrieben zu veröffentlichen. Alternativ können Sie Ihren Platz konfigurieren und ihn auf eine frühere Version zurücksetzen, in dem das Problem möglicherweise noch nicht aufgetreten ist.
