Allgemeine Verbindungsprobleme

Bitte lesen Sie sich alle folgenden Informationen durch und probieren Sie alle empfohlenen Schritte für Probleme mit der Verbindung zu Roblox >

Anzeichen für Verbindungsprobleme

  • Roblox hat Probleme mit dem Laden von Online-Spielen oder manchmal  wird eine Fehlermeldung angezeigt. Entweder " Could not connect " ( keine Verbindung )  oder "ID = 17 Failure" ( ID = 17 Fehler ).
  • Der Ladebildschirm auf der Webseite dauert sehr lange oder wird nie geladen.
  • Der Spieler kann die Spiele im Build- oder Edit-Modus besuchen, aber nicht im Play-Modus.
  • Die Spiele können Ihnen zufällig  oder ständig sagen , dass sie heruntergefahren sind.
  • HttpSendRequest fehlgeschlagen: Der HRESULT 0x80072efd-Fehler ist normalerweise darauf zurückzuführen, dass Roblox Schwierigkeiten hat, bestimmte Informationen mit unseren Servern zu kommunizieren.

Warum es passieren kann

  • Firewall “ : Dies ist sehr häufig ein Firewall-Problem, obwohl es (in sehr wenigen Fällen) das Ergebnis einer Verbindung mit niedriger Bandbreite oder einer inkonsistenten drahtlosen Verbindung sein kann.
  • Slow Internet Connection/Big Game “ ( Langsame Internetverbindung / Big Game ) : Wenn Sie eine langsamen  Internetservice haben und das Spiel ist groß, kann es etwas dauern, um die Karte tatsächlich herunterzuladen. Bitte, gedulden Sie sich ein bisschen. Um zu sehen, welche Geschwindigkeiten Roblox benötigt, sowie weitere Systemanforderungen, klicken Sie bitte hier .
  • Empty Game/Place “ ( Leeres Spiel / Platz ) : Wenn die Karte zu laufen scheint, aber ungewöhnlich leer ist, bedeutet es, dass der Ersteller des Spiels hat wahrscheinlich einfach nichts  eingebaut, also ist es leer. Sobald Ihr Avatar erscheint, können Sie feststellen, wann eine Karte geladen wurde.
  • Bad Game/Place “ ( Schlechtes Spiel / Ort ) : Das Spiel könnte schlecht sein. Es könnte ein schlechtes Skript  oder zu viele Gegenstände enthalten sein, wodurch das Spiel nicht ordnungsgemäß ausgeführt werden kann. In diesem Fall, bitte informieren Sie den Ersteller wegen des Problems, da nur er / sie in der Lage  ist, es zu reparieren.

Was ist zu tun

Bitte versuchen Sie jeden Schritt einzeln nacheinander. Sollte der erste Schritt nicht erfolgreich sein, fahren Sie bitte mit dem nächsten Schritt fort!

Stellen Sie sicher, dass Sie einen unterstützten Browser verwenden

Vergewissere dich, dass du die aktuellste Version deines Browsers verwendest, um Roblox zu spielen. Wenn Sie Probleme beim Spielen mit Ihrem aktuellen Browser haben, versuchen Sie bitte, mit einem anderen Browser zu spielen, z. B. Microsoft Edge, Firefox oder Chrome.

Überprüfen Sie die Sicherheitseinstellungen Ihres Browsers

Sie müssen sicherstellen, dass die Sicherheitseinstellungen Ihres Browsers Roblox zulassen. Wenn diese Einstellungen zu hoch sind, kann dies zu einer Vielzahl von Problemen führen, die von den oben in diesem Artikel aufgeführten bis hin zu Website-Problemen reichen, wie z. B. nicht reagierende Schaltflächen/Links.

Überprüfen Sie Ihre schnurlose Internetverbindung

Sie können die schnurlose Internetverbindung ausschließen, indem Sie möglichst auf eine Kabelverbindung umschalten. Wenn es sich nicht um die Verbindung handelt oder wenn Sie keine kabelgebundene Verbindung versuchen können, überprüfen Sie die Einstellungen Ihrer Firewall, die wir im Folgenden besprechen werden.

Entfernen / Deaktivieren Sie alle Ad-Blocker-Browser-Add-Ons

Browser Add-Ons / Extensions können eine Mengevon Problemen verursachen, insbesondere Ad-Blocker können das vollständige Laden eines Spiels verhindern. Für weitere Informationen hierzu klicken Sie bitte hier .

Stellen Sie sicher, dass die entsprechenden Ports geöffnet sind

Die von Roblox verwendeten Ports hängen vom verwendeten Betriebssystem ab. Der Portbereich ist derzeit UDP 49152 - 65535. Wenn Sie alle diese Ports offen lassen werden, sollte Roblox eine ordnungsgemäße Verbindung zu Ihrem Router herstellen können.

Hinweis: Sollten Sie ein Schuladministrator sein und Roblox Zugriff auf Ihr Klassenzimmer gewähren möchten, lesen Sie bitte diesen Hilfsartikel durch für weitere Informationen.

Konfigurieren Sie Ihre Firewall und / oder Ihren Router

Viele Antiviren- oder Internet Sicherheitssoftware versuchen zu kontrollieren, welche Programme auf das Internet zugreifen können. Oft sind diese Programme auf der sicheren Seite und standardmäßig ist kein Programm erlaubt. Klicken Sie hier , wenn Sie Unterstützung benötigen, Roblox durch Blockaden zu untersuchen.

Installieren Sie Roblox erneut

Als zusätzliche Maßnahme installieren Sie bitte Roblox erneut. Um hier Abhilfe zu schaffen, klicken Sie bitte hier .
