Um eine Verbindung zur Anwendung herzustellen, müssen Sie den Zugriff auf die folgenden URLs zulassen:

HTTP und HTTPS für diese Domänen

api.roblox.com

versioncompatibility.api.roblox.com
chat.roblox.com

assetgame.roblox.com

setup.rbxcdn.com
cdn.arkoselabs.com
roblox-api.arkoselabs.com
js.rbxcdn.com
static.rbxcdn.com

Hinweis: Das Starten des Spiels (Klicken auf die Schaltfläche " Spielen ") unterstützt derzeit keine Proxys. Bitte erlauben Sie auch: assetgame.roblox.com

Sobald das Spiel gestartet wird, verwendet es die UDP-Ports 49152 - 65535. Die IP-Bereiche können unterschiedlich sein und spezifische Informationen können möglicherweise nicht bereitgestellt werden, um einzelne IPs auf die weiße Liste zu setzen.

Häufig auftretende Probleme

Fehler beim Einloggen in Roblox Studio

Wenn Sie sich bei Roblox Studio anmelden, erhalten Sie möglicherweise eine Fehlermeldung, in der Folgendes angezeigt wird: "Es wurden zu viele Anmeldungen von diesem Netzwerk aus versucht. Sie müssen sich über einen Webbrowser anmelden ".

Um sich bei Roblox Studio anzumelden, müssen Sie einen Webbrowser öffnen und sich dort bei Ihren Konten anmelden. Dies fügt eine zusätzliche Sicherheitsebene hinzu, um das Knacken von Konten zu verhindern. Unsere Ingenieure arbeiten derzeit an der Verbesserung des Login, sodass Benutzer diesen Vorgang nur über Roblox Studio und nicht über einen Webbrowser durchführen müssen.
