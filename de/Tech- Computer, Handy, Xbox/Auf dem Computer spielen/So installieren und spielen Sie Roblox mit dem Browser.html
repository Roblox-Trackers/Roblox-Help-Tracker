So installieren und spielen Sie Roblox mit dem Browser

Wie installiere ich den Roblox Player

  1. Gehen Sie zur Roblox website und melden Sie sich bei Ihrem Konto an
  2. Wenn Sie sich bei Roblox eingeloggt haben, besuchen Sie jedes Spiel und klicken Sie auf den grünen Play -Button
  3. Ein Popup-Fenster wird angezeigt das Sie darüber informiert, dass Roblox Player installiert wird
  4. Einmal installiert, wird das Spiel automatisch geöffnet

Hinweis: Es erscheint möglicherweise ein sekundäres Popup-Fenster in dem Sie aufgefordert werden ein Programm auszuwählen. Wählen Sie Roblox und bestätigen Sie. Wenn Sie sich an Ihre Auswahl erinnern, müssen Sie diese Aktion in Zukunft nicht mehr durchführen.

So installieren Sie Roblox Studio

  1. Gehen Sie zur Roblox website und melden Sie sich bei Ihrem Konto an
  2. Besuchen Sie nach dem Anmelden bei Roblox eines Ihrer Spiele, klicken Sie auf in der oberen rechten Ecke und wählen Sie Bearbeiten
  3. Ein Popup-Fenster wird angezeigt das Sie darüber informiert, dass Roblox Studio installiert wird
  4. Einmal installiert, wird das Studio automatisch geöffnet

Hinweis: Es erscheint möglicherweise ein sekundäres Popup-Fenster in dem Sie aufgefordert werden, ein Programm auszuwählen. Wählen Sie Roblox und bestätigen Sie. Wenn Sie sich an Ihre Auswahl erinnern, müssen Sie diese Aktion in Zukunft nicht mehr durchführen.
