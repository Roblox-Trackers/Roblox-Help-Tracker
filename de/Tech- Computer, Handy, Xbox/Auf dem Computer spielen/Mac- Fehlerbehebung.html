Mac: Fehlerbehebung

Deinstallieren / Neu installieren

Wenn Sie ein Problem haben und in letzter Zeit keine Neuinstallation durchgeführt haben, versuchen Sie, das Programm zu deinstallieren und installieren Sie die folgenden Schritte neu:

  • Deinstallieren
  • Neu installieren

Andere Sachen zu versuchen

Wenn die Deinstallation und Neuinstallation nicht hilfreich war, versuchen Sie folgendes

  • Roblox Client wird auf 10.10 (Yosemite) und höher installiert, während Roblox Studio auf Mac OS 10.13 (High Sierra) und höher installiert wird.
  • Derzeit werden Apple Silicon / M1 chips unterstützt.
  • Wenn Sie derzeit den Safari-Webbrowser verwenden und ein Problem auftritt, versuchen Sie es mit einem unserer unterstützten Browser so wie Firefox oder Google Chrome.
  • Stellen Sie sicher, dass sich das Roblox-Symbol gerade in Ihrem Dock befindet und die Roblox-Applikation in Ihrem Applikationsmappe angezeigt wird.
  • Versuchen Sie, das Programm über das Dock zu starten, wenn Sie Probleme beim Spielen eines Spiels von der Website haben.
  • Versuchen Sie, Ihre Roblox-Einstellungen vollständig zurückzusetzen, indem Sie die Einstellungen Dateien löschen. Um dies zu tun gehen Sie zu Library-> Preferences und löschen Sie alle Roblox-com-Dateien. Wenn Studio / Player erneut gestartet wird, sollte es alle neu installieren.

Wenn Sie nach Befolgen der obigen Vorschläge weiterhin Probleme haben, wenden Sie sich bitte an den Roblox-Kundendienst und geben Sie die folgenden Details an:

  • Roblox-Benutzername:
  • Erhaltene Fehlermeldungen:
  • Modellname Ihres Mac:
  • Mac OS-Version:
  • Schritte zur Lösung des Problems:

Bitte geben Sie auch alle zusätzlichen Details an, die uns helfen könnten, das gemeldete Problem besser zu verstehen, sowie alle kürzlich vorgenommenen Änderungen, die Sie an Ihrem Mac oder Netzwerk vorgenommen haben.
