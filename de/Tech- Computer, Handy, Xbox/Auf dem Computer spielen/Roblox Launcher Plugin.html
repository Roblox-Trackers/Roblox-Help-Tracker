Roblox Launcher Plugin

Wenn Roblox nicht ausgeführt werden kann oder eine Aufforderung zum Herunterladen von Roblox anzeigt, obwohl es bereits installiert ist. Gehen Sie zur Überprüfung wie folgt vor:

Mozilla Firefox

  1. Klicken Sie oben rechts in Ihrem Firefox-Fenster auf die Menüschaltfläche
  2. Klicken Sie im Menü auf die Schaltfläche Add-ons
  3. Klicken Sie im angezeigten Fenster auf Plugins
  4. Suchen Sie nach "Roblox Launcher Plugin" und wählen Sie im Dropdown-Menü auf der rechten Seite die Option Immer zulassen aus
  5. Schließen Sie die Registerkarte Add-ons-Manager und starten Sie ein Roblox-Erlebnis

Apple Safari

  1. Öffnen Sie Safari und klicken Sie auf das Zahnradsymbol in der oberen rechten Ecke
  2. Klicken Sie auf Einstellungen und dann auf dem Tab Sicherheit
  3. Stellen Sie sicher, das auf der Registerkarte Sicherheit - JavaScript aktiviert ist

Google Chrome

Hinweis: Ab Version 42 unterstützt Chrome keine Plugins mehr.

Microsoft Edge

Hinweis: Microsoft Edge unterstützt keine Plugins
