Roblox Launcher Plugin

Wenn Roblox nicht ausgeführt werden kann oder eine Aufforderung zum Herunterladen von Roblox anzeigt, obwohl es bereits installiert ist. Sie sollten prüfen, ob das Roblox-Plug-in in Ihrem bevorzugten Browser zusammen mit Internet Explorer zulässig ist, da Roblox für einige Funktionen auf Internet Explorer angewiesen ist. Führen Sie zur Überprüfung Folgendes aus:

Mozilla Firefox

  1. Klicken Sie oben rechts in Ihrem Firefox-Fenster auf die Menüschaltfläche
  2. Klicken Sie im Menü auf die Schaltfläche Add-ons
  3. Klicken Sie im angezeigten Fenster auf Plugins
  4. Suchen Sie nach "Roblox Launcher Plugin" und wählen Sie im Dropdown-Menü auf der rechten Seite die Option Immer zulassen aus
  5. Schließen Sie die Registerkarte Add-ons-Manager und starten Sie ein Roblox Spiel

Internet Explorer

  1. Klicken Sie auf das Zahnradsymbol in der oberen rechten Ecke
  2. Klicken Sie auf Add-Ons verwalten
  3. Im Popup-Fenster das sich links unter 'Anzeigen' befindet, klicken Sie auf das Pulldown-Menü
  4. Wählen Sie Alle Add-ons aus
  5. Auf der rechten Seite des Popup-Fensters, verschieben Sie die Scrollleiste nach unten
  6. Unter Roblox Corporation, stellen Sie sicher das die “Launcher-Klasse” aktiviert ist

Apple Safari

  1. Öffnen Sie Safari und klicken Sie auf das Zahnradsymbol in der oberen rechten Ecke
  2. Klicken Sie auf Einstellungen und dann auf dem Tab Sicherheit
  3. Stellen Sie sicher, das auf der Registerkarte Sicherheit - Plug-ins, Java und JavaScript aktiviert sind

Google Chrome

Hinweis: Ab Version 42 unterstützt Chrome keine Plugins mehr.

Microsoft Edge

Hinweis: Microsoft Edge unterstützt keine Plugins
