Roblox-Launcher-Plugin

Wenn Roblox nicht ausgeführt werden kann oder eine Aufforderung zum Herunterladen von Roblox anzeigt, obwohl es bereits installiert ist. Um das zu überprüfen, gehe wie folgt vor:

Mozilla Firefox

  1. Klicke auf die Menü-Schaltfläche oben rechts in deinem Firefox-Fenster
  2. Klicke im Menü auf die Add-ons Schaltfläche
  3. Klicke im angezeigten Fenster auf Plugins
  4. Suche das "Roblox Launcher Plugin" und wähle Immer aktivieren im Dropdown-Menü auf der rechten Seite
  5. Schließe die Registerkarte Add-ons Manager und starte ein Roblox-Erlebnis

Apple Safari

  1. Öffne Safari und klicke auf das Zahnradsymbol in der oberen rechten Ecke
  2. Klicke auf Einstellungen und dann auf die Registerkarte Sicherheit
  3. Vergewissere dich unter der Registerkarte "Sicherheit", dass JavaScript aktiviert ist

Google Chrome

Hinweis: Ab Version 42 unterstützt Chrome keine Plugins mehr.

Microsoft Edge

Hinweis: Microsoft Edge unterstützt keine Plugins
