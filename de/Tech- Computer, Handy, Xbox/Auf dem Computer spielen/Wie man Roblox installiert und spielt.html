Wie man Roblox installiert und spielt

Wie man Roblox spielt

  1. Roblox-Player installieren
    • Rufen Sie die Roblox-Website mit einem beliebigen modernen Browser wie Microsoft Edge, Firefox oder Chrome auf
    • Nachdem Sie sich bei Roblox angemeldet haben, besuchen Sie ein beliebiges Erlebnis und klicken Sie auf die grüne Schaltfläche " Spielen "
    • Ein Pop-up-Fenster erscheint und informiert Sie darüber, dass der Roblox Player installiert wird
  2. Starten Sie den Player über einen modernen Browser wie Microsoft Edge, Firefox oder Chrome
    • Gehen Sie zu einem beliebigen Spiel auf der Roblox-Website
    • Wählen Sie Spielen
    • Es kann ein zweites Pop-up-Fenster erscheinen, in dem Sie aufgefordert werden, ein Programm auszuwählen. Wählen Sie Roblox und bestätigen Sie. Wenn Sie Ihre Wahl speichern, müssen Sie diese Aktion in Zukunft nicht mehr durchführen

Wie Sie Ihre eigenen Roblox-Spiele erstellen

Sie müssen Roblox Studio installieren, um Ihre eigenen Roblox-Erlebnisse zu erstellen. So gehen Sie vor .

  1. Rufen Sie die Roblox-Website auf und melden Sie sich bei Ihrem Konto an
  2. Wenn Sie sich bei Roblox angemeldet haben, besuchen Sie eines Ihrer Erlebnisse, klicken Sie in der oberen rechten Ecke und wählen Sie Bearbeiten
  3. Es erscheint ein Pop-up-Fenster, das Sie darüber informiert, dass Roblox Studio installiert wird
  4. Nach der Installation wird Studio automatisch geöffnet
