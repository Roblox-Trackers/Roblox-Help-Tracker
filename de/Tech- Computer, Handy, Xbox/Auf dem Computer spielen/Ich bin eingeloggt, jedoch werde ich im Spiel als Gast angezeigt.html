Ich bin eingeloggt, jedoch werde ich im Spiel als Gast angezeigt

Wenn Sie feststellen, dass Sie als Gast spielen, obwohl Sie in Ihrem Konto eingeloggt sind, gibt es ein paar Dinge, die zu beachten sind.

Was zu tun ist - Windows

Probieren Sie jeden einzelnen Schritt der Reihe nach aus. Wenn eine Lösung nicht funktioniert, fahren Sie mit der nächsten fort!

Neustart

Starten Sie zuerst Ihren Computer neu und versuchen Sie zu spielen. Manchmal ist dies alles, was benötigt wird, um Roblox- oder Roblox Studio-Probleme zu lösen.

Setzen Sie Ihre Internetoptionen in Windows zurück

Auch wenn Sie Internet Explorer nicht verwenden, kann das Problem durch das Zurücksetzen der Internetoptionen über diesen Browser behoben werden. Beachten Sie bitte, dass diese Schritte zwar mit dem Internet Explorer durchgeführt werden müssen, Sie aber nach Abschluss des Vorgangs zu Ihrem bevorzugten Browser zurückkehren können. Probieren Sie folgendes aus:

    1. Öffnen Sie Internet Explorer
    2. Öffnen Sie das Zahnradsymbol > Internet Optionen
    3. Klicken sie auf die Schaltfläche Erweitert
    4. Klicken Sie auf die Schaltfläche Zurücksetzen

Schließen Sie nach dem Zurücksetzen den Internet Explorer, damit die Änderungen wirksam werden.

Hinweis: Wenn Sie Windows 10 verwenden, stellen Sie sicher, dass Sie diese Schritte mit Internet Explorer und nicht mit Microsoft Edge ausführen. Die Symbole ähnlich sich sehr. Um IE in Windows 10 zu finden, drücken Sie die Windows-Taste + S auf Ihrer Tastatur und geben Sie "Internet Explorer" ein.

Überprüfen Sie Ihre Windows-Systemuhr

Es mag seltsam klingen, aber verschiedene Probleme können tatsächlich dadurch verursacht werden, dass Ihre Windows-Systemuhr auf die falsche Zeit oder das falsche Datum eingestellt ist. Wenn dies die Ursache ist, sollte einfach alles korrigiert werden, um das Problem zu beheben. Obwohl dies manuell durchgeführt werden kann, können Sie es auch so einstellen, dass es automatisch mit einem Server aktualisiert wird. So wird sichergestellt, dass es nicht zurückfällt und erneut Probleme verursacht. So geht's:

Windows 8.1 und älter

    1. Klicken Sie mit der rechten Maustaste auf die Windows-Uhr in der unteren rechten Ecke
    2. Klicken Sie auf Datum/Zeit anpassen
    3. Wählen Sie die Registerkarte Internetzeit
    4. Hier sollten Sie Zeit und Datum mit einem Internetserver synchronisieren können

Windows 10

    1. Klicken Sie mit der rechten Maustaste auf die Windows-Uhr in der unteren rechten Ecke
    2. Stellen Sie sicher, dass die Option Zeit automatisch einstellen aktiviert ist und Sie die richtige Zeitzone ausgewählt haben

Überprüfen Sie Ihre Proxy-Einstellungen

Wenn die obigen Schritte Ihr Problem nicht gelöst haben, müssen Sie möglicherweise sicherstellen, dass Ihr Computer keine Verbindung über einen Proxy-Server herstellt. Bitte befolgen Sie diese Schritte:

Windows 7 oder älter

    1. Gehen Sie zum Startmenü
    2. Suchen Sie "Proxy" und “konfiguriere Proxy-Server” wird angezeigt, klicken Sie darauf
    3. Klicken Sie auf LAN Einstellungen und deaktivieren Sie alle Häkchen auf dieser Registerkarte
    4. Klicken Sie auf OK

Windows 8 oder 8.1

    1. Gehen Sie zum gekachelten Startbildschirm, indem Sie auf die untere linke Ecke des Bildschirms klicken
    2. Suchen Sie "Proxy" und “konfiguriere Proxy-Server” wird angezeigt, klicken Sie darauf
    3. Klicken Sie auf LAN Einstellungen und deaktivieren Sie alle Häkchen auf dieser Registerkarte
    4. Klicken Sie auf OK

Windows 10

    1. Drücken Sie die Windows-Taste + S auf Ihrer Tastatur. Wenn die Cortanas Leiste "Fragen Sie mich etwas" in Ihrer Taskleiste verfügbar ist, können Sie alternativ auch auf diese klicken
    2. Suchen Sie "Proxy" und “konfiguriere Proxy-Server” wird angezeigt, klicken Sie darauf
    3. Stellen Sie sicher, dass "Setup Script verwenden" und "Proxy Server verwenden" deaktiviert sind
    4. Scrollen Sie nach unten und klicken Sie auf Speichern
