Wie ruft man Client-Protokolle ab

Es kann vorkommen, dass ein Roblox-Mitarbeiter Client-Protokollen über eine Kundendienst-E-Mail oder einen DevForum-Kommentar anfordert, um ein Problem zu untersuchen. Dieser Artikel beschreibt das Abrufen dieser Protokolle.

Hinweis : Bitte rufen Sie diese Protokolle nur ab und senden Sie sie, wenn Sie von einem Roblox-Mitarbeiter dazu aufgefordert werden.

Abrufen von Client-Protokollen

Der Vorgang zum Abrufen von Client-Protokollen hängt vom verwendeten Betriebssystem ab.

Bitte beachten Sie, dass die folgenden Informationen nicht für mobile Geräte gelten.

  1. Öffnen Sie Roblox Studio wie folgt:

Windows 7 / 10 : Start > Alle Programme (oder Alle Apps) > Roblox > Roblox Studio

Windows 8 : Klicken Sie auf das Windows-Logo in der unteren linken Ecke und geben Sie Roblox Studio ein, wenn der gekachelte Startbildschirm angezeigt wird. Wählen Sie Roblox Studio aus der Suchliste, die rechts auf dem Bildschirm angezeigt wird.

Mac : Befehlszeile + Leertaste > Geben Sie RobloxStudio ein > Eingeben

  1. Öffnen Sie Einstellungen , indem Sie auf "Datei" klicken > Einstellungen
  2. Klicken Sie im linken Bereich auf Netzwerk . Wird es nicht angezeigt, schließen Sie das Roblox Studio und öffnen Sie es erneut.
  3. Stellen Sie sicher, dass unter Diagnose die folgenden Kontrollkästchen aktiviert sind:

PrintEvents

PrintInstances

PrintPhysicsErrors

PrintProperties

PrintTouches

  1. Schließen Sie Roblox
  2. Drücken Sie gleichzeitig auf Ihrer Tastatur WINDOWS-TASTE und R.

Für Mac : Macintosh HD; Benutzer; [Ihr Benutzerprofil]; Bibliothek; Protokolle; Roblox

Hinweis: Überspringen Sie den siebten Schritt, wenn Sie MAC verwenden.

  1. Ein Feld mit dem Namen Run wird angezeigt. Kopieren Sie Folgendes und fügen Sie es ein oder geben Sie es ohne Anführungszeichen ein:

Wenn Sie Windows 7/8/10 verwenden : 10: "%localappdata%\Roblox\logs"

  1. Löschen Sie alles im Ordner.
  2. Versuchen Sie, Roblox wie gewohnt zu spielen.
  3. Wenn es nicht funktioniert, schließen Sie Roblox und wiederholen Sie die Schritte 6 und 7.

Der Ordner enthält jetzt mehrere Dateien. Bitte senden Sie sie an info@roblox.com und beschreiben Sie die Situation, in der die Protokolle benötigt wurden.
