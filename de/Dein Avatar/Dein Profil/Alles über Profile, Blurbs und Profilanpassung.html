Alles über Profile, Blurbs und Profilanpassung

Ein Profil anzeigen

Um Ihr eigenes Profil anzuzeigen, klicken Sie einfach auf dem Startbildschirm auf Ihren eigenen Benutzernamen oder im Navigationsmenü auf Profil . Profil einer anderen Person anzeigen:

  • Verwenden des Suchfelds: Sie können den Namen des Profils, das Sie anzeigen möchten, in das Suchfeld oben in der App eingeben.
  • Artikel- oder Erlebnisentwickler: Sie können das Profil des Erstellers eines Erlebnisses oder Gegenstands einsehen, indem Sie auf die Detailseite gehen und auf den Namen tippen. Bei Erlebnissen befindet sich dieser unter dem Titel, bei Gegenständen rechts neben dem Bild.

Die grundlegende Infobox

Die erste Sache, die Sie sehen, wenn Sie ein Profil anschauen, ist die grundlegende Informationsbox. Hier finden Sie das Spieler-Symbol des Anwenders, den Benutzernamen, die Mitgliedschaft-Stufe, die neueste Statusaktualisierung, Interaktionsoptionen und Freunde/Follower-Informationen.

Die Tab-Kreationen


Sie können die Gegenstände oder Erlebnisse, die ein Spieler erstellt hat, anzeigen, indem Sie auf die Registerkarte "Kreationen" des Spielerprofils tippen. Diese befindet sich unter dem Feld mit den Basisinformationen. Auf dieser Registerkarte wird alles angezeigt, was der Benutzer erstellt hat, getrennt nach Kategorien.

Standardmäßig werden die Erlebnisse in einer detaillierten Ansicht angezeigt, die sich jeweils auf ein Erlebnis konzentriert und durch Antippen der Pfeilsymbole links und rechts neben dem Beschreibungsfeld durchgeblättert werden kann. Sie können jedoch auch mehrere Erlebnisse auf einmal anzeigen, indem Sie auf das Symbol tippen.


Wählen Sie, was auf Ihrem Tab Kreationen angezeigt wird

Gelegentlich können Sie feststellen, dass einige Ihrer Kreationen bessere Beispiele für Ihre Fähigkeiten oder Persönlichkeit sind als andere. Wenn dies der Fall ist, können Sie nur bestimmte Schaufenster anzeigen lassen. Um dies zu tun:

  1. Besuchen Sie die Detailseite des Erlebnisses/Artikels
  2. Klicken Sie auf das Zahnradsymbol oder das Symbol, das rechts vom Titel steht und wie drei Punkte aussieht
  3. Klicken Sie im angezeigten Menü entweder auf Zum Profil hinzufügen oder Aus Profil entfernen


Die Info-Tab - Blurb und soziale Netzwerke

Gelegentlich möchten Benutzer vielleicht wissen, worum es geht oder vielleicht wollen Sie Ihre Mitspieler nur mit ein paar lustigen Zitaten unterhalten. Der Klappentext/Blurb, der auf der Registerkarte Info des Profils angezeigt wird, ist der Ort dafür. Um einen eigenen zu erstellen oder einen bereits vorhandenen zu bearbeiten:

  1. Melden Sie sich mit Ihrem Konto
  2. Gehen Sie zur Profilseite
  3. Geben Sie Ihren neuen Klappentext in das Feld Info ein
  4. Klicken Sie auf Speichern

So fügen Sie einem Konto soziale Links hinzu

Wenn Sie 13 Jahre oder älter sind, können Sie über die Registerkarte Sozial auf der Seite Kontoeinstellungen soziale Netzwerke zu Ihrem eigenen Profil hinzufügen, indem Sie die folgenden Schritte ausführen:

  1. Ins Konto einloggen
  2. Klicken Sie auf das Symbol, das wie ein Zahnrad oben rechts auf der Seite aussieht
  3. Wählen Sie im Popup-Menü Einstellungen
  4. Scrolle runter zu soziale Netzwerk
  5. Geben Sie die URL ein und klicken Sie auf Speichern

  6. Wählen Sie die Option Sichtbar für , für die Sie diese Links anzeigen möchten.

Hinweis: Social Media-Links sind nur für Konten über 13 sichtbar.

Oben rechts in Ihrem Blurb Box sehen Sie alle sozialen Netzwerke, die ein Spieler verlinkt hat.

Die Info-Tab - Derzeit tragend

Im Bereich Aktuell tragend auf der Registerkarte Info des Profils sehen Sie ein Bild, wie der Avatar des Spielers aktuell aussieht. Sie können zwischen 2D- und 3D-Ansichten wechseln, indem Sie auf die entsprechende Schaltfläche in der oberen rechten Ecke des Bildes klicken. In der blauen Box auf der rechten Seite dieses Bildes sehen Sie eine Liste von allem, was der Benutzer trägt. Wenn Sie etwas sehen, das Ihnen gefällt, können Sie darauf klicken, um eines für sich selbst zu kaufen oder weitere Details zu erhalten.



Die Info-Tab - Andere Abschnitte

Im Folgenden finden Sie die anderen Abschnitte, die Sie im Info-Tab finden:

Freunde

Bis zu neun Spieler-Freunde auflisten. Wenn sie mehr als neun haben, können Sie alle anzeigen lassen, indem Sie auf die blaue Schaltfläche Alle anzeigen klicken.

Kollektionen

Dies wird Ihnen bis zu sechs Gegenstände anzeigen, die ein Spieler gekauft hat. Standardmäßig sind dies die zuletzt erhaltenen Gegenstände, aber wenn ein Spieler möchtet, kann er wählen, was in diesem Abschnitt angezeigt wird. Um das gesamte Inventar eines Spielers nach Kategorien sortiert anzeigen zu lassen, klicken Sie auf die blaue Schaltfläche Inventar . Wir werden später in diesem Artikel besprechen, was zu tun ist und was in diesem Abschnitt ausgeführt wird.

Gruppen

In diesem Abschnitt werden alle Gruppen angezeigt, denen der Spieler angehört. Ähnlich wie im Abschnitt "Erfahrungen" auf der Registerkarte "Kreationen" werden die Gruppen in einer detaillierten Ansicht angezeigt, die sich jeweils auf eine Gruppe konzentriert. Sie können durch die Gruppen blättern, indem Sie auf die Pfeilsymbole links und rechts neben dem Beschreibungsfeld tippen. Sie können jedoch auch mehrere Gruppen auf einmal anzeigen, indem Sie auf das Symbol tippen.

Lieblingserlebnisse

Hier sehen Sie eine Auswahl von Erlebnissen, die ein Spieler favorisiert hat. Um alle Favoriten eines Spielers anzuzeigen, tippen Sie einfach auf die Schaltfläche Favoriten . Für Informationen darüber, wie Sie selbst ein Erlebnis favorisieren können, tippen Sie hier .

Roblox Abzeichen

Dies zeigt eine Liste aller von Roblox erstellten Abzeichen, die der Benutzer verdient hat. Um zu sehen, wie ein Abzeichen erhalten wurde, klicken Sie einfach darauf.

Spieler Abzeichen

Hier werden die von Spielern erstellten Abzeichen aufgelistet, die ein Spieler durch verschiedene Dinge in bestimmten Erfahrungen erhalten hat. Für weitere Informationen darüber, wie ein Abzeichen verdient wird, tippen Sie einfach darauf.

Info-Tab - Auswählen, was in Ihren Kollektionen angezeigt wird

Möglicherweise, möchten Sie manchmal bestimmte Elemente in Ihrer Kollektion Ihres Profils anzeigen lassen. Um dies zu tun:

  1. Besuchen Sie die Detailseite für einen Artikel, den Sie besitzen
  2. Klicken Sie auf das Zahnradsymbol rechts neben dem Titel
  3. In dem Menü, welche erscheinen wird, klicken Sie entweder auf zum Profil hinzufügen oder aus Profil entfernen
