Wo bearbeite ich meinen Avatar?

Tippen Sie im Navigationsmenü auf Avatar .

Wie bearbeite ich meinen Avatar?

So fügen Sie ein Objekt hinzu

  1. Klicke auf der Avatar-Bearbeitungs-Seite auf eines der verfügbaren Pulldown-Menüs und wähle eine Kategorie aus (z. B. Hut, Haare, Gesicht usw.)
  2. Wenn Sie mehrere Elemente für ein bestimmtes Teil haben, Scrollen Sie nach unten, bis das gewünschte Element gefunden haben.
  3. Sobald Sie es ausgewählt haben, klicken Sie auf das Element-Symbol, damit es grün markiert wird.

Wenn Sie mehr als das Softlimit (z. B. mehr als drei Hüte) eines bestimmten Artikels hinzufügen möchten, klicken Sie in der unteren rechten Ecke auf Fortgeschritten . Sie können bis zu 10 Elemente hinzufügen, indem Sie die Asset-ID eingeben, die Sie in der Link finden. Siehe Beispiel unten:

(https://www.roblox.com/catalog/ 62234425 /Brown-Hair <- Die Nummern in dem Link ergeben die Asset ID)

So entfernen Sie ein Objekt

  1. Klicken Sie auf der "Avatar Editor" Seite auf Vor Kurzem , oder wählen Sie die entsprechende Kategorie aus (z. B. Hut, Haare, Gesicht usw.).
  2. Im Pulldown-Menü (das Menü zum herunterziehen) "Zuletzt verwendet" sollten aktuell getragene Artikel zu den ersten gehören. Wenn Sie stattdessen die zutreffende Kategorie auswählen, scrollen Sie nach unten, bis Sie das gefunden haben, was Sie entfernen möchten.
  3. Klicke auf das grün markierte Item-Icon.

Hinweis: Wenn Sie ein Objekt hinzugefügt oder entfernt haben und Ihr Avatar es nicht korrekt anzeigt, klicken Sie auf den Link "Neu zeichnen" unter dem Bild Ihres Avatars.

Wie man die Hautfarbe ändert

Während das Ändern deiner Gegenstände / Kleidung sicherlich Spaß macht, möchtest du vielleicht manchmal die Hautfarbe deines Avatars ändern. Die folgenden Schritte helfen Ihnen dabei, die Hautfarbe Ihres Avatars über die Roblox-Website zu ändern.

  1. Gehen Sie auf den Avatar -Bereich auf der linken Seite.
  2. Bewegen Sie auf dieser Seite den Mauszeiger über das Pulldown-Menü Körper und wählen Sie Hautfarbe .
  3. Klicke auf eine Farbe, um das Aussehen deines gesamten Avatars zu ändern.
  4. Wenn Sie für einzelne Körperteile eine andere Farbe wünschen, klicken Sie in der unteren rechten Ecke auf Fortgeschritten .
  5. Klicken Sie von hier aus auf ein Körperteil (Kopf, Oberkörper, linkes Bein usw.) und wählen Sie dann die gewünschte Farbe.
