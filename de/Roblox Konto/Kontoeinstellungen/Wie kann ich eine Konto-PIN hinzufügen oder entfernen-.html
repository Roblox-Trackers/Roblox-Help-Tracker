Wie kann ich eine Konto-PIN hinzufügen oder entfernen?

Sie haben die Möglichkeit, eine vierstellige Konto-PIN hinzuzufügen, um Ihre Einstellungen zu sichern. Nachdem eine Konto-PIN hinzugefügt wurde, wird diese für jede Änderung an den Kontoeinstellungen benötigt. Diese Funktion schützt Ihr Passwort, Ihre E-Mail-Adresse sowie die Datenschutzeinstellungen.

Wie kann ich eine Konto-PIN hinzufügen?

  1. Melden Sie sich unter Ihrem Konto an
    (Wenn das nicht möglich ist, versuchen Sie diese Schritte, um Ihr Passwort wiederherzustellen .)
  2. Öffne die Kontoeinstellungen
    • Browser – suchen Sie nach dem Zahnradsymbol oben rechts auf der Seite
    • Mobile Apps – suchen Sie nach dem Symbol mit den drei Punkten, das für Mehr steht
  3. Wähle die Registerkarte Kindersicherung
  4. Schalten Sie die Option Eltern PIN auf Ein/On
  5. Erstellen und bestätigen Sie Ihre neue PIN

Teilen Sie niemandem Ihre PIN mit - diese ist sogar noch wichtiger als Ihr Passwort.

Wie kann ich eine Konto-PIN entfernen?

  1. (Wenn das nicht möglich ist, versuchen Sie diese Schritte, um Ihr Passwort wiederherzustellen .)
  2. Öffne die Kontoeinstellungen
    • Browser – suchen Sie nach dem Zahnradsymbol oben rechts auf der Seite
    • Mobile Apps – suchen Sie nach dem Symbol mit den drei Punkten, das für Mehr steht
  3. Wähle die Registerkarte Kindersicherung
  4. Schalten Sie die Option Eltern PIN aus
  5. Geben Sie die aktuelle PIN ein und wählen Sie Entsperren

Wie kann ich eine PIN ändern?

Um eine PIN zu ändern, befolgen Sie zunächst die Schritte Wie kann ich eine Konto-PIN entfernen . Dann folgen Sie den Schritten Wie kann ich eine Konto-PIN hinzufügen , um eine neue PIN zu erstellen.
