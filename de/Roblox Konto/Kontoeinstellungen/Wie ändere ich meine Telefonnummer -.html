Wie ändere ich meine Telefonnummer ?

Diese Funktion ist für Spieler ab 13 Jahren. Eine Telefonnummer darf nur für ein Konto verwendet werden.

Um eine Telefonnummer hinzuzufügen

  1. Melden Sie sich mit Ihrem Konto an
    ( Wenn Sie nicht auf das Konto zugreifen können, bitte klicken Sie hier )
  2. Gehen Sie zu den Kontoeinstellungen
    • Browser - finden Sie das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps - Finde das Drei-Punkte-Symbol für Mehr
  3. Wählen Sie die Registerkarte Kontoinformationen
  4. Wählen Sie die Schaltfläche Telefon hinzufügen
    1. Wählen Sie Ihren Standort aus dem Pulldown-Menü
    2. Geben Sie die Telefonnummer und das Kontopasswort ein
    3. Wählen Sie Telefon hinzufügen

Aktualisieren oder entfernen einer vorhandene Telefonnummer

  1. Melden Sie sich mit Ihrem Konto an
    ( Wenn Sie nicht auf das Konto zugreifen können, bitte klicken Sie hier )
  2. Gehen Sie zu den Kontoeinstellungen
    • Browser - finden Sie das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps - Finde das Drei-Punkte-Symbol für Mehr
  3. Wählen Sie die Registerkarte Kontoinformationen
  4. Klicken Sie auf die Schaltfläche Telefon aktualisieren (das Symbol sieht aus wie Bleistift und Papier)
    1. Wählen Sie Ihren Standort aus dem Pulldown-Menü
    2. Geben Sie die Telefonnummer und das Kontopasswort ein
    3. Wählen Sie Telefon bearbeiten oder Telefonnummer entfernen

Befehle für die Nachrichten

Hier sind einige nützliche Befehle zum Starten und Stoppen von Textnachrichten, die Sie möglicherweise von Roblox erhalten:

  • STOPPEN, ALLE STOPPEN, ABONNIEREN, ABBRECHEN, ENDE und BEENDEN - Diese Einzelwortbefehle stoppen zukünftige Textnachrichten. Bitte beachten Sie, dass nur einzelne Wortmeldungen den Block auslösen. Wenn Sie beispielsweise auf "STOP" antworten, wird sichergestellt, dass Sie keine weiteren Nachrichten erhalten, aber "STOPP BITTE" wird nicht funktionieren.
  • START, JA und AUFMACHEN- Mit diesen Befehlen können Sie sich wieder für den Empfang von Textnachrichten entscheiden.
  • HILFE und INFO - Diese Befehle enthalten Informationen, mit denen Sie die oben genannten Befehle zum Steuern der Zustellung von Nachrichten verwenden können.
