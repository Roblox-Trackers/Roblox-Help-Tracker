Fehlersuche für 2SV (2-Step Verification, zweistufige Überprüfung)

Ich erhalte die E-Mail für die 2-Step Verification nicht

  1. Wenn du mehrere E-Mail-Adressen hast, überprüfe, ob du den Posteingang für die E-Mail-Adresse abgerufen hast, die du für dein Roblox-Konto angegeben hast.
  2. Überprüfe deine Spam-, Junk-, Werbe-, Social Media- und sonstigen Ordner, um zu prüfen, ob die Nachricht von deinem E-Mail-Anbieter ausgefiltert wurde.
    • Wenn du die Nachricht in deinem Spam-Ordner findest, kennzeichne sie als Kein Spam , um zu verhindern, dass sie in Zukunft ausgefiltert wird.
    • Fugen Sie die E-Mail-Adresse no-reply@roblox.com zu Ihren E-Mail-Kontakten hinzu.
  3. Verwende die Funktionen Code erneut senden oder Neustart auf dem Anmeldebildschirm für die 2-Schritt-Verifizierung.

Sie können auch versuchen, eine andere von Ihnen eingerichtete 2-Schritt-Verifizierungsmethode zu verwenden, wenn Sie weiterhin Probleme haben.

Wenn du nach Ausführung dieser Schritte den 2-Stufen-Code immer noch nicht erhalten hast, wende dich bitte an den Kundendienst .

Mein 2-Schritt Verifizierung-Code funktioniert nicht

2-Schritt Verifizierung-Code verfallen nach 15 Minuten und funktionieren nicht mehr, wenn bereits ein falscher Code für dieses Konto eingegeben wurde.

Verwende die Funktionen Code erneut senden oder Neustart auf dem Anmeldebildschirm für die 2-Schritt Verifizierung, um einen neuen Code anzufordern.

Wenn du dich auch mit einem neuen Code nicht anmelden kannst, wende dich bitte an den Kundendienst .

Mein Authentifizierung-Bestätigungscode funktioniert nicht

Die Codes der Authentifizierungs-App 2-Schritt-Verifizierung funktionieren nur, wenn sie in der App angezeigt werden. Vergewissern Sie sich, dass Sie einen Code eingeben, der derzeit aktiv ist. Sie müssen den neuen Code eingeben, der in der App angezeigt wird, wenn der vorherige Code nicht funktioniert hat.

Wenn Sie weiterhin Probleme haben, können Sie versuchen, eine andere 2-Schritt-Verifizierungsmethode zu verwenden, die Sie eingerichtet haben.

Wenn Sie sich immer noch nicht mit einem neuen Code anmelden können, wenden Sie sich bitte an den Kundensupport .

Ich habe meine Wiederherstellungscodes verloren

Erstellen Sie einen neuen Satz von Wiederherstellungs-Codes, indem Sie den Anweisungen im Artikel " Deinem Konto eine 2-Schritt Verifizierung hinzufügen " folgen.
