Deinem Konto eine 2-Step Verification (2-stufige Überprüfung) hinzufügen

Wir empfehlen dringend, die 2-Schritte-Verifizierung zu aktivieren, um die Sicherheit Ihres Kontos zu erhöhen. Wenn Sie sich von einem neuen Gerät aus anmelden, geben Sie einen eindeutigen Sicherheitscode aus einer Authentifizierungs-App oder der E-Mail ein, die Roblox Ihnen sendet.

Warum ist sie wichtig?

Diese Funktion hat mehrere Vorteile für den Schutz Ihres Kontos.

  • Diese Funktion stellt sicher, dass sich keine andere Person bei Ihrem Konto anmelden kann, selbst wenn sie das Passwort kennt.
  • Da nur Sie Zugriff auf die Authenticator-App oder Ihr E-Mail-Konto haben, können auch nur Sie den Sicherheitscode erhalten.
  • Zusätzliche Sicherheit, damit du den Zugang zu deinen Erfahrungen, Robux-Limits und mehr nicht verlierst.

Aktivierung der 2-Step Verification

  • Melden Sie sich in Ihrem Konto an .
    • Wenn das nicht möglich ist, versuchen Sie es mit den Schritten für die Wiederherstellung Ihres Passworts .
  • Für die Aktivierung dieser Funktion brauchen Sie eine verifizierte E-Mail-Adresse .
      • Hier sind die Schritte für die Verifizierung deiner E-Mail-Adresse beschrieben.
  • Öffnen Sie Kontoeinstellungen .
    • Browser – suchen Sie nach dem Zahnradsymbol oben rechts auf der Seite.
    • Mobile Apps – suchen Sie nach dem Symbol mit den drei Punkten, das für Mehr steht.
  • Wählen Sie die Registerkarte Sicherheit.
  • Befolgen Sie die Schritte auf dem Bildschirm, um die Einrichtung abzuschließen.

Hinweis : Sie können nur entweder eine Authentifizierungs-App aktivieren oder eine E-Mail-Adresse für die Bestätigungscodes verwenden.

Einrichten der 2-Schritt-Verifizierung durch eine Authenticator-App

  1. Toggle Authenticator App (sehr sicher) .
    • Wenn das Konto über eine Konto-PIN verfügt, werden Sie aufgefordert, diese einzugeben
  2. Geben Sie Ihr Kontopasswort ein, um Ihre Anfrage zu bestätigen.
  3. Scannen Sie den QR-Code oder geben Sie den Einrichtungscode manuell in der Authenticator-App ein.
    • Vergewissern Sie sich, dass Sie eine Authenticator-App auf Ihrem Mobilgerät installiert haben.
        • Beispiele: Google Authenticator, Microsoft Authenticator oder Twilio Authy
  4. Die Wiederherstellungscodes werden automatisch für Sie generiert. Stellen Sie sicher, dass Sie diese an einem sicheren Ort speichern und teilen Sie sie mit niemandem. Wenn Sie Ihre Authenticator-App und Ihre Wiederherstellungscodes verlieren, können Sie nicht mehr auf Ihr Konto zugreifen.

2-Schritt-Verifizierung mit Sicherheitsschlüsseln einrichten

Beachten Sie, dass Sicherheitsschlüssel derzeit nur in Webbrowsern unterstützt werden. Sie können Ihren Sicherheitsschlüssel nur verwenden, wenn Sie die Authenticator-App aktiviert haben und sich über einen Webbrowser und nicht über die Roblox-App anmelden.

Befolgen Sie die Anweisungen unter "2-Schritt-Verifizierung mit der Authenticator-App einrichten". Authenticator 2FA muss aktiviert sein, um Sicherheitsschlüssel einzurichten.

  1. Schalten Sie "Sicherheitsschlüssel nur im Web (sehr sicher)" ein.
  2. Folgen Sie den Anweisungen Ihres Browsers, um den Sicherheitsschlüssel Ihrer Wahl zu verifizieren
    • Sie können einen physischen Schlüssel oder eine von der Hardware Ihres Geräts verwaltete Methode wie Gesichtserkennung oder einen Fingerabdruckscanner verwenden
    • Beispiele für physische Sicherheitsschlüssel: YubiKey 5, YubiKey 5C, Google Titan Security Key
    • Die Optionen für Sicherheitsschlüssel variieren je nachdem, welchen Browser (Safari, Chrome, Firefox) und welches Betriebssystem (Windows oder MacOS) Sie verwenden
  3. Geben Sie einen Namen für Ihren Schlüssel ein, damit Sie sich daran erinnern können, welchen Schlüssel Sie verwendet haben. Beispiel: "Blue YubiKey 5"
    • Example key name: “Blue YubiKey 5”
  4. Sie können diese Schritte wiederholen, um weitere Schlüssel hinzuzufügen, oder das Fenster schließen, wenn Sie fertig sind .

2-Schritt-Verifizierung nach E-Mail-Adresse einrichten

  1. Schalten Sie E-Mail (sicher) um .
    • Wenn das Konto über eine Konto-PIN verfügt, werden Sie aufgefordert, diese einzugeben.
  2. Geben Sie Ihr Kontopasswort ein, um Ihre Anfrage zu bestätigen.

Deaktivierung der 2-Step Verification

  1. Melden Sie sich in Ihrem Konto an .
    • Wenn das nicht möglich ist, versuchen Sie es mit den Schritten für die Wiederherstellung deines Passworts .
  2. Öffnen Sie die Kontoeinstellungen
    • Browser - finden Sie das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps – suchen Sie nach dem Symbol mit den drei Punkten, das für Mehr steht
  3. Wählen Sie die Registerkarte Sicherheit
  4. Deaktivieren Sie die 2-Step Verification
  5. Geben Sie das Passwort für Ihr Konto ein, um die Deaktivierung der Funktion abzuschließen.

Alle zuvor erstellten Wiederherstellungscodes werden auch deaktiviert, wenn Sie die Bestätigung in zwei Schritten deaktivieren. Hilfe zum Zurücksetzen des Kontopassworts finden Sie im Artikel Passwort vergessen Ich habe mein Passwort vergessen .

So erhalten Sie Wiederherstellungscodes

  1. Melden Sie sich in Ihrem Konto an
  2. Öffnen Sie die Kontoeinstellungen
    • Browser - finden Sie das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps – suchen Sie nach dem Symbol mit den drei Punkten, das für Mehr steht
  3. Wählen Sie die Registerkarte Sicherheit
  4. Klicken Sie im Abschnitt Wiederherstellungscodes auf Generieren.
  5. Geben Sie Ihr Kontopasswort als Teil des Einrichtungsprozesses ein.
  6. Stellen Sie sicher, dass Sie die Codes an einem sicheren Ort aufbewahren, da Sie sie nur einmal sehen können.

Jeder Code kann nur einmal verwendet werden. Wenn Sie Ihre Wiederherstellungscodes verlieren, können Sie diesen Vorgang wiederholen, um neue Codes zu erstellen. Alle zuvor erstellten Wiederherstellungscodes werden deaktiviert, wenn Sie neue Codes erstellen.

Berechtigung zur Kontowiederherstellung

Um zur Kontowiederherstellung berechtigt zu sein, müssen Sie eine Authentifizierungs-App zum Schutz Ihres Kontos eingerichtet haben. Außerdem müssen Sie Roblox innerhalb von 30 Tagen nach der Kompromittierung des Kontos benachrichtigen. Wenn wir innerhalb dieser Zeit nicht benachrichtigt werden, können wir den Bestand des Kontos in der Regel nicht wiederherstellen.
