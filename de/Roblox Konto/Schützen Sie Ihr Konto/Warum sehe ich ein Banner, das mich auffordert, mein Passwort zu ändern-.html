Warum sehe ich ein Banner, das mich auffordert, mein Passwort zu ändern?

Wenn es verdächtige Aktivitäten in Ihrem Roblox-Konto gibt oder wir feststellen, dass Ihr Passwort möglicherweise von anderen Diensten* gestohlen wurde, können wir Sie auffordern, Ihr Passwort zu ändern. Indem Sie Ihr Passwort ändern, stellen Sie sicher, dass nur Sie Ihr Konto nutzen können.

Helfen Sie, Ihr Konto sofort zu sichern

  1. Melden Sie sich bei Ihrem Konto an.
  2. Sie werden aufgefordert, Ihr Passwort zu ändern. Klicken Sie auf dem Banner auf Passwort ändern und Sie werden aufgefordert, Ihr aktuelles und neues Passwort einzugeben.
    Wenn für das Konto eine PIN existiert, müssen Sie diese eingeben, wenn Sie dazu aufgefordert werden.
  3. Erstellen Sie ein neues, sicheres Passwort, das Sie nicht für andere Dienste verwenden und das sich von den bisher verwendeten unterscheidet.
  4. Erwägen Sie die Verwendung eines Passwort-Managers, um sichere und eindeutige Passwörter zu erstellen, damit Sie sich diese nicht merken müssen.

Sichern Sie Ihr Konto mit der 2-Schritt-Verifizierung ab

Die 2-Schritt-Verifizierung sorgt dafür, dass sich keine andere Person bei Ihrem Konto anmelden kann, selbst wenn sie das Passwort kennt. Wenn Sie sich von einem neuen Gerät aus anmelden, sendet Roblox Ihnen einen Code, mit dem Sie sich nur anmelden können, wenn Sie diesen Code haben. Weitere Informationen finden Sie im Hilfeartikel 2-Schritt-Verification zu Ihrem Konto hinzufügen .

  1. Aktivieren Sie die 2-step Verification über eine Authentifizierungs-App
  2. Aktivieren Sie die 2-step Verification per E-Mail

Ihr Passwort könnte durch Datenschutzverletzungen von anderen Diensten gestohlen werden, wenn Sie dasselbe Passwort auf verschiedenen Websites verwenden. Um Ihr Konto zu schützen, erstellen Sie ein eindeutiges Passwort nur für Roblox.
