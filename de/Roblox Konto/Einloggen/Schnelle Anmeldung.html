Schnelle Anmeldung

Mit unserer Schnellanmeldungsfunktion können Sie sich von einem anderen Gerät, bei dem Sie bereits angemeldet sind, bei einem neuen Gerät anmelden, ohne dass Sie Ihr Passwort erneut eingeben müssen.

So melden Sie sich bei einem neuen Gerät an

Um sich anzumelden, müssen Sie den beiden Anweisungen folgen.

Generieren Sie einen einmaligen Code

  1. Öffnen Sie auf Ihrem neuen Gerät die Anmeldeseite.
  2. Klicken Sie auf die Schaltfläche “ Mit einem anderen Gerät anmelden” .
  3. Es wird ein einmaliger Code angezeigt, der nur 5 Minuten gültig ist.

Authentifizieren Sie den Einmalcode

  1. Gehen Sie auf Ihrem angemeldeten Gerät zu " Kontoeinstellungen " und dann zu " Schnellanmeldung ".
  2. Geben Sie nach Beachtung der Warnungen den auf Ihrem neuen Gerät angezeigten Einmalcode ein und klicken Sie auf die Eingabetaste .
  3. Überprüfen Sie alle Details, bevor Sie " Vollständigen Kontozugriff " gewähren auswählen.
  4. Sobald sich Ihr neues Gerät angemeldet hat, wird eine Erfolgsmeldung angezeigt.

QR-Code-Fehlerbehebung

Wenn Sie eine Option zum Anmelden mit einem QR-Code sehen und Probleme auftreten, versuchen Sie, diese Schritte zu folgen:

  1. Stellen Sie sicher, dass Sie die neueste Version der Roblox-App installiert haben. 1.
    1. Außerdem benötigen Sie eine weitere App, um den QR-Code zu scannen. Probieren Sie die Kamera-App Ihres Geräts oder einen beliebigen QR-Code-Scanner Ihrer Wahl aus!
  • Stellen Sie sicher, dass Sie bei dem Gerät angemeldet sind, von dem Sie den QR-Code scannen.
  • Auf einigen Telefonen (zB. Google Pixel) verarbeitet möglicherweise den QR die integrierte Kamera-App nicht. Versuchen Sie in dieser Situation, einen anderen QR-Code-Scanner zu installieren oder sich durch Eingabe einen sechsstelligen Code  anzumelden.

    • HINWEIS: Diese Funktion ist derzeit auf Xbox nicht verfügbar. Diese Funktion geht nur, wenn sich Ihre Geräte in der Nähe befinden.

    Und schließlich ist diese Funktion derzeit möglicherweise nicht für alle Spieler verfügbar.
