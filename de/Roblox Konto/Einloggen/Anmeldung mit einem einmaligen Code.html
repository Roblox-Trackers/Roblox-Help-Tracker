Anmeldung mit einem einmaligen Code

Mit einem einmaligen Code können Sie sich anmelden, ohne Ihren Benutzernamen und Ihr Passwort einzugeben. Momentan sind einmalige Codes nur für einige Nutzer verfügbar.

Hinweis: Um diese Funktion nutzen zu können, müssen Sie eine verifizierte E-Mail-Adresse in Ihrem Konto haben. Auf der Seite mit den Kontoinformationen können Sie Ihre E-Mail-Adresse hinzufügen und bestätigen (oder die E-Mail-Adresse eines Elternteils für Benutzer unter 13 Jahren). In diesem Hilfeartikel können Sie nachlesen, wie Sie die E-Mail-Adresse des Kontos verifizieren.

Wie meldet man sich mit einem einmaligen Code an

  1. Drücken Sie auf der Anmeldeseite die Schaltfläche Senden Sie mir einen einmaligen Code .
  2. Geben Sie die E-Mail-Adresse ein, die in Ihrem Konto verifiziert ist. Sie erhalten per E-Mail einen sechsstelligen, einmaligen Code.
  3. Geben Sie den einmaligen Code ein, um sich anzumelden (wenn Ihre E-Mail auf mehreren Konten verifiziert ist, können Sie eines auswählen).

Ich habe keinen einmaligen Code erhalten

Versuchen Sie diese Schritte, wenn Sie keine E-Mail mit einem einmaligen Code erhalten haben.

  • Überprüfen Sie Ihren Spam-/Junk-Ordner.
  • Klicken Sie auf die Schaltfläche erneut senden und überprüfen Sie dann erneut Ihren Posteingang.
  • Vergewissern Sie sich, dass Sie Ihre E-Mail-Adresse genau so eingegeben haben, wie sie in Ihrem Konto angegeben ist.
  • Stellen Sie sicher, dass Ihre E-Mail-Adresse verifiziert ist. In diesem Hilfeartikel können Sie nachlesen, wie Sie die E-Mail-Adresse des Kontos verifizieren.

Wenn keiner dieser Schritte funktioniert, melden Sie sich bitte mit Ihrem Passwort an. Möglicherweise müssen Sie eine Weile warten, bis einmalige Codes für Ihre E-Mail wieder funktionieren.

Ich habe einen einmaligen Code erhalten, aber er funktioniert nicht

Versuchen Sie diese Schritte, wenn Ihr einmaliger Code nicht funktioniert.

  1. Stellen Sie sicher, dass Sie den Einmalcode auf demselben Gerät eingeben, auf dem Sie ihn angefordert haben.
  2. Drücken Sie die Schaltfläche Erneut senden und versuchen Sie es noch einmal mit einem neuen Code.

Wenn keiner dieser Schritte funktioniert, melden Sie sich bitte mit Ihrem Passwort an. Möglicherweise müssen Sie einige Zeit warten, bis einmalige Codes für Ihre E-Mail wieder funktionieren.

Sicherheitstipps

  • Geben Sie einen einmaligen Code niemals an andere weiter. Roblox wird Sie niemals kontaktieren, um den einmaligen Code anzufordern.
  • Wenn Sie eine E-Mail mit einem einmaligen Code erhalten, ohne diesen anzufordern, können Sie sie getrost ignorieren.
  • Wenn Sie mehrere unerwünschte E-Mails mit Einmal-Codes erhalten, raten wir Ihnen dringend, die E-Mail-Adresse in Ihrem Konto zu ändern und die 2-Schritt-Verifizierung durch die Authenticator App einzurichten .
