Bearbeiten und Testen in Studio (Bearbeitungsmodus)

Der Bearbeitungsmodus ist eine Funktion in Roblox Studio, mit der Sie alle oder die meisten Teile eines Ortes bearbeiten können, ohne dass sich etwas bewegt (als ob die Zeit stehen geblieben wäre ). Es bietet eine Vielzahl von Werkzeugen und Optionen, mit denen Sie Ihren Ort aufbauen und anpassen können. Bevor Sie den Bearbeitungsmodus verlassen, sollten Sie das, woran Sie gearbeitet haben, auf Ihrem Computer speichern.

In den Bearbeitungsmodus wechseln

Der Bearbeitungsmodus kann auf eine der folgenden Arten erreicht werden:

  • Indem Sie die Detailseite eines Erlebnisses aufrufen, klicken Sie auf das Symbol das wie drei Punkte aussieht, über der Wiedergabe Schaltfläche, und wählen Sie Bearbeiten aus dem angezeigten Menü. Bitte beachten Sie, dass dies eines Ihrer eigenen Spiele sein muss oder, wenn es jemand anderem gehört, muss es nicht Copy Locked sein
  • Gehen Sie zu Ihrem Erstellen Abschnitt, suchen Sie die Stelle, die Sie bearbeiten möchten, und klicken Sie auf die entsprechende Schaltfläche Bearbeiten .
  • Öffnen Sie einen Platz, der auf Ihrem Computer gespeichert ist, indem Sie im Studio Menü Datei Öffnen auswählen
  • Indem Sie in Studio Datei> Neu auswählen. Dies öffnet einen leeren Platz, der in fast alles was Sie möchten, verwandelt werden kann

Die Tabs

Studio bietet eine Vielzahl von verschiedenen Optionen in schnell anklickbaren Schaltflächen in verschiedenen Registerkarten an. Diese Registerkarten sind:

  • Startseite : Diese enthält eine Vielzahl der am häufigsten verwendeten Tasten für Kurzanleitung
  • Model : Hier finden Sie Schaltflächen, die die meisten Optionen zum Einfügen und Ändern von Objekten darstellen
  • Gelände : Mit diesen Werkzeugen können Sie das Gelände modifizieren
  • Test : Alle Dinge, die Sie benötigen, um zu testen, wie Ihre Änderungen innerhalb des Erlebnisses funktionieren, finden Sie hier. Darüber hinaus können Sie im Bereich Emulation auf dieser Registerkarte sehen, wie Ihr Spiel auf einem von verschiedenen Mobilgeräten oder Fernsehgeräten aussieht.
  • Ansicht : Ermöglicht das Ein- oder Ausblenden der verschiedenen Bereiche/Fenster in Studio
  • Plugins : Hier finden Sie alle Plugins, die Sie heruntergeladen oder Ihrem Konto hinzugefügt haben. Weitere Informationen zu dieser Funktion finden Sie im Artikel Plugins .

Die Abschnitte Explorer und Eigenschaften

Sobald Sie sich im Bearbeitungsmodus befinden ( entweder an einem neuen Ort oder an einem bereits gebauten ) sehen Sie auf der rechten Seite des Bildschirms zwei Abschnitte mit den Bezeichnungen Explorer und Eigenschaften. Wenn Sie eines oder beide nicht sehen, können Sie sie wieder öffnen, indem Sie auf die entsprechenden Symbole auf der Registerkarte Ansicht klicken.

  • Explorer
    Dieser Bereich listet alles auf, was sich innerhalb deines Ortes befindet. Es wird in verschiedene Gruppen unterteilt, wie z.B. Arbeitsbereich, Beleuchtung, Anfängerpaket, Spieler...usw. Um zu sehen, was sich in einer Dienstgruppe befindet, klicken Sie einfach auf den kleinen grauen Pfeil links neben der Gruppe. Alle Objekte, die Sie an Ihrem Platz sehen, und die meisten Skripte befinden sich in der Gruppe Arbeitsbereich. Wenn ein Objekt andere Objekte ( wie zum Beispiel Abziehbilder ) enthält, wird es auch einen kleinen Pfeil haben. Um eine Gruppe zu schließen und ihren Inhalt zu verbergen, klicken Sie einfach erneut auf den Pfeil.
  • Eigenschaften
    In diesem Abschnitt finden Sie eine Liste der Eigenschaften eines Objekts, wie z. B. Größe, Drehung, ob es verschoben werden kann ( verankert )...usw. Das Objekt muss ausgewählt sein, um seine Eigenschaften anzuzeigen.

Die Kamera bedienen

Die Kamera ist einfach zu bedienen. Anders als wenn Sie Ihr Erlebnis online spielen, ist die Kamera nicht nur auf ein Objekt beschränkt ( z.B. Ihr Kopf) . Stattdessen können Sie es überall und in jedem Winkel bewegen. Verwenden Sie entweder W, A, S, D oder die Pfeiltasten, um die Kamera vorwärts, rückwärts, links und rechts zu bewegen. Klicken und halten Sie die rechte Maustaste und bewegen Sie die Maus, um Ihre Ansicht zu drehen. Halten Sie die rechte Maustaste gedrückt und bewegen Sie die Maus, um Ihre Ansicht zu drehen. Wenn Sie das Mausrad mit dem Mausrad nach oben rollen, wird es vergrößert, während es beim Herunterrollen verkleinert wird.

Objekte hinzufügen und verschieben

Das Hinzufügen von Bausteinen, Skripten und Modellen ( sowie vieler anderer Objekte ) im Bearbeitungsmodus ist einfach und es gibt mehrere Möglichkeiten, dies zu erreichen. Die erste Option besteht darin, einfach mit der rechten Maustaste auf ein leeres Feld im Erlebnisfenster zu klicken, Objekt einfügen auszuwählen und das Objekt selektieren, das Sie zu Ihrem Platz hinzufügen möchten .

Alternativ können Sie auch die vielen Schaltflächen im Modell-Tab verwenden, um schnell eine Vielzahl von Dingen wie Teile, Steine, Effekte, Spawn-Punkte, Skripte usw. einzufügen. Um die Platzierung zu vereinfachen, können Sie viele verschiedene Bausteine, Modelle oder Skripte per Drag & Drop in das Erlebnisfenster ziehen, indem Sie die Toolbox auf der linken Seite des Bildschirms verwenden. Wenn diese nicht sichtbar ist, können Sie sie auf der Registerkarte Ansicht erneut öffnen.

Das Verschieben von Dingen, die Sie bereits zu Ihrem Erlebnis hinzugefügt haben, ist sogar noch einfacher! Zuerst klicken Sie einfach auf das Symbol Verschieben auf der Registerkarte Modell am oberen Rand des Bildschirms. Sie können dann ein Objekt frei bewegen, indem Sie es anklicken und im Erlebnisfenster verschieben. Wenn Sie es nur nach oben/unten, nach links/rechts oder nach innen/außen verschieben möchten, klicken Sie einfach auf das Objekt, um eine Reihe von Pfeilen anzuzeigen. Klicken Sie auf das Symbol, das in die gewünschte Richtung zeigt, und ziehen Sie die Maus.

Hinweis: Einige Objekte, wie Glitzern und Skripts, sind nicht sichtbar und können daher nicht verschoben werden.

Der Werkzeugkasten

Mit der Werkzeugkasten können Entwickler Modelle, Aufkleber und mehr, die von anderen Spielern erstellt wurden, per Ziehen & Ablegen an ihren Platz ziehen. Der Werkzeugkasten ermöglicht Entwickler Modelle, Aufkleber und mehr, die von anderen Spielern erstellt wurden, durch Ziehen & Ablegen an ihren Platz ziehen. Dies ähnelt der Option Objekt Einfügen innerhalb des Rechtsklick-Menüs, bietet jedoch vorgefertigtes Material und von Spielern erstellte Inhalte an.

Größe ändern und Drehen von Objekte

Um die Größe der Bausteine ​​zu ändern, klicken Sie auf die Schaltfläche Skalieren auf der Registerkarte Modell, wählen Sie einen Baustein aus, und kleine Punkte sollten oben, unten, links, rechts, vorne und hinten am Objekt erscheinen. Klicken und ziehen Sie den Punkt, der der Seite entspricht, die Sie verkleinern oder dehnen möchten.

Hinweis: Die Größe der Modelle kann nicht geändert werden.

Um ein Objekt zu drehen, klicken Sie einfach darauf und anschließend auf die Schaltfläche Drehen auf der Registerkarte Modell. Dadurch wird das ausgewählte Objekt mit sechs Punkten und drei Kreisen umgeben. Klicken Sie auf einen der Punkte und ziehen Sie die Maus, um Ihre Auswahl innerhalb des Kreises zu drehen oder zu neigen, der der Farbe des Punktes entspricht .

Sie finden die entsprechenden Werkzeuge auf der Registerkarte Gelände. Die entsprechenden Werkzeuge finden Sie auf der Registerkarte Gelände. Einzelheiten zu den einzelnen Funktionen dieser Tools finden Sie im Artikel Gelände .

Testen Sie Ihren Platz

Wenn Sie mit Ihrem Platz fertig sind, oder wenn Sie vor der Veröffentlichung sehen wollen, wie etwas funktioniert, klicken Sie auf die Schaltfläche Abspielen auf der Registerkarte Test . Dadurch wird Ihr Avatar in das Erlebnisfenster geladen und Sie können herumlaufen und spielen. Um den Test zu beenden und den Zustand vor dem Klicken auf Play wiederherzustellen, klicken Sie auf die Schaltfläche Stop .

Hinweis: Wenn Sie Ihren Platz testen möchten, indem Sie ihn über die normale Spielfunktion auf der Roblox-Seite spielen, können Sie dies tun, ohne dass andere Spieler Ihrem Erlebnis beitreten, indem Sie Ihr Erlebnis auf Privat setzen. Testen, wie Ihre Erlebnisse mit anderen Spielern funktionieren

Wenn Sie sehen wollen, wie Ihr Platz mit mehreren Teilnehmern aussehen würde, können Sie einen Server emulieren. Gehen Sie dazu auf die Registerkarte Test und wählen Sie im Dropdown-Feld im Bereich Kunden und Servern die Anzahl der Spieler aus, die Sie emulieren möchten. Danach klicken Sie auf Start. Daraufhin werden mehrere andere Studio-Fenster geöffnet. Eines dieser Fenster funktioniert als Server, während die anderen es Ihnen ermöglichen, einen der verschiedenen Fake-Spieler zu steuern, die dem Erlebnis hinzugefügt werden. Um den Test zu beenden, klicken Sie im ursprünglichen Studio-Fenster auf die Schaltfläche Bereinigen.

Hinweis: Wenn Sie Ihren Platz testen möchten, indem Sie ihn mit anderen von Ihnen ausgewählten Entwicklern über die normale Spielfunktion auf der Roblox-Seite spielen, können Sie dies tun, ohne dass unberechtigte Spieler Ihrem Erlebnis beitreten, indem Sie es als Gruppenerlebnis hochladen, den Zugriff für editieren auf vorgegebene Ränge beschränken und Ihr Erlebnis auf privat setzen. Weitere Informationen finden Sie in den Hilfeartikeln Erlebnisse schaffen und öffentlich machen und Gruppenerlebnisse .
