Verbieten Sie das Kopieren Ihrer Erfahrung

Wenn Sie das Kopieren Ihrer Erfahrung verbieten, wird verhindert, dass andere Personen alles, was Sie erstellt haben, direkt kopieren. Wenn Sie jedoch anderen erlauben möchten, Ihr Erlebnis als Vorlage für ihre eigenen Erlebnisse zu kopieren, können Sie das Kopieren zulassen.

Es gibt zwei Methoden, mit denen Sie das Kopieren zulassen oder verbieten können.

Methode eins :

    1. Melden Sie sich an und klicken Sie auf den Tab Erstellen
    2. Klicken Sie auf das Zahnradsymbol rechts neben dem Namen Ihres Erlebnisses. Wählen Sie dann im angezeigten Menü die Option Startpunkt konfigurieren
    3. Klicken Sie auf Berechtigungen
    4. Wenn Sie unter Andere Berechtigungen das Kopieren zulassen möchten, aktivieren Sie das Kontrollkästchen neben Kopieren zulassen . Wenn Sie das Kopieren nicht zulassen möchten, lassen Sie das Kästchen deaktiviert.
      • Hinweis: Wenn Sie dieses Kontrollkästchen aktivieren, gewähren Sie jedem anderen Benutzer von Roblox das Recht, den Inhalt, den Sie gerade teilen, auf verschiedene Arten zu verwenden. Wenn Sie dieses Recht nicht gewähren möchten, aktivieren Sie dieses Kontrollkästchen bitte nicht. Weitere Informationen zum Teilen von Inhalten finden Sie in den Roblox-Nutzungsbedingungen
    5. Klicken Sie auf Speichern

Methode zwei :

    1. Navigieren Sie zur Seite Ihres Erlebnisses
    2. Klicken Sie auf das Symbol mit den drei Punkten, um das Einstellungsmenü des Erlebnisses aufzurufen
    3. Wählen Sie Diesen Ort konfigurieren
    4. Gehen Sie auf dem angezeigten Bildschirm zur Registerkarte Berechtigungen
    5. Wenn Sie unter Andere Berechtigungen das Kopieren zulassen möchten, aktivieren Sie das Kontrollkästchen neben Kopieren zulassen . Wenn Sie das Kopieren nicht zulassen möchten, lassen Sie das Kästchen deaktiviert.
      • Hinweis: Wenn Sie dieses Kontrollkästchen aktivieren, gewähren Sie jedem anderen Benutzer von Roblox das Recht, den Inhalt, den Sie gerade teilen, auf verschiedene Arten zu verwenden. Wenn Sie dieses Recht nicht gewähren möchten, aktivieren Sie dieses Kontrollkästchen bitte nicht. Weitere Informationen zum Teilen von Inhalten finden Sie in den Roblox-Nutzungsbedingungen .)
    6. Klicken Sie auf Speichern
