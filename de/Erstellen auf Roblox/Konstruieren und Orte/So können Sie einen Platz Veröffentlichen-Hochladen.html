So können Sie einen Platz Veröffentlichen/Hochladen

Veröffentlichen ist, wenn  Sie eine Kopie Ihres Ortes auf den Roblox-Servern speichern. Dies bedeutet, dass es Ihren Platz speichert und ihn dann auf die Roblox-Server hochlädt, wo er gespeichert wird, damit andere Ihren Platz leichter genießen können.

Bitte beachten Sie, dass Sie Ihr Erlebnis auf Roblox (und/oder als lokale Kopie) speichern können, ohne Ihr Erlebnis zu veröffentlichen.  Das Speichern auf Roblox wird Ihren Fortschritt gespeichert, ohne dass er öffentlich zugänglich ist, während das Veröffentlichen auf Roblox Ihren Fortschritt speichert und zum Spielen verfügbar macht.

Weitere Informationen zum Veröffentlichen Ihres Ortes auf Roblox finden Sie im Artikel Studio-Übersicht .
