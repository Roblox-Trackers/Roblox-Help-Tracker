So kopieren Sie eine Erfahrung

Manchmal stellen Sie möglicherweise fest, dass Sie ein neues Erlebnis starten möchten, aber ein bereits vorhandenes Erlebnis bearbeiten möchten, um Zeit zu sparen. Vielleicht möchten Sie auch nur eines Ihrer Erlebnisse auf ein anderes Konto kopieren. Die folgenden Schritte helfen Ihnen dabei.

So kopieren Sie Ihre eigene Erfahrung

  1. Führen Sie Roblox Studio aus
  2. Melden Sie sich in Ihrem Konto im Studio an
  3. Klicken Sie auf Meine Erfahrung
  4. Suchen Sie das Erlebnis, das Sie kopieren möchten, klicken Sie darauf und dann unten auf dem Bildschirm auf Bearbeiten
  5. Nachdem Sie das Erlebnis bearbeitet haben, können Sie es auf zwei Arten kopieren:
  • Wenn Sie nur ein Backup auf Ihrem Computer speichern möchten, wählen Sie Datei > Speichern unter
  • Wenn Sie eine Kopie des Erlebnisses auf Roblox hochladen möchten, wählen Sie Datei > Auf Roblox veröffentlichen als und laden Sie ein vorhandenes Erlebnis hoch oder erstellen Sie ein neues

Hinweis: Um das Erlebnis in eines Ihrer anderen Konten zu kopieren, wählen Sie in Schritt 5 die Methode Speichern unte r. Melden Sie sich nach dem Speichern des Erlebnisses von Studio ab und dann mit Ihrem neuen Konto wieder an. Sobald das Erlebnis gespeichert ist, melden Sie sich bei Studio ab und melden sich dann mit Ihrem neuen Konto wieder an. Sie können dann Auf Roblox veröffentlichen verwenden, um auf dieses Konto hochzuladen.

So verwenden Sie die Kopiererlaubnis einer anderen Person als Vorlage

Möglicherweise sehen Sie eine Erfahrung, von der Sie glauben, dass sie als guter Ausgangspunkt für Ihre eigene Erfahrung dienen würde. Die gute Nachricht ist, dass einige Bauherren Sie ermutigen, ihre Erfahrungen für diesen Zweck zu nutzen! Ein Erlebnis kann jedoch nur kopiert werden, wenn der Hinweis „Die Quelle dieses Erlebnisses kann kopiert werden“ auf der Registerkarte Info des Erlebnisses aufgeführt ist. Weitere Informationen finden Sie in unserem Hilfeartikel zum Zulassen oder Verbieten des Kopierens Ihrer Erfahrung.

Wenn das Erlebnis kopiert werden darf, helfen Ihnen die folgenden Schritte, es als Ausgangspunkt für Ihr eigenes Erlebnis zu verwenden:

  1. Rufen Sie die Detailseite des Erlebnisses auf
  2. Klicken Sie auf das Symbol, das rechts oben im Titel wie drei Punkte aussieht
  3. Klicken Sie im angezeigten Menü auf Bearbeiten
  4. Dadurch wird das Erlebnis in Studio geöffnet, wo Sie eines von zwei Dingen tun können:
  • Wählen Sie Date > Speichern unter und speichern Sie das Erlebnis auf Ihrem Computer. Sie können das Erlebnis dann nach Ihren Wünschen bearbeiten.
  • Wählen Sie Datei > Auf Roblox veröffentlichen als und laden Sie dann entweder ein vorhandenes Erlebnis hoch oder erstellen Sie ein neues.

Hinweis: Bevor Sie sich entscheiden, ein kopiertes Erlebnis zu veröffentlichen, fragen Sie zuerst den Ersteller um Erlaubnis. Wenn die Erfahrung des Spielers nicht kopiert werden darf und Sie Ihre eigene Version erstellen möchten, fragen Sie unbedingt zuerst den Ersteller um Erlaubnis. Es ist höflich.
