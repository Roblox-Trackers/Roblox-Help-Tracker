Wie man ein Bündel erstellt

Ein Bündel besteht aus mehreren Objekten, die zu einem Objekt zusammengefasst sind. Bündel können alles von Ziegeln bis zu HopperBins enthalten und die Leichtigkeit, mit der Sie eine Gruppe von Objekten manipulieren können, erheblich verbessern. Darüber hinaus haben Sie durch Erstellen eines Bündels auch die Möglichkeit, es auf Roblox hochzuladen. Diese Funktion ermöglicht jedem anderen Spieler den Zugriff auf das Bündel über die Option Alle Bündel in Studio Werkzeugkasten oder beim Durchsuchen der Bibliothek .

Wie man ein Bündel erstellt

Um ein Bündel zu erstellen, folgen Sie diesen fünf einfachen Schritten.

  1. Öffnen Sie einen Platz in Roblox Studio.
  2. Erstellen Sie die Objekte, die Sie gruppieren möchten, und ordnen Sie sie so an, wie Sie sie sehen möchten.
  3. Gruppieren Sie Ihr Bündel, indem Sie CTRL + G auf Ihrer Tastatur drücken.
  4. Wählen Sie in der Menüleiste Datei und dann Roblox veröffentlichen .
  5. Vervollständigen Sie die erforderlichen Informationen und wählen Sie Fertig stellen .

Wenn diese fünf Schritte ordnungsgemäß befolgt wurden, sollten Sie ein Bündel erfolgreich erstellt haben.
