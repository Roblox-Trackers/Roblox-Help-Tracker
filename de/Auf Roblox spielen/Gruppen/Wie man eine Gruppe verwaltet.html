Wie man eine Gruppe verwaltet

Nachdem Sie eine Gruppe erstellt haben, wird sie im nächsten Schritt verwaltet. Klicken Sie dazu auf den Link Gruppen , der sich im linken Menü der Seite befindet. Von hier aus können Sie die Gruppenaktivität und das Gruppensteuerfeld anzeigen.

Die Gruppen Hauptseite

Dies sehen Sie, sobald Sie Ihre Gruppe auswählen. Diese folgenden Optionen sind verfügbar:

  • Ruf :Wichtige Gruppe-Updates können hier gepostet werden. Der letzte Schrei wird in einer gelben Wortblase unter der Gruppenbeschreibung erscheinen.
  • Mitglieder : Auf dieser Registerkarte können Mitglieder nach Rang angezeigt werden.
  • Verbündete und Feinde : Diese Registerkarten sind nicht sichtbar, wenn deine Gruppe keine Verbündeten oder Feinde hat. Sollte dies der Fall sein, werden auf diesen Registerkarten andere Gruppen angezeigt, die entweder deine Verbündeten oder Feinde sind.
  • Store : Auf dieser Registerkarte können Sie Kleidung zum Verkauf anbieten und Kleider anschauen, die bereits zum Verkauf angeboten werden. Für weitere Informationen klicken Sie hier .
  • Spiele : Wenn Sie oder Ihre Mitglieder Orte / Spiele für die Gruppe erstellt haben, werden sie hier angezeigt. Für weitere Informationen klicken Sie hier .
  • Wall : Dieser Abschnitt ist wie ein fortlaufender Forenthread für die Gruppe. Gruppendiskussionen können hier veröffentlicht werden. Beiträge können von Gruppenmitgliedern mit dieser Berechtigung gelöscht werden. Mitglieder können auch aus der Gruppe innerhalb der Group Admin-Seite ausgeschlossen werden. Sollten unangemessene Kommentare abgegeben werden (siehe Roblox Community Regeln ), befindet sich die Option Missbrauch melden auf der rechten Seite.

Gruppen Steuerelemente

Diese Steuerelemente befinden sich oben rechts auf der Gruppenseite und bieten viele Optionen zum Verwalten Ihrer Gruppe.

  • Gruppen Admin : Auf dieser Seite können Sie viele grundlegende Informationen und Funktionen Ihrer Gruppe anpassen. Diese Seite wird im Folgenden näher beschrieben.
  • Gruppen Werbung : Auf dieser Seite können Sie eine Anzeige für Ihre Gruppe erstellen. Für weitere Informationen klicken Sie hier .
  • Make Primary : Mit dieser Schaltfläche können Sie eine Gruppe zu Ihrer primären Gruppe machen. Klicken Sie hier , um weitere Informationen zu primären Gruppen zu erhalten.
  • Gruppe verlassen :  Mit dieser Schaltfläche können Sie eine Gruppe verlassen, der Sie nicht mehr angehören möchten.
    • Bitte beachten Sie: Wenn Sie eine Gruppe verlassen, die Sie besitzen, können Sie später nicht wieder der Besitzer werden. Wenn Sie das Eigentumsrecht nicht auf einen anderen Spieler übertragen (siehe Einstellungen im Abschnitt Gruppenadministrationsseite weiter unten), bevor Sie eine Gruppe verlassen, die Sie besitzen, wird der besitz auf ein anderes Gruppenmitglied mit Mitgliedschaft übertragen. Wenn es keine anderen Mitglieder mit Mitgliedschaft gibt, gehört die Gruppe niemandem.
  • Audit Log : Hier sehen Sie alles, was Ihre Mitglieder in Bezug auf die Gruppe getan haben. Für weitere Informationen klicken Sie hier .

Die Gruppen-Admin-Seite

Auf dieser Seite finden Sie die Tools, die Sie benötigen, um Ihre Mitglieder und die Gruppe selbst effektiv zu verwalten. Um dorthin zu gelangen, klicken Sie auf die Schaltfläche Gruppen Admin im Feld Controls auf der rechten Seite der Gruppen-Seite. Dort sehen Sie die folgenden Optionen:

  • Mitglieder : Hier können Sie Ihre aktuellen Mitglieder suchen oder durchsuchen. Sie können auch nach Rang filtern. Darüber hinaus können Sie hier Mitgliederranglisten ändern und ausstehende Beitrittsanfragen akzeptieren oder ablehnen. Zuletzt können Sie ein Mitglied aus der Gruppe entfernen, indem Sie auf das Zahnrad neben dem Symbol klicken und "Benutzer entfernen" wählen.
  • Gruppen Info : Hier können Sie das Bild und die Beschreibung Ihrer Gruppe ändern.
  • Einstellungen : Hier können Sie Voraussetzungen für den Beitritt zur Gruppe festlegen. Dies beinhaltet, dass Sie eine manuelle Genehmigung benötigen und den Zugriff auf Mitglieder des Builders Club beschränken müssen. Sie werden auch die Optionen finden, die die Deklaration von Feinden erlauben und die Sichtbarkeit von Gruppenbestände und Spielen festlegen. In diesem Abschnitt können Besitzer auch Eigentumsrechte an ein anderes Mitglied übertragen. Denken Sie daran, dass der Spieler, dem Sie die Besitzrechte übertragen, auch eine Mitgliedschaft im Builders Club haben muss, um die Gruppe zu besitzen.
    • Bitte beachten Sie, dass dieser Prozess nicht rückgängig zu machen ist.
  • Beziehungen :  In diesem Bereich können Sie andere Gruppen zu offiziellen Feinden oder Verbündeten Ihrer Gruppe erklären. Für weitere Informationen klicken Sie bitte hier .
  • Rollen : Hier können Sie neue Ränge hinzufügen sowie den Namen und die Berechtigungen Ihrer bereits vorhandenen ändern. Für weitere Informationen klicken Sie bitte hier .
  • Umsatz : In diesem Bereich können Sie den Robux, den Ihre Gruppe am letzten Tag, in der Woche, im Monat oder im Jahr verdient hat, verfolgen.
  • Auszahlung :  In diesem Bereich können Sie bestimmte Gelder Ihrer Gruppe bestimmten Mitgliedern durch eine einmalige Auszahlung sowie eine wiederkehrende Auszahlung zuweisen.
