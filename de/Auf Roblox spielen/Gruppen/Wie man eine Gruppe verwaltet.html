Wie man eine Gruppe verwaltet

Nachdem Sie eine Gruppe erstellt haben, ist der nächste Schritt die Verwaltung der Gruppe. Klicken Sie dazu auf den Link Gruppen , der sich im linken Menü der Seite befindet. Von hier aus können Sie die Gruppenaktivität und das Gruppensteuerfeld anzeigen.

Die Gruppen Hauptseite

Dies sehen Sie, sobald Sie Ihre Gruppe auswählen. Diese folgenden Optionen sind verfügbar:

  • Ruf : Wichtige Gruppe-Updates können hier gepostet werden
  • Beschreibung: Dieser Abschnitt zeigt, worum es in der Gruppe geht.
  • Erlebnisse : Wenn Sie oder Ihre Mitglieder Orte / Erlebnisse für die Gruppe erstellt haben, werden sie hier angezeigt. Weitere Informationen finden Sie unter Gruppenerlebnisse .
  • Mitglieder : Auf dieser Registerkarte können Mitglieder nach Rang angezeigt werden.
  • Sozial Media Links: In diesem Abschnitt werden Ihre hinzugefügten Social Media Links angezeigt. Für weitere Informationen, siehe Social Media Links zu Gruppenbeschreibungen .
  • Wand : Dieser Abschnitt ist wie ein fortlaufender Forumsbeitrag für die Gruppe. Gruppendiskussionen können hier veröffentlicht werden. Beiträge können von Gruppenmitgliedern mit dieser Berechtigung gelöscht werden. Mitglieder können auch aus der Gruppe innerhalb der Group Admin-Seite ausgeschlossen werden. Sollten unangemessene Kommentare abgegeben werden (siehe Roblox-Gemeinschaftsstandards ), befindet sich die Option Missbrauch melden auf der rechten Seite.

Gruppen Steuerelemente

Diese Steuerelemente befinden sich oben rechts auf der Gruppenseite und bieten viele Optionen zum Verwalten Ihrer Gruppe.

  • Gruppe konfigurieren : Auf dieser Seite können Sie viele grundlegende Informationen und Funktionen Ihrer Gruppe anpassen. Diese Seite wird im Folgenden näher beschrieben.
  • Als Hauptgruppe festlegen : Mit dieser Schaltfläche können Sie eine Gruppe zu Ihrer primären Gruppe machen. Weitere Informationen über Hauptgruppen finden Sie unter Hauptgruppe .
  • Gruppe verlassen :  Mit dieser Schaltfläche können Sie eine Gruppe verlassen, der Sie nicht mehr angehören möchten.
    • Bitte beachten Sie: Wenn Sie eine Gruppe verlassen, die Sie besitzen, können Sie später nicht wieder der Besitzer werden. Wenn Sie eine Gruppe, die Ihnen gehört, nicht auf einen anderen Spieler übertragen (siehe Einstellungen im Abschnitt Gruppenverwaltungsseite unten), bevor Sie sie verlassen, wird die Gruppe niemandem gehören.
  • Gruppen bewerben : Auf dieser Seite können Sie eine Anzeige für Ihre Gruppe erstellen. Weitere Informationen finden Sie unter Wie man für eine Gruppe wirbt .
  • Missbrauch melden : Um den Namen oder die Beschreibung , den Ruf und die Rolle einer Gruppe zu melden, wählen Sie die Schaltfläche Verstoß melden , füllen Sie sie aus und wählen Sie die Schaltfläche Melden .

Die Gruppen-Admin-Seite

Auf dieser Seite finden Sie die Werkzeuge, die Sie für die effektive Verwaltung Ihrer Mitglieder und der Gruppe benötigen. Um dorthin zu gelangen, klicken Sie auf die Schaltfläche Gruppe konfigurieren im Feld Steuerelemente rechts auf der Seite der Gruppe. Dort sehen Sie die folgenden Optionen:

  • Informationen : Hier können Sie den Namen, den Besitzer, das Bild und die Beschreibung Ihrer Gruppe ändern.
  • Einstellungen : Hier können Sie Voraussetzungen für den Beitritt zur Gruppe festlegen. Dazu gehört auch das Erfordernis einer manuellen Genehmigung. Sie werden auch die Optionen finden, die die Deklaration von Feinden erlauben und die Sichtbarkeit von Gruppengeldern und Erlebnissen festlegen.
    • In diesem Abschnitt können Besitzer auch Eigentumsrechte an ein anderes Mitglied übertragen.
      • Bitte beachten Sie, dass dieser Prozess nicht rückgängig zu machen ist.
  • Soziale Medien Links: In diesem Abschnitt können Sie soziale Links zu Ihren Gruppenbeschreibungen hinzufügen. Weitere Informationen finden Sie unter Social Media Links zu Gruppenbeschreibungen.
  • Einnahmen : In diesem Bereich können Sie den Robux, den Ihre Gruppe am letzten Tag, in der Woche, im Monat oder im Jahr verdient hat, verfolgen. Außerdem wurde die Option Auszahlungen im Abschnitt Einnahmen hinzugefügt, um das Geld der Gruppe durch eine einmalige Auszahlung an bestimmte Mitglieder weiterzugeben oder eine wiederkehrende Auszahlung einzurichten.
  • Mitglieder : Hier können Sie Ihre aktuellen Mitglieder suchen oder durchsuchen. Sie können auch nach Rang filtern. Darüber hinaus können Sie hier Mitgliederranglisten ändern und ausstehende Beitrittsanfragen akzeptieren oder ablehnen. Zuletzt können Sie ein Mitglied aus der Gruppe entfernen, indem Sie auf das Zahnrad neben dem Symbol klicken und "Benutzer entfernen" wählen.
  • Rollen : Hier können Sie neue Ränge hinzufügen sowie den Namen und die Berechtigungen Ihrer bereits vorhandenen ändern. Für weitere Informationen siehe Gruppenrollen/Ränge und Berechtigungen .
  • Beziehungen : In diesem Bereich können Sie andere Gruppen zu offiziellen Feinden oder Verbündeten Ihrer Gruppe erklären. Weitere Informationen finden Sie unter So erstellen Sie Verbündete und Feinde für Ihre Gruppe .
  • Prüfungsprotokoll : Hier sehen Sie alles, was Ihre Mitglieder in Bezug auf die Gruppe getan haben. Weitere Informationen finden Sie unter Aktivität Ihrer Gruppe anzeigen .
