Mobile Geräte: Gruppen

Die Roblox Mobile App ermöglicht Spielern, sich über ihre Gruppen auf dem Laufenden zu halten und auf der Gruppenwand zu posten.

Zugreifen auf Gruppen

Gehen Sie folgendermaßen vor, um über die Mobile App auf Gruppen zuzugreifen:

  1. Tippen Sie auf die Schaltfläche Mehr ... am unteren Bildschirmrand
  2. Tippen Sie hier auf Gruppen
  3. Wählen Sie die Gruppe aus, die Sie anzeigen möchten

Hinweis: Sie können auch nach Gruppen suchen, indem Sie in der Suchleiste oben auf dem Bildschirm einen Gruppennamen eingeben.

Die Hauptseite der Gruppe

Dies sehen Sie, sobald Sie Ihre Gruppe ausgewählt haben. Die folgenden Optionen sind verfügbar:

  • Ruf : Wichtige Gruppen-Updates können hier gepostet werden.
  • Beschreibung : Dieser Abschnitt zeigt, worum es in der Gruppe geht.

  • Erlebnisse : Wenn Sie oder Ihre Mitglieder Orte/Erlebnisse für die Gruppe erstellt haben, werden diese hier angezeigt. Für weitere Informationen klicken Sie bitte hier.

  • Mitglieder : Auf dieser Registerkarte können die Mitglieder nach Rang angezeigt werden.

  • Soziale Links: In diesem Abschnitt werden Ihre hinzugefügten sozialen Links angezeigt. Für weitere Informationen, klicken Sie hier .

  • Wand : Dieser Bereich ist wie ein laufendes Forumsthread für die Gruppe. Hier können Gruppendiskussionen gepostet werden. Beiträge können von Gruppenmitgliedern, die diese Berechtigung haben, gelöscht werden. Mitglieder können auch über die Gruppenverwaltungsseite aus der Gruppe ausgeschlossen werden. Sollten unangemessene Kommentare geäußert werden(siehe Roblox-Community-Standards ), befindet sich rechts die Option, Missbrauch zu melden.

Gruppen-Steuerelemente

Diese Steuerelemente befinden sich oben rechts auf der Gruppenseite und bieten viele Optionen zur Verwaltung Ihrer Gruppe.

  • Gruppe konfigurieren : Auf dieser Seite können Sie zahlreiche Anpassungen an den grundlegenden Informationen und Funktionen Ihrer Gruppe vornehmen. Diese Seite wird weiter unten im Detail beschreiben.
  • Primärgruppe machen : Mit dieser Schaltfläche können Sie eine Gruppe zur Primäre Gruppe machen. Weitere Informationen über primäre Gruppen finden Sie hier .
  • Gruppe verlassen : Mit dieser Schaltfläche können Sie eine Gruppe verlassen, der Sie nicht mehr angehören möchten .
    • Bitte beachten Sie: Wenn Sie eine Gruppe verlassen, deren Eigentümer Sie sind, können Sie die Eigentümerschaft nicht wiedererlangen, indem Sie der Gruppe später wieder beitreten.Wenn Sie vor dem Verlassen einer Gruppe, deren Eigentümer Sie sind, die Eigentümerschaft nicht auf einen anderen Spieler übertragen (siehe Einstellungen im Abschnitt Gruppenadministrationsseite unten), wird die Gruppe niemandem gehören.
  • Missbrauch melden : Um den Namen oder die Beschreibung, den Ruf und die Rolle einer Gruppe zu melden, wählen Sie die Schaltfläche Missbrauch melden , fülle diese aus und wählen die Schaltfläche Melden .

Die Gruppenverwaltungsseite

Auf dieser Seite finden Sie dieTools, die Sie für die effektive Verwaltung Ihrer Mitglieder und der Gruppe selbst benötigen. Um dorthin zu gelangen, klicken Sie auf die Schaltfläche Gruppe konfigurieren im Feld Steuerelemente rechts auf der Seite der Gruppe. Dort sehen Sie die folgenden Optionen:

  • Informationen: Hier können Sie den Namen, den Besitzer, das Bild und die Beschreibung Ihrer Gruppe ändern.
  • Einstellungen : Hier können Sie die Voraussetzungen für den Beitritt zur Gruppe festlegen. Dazu gehört auch das Erfordernis einer manuellen Genehmigung. Hier finden Sie auch die Optionen, um die Deklaration von Feinden zu erlauben und die Sichtbarkeit von Gruppengeldern und Spielen festzulegen. In diesem Abschnitt können die Besitzer auch das Eigentum an ein anderes Mitglied übertragen.
    • Bitte beachten Sie, dass dieser Vorgang nicht umkehrbar ist.
  • Soziale Links: In diesem Abschnitt können Sie soziale Links zu Ihren Gruppenbeschreibungen hinzufügen. Für weitere Informationen klicken Sie bitte hier.
  • Einnahmen: In diesem Abschnitt können Sie die Robux verfolgen, die Ihre Gruppe im Laufe des letzten Tages, der letzten Woche, des letzten Monats oder des letzten Jahres verdient hat. Außerdem wurde im Abschnitt Einnahmen die Option Auszahlungen hinzugefügt, mit der Sie das Geld der Gruppe durch eine einmalige Auszahlung an bestimmte Mitglieder weitergeben oder eine wiederkehrende Auszahlung einrichten können.
  • Mitglieder: Hier können Sie Ihre aktuellen Mitglieder suchen oder durchsuchen. Sie haben auch die Möglichkeit, nach Rang zu filtern. Außerdem können Sie hier die Rangfolge der Mitglieder ändern und ausstehende Beitrittsanträge annehmen oder ablehnen. Schließlich können Sie ein Mitglied aus der Gruppe entfernen, indem Sie auf das Zahnrad neben dem Symbol des Mitglieds klicken und "Benutzer kicken" auswählen.
  • Rollen : Hier können Sie neue Ränge hinzufügen sowie den Namen und die Berechtigungen der bereits vorhandenen Ränge ändern. Für weitere Informationen klicken Sie bitte hier .

Hinweis: Sie können sich mit Ihrem Desktop-/Laptop-Computer anmelden, um die Gruppen-Audit-Protokolle zu überprüfen und mit dem Bereich Affiliates Ihrer Gruppe zu wechseln.

Verwenden Sie die Schaltfläche Missbrauch melden , wenn es Pinnwand-Posts gibt, die gegen die Roblox-Community-Standards verstoßen.
