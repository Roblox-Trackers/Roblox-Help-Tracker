Wie man für eine Gruppe wirbt

Sobald Sie eine tolle neue Gruppe erstellt haben, möchten Sie einen Aufruf starten um neue Mitglieder zu gewinnen! Anzeigen sind ein guter Weg, um das zu tun und Werbung für eine Gruppe ähnelt der Werbung für andere Artikel auf Roblox.

Wie man eine Anzeige erstellt

Jedes Gruppenmitglied, dem die Berechtigung zum Werben der Gruppe erteilt wurde (weitere Informationen über Berechtigungen finden Sie unter Gruppenrollen/Ränge und Berechtigungen ), kann eine Anzeige erstellen.

Um zu beginnen, gehen Sie auf die Seite der Gruppe und klicken Sie auf die Schaltfläche Gruppe werben auf der rechten Seite, um die Seite Benutzeranzeige erstellen zu öffnen. Von dort aus:

  1. Wählen Sie die Anzeigegrößen aus, die Sie erstellen möchten. Es gibt derzeit drei Optionen. Jede Option enthält eine Vorlage zum Herunterladen, die Ihnen den Einstieg erleichtert:

    Download 728 x 90 Banner Vorlage

    Download 160 x 600 Wolkenkratzer-Vorlage

    Download 300 x 250 große Rechteckvorlage

  2. Verwenden Sie Bildbearbeitungssoftware, um eine verlockende Anzeige zu erstellen.
  3. Speichern Sie die benutzerdefinierte Anzeige auf Ihrem Computer.
  4. Klicken Sie auf die Seite Benutzeranzeige erstellen auf die Schaltfläche Datei auswählen .
  5. Suchen Sie das Bild, das Sie in Schritt 2 erstellt haben, und wählen Sie es aus.
  6. Geben Sie einen Namen ein und klicken Sie auf die grüne Schaltfläche Hochladen .

Die von dir hochgeladene Anzeige wird von unseren Moderatoren-Team überprüft, um sicherzustellen, dass sie den Roblox Gemeinschaftsstandards entspricht. Bitte erlauben Sie mehrere Stunden für diesen Prozess. Sobald Ihre Anzeige genehmigt wurde, können Sie die Anzeige auf der Seite Anzeige Inventar starten. Weitere Informationen über das Schalten von Anzeigen und die Funktionsweise des Anzeigensystems finden Sie unter Wie man eine Anzeige erstellt .

Hinweis: Sie können mit Gruppengeldern für Ihre Anzeige anzahlen, wenn Ihnen die “Spenden von Gruppengelder” Berechtigung gewährt wurde.
