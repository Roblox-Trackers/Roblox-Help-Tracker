Wie man Gruppen erstellt und beitritt

Gruppen bieten eine Möglichkeit für Spieler und ihre Freunde, eine eigene Mini-Gemeinschaft mit anderen gleichgesinnten Spielern zu bilden. Gruppen können gegen andere Gruppen kämpfen, beim Bauen helfen, sich an Diskussionen beteiligen und vieles mehr.

Voraussetzungen

Die Anzahl der Gruppen, denen ein Spieler beitreten kann, ist auf 100 Gruppen begrenzt. Die Spieler können auch bis zu 100 eigene Gruppen erstellen. Die Gruppenerstellung erfordert eine einmalige Zahlung von 100 Robux pro Gruppe.


Wie man eine Gruppe erstellt

Wenn die Voraussetzungen erfüllt sind, ist das Erstellen einer Gruppe ziemlich einfach.

  1. Melden Sie sich bei Ihrem Konto an
  2. Klicken Sie im Navigationsbereich auf der linken Seite auf Gruppen
  3. Klicken Sie auf die Schaltfläche Erstellen links auf der Seite

Sobald Sie auf diese  Seite gelangt sind, füllen Sie die Abschnitte Name, Beschreibung und Emblem aus. Jeder Abschnitt sollte die Interessen der Gruppe widerspiegeln. Wenn Sie fertig sind, wählen Sie im Feld Einstellungen rechts auf der Seite, die Zugänglichkeit der Gruppe beizutreten. Wählen Sie für die größtmögliche Kontrolle die Option Manuelle Genehmigung aus.

Hinweis: Gruppennamen dürfen nicht länger als 50 Zeichen sein.

Sobald alles so eingestellt wurde, wie Sie es möchten, klicken Sie auf die Schaltfläche ( ) Kaufen , um die Erstellung der Gruppe abzuschließen.

Wie Sie eine Gruppe finden und beitreten können

Eine Gruppe finden:

  1. Klicken Sie auf die Suchleiste oben auf der Seite
  2. Geben Sie ein Schlüsselwort ein (z. B. "Ninjas", wenn Sie nach einer Gruppe über Ninjas suchen)
  3. Schränken Sie die Suche auf Gruppen ein

Von hier aus enthält die Suchseite Informationen wie eine kurze Beschreibung der Gruppe, die beim Identifizieren von Gruppeninteressen hilfreich ist.

Sobald eine Gruppe ausgewählt wurde, wechseln Sie zur Seite dieser Gruppe und klicken Sie auf die Schaltfläche Gruppe beitreten . Wenn für den Beitritt zur Gruppe eine manuelle Genehmigung erforderlich ist, warten Sie, bis die Beitrittsanfrage genehmigt wurde. Abhängig von der Gruppe kann der Besitzer bestimmte Anforderungen haben, wie z. B. das Senden einer persönlichen Nachricht, bevor eine Anfrage zur Teilnahme an der Gruppe genehmigt wird.

Häufig gestellte Fragen

Frage : "Kann ich meine Gruppe löschen?"

Antwort : Derzeit gibt es keine Möglichkeit, eine Gruppe zu löschen. Stellen Sie sicher, dass der Name korrekt ist, bevor Sie die Gruppe erstellen, da es derzeit keine Option gibt, den Titel nach der Erstellung zu bearbeiten.
