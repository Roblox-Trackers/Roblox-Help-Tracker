Gruppenrollen / Ränge und Berechtigungen

Wenn Sie bestimmten Mitgliedern spezielle Privilegien gewähren möchten, dann ist das RoleSet-System die richtige Lösung! Die Standardgruppen-Rollen sind Eigentümer, Administratoren, Mitglieder und Gäste. Benutzerdefinierte Ränge können erstellt werden, indem Sie im Bereich " Rollen " der Seite " Gruppenadministrator " auf die Schaltfläche " Neu erstellen " klicken. Für weitere Informationen zu dieser Seite, klicken bitte Sie hier .

So weisen Sie Rollen Mitgliedern zu

Solange Sie der Gruppenbesitzer sind oder eine Rolle vom Berechtigten der Gruppe mit den richtigen Berechtigungen zugewiesen bekommen haben, können Sie die Rolle eines anderen Benutzers folgendermaßen ändern:

  1. Klicken Sie in der Navigationsleiste links auf dem Bildschirm auf Gruppen
  2. Finde die fragliche Gruppe und klicke darauf
  3. Klicken Sie rechts auf der Seite auf die Schaltfläche Gruppenadministrator
  4. Klicken Sie auf Mitglieder
  5. Suchen Sie einen Benutzer, den Sie ändern möchten
  6. Klicken Sie auf das Dropdown-Feld unter ihrem Bild
  7. Wählen Sie die Rolle aus, die Sie diesem Benutzer zuweisen möchten

Hinweis: Sie können nur die Rollen von Benutzern ändern, die sich derzeit in Rollen befinden, die niedriger eingestuft sind als Ihre. Darüber hinaus können Sie Nutzer nur für Rollen befördern, die niedriger eingestuft sind als Ihre.

Erstellen einer neuen Rolle

Wenn Sie der Gruppenbesitzer sind, können Sie eine brandneue Rolle für Ihre Mitglieder erstellen. Jede neue Rolle kostet 25 Robux, und Namen können nicht für mehr als eine Rolle gleichzeitig verwendet werden. So erstellen Sie eine neue Rolle:

  1. Klicken Sie in der Navigationsleiste links auf dem Bildschirm auf Gruppen
  2. Finde die fragliche Gruppe und klicke darauf
  3. Klicken Sie rechts auf der Seite auf die Schaltfläche Gruppenadministrator
  4. Klicken Sie in der linken Spalte auf Rollen
  5. Klicken Sie auf die grüne Schaltfläche Neu erstellen
  6. Geben Sie den Namen, die Beschreibung und den Rang der Rolle ein
    • Hinweis: Die Ränge dürfen nur die Nummern 1-254 enthalten. Fehler treten am häufigsten auf, wenn zwei oder mehr Ränge dieselbe Rangnummer oder denselben Rangnamen haben. Sonderzeichen können auch dazu führen, dass Gruppenränge nicht korrekt aktualisiert werden.
  7. Klicken Sie auf die grüne Schaltfläche Kaufen

Dadurch wird die neue Rolle mit den Standardberechtigungen erstellt. Informationen zum Bearbeiten dieser Berechtigungen und zum Anzeigen der einzelnen Funktionen finden Sie im folgenden Abschnitt Bearbeiten einer Rolle und deren Berechtigungen.

Hinweis: Für jede Gruppe gibt es eine feste Obergrenze von 40 Rängen. Das beinhaltet, dass der "Gäste-Rank" automatisch für alle Gruppen zugewiesen wird.

Bearbeiten einer Rolle und deren Berechtigungen

Gruppeneigentümer können Rollen und deren Berechtigungen über die Registerkarte " Rollen " der Seite " Gruppen-Administration " bearbeiten. Hier können Sie den Namen, die Beschreibung und das Ranking ändern, indem Sie die entsprechenden Felder bearbeiten. Wenn Sie die Berechtigungen der Rolle ändern möchten, klicken Sie auf die Schaltfläche Berechtigungen . Dies zeigt ein Fenster an, in dem Sie spezielle Privilegien vergeben können. Diese sind:

Beiträge

  • Gruppenwand anzeigen : Zeigt Diskussionen an der Gruppenwand an.
  • Gruppenstatus anzeigen : Siehe Gruppenruf.
  • Post an der Gruppenwand : Nehmen Sie an Gesprächen Ihrer Grupp teil.
  • Postgruppenstatus : Aktualisieren Sie den Gruppenruf.
  • Gruppen-Posteinträge löschen : Löscht Einträge, die andere Mitglieder an der Gruppenwand erstellt haben.

Mitglieder

  • Beitrittsanfragen akzeptieren : Wenn für die Gruppe derzeit eine manuelle Genehmigung erforderlich ist, können Rollen mit dieser Berechtigung alle ausstehenden Anforderungen zum Beitritt zur Gruppe annehmen oder ablehnen.
  • Kick-Lower-Ranked-Mitglieder : Die Möglichkeit, Mitglieder rauszuschmeißen, die Rollen mit niedrigerem Rang zugewiesen wurden.
  • Untergeordnete Rangstufen verwalten : Die Möglichkeit, die Rollen anderer Mitglieder zu ändern, die derzeit einer niedrigeren Rangliste zugeordnet sind. Weitere Informationen finden Sie im Abschnitt "Zuweisen von Rollen zu Mitgliedern" oben in diesem Artikel.
  • Verbündete und Feinde verwalten : Die Fähigkeit, Gruppengegner und Verbündete zu deklarieren oder zu entfernen.

Vermögenswerte

  • Gruppenartikel erstellen : Erlaubt das Erstellen von Gruppen-Bekleidung. Für weitere Informationen klicken Sie hier .
  • Konfigurieren von Gruppenobjekten : Möglichkeit, bereits vorhandene Gruppen-Bekleidung zu bearbeiten. Für weitere Informationen klicken Sie hier .
  • Werben Sie die Gruppe : Möglichkeit, Anzeigen für die Gruppe zu erstellen. Für weitere Informationen klicken Sie hier .
  • Gruppenfonds ausgeben : Die Möglichkeit, die Mittel der Gruppe für bestimmte gruppenbezogene Aktionen zu verwenden.
  • Gruppenspiele erstellen und bearbeiten : Die Möglichkeit, Spiele zu erstellen und zu bearbeiten, die speziell der Gruppe gehören.

Sonstiges

  • Gruppen-Überwachungsprotokoll anzeigen : Ermöglicht die Anzeige von Gruppenaktivitäten. Für weitere Informationen zu dieser Funktion klicken Sie hier.

Löschen von Rollen

Gruppeneigentümer haben die Möglichkeit Rollen zu löschen, außer Eigentümer und Gäste. Das heißt, der Name und die Beschreibung der Besitzer-Rolle können bearbeitet werden. Um eine Rolle zu löschen, klicken Sie auf das rote X , das sich rechts davon befindet. Wenn ihm derzeit jedoch Benutzer zugewiesen sind, ist das X grau und Sie können es nicht löschen.
