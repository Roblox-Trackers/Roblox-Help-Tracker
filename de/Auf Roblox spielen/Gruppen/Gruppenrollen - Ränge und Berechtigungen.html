Gruppenrollen / Ränge und Berechtigungen

Wenn Sie bestimmten Mitgliedern besondere Privilegien gewähren möchten, dann ist das RoleSet-System die richtige Lösung! Die Standardgruppen-Rollen sind Eigentümer, Administrator, Mitglied und Gast. Benutzerdefinierte Ränge können erstellt werden, indem Sie im Bereich Rollen der Seite Gruppe konfigurieren auf die Schaltfläche Rolle erstellen klicken. Weitere Informationen finden Sie unter Wie man eine Gruppe verwaltet .

Wie man Mitgliedern Rollen zuweist

Wenn Sie der Gruppenbesitzer sind oder Ihnen vom Gruppenbesitzer eine Rolle mit den richtigen Berechtigungen zugewiesen wurde, können Sie die Rolle eines anderen Benutzers wie folgt ändern:

  1. Klicken Sie in der Navigationsleiste links auf dem Bildschirm auf Gruppen
  2. Finde die fragliche Gruppe und klicke darauf
  3. Klicken Sie rechts auf der Seite auf die Schaltfläche Gruppe konfigurieren
  4. Klicken Sie auf Mitglieder
  5. Suchen Sie einen Benutzer, den Sie ändern möchten
  6. Klicken Sie auf das Dropdown-Feld unter ihrem Bild
  7. Wählen Sie die Rolle aus, die Sie diesem Benutzer zuweisen möchten

Hinweis: Sie können nur die Rollen von Benutzern ändern, die sich derzeit in Rollen befinden, die niedriger eingestuft sind als Ihre. Darüber hinaus können Sie Nutzer nur für Rollen befördern, die niedriger eingestuft sind als Ihre.

Erstellen einer neuen Rolle

Wenn Sie der Gruppenbesitzer sind, können Sie eine brandneue Rolle für Ihre Mitglieder erstellen. Jede neue Rolle kostet 25 Robux, und Namen können nicht für mehr als eine Rolle gleichzeitig verwendet werden. So erstellen Sie eine neue Rolle:

  1. Klicken Sie in der Navigationsleiste links auf dem Bildschirm auf Gruppen
  2. Finde die fragliche Gruppe und klicke darauf
  3. Klicken Sie rechts auf der Seite auf die Schaltfläche Gruppe konfigurieren
  4. Klicken Sie in der linken Spalte auf Rollen
  5. Klicken Sie auf die grüne Schaltfläche Rolle erstellen
  6. Geben Sie den Namen, die Beschreibung und den Rang der Rolle ein
    • Hinweis: Die Ränge dürfen nur die Nummern 1-254 enthalten. Fehler treten am häufigsten auf, wenn zwei oder mehr Ränge dieselbe Rangnummer oder denselben Rangnamen haben. Sonderzeichen können auch dazu führen, dass Gruppenränge nicht korrekt aktualisiert werden.
  7. Klicken Sie auf die grüne Schaltfläche Kaufen

Dadurch wird die neue Rolle mit den Standardberechtigungen erstellt. Informationen zum Bearbeiten dieser Berechtigungen und zum Anzeigen der einzelnen Funktionen finden Sie im folgenden Abschnitt Bearbeiten einer Rolle und deren Berechtigungen.

Hinweis: Für jede Gruppe gibt es eine feste Obergrenze von 40 Rängen. Das beinhaltet, dass der "Gäste-Rank" automatisch für alle Gruppen zugewiesen wird.

Bearbeiten einer Rolle und deren Berechtigungen

Gruppenbesitzer können Rollen und deren Berechtigungen auf der Registerkarte "Rollen" der Seite " Gruppe konfigurieren " bearbeiten. Hier können Sie den Namen, die Beschreibung und den Rang ändern, indem Sie die entsprechenden Felder bearbeiten. Daraufhin wird ein Fenster angezeigt, in dem Sie besondere Berechtigungen zuweisen können. Diese sind:

Beiträge

  • Gruppenwand anzeigen : Diskussionen an der Gruppenwand anzeigen.

  • Auf Gruppenwand posten : Nehmen Sie an Unterhaltungen an der Gruppenwand teil.

  • Beiträge an der Gruppenwand löschen : Löschen Sie Beiträge, die andere Mitglieder auf der Gruppenpinnwand verfasst haben.

  • Gruppenrufe anzeigen : Sehen Sie den Gruppenstatus.

  • Gruppenrufe posten : Aktualisieren Sie den Gruppenstatus.

Mitglieder

  • Niedrigere Mitgliedsränge verwalten : Die Möglichkeit, die Rollen anderer Mitglieder zu ändern, die derzeit einer Rolle mit niedrigerem Rang zugewiesen sind. Weitere Informationen finden Sie im Abschnitt "Wie man Mitgliedern Rollen zuweist" am Anfang dieses Artikels.

  • Beitrittsanfragen annehmen : Wenn die Gruppe derzeit so eingestellt ist, dass eine manuelle Genehmigung erforderlich ist, können Rollen mit dieser Berechtigung ausstehende Anfragen zum Beitritt zur Gruppe annehmen oder ablehnen.

  • Niederrangige Mitglieder entlassen : Die Möglichkeit, Mitglieder, die einer niedrigeren Rolle zugewiesen sind, rauszuwerfen.

Sonstiges

  • Verbündete und Feinde verwalten : Die Fähigkeit, Gruppenfeinde und Verbündete zu deklarieren oder zu entfernen.
  • Gruppen-Überwachungsprotokoll anzeigen : Ermöglicht die Anzeige der Gruppenaktivitäten. Weitere Informationen zu dieser Funktion finden Sie unter Aktivität Ihrer Gruppe anzeigen (Gruppen Überwachungsprotokoll)

Vermögenswerte

  • Gruppen-Mittel ausgeben : Die Möglichkeit, die Mittel der Gruppe für bestimmte gruppenbezogene Aktionen zu verwenden.
  • Die Gruppe werben : Die Möglichkeit, Anzeigen für die Gruppe zu erstellen. Weitere Informationen finden Sie unter Wie man für eine Gruppe wirbt .
  • Gruppenartikel erstellen : Ermöglicht die Erstellung von Gruppenkleidung. Weitere Informationen finden Sie unter Gruppenbekleidung, Verwaltung der Gewinne und Auszahlungen .
  • Gruppenartikel konfigurieren : Möglichkeit, bereits vorhandene Gruppenkleidung zu bearbeiten. Weitere Informationen finden Sie unter Gruppenbekleidung, Verwaltung der Gewinne und Auszahlungen .
  • Gruppenerlebnisse erstellen und bearbeiten : Die Möglichkeit, Erlebnisse zu erstellen und zu bearbeiten, die der Gruppe gehören.

Löschen von Rollen

Gruppenbesitzer haben die Möglichkeit, andere Rollen als die des Besitzers und des Gastes zu löschen. Das heißt, der Name und die Beschreibung der Besitzer-Rolle können bearbeitet werden. Um eine Rolle zu löschen, klicken Sie auf das rote X , das sich rechts davon befindet. Wenn ihm derzeit jedoch Benutzer zugewiesen sind, ist das X grau und Sie können es nicht löschen.
