So hängen Sie ein Bild an eine Gruppe an

Als Gruppeninhaber können Sie ein Bild (auch als Emblem bezeichnet) an eine Gruppe anhängen, indem Sie die folgenden Schritte ausführen:

    1. Klicken Sie die Schaltfläche Gruppen in der Navigationsleiste auf der linken Seite des Bildschirms
    2. Finden Sie die Gruppe und klicken Sie darauf
    3. Klicken Sie oben rechts auf der Seite auf die Schaltfläche Gruppe konfigurieren
    4. Klicken Sie links auf Informationen
    5. Klicken Sie auf Ein Bild von Ihrem Computer auswählen und suchen Sie das gewünschte Bild aus.
    6. Sobald ein Bild ausgewählt wurde, klicken Sie auf Öffnen
    7. Klicken Sie dann auf die Schaltfläche Speichern

Bilddatei-Einschränkungen:

  • Müssen die Roblox Gemeinschaftsstandards erfüllen.
  • Dateityp sollte jpg, png oder jpeg sein.
  • Die maximale Bildgröße beträgt 256 x 256. Sie können größere Dateien hochladen, dennoch wird  das Bild auf 256 x 256 skaliert.
  • Wenn das Bild nicht quadratisch ist, wird es ausgefüllt, so dass es ein Quadrat wird
