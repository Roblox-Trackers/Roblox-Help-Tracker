So hängen Sie ein Bild an eine Gruppe an

Als Gruppeneigentümer kannst du ein Bild (auch Emblem genannt) an eine Gruppe anhängen, indem du diese Schritte befolgst:

  1. Klicke auf Gruppen in der Navigationsleiste auf der linken Seite des Bildschirms
  2. Finde die betreffende Gruppe und klicke sie an
  3. Klicke auf das Menü mit den drei Punkten rechts auf der Seite und klicke auf Gruppe konfigurieren
  4. Klicke auf Informationen auf der linken Seite
  5. Klicke auf Wähle ein Bild von deinem Computer und wähle das Bild aus, das du verwenden möchtest
  6. Sobald ein Bild ausgewählt ist, klicke auf Öffnen
  7. Klicke dann auf die Schaltfläche Speichern

Bilddatei-Einschränkungen:

  • Muss den Roblox Community Standards entsprechen.
  • Dateityp sollte jpg, png oder jpeg sein.
  • Die maximale Bildgröße beträgt 256 x 256. Du kannst ein größeres Bild hochladen, aber das Bild wird auf 256 x 256 skaliert.
  • Wenn das Bild nicht quadratisch ist, wird es ausgefüllt, so dass es ein Quadrat wird
