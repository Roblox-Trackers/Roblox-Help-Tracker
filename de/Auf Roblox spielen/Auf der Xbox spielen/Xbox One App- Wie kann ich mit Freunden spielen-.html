Xbox One App: Wie kann ich mit Freunden spielen?

Um mit Ihren Roblox-Freunden auf Xbox zu spielen, können Sie Folgendes tun:

  • Wenn beide Benutzer gleichzeitig auf derselben Spielseite auf Spielen drücken, haben sie eine größere Chance, auf demselben Server zu spielen
  • Um sicherzustellen, dass Sie dieselbe Erfahrung machen, muss der Xbox-Benutzer zuerst beitreten, dann kann der Benutzer auf der anderen Plattform Ihr Profil öffnen und auf Erlebnis beitreten klicken
  • Wenn beide Xbox-Benutzer sind und beide auf der Xbox verbunden sind, können sie sich über den Freunde -Bildschirm verbinden, indem sie sie auswählen und ihrer Erfahrung beitreten

Kann ich mit meinem Freunden online von ihrem PC, Mac oder Telefon/Tablet aus spielen?

Ja! Sie können mit Ihren Freunden auf demselben Server spielen, auch wenn sie auf einer anderen Plattform sind. Während Xbox-Spieler den In-Experience-Chat nicht sehen können, können Sie über Xbox Voice Chat mit anderen Spielern kommunizieren.

Sind andere Funktionen der Roblox Website wie Freunde, Chat, Gruppen, usw. verfügbar auf Xbox?

Sie können jederzeit Freundschaftsanfragen erhalten, wenn Sie Roblox auf einer beliebigen Plattform spielen. Sie können Ihre Online-Xbox-Freunde auf der Registerkarte Freunde in Xbox anzeigen und eine Party erstellen, um mit ihnen zu spielen. Um alle Ihre Freunde anzuzeigen oder Freundschaftsanfragen anzunehmen/abzulehnen, können Sie dies tun, indem Sie sich auf einer anderen Plattform wie PC oder Handy bei Ihrem Roblox-Konto anmelden.
