Xbox One App: Wie kann ich mit Freunden spielen?

Um mit Ihren Roblox-Freunden auf Xbox zu spielen, können Sie Folgendes tun:

  • Wenn beide Benutzer gleichzeitig auf derselben Spielseite auf Spielen drücken, haben sie eine größere Chance, auf demselben Server zu spielen
  • Um sicherzustellen, dass Sie dem selben Spiel beitreten, muss der Xbox-Benutzer zuerst beitreten, dann kann der Benutzer auf der anderen Plattform Ihr Profil öffnen und auf Spiel beitreten klicken
  • Wenn beide Xbox-Benutzer sind und beide mit Xbox verbunden sind, können sie sich über den Bildschirm Freunde miteinander verbinden, indem sie sie auswählen und ihrem Spiel beitreten

Kann ich mit meinem Freunden online von ihrem PC, Mac oder Telefon/Tablet aus spielen?

Ja! Sie können mit Ihren Freunden auf demselben Server spielen, auch wenn sie auf einer anderen Plattform sind. Während Xbox-Spieler den Chat im Spiel nicht sehen können, können Sie über den Xbox Sprach-Chat mit anderen Spielern kommunizieren.

Sind andere Funktionen der Roblox Website wie Freunde, Chat, Gruppen, usw. verfügbar auf Xbox?

Sie können immer Freundschaftsanfragen erhalten, wenn Sie Roblox auf einer beliebigen Plattform spielen. Sie können Ihre Xbox Freunde die online sind, im Tab Freunde der Xbox anzeigen und eine Party gründen, um mit ihnen zu spielen. Um alle Ihre Freunde anzuzeigen oder Freundschaftsanfragen zu akzeptieren/abzulehnen, können Sie dies tun, indem Sie sich in Ihrem Roblox-Konto auf einer anderen Plattform wie PC oder Mobile Geräte anmelden.
