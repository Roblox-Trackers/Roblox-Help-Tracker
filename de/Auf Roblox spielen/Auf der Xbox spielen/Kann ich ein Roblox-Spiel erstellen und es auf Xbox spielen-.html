Kann ich ein Roblox-Spiel erstellen und es auf Xbox spielen?

Ja, Sie können Roblox-Spiele mit den Roblox Studio-Werkzeuge auf einem PC oder Mac erstellen und diese dann überall spielen. Sie können nun auch den Konsolenzugriff für Ihr eigenes Spiel und das gesamte Roblox Games Multiversum erlauben.

Wie kann ich mein Spiel auf Xbox verfügbar stellen?

Um Ihr Spiel für die Xbox-Wiedergabe verfügbar zu machen, führen Sie folgende Schritte auf einem Computer aus:

  1. Gehe zur Seite deines Spiels
  2. Klicke oben rechts auf der Spielseite auf die Schaltfläche " Menü ", die aussieht wie 3 horizontal ausgerichtete Punkte
  3. Klicken Sie im angezeigten Menü auf diese Stelle konfigurieren
  4. Klicken Sie auf die Registerkarte Zugriff auf der linken Seite
  5. Stellen Sie unter Spielbare Geräte sicher, dass die Konsole aktiviert ist (sowie jede andere Plattform, auf der Ihr Spiel verfügbar sein soll)
  6. Überprüfen Sie das angezeigte Dialogfeld " Inhalt-Vereinbarung " und klicken Sie auf " Zustimmen ", wenn Ihr Spiel alle angegebenen Anforderungen erfüllt
  7. Scrollen Sie nach unten und klicken Sie auf die blaue Schaltfläche Speichern am unteren Bildschirmrand

Nach Abschluss dieser Schritte sollte dein Spiel verfügbar sein, um auf Xbox zu spielen!

Wie kann ich meine Spiele mit Freunden spielen?

Wenn Sie Ihre ursprünglichen Spiele einfach mit Freunden spielen möchten, können Sie Meine Spiele verwenden, die ganz rechts auf der Registerkarte Spiele zu finden ist. Die Einrichtung von Meine Spiele ist schnell und einfach. Jedes auf der Roblox-Webseite veröffentlichte Spiel erscheint automatisch auf Ihrem Xbox-Konto. Alle online veröffentlichten Updates werden automatisch mit Ihrem My Games-Platz auf der Xbox synchronisiert.

Von dort aus ist es genauso einfach, Ihr Spiel mit Freunden auf der Xbox zu spielen. Der Besitzer des Ortes kann mit seinen Xbox-Freunden eine Xbox-Party erstellen. Danach können Sie direkt mit allen Beteiligten in das Spiel springen. Soziales ist der Kern dessen, um was es bei Roblox geht und wir hoffen, dass jeder seine eigenen Spiele auf Xbox spielen und die Erfahrung mit Freunden teilen kann!

Wie kann ich mein Spiel vorgestellt bekommen?

Wenn Sie der Meinung sind, dass Ihr Spiel auf der Featured -Liste hervorgehoben werden kann, senden Sie eine E-Mail an featuredgames@roblox.com mit einem Link zu unserem Platz.

Ausgewählte Spiele werden auf dem Xbox One-Startbildschirm angezeigt, was die Klickraten erheblich erhöht. Denken Sie daran, dass alle Spiele auf Xbox getestet werden und den Microsoft Xbox-Anforderungen-Standards entsprechen sollen. Stellen Sie daher sicher, dass Ihr Spiel und Ihre GUIs auf das Gamepad vorbereitet sind!

Weitere Informationen zum Erstellen von Roblox-Spielen finden Sie im Developer Hub Seite . Wenn Sie weitere Informationen zur Implementierung von Gamepad-Steuerelementen benötigen, lesen Sie das Gamepad-Tutorial .
