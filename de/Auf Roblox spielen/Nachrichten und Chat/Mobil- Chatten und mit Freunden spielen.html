Mobil: Chatten und mit Freunden spielen

Sie können mit Freunden auf Mobilgeräten chatten und einfach mit Freunden direkt aus dem Chat an Spiele teilnehmen.

Wie kann ich  mit meinen Freunden auf dem Handy chatten?

Der Chat mit deinen Freunden auf Mobilgeräten  ähnelt einem Android- oder iOS-Gerät, es können jedoch einige visuelle Unterschiede bestehen.

Um mit Freunden auf Mobilgeräten zu chatten, folgen Sie, bitte  diesen Anweisungen:

  1. Öffnen Sie die Roblox App
  2. Wählen Sie in der Navigationsleiste am unteren Bildschirmrand das Chat-Symbol aus:
  3. Auf dem Bildschirm wird der Verlauf Ihrer letzten Chats und Freunde angezeigt, mit denen Sie chatten können. Sollte einer von Ihren Freunden derzeit ein Spiel spielen, können Sie die Spiele sehen, wenn die Datenschutzeinstellungen  Ihrer Freunde dies erlauben.
  4. Wählen Sie den Freund oder die Gruppe aus, mit der / dem Sie chatten möchten, und das Chat-Fenster wird geöffnet.

Wie starte ich eine neue Chatgruppe auf meinem Handy?

  1. Wählen Sie in der Navigationsleiste am unteren Bildschirmrand das Chat-Symbol aus
  2. Wählen Sie das Symbol aus, das wie Bleistift und Papier aussieht
  3. Wählen Sie bis zu 5 Freunde aus, um deiner Gruppe beizutreten, indem du durch deine Freundesliste blättern oder deinen Benutzernamen suchst
  4. Oben rechts auf Chat klicken, um den Chat zu starten
  5. Die neue Chatgruppenseite wird geladen

Hinweis : Wenn Ihr Freund Sie zu einer Chat-Gruppe hinzufügt, werden Sie automatisch hinzugefügt. Wenn Sie zu einer Chat-Gruppe hinzugefügt werden, sehen Sie möglicherweise eine ungelesene Nachrichten Benachrichtigung , die angibt, dass ungelesene Nachrichten aus dem neuen Chat vorhanden sind. Wenn Sie nicht im Chat sein möchten, können Sie den Chat jederzeit verlassen. Wenn Sie weiterhin zu Chat-Gruppen hinzugefügt werden, denen Sie nicht sein möchten, empfehlen wir Ihnen, diese zu blockieren. Weitere Informationen, finden Sie in unseren Hilfsartikel Wie kann man die Regelverletzungen melden und Wie kann man einen Benutzer blockieren.

Wie kann ich den Namen einer Chat-Gruppe ändern ?

Um den Namen einer Chat-Gruppe zu ändern, gehen Sie bitte folgendermaßen vor:

  1. Gehen Sie zu der Chat-Gruppe, die Sie bearbeiten möchten

  2. Wählen Sie das Info-Symbol oben rechts auf dem Bildschirm

  3. Klicken Sie auf Nicht festgelegt
  4. Geben Sie einen neuen Namen für die Chat-Gruppe ein
  5. Wählen Sie Speichern aus

Wie kann ich jemanden aus einer von mir erstellten Chat-Gruppe melden oder entfernen?

  1. Gehen Sie zu der Chat-Gruppe, die Sie bearbeiten möchten
  2. Wählen Sie das Info-Symbol oben rechts auf dem Bildschirm
  3. Wählen Sie die Schaltfläche aus, die wie drei Punkte neben dem Benutzer aussieht, den Sie melden und / oder aus der Chat-Gruppe entfernen möchten
  4. Wählen Sie entweder Benutzer melden or Aus Gruppe entfernen

Hinweis : Wenn der Benutzer unhöflich, unangemessen oder in irgendeiner Weise gegen die Nutzungsbedingungen oder Verhaltensregeln von Roblox verstoßen hat, wählen Sie bitte Benutzer melden und anschließend Aus Gruppe entfernen aus.

Wie füge ich einen Freund zu einer zuvor erstellten Chatgruppe hinzu?

  1. Gehen Sie zu der Chat-Gruppe, die Sie bearbeiten möchten
  2. Wählen Sie das Info-Symbol oben rechts auf dem Bildschirm
  3. Wählen Sie Freunde hinzufügen aus
  4. Wählen Sie einen Benutzer aus, der Ihrer Gruppe beitritt, indem Sie in Ihrer Freundesliste blättern oder nach deren Benutzernamen suchen
  5. Klicken Sie auf Hinzufügen , wenn Sie die Benutzer ausgewählt haben, die Sie hinzufügen möchten

Wie verlasse ich eine Chat-Gruppe auf meinem Handy?

  1. Gehen Sie zu der Chat-Gruppe, die Sie verlassen möchten
  2. Wählen Sie das Info-Symbol oben rechts auf dem Bildschirm
  3. Wählen Sie Gruppe verlassen aus
