Chatten und mit Freunden spielen

Außer mit Ihren Roblox-Freunden in Erlebnisse zu sprechen , können Sie auch direkt auf der Roblox Website mit ihnen chatten! Sie können Erlebnisse die Ihre Freunde gerade spielen, direkt aus dem Chat ansehen und ihnen beitreten. Sie können eine Party aus einem Chat erstellen, wenn Sie und Ihre Freunde schnell gleichzeitig einem Erlebniss beitreten möchten.

Web-Chat-Grundlagen

Wenn Sie sich bei Roblox anmelden, sehen Sie eine blaue Chat-Feld in der unteren rechten Ecke des Bildschirms. Klicken Sie auf dieses Feld, um das Chat-Menü anzuzeigen, wie in der Abbildung unten zu sehen. Hier findest du eine Liste deiner Freunde, Chat-Gruppen und Parties, einschließlich der Möglichkeit, mit deinen Freunden zu spielen.

  • Um mit dem Chatten zu beginnen, klicken Sie einfach auf die gewünschte Person / Gruppe / Party in Ihrer Liste. Wenn die Liste besonders lang ist, können Sie diese über das Suchfeld durchsuchen.
  • Nachdem Sie auf die gewünschte Person geklickt haben, erscheint ein kleines Kästchen links von Ihrer Chat-Liste.
  • Geben Sie Ihre Nachricht in das Feld Nachricht senden ein und drücken Sie die Eingabetaste auf Ihrer Tastatur, um sie zu senden.
  • Sie können eine Unterhaltung ein- oder ausblenden, indem Sie auf die blaue Titelleiste klicken. Um alles zu schließen, klicken Sie auf das X -Symbol.

Chat-Gruppe

Sie können eine Chat-Gruppe erstellen, wenn Sie und mehrere Ihrer Freunde miteinander chatten möchten.

  • Klicken Sie dazu oben in der Chat-Liste (Chatfenster rechts) auf die Schaltfläche Chat-Gruppe erstellen . Dies öffnet ein neues Chatfenster (links oben).

  • Klicken Sie auf den Namen der Benutzer, die Sie hinzufügen möchten, und klicken Sie auf die blaue Schaltfläche Erstellen , wenn Sie alle ausgewählt haben.
  • Das Fenster "Neue Chat-Gruppe" wird wie unten in Ihr Chat-Gruppenfenster umgewandelt. Geben Sie einfach das, was Sie sagen möchten, in das Feld Nachricht senden ein und drücken Sie die Eingabetaste auf Ihrer Tastatur, um es zu senden. Viel Spaß beim Chatten mit Ihren Freunden!

Hinweis: Wenn Ihr Freund Sie zu einer Chat-Gruppe hinzufügt, werden Sie automatisch hinzugefügt. Wenn Sie nicht im Chat sein möchten, können Sie den Chat jederzeit verlassen. Wenn Sie weiterhin zu Chat-Gruppen hinzugefügt werden, in denen Sie nicht sein möchten, empfehlen wir Ihnen, diese zu blockieren. Weitere Informationen finden Sie in unseren Hilfeartikeln zum Melden von Regelverletzungen und zum Blockieren eines anderen Benutzers.

Wie ändere ich den Namen einer Chat-Gruppe?

  • Navigieren Sie zu der Chat-Gruppe, die Sie bearbeiten möchten
  • Wählen Sie das Zahnradsymbol in der oberen rechten Ecke des Chatfensters
  • Wählen Sie Chat-Gruppenname

  • Geben Sie einen neuen Namen für die Chat-Gruppe ein


  • Wenn Sie fertig sind, wählen Sie die Option Speichern

Hinweis: Dies funktioniert unabhängig davon, ob Sie die Chat-Gruppe erstellt haben oder nicht.

Wie kann ich jemanden aus einer von mir erstellten Chat-Gruppe melden oder entfernen?

  • Navigieren Sie zu der Chat-Gruppe, die Sie bearbeiten möchten
  • Wählen Sie das Symbol
  • Wählen Sie die Schaltfläche aus, die wie drei Punkte aussieht, neben dem Benutzer den Sie melden und / oder aus der Chat-Gruppe entfernen möchten

  • Wählen Sie entweder Benutzer melden oder Aus Gruppe entfernen .

Hinweis: Wenn der Benutzer unhöflich oder unangemessen oder in irgendeiner Weise gegen die Nutzungsbedingungen oder Verhaltensregeln von Roblox verstoßen hat, wählen Sie bitte Benutzer melden und anschließend Aus Gruppe entfernen aus.

Wie füge ich einen Freund zu einer zuvor erstellten Chat-Gruppe hinzu?

  • Navigieren Sie zu der Chat-Gruppe die Sie bearbeiten möchten
  • Wählen Sie das Symbol
  • Wählen Sie Freunde hinzufügen

  • Wählen Sie einen Freund aus, der Ihrer Gruppe beitreten soll, indem Sie durch Ihre Freundesliste scrollen oder deren Benutzernamen suchen

  • Klicken Sie auf Hinzufügen , wenn Sie die Freunde ausgewählt haben, die Sie hinzufügen möchten

Wie verlasse ich eine Chat-Gruppe?

  • Navigieren Sie zu der Chat-Gruppe, die Sie verlassen möchten
  • Wählen Sie das Symbol
  • Wählen Sie die Option Gruppe verlassen

Mit deinen Freunden spielen

Während du mit deinen Freunden chattest, kannst du sehen, welche Erlebnisse deine Freunde gerade spielen, wenn ihre Datenschutzeinstellungen dies zulassen. Wenn Sie durch Ihre Chatliste scrollen, können Sie sehen, welche Erlebnisse gerade gespielt werden, und sich Ihren Freunden in den Erlebnisse  anschließen, auf die Sie Zugriff haben.

Wie kann ich sehen, welche Erlebnisse meine Freunde spielen?

Um auf die Schnelle zu sehen, welche Erlebnisse Ihre Freunde spielen, können Sie durch Ihre Liste von Chats / Chat-Gruppen scrollen, um auf einen Blick zu sehen welche Erlebnisse gespielt werden.

Wie kann ich einem Erlebnis mit Freunden in einer Chat-Gruppe beitreten

In einer Chat-Gruppe können Sie sehen, welche Erlebnisse von allen in der Gruppe gespielt werden und Sie können auswählen, welchem ​​Erlebnis Sie beitreten möchten. Wenn ein Erlebnis Symbol neben einer Chat-Gruppe einen Schlagschatten hat, bedeutet dies, dass im Chat mehrere aktive Erlebnisse vorhanden sind. Wenn das Chat-Fenster angezeigt wird, können Sie auswählen, welches Erlebnis Sie spielen möchten.

Hinweis: Wenn Sie ein bestimmtes Erlebis nicht spielen können (z. B. bezahlter Zugang, nicht auf Ihrer Plattform verfügbar), wird für dieses Erlebnis keine Beitrittsschaltfläche angezeigt.

Wie kann ich ein Erlebnis in einer Chat-Gruppe speichern, um jederzeit zu spielen?

Sie können jedes Erlebnis an die Spitze einer Chat-Gruppe setzen, so dass jeder in der Chat-Gruppe es jederzeit spielen kann, auch wenn niemand aus der Gruppe zu einem bestimmten Zeitpunkt im Erlebnis war.

Wie kann ich meinen Freunden im Erlebnis beitreten, oder meine Freunde mir?

Jeder in einem Einzelchat oder einer Chat-Gruppe kann eine Spielsitzung mit Freunden beginnen, indem er einem Erlebnis beitritt. Dadurch fügen Sie Ihrem Chat eine Beitreten-Schaltfläche für das Erlebnis hinzu, sodass jeder im Chat jederzeit teilnehmen kann. Sie können einstellen, ob die Spielinformationen angezeigt werden, indem Sie in der blauen Chatleiste auf das Symbol des Erlebnisses klicken.

Eine Gruppe verlassen

Wenn Sie Teil einer Gruppe / Party sind und Sie diese verlassen möchten, ist dies problemlos möglich.

Alles was Sie tun müssen ist auf das Symbol oben in der Konversationsbox, gefolgt von Verlassen der Gruppe zu klicken.

So deaktivieren Sie den Chat

Um die oben genannten Methoden des Chats zusammen mit der Möglichkeit auf Gruppen-Wänden zu posten, zu deaktivieren, müssen Sie Folgendes ausführen:

    1. Anmelden
    2. Gehen Sie zur Seite mit den Kontoeinstellungen , indem Sie auf das Zahnradsymbol in der oberen rechten Ecke der Seite und anschließend auf Einstellungen klicken. Alternativ können Sie hier klicken .
    3. Klicken Sie auf den Tab Datenschutz und wählen Sie unter "Wer kann mit mir in der App chatten?" Die Option Keiner
    4. S crollen Sie zum unteren Bildschirmrand und klicken Sie auf Speichern

Hinweis: Wenn Sie den Chat für "In-App" deaktivieren, wird die Option Mannschaft Erstellen in Roblox Studio deaktiviert.
