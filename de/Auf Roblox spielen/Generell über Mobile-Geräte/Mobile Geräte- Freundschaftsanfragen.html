Mobile Geräte: Freundschaftsanfragen

Die Möglichkeit, Freundschaftsanfragen zu akzeptieren und zu senden, ist auch in der Roblox Mobile App verfügbar.

Freundschaftsanfragen akzeptieren oder ablehnen:

Handys:

  1. Tippen Sie auf die Schaltfläche Mehr am unteren Bildschirmrand
  2. Tippen Sie auf Freunde (die aktuelle Gesamtzahl der Anfragen wird auf der rechten Seite angezeigt)
  3. Von hier aus, tippen Sie auf Anfragen
  4. Tippen Sie auf Akzeptieren oder Ablehnen

Tablets:

  1. Tippen Sie auf die Schaltfläche Freunde am unteren Bildschirmrand
  2. Tippen Sie auf die Registerkarte Anfragen
  3. Tippen Sie entweder auf die Schaltfläche Annehmen oder Ignorieren , die der Anfrage entspricht, die Sie sehen möchten

Freundschaftsanfragen senden:

Handys:

  1. Tippen Sie auf die Schaltfläche Mehr am unteren Bildschirmrand
  2. Tippen Sie auf Freunde
  3. Tippen Sie das Suchleiste-Symbol ganz oben
  4. Tippen Sie den Benutzernamen Ihres Freundes ein und suchen Sie
  5. Klicken Sie rechts neben dem Benutzernamen auf die Schaltfläche Hinzufügen

Hinweis: Wenn Sie bereits eine Freundschaftsanfrage gesendet haben, können Sie keine weitere Anfrage senden und deren Profil wird "Ausstehend" anzeigen oder es wird das Symbol "Wird geladen" angezeigt.

Tablets:

  1. Tippen Sie auf die Home -Taste am unteren Bildschirmrand
  2. Tippen Sie auf das Symbol am oberen Bildschirmrand
  3. Geben Sie den Namen des Spielers ein, mit dem Sie sich anfreunden möchten
  4. Tippe auf den Spielernamen in der Liste
  5. Tippen Sie auf die Schaltfläche Freundschaftsanfrage senden

Hinweis: Wenn Sie bereits eine Freundschaftsanfrage gesendet haben, können Sie keine weitere Anfrage senden und deren Profil wird "Freundschaftsanfrage ausstehend" anzeigen.
