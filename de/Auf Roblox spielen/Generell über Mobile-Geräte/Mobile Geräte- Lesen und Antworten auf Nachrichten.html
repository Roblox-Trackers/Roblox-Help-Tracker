Mobile Geräte: Lesen und Antworten auf Nachrichten

Ihre Roblox-Nachrichten sind in der Roblox Mobile App verfügbar und können mit der Navigationsleiste am unteren Bildschirmrand aufgerufen werden.

Um persönliche Nachrichten zu lesen

  1. Tippen Sie auf das Symbol " ... " in der unteren rechten Ecke des Bildschirms
  2. Wählen Sie Nachrichten
  3. Blättern Sie durch Ihre Nachrichten. Wenn Sie unten angekommen sind, werden automatisch weitere Nachrichten geladen.
  4. Wählen Sie die Nachricht, die Sie lesen möchten.

Antworten auf persönliche Nachrichten

  1. Tippen Sie auf das Symbol " ... " in der unteren rechten Ecke des Bildschirms
  2. Wählen Sie Nachrichten
  3. Blättern Sie durch Ihre Nachrichten. Wenn Sie unten angekommen sind, werden automatisch weitere Nachrichten geladen.
  4. Wählen Sie aus, auf welche Nachricht Sie eine Antwort senden möchten.
  5. Tippen Sie auf Antworten , geben Sie Ihre Nachricht ein und tippen Sie anschließend auf Senden .

Hinweis: Persönliche Nachrichten können gemeldet werden. Tippen Sie auf Diese Nachricht melden , um einen Bericht an unsere Moderatoren zu senden.
