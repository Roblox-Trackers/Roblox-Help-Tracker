Mobile Geräte: Gruppen

Die Roblox Mobile App ermöglicht Spielern, sich über ihre Gruppen auf dem Laufenden zu halten und auf der Gruppenwand zu posten.

Zugreifen auf Gruppen

Gehen Sie folgendermaßen vor, um über die Mobile App auf Gruppen zuzugreifen:

  1. Tippen Sie auf die Schaltfläche Mehr ... am unteren Bildschirmrand
  2. Tippen Sie hier auf Gruppen
  3. Wählen Sie die Gruppe aus, die Sie anzeigen möchten

Hinweis: Sie können auch nach Gruppen suchen, indem Sie in der Suchleiste oben auf dem Bildschirm einen Gruppennamen eingeben.

Interaktion mit Gruppen

Tablet

Wenn Sie ein Tablet verwenden, können Sie mit Gruppen genauso interagieren wie auf einem Desktop/Laptop. Weitere Informationen zu Gruppen- und Gruppenfunktionen auf Tablets und Computern finden Sie hier .

Handy

Wenn Sie die Gruppenseite auf einem Telefon anzeigen, können Sie die Beschreibung, die Mitglieder, die Pinnwand, den Shop und die Partner anzeigen und folgendes ausführen (abhängig von den Gruppenberechtigungen):

  • An einer Wand postieren - Tippen Sie dazu auf das Feld "Sagen Sie etwas ...", geben Sie Ihre Nachricht ein und klicken Sie auf den Post .
  • Gruppe verlassen - Tippen Sie dazu auf das Symbol "..." in der oberen rechten Ecke des Bildschirms. Lesen Sie die Nachricht im Pop-up-Fenster und tippen Sie auf Ja , wenn Sie sicher gehen möchten.
  • Primär machen - Tippen Sie dazu auf das Symbol "..." in der oberen rechten Ecke des Bildschirms. Tippen Sie im Dropdown-Menü auf die Schaltfläche "Primär machen".

Hinweis: Verwenden Sie die Schaltfläche Missbrauch melden , wenn Pinnwände vorhanden sind, die gegen die Roblox-Gemeinschaftsregeln verstoßen.
