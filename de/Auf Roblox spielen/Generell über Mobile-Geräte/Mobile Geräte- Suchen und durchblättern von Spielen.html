Mobile Geräte: Suchen und durchblättern von Spielen

Sie können auf Ihrem Computer nach Spielen auf Roblox suchen. Wenn Sie jedoch ein Tablet oder ein Handy verwenden, können Sie dies auch mit der Mobile App-Version von Roblox tun.

So finden Sie Spiele

Es gibt ein paar Möglichkeiten, wie man auf Roblox nach Spielen suchen kann.

Kategorien durchsuchen

Tippen Sie dazu auf das Spiel-Symbol in der Navigationsleiste am unteren Bildschirmrand. Von dort aus können Sie eine der Standardlisten durchsuchen: Beliebt, Top-Verdienst, Top-Bewertet, Beliebt in Ihrer Umgebung und Empfohlen.

Hinweis: Sie können auch die Dropdown-Felder oben auf dem Bildschirm verwenden, um die Spiele auf eine bestimmte Kategorie zu beschränken oder eine andere Liste auszuwählen, z. B. Ihre Favoriten, Freundschaftsaktivitäten usw.

Verwenden Sie die Suchfunktion

Sie können auch nach Titeln oder Schlüsselwörtern suchen. Befolgen Sie einfach diese Schritte:

  1. Gehen Sie zum Abschnitt Spiele
  2. Tippen Sie auf das Symbol am oberen Bildschirmrand.
  3. Geben Sie den Titel eines Spiels oder ein Stichwort in die Suchleiste ein.
  4. Tippen Sie auf Zurück oder Suchen auf Ihrer Tastatur. Dies kann wie ein Pfeil nach links oder eine Lupe aussehen.

Sobald Sie ein Spiel gefunden haben, das Sie spielen möchten, tippen Sie auf das Symbol. Dies lädt dessen Seite. Hier können Sie über das Spiel lesen, siehe Bestenlisten usw. Um das Spiel zu spielen, tippen Sie auf die Spielen- Schaltfläche.
