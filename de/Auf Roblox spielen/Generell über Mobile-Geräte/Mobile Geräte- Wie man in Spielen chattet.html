Mobile Geräte: Wie man in Spielen chattet

Während Sie mit Ihren Freunden auf Roblox von Ihrem Computer aus chatten können, haben Sie auch die Möglichkeit, im Spiel auf der Roblox Mobile App zu chatten.

Wie man chattet

Wenn Sie im Spiel chatten möchten, tun Sie Folgendes:

  1. Wählen Sie ein Spiel aus. Für Informationen wie Sie Spiele suchen können, bitte klicken Sie hier.
  2. Um eigene Nachrichten zu senden, tippen Sie auf das Symbol oben links auf dem Bildschirm.
  3. Geben Sie Ihre Nachricht über die Pop-up-Tastatur ein und tippen Sie dann auf die Eingabetaste . Dies kann wie ein Pfeil aussehen, der ganz rechts auf der Tastatur nach links zeigt.

Hinweis: Sie können die Chat-Box auch nach Belieben öffnen und schließen, indem Sie auf das Symbol in der oberen linken Ecke tippen.

Wie deaktiviere ich den Chat im Spiel

Führen Sie folgende Schritte aus, um die Chat-Fähigkeit im Spiel zu deaktivieren:

  1. Melden Sie sich in dem Konto an
  2. Klicken Sie auf das Symbol Mehr ... in der unteren rechten Ecke des Bildschirms
  3. Einstellungen auswählen
  4. Wählen Sie im Pulldown-Menü Datenschutz aus
  5. Wählen Sie unter "Wer kann mit mir chatten?" Die Option Niemand aus

Hinweis: Bitte beachten Sie, dass dieser Chat dem Datenschutzmodus/-filter des Kontos unterliegt. Weitere Informationen zu diesen Filtern finden Sie hier .
