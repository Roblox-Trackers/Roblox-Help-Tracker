Mobile Geräte: Suchen und durchblättern von Erlebnissen

Sie können zwar über Ihren Computer nach Erlebnissen auf Roblox suchen, aber wenn Sie ein Tablet oder ein Telefon benutzen, können Sie dies auch über die mobile App-Version von Roblox tun.

So finden Sie Erlebnisse

Es gibt ein paar Möglichkeiten, wie man auf Roblox nach Erlebnissen suchen kann.

Kategorien durchsuchen

Tippen Sie dazu auf das Erlebnis -Symbol in der Navigationsleiste am unteren Bildschirmrand. Von dort aus können Sie eine der Standardlisten durchsuchen: Beliebt, Top-Verdienst, Top-Bewertet, Beliebt in Ihrer Umgebung und Empfohlen.

Hinweis: Sie können auch die Dropdown-Felder oben auf dem Bildschirm verwenden, um die Erlebnisse auf eine bestimmte Kategorie zu beschränken oder eine andere Liste auszuwählen, z. B. Ihre Favoriten, Freundschaftsaktivitäten usw.

Verwenden Sie die Suchfunktion

Sie können auch nach Titeln oder Schlüsselwörtern suchen. Befolgen Sie einfach diese Schritte:

  1. Gehen Sie zum Abschnitt Erlebnisse
  2. Tippen Sie auf das Symbol am oberen Bildschirmrand.
  3. Geben Sie den Titel eines Erlebnisses oder ein Stichwort in die Suchleiste ein.
  4. Tippen Sie auf Zurück oder Suchen auf Ihrer Tastatur. Dies kann wie ein Pfeil nach links oder eine Lupe aussehen.

Sobald Sie ein Erlebnis gefunden haben, das Sie spielen möchten, tippen Sie auf das Symbol. Dies lädt dessen Seite. Hier können Sie über das Erlebnis lesen, siehe Bestenlisten usw. Um das Erlebnis zu spielen, tippen Sie auf die Schaltfläche Abspielen .
