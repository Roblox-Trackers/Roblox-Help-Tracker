Roblox VR

Erweitere dein Roblox Erlebnis mit virtueller Realität! Roblox unterstützt Oculus Rift und HTC Vive.

So aktivieren Sie VR

Um Ihre VR Erfahrung mit Roblox zu ermöglichen, müssen Sie zuerst sicherstellen, dass Sie Roblox installiert haben (http://www.roblox.com/download) und dann die folgenden Schritte ausführen:

    1. Öffnen Sie die Oculus App auf Ihrem Computer und wählen Sie das Zahnradsymbol oben rechts aus
    2. Wählen Sie Ausrüstung Einstellungen
    3. Wählen Sie im linken Menü Allgemein
    4. Schalten Sie die Einstellung "Unbekannte Quellen" so um , dass anstelle von X ein Häkchen angezeigt wird
    5. Bestätigen Sie Ihre Auswahl, um diese Einstellung zu aktivieren
    6. Wählen Sie ein Spiel auf Roblox und drücken Sie Spielen : http://www.roblox.com/games
    7. Sie werden einen quadratischen Bildschirm auf Ihrem Monitor sehen, der den VR-Modus anzeigt (Beispiel: Bild unten)
    8. Setzen Sie Ihr Headset auf und spielen Sie!

So verwenden Sie das VR Radialmenü

Alle wichtigen Systemeinstellungen können über das VR Radialmenü zugegriffen werden.

    1. Tippen Sie auf die Schaltfläche Menü, um das VR-Radialmenü aufzurufen
    2. Wählen Sie eine der folgenden Optionen:
      • Systemmenü - Zugriff auf Systemoptionen
      • Spielerliste - Zeigt die Spielerliste des Spielservers an
      • Kamera positionieren - Richten Sie die Kamera auf Ihren Charakter aus
      • Benachrichtigungen - Roblox-Benachrichtigungen anzeigen
      • Spiel beenden - Beenden Sie das Spiel
      • Rucksack - Sehen Sie sich Ihr im Spiel-Inventar an
      • 2D UI Toggle - 2D-UI-Funktion ein-/ausschalten, sollte für ein optimales Roblox Spiel eingeschaltet bleiben

Die oben beschriebenen VR Radialmenü Optionen entsprechen den Schaltflächen in der folgenden Abbildung, beginnend mit "Systemmenü" oben und weiter gegen den Uhrzeigersinn.

So schalten Sie den VR-Modus um

Sie können den VR-Modus im Spiel ein- oder ausschalten, indem Sie den Bildschirm zwischen Ihrem VR-Headset und Ihrem Computermonitor umschalten.

Wenn der VR-Modus Eingeschaltet ist, können Sie ihn wie folgt Ausschalten :

    1. Tippen Sie auf die Schaltfläche Menü , um das VR-Radialmenü aufzurufen
    2. Wählen Sie die Option Systemmen ü
    3. Schalten Sie die Option VR Aus

Wenn der VR-Modus Ausgeschaltet ist, können Sie ihn wie folgt Einschalten :

    1. Rufen Sie das Systemmenü auf, indem Sie ESC auf Ihrer Tastatur drücken oder indem Sie auf die Schaltfläche "Hamburgermenü" in der oberen linken Ecke des Bildschirms klicken
    2. Schalten Sie die Option VR Ein
