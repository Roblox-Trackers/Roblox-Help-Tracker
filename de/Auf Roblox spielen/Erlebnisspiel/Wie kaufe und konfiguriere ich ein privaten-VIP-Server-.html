Wie kaufe und konfiguriere ich ein privaten/VIP-Server?

Wenn der Ersteller des Erlebnisses private Server für sein Erlebnis aktiviert hat, kannst du Robux verwenden, um einen privaten Server zu kaufen, den du mit Freunden, die du einlädst, nutzen kannst.

Wie erstelle und ändere ich meinen Server?

Wie erstelle ich einen?

  • Klicke auf die Registerkarte Server auf der Detailseite des Erlebnisses.
  • Wenn diese Funktion aktiviert wurde, wird ein Abschnitt mit dem Titel Private Server angezeigt. Hier werden alle Server angezeigt, die du erstellt hast oder an denen du beteiligt bist. Wenn du welche siehst, kannst du sofort loslegen und spielen!
  • Um einen neuen Server zu erstellen, klicke auf die Schaltfläche Privaten Server erstellen .
  • Gib deinem neuen Server einen Namen. Lies den Abonnementvertrag. Wenn du alle Informationen auf der Box durchgelesen und den perfekten Namen ausgewählt hast, kannst du den Kauf abschließen, indem du auf Jetzt kaufen klickst.
  • Nach dem Kauf kannst du den Server sofort oder später konfigurieren.

Wie konfiguriere ich meinen VIP-Server?

  • Vom Fenster der Kaufbenachrichtigung aus oder indem du Konfigurieren aus dem Menü (sieht aus wie drei Punkte) für den Server auf der Registerkarte Server wählst, gelangst du zur Konfigurationsseite.
  • Du kannst den Server umbenennen, festlegen, wer auf ihm spielen darf, einschließlich bis zu 50 einzelner Benutzer oder deiner Freundesliste, und sogar einen Link zu deinem Server erstellen.
  • Du kannst auch den Zugriff auf den Server auch ein- und ausschalten und dein Abonnement für den Server kündigen.

Wie funktionieren Private Server Abonnements und wie viel kosten sie?

Wie funktionieren die Abonnements für private Server?

  • Die Preise für private Server werden vom Entwickler des Erlebnisses festgelegt.
  • Dies wird bei der ersten Bestellung in Rechnung gestellt und wird jeden Monat erneut in Rechnung gestellt, damit es aktiv bleibt. Die monatliche Verlängerungsgebühr wird automatisch von deinem Robux-Guthaben abgezogen, so dass du dir keine Sorgen machen musst, dass du jeden Monat die Verlängerung vergisst.
  • Wenn der Server nicht verlängert werden soll, kann dies jederzeit vor dem Erneuerungsdatum erfolgen. Um deinen VIP-Server zu kündigen, lies den Abschnitt Wie kündige ich meinen VIP-Server unten.

Wie storniere ich meinen Server?

  • Navigiere zu deinem privaten Server für das Erlebnis. Klicke auf die Schaltfläche Menü (sieht aus wie drei Punkte) ganz rechts im Abschnitt.
  • Klicke auf Konfiguration .
  • Um deine monatliche Gebühr zu stornieren, klicke auf das grüne Symbol Aktiv neben Abonnementstatus .
  • Klicke auf Zahlungen stornieren im Bestätigungsfenster, um das Abonnement zu stornieren.

Hinweis: Du kannst deinen Server noch für den Rest des Monats nutzen, für den du bereits bezahlt hast, aber er läuft am Ende dieser Zeit aus.

Was passiert, wenn ich am Verlängerungsdatum nicht genug Robux habe?

In diesem Fall läuft der Server ab und kann nicht mehr verwendet werden. Er kann jedoch jederzeit wieder erworben werden.

Wie stelle ich meinen Server auf inaktiv?

  • Navigiere zu deinem privaten Server für das Erlebnis. Klicke auf die Schaltfläche Menü (sieht aus wie drei Punkte) ganz rechts im Abschnitt.
  • Klicke auf Konfigurieren .
  • Um den Server vorübergehend inaktiv zu machen, klicke auf das grüne Symbol Aktiv neben VIP Server Status .
  • Klicke auf Deaktivieren im Bestätigungsfenster, um den Server zu deaktivieren.
  • Du kannst es jederzeit wieder anklicken, um es zu reaktivieren. Es ist jedoch wichtig, daran zu denken, dass deine Verlängerungszeit nicht für diesen Zeitraum der Inaktivität verlängert wird und trotzdem zum ursprünglich geplanten Datum verlängert wird oder abläuft.
