Computer Hardware- & Betriebssystemanforderungen

Die Mindestsystemanforderungen für Roblox sind unten aufgeführt. Wenn beim Spielen Probleme auftreten, musst du möglicherweise deinen Computer aufwerten. Lies diesen Hilfsartikel für Systemanforderungen für mobile Geräte.

Betriebssystemanforderungen

  • PC/Windows: Roblox kann auf 64-Bit-Versionen von Windows 7, Windows 8.1 oder Windows 10 installiert werden. Für Windows 8.1 musst du Roblox im Desktop-Modus ausführen, da der Metro-Modus (der gekachelte Startbildschirm) derzeit nicht unterstützt wird.
    Hinweis: Die Roblox-Website unterstützt die folgenden Browser für PC/Windows: Chrome , Firefox , und Microsoft Edge .
  • Mac: Der Roblox-Client kann unter 10.10 (Yosemite) und höher installiert werden, während das Roblox Studio unter Mac OS 10.13 (High Sierra) und höher installiert werden kann.
    Hinweis: Die Roblox-Website unterstützt die folgenden Browser für Mac: Chrome , Firefox , und Safari .
  • Linux: Roblox wird auf Linux nicht unterstützt.
  • Mobil: Klicke hier für Systemanforderungen für Roblox auf Mobilgeräten.
  • Chrome OS: Chrome OS Version 53 oder höher ist erforderlich und du musst Google Play Store zuerst auf deinem Gerät aktivieren.

Hinweis: Die Roblox-Videoaufzeichnungsfunktion funktioniert nur unter Windows 7, Windows 8.1 und Windows 10. Neu veröffentlichte Betriebssysteme oder vorab veröffentlichte Versionen von Betriebssystemen werden möglicherweise nicht sofort unterstützt.

System-Hardwareanforderungen

  • Grafikkarte: Auf PC/Windows benötigt die Roblox-App DirectX 10 oder eine höhere Funktionsstufe. Für die beste Leistung empfehlen wir entweder einen Computer unter 5 Jahren mit einer entsprechenden Grafikkarte oder einen Laptop unter 3 Jahren mit einer integrierten Grafikkarte.
  • Prozessor: Roblox empfiehlt, dass du einen aktuellen Prozessor (2005+) mit einer Taktrate von 1,6 Ghz oder besser hast. Es gab einige Probleme mit älteren AMD-Prozessoren.
  • RAM oder Speicher: Roblox empfiehlt, dass du auf Windows 7, Windows 8.1 oder Windows 10 mindestens 1 GB Arbeitsspeicher hast.
  • Speicherplatz: Roblox empfiehlt, dass du mindestens 20 MB Systemspeicher für die Installation von Roblox zur Verfügung hast.
  • Mobilgeräte: Klicke hier für Systemanforderungen für Roblox auf Mobilgeräten.

Internetanforderungen

Für einen Einzelhaushalt empfiehlt Roblox eine Internetverbindung von mindestens 4-8 Mb/s. In größeren Haushalten ist eine größere Bandbreite empfehlenswert.
