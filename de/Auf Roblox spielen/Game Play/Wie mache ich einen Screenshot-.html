Wie mache ich einen Screenshot?

Die folgenden Schritte führen Sie durch die Aufnahme eines Bildes von dem, was auf Ihrem Bildschirm geschieht.

Hinweis: Wenn Sie ein Bild an den Kundendienst senden, speichern Sie die Datei im .png- oder .jpg-Format und hängen Sie sie an Ihre E-Mail an.

So machen Sie einen Screenshot von Ihrem Bildschirm

  • Klicken Sie auf die Schaltfläche Menü in der oberen linken Ecke
  • Klicken Sie in dem sich öffnenden Menü auf die Registerkarte Aufnahme
  • Klicken Sie auf die Schaltfläche Screenshot machen

Alternativ können Sie die ersten beiden Schritte überspringen und die Taste PrntScn (PrintScreen) auf Ihrer Tastatur drücken.

Wenn Sie dies zum ersten Mal tun, erscheint ein Popup-Fenster, das Sie darüber informiert, dass Sie einen Screenshot gemacht haben und wo Sie ihn finden können. Unten rechts finden Sie die Option, dass diese Meldung nie wieder angezeigt werden soll.

So machen Sie einen Screenshot Ihres gesamten Bildschirms/der Website

Für Windows

  1. Richten Sie Ihre Aufnahme ein und drücken Sie diese Tasten: PrntScn (Bildschirm drucken). Diese Taste befindet sich normalerweise ganz oben auf der Tastatur, manchmal in sehr kleiner Schrift. Auf manchen Systemen kann es notwendig sein, die Strg-Taste gedrückt zu halten, wenn Sie PrntScn drücken.
  2. Fügen Sie das Bild in ein Bildprogramm wie MS Paint oder paint.net ein, indem Sie Strg + V drücken oder mit der rechten Maustaste klicken und Einfügen in eine neue Datei wählen.

Für Macs

Es gibt verschiedene Tastenkombinationen, um einen Screenshot zu erstellen.  Die folgenden sind zwei der gebräuchlicheren Methoden.
    • Drücken Sie die Befehlstaste-Umschalttaste-3: Es wird ein Screenshot des gesamten Bildschirms erstellt.
    • Drücken Sie Befehl-Umschalt-4: Damit können Sie einen Teil des Bildschirms auswählen.
