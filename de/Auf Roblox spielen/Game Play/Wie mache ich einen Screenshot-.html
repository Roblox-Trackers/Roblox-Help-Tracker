Wie mache ich einen Screenshot?

Die folgenden Schritte führen Sie durch, wie man ein Bild von dem macht, was auf Ihrem Bildschirm passiert.

Hinweis: Wenn Sie ein Bild an den Kundendienst senden, speichern Sie die Datei im PNG- oder JPG-Format und hängen Sie sie an Ihre E-Mail an.

Einen Screenshot im Spiel erstellen

  • Klicken Sie auf die Schaltfläche Menü in der oberen linken Ecke

  • Klicken Sie auf die Option Aufnehmen in dem Menü das sich öffnet

  • Klicken Sie die Schaltfläche Screenshot machen

Alternativ können Sie die ersten beiden Schritte überspringen und die Druck Taste (PrtScn-Taste) (PrintScreen/Druckbildschirm) auf Ihrer Tastatur drücken.

Wenn Sie das zum ersten Mal machen, erscheint ein Popup-Fenster in dem Sie darüber informiert werden, dass Sie einen Screenshot gemacht haben und wo Sie ihn finden können. Unten rechts ist die Option, diese Nachricht nie wieder anzuzeigen.

Um einen Screenshot Ihres gesamten Bildschirms / der Website zu erstellen

Für Windows

  1. Richten Sie Ihre Aufnahme ein und drücken Sie diese Taste: Druck (PrtScn). Dies befindet sich normalerweise ganz oben auf der Tastatur, manchmal in sehr kleiner Schrift. Auf einigen Systemen kann es notwendig sein, die Ctrl oder Strg-Taste gedrückt zu halten, wenn Sie Druck (PrntScn) drücken.
  2. Fügen Sie das Bild in ein Bildprogramm wie MS Paint oder paint.net ein, indem Sie Strg + V (Ctrl + V für englische Tastatur) verwenden oder rechte Maustaste klicken und in eine neue Datei einfügen.

Für Macs

Bei der Erstellung eines Screenshots stehen mehrere Tastaturkombinationen zur Verfügung.  Die folgenden sind zwei der häufigsten Methoden.

    • Drücken Sie strg-shift (Umschalttaste)-3 (oder ctrl-shift-3): Erstellt einen Screenshot des gesamten Bildschirms.
    • Drücken Sie strg-shift (Umschalttaste)-4 (oder ctrl-shift-4): Ermöglicht Ihnen, einen Teil des Bildschirms auszuwählen.
