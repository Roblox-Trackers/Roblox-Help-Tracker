A|B Tests - Warum gibt es nur für bestimmte Spieler ein neues Feature?

Bei Roblox arbeiten wir ständig daran, das Spielerlebnis zu verbessern. Teil dieses Prozesses ist das Testen neuer Funktionen auf unserer Webseite. Da unser Team diese Tests durchführt, können einige Konten zu unterschiedlichen Zeiten unterschiedliche Funktionen sehen. Wenn Sie sich fragen, wann eine neue Funktion für Sie verfügbar sein könnte, beachten Sie bitte unseren Blog für offizielle Ankündigungen.


Hinweis: Ihr Konto wird zufällig einer Testgruppe zugeordnet. Die Rolle Ihres Kontos in einem Test kann nicht geändert werden.
