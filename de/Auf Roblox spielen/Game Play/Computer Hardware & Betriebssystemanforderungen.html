Computer Hardware & Betriebssystemanforderungen

Die Mindestsystemanforderungen für Roblox sind unten aufgeführt. Wenn beim Spielen Probleme auftreten, müssen Sie möglicherweise Ihren Computer aktualisieren. Lesen sie diesen Hilfsartikel für Systemanforderungen für mobile Geräte.

Betriebssystemanforderungen

  • PC/Windows: Roblox wird unter Windows 7, Windows 8/8.1 oder Windows 10 installiert; Für Windows 8/8.1 müssen Sie Roblox im Desktop-Modus ausführen, da der Metro-Modus (der gekachelte Startbildschirm) derzeit nicht unterstützt wird. Hinweis: Die Roblox-Website unterstützt die folgenden Browser für PC/Windows: Chrome , Firefox , und Microsoft Edge .
  • Mac: Roblox Client wird auf 10.10 (Yosemite) und höher installiert, während Roblox Studio auf Mac OS 10.11 ( El Capitan ) und höher installiert wird.
  • Linux: Roblox wird auf Linux nicht unterstützt.
  • Mobile: Klicken Sie hier für Systemanforderungen für Roblox Mobile
  • Chrome OS: Chrome OS Version 53 oder höher ist erforderlich und Sie müssen Google Play Store zuerst auf Ihrem Gerät aktivieren.

Hinweis: Die Roblox-Videoaufzeichnungsfunktion funktioniert nur unter Windows 7, Windows 8/8.1 und Windows 10. Neu veröffentlichte Betriebssysteme oder vorab veröffentlichte Versionen von Betriebssystemen werden möglicherweise nicht sofort unterstützt.

System-Hardwareanforderungen

  • Grafikkarte: Auf PC/Windows benötigt die Roblox-App DirectX 10 oder eine höhere Funktionsstufe. Für die beste Leistung empfehlen wir entweder einen Computer unter 5 Jahren mit einer entsprechenden Grafikkarte oder einen Laptop unter 3 Jahren mit einer integrierten Grafikkarte.
  • Prozessor: Roblox empfiehlt, dass Sie einen aktuellen Prozessor (2005+) mit einer Taktrate von 1,6 Ghz oder besser haben. Es gab einige Probleme mit älteren AMD-Prozessoren.
  • RAM oder Speicher: Roblox empfiehlt, dass Sie unter Windows 7, Windows 8 oder Windows 10 mindestens 1 GB Arbeitsspeicher haben.
  • Speicherplatz: Roblox empfiehlt, dass Sie mindestens 20 MB Systemspeicher für die Installation von Roblox zur Verfügung haben.
  • Handy: Klicken Sie hier für Systemanforderungen für Roblox Mobile.

Internetanforderungen

Für einen Single-Haushalt empfiehlt Roblox eine Internetverbindung von mindestens 4-8 Mb/s. In größeren Haushalten ist eine größere Bandbreite empfehlenswert.
