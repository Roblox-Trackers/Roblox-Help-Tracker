Roblox Beta-App

Die Roblox-App ändert den Ablauf, wie Spieler Spielen beitreten und wo sie Premium und Robux kaufen können.

Wie installiert man

  • Wenn Ihr Konto Zugriff auf die Roblox Beta-App hat, sehen Sie ein Banner für die App.
  • Klicken Sie im Banner auf die Schaltfläche Jetzt ausprobieren .
  • Es erscheint ein Popup-Fenster mit der Meldung, dass der Roblox Player installiert wird
  • Möglicherweise erscheint ein zweites Popup-Fenster, in dem Sie aufgefordert werden, ein Programm auszuwählen, wählen Sie Roblox öffnen .

So verlassen Sie die Beta

Wenn Sie sich von der Roblox Beta-App abmelden möchten, befolgen Sie die nachstehenden Anweisungen:

  • Öffnen Sie die Beta-App
  • Klicken Sie auf Mehr und gehen Sie dann zu den Einstellungen
  • Am unteren Rand des Einstellungsbildschirms sehen Sie die Option zum Deaktivieren der Beta-App.

  • Wenn Sie auf die Option „ Beta deaktivieren “ klicken, wird beim Verlassen eine Roblox-Umfrage geöffnet. Bitte teilen Sie uns Ihre Meinung mit.

Hinweis: Wenn Sie auf das Banner auf der Startseite klicken, gelangen Sie zurück zur Beta-App.

Beta-App-Unterschiede

Während Sie mit dem größten Teil der Benutzeroberfläche vertraut sein können, wenn Sie Roblox normal gespielt haben, gibt es einige wichtige Unterschiede, die Sie beachten sollten, wenn Sie mit der Beta-Version spielen.


Hauptmenü

Sie werden sehen, dass mehrere Funktionen direkt im Hauptmenü der Beta-App verfügbar sind, wie z. B. die Startseite, die Spieleseite, der Avatar-Editor und das Chat-Fenster. Sie können jede davon auf der linken Seite öffnen, wenn Sie die App zum ersten Mal öffnen.

Spiel verlassen (L)

Diese Schaltfläche veranlasst den Spieler, das Spiel zu verlassen, wodurch der Spieler die Verbindung zu dem Server, auf dem er sich befand, trennt. Nach dem Klicken auf die Schaltfläche gibt es eine weitere Nachricht, um zu bestätigen, ob sie gehen möchten. Wenn Sie Spiele auf diese Weise verlassen, wird das Spielfenster nicht geschlossen, sondern Sie kehren stattdessen zum Hauptmenü zurück.

Um das Spiel sofort vollständig zu beenden, können Sie oben rechts im Spielfenster auf das X klicken.
