Wie kaufe und konfiguriere ich ein privaten/VIP-Server?

Wenn der Entwickler des Erlebnisses private Server für sein Erlebnis aktiviert hat, können Sie Robux verwenden, um einen privaten Server zu kaufen, den Sie mit Ihren eingeladenen Freunden nutzen können.

Wie erstelle und ändere ich meinen Server?

Wie erstelle ich einen?

  • Klicken Sie auf der Detailseite eines Erlebnisses auf die Registerkarte Server .
  • Wenn diese Funktion aktiviert wurde, wird ein Abschnitt mit dem Titel Private Server angezeigt. Hier finden Sie alle privaten Server, die Sie erstellt haben oder zu denen Sie gehören. Wenn Sie welche sehen, können Sie sofort spielen!
  • Zum Erstellen eines neuen Servers, klicken Sie auf die Schaltfläche Privaten Server erstellen .
  • Geben Sie Ihrem neuen Server einen Namen. Lesen Sie den Abonnementvertrag. Nachdem Sie alle Informationen auf der Box gelesen und den gewünschten Namen ausgewählt haben, können Sie den Kauf abschließen, indem Sie auf Jetzt kaufen klicken.
  • Nach dem Kauf können Sie den Server sofort oder später konfigurieren.

Wie konfiguriere ich meinen VIP-Server?

  • Über das Kaufbenachrichtigungsfenster oder durch Auswahl von Konfigurieren über die Menüschaltfläche (sieht aus wie drei Punkte) für den Server auf der Registerkarte Server gelangen Sie zur Konfigurationsseite.
  • Sie können den Server umbenennen, anpassen, wer darauf spielen kann, einschließlich bis zu 50 einzelner Benutzer, oder Ihre Freundesliste zulassen, und sogar einen Link zu Ihrem Server erstellen.
  • Sie können den Zugriff auf den Server auch ein- und ausschalten sowie Ihr Abonnement für den Server kündigen.

Wie funktionieren Private Server Abonnements und wie viel kosten sie?

Wie funktionieren die Abonnements für private Server?

  • Die Preise für private Server werden vom Entwickler des Erlebnisses festgelegt.
  • Dies wird bei der ersten Bestellung in Rechnung gestellt und wird jeden Monat erneut in Rechnung gestellt, damit es aktiv bleibt. Die monatliche Verlängerungsgebühr wird automatisch aus Ihrem Robux-Konto abgebucht, so dass Sie sich nicht darum sorgen müssen, es jeden Monat zu erneuern.
  • Wenn der Server nicht verlängert werden soll, kann dies jederzeit vor dem Erneuerungsdatum erfolgen. Um den VIP-Server abzubrechen, lesen Sie den Abschnitt Wie kann ich meinen VIP-Server kündigen? , der weiter unten zu finden ist.

Wie storniere ich meinen Server?

  • Navigiere zu deinem VIP-Server für das Erlebnis. Klicken Sie auf die Schaltfläche Hauptmenü (sieht wie drei Punkte aus) ganz rechts im Abschnitt.
  • Klicke auf Konfiguration .
  • Um die monatliche Gebühr zu stornieren, klicken Sie auf die grüne Schaltfläche Aktiv neben dem Abonnementstatus .
  • Sie müssen im Bestätigungsfenster auf Zahlungen stornieren klicken, damit das Abonnement storniert wird.

Hinweis: Sie werden Ihren Server für den Rest des Monats, für den Sie bereits bezahlt haben, weiterhin nutzen, aber er läuft nach Ablauf dieser Zeit ab.

Was passiert, wenn ich am Verlängerungsdatum nicht genug Robux habe?

In diesem Fall läuft der Server ab und kann nicht mehr verwendet werden. Er kann jedoch jederzeit wieder erworben werden.

Wie stelle ich meinen Server auf inaktiv?

  • Navigiere zu deinem privaten Server für das Erlebnis. Klicken Sie auf die Schaltfläche Hauptmenü (sieht wie drei Punkte aus) ganz rechts im Abschnitt.
  • Klicke auf Konfigurieren .
  • Um den Server vorübergehend inaktiv zu halten, klicken Sie auf die grüne Schaltfläche Aktive neben dem VIP Server Status .
  • Klicken Sie im Bestätigungsfenster auf Deaktivieren , um den Server zu deaktivieren.
  • Sie können es jederzeit erneut anklicken, um es zu reaktivieren. Bitte beachten Sie jedoch, dass die Verlängerungszeit für diesen Zeitraum der Inaktivität nicht verlängert wird, sondern weiterhin zum ursprünglich geplanten Datum verlängert wird oder abläuft.
