Wie kann ich eine Konto-PIN hinzufügen oder entfernen?

Du hast die Möglichkeit, eine vierstellige PIN hinzuzufügen, um deine Einstellungen zu sichern. Sobald eine PIN hinzugefügt wurde, wird sie benötigt, um Änderungen an den Kontoeinstellungen vorzunehmen. Diese Funktion schützt dein Passwort, deine E-Mail-Adresse und deine Datenschutzeinstellungen.

Wie kann ich eine Konto-PIN hinzufügen?

  1. Logge dich in dein Konto ein
    (Wenn du dich nicht einloggen kannst, versuche diese Schritte, um dein Passwort wiederzufinden .)
  2. Öffne die Kontoeinstellungen
    • Browser - suche das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps - finde das Symbol mit den drei Punkten für Mehr
  3. Wähle die Registerkarte Kindersicherung
  4. Umschalttaste zum Einschalten der PIN
  5. Erstelle und bestätige deine neue PIN

Gib niemals deine PIN weiter - sie ist sogar noch wichtiger als dein Passwort.

Wie kann ich eine Konto-PIN entfernen?

  1. Logge dich in dein Konto ein
    (Wenn du dich nicht einloggen kannst, versuche diese Schritte, um dein Passwort wiederzufinden .)
  2. Öffne die Kontoeinstellungen
    • Browser - suche das Zahnrad-Symbol in der oberen rechten Ecke der Website
    • Mobile Apps - finde das Symbol mit den drei Punkten für Mehr
  3. Wähle die Registerkarte Kindersicherung
  4. Umschalttaste zum Ausschalten der PIN
  5. Gib die aktuelle PIN ein und wähle Entsperren

Wie kann ich eine PIN ändern?

Um eine PIN zu ändern, befolge zuerst die Wie kann ich eine Konto-PIN entfernen Schritte. Folge dann den Wie kann ich eine Konto-PIN hinzufügen Schritten, um eine neue PIN zu erstellen.
