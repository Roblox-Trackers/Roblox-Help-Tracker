2-Schritt-Verifizierung zu deinem Konto hinzufügen

Die 2-Schritt-Verifizierung ist eine zusätzliche Sicherheitsebene für dein Konto. Dies ist besonders hilfreich, um zu verhindern, dass Angreifer Zugriff auf dein Konto erhalten, falls dein Passwort gestohlen wird.

2-Schritt-Verifizierungsmethoden

Roblox bietet verschiedene Methoden zur 2-Schritt-Verifizierung an. Derzeit kannst du jeweils nur eine Methode zur 2-Schritt-Verifizierung auswählen.

Zur Aktivierung eine 2-Schritt-Verifizierung deines Kontos:

  1. Melde dich in deinem Roblox-Konto an
    • Wenn du dich nicht anmelden kannst, versuche diese Schritte zum Wiederherstellen deines Passworts
  2. Gehe zu Kontoeinstellungen
    • Browser : Suche das Zahnradsymbol in der oberen rechten Ecke der Website
    • Mobile App : Suche das Symbol mit den drei Punkten und tippe auf das Zahnradsymbol
  3. Gehe zur Registerkarte Sicherheit
  4. Aktiviere den Schalter für die gewünschte Methode zur 2-Schritt-Verifizierung und befolge die Anweisungen auf dem Bildschirm. Die Anweisungen variieren je nach deiner Wahl der 2-Schritt-Verifizierung
    • WICHTIG : Du musst eine bestätigte E-Mail-Adresse in deinem Konto haben, um die 2-Schritt-Verifizierung zu aktivieren

2-Schritt-Verifizierung über die E-Mail-Adresse einrichten

Sobald du die 2-Schritt-Verifizierung per E-Mail aktiviert hast, wirst du jedes Mal, wenn du dich mit deinem Passwort anmelden, zur Eingabe eines Codes aufgefordert.

Du findest den Code im Posteingang der verifizierten E-Mail-Adresse deines Roblox-Kontos. Sobald du den Code auf deinem Roblox-Bildschirm eingegeben hast, wirst du erfolgreich angemeldet.

Bitte beachte, dass deine 2-Schritt-Verifizierung umgangen wird, wenn du dich per E-Mail-OTP oder Passkey anmeldest.

Einrichten der 2-Schritt-Verifizierung durch eine Authentifizierungs-App

Um die 2-Schritt-Verifizierung per Authenticator-App zu aktivieren, musst du über eine Authenticator-App auf deinem Mobilgerät verfügen. Wenn du keine Authenticator-App hast, kannst du eine aus dem App Store herunterladen. Einige Apps, die du auswählen kannst, sind Google Authenticator, Microsoft Authenticator, Duo und Twilio Authy.

Sobald du die 2-Schritt-Verifizierung durch die Authenticator-App aktiviert hast, wirst du jedes Mal, wenn du dich mit deinem Passwort oder mit E-Mail-OTP anmeldest, zur Eingabe eines Codes aufgefordert.

Du findest den Code in der von dir gewählten Authenticator-App auf deinem Mobilgerät. Sobald du den Code auf deinem Roblox-Bildschirm eingegeben hast, wirst du erfolgreich angemeldet.

Bitte beachte, dass deine 2-Schritt-Verifizierung umgangen wird, wenn du dich über einen Passkey anmeldest.

2-Schritt-Verifizierung mit Sicherheitsschlüsseln einrichten

Um die 2-Schritt-Verifizierung durch Sicherheitsschlüssel zu aktivieren, muss in deinem Konto auch die 2-Schritt-Verifizierung durch die Authenticator-App aktiviert sein. Dies liegt daran, dass Sicherheitsschlüssel nur von Webbrowsern unterstützt werden.

Dein Sicherheitsschlüssel kann ein physischer Schlüssel wie Yubikey 5, Google Titan Security Key oder eine biometrische Methode sein, die von der Hardware deines Geräts verwaltet wird, wie zum Beispiel Gesichtserkennung oder Fingerabdruckscanner. Deine Sicherheitsschlüsseloption hängt davon ab, über welchen Browser, welches Betriebssystem und welches Gerät du dich anmeldest.

Bitte beachte, dass deine 2-Schritt-Verifizierung umgangen wird, wenn du dich über einen Passkey anmeldest.

2-Schritt-Verifizierung durch geräteübergreifende Eingabeaufforderungen

Hinweis : Dies ist eine experimentelle Funktion und steht noch nicht allen Benutzern zur Verfügung.

Als Benutzer hast du nicht die Möglichkeit, die 2-Schritt-Verifizierung durch geräteübergreifende Eingabeaufforderungen explizit zu aktivieren. Diese Eingabeaufforderungen werden angezeigt, wenn in deinem Konto keine der oben aufgeführten Methoden zur 2-Schritt-Verifizierung aktiviert ist und wenn Roblox einen verdächtigen Anmeldeversuch bei deinem Konto feststellt.

Wenn du eine geräteübergreifende 2-Schritt-Verifizierung erhalten, wirst du aufgefordert, ein Mobilgerät zu verwenden, auf dem du bereits in deinem Roblox-Konto angemeldet bist, um den Anmeldeversuch zu genehmigen oder abzulehnen. Du hast die Möglichkeit, Details zum Anmeldeversuch wie Gerätetyp, regionaler Standort, IP-Adresse, Datum und Uhrzeit zu überprüfen.

Eine andere Möglichkeit zur Überprüfung deines Anmeldeversuchs wählen

Wenn du mit einer geräteübergreifenden 2-Schritt-Verifizierung konfrontiert wirst, hast du die Möglichkeit, deine Anmeldung auf andere Weise zu überprüfen.

Einen Code per E-Mail senden

Wenn dein Konto über eine bestätigte E-Mail-Adresse verfügt, wird die Option „Einen Code per E-Mail senden“ angezeigt. Wenn du diese Option wählst, wird ein eindeutiger Code an deine E-Mail-Adresse gesendet. Gib diesen Code auf deinem Roblox-Bildschirm ein, um dich erfolgreich anzumelden. Denke daran, diesen Code nicht an Dritte weiterzugeben.

Wiederherstellungscode

Du hast auch die Möglichkeit, deinen gespeicherten Wiederherstellungscode einzugeben, um dich erfolgreich anzumelden. Denke daran, diesen Code nicht an Dritte weiterzugeben.

Kundendienst kontaktieren

Wenn du keine Optionen siehst, mit denen du dich erfolgreich anmelden kannst, wende dich an den Kundendienst .

Wir empfehlen dir dringend, deinem Konto eine verifizierte E-Mail-Adresse hinzuzufügen. Dies hilft dir, wenn du deinen Anmeldeversuch mit einem E-Mail-Code anstelle der geräteübergreifenden 2-Schritt-Verifizierung verifizieren möchtest.

Anmeldeversuch abgelehnt

Wenn du eine geräteübergreifende 2-Schritt-Verifizierung erhältst, aber keinen Anmeldeversuch unternommen hast, lehne den Anmeldeversuch ab.

Dies bedeutet, dass möglicherweise jemand dein Passwort hat und es verwendet, um sich bei deinem Konto anzumelden. Wir empfehlen dir, dein Passwort zu ändern um dein Konto zu schützen.

Anmeldeversuch abgelaufen

Wenn du zu lange mit der Genehmigung oder Ablehnung des Anmeldeversuchs wartest, verfällt dieser. Versuche einfach den Anmeldeversuch erneut.

Wiederherstellungscodes abrufen

  1. Melde dich in deinem Konto an.
  2. Gehe zu Kontoeinstellungen .
    • Webbrowser : Finde das Zahnradsymbol in der oberen rechten Ecke der Website.
    • Mobile Apps : Suche nach dem Symbol mit den drei Punkten für Mehr .
  3. Wähle die Registerkarte Sicherheit .
  4. Klicke im Abschnitt Wiederherstellungscodes auf Generieren.
    • Webbrowser : Dieser befindet sich auf der rechten Seite der Seite
    • Mobile Apps : Diese findest du im Abschnitt Wiederherstellungscodes
  5. Gib dein Kontopasswort als Teil des Einrichtungsprozesses ein.
  6. Stelle sicher, dass du die Codes an einem sicheren Ort speicherst, weil du sie nur einmal sehen kannst.

Jeder Code kann nur einmal verwendet werden. Wenn du deine Wiederherstellungscodes verlierst, kannst du diesen Vorgang wiederholen, um neue Codes zu erstellen. Alle zuvor erstellten Wiederherstellungscodes werden deaktiviert, wenn du neue Codes erstellst.
