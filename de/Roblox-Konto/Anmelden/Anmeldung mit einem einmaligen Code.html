Anmeldung mit einem einmaligen Code

Mit einem einmaligen Code kannst du dich anmelden, ohne deinen Benutzernamen und dein Passwort einzugeben. Momentan sind einmalige Codes nur für einige Nutzer:innen verfügbar.

Hinweis: Um diese Funktion nutzen zu können, musst du eine verifizierte E-Mail-Adresse mit deinem Konto assoziiert haben. Auf der Seite mit den Kontoinformationen kannst du deine E-Mail-Adresse hinzufügen und verifizieren. In diesem Hilfeartikel kannst du nachlesen, wie man die E-Mail-Adresse des Kontos verifiziert.

Wie meldet man sich mit einem einmaligen Code an

  1. Drücke auf der Anmeldeseite die Schaltfläche Einmaligen Code per E-Mail senden .
  2. Gib die E-Mail-Adresse ein, die für dein Konto verifiziert ist. Du erhälts per E-Mail einen sechsstelligen, einmaligen Code.
  3. Gib den einmaligen Code ein, um dich anzumelden (wenn deine E-Mail auf mehreren Konten verifiziert ist, kannst du eines auswählen).

Ich habe keinen einmaligen Code erhalten

Versuche diese Schritte, wenn du keine E-Mail mit einem einmaligen Code erhalten hast.

  • Überprüfe deinen Spam-/Junk-Ordner.
  • Klicke auf die Schaltfläche "Erneut senden" und überprüfe dann erneut deinen Posteingang.
  • Vergewissere dich, dass du deine E-Mail-Adresse genau so eingegeben hast, wie sie in deinem Konto angegeben ist.
  • Stelle sicher, dass deine E-Mail-Adresse verifiziert ist. In diesem Hilfeartikel kannst du nachlesen, wie man die E-Mail-Adresse des Kontos verifiziert.

Wenn keiner dieser Schritte funktioniert, melde dich bitte mit deinem Passwort an. Möglicherweise musst du eine Weile warten, bis einmalige Codes für deine E-Mail wieder funktionieren.

Ich habe einen einmaligen Code erhalten, aber er funktioniert nicht

Versuche diese Schritte, wenn dein einmaliger Code nicht funktioniert.

  1. Stelle sicher, dass du den einmaligen Code auf demselben Gerät eingibst, auf dem du ihn angefordert hast.
  2. Drücke die Schaltfläche Erneut senden und versuche es noch einmal mit einem neuen Code.

Wenn keiner dieser Schritte funktioniert, melde dich bitte mit deinem Passwort an. Möglicherweise musst du eine Weile warten, bis einmalige Codes für deine E-Mail wieder funktionieren.

Sicherheitstipps

  • Gib einmalige Codes niemals an andere weiter. Roblox wird dich niemals kontaktieren, um den einmaligen Code anzufordern.
  • Wenn du eine E-Mail mit einem einmaligen Code erhältst, ohne diesen angefordert zu haben, kannst du sie getrost ignorieren.
  • Wenn du mehrere unerwünschte E-Mails mit einmaligen Codes erhältst, raten wir dir dringend, die E-Mail-Adresse in deinem Konto zu ändern und die 2-Schritt-Verifizierung durch die Authenticator App einzurichten .
